## §0 — WHAT IS MODALITY

```
Modality = ENCODING CHANNEL — how the brain stores information into chunks.

Chunk system (Core-Software.md §4) = sole substrate.
Chunks are NOT stored in a single channel — they are stored in MANY channels SIMULTANEOUSLY.
Each channel = 1 modality.

EXAMPLE — chunk "dog":
  Visual:    image of a dog (4 legs, tail, fur)
  Auditory:  the sound of barking
  Somatic:   sensation of fur against the hand
  Motor:     the movement of patting the dog's head
  Emotional: fear (if ever bitten) or joy (if raised since puppyhood)
  Communication: label "dog"

→ 1 chunk = COMBINATION of many modalities
→ Deeper chunk = more modalities
→ Shallow chunk = fewer modalities (often verbal only)

MODALITY IN V7.8:

  Hardware Map (§1.2):
    Zone B = CORTICAL MODALITY — PFC trainable
    Visual cortex, Auditory cortex, Somatosensory, Motor, Insula,...
    → Modality channels = physical brain regions (T1 hardware)

  Software Map (§1.3):
    Chunks compile WITHIN modalities (T2 software)
    → Same hardware (zone B), different compiled chunks = different skills

  Observation:
    Modality Balance = hardware profile (Core-Software.md §6.3)
    → Affects how chunks compile (multi-modal richness)
    → Affects how PFC observes (which modality is dominant)
    → Brain-wide property, NOT exclusive to PFC or Unconscious

KEY DISTINCTIONS:
  → Modality channels = T1 (hardware, brain regions, change slowly)
  → Chunks in modalities = T2 (software, change through experience)
  → Hardware sets the RANGE → chunks select POSITION within that range
```

## §1 — 6 MODALITIES (5 Experience + 1 Communication)

```
NOT 6 equal modalities — but 2 FUNDAMENTALLY DIFFERENT types:

  5 EXPERIENCE MODALITIES (encode DIRECT EXPERIENCE — primary data):
    ① Visual:    images, shapes, spatial information (Visual cortex V1→IT, FFA)
    ② Auditory:  sounds, voice, rhythm               (Auditory cortex, Heschl's)
    ③ Somatic:   body sensations, gut feeling        (Insula + Somatosensory)
    ④ Motor:     movement, muscle memory             (Motor cortex + Cerebellum + BG)
    ⑤ Emotional: emotions, fear, joy, love           (Amygdala + Insula + Limbic)

    = Present in ALL neural organisms (to varying degrees)
    = Encodes experience directly — body senses → brain stores

  1 COMMUNICATION MODALITY (encode LABELS + TRANSFER — INDEX):
    ⑥ Verbal / Gesture / Writing / Code /...
    = Does NOT encode experience → encodes REFERENCES to experience
    = Only humans have it at this level of complexity (language, writing, symbols)

    Brain regions: Broca's (production) + Wernicke's (comprehension) + Arcuate fasciculus
    🟢 Deaf people using sign language: ALSO use Broca + Wernicke
       (Petitto et al. 2000) → brain regions for LANGUAGE, not just for speech

    4 FUNCTIONS:
      LABEL:    assigns a name to the multi-modal chunk → PFC holds it more easily (WM compression)
      TRANSFER: transmits knowledge to others WITHOUT requiring direct experience
      COMBINE:  combines labels → expands domain QUICKLY without needing experience
      SEQUENCE: processes SEQUENTIAL + HIERARCHICAL structure (Broca's, see below)


WHY SEPARATE 2 TYPES:

  🟢 Strongest evidence — Fedorenko et al. (MIT, 2011-2024):
    The brain has 2 DISTINCT networks (dissociable):
      Language network (Broca + Wernicke): processes LANGUAGE
      Multiple demand network (PFC + parietal): processes THINKING / REASONING
    → Language damaged: cannot speak, can STILL calculate, solve puzzles ✅
    → Multiple demand damaged: cannot reason, can STILL speak fluently ✅
    → THINKING ≠ LANGUAGE — 2 SEPARATE systems

  Additional evidence:
    → Children 0-2: no verbal yet → STILL chain (160+ strategies)
    → Aphasia: loss of language → can STILL calculate
    → Meditation stops inner voice: STILL thinking

  → Verbal = COMMENTATOR: describes the match, does NOT play
  → "Thinking in words" = verbal NARRATING results the Unconscious already generated

  BUT — Broca's has 1 SMALL function related to thinking:
    🟢 Broca's area ALSO processes SEQUENTIAL / HIERARCHICAL structure:
      Action sequences (Fadiga et al.)
      Musical structure (musical syntax — Maess et al. 2001)
      → Broca's does NOT reason → but DOES process SEQUENCE
      → Language = 1 type of sequence (a sentence = an ordered sequence of words)


COMMUNICATION MODALITY — POWERFUL DESPITE NOT BEING A "REASONER":

  ✅ TRANSFER:  "fire is hot" → transmits knowledge WITHOUT needing to be burned
  ✅ COMBINE:   "forest fire" = combined labels → NEW schema nobody has experienced
  ✅ WM BOOST:  label compresses chunk → PFC holds MORE → workspace WIDER
  ✅ SEQUENCE:  arranges steps, plans order, hierarchical structure

  ❌ BUT encoding = SHALLOW (labels only, not experience):
    "Knowing fire is hot" (verbal) ≠ "FEELING fire is hot" (somatic + emotional)
    Verbal schema = PREVIEW (right direction, low fidelity)
    Experience schema = CONFIRM (high fidelity, body responds for real)
    "Knows but can't do it" = verbal HAS labeled → body has NOT experienced


═══════════════════════════════════════════════════════
§1.1 — 5 Experience Modalities — Detail
═══════════════════════════════════════════════════════

① VISUAL — Images
  Brain regions: V1-V5, Fusiform (faces), Parietal (spatial)
  Encodes: images, colors, shapes, spatial relations, motion
  2 sub-types:
    Visual-sequential: "flow chart in the mind" → VERTICAL
    Visual-spatial:    "3D model in the mind" → HORIZONTAL (cross-domain structural match)
  Cross-domain: YES but FILTERED (requires structural similarity)
    "Atom looks like solar system" = YES → match ✅
    "Team conflict is like a code bug" = NO structural similarity → MISS
  ⚠️ Strongest in children age 4-7 → diminishes as verbal takes over (schooling)
  ⚠️ Aphantasia ~2-5% have no visual imagination

② AUDITORY — Sound
  Brain regions: Primary auditory cortex, Heschl's gyrus, Wernicke (speech)
  Encodes: sounds, voice, music, rhythm, tone
  2 sub-types:
    Inner voice:      self-talk in the mind → VERTICAL (sequential self-talk)
    Pattern listening: hears PATTERN → HORIZONTAL (rhythm, melody across contexts)
  Cross-domain: mild (pattern match via rhythm)
  Strengths: continuous (ears always open), strong emotional power (music → body responds)
  → Deep dive on 6 unique hardware properties: Auditory-Hardware.md

③ SOMATIC — Body Sensations
  Brain regions: Insula (interoception) + Somatosensory + body-wide receptors
  Encodes: body sensations, gut feeling, tension, warmth, pain
  Characteristics:
    → NON-VERBAL: "can feel but can't articulate"
    → CROSS-DOMAIN: body detects patterns from ANY domain
    → FAST: body responds ~50ms vs PFC ~200ms
    → HONEST: hard to fake a somatic response
  Cross-domain: EXTREMELY HORIZONTAL (unfiltered body state match)
    Body doesn't encode "team" or "code" → encodes SENSATION
    Same sensation = "SAME" → regardless of domain
    = Universal matching key → cross-domain insight engine
  ⚠️ vmPFC = bridge somatic → conscious. Weak vmPFC → alexithymia
  ⚠️ Somatic → verbal is difficult: language LACKS WORDS for body states
    Insula lies DEEP → connection to Broca/Wernicke is LONG + WEAK

④ MOTOR — Movement
  Brain regions: Motor cortex + Premotor + Cerebellum (timing) + BG (habit)
  Encodes: movements, skills, procedures, muscle memory
  Characteristics:
    → PROCEDURAL: "hands know how, mind can't explain"
    → COMPILED: enough repetition → BG takes over → automatic → PFC freed
    → DURABLE: cycling, swimming, typing → not forgotten for years
  Cross-domain: VERY LOW (domain-specific, little transfer)
  Direction: VERTICAL (sequential skills within domain → goes DEEP)

⑤ EMOTIONAL — Emotions
  Brain regions: Amygdala (threat/reward) + Insula (feeling state) + Limbic
  Encodes: emotional associations, fear, joy, disgust, love
  Characteristics:
    → FASTEST: Amygdala ~12ms (fastest in the brain)
    → STRONGEST: emotional encoding = strong + durable (flashbulb)
    → SOCIAL-DOMINANT: primarily encodes social/relational patterns
    → CONTAGIOUS: detects + mirrors others' emotions
  Cross-domain: HORIZONTAL favoring SOCIAL (feeling match across social contexts)
  ⚠️ Distinguishing from somatic:
    Somatic = body PATTERN from any domain (can be non-emotional)
    Emotional = FEELING VALENCE (always has +/−)
    Overlap: emotional HAS somatic component (fear → heart pounds)
    But: somatic CAN be non-emotional ("this structure feels similar")
  ⚠️ Trauma = EXTREMELY STRONG emotional encoding → doesn't fade → intrusive
```

## §2 — HARDWARE BASIS (Sensor → Cortex)

```
⭐ REFRAME: Sensor → Modality mapping is NOT 1:1. It is MANY-TO-MANY.

  1 SENSOR → MANY cortical areas:
    Eyes → Visual cortex (images) + Amygdala (threat, ~12ms) + SCN (circadian)
    Ears → Auditory cortex + Amygdala (emotional) + Wernicke (speech)
    Skin → Somatosensory (where/what) + Insula (pleasant/unpleasant)
    Nose → Olfactory cortex + Amygdala + Hippocampus (BYPASSES thalamus!)

  1 CORTICAL AREA ← MANY sensors:
    Amygdala ← eyes + ears + skin + nose + visceral + hormones + mirror

  → "Modality" = PROCESSING AREA, not "input source"
  → Visual modality = Visual cortex processing (receives mainly from eyes + top-down PFC)

HARDWARE:
  Most input passes through THALAMUS (gateway + gate) before reaching cortex.
  Exceptions: olfaction bypasses thalamus, amygdala subcortical (~12ms).
  PFC controls attention via TRN (Thalamic Reticular Nucleus) — opens/closes gate.
  Entire neocortex = SAME 6-LAYER STRUCTURE (Mountcastle 1957),
  differing in wiring + layer thickness + receptor density.

  Hardware Map (Core-Software.md §1.2):
    Zone B = CORTICAL MODALITY — PFC trainable
    Includes: Visual, Auditory, Language, Somatosensory, Motor,
              Cerebellum, Insula, Parietal, Temporal
    → PFC (Zone A) reaches Zone B: STRONG (Layer 2/3 direct, top-down bias)

BINDING — How modalities sync (5 concurrent mechanisms):
  ① Gamma oscillation ~40Hz (fire in same phase = same object — Singer 1995)
  ② Multisensory neurons (hardware, present from birth — Stein & Meredith 1993)
  ③ Convergence zones (Parietal + STS + Insula — Damasio 1989)
  ④ Amygdala affective tagging (emotional → binds strongly — McGaugh 2004)
  ⑤ DMN scaffolding (long-range connectivity — Doria 2010)

  → NO single binder. Binding = emergent from 5 mechanisms.
  → Detail: Neural-Architecture.md

🟢 Many-to-many mapping: established neuroanatomy
🟢 Thalamic gating: established (TRN, Sherman & Guillery)
🟢 6-layer cortex: Mountcastle 1957
🟢 Binding mechanisms: Singer 1995, Stein 1993, Damasio 1989, McGaugh 2004
🟡 "Modality = processing area" framing = framework synthesis
```

## §3 — CHUNK × MODALITY (Depth = Modality Count)

```
CHUNK DEPTH = NUMBER OF MODALITIES THE CHUNK IS ENCODED IN.

This is NOT a metaphor — it is LITERAL multi-modal encoding:
  A chunk stored in MANY brain regions = many modalities = DEEPER chunk.

  1 modality  → ★☆☆☆☆ SURFACE
    "Water boils at 100°C" (verbal only)
    Learn: read once. Forget: quickly. Change: change the verbal → done.

  2 modalities → ★★☆☆☆
    "Red light means stop" (verbal + visual)
    Learn: see + hear. Forget: slower. Change: requires 2 modalities.

  3 modalities → ★★★☆☆
    "Guitar Am chord" (motor + auditory + visual)
    Learn: look + hear + practice with hands. Forget: fairly slowly.

  4 modalities → ★★★★☆
    "Fear of dogs" (visual + auditory + somatic + emotional)
    Learn: being bitten once = enough. Forget: very slowly.

  5-6 modalities → ★★★★★ DEEP
    "I am worthless" (ALL modalities)
    Learn: years of experience. Forget: near-impossible. Change: years of therapy.


CHUNK DEPTH × FRAMEWORK DYNAMICS:

  COMPILE (Chunk.md v2.0 §2):
    Multi-modal encoding = 1 of the 5 compile mechanisms
    → Chunks fire in MANY brain regions simultaneously → bind strongly
    → Emotional peak + multi-modal = compile FASTEST (1 time = enough)

  RESISTANCE TO CHANGE:
    Fix 1 modality → the other 4-5 modalities STILL hold the old version → pull back
    → = Why therapy hits "I know, but I still act the same way":
      verbal fixed (1 modality), somatic + emotional not yet (3-4 modalities)
    → Fix: repair ENOUGH modalities to form a majority → behavior changes

  FEELING THRESHOLD (Core-Software.md §5):
    magnitude × clarity → PFC-observable
    → Deeper chunk (more modalities) = STRONGER signal
    → Expert domain (high clarity) = detect more SUBTLE signals
    → Modality richness DIRECTLY affects the feeling threshold

  BODY-FEEDBACK:
    Chunks stored in somatic + emotional modalities → body CAN EVALUATE
    Chunks stored only verbally → body CANNOT evaluate → no body-confirm
    → Verbal-only chunks = "pseudo-explicit" (Somatic-Articulation-Loop §1.3)
    → Body-confirmed chunks = "genuine explicit"

  LABEL ≠ 6TH MODALITY:
    🟡 Label = retrieval path, NOT modality (Chunk.md §6 Label-As-Handle)
    Label "dog" = INDEX into the multi-modal chunk {visual+auditory+somatic+...}
    Label does NOT contain experience → contains REFERENCE to experience
    → WM compression: 1 label = 1 slot → holds MORE chunks than raw experience

🟢 Multi-modal encoding strengthens memory: Shams & Seitz 2008
🟢 Emotional encoding = strongest: McGaugh 2004
🟢 Resistance to change via modalities: implicit learning research
🟡 Depth = modality count framing = framework synthesis
🟡 "Pseudo-explicit vs genuine explicit" = framework extension (Somatic-Articulation-Loop §1.3)
```

## §4 — MODALITY INTERACTION (3 Types)

```
When a chunk is stored in MANY modalities, those modalities can:

① REINFORCEMENT (same direction → chunk becomes STRONGER):

  Verbal "dogs are dangerous" + Visual of a snarling dog + Emotional fear
  = 3 modalities all saying "DANGEROUS" → EXTREMELY STRONG chunk

  → Multi-sensory learning is effective BECAUSE: many modalities reinforce each other
  → Why experience > lecture: experience = multi-modal, lecture = verbal only

② CONFLICT (different directions → "knows but doesn't act"):

  Verbal "dogs are friendly" (CBT has corrected this)
  BUT: Emotional STILL fears + Somatic STILL tenses + Visual STILL sees threat
  = 1 modality says A, 3 say B → B WINS (stronger signal wins)

  → "Knows but doesn't change" = MODALITY CONFLICT
  → Verbal (1) vs Somatic+Emotional+Motor (3-4) → the majority wins
  → Fix: repair ENOUGH modalities to form a majority → behavior changes
  → = Body-Feedback dissonance (Core-Software §4):
    body evaluates conflict → fires DISSONANCE signal

③ COMPENSATION (1 modality compensates for a weak modality):

  Aphantasia (visual = 0) → compensates with verbal + somatic
  Deafness (auditory = 0) → compensates with visual + somatic
  → Neuroplasticity: resources reallocate to the available modality
  → = No loss of ability, just a change of CHANNEL


⭐ MODALITY CONFLICT × THERAPY (preview §7):

  "Therapy isn't working" MAY = right therapy, WRONG modality target:
    CBT targets verbal → verbal-only chunk → effective ✅
    CBT targets somatic+emotional chunk → verbal fixed, body NOT YET → ❌

🟢 Multi-sensory integration: Shams & Seitz 2008
🟢 Modality conflict in therapy: exposure therapy research
🟢 Compensation/neuroplasticity: Merabet & Pascual-Leone 2010
🟡 "Modality majority wins" = framework synthesis
```

## §5 — MODALITY × THINKING DIRECTION (Vertical / Horizontal)

```
Dominant modality → THINKING DIRECTION → DEVELOPMENT DIRECTION:

  FULLY VERTICAL ←———————————————————→ FULLY HORIZONTAL
  Verbal    Motor   Visual-seq  Auditory  Emotional  Visual-spatial  Somatic

  VERTICAL (specialist — deep within 1 domain):
    Verbal + Motor dominant → chains SEQUENTIALLY within domain
    → Step 1 → step 2 → step 3 → drills DEEP
    → School trains verbal 12-16 years → majority = vertical

  HORIZONTAL (improviser — broad across many domains):
    Somatic + Visual-spatial dominant → pattern matches CROSS-DOMAIN
    → "This FEELS like that" (somatic)
    → "This LOOKS like that" (visual-spatial)
    → Cross-domain insight, framework building, intuition


═══════════════════════════════════════════════════════
§5.1 — Concept-First vs Label-First
═══════════════════════════════════════════════════════

Depending on somatic vs verbal dominance, the ORDER of understanding DIFFERS:

SOMATIC-DOMINANT (concept first, label second):
  ① Perceives the concept FIRST (body/pattern)
  ② Label = found LATER, or not needed
  ③ Wrong label → body says "doesn't fit" → searches for another
  ④ Many concepts have NO label → understands but hard to EXPLAIN
  → = Somatic-Articulation Loop (Somatic-Articulation-Loop §2): body knows → words come after

VERBAL-DOMINANT (label first, concept through label):
  ① Receives the label FIRST (word, definition)
  ② Understands concept VIA label (label = gateway)
  ③ Wrong label → concept is also wrong → must fix label first
  ④ Every concept HAS a label → easy to explain, but misses nuance

BOTH work. They only differ in order. Everyone has both, just in different ratios.


═══════════════════════════════════════════════════════
§5.2 — Different Encode Types → Different Cross-Domain Reach
═══════════════════════════════════════════════════════

  MODALITY       │ ENCODE TYPE                │ CROSS-DOMAIN
  ═══════════════╪════════════════════════════╪════════════════════════════
  Visual         │ Discrete points + spatial  │ Filtered (structural match)
  Auditory       │ Temporal patterns (rhythm) │ Mild (pattern match)
  Motor          │ Action sequences           │ Very low (domain-specific)
  Emotional      │ Valence + intensity        │ Horizontal social (feeling match)
  Somatic        │ Continuous field (body)    │ EXTREMELY HORIZONTAL (unfiltered)
  Communication  │ Labels + references        │ Horizontal via label combining

Why SOMATIC cross-domain reach is STRONGEST:
  → Body state = ABSTRACT representation (no fixed structure)
  → Body encodes SENSATION, not domain-labels
  → Same sensation = "SAME" → regardless of domain
  → + Body-wide = MOST data (whole body vs 1 brain region)
  → + Continuous analog field = RICHER than discrete points
  → BUT: NOISY — body feels similar ≠ actually similar → MUST verify

Why VISUAL cross-domain reach is YES but FILTERED:
  → Visual encodes DISCRETE structures → requires structural similarity
  → "Atom LOOKS like solar system" → YES structural match ✅
  → "Team conflict is like a code bug" → NO structural match → MISS
  → Somatic CATCHES it (body state match) but visual MISSES it (no structure match)
  → Visual = PRECISION horizontal (when structures are similar)
  → Somatic = BREADTH horizontal (any domain)


═══════════════════════════════════════════════════════
§5.3 — Expert = Balanced Modalities Within Domain
═══════════════════════════════════════════════════════

"Expert" does NOT mean strong in 1 modality → BALANCED in RELEVANT modalities:

  Each domain requires a DIFFERENT modality mix:
    Surgery:  visual-spatial + motor + somatic + emotional regulation
    Coding:   sequential + visual-spatial + somatic gut feel
    Therapy:  emotional + somatic + communication
    Music:    auditory + motor + emotional

  Expert = ALL relevant modalities are STRONG ENOUGH:
    → A skilled surgeon: sees + does + feels + stays calm = ALL strong
    → Strong visual alone but shaky hands = NOT an expert surgeon

  DOMAIN FIT = modality profile MATCH domain requirements:
    → "Poor performance" may = modalities DO NOT match domain (a fish climbing a tree)
    → Career guidance: find domain that MATCHES modality profile → natural advantage

  5 THINKING TYPES (from 5 experience modalities):
    Visual-spatial: "structures are similar" → architect, physicist
    Somatic:        "feels similar" → cross-domain theorist, therapist
    Motor:          "hands know how" → craftsman, athlete, surgeon
    Emotional:      "emotions resonate" → leader, artist, counselor
    Sequential:     "what's next" → analyst, programmer, manager

🟡 Thinking direction framework = framework synthesis
🟡 Expert = balanced modalities = framework synthesis
🟢 Cross-domain analogy research: Gentner & Markman 1997
🟢 Implicit learning cross-domain: Reber 1967
```

```

  Age 0-2:   Somatic●●●●● + Emotional●●●●●
    → Body feel + attachment = foundation

  Age 2-4:   + Motor●●●● + Visual●●●●
    → Running + pretend play + imagination

  Age 4-7:   + Verbal●●●

  Age 13-18: Emotional SPIKE●●●●●
    → Hormones → emotional encoding rises sharply
    → Verbal still strong + Emotional RISING → conflict
    → "Knows it's wrong but still does it" = verbal says A, emotional says B (§4 conflict)

  Age 18+:   Verbal DOMINANT for the majority
    → 12+ years of school training → verbal leads
    → Exceptions: people who train other modalities (artists, athletes, meditators)

⚠️ SCHOOL = VERBAL TRAINING MACHINE:
  12-16 years × verbal → majority = verbal dominant → specialist mode
  Other modalities ATROPHY (less use → weakened)
  CAN be re-trained: art → visual↑, meditation → somatic↑, sports → motor↑
  Neuroplasticity: "use it or lose it" — but CAN RECOVER with re-training

CONNECTION TO V7.8:
  Development = SAME architecture, chunk density BUILDS GRADUALLY (Core §9)
  Modality development = chunks compile INTO MORE modalities over time
  School bias = chunks compile TOWARD verbal → other modalities are chunk-poor

🟢 Attachment encoding: Bowlby 1969, Ainsworth 1978
🟢 Visual-verbal shift in school: educational development research
🟢 Neuroplasticity: Merabet & Pascual-Leone 2010

## §7 — MODALITY × THERAPY (Fix the Right Channel)

```
EACH therapy approach targets a DIFFERENT modality:

  THERAPY               │ MODALITY TARGET          │ BEST FOR
  ══════════════════════╪══════════════════════════╪════════════════════════
  CBT                   │ Verbal                   │ Verbal-dominant chunks
  Exposure therapy      │ Emotional + Somatic      │ Phobia, anxiety (body)
  EMDR                  │ Visual + Emotional        │ Trauma (flashbulb)
  Somatic Experiencing  │ Somatic                  │ Body-stored trauma
  Art therapy           │ Visual + Emotional        │ Non-verbal processing
  Music therapy         │ Auditory + Emotional      │ Emotional regulation
  Dance/Movement        │ Motor + Somatic + Emot   │ Body-embedded patterns
  Meditation            │ Somatic + Emotional       │ Body awareness
  Focusing (Gendlin)    │ Somatic                  │ Felt sense → articulation

PRINCIPLE: Match therapy modality TO chunk modality profile.

  Verbal-only chunk → CBT effective ✅
  Somatic+emotional chunk → CBT NOT effective ❌ → body-based therapy needed
  "Therapy isn't working" MAY = right therapy, WRONG modality target

  Framework predicts:
    Identify chunk modality profile FIRST → select RIGHT therapy → effective

CONNECTION TO FRAMEWORK:
  → Somatic-Articulation-Loop (Somatic-Articulation-Loop.md):
    Stage 5 body-confirm = somatic modality evaluates articulation
    Effective therapy = therapy that reaches the CORRECT modalities the chunk is encoded in
  → Feeling (Feeling.md v2.0):
    Feeling threshold (magnitude × clarity) → modality richness affects magnitude
    Deeper chunks (more modalities) → stronger signal → PFC can observe more readily
  → Cortisol (Cortisol-Amplifier-Not-Cause.md):
    Chronic cortisol → overrides body signals → somatic modality gets muffled
    → Therapy must address cortisol FIRST before somatic work becomes effective

🟢 Therapy modality matching: exposure therapy research
🟢 CBT limitations for body-encoded trauma: van der Kolk 2014
🟢 EMDR for visual+emotional: Shapiro 2001
🟢 Focusing effectiveness: Hendricks 2001 meta-analysis
🟡 "Modality profile → therapy selection" = framework prediction
