## §3 — 9 PIECES OF EVIDENCE AGAINST "DOPAMINE = REWARD"

### §3.1 — Evidence 1: Berridge rat lesion (decisive)

🟢 **Berridge & Robinson 1998, Peciña & Berridge 2005**:

> Rats with dopamine system blocked/lesioned:
> - **STILL** show "liking" facial reactions when given sugar (lip lick, tongue protrusion)
> - Do **NOT** seek sugar (don't press lever, don't go look for it)
>
> Rats with dopamine stimulation (amphetamine, optogenetic):
> - Seek sugar NON-STOP (seeking increases dramatically)
> - Facial pleasure reactions do NOT increase (liking does not increase)

**Implication**:
- Dopamine = wanting system (seeking, motivation)
- Opioid = liking system (actual pleasure)
- → **Dopamine and opioid are 2 DIFFERENT systems**. They cannot be equated.

**Framework verdict**: ✅ Decisive evidence. Framework's "doorbell ≠ gift" is precisely a description of this phenomenon.

### §3.2 — Evidence 2: Schultz PE = salience, not pleasure

🟢 **Schultz 1997 classic finding**:
- Monkey VTA neuron fires when receiving unexpected juice (positive reward prediction error)
- After training, VTA fires when receiving CUE (not the juice itself)
- When reward expected but doesn't come → VTA firing DECREASES (negative prediction error)

**Classical interpretation**: Dopamine = reward signal tracking value.

**Framework reinterpretation** 🟡 (PFC-Analysis §8.3 + 03-Reward §2.4):
- VTA fires on **DELTA** (habituation-based detection), no need to compute prediction error
- "Unexpected juice" = large delta → VTA fires
- "Expected but missing" = delta from expected pattern → VTA fires (direction reversed)
- Habituation model **computationally cheaper** but gives same observable result

**Key**: Schultz's observations are **still correct**. Framework does not refute them. Framework reinterprets:
- Dopamine signal = salience/delta detection (attention direction)
- NOT a reward value signal
- The "feel" of reward = opioid downstream (step 5 after VTA)

### §3.3 — Evidence 3: Parkinson's anhedonia incomplete

🟢 **Clinical observation** (Sienkiewicz-Jarosz 2005, Loas 2012):
- Parkinson's = dopamine neuron loss in substantia nigra + VTA
- Expected (if dopamine = pleasure): lose ability to enjoy anything
- Actual: **motivation/seeking impaired** (apathy common), **enjoyment partially preserved**
  - Patients still enjoy food when eating (liking)
  - But no longer drive to go find food (wanting)

**Implication**:
- Dopamine depletion → wanting impaired (matches framework: no VTA alert → no attention boost)
- Dopamine depletion → liking preserved (matches framework: opioid system intact)

**Framework verdict**: ✅ Consistent with 7-step mechanism. Block VTA → steps 2-4 fail, step 5 (opioid) can still fire if stimulus is strong enough to bypass the attention gate.

**v1.2 NOTE** (Parkinson-Analysis.md §6.2): 3 dopamine pathways affected at different rates (nigrostriatal FIRST → mesolimbic → mesocortical LAST). Wanting/liking separation maps PRECISELY to 7-step: Steps 2-4 (VTA → PFC) weakened, Step 5 (opioid body-base) preserved. = CLEANEST clinical dissociation. Detail + 3-way comparison: §5.5.

### §3.3b — Evidence 3b: Musical anhedonia = STRONGEST pipeline integrity test

🟢 **Martínez-Molina et al. 2016 (PNAS 113:E7337-E7345)**:

Musical anhedonics (~3-5% of population):
- Auditory cortex: **NORMAL** (hears music normally)
- Reward system (NAcc): **NORMAL** (food/money/sex pleasure normal)
- CONNECTION auditory cortex ↔ NAcc: **REDUCED**
- → Domain-specific **DISCONNECT**, not global deficit

**Why this is the STRONGEST test case — proof-by-contradiction**:

> IF dopamine = reward (single step):
> → Auditory cortex works fine → VTA detects delta → dopamine fires
> → **MUST** feel pleasure → Anhedonia **CANNOT** occur
>
> MULTI-STEP MODEL:
> → Steps 1-3 (VTA detect + dopamine + spreading activation): **OK**
>   (auditory cortex processes music normally)
> → Step 4 pathway (auditory cortex → NAcc): **BROKEN**
>   (connectivity reduced — signal DOES NOT reach reward system)
> → Steps 5-6 (body-base vote + opioid): **NEVER FIRE**
> → Anhedonia **CAN** occur. **CONFIRMED.**

**Comparing §3.3 Parkinson vs §3.3b Musical anhedonia**:

| Dimension | Parkinson (§3.3) | Musical anhedonia (§3.3b) |
|---|---|---|
| **Mechanism** | GLOBAL dopamine loss | DOMAIN-SPECIFIC disconnect |
| **Scope** | Entire reward system | ONLY music → reward pathway |
| **Steps affected** | Steps 2-4 WEAKENED | Step 4 pathway BROKEN |
| **Step 5 (opioid)** | PRESERVED (food ok) | PRESERVED (food/money ok) |
| **Wanting** | Impaired (global) | Normal (non-music domains) |
| **Liking** | Partly preserved | Normal (non-music domains) |
| **Specificity** | Low (global) | HIGH (domain-specific) |
| **Test strength** | Strong (clinical) | **STRONGEST** (proof-by-contradiction) |

→ Parkinson = global disruption → wanting/liking SEPARATED (consistent with 7-step)
→ Musical anhedonia = domain-specific disconnect → pipeline integrity **PROVEN**
→ 2 clinical cases, 2 types of disruption, **BOTH confirm multi-step model**

**Motor entrainment insight**:
- Motor urge to move CAN remain intact in musical anhedonia
- → Entrainment pathway (auditory → motor cortex) ≠ reward pathway (auditory → NAcc)
- → = 2 **INDEPENDENT** pathways from auditory processing
- → Entrainment ≠ pleasure: rhythm WITHOUT reward possible

**Framework verdict**: ✅ Musical anhedonia = strongest test case for the 7-step mechanism. If PE = reward (1 step), anhedonia is impossible. Multi-step model predicts AND explains anhedonia. §3.3 (Parkinson) + §3.3b (musical anhedonia) = 2 complementary clinical dissociations.

### §3.4 — Evidence 4: Opioid = direct pleasure (morphine, endorphin)

🟢 **Pharmacological evidence**:
- Morphine/heroin (opioid agonist): **direct pleasure** signal, does NOT require a dopamine surge first
- Naloxone (opioid antagonist): **blocks pleasure** clearly even when dopamine is normal
- Endogenous opioids (endorphin, enkephalin): fire when eating, during sex, social bonding, flow state
- Runner's high = endorphin (established), not dopamine

**Critical experiment** (Peciña & Berridge 2005):
- Inject opioid agonist into NAcc "hedonic hotspot" → liking reactions increase
- Inject dopamine agonist into same region → liking does NOT increase (wanting increases)

**Framework verdict**: ✅ Opioid = actual reward. Dopamine surrounds the opioid system but has a different function.

### §3.5 — Evidence 5: Addiction wanting/liking dissociation

🟢 **Addiction research** (Robinson & Berridge 2003, Koob & Volkow 2010):
- Addicts **WANT drug intensely** (craving, seeking dominates life)
- BUT **LIKE drug decreasingly** (tolerance, pleasure diminishes)
- → Classical "dopamine = pleasure" cannot explain this

**Framework mechanism**:
- Dopamine system sensitized (hyperreactive to drug cues) → intense wanting
- Opioid response attenuated (tolerance) → decreased liking
- Body-need match DOES NOT occur (drug hijacks VTA without fulfilling body needs)
- → Loop: VTA fires → attention → seek → consume → no body satisfaction → VTA fires again

**Implication for framework**:
- Scrolling phone, slot machine, social media = **mini-addiction pattern**
- Micro-delta triggers VTA (new posts, notifications) → dopamine + wanting
- Body-need DOES NOT match (doesn't fulfill social, meaning, rest needs) → no opioid
- "Empty" feel = VTA active + opioid absent

### §3.6 — Evidence 6: Novelty alone without pleasure (scroll empty)

🟡 **Daily observable phenomenon** (Novelty.md §2.4 scroll case):
- Scroll phone: each post = micro-delta (new image, new text)
- VTA fires lightly each swipe → dopamine micro-dose
- **But**: body feels "tired, hollow, bored but can't stop"
- → Dopamine present (seeking continues) + opioid absent (no body satisfaction)

**Classical explanation fails**:
- If dopamine = pleasure → scrolling should feel good
- Reality: scrolling feels "boringly empty" but continues

**Framework explanation**:
- VTA detects delta (satisfies Precondition-3) → dopamine fires
- Body-need DOES NOT match (no pending schema to fill, no connection, no meaning) → Precondition-1 fails
- → No opioid → no actual reward
- Loop stuck: VTA keeps firing (new posts) + opioid never fires → "want more" without "getting satisfaction"

**Critical insight**: The pop "dopamine detox" logic is wrong where? Correct intuition (scrolling = problem) but wrong mechanism (not "too much dopamine", but **dopamine-opioid mismatch** + body-need bypassed).

### §3.7 — Evidence 7: Eureka moment = opioid, not dopamine spike

🟡 **Framework N+10 correction** (Chunk-Internal-Processing/01 §86):

> OLD mainstream: "Eureka = dopamine spike + cortisol drop"
> NEW framework: Eureka reward = **opioid** (schema match). Dopamine = salience alert. Cortisol drop = relief bonus (if there was a threat context).

**Mechanism**:
- Schema has dissonance (Precondition-1 met)
- New chunk appears → VTA detects delta → dopamine → PFC attends
- PFC simulates chunk → body-base check → chunk fits gap → **opioid RELEASE**
- Subjective: "AHA! EUREKA!" = opioid burst
- If there was accompanying threat (thinking under stress): cortisol drop added → explosive Eureka
- If no threat (pure curiosity): opioid alone → deep satisfaction but less explosive

**4 Eureka intensity contexts** (01-Chunk-Connection-Logical.md §86):
1. Deep problem mode (threat + schema dissonance): opioid + cortisol drop = EXTREMELY REWARDING
2. Deep research (no threat, deep gap): opioid strong, less explosive
3. Light puzzle solve: mild opioid + minor cortisol drop
4. Routine task complete: body-need "safe" DOES NOT match → NO proper opioid reward

**Implication**: Pop "dopamine rush = insight" = WRONG. Insight reward = opioid-based.

### §3.8 — Evidence 8: Naltrexone separates dopamine from euphoria (stimulant case)

🟢 **Naltrexone (opioid blocker) REDUCES stimulant euphoria — multiple studies, multiple labs**:

| Study | n | Stimulant | Finding |
|---|---|---|---|
| Jayaram-Lindström 2004 (J Clin Psychopharmacology) | 12 | Dexamphetamine 30mg | Naltrexone reduces subjective effects |
| Jayaram-Lindström 2008 (Neuropsychopharmacology) | clinical | Dexamphetamine challenge | Reduces effects (p<0.001) + craving (p<0.001) |
| Jayaram-Lindström 2008 (Am J Psychiatry) | 80 | 12-week maintenance | Reduces relapse |
| Ray 2015 (Neuropsychopharmacology) | 30 | IV methamphetamine | Reduces craving |
| Spencer 2018 (J Clin Psychopharmacology) | 37 | Methylphenidate 60mg (ADHD) | Reduces euphoria ("Liking Effect") |

**⭐ KEY study — Jayaram-Lindström 2017** (Translational Psychiatry):
- PET [11C]raclopride in humans + microdialysis in rats
- Naltrexone blocks euphoria BUT **DOES NOT block dopamine release**
- → Dopamine still fires normally → euphoria STILL GONE
- → Opioid system contributes to hedonic experience **INDEPENDENTLY** of dopamine signal

**PET evidence — amphetamine causes opioid displacement** 🟡:
- Colasanti 2012 (Biological Psychiatry, n=12): [11C]carfentanil PET, amphetamine causes decrease in opioid binding
- Mick 2014 (n=9): replication confirmed
- Mick 2016 (n=14 gamblers vs 15 controls): opioid release **blunted** in gamblers
- ⚠️ Guterstam 2013 (n=10): IV route → null finding (different route + timing)
- Caveat: replications mainly from 1 lab group (Imperial/King's College London)

**Why this evidence is STRONG — proof by pharmacological dissociation**:

> IF dopamine = reward (1 signal):
> → Block opioid (naltrexone) SHOULD NOT affect euphoria
>
> REALITY: Block opioid = euphoria CLEARLY DECREASES (5+ studies)
> AND: Dopamine release is NOT blocked (Jayaram-Lindström 2017)
> → 2 INDEPENDENT systems. Dopamine ≠ reward. QED.

**Adderall misconception**:
- Pop science: "Adderall supplements dopamine → euphoria → dopamine = reward"
- Wrong in 2 places: (1) Adderall acts on dopamine + norepinephrine + mild serotonin, NOT dopamine alone. (2) Euphoria at high doses requires the opioid system (naltrexone block → euphoria decreases)
- Therapeutic doses in ADHD: usually do NOT cause euphoria — only normalize PFC dopamine
- → Same drug, different doses = different outputs = dopamine ≠ simple reward signal

**2 competing models for opioid role — NOT YET SETTLED** 🟡:

| Model | Mechanism | Main evidence | Limitation |
|---|---|---|---|
| **Active trigger** | Dopamine flood → triggers new opioid release | PET (Colasanti 2012) | 1 lab group, 1 null |
| **Permissive tone** | Tonic opioid already present = necessary condition | Jayaram-Lindström 2017, OPRM1 genetics | Hard to explain dose-dependence |

- Likely combination: tonic opioid = baseline + dopamine stimulation = boost
- New tools (opioid biosensors — Bhatt 2024, Nature Neuroscience) may settle this

**Framework verdict**: ✅ Naltrexone evidence = **strongest pharmacological proof** for "dopamine ≠ reward". The exact mechanism (active trigger vs permissive tone) is unsettled, but the conclusion is CLEAR: euphoria requires the opioid system, dopamine alone is insufficient. Confirms 7-step: Step 5 (opioid) ≠ Steps 2-4 (dopamine).

## §4 — FRAMEWORK ALTERNATIVE: 7-STEP MECHANISM

### §4.1 — Overview: 2-level reward

🟡 **Framework core claim** (03-Reward.md §1.2, Feeling-Mechanism-Deep §4 definitive):

> **Reward = 2-level mechanism**, NOT 1 signal:
>
> **Level 1 — DOPAMINE ALERT (VTA)**: "THERE IS A CHANGE — pay attention!" — salience, not value
> **Level 2 — OPIOID REWARD (body-base)**: "chunk MATCHES body-need → actually valuable" — actual reward
>
> VTA dopamine = **doorbell**. Opioid = **gift behind the door**.

### §4.2 — 7-Step Mechanism (consolidated from PFC-Analysis §8.3 + 03-Reward §2)

```
┌────────────────────────────────────────────────────────────────┐
│ Step 1: UNCONSCIOUS FIRE (24/7)                                │
│   86 billion neurons fire continuously + Hebbian synchronize   │
│   Patterns (chunks) emerge self-organizing                     │
│   Cortisol baseline amplifies/dampens (Cortisol-Baseline §1)   │
```

### §4.3 — Core difference: Body-base check (Step 5)

⭐ **Step 5 is where the framework differs from all other positions**:
- Classical: skips this step (dopamine = pleasure, no body check needed)
- Berridge: acknowledges opioid differs from dopamine, does not specify WHEN/HOW opioid fires
- Framework: opioid fires WHEN chunk fits body-need (Body-Feedback-Precondition 5 preconditions — 03-Reward §3)

**Body-Feedback-Precondition 5 preconditions for opioid to fire**:
1. Precondition-1 Directed-Gap (body has a need waiting to be filled)
2. Precondition-2 Chunk-Substrate (sufficient substrate to decode)
3. Precondition-3 Delta-Gate (change large enough)
4. Precondition-4 Match-Range (Goldilocks zone — dynamic)
5. Precondition-5 Compile-Gate (opioid-tagged not threat-tagged)

**ALL 5 must be met** → full reward. Miss ANY → weak/absent/wrong-direction signal.

### §4.4 — Addiction explained through 7-step

⚠️ **3 addiction types, 3 different mechanisms within 7-step**:

**Type A — Opioid drugs (morphine, heroin)**:
- Direct μ-receptor activation → Step 5 **COMPLETELY BYPASSED**
- Drug fires opioid receptors directly, no body-need match required
- Also: μ-receptor → disinhibits VTA GABA → dopamine surge (Steps 2-4 fire in reverse)
- Step 6: reinforces drug-seeking chunks (strong Hebbian due to direct opioid hit)
- Step 7: tolerance — opioid receptor adaptation → need more for same effect

**Type B — Stimulant drugs (amphetamine, cocaine, Adderall at high doses)** 🟡:
- Floods dopamine at Steps 2-4 (blocks reuptake + reverses transporters)
- + Norepinephrine + mild serotonin (NOT dopamine alone)
- Step 5 opioid involvement: **YES** (naltrexone blocks euphoria 🟢 §3.8) but mechanism NOT YET SETTLED:
  - Model 1: dopamine flood triggers new opioid release (Colasanti 2012)
  - Model 2: tonic opioid = permissive condition (Jayaram-Lindström 2017)
- KEY: naltrexone blocks euphoria WITHOUT blocking dopamine release (2017)
  → Opioid contribution INDEPENDENT, not simple downstream
- Dynorphin feedback: D1-MSN activation → dynorphin (κ-opioid) → REDUCES dopamine
  = negative feedback loop, explains tolerance + withdrawal dysphoria
- Step 7: D2 receptor down-regulation → need more stimulant for same dopamine flood

**Type C — Behavioral addictions (scrolling, slot machines, social media)**:
- Micro-delta per stimulus → VTA constant firing → dopamine micro-doses
- Body-need rarely/never met → Step 5 opioid absent → "empty chase"
- Intermittent reinforcement (random reward schedule) → VTA sensitized
- = Wanting active + liking absent → "bored but can't stop"

**Common result**: Wanting sensitized (Steps 2-4 boost) + Liking habituates/absent + Body-need never actually fulfilled
- → Pattern: "need more, enjoy less, never satisfied"
- Classical "dopamine = reward" cannot explain. Framework 7-step can.
- 3 types, 3 disruption points, ALL with the same output pattern
