## 1. Resonance Chi Tiết — Signal vs Noise

```
🟡 Search KHÔNG phải "hit or miss" — mà là RESONANCE:

  Chunks ĐÚNG (compiled, khớp): HẤP THỤ dao động → fire RÕ NÉT → SIGNAL
    → "Ồ, có vẻ đúng" → body feel TRƯỚC PFC label
    → PFC chú ý → focus → chunk RÕ THÊM → "đúng rồi!"

  Chunks SAI (random, không khớp): dao động PHẢN LẠI → NOISE
    → Quá nhiều noise → "căng đầu, rối"

  SIGNAL-TO-NOISE RATIO:
    Cortisol CAO + ÍT chunks = noise >> signal → RỐI, đau đầu
      → Fix: DỪNG, hạ cortisol, HOẶC nạp chunks trước
    Cortisol CAO + NHIỀU chunks = signal >> noise → INSIGHT
      → Kết nối lờ mờ → dao động kích → fire rõ → chunk MỚI
    Cortisol THẤP + NHIỀU chunks = signal yếu → "biết nhưng lười nghĩ"
    Cortisol VỪA + NHIỀU chunks = SWEET SPOT → Flow state

  → = "Sáng tạo" = signal-to-noise ratio ĐỦ CAO
  → = Chunks đủ + dao động đủ → hit. Thiếu 1 = không ra gì.
```

## 2. "Aha!" = Link MỚI Giữa Chunks CŨ

```
🟡 "Aha" KHÔNG phải tạo chunk MỚI — mà NỐI chunks cũ THEO CÁCH MỚI:

  → 2+ chunks ĐÃ TỒN TẠI → chưa bao giờ LINKED → bỗng connection FOUND

  VD Archimedes:
    TRƯỚC: PFC hold chunk[nhiệm vụ vua] = {vàng, vương miện, thể tích,...}
      Cortisol CAO → dao động MẠNH → search rộng → NHƯNG chưa có trigger
    TRONG BỒN: body input [nước ĐẨY] → chunk [thể tích] CỘNG HƯỞNG
      → Somatic input + stored chunk = RESONANCE → fire RÕ NÉT
      → PFC focus: [thể tích nước] + [thể tích vàng] + [thể tích vương miện]
      → Chuỗi chunks fire → body simulate: "mượt, hợp lý" → "EUREKA!"

  Tại sao "aha" feel CỰC MẠNH:
    → 1 link mới → RESOLVE nhiều loose threads CÙNG LÚC
    → Signal/noise ratio NHẢY cực mạnh trong 1 khoảnh khắc
    → = "Mọi thứ CLICK vào nhau"
    → Body-Reward CỰC: dopamine (mới!) + opioid (coherent!) + cortisol DROP
    → 🟢 Aha = gamma burst (Kounios & Beeman 2009): sudden neural synchrony

  → = "Sáng tạo" = tìm LINKS MỚI, không tạo content mới
  → = Nhiều chunks → nhiều combos link → nhiều aha tiềm năng
  → = Cross-domain chunks → link CHƯA AI THỬ → aha LỚN nhất
```

## 3. "Tip of the Tongue" — Chunk Có Nhưng Search Chưa Hit

```
🟡 "Biết mà... gần ra rồi... KHÔNG nhớ!"

  → Body FEEL: "chunk NÀY tồn tại" (resonance NHẸ — signal có nhưng YẾU)
  → PFC search: gần hit nhưng KHÔNG ĐỦ mạnh để fire hoàn toàn
  → = Chunk TỒN TẠI nhưng retrieval path TẠM yếu

  → 10 phút sau: "À! TOM HANKS!" → bỗng fire
  → = Incubation: PFC thả → nền vẫn search → tìm thấy → fire

  → Tại sao "gợi ý 1 chữ cái" → bỗng nhớ:
    thêm 1 search term → ĐỦ để hit (signal vượt threshold)
  → 🟢 Tip-of-the-tongue (Brown & McNeill 1966)
```

## 4. Incubation Hold — "Ngủ Đi Rồi Sáng Mai Rõ"

```
🟡 "Hold" KHÔNG chỉ working memory VÀI GIÂY:

  WORKING MEMORY (giây): PFC active hold → tốn energy → MỆT
  INCUBATION (giờ-ngày): PFC THẢ → chunks vẫn PARTIALLY ACTIVE
    → DMN (Default Mode Network) TIẾP TỤC process khi PFC offline
    → = "Nền" vẫn search — PFC không biết
    → Sáng mai: chunks đã ARRANGE tốt hơn → "ồ, rõ hơn!"

  "NGỦ → SÁNG RÕ":
    → Ngủ: hippocampus REPLAY → consolidate → sáng = compiled TỐT hơn
    → 🟢 Sleep → memory consolidation (Walker 2017)
    → 🟢 Sleep → 2x insight rate (Wagner 2004)

  "ĐI BỘ → NGHĨ RA":
    → Body relax → cortisol giảm → noise GIẢM
    → External input ĐƠN GIẢN → PFC freed → search NỀN sạch hơn
    → 🟢 Walking boosts creativity (Oppezzo & Schwartz 2014)

  TIẾN TRÌNH ITERATIVE:
    Mâu thuẫn → hold → seek → mini-aha → hold → incubate → rõ hơn → seek → ...
    = NHIỀU VÒNG: [hold → trigger → incubate → rõ hơn] × N
    = Mỗi vòng: signal/noise TĂNG → đến khi "mượt" (noise ≈ 0)
```

## 5. Priming — Môi Trường BIAS Search

```
🟡 "Nghĩ ở café KHÁC nghĩ ở văn phòng" — không chỉ ít distraction:

  PRIMING:
    → Chunk fire → HẠ ngưỡng cho chunks CONNECTED → dễ fire hơn
    → 🟢 Priming effect (Meyer & Schvaneveldt 1971)

  MÔI TRƯỜNG = PASSIVE PRIMER — "search terms NGẦM":
    Văn phòng: [máy tính, deadline] PRIME → search bias về [work, stress]
    Rừng: [cây, trời] PRIME → search bias về [calm, open-ended]
    Quán café: [nhạc, người] PRIME → search bias về [social, explore]
    Phòng MỚI: [đồ lạ] PRIME → search bias về [novelty, fresh]

  → = CÙNG PFC, CÙNG database → KHÁC environment → KHÁC output
  → = "Đổi chỗ ngồi = đổi search terms NGẦM"

  PRACTICAL:
    Deep focus: phòng QUEN, ÍT input → priming ÍT → search FOCUSED
    Creative: chỗ MỚI → priming RỘNG → combos MỚI
    Insight: thiên nhiên, đi bộ → priming NHẸ + noise THẤP → hit BẤT NGỜ
```

```

  Compile lúc SỢ → tag [threat] → fire NHANH khi context tương tự
  Compile lúc VUI → tag [reward] → fire NHANH khi context tương tự

  Emotional state hiện tại = THÊM 1 search term NGẦM:
    Đang sợ → chunks [threat] fire MẠNH → "thấy nguy hiểm KHẮP NƠI"
    Đang vui → chunks [reward] fire MẠNH → "thấy cơ hội KHẮP NƠI"
    → = "Khi cầm búa, mọi thứ giống cái đinh"

  → = PFC TƯỞNG search neutral → emotional tag ĐÃ filter SẴN
  → = "Thinking with your heart" = emotional tags AMPLIFY — không phi logic
  → = CÙNG data → người sợ thấy RISK, người vui thấy OPPORTUNITY
  → = Body-Dissonance cao → search thiên threat → vicious cycle
```
