## §1 — ĐỊNH NGHĨA FEELING

### 1.1 Formal Definition

```
⭐ FEELING — ĐỊNH NGHĨA CHÍNH THỨC:

  FEELING = signal phản hồi đã được INTEGRATED về state của body systems
            (L0-L3 body-base + agent input + mirror + schema + imagine-final
            + valence + cognitive prediction) → VERBALIZABLE (hoặc gần
            verbalizable) ở PFC level.

  Nói ngắn gọn: Feeling là CÁCH body tự làm cho chính mình trở thành
  noticeable với PFC.


  PHÂN TÍCH THEO TỪNG PHẦN:

  ① "signal phản hồi" — feedback, không phải input mới
     Feeling KHÔNG tạo ra thông tin mới. Feeling REPORT về trạng thái
     body đã tính toán. Body là source, feeling là messenger.

  ② "đã được INTEGRATED" — unified, không phải list
     Feeling KHÔNG phải danh sách sensation. Feeling = ONE unified
     sense về "tôi đang ở trạng thái nào". Multi-source → single stream.

  ③ "state của body systems" — multi-source
     Body có NHIỀU systems đang fire đồng thời (L0-L3, mirror, schema,
     imagine-final, valence, cognitive). Feeling tích hợp TẤT CẢ.

  ④ "VERBALIZABLE ở PFC level" — PFC-accessible
     Feeling phải ĐỦ mạnh để PFC nhận biết. Body tính toán vô thức
     liên tục — chỉ 1 phần nhỏ nổi lên thành feeling.
     "Verbalizable" không có nghĩa "có từ sẵn" — có nghĩa PFC CÓ THỂ
     tương tác được (observe, locate, label, explain — dù imperfect).


  CÔNG THỨC ĐƠN GIẢN:
    Body fire + Integrated + PFC-noticeable = FEELING
```

### 1.4 Phân Biệt Nhanh Với Concepts Khác

```
🟡 FEELING vs CONCEPTS KHÁC (tóm tắt — detail ở Feeling-Research.md §1):

  FEELING vs EMOTION:
    Emotion = limbic valence narrow (vui, buồn, giận, sợ)
    Feeling = broader integrated body sense
    Relationship: Emotion ⊂ Feeling

  FEELING vs SENSATION:
    Sensation = single-modality (touch, sound, sight)
    Feeling = multi-modal integration
    Example: "mùi thơm" = sensation, "cảm thấy ấm cúng" = feeling

  FEELING vs THOUGHT:
    Thought = PFC symbolic processing
    Feeling = body-based, pre-symbolic often
    Note: Thought CÓ THỂ tạo feeling (schema fire), nhưng feeling
          KHÔNG phải thought

  FEELING vs INTUITION:
    Intuition = compiled feeling firing fast
    Intuition ⊂ Feeling (1 subset: expert-compiled)

  FEELING vs MOOD:
    Mood = sustained feeling state (hours-days)
    Feeling = can be acute or sustained
    Relationship: Mood = extended feeling

  FEELING vs AFFECT:
    Affect = psychology technical term, overlapping
    Different academic lineage, similar meanings

  → Framework dùng "FEELING" vì rộng nhất, inclusive nhất,
    và gần với everyday experience nhất.
```

### 2.2 Layer 1: Raw Body Signals

```
🟢 LAYER 1 — RAW BODY SIGNALS (unconscious)

  Nhiều body systems ĐANG fire đồng thời — vô thức, liên tục:

    → Interoceptive: HR, BP, gastric, respiration, temperature
    → Somatic: proprioception, tension, posture
    → Autonomic: sympathetic/parasympathetic balance
    → Neuroendocrine: cortisol, dopamine, serotonin
    → L0-L3 body-base monitoring
    → Mirror circuits firing when observing agents
    → Schema firing when detecting familiar patterns
    → Imagine-final preview running constantly
    → Valence compiled assessments firing
    → Cognitive prediction + error detection

  Các signals này KHÔNG phải feeling YET. Chúng là RAW INPUT.

  Scale: hàng NGHÌN signals/giây ở thân thể.
         Body handle vô thức, PFC không access trực tiếp.

  Research:
    🟢 Craig 2002, 2013 — interoception afferent pathways
    🟢 Sherrington proprioception
    🟢 Porges polyvagal (🟡 controversial)

  Fidelity: 100% (đây LÀ body truth, chưa có loss)
  PFC access: KHÔNG (quá nhiều, quá nhanh, đa dạng)
```

### 2.3 Layer 2: Integration

```
🟢 LAYER 2 — INTEGRATION (insula, ACC, VMPFC)

  Raw signals được TÍCH HỢP thành unified body state.

  Hub chính (research-established):
    → Insula: interoceptive integration (Craig 2002)
    → ACC: conflict detection, emotional salience
    → VMPFC: value integration, decision preparation
    → Thalamus: relay + preprocessing

  Cơ chế:
    → Hàng nghìn raw signals converge
    → Brain compress thành 1 unified representation
    → Representation này = "tôi đang ở state X"
    → State X KHÔNG phải 1 number — là multi-dimensional
      nhưng COHERENT

  Analogy:
    Giống như CPU integrate hàng triệu bit → system state.
    PFC KHÔNG cần biết từng bit — chỉ cần biết "state X vs state Y".

  Research:
    🟢 Craig 2009 "How do you feel — now?"
    🟢 Damasio 1999 "The Feeling of What Happens"
    🟢 Seth 2013 interoceptive inference

  Fidelity: ~95% (integration lossy nhẹ, nhưng body truth vẫn rõ)
  PFC access: INDIRECT (qua layer 3+)
```

### 2.4 Layer 3: Consciousification

```
🟢 LAYER 3 — CONSCIOUSIFICATION (body state trở nên noticeable)

  Integrated state → crossing threshold of awareness.

  TRANSITION quan trọng nhất:
    → Trước Layer 3: body computing vô thức
    → Sau Layer 3: PFC có thể "hướng attention" vào feeling

  Cơ chế:
    → Signal đủ mạnh (threshold)
    → Hoặc attention hướng vào (top-down query)
    → Hoặc contrast/change đủ để trigger
    → → PFC nhận thức "something is happening in me"

  ĐIỂM ĐẶC BIỆT:
    Layer 3 là điểm body đã compute XONG — nhưng PFC CHƯA
    label gì cả. Đây là "felt sense" (Gendlin term).

    → "Có gì đó"
    → "Cảm thấy sao sao ấy"
    → "Không biết nhưng cảm thấy..."

    = TRUTH đã sẵn sàng. PFC chưa có từ.

  Wise practitioners (meditators, Focusing trainees, expert
  debuggers) HỌC để dừng ở Layer 3 — TRUST raw felt sense,
  KHÔNG vội lên label/explanation.

  Research:
    🟢 Gendlin 1978 Focusing — pre-verbal felt sense
    🟢 Farb 2013 meditation → improved layer 3 access
    🟢 Damasio — "core self" awareness

  Fidelity: ~90% (body truth vẫn gần nguyên, chỉ mất compression nhẹ)
  PFC access: YES — nhưng cần training để trust at this layer
```

### 2.5 Layer 4: Observation

```
🟡 LAYER 4 — OBSERVATION ("something is happening")

  PFC bắt đầu actively observe feeling.

  Câu PFC tự nói với chính mình:
    → "Something is happening in me"
    → "I'm feeling something"
    → "There's some state"

  KHÁC với Layer 3 ở điểm: Layer 3 = body đã make state noticeable
                              Layer 4 = PFC actively pay attention

  Layer 3 có thể happen không cần Layer 4 (background awareness).
  Layer 4 requires PFC allocation (attention budget).

  Ví dụ:
    Layer 3: Background unease khi đi vào phòng lạ
    Layer 4: "Ồ, tôi đang cảm thấy gì đó"

    Layer 3: Body khó chịu nhẹ khi nghe 1 tên
    Layer 4: "Tôi đang có reaction với tên này"

  ERROR MODE: Layer 4 có thể BỎ SÓT Layer 3.
    → Busy, distracted, intellectualize → không notice
    → Alexithymia → Layer 3 active nhưng Layer 4 không engage
    → Ignore body signals → silent suffering

  Fidelity: ~85% (attention bias introduces small distortion)
  PFC access: YES
```

### 2.6 Layer 5: Location

```
🟡 LAYER 5 — LOCATION ("where is this feeling?")

  PFC thử xác định feeling "ở đâu" trong body.

  Câu hỏi PFC tự đặt:
    → "Where is this feeling coming from?"
    → "Chỗ nào trong body?"
    → "Throat? Chest? Stomach? Head?"

  Ví dụ:
    → "Anxiety ở ngực" (tight chest)
    → "Đau bụng chỗ này" (location approximate)
    → "Tension ở shoulders"
    → "Heat ở mặt" (embarrassment/anger)
    → "Empty feeling in chest" (grief)

  ACCURACY VARIES HUGE:
    → Somatic pain: HIGH accuracy (well-mapped cortex)
    → Visceral pain: LOW accuracy (referred pain phenomenon)
    → Emotional feelings: MEDIUM accuracy (cultural variation)

    VD: heart attack → đau tay trái (referred, không phải ở tay)
        Đau ruột thừa → đầu tiên "đau bụng chung chung",
        sau đó localize về bên phải

  TRAINING effect:
    → Expert doctors learn to distinguish visceral qualities
    → Meditators learn finer-grained body scan
    → Gendlin Focusing practitioners dwell at Layer 5

  NGÔN NGỮ cung cấp constraints:
```

```
  PFC access: HIGH (mainstream "feeling" stops here)
```

### 2.8 Layer 7: Explanation

```
🟡 LAYER 7 — EXPLANATION ("because ___")

  PFC generates CAUSAL story for feeling.

  Câu PFC tự nói:
    → "I feel anxious BECAUSE I have a deadline"
    → "I feel sad BECAUSE he left"
    → "I feel angry BECAUSE traffic was bad"

  LAYER RISKY NHẤT:
    Layer 7 = PFC narrative-making engine activated.
    → Confabulation risk HIGH
    → Rationalization common
    → Displacement frequent

  Ví dụ thực tế:
    "Tôi giận chồng vì anh ấy không rửa bát" (Layer 7)
    Thực ra: Mệt + đói + stress công việc compile thành irritable
              → any stimulus trigger anger
              → chồng + bát là SAFE target
              → PFC tạo câu chuyện post-hoc

  ERROR MODES (Feeling-Accuracy.md §3 detail):
    → RATIONALIZATION: Feeling exists → create plausible story
    → DISPLACEMENT: Feel angry at A, blame B (safer target)
    → MISATTRIBUTION: Physical → emotional story
      ("anxious" = actually hungry)
    → OVER-COMPLEXITY: Simple body need → grand narrative

  WHY PFC misleads ở Layer 7:
    → Linguistic structure forces subject-verb-object
    → Cultural scripts limit valid explanations
    → Social pressure to explain simply
    → Self-concept protection filters honest causes
    → PFC likes single-cause > multi-cause


  CRITICAL FOR DECISION-MAKING:
    Decisions based on Layer 7 explanations = often WRONG actions.
    → "I feel tired BECAUSE work is too hard" → quit job
      (actually: bad sleep + no exercise)
    → "I feel empty BECAUSE I need new relationship" → chase partner
      (actually: lost meaning at work, unaddressed)


  WISE approach (Feeling-Accuracy.md §5-7 literacy):
    → Notice Layer 3 (pre-labeling body state)
    → Stay curious at Layer 4-5 (where, not why)
    → Be SUSPICIOUS of Layer 7 explanations
    → Verify hypotheses instead of believing stories

  Fidelity: 20-70% (worst layer, highest confabulation risk)
  PFC access: MAXIMUM — which is why it's most trusted + most wrong
```

```
     Ancestral: decisions measured in seconds-hours

     Ancestral: stress = acute (escape tiger, then rest)

     Ancestral: 50-150 close relationships (Dunbar)

     → Agent input "deficit" despite hyper-connection
     → "Lonely in crowd" phenomenon
     → → Literacy cần: distinguish real vs mediated agents

  ⑤ INFORMATION OVERLOAD
     Ancestral: threats were local, proximate

     → Body treats "earthquake in other continent" như

     → Cortisol for things you cannot act on
     → Learned helplessness + chronic activation
     → → Literacy cần: filter threat relevance
```
