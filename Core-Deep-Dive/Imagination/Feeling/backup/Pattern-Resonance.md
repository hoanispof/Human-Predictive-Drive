## §1 Definition

[Phase 3 fill]

**Core definition**: Interpersonal mechanism by which Self-Pattern
Match library is used as a SIMULATOR to model and infer other
entity's state, outputting fit-judgment (match/mismatch) between
self state and inferred other state.

**Unpacking:**
- Simulator: own chunks used as template for modeling
- Inference: not direct sensing, reasoning-based
- Fit-judgment: output is relational (alignment or mismatch)
- Drives: social feelings + social actions

**What it's NOT:**
- NOT hardware mirror neuron module (REJECTED)
- NOT direct mind reading (always inferential)
- NOT accurate by default (limited by self-library quality)
- NOT same as Self-Pattern Match (different application)

## §2 Mechanism

[Phase 3 fill]

### §2.1 Input
- Observation of other entity (visual, auditory, contextual)
- Features extracted, passed upstream

### §2.2 Self-library activation
- External observation pattern-matches own chunks
- Own chunks activate as if simulating the observed situation

### §2.3 Simulation
- Activated chunks run as model of other's state
- "If I were in that situation, showing those signals, I'd feel X"
- NOT accurate to actual other-state, but inferred via own template

### §2.4 Fit judgment
- Compare inferred-other-state with current self-state
- Output categories:
  - Match positive → connection, joy, belonging
  - Match negative → solidarity, shared struggle
  - Mismatch → discomfort, distance
  - Anti-match → conflict, irritation
  - Inferred distress → concern, comforting urge

### §2.5 PFC observation
- PFC observes the fit-judgment output as feeling
- May verbalize ("tôi cảm thấy bạn buồn", "chúng ta không hợp")
- May remain pre-verbal (intuitive sense)

### §2.6 Action
- Feed into Response Channel
- Social actions: approach, withdraw, comfort, share, confront

## §3 Developmental Stages

[Phase 3 fill]

### §3.1 Stage framework

9 stages from reflexive contagion to refined practice.

### §3.2 Stage 0 — Reflexive Contagion (0-3 months)
- Newborn cry response to other cries
- Chunk-based (own-cry experience) + novelty/dissonance
- NOT hardware mirror
- Examples: cry contagion in nursery

### §3.3 Stage 1 — Social Engagement (3-6 months)
- Smile contagion (after 6-8 weeks cortical maturation)
- Mutual gaze
- Affect resonance with primary caregiver
- Requires: face visual exposure + reward association built

### §3.4 Stage 2 — Joint Attention (6-12 months)
- Look where mom looks
- Social referencing (look to mom for danger info)
- Shared attention circuits developing

### §3.5 Stage 3 — Comforting Behavior (18-24 months)
- Hoffman empathy stage 3
- Bring own comfort object to distressed adult
- Using own pattern as template for other's need
- First true simulation-based action

### §3.6 Stage 4 — Theory of Mind Basic (3-4 years)
- False belief task passing
- Understand other has different knowledge/beliefs
- Mental state attribution

### §3.7 Stage 5 — Multi-Perspective (5-7 years)
- "He thinks I think..." recursive ToM
- Understand others' motivations from their view

### §3.8 Stage 6 — Complex Social Inference (7-12 years)
- Read implicit motives, sarcasm
- Understand multi-layered social situations

### §3.9 Stage 7 — Abstract Social Reasoning (adolescence)
- Hypothetical scenarios
- Ethical reasoning
- Identity and belonging

### §3.10 Stage 8 — Refined Practice (adult+ with training)
- Deep listening (therapist-level)

## §5 Bird & Cook Alexithymia Insight

[Phase 3 fill — this is the DECISIVE section]

### §5.1 The finding
Bird & Cook 2013 — empathy deficits in autism driven by alexithymia,
not autism per se. Those without alexithymia show intact empathy.

### §5.2 Mechanism implication
Pattern Resonance requires Self-Pattern Match library as prerequisite.
If self-library is shallow (alexithymia), simulator has no templates
to run → no resonance → no empathy.

### §5.3 Decisive against hardware mirror
Hardware mirror would predict automatic empathy regardless of self-
awareness. Reality: empathy depends on self-learning. Therefore
learning-based mechanism, not hardware.

### §5.4 Framework implication
Self-Pattern Match is foundation for Pattern Resonance. You cannot
understand others without first understanding yourself. Empathy is
cultivated through self-literacy.

## §6 Multi-Axis Profile Within Pattern Resonance

[Phase 3 fill]

5 axes applied to Pattern Resonance specifically:

- Axis 1 depth: how refined the social chunks (clinician vs beginner)
- Axis 2 breadth: how many types of others can simulate (in-group
  only vs cross-cultural)
- Axis 3 speed: automatic reading vs effortful analysis
- Axis 4 integration: single-dimension (just emotion) vs multi
  (emotion + motivation + context + history)
- Axis 5 action coupling: insight without action vs tightly coupled

### Profile examples
- Therapist (deep, broad, fast, integrated, moderate coupling)
- Psychopath (decent cognitive depth, shallow affective, no coupling)
- Alexithymic (shallow across)
- Typical adult (moderate across)
- Autistic with alexithymia (variable, often shallow emotional)
- Autistic without alexithymia (can be normal Pattern Resonance)

## §7 Accuracy Problems

[Phase 3 fill]

### §7.1 Simulation fallibility
Output only as accurate as simulator. Own library may not match
other's actual library. Systematic errors emerge.

### §7.2 Projection error
Assuming other feels exactly as you would. Ignores individual
differences.

### §7.3 Cultural errors
Using own culture's chunks to simulate other culture's person.
Missing/wrong chunks.

### §7.4 Temperament blindspot
Introvert modeling extrovert (or vice versa). Mismatched base states.

### §7.5 Trauma projection
Trauma survivors may over-detect threats in others' behavior.

### §7.6 Cognitive vs affective split
Psychopathy: cognitive empathy intact, affective disconnected.
High-functioning manipulation possible.
