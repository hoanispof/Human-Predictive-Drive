## §1 Definition

[Phase 3 fill]

**Core definition**: Intrapersonal mechanism by which PFC observes
own internal state (body + unconscious processing) through activation
of self-built pattern chunks.

**Unpacking:**
- Self-built: chunks formed through own experience
- Pattern chunks: schemas, labels, templates
- Activation: chunks fire when matching signals arrive
- Observation: PFC reads the activated pattern (observer role)
- Internal state: body interoception + schema activation + predictive
  processes + memory retrieval + all unconscious processing

**What it's NOT:**
- NOT direct raw body sensing (PFC no raw access)
- NOT external influence (no "mirror" from outside)
- NOT automatic hardware trigger (requires learned chunks first)
- NOT single-shot (stage-gated, state-gated, probabilistic)

## §2 Mechanism

[Phase 3 fill]

### §2.1 Input side
- Channels feeding signals upstream (body interoception, schema
  activation, predictive systems, memory, cognitive processes)
- Integration happens upstream before PFC access

### §2.2 Chunk library
- Built through experience + repetition (ritual formation)
- Operant + classical conditioning
- Cross-channel scaffolding (similar chunks transfer)
- Math formula analogy: cumulative + branching

### §2.3 Match operation
- Pattern matching: signals ↔ chunks (fuzzy, probabilistic)
- State-gated: current state modulates what activates
- Near-miss triggers strongest (predictive coding amplifies dissonance)
- Exact match weakest (familiar, resolved)

### §2.4 PFC observation
- Read access to integrated outputs
- Layer 3-4 of 7-layer structure = genuine felt sense
- NOT raw body, NOT pure labeling — integrated middle ground
- This is Q3 partial resolution

### §2.5 Labeling
- Chunks provide labels when available
- Missing chunks = diffuse feeling ("vui chung chung không rõ vì sao")
- Rich chunks = specific verbalization ("ấm ức vì bị hiểu lầm")

### §2.6 Output
- Verbalization (if chunks support it)
- Internal subjective experience (regardless of verbalization)
- Informs Response Channel (action generation)

## §3 Developmental Stages

[Phase 3 fill with extensive detail]

### §3.1 Stage framework

8 stages from crude to refined. Stages are not strict ages — depend
on scaffolding + practice + individual variation.

### §3.2 Stage 0 — Crude Distress (0-3 months)
- Capability: distress vs comfort binary
- PFC milestone: subcortical + immature PFC
- Examples: undifferentiated crying (đói/đau/ướt/mệt all → khóc)
- Chunks: minimal, seed patterns only

### §3.3 Stage 1 — Behavioral Discrimination (3-12 months)
- Capability: different distress types (parents recognize)
- PFC milestone: OFC begin functioning
- Examples: different cry patterns for different needs
- Chunks: beginning differentiation

### §3.4 Stage 2 — Body Location (12-24 months)
- Capability: point to body parts, basic interoception, bladder
  awareness emerging
- PFC milestone: sensorimotor integration
- Examples: "đau" + pointing to location, gesturing for needs
- Chunks: body map forming

### §3.5 Stage 3 — Verbal Labels Primitive (2-3 years)
- Capability: "đói", "đau", "buồn đái", "sợ"
- PFC milestone: language network + medial PFC
- Examples: verbal request before event ("buồn đái" before wetting)
- Chunks: first verbal anchors

### §3.6 Stage 4 — Emotion Words (3-5 years)
- Capability: "vui", "buồn", "sợ", "giận"
- PFC milestone: self-other distinction
- Examples: can tell caregiver about emotional state
- Chunks: emotional vocabulary building

### §3.7 Stage 5 — Complex Emotions (5-10 years)
- Capability: "ấm ức", "ghen tị", "tự hào", "xấu hổ"
- PFC milestone: social-emotional integration
- Examples: articulates mixed feelings, understands context
- Chunks: compound emotional schemas

### §3.8 Stage 6 — Abstract Reasoning (10-18 years)
- Capability: "tôi cảm thấy bị phản bội", "tôi cảm thấy mất ý nghĩa"
- PFC milestone: abstract thinking, identity formation
- Examples: articulates complex inner states
- Chunks: abstract emotional chunks

### §3.9 Stage 7 — Refined Practice (adult+, with training)
- Capability: felt sense (Gendlin), subtle body states, metacognitive
  feeling, predictive interoception
- PFC milestone: continues with practice
- Examples: meditators, artists, expert therapists
- Chunks: deep refinement in specific channels

## §4 Multi-Axis Profile Within Self-Pattern Match

- Axis 1: Depth per channel (chunk richness)
- Axis 2: Breadth (number of channels)
