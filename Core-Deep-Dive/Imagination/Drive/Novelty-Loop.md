## 1. Novelty Loop Là Gì

```
DEFINE:

  Novelty Loop = vòng lặp KHÔNG CHỦ ĐỘNG:
    Mismatch (chưa xong) → cortisol (Cortisol-Baseline) → imagine (tìm giải pháp)
    → chưa ra → mismatch VẪN CÒN → cortisol VẪN CÒN → imagine TIẾP → ...

  ĐẶC ĐIỂM:
    → KHÔNG chọn vào → VÔ TÌNH bị cuốn (vì mismatch → cortisol TỰ ĐỘNG)
    → KHÔNG dễ dừng → dừng imagine ≠ dừng mismatch → cortisol VẪN CÒN → loop quay lại
    → VỪA khó chịu VỪA hứng thú → Body-Dissonance (mismatch) + dopamine (khi gần insight)
    → CÓ GIÁ TRỊ → mỗi vòng DẦN gần domain reality hơn (nếu chunks đủ)
    → CÓ HẠI NẾU → kéo dài + thiếu repair → Neural Wear tích lũy

  = Tính năng MẠNH NHẤT của Imagine:
    → Cho phép con người tìm ra E=mc², sáng tác nhạc, build framework
    → = Cái MÀ trial-error KHÔNG BAO GIỜ tới được (Imagination-Analysis §2)
  = NHƯNG CŨNG nguy hiểm nhất:
    → Newton mental health decline, Tesla OCD, burnout, mất ngủ
    → = Cùng mechanism → sáng tạo HOẶC tự hại → tùy cách quản lý loop
```

## 2. Cơ Chế — Tại Sao Loop KHÔNG Tự Dừng

```
LOOP CHI TIẾT:

  ① MISMATCH khởi tạo:
     → "Muốn X" (body-need) ≠ "chưa có X" (reality)
     → = Schema detect gap → body signal "THIẾU"
     → Ví dụ: "code chưa xong", "framework chưa mượt", "bài hát chưa hay"

  ② CHANGE-READINESS tăng (cortisol):
     → Mismatch → body: "cần thay đổi" → cortisol tăng
     → Cortisol → neurons rung → vô thức THỬ pattern mới
     → VTA detect biến động → dopamine → PFC tham gia
     → = Imagine ACTIVATE

  ③ IMAGINE tìm giải pháp:
     → PFC draft → body simulate → "khớp không?"
     → NẾU RA: mismatch RESOLVE → cortisol GIẢM → loop DỪNG ✅
     → NẾU CHƯA RA: mismatch VẪN CÒN → quay lại ②

  ④ LOOP DUY TRÌ vì:
     → Dừng imagine ≠ dừng mismatch
     → Mismatch vẫn → cortisol vẫn → vô thức VẪN rung → VTA VẪN fire
     → = "Cố KHÔNG nghĩ" → nhưng vô thức VẪN process → PFC VẪN nhận signal
     → = Giống cố KHÔNG nghe nhạc dính (earworm) → càng cố → càng nghe rõ

  ⑤ CẢ KHI NGỦ:
     → Cortisol cao → neurons rung CẢ khi ngủ → REM quality GIẢM
     → Vô thức VẪN "brainstorm" (🟢 REM = vô thức process tự do)
     → NHƯNG: sleep quality KÉM → repair THIẾU → sáng dậy: MỆT + chưa ra
     → = "Nằm giường mà đầu vẫn chạy" = Cortisol-Baseline CAO + VTA vẫn fire
     → = Neural Wear TÍCH LŨY qua mỗi đêm thiếu repair

  TẠI SAO KHÔNG TỰ DỪNG:
    → Loop CÓ reward (gần insight = dopamine) → DUY TRÌ drive
    → Loop CÓ threat (chưa xong = cortisol) → DUY TRÌ urgency
    → Reward + Threat CÙNG LÚC = "vừa muốn dừng vừa không thể dừng"
    → = "Nghiện sáng tạo" = ĐÚNG cơ chế — reward loop + threat loop CHỒNG
    → Dừng = MẤT reward (dopamine) + VẪN CÓ threat (mismatch)
    → = "Dừng = TỆ HƠN tiếp" → não CHỌN tiếp (dù body mệt)
```

### 3.0 — "Tension" — Trải Nghiệm Cốt Lõi Của Loop

```
TENSION = trạng thái KHÓ CHỊU + KHÔNG THỂ BỎ trong novelty loop.

  Thuật ngữ: Einstein mô tả giai đoạn stuck là "a feeling of tension"
  — trạng thái căng thẳng nội tại khi vấn đề CHƯA giải quyết được.
  Poincaré dùng "agitation" (kích động). Nhiều nhà khoa học dùng "nagging" (dai dẳng).
  File này dùng "TENSION" theo nghĩa Einstein: căng thẳng sáng tạo.

  TENSION ≠ stress (stress = cortisol quá tải, tiêu cực)
  TENSION = Body-Dissonance VỪA ĐỦ giữ não ở change-mode, vừa khó chịu vừa productive
  (cortisol sustain trạng thái, nhưng Body-Dissonance = NE + adrenaline + CRH tạo cảm giác)

  ⭐ TENSION CẢM NHẬN KHÁC TÙY MODALITY DOMINANT:

    Somatic-dominant:
      → Cảm nhận TRONG CƠ THỂ: bồn chồn, ngồi không yên, "ngứa" bên trong
      → "Biết có gì đó chưa đúng nhưng KHÔNG NÓI ĐƯỢC là gì"
      → Đi lại, vận động, thay đổi tư thế → cố giải tỏa tension qua body
      → Đầu có thể NHẸ (vì PFC không overwork — vô thức process ở somatic)

    Verbal-dominant:
      → Cảm nhận TRONG ĐẦU: suy nghĩ lặp lại, "quay vòng", nặng đầu
      → "Cứ nghĩ mãi cùng 1 vấn đề, mỗi lần hơi khác"
      → Nói ra, viết ra, thảo luận → cố giải tỏa tension qua language
      → Body có thể bình thường (vì process ở vùng ngôn ngữ gần PFC)

    Visual-dominant:
      → Cảm nhận HÌNH ẢNH: "vấn đề cứ hiện lên", "nhìn đâu cũng thấy nó"
      → "Thấy hình ảnh giải pháp nhưng chưa rõ, mờ mờ"
      → Vẽ, phác thảo, hình dung → cố giải tỏa tension qua visual
      → Body + đầu có thể bình thường (process ở visual cortex)

    → = CÙNG mechanism (cortisol + mismatch + VTA fire)
    → = KHÁC trải nghiệm chủ quan tùy modality
    → = Tại sao mô tả tension KHÁC NHAU giữa các nhà khoa học/creator:
        Einstein (visual+verbal): "feeling of tension" + "thought experiments" (visual)
        Poincaré (verbal): "ideas collide" (verbal processing)
        Tesla (visual extreme): "images appear before my eyes" (vivid visual)
        Nhiều nghệ sĩ (somatic): "restless", "can't sit still", "something burning inside"
```

### 3.1 — Hai Loại Novelty Trigger

```
LOẠI 1 — NHÌN THẤY TRỰC TIẾP (perception-triggered):
  Thấy con cá đẹp → tiến lại → sờ → cho ăn → vui → chán → đi
  → Novelty reward: CÓ (opioid)
  → Threat: GẦN NHƯ KHÔNG
  → Thời gian: PHÚT
  → Depth: NÔNG
  → = "Novelty snack" — body satisfy NHANH → dừng
  → = Mọi người đều trải nghiệm hàng ngày (scroll feed, thấy cái hay, xem xong quên)

LOẠI 2 — TƯỞNG TƯỢNG (imagination-triggered):
  Chunks tích lũy đủ → PFC detect mismatch giữa chunks → imagine giải pháp
  → BẮT BUỘC có chunks tích lũy MỨC NÀO ĐÓ mới trigger được
  → Novelty reward: CÓ nhưng KHÔNG LIÊN TỤC (sướng → tension → sướng → ...)
  → Thời gian: GIỜ → NGÀY → THÁNG → NĂM
  → Depth: SÂU
  → = "Novelty loop" thật sự — cái tạo ra breakthrough

  Loại 2 CẦN gì để KHÔNG dừng giữa chừng?
  → Cần CÁI GÌ ĐÓ giữ drive DƯƠNG qua giai đoạn tension
  → = Threat-schema cung cấp cái SÀN
```

### 3.2 — Threat-Schema = Sàn Giữ Loop

```
KHI NOVELTY REWARD TẠM = 0 (giai đoạn tension):

  KHÔNG CÓ threat nền:
    Drive = novelty(0) - cost(effort) = ÂM
    → Body: "bỏ đi, làm gì khác sướng hơn" → DỪNG
    → = Phần lớn mọi người dừng ở đây

  CÓ threat nền:
    Drive = novelty(0) + threat-push(dương) - cost(effort)
    → Threat-push = cortisol từ threat-schema → "KHÔNG THỂ dừng vì..."
    → NẾU threat-push > cost → VẪN TIẾP TỤC dù không sướng
    → Tới khi novelty spike lại → drive DƯƠNG MẠNH → euphoria
    → Euphoria CÀNG MẠNH vì vừa thoát giai đoạn threat → contrast effect

  → = Threat-schema là SÀN: giữ drive KHÔNG RƠI xuống ÂM
  → = Khi novelty spike → sàn + spike = TỔNG rất cao → euphoria

  ⭐ SCALING + RÚT SÀN (xem Personal-Melody.md §5):
  → Investment cost NHỎ → không cần threat sàn (curiosity/carrot đủ)
  → Investment cost LỚN → threat sàn CẦN THIẾT (carrot không đủ mạnh)
  → Khi chunks ĐỦ → melody mới emerge → intrinsic reward take over
  → LÚC ĐÓ: RÚT sàn threat → cho tiếp = cortisol thừa, cho carrot tiếp = lạm phát
  → = Sàn là TẠM → mục tiêu = intrinsic reward thay thế sàn
```

### 3.3 — 4 Mức Depth Của Loop

```
MỨC 1 — PURE NOVELTY (không threat):
  Trigger:  Nhìn thấy trực tiếp (con cá, video hay, đồ chơi mới)
  Threat:   KHÔNG
  Cortisol: Chỉ micro-mismatch thoáng qua
  Duration: PHÚT
  Output:   Trải nghiệm thú vị, không breakthrough
  Ví dụ:    Scroll feed, ngắm cảnh, chơi đồ chơi
  → = "Ăn vặt Novelty" — vui nhưng không sâu

MỨC 2 — NOVELTY + MICRO-MISMATCH NỘI TẠI:
  Trigger:  Bắt đầu làm gì đó → "chưa xong" = mismatch NHỎ tự nhiên
  Threat:   Không external, chỉ "chưa xong" → cortisol NHỎ
  Cortisol: Thấp-trung bình, TỰ SINH từ quá trình novelty
  Duration: GIỜ
  Output:   Hoàn thành task, học skill, giải puzzle
  Ví dụ:    Giải sudoku, chơi game, hobby cuối tuần, đọc sách hấp dẫn
  → = "Bữa chính Novelty" — có depth, nhưng dừng được khi mệt

MỨC 3 — NOVELTY + THREAT-SCHEMA NỀN (external, ongoing):
  Trigger:  Chunks đủ + threat cuộc sống tạo cortisol nền
  Threat:   External: tài chính, cạnh tranh, kỳ vọng, deadline, hardship
  Cortisol: Trung bình-cao, LIÊN TỤC từ threat (không phụ thuộc novelty)
  Duration: THÁNG → NĂM
  Output:   Breakthrough, tác phẩm lớn, phát minh
  Ví dụ:    Einstein (nghèo + academic pressure), Newton (plague isolation),
            Tesla (tài chính + cạnh tranh), startup founders
  → = "Marathon Novelty" — sâu, dài, nhưng KHÔNG KIỂM SOÁT ĐƯỢC threat
  → = Threat giữ loop DÙ KHI KHÔNG MUỐN → risk burnout/mental health

MỨC 4 — NOVELTY + THREAT-SCHEMA TỰ TẠO (deliberate):
  Trigger:  Chunks đủ + TỰ TẠO threat ở L3 (controllable)
  Threat:   Self-generated: "đối thủ đang tiến", "đời ngắn", "chưa đủ tốt"
  Cortisol: Trung bình, BẬT/TẮT ĐƯỢC (vì L3 = PFC accessible)
  Duration: NĂM → CẢ ĐỜI (sustainable)
  Output:   Đế chế, legacy, sustained innovation
  Ví dụ:    Jensen Huang, Jeff Bezos, researchers chiến lược
  → = "Lifestyle Novelty" — sâu + dài + KIỂM SOÁT ĐƯỢC
  → = TỐI ƯU NHẤT vì: threat VỪA ĐỦ + tắt được khi cần repair
```

### 4.2 — Tại Sao CEO BẬT/TẮT Được Mà Người Thường Khó

```
CEO threat = L3 (PFC accessible):
  → PFC TẠO threat → PFC CÓ THỂ TẮT threat
  → Họp chiến lược: "chúng ta sắp thua" → cortisol ON → team chạy
  → Về nhà cuối tuần: tắt → thư giãn → repair → recovery
  → = Giống bật/tắt công tắc (vì L3 = PFC territory)

Người bị anxiety: threat ở L1-L2 (compiled SÂU, vô thức):
  → Body tự threat → PFC KHÔNG can thiệp được → không tắt được
```
