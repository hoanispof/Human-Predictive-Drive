```

      → Con nghe melody A + B hàng ngày → chunks từ A và B
```

## 3. Cultural Melody — Mỗi Văn Hóa 1 KEY

```
🟡 Mỗi văn hóa = 1 KEY (giọng nhạc) mà personal melodies CHƠI TRONG:

  KEY = "luật chơi" chung của dàn nhạc lớn:
    → Mỗi KEY cho phép 1 số notes → CẤM 1 số notes khác
    → = "Trong văn hóa NÀY, hành vi NÀY ok → hành vi KIA KHÔNG ok"


  VÍ DỤ cultural KEYs:

    Việt Nam (key: Family + Hierarchy + Education):
      → Notes OK: tôn trọng lớn tuổi, gia đình ưu tiên, học giỏi
      → Notes KHÔNG OK: nói trái ý bề trên, bỏ học, sống 1 mình
      → Melody TRONG key: "con ngoan, học giỏi" → harmony → status ok
      → Melody NGOÀI key: "con bỏ học, đi làm riêng" → bị phạt

    Mỹ (key: Individual + Innovation + Competition):
      → Notes OK: tự lập, dám khác, thắng
      → Notes KHÔNG OK: phụ thuộc, copy, thua
      → Melody TRONG key: "startup founder" → harmony
      → Melody NGOÀI key: "sống nhờ bố mẹ 30 tuổi" → dissonance

    Nhật (key: Group Harmony + Precision + Respect):
      → Notes OK: hòa hợp nhóm, chính xác, kính trọng
      → Notes KHÔNG OK: nổi bật cá nhân, sai sót
      → Melody TRONG key: "nhân viên tận tụy" → harmony
      → Melody NGOÀI key: "nổi loạn cá nhân" → dissonance MẠNH


  CULTURE SHOCK = KEY CHANGE đột ngột:
    → Người Việt → sang Mỹ: key Family → key Individual
    → Personal melody VN chơi trong key Mỹ = LỆCH notes → cortisol
    → Phải learn key mới: thêm notes (tự lập) + giảm notes (chờ ý kiến)
    → Thời gian adapt: tháng → năm (dần shift melody cho fit key mới)
    → = KHÔNG phải "yếu đuối" → melody ĐANG ở KEY KHÁC

  KEY CHANGE TRONG CÙNG NƯỚC:
    → Nông thôn → thành phố: key KHÁC (tốc độ, cạnh tranh)
    → THPT → đại học: key KHÁC (tự do hơn)
    → Đi làm → nghỉ hưu: key KHÁC (mất vai trò, mất nhóm)
    → = MỖI CONTEXT SWITCH = mini key change → cần adjust melody
```
