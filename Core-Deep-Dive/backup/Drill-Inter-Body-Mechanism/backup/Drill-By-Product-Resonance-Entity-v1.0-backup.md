```

    Duration: months-years of regular interaction
    = "Bắt đầu quen, gặp thoải mái hơn, ít effort"

      → Interaction patterns COMPILE → automatic
```

## §2 — RESONANCE × MẸ-CON (asymmetric)

### §2.1 — Mẹ → Con

```
⭐⭐ MẸ→CON = UNIQUE RESONANCE PROFILE:

  GAP TYPE: EXISTENCE-BASED (Drill-Individual-Gap §7)
    Con tồn tại = fill parental drive gap LIÊN TỤC
    = Type B satiation (slow habituation) + hardware oxytocin ANTI-HABITUATION
    → Duration = INFINITE while child alive
    → No satiation point (unlike hunger: full=stop)
    
  COMPILATION CHAIN:
    Mẹ SKIP Stage 1-2 (partially):
      Prenatal: hormone priming → brain restructure (Hoekzema 2017)
      Birth: oxytocin surge → IMMEDIATE baseline precursor
      → F1 ALREADY partially compiled before significant interaction
      → Stage 3 reached FASTER than any other bond
      → = "Con chưa làm gì, mẹ đã thương"
      
  RESONANCE MECHANISM:
    MẸ'S GAP: "con sống, con khỏe, con phát triển" (parental drive)
    CON'S BY-PRODUCT: tồn tại, cười, lớn lên (just being alive = by-product)
    → CON'S by-product match MẸ'S gap = AUTOMATIC
    → = RESONANCE GUARANTEED BY ARCHITECTURE (child exists → mẹ rewarded)
    → NHƯNG: one-sided guarantee (only mẹ→con direction)
    
  HARDWARE BOOST:
    Oxytocin: sustain reward → slow habituation
    Baby schema: visual trigger → nucleus accumbens within 140ms (Glocker 2009)
    Biobehavioral synchrony: 4-system sync (Feldman 2012)
    → Resonance AMPLIFIED beyond any non-kin bond
    
  VTA DYNAMICS:
    Con present = baseline → VTA habituated → "bình thường"
    Con absent → Chunk-Miss → "nhớ con"
    Con achievement → prediction-delta > 0 → dopamine → "tự hào"
    Con threat (ốm, gặp nguy) → negative delta → cortisol → "lo lắng cực độ"
    
  🟢 Prenatal attachment: Cranley 1981, Feldman 2007
  🟢 Brain restructure: Hoekzema 2017 (Nature Neuroscience)
  🟢 Baby schema: Glocker 2009 (PNAS), Lorenz 1943
  🟡 Existence-based resonance as unique type = framework synthesis
```

```

    CON'S GAPS: hunger + comfort + safety + emotional + guidance

      Con reward: MULTI-GAP fill (concrete: food, warmth, safety)

  ⭐ ASYMMETRY SUMMARY:
    MẸ→CON: existence-based + hardware-boosted + skip stages → STABLE POSITIVE
    CON→MẸ: episodic + interaction-shaped + lifecycle shift → MIXED + SHIFTING
    
  🟢 Schaffer & Emerson 1964: specific attachment ~7 months
  🟢 Harlow 1958: attachment ≠ food reward (contact comfort)
  🟢 Lüscher & Pillemer 1998: 50% parent-child ambivalence = normative
  🟡 Asymmetric resonance profile = framework synthesis
```

```
    → "Không thể sở hữu bạn" = architecturally correct
```

## §4 — RESONANCE × ROMANTIC

```

      → Dopamine create artificial gap: "need THIS PERSON"
      → NOT body-need gap → HORMONE gap
      → Fill = partner presence → dopamine relief
      → Duration: ONGOING while limerence active
      → Satiation: NO (hormone sustain drive)
      
    DANGER: compile Entity-Owned UNDER HORMONE → NOT representative
      → Compile: "partner = fill ALL gaps" → INFLATED
      → Reality post-hormone: partner fill SOME gaps → mismatch
      → = "Hết yêu" = hormone gone, Entity-Owned INFLATED → reality < baseline
      
  PHASE 2 — POST-LIMERENCE (attachment or dissolution):
    Hormone FADED → resonance MUST carry the bond:
    
    TRAJECTORY A — GENUINE RESONANCE:
      By-product match existed underneath limerence
      → Now visible without hormone mask
      → Both grow riêng → new by-products → sustained reward

      By-product match was NEVER real (hormone-only bond)

      → Entity-Compiled built nhưng valence SHIFT negative

    EC deep (years compiled) + VTA habituated + no hormone boost
    → Resonance depends ENTIRELY on:
      ① Drive riêng maintained (both keep growing)
      ② Shared novel experiences (new by-products)
      ③ No compiled suppress gap riêng
    → Missing ANY → trajectory B or C
    → Missing ALL → trajectory C inevitable
    
  🟢 Fisher 2004: limerence = drive, not attachment
  🟢 Fisher 2010: rejected lovers → cocaine-like circuits
  🟢 Acevedo 2012: long-term love → VTA + vasopressin/oxytocin
```
