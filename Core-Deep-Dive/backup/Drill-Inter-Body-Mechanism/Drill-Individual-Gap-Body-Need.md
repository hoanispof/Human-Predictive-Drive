### §2.1 — CYCLIC (cũ: "Type A" — Sharp Satiation)

```
⭐ CYCLIC = Fill → satiation CLEAR → gap DORMANT → return after interval:

  Đặc điểm:
    → Hardware mechanism tắt reward khi fill đủ
    → "Full = stop" rõ ràng (hormonal signal: leptin/ghrelin, thermoregulation)
    → Gap return theo hardware clock (predictable interval)
    → Entity fill cyclic gap = needed in BURSTS, not continuously
  
  VÍ DỤ:
    Hunger: fill 15-30 phút → leptin/ghrelin signal → stop
            → Return 3-6h → gap re-activate → CYCLE
    Temperature: hot→cool = SHARP relief → baseline met → INVISIBLE
            → Shift environment = immediate return
    Safety: threat removed → safe = baseline → DORMANT
            → Threat appears = gap INSTANT activation
    Sexual: fill minutes → refractory period → return hours/days
    
  REWARD COMPOSITION (RSA integration):
    Hunger = Type A dominant (~80A/20B): taste EVALUATION + physical satiation
    Temperature relief = Type B dominant (~20A/80B): thermoregulator direct
    Safety relief = Type A dominant (~70A/30B): threat EVALUATION resolves
    → Cyclic gaps VARY in A/B ratio — satiation profile ≠ reward type
    
  🟢 Satiety hormones: leptin, ghrelin (established endocrinology)
  🟢 Thermoregulation: homeostatic baseline (established physiology)
```

### §2.2 — TONIC (cũ: "Type B" — Slow Habituation)

```
⭐ TONIC = Fill → habituate SLOWLY (Weber-Fechner) → NO sharp cutoff:

  Đặc điểm:
    → Opioid system: sustained low-level reward
    → "Mãi không chán" WHILE entity present
    → Weber-Fechner CHẬM (hours not minutes for social/touch)
    → No satiation hormone equivalent (unlike hunger)
    → Entity fill tonic gap = needed ONGOING, not in bursts
    → Reset qua ABSENCE + return (de-habituate → re-fill = "ấm hơn")
  
  VÍ DỤ:
    Touch/comfort (ôm): ongoing opioid → hours → gradual decline
      → "Ôm mẹ mãi không chán" = no sharp cutoff
      → INTERRUPT bởi cyclic gap (đói) hoặc generative gap (curiosity)
    Social connection: ongoing while with agent → gradual decline
      → Person-dependent: hardware need varies (extrovert > introvert)
    Maintained temperature: initially cyclic (relief) → transitions to tonic
      → "Mặc áo mát mẻ": first = sharp relief, then = baseline = invisible
  
  REWARD COMPOSITION (RSA integration):
    Touch/comfort = Type B dominant (~20A/80B): CT afferents direct
    Social presence = Type A+B compound (~50A/50B): evaluation + proximity hardware
    → Tonic gaps TEND toward Type B nhưng KHÔNG tuyệt đối
    
  ⭐ CYCLIC → TONIC TRANSITION:
    Nhiều gaps START as cyclic (sharp relief) → TRANSITION to tonic (baseline):
      Temperature: hot→cool (cyclic) → maintained cool (tonic → baseline → invisible)
      Safety: threat→safe (cyclic) → maintained safe (tonic → baseline → invisible)
    = "Relief → Baseline → Invisible" trajectory
    
  🟢 Weber-Fechner law: established psychophysics
  🟢 Opioid system + touch: CT afferent fibers (Löken 2009)
  🟢 Hedonic adaptation: Bao & Lyubomirsky 2013
```

### §2.3 — GENERATIVE (cũ: "Type C" — Self-Generating)

```
⭐ GENERATIVE = Fill → CREATE new gaps → never "done" → perpetual cycle:

  Đặc điểm:
    → Fill gap → discover new gaps → chase → fill → discover more
    → Reward = VARIABLE (depends on prediction-delta per discovery)
    → High discovery = HIGH reward (dopamine burst)
    → Repeated known = LOW reward (prediction confirmed, no delta)
    → Entity fill generative gap = needed for NOVELTY, not for presence
  
  VÍ DỤ:
    Curiosity/novelty: learn X → see Y unknown → gap for Y → chase
      → "Học toán → thấy bài khó mới → muốn giải" = self-generating
    Domain mastery: solve problem → see bigger problem → gap expands
      → PhD: chapter → experiment → thesis = mini-arc decomposition (BFM §3.3d)
    Status (comparative): achieve → baseline shift → need MORE
      → "Lương $1M → adapt → $10M → vẫn thiếu" = escalating baseline
      
  ⭐ GENERATIVE + ENTITY:
    Entity RELEVANT khi entity CONTRIBUTE to novelty:
      → Bạn thân chơi cùng: nếu tạo NEW discoveries → generative reward
      → Bạn thân same routine: no new discovery → generative ↓ → "chán"
      → Partner support drive riêng: new by-products → generative reward
      → Partner suppress drive riêng: no new by-products → "chán"
  
  REWARD COMPOSITION (RSA integration):
    Curiosity/insight = Type A dominant (~95A/5B): pattern EVALUATION
    Status comparison = Type A dominant (~90A/10B): social EVALUATION
    Physical mastery (sport) = Type A+B compound (~60A/40B): skill eval + body state
    → Generative gaps TEND toward Type A — requires cortical evaluation
    
  🟢 VTA habituation + prediction-delta: Schultz 1997
  🟢 Mini-arc decomposition: BFM §3.3d
  🟢 Self-expansion model: Aron & Aron 1996, 2000
  🟡 "Generative" as explicit satiation profile = framework synthesis (renamed v2.0)
```

### §2.4 — Profile Transitions + Compounds

```
⭐ SATIATION PROFILES KHÔNG TĨNH — CÓ THỂ TRANSITION VÀ COMPOUND:

  TRANSITIONS:
  
    CYCLIC → TONIC (§2.2 đã nói):
      Temperature: relief (cyclic) → maintained (tonic → invisible)
      Safety: resolved (cyclic) → maintained (tonic → invisible)
      = "Relief → Baseline → Invisible" trajectory
      
    TONIC → GENERATIVE (less common):
      Social connection: ongoing (tonic) + shared novel experiences → generative
      Touch: ongoing (tonic) nhưng KHÔNG tự generate new gaps
      = Transition CẦN external novelty input — tonic KHÔNG tự generate
      
    GENERATIVE → TONIC (possible):
      Curiosity domain: active pursuit (generative) → mastery → domain "hết gap"
      → Shifts to maintenance mode (tonic) — "still enjoy but no new frontiers"
      → RARE vì generative BY DEFINITION create new gaps
      → CAN happen: domain FINITE (all chess openings memorized) nhưng rất hiếm
      
  COMPOUNDS:
  
    CYCLIC + GENERATIVE (phổ biến):
      Hunger cho cuisine cụ thể = cyclic (physical hunger) + generative (novelty seeking)
      → "Đói" = cyclic. "Muốn ăn món mới" = generative ON TOP OF cyclic.

      → Mất generative → chỉ còn tonic → "gặp thì ổn nhưng không exciting"
      
    CYCLIC + TONIC (less common):
      Sexual + emotional bonding = cyclic (physical) + tonic (emotional ongoing)
      
  ⭐ COMPOUND MODEL GIẢI THÍCH "CHÁN NHAU":
    Romantic relationship ban đầu: tonic (comfort) + generative (novelty)
    Sau nhiều năm: generative HABITUATED → chỉ còn tonic
    → "Vẫn thoải mái nhưng không exciting" = tonic intact, generative dead
    → Drill-Boredom-Mechanism §3: Source ⑥ = by-product match dừng
    → FIX: rebuild generative (new shared experiences, maintain drive riêng)
    
  🟡 Profile transitions + compounds = framework synthesis
  🟢 Consistent with: Aron 2000 (novel activities restore satisfaction)

### §3.2 — Reward intensity × gap depth × satiation state

```
⭐ REWARD ≠ FIXED. REWARD = f(gap_depth × fill_quality × satiation_state):

  Hunger example:
    Gap depth SHALLOW (ăn 2h trước): fill reward = THẤP (not very hungry)
    Gap depth DEEP (nhịn 12h): fill reward = CAO (very hungry)
    Fill quality CAO (món ngon): reward AMPLIFIED (A₂ evaluation + B satiation)
    Fill quality THẤP (cơm nguội): reward REDUCED (A₁ basic + B satiation)
    NHƯNG: satiation OVERRIDE → "no rồi thì không muốn ăn dù món ngon"
    → Reward CAPPED by satiation state (cyclic = clear cap)
    
  Touch/comfort example:
    Gap depth varies by person (hardware: CT afferent density)
    HIGH CT density: cần ôm NHIỀU → gap deeper → reward per ôm HIGHER
    LOW CT density: cần ôm ÍT → gap shallower → reward per ôm LOWER
    NO sharp satiation → reward continues (slow decline, tonic profile)
    → Person-dependent "enough" point (hardware variation)
    
  Curiosity example:
    Gap depth depends on SURROUNDING CHUNKS (how much you already know)
    Expert: deep surrounding chunks → deep gap → fill = HIGH reward (A₃)
    Novice: shallow surrounding → shallow gap → fill = MODERATE reward (A₁)
    → "Chưa biết = không có gap" → expert has DEEPER gaps than novice
    → Paradox: expert MORE curious, not less (deeper gaps, more borders)
    
  ⭐ RSA DIMENSION ADDS:
    Cùng gap, cùng depth → REWARD QUALITY khác tùy A:B ratio:
    → Touch (B dominant): reward = "warm, soothing" (body-state, no evaluation)
    → Insight (A dominant): reward = "eureka, exciting" (evaluative, cortical)
    → Food (A+B compound): reward = "ngon" (taste evaluation + body satiation)
    → RSA §4: 5 reward profiles (Sensory, Coherence, Social, Relief, Preview)
    → Mỗi gap fill = 1 hoặc nhiều profiles COMPOUND
    
  🟢 Effort-proportional reward: BFM §3.3a (03-Reward.md §4.7)
  🟢 Satiation override: leptin/ghrelin system
  🟢 5 reward profiles: RSA v2.0 §4 (Tik et al. 2020, Kounios & Beeman 2009)

  🟡 Gap lifecycle model = framework synthesis (v2.0)
