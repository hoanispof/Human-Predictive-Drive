# Plan — Consciousness Control Drill (Phase C)

> **Phase A: Can consciousness ENTER in unusual conditions?**
> **Phase B: How does consciousness QUALITY vary in extremes?**
> **Phase C: What happens when consciousness CONTROL fails?**
>
> Hallucination, intrusive thoughts, rumination, flashbacks —
> patterns that get ON the stage and WON'T LEAVE.
> The framework says PFC = gate + resolve + sustain (§6.1).
> What happens when gate FAILS, resolve FAILS, or sustain OVERRIDES?

### Session 1: Hallucination × Consciousness Architecture
**Topic**: False/involuntary patterns on consciousness stage
**Status**: `DONE (2026-07-01)`

```
MỤC TIÊU:
  ① Map hallucination through 5-Checkpoint model:
     → Which CPs involved? All 5 PASS (that's why it feels REAL)
     → Where does the FALSE signal originate?
     → Why does TRN gate (CP1) let a non-existent stimulus through?

  ② Map hallucination through 3-Parameter model:
     → Signal: INTERNAL source, no external input — but strong enough to ignite
     → Anchor: may or may not have verbal anchor (depends on type)
     → Attention: CAPTURED involuntarily (bottom-up pathway hijacked?)

  ③ Map hallucination through sustain hierarchy:
     → Primary sustain (thalamo-cortical loop): fires on false signal?
     → Additional sustain (PFC): cannot suppress → WHY?
     → "Cứ chạy mãi rồi tự tắt" → what EXIT mechanism?
        Signal decay? Competing signal wins? Oscillatory shift?

  ④ Types of hallucination → different pathways?
     → Schizophrenic (chronic, auditory dominant, dopamine excess)
     → Psychedelic (temporary, visual dominant, 5-HT2A → thalamic gating)
     → Hypnagogic/hypnopompic (sleep boundary, tonic→burst transition)
     → Fever/delirium (metabolic, arousal unstable)
     → Sensory deprivation (Ganzfeld effect — brain fills the void)
     → Charles Bonnet syndrome (visual loss → cortex generates)
     → KEY: Are these the SAME mechanism in different contexts,
       or DIFFERENT mechanisms at different checkpoints?

  ⑤ Hallucination vs dreaming:
     → Dreaming = "structured hallucination with PFC offline" (Phase A S2)
     → Waking hallucination = "dreaming with PFC ONLINE but UNABLE to suppress"
     → Same primary sustain, different PFC involvement?
     → Does this CONFIRM or CHALLENGE the sustain hierarchy?

  ⑥ Hallucination vs normal imagination:
     → "Tưởng tượng con voi hồng" = voluntary, PFC-initiated, can dismiss
     → Hallucination = involuntary, cannot dismiss
     → What distinguishes VIVID imagination from mild hallucination?
     → Framework prediction: signal strength + source attribution
       Imagination: PFC KNOWS source = internal → "I'm imagining this"
       Hallucination: brain DOES NOT tag source → "this is real"
     → Source monitoring failure? (Johnson, Hashtroudi & Lindsay 1993)

  ⑦ Framework predictions (testable)

QUESTIONS:
  Q1: Hallucination = "sân khấu sáng + kịch bản sai"
      hay "sân khấu bình thường + diễn viên giả"?
      (= thalamic state problem or content problem?)
  Q2: Tại sao ảo giác "thật" hơn tưởng tượng?
      (= what makes hallucination feel REAL vs imagination feel IMAGINED?)
  Q3: Tại sao PFC không suppress được ảo giác?
      (= PFC suppress fails against what kind of signal?)
  Q4: Schizophrenia vs psychedelic vs hypnagogic — cùng mechanism?
  Q5: Predictive coding: hallucination = "strong prior, weak/no data"?
      (= connection to prediction-delta framework §7.3, §8.2?)
```

### Session 2: Intrusive/Persistent Patterns × Consciousness Control
**Topic**: Patterns that resist PFC control — suppress fails, persist, loop
**Status**: `DONE (2026-07-01)`

```
MỤC TIÊU:
  ① Ironic process through 3-Parameter model:
     → "Đừng nghĩ con voi hồng" → PFC suppress requires HOLD
     → Hold = monitoring = target ACTIVE on stage
     → Suppress command CONTAINS the thing being suppressed
     → = Structural paradox, not willpower failure
     → Wegner 1987: ironic process theory → map to framework

  ② PFC suppress mechanism — when works vs when fails:
     → Success cases: dismiss unimportant thought, redirect attention
     → Fail cases: ironic process, OCD, rumination, PTSD
     → What determines success vs failure?
     → Hypothesis: signal strength × anchor strength × emotional charge
       Low signal + weak anchor + no emotion → suppress succeeds
       High signal + strong anchor + high emotion → suppress fails
     → Map to PFC-Operations.md §2 (Hold/Suppress)

  ③ Intrusive thoughts taxonomy (framework terms):
     → Normal intrusive thoughts (everyone has — DMN default, ~50% of time)
     → Ironic process (paradox: suppress = sustain)
     → Rumination (cortisol loop: negative thought → cortisol → PFC ↓ →
       cannot redirect → thought persists → more cortisol)
     → OCD intrusive thoughts (threshold too low? or suppress broken?)
     → PTSD flashbacks (emotional anchor SO strong → re-ignites involuntarily)
     → Earworms (compiled musical pattern loops on stage)
     → KEY: Same mechanism or different mechanisms?

  ④ Rumination × body-state loop (connect to Session 2 Phase B):
     → Rumination = vicious cycle: thought → cortisol → PFC ↓ → 
       thought persists → more cortisol
     → = Self-referential paradox (Session 2) applied to THOUGHT not body
     → = Body-state modulates consciousness (Session 2 finding)
       applied specifically to negative thought patterns

  ⑤ PTSD flashback × consciousness architecture:
     → Emotional anchor (amygdala-tagged) = EXTREMELY strong
     → Trigger → re-ignites entire pattern involuntarily
     → = Bottom-up push (§5.3) so strong PFC cannot override
     → = Signal pushes onto stage AGAINST PFC suppress
     → Connects to: limbic drive as sustain mechanism (Session 3 Phase B)

  ⑥ Exit mechanisms — HOW do persistent patterns eventually STOP?
     → "Con voi hồng": competing signals win → gradual displacement
     → Rumination: body-state change (sleep, exercise) → cortisol drops → 
       PFC capacity restored → redirect succeeds
     → OCD: exposure therapy = deliberately NOT suppressing →
       removes ironic process → habituation → signal weakens
     → PTSD: EMDR, prolonged exposure → emotional anchor weakened
     → Earworm: new music, engaging task → competitive displacement
     → = EXIT mechanisms map to framework:
       competitive displacement (§5.2), signal decay (§8.1),
       anchor weakening, cortisol cycle break

  ⑦ Framework predictions (testable)

QUESTIONS:
  Q1: Ironic process = PFC HOLD required for SUPPRESS = structural paradox?
      Hay có suppress pathway mà KHÔNG cần hold target?
  Q2: OCD intrusive = threshold quá thấp (everything enters) hay
      suppress mechanism broken (entries normal, exits blocked)?
  Q3: PTSD flashback = re-ignition from emotional anchor hay
      compiled pattern re-fires automatically?
  Q4: Tại sao earworms thường là nhạc? (compiled auditory patterns
      có property gì đặc biệt về looping?)
  Q5: Meditation = deliberate TRAINING of suppress/redirect capacity?
      (connects to attention parameter training, Session 2 interoception)

RESEARCH ĐỌC THÊM:
  → PFC-Operations.md v1.3 (§2 Hold/Suppress, §6 Override limits)
  → Consciousness.md v1.4 (§5.2 biased competition, §6.1 PFC 3 roles)
  → Background-Pattern.md v2.0 (§5 PFC invisibility, §6 triple bias)
  → Cortisol-Baseline.md v2.4 (§9 PFC damage timeline, §3.10 silent cortisol)
  → ADHD-Observation.md v1.0 (§8 DMN interference, §4.5 task switching)
  → Drill-Body-State-Consciousness.md v1.0 (self-referential paradox, vicious cycle)
  → Wegner 1987, 1994: ironic process theory
  → OCD intrusive thoughts: Rachman 1997, Salkovskis 1985
  → PTSD re-experiencing: Brewin et al. 2010 (dual representation theory)
  → Earworms: Williamson et al. 2012, Jakubowski et al. 2015
  → Rumination: Nolen-Hoeksema 1991, cortisol × rumination loop

OUTPUT:
  → Drill-Intrusive-Consciousness.md v1.0
  → Focus: intrusive/persistent patterns mapped to consciousness control model
```

## BIAS CHECKLIST (kiểm tra mỗi session)

```
⚠️ Biases cần cảnh giác:

  □ Pathologization bias: ảo giác/intrusive thoughts = "broken brain"
    → Nhiều hiện tượng này = NORMAL (DMN default, hypnagogic, earworms)
    → Chỉ pathological khi disrupts daily function

  □ PFC-centric bias: mọi thứ = "PFC fail"
    → PFC = controller, not only mechanism
    → Thalamic gating, limbic drive, signal strength = independent factors

  □ Mechanism conflation: treat different phenomena as same mechanism
    → OCD ≠ PTSD ≠ rumination ≠ earworm
    → May share components but different COMBINATIONS

  □ Dopamine oversimplification: "dopamine = answer"
    → Schizophrenia: dopamine excess (mesolimbic)
    → ADHD: dopamine deficit (mesocortical)
    → Same molecule, different pathway, different effect

  □ Post-hoc rationalization: ép phenomena vào model SAU KHI thấy
    → Test: can the framework PREDICT aspects of these phenomena
      BEFORE looking at the evidence?

  □ Clinical claim overreach: framework = theoretical, NOT clinical guide
    → Drill = understanding mechanism, NOT treatment recommendation
    → Always note when framework synthesis ≠ clinical evidence
```
