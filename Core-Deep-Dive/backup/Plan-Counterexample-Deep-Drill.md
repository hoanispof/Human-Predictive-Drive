## NGUYÊN TẮC DRILL

```
① THỰC TẾ LÀ ĐÚNG DUY NHẤT
   → Counterexample = sự thật lâm sàng/thực nghiệm
   → Framework = mô hình — mô hình có thể sai
   → Nếu mâu thuẫn: sửa mô hình, KHÔNG ép thực tế

② ADVERSARIAL APPROACH
   → Mỗi session: TÌM evidence PHÁ hypothesis trước
   → Chỉ khi không phá được → hypothesis đứng vững hơn
   → Tránh confirmation bias: không chỉ tìm evidence ủng hộ

③ 3 KẾT QUẢ HỢP LỆ
   → RESOLVED: hòa giải thuyết phục, evidence đủ mạnh hai phía
   → UNRESOLVED: evidence không đủ, cần ghi nhận open question
   → FRAMEWORK NEEDS FIX: counterexample đúng, framework sai → phải sửa

④ CHẤT LƯỢNG HƠN TỐC ĐỘ
   → 1 counterexample per session
   → Research kỹ → phân tích kỹ → ghi nhận trung thực
   → Compact session → session sau tiếp counterexample tiếp
```

## SESSION 1: HYDRANENCEPHALY — Consciousness Without Cortex?

### Tại sao quan trọng nhất

```
Nếu trẻ hydranencephaly THẬT SỰ có consciousness:
  → Toàn bộ thalamo-CORTICAL model bị thách thức
  → "Knowing" có thể KHÔNG cần cortex
  → Framework phải mở rộng scope radically

Nếu KHÔNG phải consciousness (chỉ là complex reflexes):
  → Model giữ nguyên
  → Nhưng phải giải thích CONVINCINGLY tại sao behaviors phức tạp ≠ consciousness
```

### Câu hỏi cần trả lời

```
Q1: Cấu trúc não CÒN LẠI chính xác là gì?
    → Thalamus: intact hay partial? Nuclei nào?
    → Brainstem: intact tới mức nào?
    → Cortical remnants: có hay không? Bao nhiêu?
    → Variation: tất cả cases giống nhau hay khác nhau đáng kể?

Q2: Behaviors quan sát được CỤ THỂ là gì?
    → Discrimination familiar vs stranger: mechanism?
    → Musical preference: chỉ arousal response hay real preference?
    → Social interaction: reactive hay intentional?
    → So sánh: insects cũng có behaviors phức tạp — consciousness?

Q3: "Affective consciousness" vs "complex reflexes" — phân biệt thế nào?
    → Panksepp's evidence cho affective consciousness ở subcortical level
    → Counterargument: Giả thuyết Morgan's Canon (parsimony)
    → Có cách test nào không? (khó — subjects non-verbal)

Q4: Nếu thalamus intact → thalamo-cortical loop CÓ THỂ chạy không?
    → Cortical remnants + thalamus = partial loop?
    → Thalamo-brainstem loop thay thế?

Q5: Merker's position hiện tại trong neuroscience mainstream?
    → Minority view hay gaining traction?
    → Phản bác chính từ mainstream là gì?
    → Papers phản bác Merker?

Q6: Framework implications nếu mỗi answer khác nhau?
    → Scenario A: consciousness = yes → framework phải sửa gì?
    → Scenario B: consciousness = no → framework confirm, nhưng giải thích gì?
    → Scenario C: unknown → framework ghi nhận open question thế nào?
```

### Evidence cần tìm

```
TÌM ỦNG HỘ "consciousness exists without cortex":
  → Aleman & Merker 2014 — chi tiết survey results
  → Merker 2007 — brainstem consciousness full argument
  → Panksepp — affective neuroscience evidence
  → Damasio — evolution of thought trên brainstem's role
  → Case reports chi tiết: specific behaviors, video evidence

TÌM PHẢN BÁC "this is NOT consciousness":
  → Mainstream criticisms of Merker
  → Papers arguing behaviors = reflexes
  → Morgan's Canon / parsimony arguments
  → Comparative: insect/simple organism behaviors
  → Papers on "philosophical zombie" problem in clinical neuroscience
```

→ File: Drill-CE-Hydranencephaly.md
→ Sections: Evidence, Challenges, Counterarguments, Framework Scenarios, Resolution

## SESSION 2: DREAMING — Consciousness with PFC Offline

### Tại sao quan trọng

```
Nếu dreaming = full consciousness mà KHÔNG CẦN PFC:
  → PFC role trong framework bị GIẢM đáng kể
  → "Knowing" definition cần revision (knowing without PFC?)
  → Sustain mechanism hierarchy cần rethink

Nếu dreaming = "degraded" consciousness (content without access):
  → Confirms hybrid model
  → PFC = access enabler (not source)
  → But: cần drill sâu WHAT sustains dreaming consciousness
```

### Câu hỏi cần trả lời

```
Q1: PFC CÓ THẬT SỰ offline hoàn toàn không?
    → dlPFC vs vmPFC vs mPFC — region-specific?
    → "Mostly quiescent" = bao nhiêu % activity?
    → NREM dreams vs REM dreams — PFC activity khác nhau?

Q2: Sustain mechanism cụ thể during dreaming?
    → Brainstem REM generator → thalamus → cortex: đủ giải thích?
    → Thalamo-cortical resonance alone (no PFC WM hold)?
    → Limbic drive (amygdala hyperactive) = alternative sustain?

Q3: Dreaming = consciousness ĐÚNG KHÔNG?
    → Dreaming clearly = subjective experience
    → Nhưng: "knowing" trong dream khác "knowing" lúc thức thế nào?
    → Dream reports: reliable evidence? (recalled after waking = biased?)
    → Non-reporting dreams: consciousness hay processing?

Q4: Lucid dreaming nói gì về PFC role?
    → PFC reactivates → meta-awareness appears
    → = PFC adds meta-level, not consciousness itself?
    → Hoặc: PFC UPGRADES consciousness from Level 1 → Level 2?

Q5: Anesthesia vs dreaming: same mechanism, different degree?
    → Propofol targets thalamo-cortical → abolishes dreaming
    → Ketamine preserves thalamic activity → preserves dreaming
    → = Evidence for thalamus as key, not PFC?

Q6: Framework implications?
    → Sustain mechanism hierarchy: primary vs additional?
    → "Knowing" definition: needs "internal vs external" dimension?
    → PFC role: access enabler or consciousness contributor?
```

### Evidence cần tìm

```
TÌM ỦNG HỘ "dreaming = consciousness without PFC":
  → fMRI/PET during REM sleep: exact PFC regions activity levels
  → Dream reports correlating with neural activity
  → Lucid dreaming studies: PFC reactivation → meta-awareness
  → Ketamine vs propofol: thalamic preservation → consciousness preservation

TÌM PHẢN BÁC:
  → PFC not truly offline (residual activity sufficient?)
  → Dreams may not be "real-time" experiences (reconstructed at waking?)
  → REM without dreams: dissociation between REM and dreaming
  → Arguments that dream consciousness IS access consciousness (just internal)
```

## SESSION 3: SPLIT-BRAIN — Unified Consciousness Without Corpus Callosum

### Tại sao quan trọng

```
Nếu consciousness UNIFIED sau callosotomy:
  → "Global" trong Global Workspace bị thách thức
  → Thalamus (subcortical) > cortical tracts cho consciousness unity
  → Framework's broadcast model cần revision

Nếu consciousness SPLIT (Sperry's view correct):
  → Model giữ nguyên
  → Corpus callosum = necessary for unity
  → Pinto 2017 findings = methodological artifact?
```

### Câu hỏi cần trả lời

```
Q1: Pinto 2017 — methodology có solid không?
    → Sample size?
    → Replication?
    → Methodological criticisms?
    → Response from Gazzaniga camp?

Q2: Subcortical pathways CỤ THỂ:
    → Thalamic commissures (Szczupak 2023) — replicated? In humans confirmed?
    → Anterior commissure — spared in all callosotomies?
    → Brainstem bilateral connections — carry consciousness info?
    → How much information can subcortical routes carry?

Q3: "Unified consciousness" vs "unified behavior"?
    → Cross-cueing (body language, context) → behavioral unity without consciousness unity?
    → Methodological challenge: how to test consciousness unity vs behavior unity?

Q4: Perception split vs consciousness unity — coherent?
    → Can consciousness be unified while perception is split?
    → What does "unified consciousness" mean exactly?
    → Philosophical clarity needed

Q5: Thalamus as unifier — mechanism?
    → Each hemisphere → own thalamus → how does that UNIFY?
    → Brainstem connects both thalami → unity through brainstem?
    → Intralaminar nuclei → bilateral projections?

Q6: Framework implications?
    → "Global broadcast" = via thalamic hub or cortical highways?
    → White matter tracts (SLF etc.): necessary or enhancing?
    → Thalamus role upgrade needed?
```

### Evidence cần tìm

```
TÌM ỦNG HỘ "consciousness unified via subcortical":
  → Pinto 2017 + follow-up papers
  → Thalamic commissures (Szczupak 2023, Cerebral Cortex 2024)
  → fMRI in callosotomy patients: subcortical connectivity
  → Volz & Gazzaniga 2017 response

TÌM PHẢN BÁC:
  → Gazzaniga's critique of Pinto's methodology
  → Arguments for split consciousness (classic view)
  → Cross-cueing as alternative explanation
  → Limitations of thalamic commissures research (new, limited replication)
```

## SESSION 5: STRESS-TEST — Thalamus WITHOUT Consciousness

### Tại sao cần drill này

```
Sessions 1-4 đã establish: thalamus = backbone/sufficient/unifier/relay.
NHƯNG: phần lớn thông tin qua thalamus KHÔNG thành conscious.
→ Nếu "thalamus = backbone" mà thalamus relay hàng triệu tín hiệu
  unconsciously → claim quá lỏng? Cần PRECISE hóa.

Đây KHÔNG phải counterexample mới — đây là REVERSE STRESS-TEST:
  Sessions 1-4: "consciousness WITHOUT [X]?" (cortex, PFC, CC, V1)
  Session 5: "thalamus WITHOUT consciousness?" (thalamic gating)

Mục tiêu: buộc framework phải chính xác về WHAT EXACTLY thalamus
contributes vs what it just relays.
```

### Câu hỏi cần trả lời

```
Q1: Bao nhiêu thông tin qua thalamus là unconscious?
    → Sensory gating: mọi giác quan (trừ khứu giác) qua thalamus
    → >95% bị filter — bởi cơ chế nào?
    → TRN (thalamic reticular nucleus) — role chính xác?

Q2: Cerebellum loops qua thalamus (VL/VA) — tại sao no consciousness?
    → Đã address sơ bộ (wrong architecture) nhưng cần drill sâu hơn
    → Basal ganglia → thalamus → cortex: cùng vấn đề

Q3: Core vs Matrix thalamic cells — cái nào route cho consciousness?
    → Jones 1998: core = specific content, matrix = arousal/readiness
    → Tại sao core relay KHÔNG đủ cho consciousness?
    → Matrix cells → widespread projection → consciousness?

Q4: Specific vs non-specific thalamic nuclei — role khác nhau?
    → Specific (LGN, VPL, MGB): sensory relay → mostly unconscious
    → Non-specific (CM-Pf, CL, MD): consciousness modulation
    → Tại sao cùng thalamus nhưng nuclei khác nhau → role khác?

Q5: Subliminal stimuli — thalamus activate mà no consciousness?
    → fMRI: LGN responds to subliminal visual stimuli
    → Signal tới thalamus → cortex nhưng BELOW threshold
    → Threshold ở đâu? Thalamus hay cortex?

Q6: Framework implications — thalamus claim cần precise thế nào?
    → "Thalamus = backbone" → QUÁ RỘNG?
    → Cần specify: WHICH nuclei, WHICH cell types, WHICH loop configurations?
    → Thalamus = NECESSARY GATE vs thalamus = GENERATOR?
```

### Evidence cần tìm

```
TÌM EVIDENCE "thalamus processes unconsciously":
  → TRN gating mechanism — how it selects what passes
  → Subliminal perception fMRI — thalamic activation patterns
  → Cerebellum/BG thalamic loops — why no consciousness
  → Proprioception gating — constant input, rarely conscious
  → Sleep gating — thalamic mode switching (tonic vs burst)

TÌM EVIDENCE "specific thalamic configurations → consciousness":
  → Intralaminar nuclei stimulation → awakening
  → Core vs matrix projection patterns
  → Thalamic lesion studies — which nuclei → consciousness disorders
  → Anesthesia — which thalamic mechanism disrupted
```
