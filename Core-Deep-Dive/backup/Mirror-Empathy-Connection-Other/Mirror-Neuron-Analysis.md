## Giả thuyết Framework: Mirror = Prediction Model Cho Agent Phi Logic

### Core idea:
KHÔNG CÓ neuron CHUYÊN cho mirror. Mirroring = LEARNED PATTERN khi
brain cần predict "vật thể phi logic" (agent có state riêng).

### Mechanism đề xuất:

```
BƯỚC 1 — Trẻ tương tác ĐỒ VẬT:
  → Input → action → output CỐ ĐỊNH (logic)
  → Bút: cầm → viết → hỏng → sửa. Pattern predict được.
  → Schema compiled → predict tốt → xong

BƯỚC 2 — Trẻ tương tác MẸ (agent):
  → Input → action → output PHI LOGIC
  → Cùng hành động (vẫy tay) → mẹ PHẢN ỨNG KHÁC tùy lúc
  → Schema đồ vật KHÔNG work → PHẢI BUILD model mới

BƯỚC 3 — Build "agent model":
  → Trẻ observe: "khi MÌNH vui → MÌNH muốn chơi"
  → Trẻ observe: "khi MẸ vui → MẸ chơi với mình"
  → Mapping: state MẸ ≈ state MÌNH (vì cùng modality: mặt, giọng, body)
  → → "MẸ có state riêng, state đó GIỐNG cách MÌNH có state"
  → → = Mirror = PREDICTION MODEL dùng own-state để predict other-state

BƯỚC 4 — Compile + generalize:
  → Lặp lại hàng nghìn lần → compiled thành pattern vô thức
  → Generalize: mẹ → bố → người khác → con chó → bất kỳ agent nào
  → Threshold: modality giống càng nhiều → map càng dễ → mirror càng mạnh
```

### Tại sao giải thuyết này GIẢI THÍCH ĐƯỢC nhiều hơn "innate module":

1. **Trẻ sơ sinh CHƯA có mirror thật**
   - Contagious crying = arousal contagion (chưa phân biệt self/other)
   - Neonatal imitation (Meltzoff) đang bị challenge (Oostenbroek 2016)
   - → Consistent: chưa có data → chưa build được model

2. **Mirror phát triển DẦN theo tuổi**
   - 14 tháng: helping behavior xuất hiện
   - 18-24 tháng: self-recognition (rouge test)
   - → Consistent: model CẦN thời gian build từ interaction data

3. **Chó mirror người TỐT nhưng không mirror cá**
   - 30.000 năm co-evolution → brain chó CÓ thời gian build model "agent-người"
   - Modality overlap chó-người: face expression, body language, voice
   - Mèo ít social co-evolution → mirror kém hơn
   - → Consistent: mirror = learned từ interaction, không innate cho MỌI species

4. **Tượng/ảnh thần CÓ THỂ "trigger empathy"**
   - Không có real input → nhưng compiled schema đủ mạnh → PFC simulate agent
   - Mẫu ảnh nhỏ ở cổ: schema trigger qua somatic (chạm), không cần visual input lớn
   - → Consistent: mirror = prediction model → model compiled → không cần real input

5. **Mirror GATE — suppress khi target = threat**
   - Hạ gục hổ/đối thủ → mirror suppressed
   - Sau chiến tranh → gate mở → PTSD
   - → Consistent: model predict "state agent" → body CHỌN có respond hay không

6. **Gương — trẻ ban đầu treat reflection như Other**
   - 0-5 tháng: social smile với reflection (= treat như agent)
   - 18-24 tháng: nhận ra = chính mình
   - → Consistent: model "agent" apply lên MỌI input visual có face → refine dần

7. **Screen time dưới 18 tháng**
   - Screen = visual input KHÔNG có real-time response
   - Trẻ chưa có chunks → không build được agent model từ screen
   - Screen characters: phi logic (biến mất, teleport) → model không convergence
   - → Không thêm chất liệu cho mirror development

  CƠ CHẾ (hypothesis):

    → Khóc người khác = UNFAMILIAR version → "pattern distress CHƯA resolve"
    → = Self-other discrimination RẤT SƠ KHAI

  Research confirm (Vreden 2025, thermal imaging, cross-cultural):

## Object-Agent Distinction — Research đã tách rõ

### Brain hardware cho animate detection (NOT mirror):
- VTC (ventral temporal cortex) có tổ chức RIÊNG cho animate vs inanimate
- 2 giai đoạn: ① categorize by shape (temporoparietal) → ② attention animate (frontal)

- Animate stimuli capture attention NHANH HƠN + GIỮ LÂU HƠN

### Phân biệt quan trọng:
- Hardware: "CÓ agent hay không" = BINARY FILTER (VTC, rất sớm, gần innate)

- = 2 thứ KHÁC NHAU — mainstream GỘP thành "mirror neuron"

## Arguments tại sao mirror KHÔNG THỂ là hardware chuyên biệt

1. **Cross-species empathy**: empathy với chó, mèo, wildlife
   → Hardware "chỉ compile hình người" → KHÔNG giải thích được

2. **Software design analogy**: dedicated module = đóng khung = khó mở rộng
   Evolution favor flexibility (general neurons + learned patterns)

3. **Training override**: 15% → 90% firing rate → schema overrideable
   Nếu hardware → training không override được
