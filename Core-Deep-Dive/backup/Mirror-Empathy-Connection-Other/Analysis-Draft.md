## Update 2026-04-12: Object-Agent Processing Modes

### Phát hiện chính (đã xác thực):
Não có 2 PROCESSING MODES cơ bản (Spelke 2007, VTC imaging 2019):
- **Object processing**: logic, deterministic, physics-based (cohesion, contact)
- **Agent processing**: model-based, non-deterministic, goal-directed

"Mirror" = BYPRODUCT của agent processing (own-state mapping lên other)
= Không phải module riêng — là CÁCH agent processing hoạt động.

### Hypothesis đang hình thành (chưa verify):
Empathy + Connection có thể là các NHÁNH/PATTERNS emerge từ agent processing:
- **Empathy** = agent processing APPLIED (detect + model state of other)
- **Connection** = agent processing SUSTAINED (2 agents model nhau liên tục)
- **Mirror** = mechanism agent processing DÙNG (own-state mapping)

Nhưng kết nối cụ thể CHƯA RÕ — cần phân tích thêm.
