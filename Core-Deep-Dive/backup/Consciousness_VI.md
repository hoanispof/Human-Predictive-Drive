## §1 — DEFINITION: Ý THỨC = "BIẾT"

### §1.1 Functional Definition

```
⭐ ĐỊNH NGHĨA:

  Ý THỨC (consciousness) =
    Phần processing mà OBSERVER CÓ THỂ ACCESS.
    = "Biết."

  VÔ THỨC (unconscious) =
    Phần processing mà observer KHÔNG access được.
    = "Không biết."

  "Biết" = observer có thể:
    → Report: nói ra được (dù mờ: "có gì đó không đúng")
    → Remember: ghi nhớ trải nghiệm
    → Use in decision: dùng trong quyết định
    → Act on it: hành động dựa trên nó

  Không cần ĐỒNG THỜI tất cả — SPECTRUM:
    → "Sao sao ấy" = report mờ + remember ngắn + khó decide
    → "Tôi buồn vì mất việc" = report + remember + decide + act
    → = MỨC ĐỘ "biết" khác nhau, không phải binary on/off

  ⭐ TOÀN BỘ BODY + BRAIN = VÔ THỨC (default state).
    Ý thức = SÂN KHẤU = phần NHỎ mà "bạn biết."
    ~95% processing = vô thức (Core-Software.md §4).

  🟢 ACCESS CONSCIOUSNESS (Block 1995):
    "Information available for reasoning, reporting, and action control"
    → "Biết" = framework's functional reframe of access consciousness

  ⚠️ GIỚI HẠN:
    → "Biết" = access consciousness ONLY
    → Phenomenal consciousness ("tại sao biết thì CẢM THẤY như vậy?")
      = Hard Problem (Chalmers 1995) → vẫn OPEN QUESTION
    → File này KHÔNG giải quyết Hard Problem

  🟢 Block 1995 | 🟢 Dehaene 2014: "global availability"
  🟡 "Biết" = framework reframe — tương đương access consciousness
```

### §1.3 "Biết" vs "Broadcast" — Compatible, Different Level

```
⭐ "BROADCAST" = CƠ CHẾ (HOW). "BIẾT" = CHỨC NĂNG (WHAT):

  GWT nói: ý thức = thông tin BROADCAST rộng đến nhiều vùng não.
  Framework nói: ý thức = BẠN BIẾT thông tin đó.

  CẢ HAI COMPATIBLE:
    → Broadcast = HOW (cơ chế: tín hiệu lan rộng → ignition)
    → Biết = WHAT (kết quả: observer access được)
    → GWT giải thích CƠ CHẾ bên dưới "biết"
    → "Biết" mô tả TRẢI NGHIỆM từ góc nhìn observer

  TẠI SAO "BIẾT" TỐT HƠN LÀM DEFINITION:
    ① "Biết" tự nhiên là SPECTRUM (biết mờ / biết rõ = tự nhiên)
    ② "Biết" không phụ thuộc push vs pull (cả hai → "biết")
    ③ "Biết" giải quyết counterexample:
       Chronic stress = cortisol broadcast khắp cơ thể
       → Nhưng KHÔNG "biết" → vô thức
       → "Broadcast" model phải giải thích thêm
       → "Biết" model: không biết → vô thức. Đơn giản.
    ④ "Biết" tương thích local → ý thức
       "Cái chấm kia xấu bức tranh" = local visual → nhưng biết → ý thức

  ⭐ ANALOGY:
    "Trọng lực" = WHAT (vật rơi xuống)
    General Relativity = HOW (spacetime curves)

    "Biết" = WHAT (tôi biết thông tin này)
    GWT = HOW (broadcast, ignition, thalamo-cortical loop)

  🟢 GWT: Baars 1988, Dehaene 2014
  🟡 "Biết" as primary definition = framework reframe
  Chi tiết test cases: Drill-Consciousness-Knowing.md §1.3
```
