## §3.3 TRN — The Physical Gatekeeper (NEW v2.0)

```
🟢 TRN (THALAMIC RETICULAR NUCLEUS) — FROM SESSION 5:

  ANATOMY:
    → Thin, shell-shaped, entirely GABAergic structure
    → Wraps around lateral and anterior thalamus
    → Every thalamocortical and corticothalamic fiber PASSES THROUGH TRN

  MECHANISM — BANDPASS FILTER:
    → TRN fires → INHIBITS corresponding thalamic relay neuron
    → TRN silent → thalamic relay PASSES signal to cortex

  HOW TRN SELECTS:
    ① TOP-DOWN: PFC → TRN → selective channel opening/closing
    ② BOTTOM-UP: strong/novel signals override TRN locally
    ③ AROUSAL STATE: brainstem → TRN modulates overall bandwidth

  TRN AND CONSCIOUSNESS:
    → TRN GABAergic neurons regulate consciousness under anesthesia
    → TRN generates ALPHA rhythm (~8-12Hz) = inhibitory pacemaker
    → Alpha increase → cortical suppression → content suppressed

  🟢 PMC 2024: TRN and anesthesia; eLife 2015: TRN arousal modulation
```

# GAP 5 — PHYSICAL BROADCAST: How Does "Global" Actually Work?

## §5.1 L2/3 Pyramidal Neurons

```
🟢 THE PHYSICAL ARCHITECTURE OF BROADCAST:

  Layer 2/3 pyramidal cells = KEY broadcast neurons.
    → LONG axons spanning distant cortical areas.
    → PFC + parietal = HIGHEST density of L2/3 long-range neurons.
    → = WHY these regions are workspace "hubs."

  Ignition mechanism (physical):
    → Signal activates local area → spreads to L2/3 neurons
    → L2/3 broadcast to DISTANT areas simultaneously
    → Distant areas receive → process → send back
    → = Reverberant loop = sustained broadcast

  Dehaene (PNAS 1998): "Core set of highly interconnected areas
    contains a high density of large pyramidal neurons with
    long-distance axons."

  🟢 Dehaene & Naccache 2001; Mountcastle 1957 (6-layer architecture)
```

## §5.3 Transthalamic Relay

```
🟢 2 TYPES OF CORTICO-CORTICAL COMMUNICATION:

  ① DIRECT: Cortex A → Cortex B (L2/3 long-range axons)
     Fast, but limited to areas with direct connections.

  ② TRANSTHALAMIC: Cortex A → Thalamus → Cortex B
     Slower, but thalamus can AMPLIFY / GATE / SYNCHRONIZE.

  PULVINAR = LARGEST thalamic nucleus:
    → Hub for visual + multimodal processing
    → Coordinates cortico-cortical communication
    → Modulates attention, feature binding, predictive coding
    → Damage → consciousness + attention impairment

  🟢 Sherman & Guillery 2006; eLife 2024 (pulvinar organization)
```

## §5.4 Oscillatory Coordination

```
🟢 3 FREQUENCY BANDS WORK TOGETHER:

  GAMMA (~30-100Hz) = CONTENT BINDING
    Neurons in same phase = "same object."

  THETA (~4-8Hz) = TEMPORAL ORGANIZATION
    Perceptual switching, WM maintenance rhythm.

  ALPHA (~8-12Hz) = INHIBITORY GATING
    TRN = alpha pacemaker. Alpha increase → content suppressed.

  ANESTHESIA TRAJECTORY:
    Awake: gamma → sedation: beta → LOC: alpha → unconscious

  🟢 Fries 2005; Klimesch 2007; PNAS 2011
```
