# Drill: Counterexample — Blindsight (Processing Without Knowing)

---

## EXECUTIVE SUMMARY

```
⭐ VERDICT: MOSTLY RESOLVED — blindsight CONFIRMS framework's "knowing"
   definition AND adds critical detail: RECURRENT PROCESSING through primary
   sensory cortex = necessary for conscious awareness in each modality.

  EVIDENCE STATUS:
    → Blindsight = genuine unconscious visual processing after V1 damage.
      Patient DB (Weiskrantz 1986): above-chance forced-choice in blind field.
      Patient TN (de Gelder 2008): navigates obstacle course, recognizes
      facial expressions — bilateral V1 destruction, reports "I see NOTHING."
    → Subcortical pathway confirmed: retina → SC → pulvinar → extrastriate.
      Inactivation of SC or pulvinar ABOLISHES blindsight (Kato et al. 2011).
    → V1's special role: RECURRENCE ANCHOR for visual consciousness.
      Feedback loops V4→V2→V1 enable Lamme Stage 2 (local recurrent processing).
      Without V1: no recurrence target → no visual awareness.
      TMS at ~100ms disrupts recurrent feedback to V1 → disrupts awareness.
    → Type 2 blindsight: "vague feeling" without visual content.
      Maps to framework's "vague knowing" and Lamme's Stage 2 boundary.
    → ⚠️ De Gelder, Humphrey & Pegna 2025 (Cerebral Cortex):
      Patient TN detects COLOR (especially red) with FULL AWARENESS.
      Cannot be explained by Type 1 or Type 2 blindsight.
      Challenges strict "V1 = absolutely necessary for ALL visual awareness."
    → Cross-modal equivalents CONFIRMED:
      Numbsense (S1 damage → tactile processing without awareness)
      Deaf hearing (A1 damage → sound processing without awareness)
      = GENERAL PRINCIPLE: primary cortex = recurrence anchor per modality.
    → Affective blindsight: SC → pulvinar → amygdala → emotional responses
      without visual awareness. Emotion processing bypasses consciousness.

  WHAT THIS MEANS FOR FRAMEWORK:
    → "Knowing" definition CONFIRMED from reverse direction:
      Blindsight = processing WITHOUT knowing. ✅
      Framework says knowing = f(signal × anchor × attention).
      Blindsight: signal exists (subcortical), but ANCHOR missing
      (no V1 recurrence) → no knowing.
    → ADD: Recurrent processing = necessary mechanism for conscious access.
      Not just "broadcast" (GWT) but "recurrent loop through primary cortex
      → THEN broadcast." Feedforward alone = insufficient.
    → REFINE: Consciousness requires BOTH broadcast (global) AND recurrence
      (local). Two complementary requirements, not one.
    → CHALLENGE: De Gelder 2025 suggests subcortical routes CAN sometimes
      support partial feature-specific awareness. Strict necessity of V1
      may need nuance → "V1 normally necessary, but not absolutely necessary
      for ALL visual features in ALL conditions."

  REMAINING UNCERTAINTY:
    → De Gelder 2025: n=1 (patient TN), naturalistic observation, not
      controlled experiment. Needs replication.
    → Type 2 blindsight: "feeling" vs "degraded seeing" (Phillips debate).
      Response criterion vs genuine unconscious perception not fully settled.
    → Lamme Stage 2 (local recurrence = phenomenal consciousness):
      debated, hard to test directly.
    → Mouse model 2026 (hippocampus involvement): may add unexpected
      complexity to the V1-centric picture.
    → Confidence: 🟡 Strong evidence but nuances from 2025 unresolved.
```

# Q1: V1 TẠI SAO ĐẶC BIỆT CHO VISUAL CONSCIOUSNESS?

## §1.1 V1 as Recurrence Anchor

```
🟢 V1'S UNIQUE ROLE — RECURRENCE TARGET FOR VISUAL FEEDBACK:

  ARCHITECTURE:
    → V1 (primary visual cortex, striate cortex) sits at the BASE of
      the visual hierarchy.
    → Receives input from LGN (thalamus) → Layer 4.
    → Sends FEEDFORWARD projections: V1 → V2 → V4 → IT (temporal).
    → Receives FEEDBACK projections: V4 → V2 → V1 (recurrent).
    → V1 = the CONVERGENCE POINT for visual feedback loops.

  WHY V1 AND NOT JUST ANY CORTEX:
    → V1 has unique columnar architecture:
      Orientation columns, ocular dominance columns, hypercolumns.
    → Highest-resolution retinotopic map in the brain.
    → All visual features ORIGINATE here before being elaborated higher.
    → Feedback FROM higher areas returns TO V1 for "verification" —
      higher-level interpretations checked against original input.
    → = V1 is the "ground truth" against which visual predictions are tested.

  LAMME'S ARGUMENT:
    → Feedforward sweep (V1→V2→V4→IT): ~50-150ms, UNCONSCIOUS.
      Features extracted: edges → contours → shapes → objects.
      Information flows forward ONLY. No awareness.
    → Local recurrent processing (V4→V2→V1): ~150-200ms, DEBATED awareness.
      Higher areas FEED BACK to V1, creating reverberant loops.
      This recurrence BINDS features into coherent percepts.
      Lamme: THIS is when phenomenal consciousness emerges.
    → Without V1: feedback has NO TARGET → loop cannot form →
      no local recurrence → no phenomenal awareness → no access.

  🟢 Lamme 2003, 2006; Lamme & Roelfsema 2000
```

## §1.2 TMS Evidence: Critical Time Window

```
🟢 TMS STUDIES CONFIRM V1'S RECURRENCE ROLE:

  METHOD:
    → Apply TMS pulse to V1 at different times after visual stimulus.
    → If TMS disrupts awareness → V1 activity at that moment is
      necessary for consciousness.

  RESULTS:
    → TMS at 0-80ms: no effect on awareness → feedforward signals
      haven't reached V1 yet.
    → TMS at ~80-100ms: disrupts initial feedforward processing in V1.
      = Disrupts BOTH conscious and unconscious processing.
    → TMS at ~100-120ms: SELECTIVELY disrupts AWARENESS without
      disrupting unconscious processing.
      = This is the RECURRENT FEEDBACK window.
      = Higher areas have sent feedback back to V1.
      = TMS disrupts this feedback → awareness lost.
      = BUT feedforward information already forwarded → unconscious
        processing continues → BLINDSIGHT-LIKE behavior.

  INTERPRETATION:
    → Two distinct time windows for V1 activity:
      EARLY (~80-100ms): feedforward, necessary for ALL processing.
      LATE (~100-120ms): recurrent feedback, necessary for AWARENESS ONLY.
    → TMS at late window = "artificial blindsight" in healthy subjects.
    → CONFIRMS: recurrent processing through V1 = critical for awareness.

  ⭐ KEY INSIGHT:
    → Blindsight patients = PERMANENT loss of the recurrence target.
    → TMS subjects = TEMPORARY loss of recurrent activity.
    → SAME RESULT: processing without awareness.
    → = CONVERGENT EVIDENCE from lesion studies AND causal manipulation.

  🟢 Boehler et al. 2008 (PNAS); de Graaf et al. 2012
  🟢 Fahrenfort et al. 2007 (backward masking disrupts reentrant processing)
```

## §1.3 Visual Backward Masking Confirmation

```
🟢 BACKWARD MASKING = NATURAL EXPERIMENT ON RECURRENCE:

  PARADIGM:
    → Show target briefly (~30ms).
    → Show mask shortly after (~50-100ms delay).
    → Result: target becomes INVISIBLE (masked).
    → BUT: target features ARE processed (priming effects remain).
    → = Processing without awareness. Same dissociation as blindsight.

  MECHANISM:
    → Target arrives in V1 (feedforward).
    → Feedforward cascade begins: V1→V2→V4→IT.
    → Before FEEDBACK returns to V1, MASK arrives.
    → Mask's feedforward signal OVERWRITES V1 activity.
    → Feedback from higher areas returns but finds mask, not target.
    → Recurrent loop disrupted → target never reaches awareness.
    → Target's feedforward information already forwarded → priming works.

  WHAT THIS PROVES:
    → Awareness requires recurrent processing THROUGH V1.
    → Feedforward alone = sufficient for unconscious processing.
    → Feedback to V1 = necessary for awareness.
    → SAME principle as blindsight, different mechanism of disruption.
    → Blindsight: V1 destroyed → no target for feedback.
    → Masking: V1 occupied by mask → feedback finds wrong content.
    → Both: recurrence disrupted → no awareness.

  🟢 Lamme 2003; Fahrenfort et al. 2007; Boehler et al. 2008
  🟡 Interpretation debated: some argue masking disrupts feedforward,
     not recurrent processing. BUT late-window TMS evidence supports
     recurrent interpretation.
```

## §1.4 Gradual vs Sudden V1 Damage

```
🟡 DOES THE BRAIN ADAPT TO V1 LOSS?

  SUDDEN DAMAGE (stroke, surgery):
    → Immediate blindness in corresponding visual field.
    → Blindsight may develop over weeks to months.
    → Subcortical pathway (SC→pulvinar) already exists but is normally
      "suppressed" by V1 dominance.
    → After V1 loss: subcortical pathway becomes MORE active.
    → But: awareness does NOT return (in typical cases).

  GRADUAL DAMAGE vs TRAINING:
    → Chokron & Perez 2008: visual training in hemianopia patients.
      22 weeks of training → some recovery of visual awareness.
    → Cavanaugh et al. 2019: training-induced recovery depends on
      SPARED PERILESIONAL V1.
      → Recovery = enlargement of population receptive fields in
        REMAINING V1 cortex.
      → NOT recovery through subcortical routes.
      → = V1 REMAINS necessary — recovery = using remaining V1 better.

  AGENESIS COMPARISON (never had V1):
    → Very rare. Most severe: hydranencephaly (Session 1).
    → If born without V1: developmental compensation differs from
      adult-acquired damage.
    → No clear cases of "born without V1 only" to compare.

  ⭐ IMPLICATION:
    → Recovery from V1 damage depends on REMAINING V1 tissue.
    → Subcortical routes support behavior but NOT awareness.
    → CONFIRMS V1's role as recurrence anchor is NOT replaceable
      (at least in adult brain with sudden damage).
    → BUT de Gelder 2025 challenges this — see §2.3.

  🟢 Chokron & Perez 2008; Cavanaugh et al. 2019 (bioRxiv/PNAS)
  🟡 Gradual damage cases limited; long-term adaptation unclear.
```

# Q2: BLINDSIGHT TYPE 2 — VAGUE AWARENESS KHÔNG CẦN V1?

## §2.1 Type 1 vs Type 2 Distinction

```
🟢 TWO TYPES OF BLINDSIGHT:

  TYPE 1 — "CLASSIC" BLINDSIGHT:
    → NO awareness whatsoever.
    → Forced-choice performance above chance.
    → Patient reports: "I see NOTHING. I'm just guessing."
    → Domains: motion direction, location, orientation, color (forced-choice).
    → = Processing without ANY subjective experience.

  TYPE 2 — "VAGUE FEELING" BLINDSIGHT:
    → Some subjective experience WITHOUT visual content.
    → Patient reports: "I don't SEE anything, but I FEEL something is there."
    → OR: "Something changed" without knowing WHAT changed.
    → NOT visual imagery — described as a "sense," "hunch," "feeling."
    → Performance typically BETTER than Type 1.

  KEY DIFFERENCE:
    → Type 1: no experience → no awareness → above-chance behavior.
    → Type 2: SOME experience → vague awareness → better behavior.
    → Type 2 = NOT "degraded seeing" (no visual quality).
    → Type 2 = "sensing without seeing" — a different KIND of experience.

  MAPPING TO FRAMEWORK'S SPECTRUM:
    → Type 1 → BELOW threshold. No knowing. Fully unconscious processing.
    → Type 2 → AT threshold boundary. "Vague knowing."
      Framework §2.1: f(signal × anchor × attention)
        Signal: present (subcortical), Anchor: PARTIAL, Attention: variable.
      = Maps to Lamme's Stage 2 BOUNDARY — some recurrence somewhere,
        but insufficient for full visual awareness.

  🟢 Weiskrantz 1998, 2009; Overgaard et al. 2008
```

## §2.2 Phillips' Challenge: Is Blindsight Real?

```
🟡 PHILLIPS (2021): BLINDSIGHT = DEGRADED CONSCIOUS VISION?

  ARGUMENT:
    → Phillips (Psychological Review 2021): "Blindsight Is Qualitatively
      Degraded Conscious Vision."
    → Claims: patients DO have visual experience but it's SO degraded
      they set an extremely conservative response criterion.
    → Under signal detection theory: they HAVE signal, but criterion
      is too high → report "I see nothing."
    → = Blindsight might be a REPORTING artifact, not absence of awareness.

  COUNTERARGUMENT — Michel & Lau (2021):
    → Residual awareness CANNOT explain feature-specific discrimination.
    → Performance matching: patients' d' in forced-choice exceeds what
      degraded conscious vision could support.
    → 2AFC > YN discrepancy (Azzopardi & Cowey 1997): patients perform
      better in 2AFC than YN even after mathematical correction.
    → This discrepancy = blindsight-specific, not explainable by
      conservative criterion alone.
    → "Not all criterion effects are nonperceptual" — true, but the
      PATTERN of results is inconsistent with simple criterion shift.

  PHILLIPS' OWN REVISION:
    → Phillips acknowledges: Campion et al. (1983) were WRONG to say
      blindsight = degraded vision due to conservative criteria.
    → Phillips' NEW claim: blindsight IS conscious but QUALITATIVELY
      different from normal vision — so degraded it's a different KIND
      of experience.
    → = Shifts from "reporting artifact" to "different experience quality."

  ⭐ ASSESSMENT:
    → Type 1 blindsight: strong evidence for GENUINE absence of awareness.
      Feature-specific discrimination without subjective report.
      Consistent across multiple patients, multiple paradigms.
    → Type 2: more ambiguous — could be degraded conscious vision
      OR could be subcortical "feeling" without visual content.
    → The debate CONTINUES but does NOT undermine the core finding:
      V1 damage → DRAMATIC reduction in visual awareness despite
      preserved visual processing. Whether "zero" or "near-zero"
      awareness, the DISSOCIATION between processing and knowing is real.

  🟡 Phillips 2021; Michel & Lau 2021; Azzopardi & Cowey 1997
  🟡 Debate ongoing, but core dissociation (processing ≠ knowing) stands.
```

## §2.3 De Gelder 2025: The Challenge Case

```
⚠️ DE GELDER, HUMPHREY & PEGNA 2025 — AWARENESS IN BLINDSIGHT:

  STUDY (Cerebral Cortex, Volume 35, Issue 1, January 2025):
    → Patient: TN — bilateral striate cortex destruction (2 strokes).
    → Same patient as famous obstacle navigation study (2008).
    → Naturalistic observations over extended period.

  SURPRISING FINDING:
    → TN can detect COLORED objects, especially RED.
    → Despite slow and effortful process, TN reports FULL AWARENESS
      of the color aspect.
    → TN spontaneously comments on colored stimuli without prompting.
    → "He does not see a red cube and would not perceive a red flower
      or red book, but CLEARLY SEES THE COLOR RED."
    → = Feature-specific conscious awareness WITHOUT V1.

  WHY THIS CHALLENGES EXISTING MODELS:
    → Type 1 blindsight: TN has AWARENESS → not Type 1.
    → Type 2 blindsight: TN has SPECIFIC feature awareness (color),
      not just vague feeling → not classic Type 2.
    → = A THIRD category? Or fundamentally different phenomenon?
    → Challenges: "V1 is ABSOLUTELY NECESSARY for ALL visual awareness."

  POSSIBLE EXPLANATIONS:
    ① Subcortical color pathway:
       → Koniocellular (K) pathway through LGN carries color information.
       → Some K-cells project to extrastriate cortex, bypassing V1.
       → Could support feature-specific (color) awareness without V1.
       → BUT: would need RECURRENCE somewhere — where?

    ② Residual V1 tissue:
       → Despite "bilateral destruction," there may be microscopic remnants.
       → MRI resolution may not detect small islands of functioning V1.
       → These remnants could support minimal recurrent loops.
       → BUT: TN's lesions well-documented, repeatedly imaged.

    ③ Extrastriate recurrence without V1:
       → Feedback loops between V4 and V2 (without V1 as target).
       → V2 could serve as alternative recurrence anchor for color.
       → V4 (color processing area) ↔ V2 recurrence = sufficient?
       → 🔴 Hypothesis — not tested.

    ④ Subcortical awareness (Merker-type):
       → SC → pulvinar pathway supports some feature processing.
       → If pulvinar acts as higher-order thalamic relay (Sherman model):
         pulvinar ↔ extrastriate recurrence = alternative loop?
       → = Thalamic involvement in consciousness again (Sessions 1-3).
       → 🔴 Hypothesis — but consistent with thalamus convergence.

  ⭐ ASSESSMENT:
    → De Gelder 2025 is n=1, naturalistic, not controlled experiment.
    → Single most famous blindsight patient (TN) — unique case.
    → Cannot generalize yet.
    → BUT: if confirmed, forces revision from:
      "V1 NECESSARY for visual consciousness" →
      "V1 normally necessary; alternative routes CAN sometimes support
       limited feature-specific awareness."
    → This is EXACTLY the kind of nuance "chậm mà chắc" requires.
    → DO NOT dismiss (real observation). DO NOT overgeneralize (n=1).

  🟡 De Gelder, Humphrey & Pegna 2025 (Cerebral Cortex)
  🔴 Mechanism for TN's color awareness = unresolved.
```

# Q3: SUBCORTICAL BYPASS — TẠI SAO KHÔNG TẠO CONSCIOUSNESS?

## §3.1 The SC → Pulvinar → Extrastriate Pathway

```
🟢 SUBCORTICAL VISUAL PATHWAY — ANATOMY:

  NORMAL VISUAL PATHWAY (geniculostriate):
    Retina → LGN (thalamus) → V1 → V2 → V4 → IT
    → Layer 4 input → Layer 2/3 processing → feedback → awareness
    = Primary route for conscious vision.

  SUBCORTICAL BYPASS (retino-tectal):
    Retina → Superior Colliculus (SC, midbrain)
           → Pulvinar (thalamus)
           → Extrastriate cortex (V5/MT, V4, etc.)
    → Bypasses V1 entirely.
    → ALSO: Retina → SC → Pulvinar → Amygdala (emotional processing).
    → = Ancient pathway, shared with reptiles and birds.

  WHAT THE SUBCORTICAL PATH CARRIES:
    ┌─────────────────────┬──────────────────────────────────┐
    │ Feature             │ Subcortical Pathway              │
    ├─────────────────────┼──────────────────────────────────┤
    │ Spatial location    │ YES — SC specialized for spatial │
    ├─────────────────────┼──────────────────────────────────┤
    │ Motion              │ YES — fast motion detection      │
    ├─────────────────────┼──────────────────────────────────┤
    │ Low spatial freq    │ YES — coarse shapes/blobs        │
    ├─────────────────────┼──────────────────────────────────┤
    │ High spatial freq   │ NO — fine detail requires V1     │
    ├─────────────────────┼──────────────────────────────────┤
    │ Color (simple)      │ PARTIAL — some via K-pathway     │
    ├─────────────────────┼──────────────────────────────────┤
    │ Emotional valence   │ YES — SC → amygdala (fear/threat)│
    ├─────────────────────┼──────────────────────────────────┤
    │ Object identity     │ VERY LIMITED                     │
    └─────────────────────┴──────────────────────────────────┘

  BANDWIDTH COMPARISON:
    → Geniculostriate (LGN→V1): ~1 million fibers (optic radiation).
      High bandwidth. Full feature set. SLOW (cortical processing).
    → Retino-tectal (SC→pulvinar): ~100,000 fibers (estimate).
      Low bandwidth. Limited features. FAST (subcortical shortcut).
    → Ratio: ~10:1 bandwidth advantage for geniculostriate.

  🟢 Tamietto & de Gelder 2010 (Nature Reviews Neuroscience)
  🟢 Kato et al. 2011 (Nature Communications 2019)
```

## §3.2 Why Subcortical Pathway Doesn't Generate Awareness

```
🟢 3 REASONS SUBCORTICAL BYPASS = NO CONSCIOUSNESS:

  REASON 1 — NO RECURRENT PROCESSING:
    → SC → pulvinar → extrastriate = FEEDFORWARD chain.
    → No V4→V2→V1 feedback loop (V1 is destroyed or bypassed).
    → Extrastriate cortex receives input but cannot VERIFY against V1.
    → = Lamme Stage 1 (feedforward) without Stage 2 (recurrent).
    → Stage 1 = sufficient for unconscious processing.
    → Stage 2 = required for awareness (even without global broadcast).
    → WITHOUT recurrence: processing yes, awareness no.

  REASON 2 — WRONG ARCHITECTURE:
    → SC neurons: organized in layers but NOT 6-layer cortical architecture.
    → SC lacks Layer 2/3 pyramidal neurons = NO long-range broadcast capability.
    → Pulvinar: thalamic nucleus, NOT cortex. No cortical columns.
    → Information reaches extrastriate cortex but arrives via NON-STANDARD
      input (not through Layer 4 → Layer 2/3 canonical circuit).
    → May activate extrastriate areas but not in the way that supports
      recurrent loops and sustained broadcast.

  REASON 3 — INSUFFICIENT INFORMATION CONTENT:
    → Subcortical pathway = coarse, fast, limited features.
    → "Something is there, moving, possibly threatening" — enough for
      behavioral response (duck, flinch, orient).
    → NOT enough for: "I see a red ball rolling left."
    → Consciousness may require MINIMUM INFORMATION CONTENT (Tononi's
      phi) — subcortical pathway below threshold.
    → BUT: this reason alone doesn't explain why NO awareness at all
      (could expect "vague" awareness from coarse information).
    → = Reason 1 (no recurrence) is the PRIMARY explanation.

  ⭐ INTEGRATED EXPLANATION:
    → Subcortical pathway provides: input signal (✓), fast processing (✓),
      behavioral output (✓).
    → Subcortical pathway LACKS: recurrent loop architecture (✗),
      V1 as verification target (✗), sustained broadcast (✗).
    → = Processing without the MECHANISM for knowing.
    → = Framework: signal exists, but the ANCHOR for knowing (recurrent
      loop) is missing → f(signal × anchor × attention): anchor = 0 → no knowing.

  🟢 Lamme 2006; Kato et al. 2011/2019
  🟡 Reason 3 (information content) is complementary but not primary.
```

## §3.3 Affective Blindsight — Emotion Without Awareness

```
🟢 AFFECTIVE BLINDSIGHT — THE EMOTIONAL EXCEPTION:

  WHAT IT IS:
    → Patients with V1 damage can:
      Correctly guess emotional expressions on faces in blind field.
      Show appropriate physiological responses (skin conductance, startle).
      Show amygdala activation to fearful faces.
    → BUT: report NO visual awareness of the face.
    → = EMOTIONAL processing without visual consciousness.

  PATHWAY:
    SC → pulvinar → amygdala (subcortical emotion route).
    → Amygdala processes low-spatial-frequency emotional cues.
    → Fearful faces: amygdala RIGHT hemisphere more active.
    → High-spatial-frequency components: NO amygdala activation.
    → = Pathway specialized for COARSE emotional signals.

  CLINICAL EVIDENCE:
    → Patient TN (de Gelder 2008): recognizes facial expressions.
    → fMRI: amygdala activation to unseen fearful faces.
    → Pulvinar lesions DISRUPT affective blindsight:
      Hemianopic patients + pulvinar damage → NO implicit fear processing.
      = Pulvinar is NECESSARY relay for emotion pathway.

  WHAT THIS MEANS:
    → Emotion processing CAN bypass consciousness entirely.
    → = Consistent with framework: emotional responses (threat detection)
      are often unconscious (Core-Software §4: unconscious processing).
    → SC → amygdala pathway = framework's "threat bypass" (~12ms).
    → ⚠️ But affective blindsight raises question:
      Is there AFFECTIVE awareness (feeling fear) without VISUAL awareness?
    → If yes: dissociation between modality-specific consciousness
      and affective consciousness.
    → Links to Session 1 (hydranencephaly): affective consciousness
      may be subcortical and INDEPENDENT of cortical visual consciousness.

  🟢 Tamietto & de Gelder 2010; de Gelder et al. 2001
  🟡 Affective awareness vs affective processing distinction = debated.
```

## §3.4 Animal Evidence: Circuit Dissection

```
🟢 ANIMAL STUDIES — CAUSAL EVIDENCE:

  MONKEY STUDIES (V1 LESION):
    → Monkeys with V1 lesions show blindsight-like behavior.
    → Saccade to target in blind field: above chance.
    → SC inactivation (muscimol): abolishes this ability.
    → = SC is NECESSARY for blindsight behavior.
```

# Q5: OTHER SENSORY MODALITIES — EQUIVALENTS TO BLINDSIGHT?

## §5.1 Numbsense (Tactile Blindsight)

```
🟢 NUMBSENSE — UNCONSCIOUS TOUCH:

  DISCOVERY:
    → Paillard, Michel & Stelmach 1983: first documented case.
    → Patient with left parietal damage.
    → Unable to FEEL touch on right hand (reported no sensation).
    → BUT: when asked to POINT to where she was touched,
      performance ABOVE CHANCE.
    → = Tactile processing without tactile awareness.
    → = "Blindsight for touch."

  TMS CONFIRMATION:
    → Ro 2021 (Current Biology): TMS to primary somatosensory cortex (S1).
    → 12 healthy participants.
    → TMS transiently disrupts S1 → loss of tactile awareness
      in contralateral hand.
    → Despite no awareness: above-chance localization of touch.
    → = TMS-induced numbsense in healthy subjects.
    → = Parallel to TMS-induced blindsight (V1 disruption).

  MECHANISM:
    → Touch signal reaches S1 (primary somatosensory cortex).
    → S1 disrupted (lesion or TMS) → no recurrent processing in S1.
    → BUT: touch information reaches S2 and posterior parietal cortex
      via PARALLEL pathways (through thalamus: VPL → S1 AND VPL → S2).
    → S2 processing = sufficient for localization behavior.
    → S2 processing = insufficient for awareness (no S1 recurrence).
    → = SAME PRINCIPLE as blindsight: primary cortex → recurrence anchor.

  🟢 Paillard et al. 1983; Ro 2021 (Current Biology)
```

## §5.2 Deaf Hearing (Auditory Blindsight)

```
🟢 DEAF HEARING — UNCONSCIOUS SOUND:

  CASE REPORT:
    → Garde & Cowey 2000 (Cortex): "Deaf hearing."
    → Patient with bilateral damage to primary auditory cortex (A1).
    → Extensive cortical deafness — no awareness of ANY sound.
    → When forced to guess sound LOCATION: above-chance performance.
    → = Sound processing without auditory awareness.
    → = "Blindsight for hearing."

  NEURAL EVIDENCE:
    → Cortical deafness patients show preserved P300-like potentials.
    → P300 = late ERP component, usually associated with conscious
      detection of novel/significant stimuli.
    → In cortical deafness: P300-LIKE potentials present but MODIFIED.
    → = Some auditory processing continues subcortically.
    → Subcortical route: inferior colliculus → medial geniculate (MGB)
      → residual cortex or association areas.

  RARITY:
    → Cortical deafness is VERY RARE (bilateral A1 damage uncommon).
    → Far fewer cases than blindsight.
    → = Less studied, less replicated.
    → But the cases that exist show SAME PATTERN as blindsight.

  🟢 Garde & Cowey 2000 (Cortex)
  🟡 Very limited case reports. Pattern consistent but sample tiny.
```

## §5.3 Cross-Modal Generalization

```
🟢 GENERAL PRINCIPLE CONFIRMED:

  PATTERN ACROSS MODALITIES:
    ┌──────────────┬──────────────┬──────────────┬───────────────────┐
    │ Modality     │ Primary      │ Phenomenon   │ Processing        │
    │              │ Cortex       │ Name         │ Without Awareness │
    ├──────────────┼──────────────┼──────────────┼───────────────────┤
    │ Vision       │ V1           │ Blindsight   │ Location, motion, │

  COMMON MECHANISM:
    → Primary cortex (V1/S1/A1) = RECURRENCE ANCHOR for each modality.
    → Damage to primary cortex → loss of recurrent processing in
      that modality → loss of conscious awareness.
    → BUT: parallel subcortical/alternative cortical pathways remain →
      behavioral processing preserved → above-chance performance
      without awareness.

  WHAT THIS PROVES:
    → Consciousness is NOT generated by primary cortex PER SE.

    → This is NOT a quirk of V1 or vision.
    → = Strengthens the theoretical claim considerably.

      mechanism for modality-specific consciousness.
    → Each modality has its own recurrence anchor: V1, S1, A1.
    → = Maps elegantly to framework's architecture-based explanation.

  🟢 Cross-modal evidence consistent (vision, touch, hearing).
  🟡 Touch and hearing cases far fewer than vision — less studied.
```
