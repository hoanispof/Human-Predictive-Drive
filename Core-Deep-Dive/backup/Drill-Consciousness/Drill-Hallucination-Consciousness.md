# §1 — HALLUCINATION THROUGH 5-CHECKPOINT MODEL

## §1.1 Normal Perception vs Hallucination — Pathway Comparison

```
⭐ THE SAME 5 CHECKPOINTS — DIFFERENT SIGNAL SOURCE:

  NORMAL PERCEPTION:
    External stimulus → receptor → thalamus relay
    → CP1 (TRN gate): TRN allows signal through ✅
    → CP2 (Tonic mode): thalamus in tonic firing ✅
    → CP3 (High-order nuclei): intralaminar/MD activate ✅
    → CP4 (Cortical recurrence): V1/S1/A1 feedback loops ✅
    → CP5 (Global broadcast): L2/3 → PFC + parietal ✅
    → RESULT: "knowing" — and SOURCE = external → CORRECT attribution

  HALLUCINATION:
    NO external stimulus → INTERNAL source (cortical/subcortical)
    → Signal enters thalamo-cortical loop from INSIDE
    → CP1: TRN may or may not be involved (type-dependent) ✅/⚠️
    → CP2: tonic mode required (hallucinations = waking/REM) ✅
    → CP3: high-order nuclei engage (feels conscious) ✅
    → CP4: cortical recurrence active (vivid perceptual content) ✅
    → CP5: global broadcast (the person KNOWS the content) ✅
    → RESULT: "knowing" — but SOURCE = internal → WRONG attribution

  ⭐ KEY INSIGHT:
    The 5-CP model is NECESSARY but NOT SUFFICIENT for distinguishing
    real perception from hallucination. All 5 CPs can pass with either
    external or internal signal source.

    The missing dimension: SOURCE ATTRIBUTION (source monitoring)
    = A judgment process AFTER broadcast, evaluating WHETHER the
    content originated externally or internally.

    This is NOT a 6th checkpoint. It is a QUALITY assessment that
    operates ON the broadcast content, using criteria like:
    → Perceptual detail richness
    → Spatial/temporal context consistency
    → Cognitive operation traces ("did I produce this?")
    → Cross-modal confirmation (do other senses confirm?)
    → Prior expectation consistency

  🟡 5-CP + source attribution mapping = framework synthesis
  🟢 Individual CPs: established (Drill-Thalamo-Cortical-Mechanism.md §6)
  🟢 Source monitoring: Johnson, Hashtroudi & Lindsay 1993
```

# §2 — HALLUCINATION THROUGH 3-PARAMETER MODEL

## §2.1 Parameter Mapping

```
⭐ f(signal_strength × anchor_quality × attention) — HALLUCINATION PROFILE:

  ① SIGNAL STRENGTH: INTERNAL but STRONG
    → Normal perception: signal from external stimulus
    → Hallucination: signal from INTERNAL source
    → But signal strength parameter doesn't distinguish source!
    → Hallucination signal is strong ENOUGH to ignite → enters workspace
    → In some types, signal is STRONGER than normal perception:
      Schizophrenia: dopamine amplifies salience of internal signals
      Psychedelics: cortical excitability enhanced
      Charles Bonnet: deafferented neurons hyperexcitable

    ⭐ IMPORTANT: The 3-parameter model measures signal
    STRENGTH, not signal SOURCE. This is correct behavior —
    the parameters determine WHETHER something becomes
    conscious, not WHETHER it is veridical.

  ② ANCHOR QUALITY: TYPE-DEPENDENT
    → Schizophrenic auditory: often VERBAL (voices have words)
      = Strong verbal anchor → easily re-fired → persistent
    → Psychedelic visual: often EMOTIONAL (awe, fear)
      = Emotional anchor → vivid but hard to re-fire later
    → Hypnagogic: brief, fragmentary → WEAK anchor
      = Quick decay, often forgotten on full waking
    → Charles Bonnet: visual images, often objects/faces
      = Contextual anchor → recognizable but patient labels "not real"
    → Fever/delirium: confused, shifting → UNSTABLE anchor
      = Cannot maintain consistent content

  ③ ATTENTION: CAPTURED INVOLUNTARILY
    → In normal knowing: attention can be top-down (PFC pull)
      or bottom-up (strong signal pushes)
    → In hallucination: attention is CAPTURED via bottom-up push
      = §5.3 bottom-up pathway — signal pushes onto stage
    → PFC tries to redirect attention → FAILS because:
      Signal self-sustains in thalamo-cortical loop
      Signal strength > PFC suppress capacity
      (Connects to PFC-Operations §2.2: suppress ≠ sustainable)

  🟡 3-parameter hallucination mapping = framework synthesis
```

## §2.2 Why 3-Parameter Model Is ROBUST for Hallucination

```
⭐ THE MODEL WORKS — BUT REVEALS A BOUNDARY:

  The 3-parameter model correctly predicts:
    → High signal + any anchor + captured attention → conscious ✅
    → Same model explains WHY hallucination enters workspace ✅
    → Same model explains WHY different types have different persistence:
      Strong anchor (schizophrenic voices) → persistent
      Weak anchor (hypnagogic fragments) → brief

  The model does NOT predict:
    → Whether the "knowing" is veridical or false ❌
    → This is CORRECT BEHAVIOR: the model describes consciousness
      (whether you KNOW), not accuracy (whether what you know is TRUE)

  ⭐ IMPLICATION:
    → "Knowing" can be about something FALSE
    → This was ALREADY implicit in the framework:
      §7.3: "Feel-Explanation CAN BE WRONG (PFC = Lawyer, confabulation)"
    → Hallucination extends this principle to PERCEPTION level:
      Not just wrong explanations, but wrong PERCEPTS
    → The model handles this cleanly: consciousness ≠ accuracy

  🟡 Analysis = framework synthesis
  🟢 3-parameter model robustness confirmed across 9+ drill sessions
```

# §3 — HALLUCINATION THROUGH SUSTAIN HIERARCHY

## §3.1 Primary Sustain in Hallucination

```
⭐ THALAMO-CORTICAL RESONANCE = HOW HALLUCINATION PERSISTS:

  The sustain hierarchy (Drill-Thalamo-Cortical-Mechanism.md §3.2):
    PRIMARY: thalamo-cortical resonance + CM-Pf
    ADDITIONAL: PFC WM hold + limbic drive

  In HALLUCINATION:
    → PRIMARY sustain (thalamo-cortical loop): ✅ ACTIVE
      The hallucinatory signal enters the loop → self-sustains
      Loop cycles at ~40Hz gamma → maintains signal
      Signal re-enters thalamus from cortex (corticothalamic > thalamocortical)
      = SAME mechanism as real perception sustain

    → ADDITIONAL sustain — PFC: ❌ OPPOSING (trying to suppress)
      PFC is NOT holding the hallucination → PFC is trying to DISMISS it
      But suppress FAILS: signal in loop > PFC suppress capacity
      PFC suppress requires CONTINUOUS effort (PFC-Operations §2.2)
      = PFC = exhaustible resource vs self-sustaining loop = mismatch

    → ADDITIONAL sustain — LIMBIC: ⚠️ VARIABLE
      If hallucination has emotional content → limbic ENHANCES sustain
      Fear/distress about hallucination → amygdala activates →
        adds emotional sustain signal → HARDER to suppress
      = Vicious cycle: hallucination → fear → limbic enhances → stronger

  ⭐ KEY INSIGHT — WHY PFC CANNOT SUPPRESS HALLUCINATION:

    PFC suppress = finite, effortful, non-compilable process
    (PFC-Operations §2.2: suppress cost = ② efference mismatch)
    Thalamo-cortical loop = self-sustaining, no effort cost
    = PFC is fighting a self-sustaining loop with a finite resource
    = Mismatch of mechanism: exhaustible vs self-regenerating

    This is DIFFERENT from suppressing a THOUGHT:
    → Thought suppress: PFC blocks compiled pattern's OUTPUT
    → Hallucination suppress: PFC tries to break a RESONANCE LOOP
    → Loop regenerates each cycle → suppress must be continuous →
      PFC budget depletes → suppress fails

  🟡 Sustain hierarchy × hallucination = framework synthesis
  🟢 Primary sustain established: Llinás 1998
  🟢 PFC suppress cost: PFC-Operations §2.2, §9
```

## §3.2 Exit Mechanisms — How Hallucination Eventually Stops

```
⭐ "CỨ CHẠY MÃI RỒI TỰ TẮT" — 4 EXIT MECHANISMS:

  The user's observation: hallucination runs until it stops on its own.
  Framework predicts 4 possible exit mechanisms, all map to §3.6 decay:

  ① SIGNAL DECAY:
    → Internal signal source weakens over time
    → Without reinforcing external input, the loop gradually decays
    → Each cycle: signal slightly weaker → below sustain threshold → stops
    → = Natural damping of a self-sustaining oscillation
    → Explains: why many hallucinations are SELF-LIMITING
    → Maps to: §3.6 mechanism ① (signal strength drops)

  ② COMPETITIVE DISPLACEMENT:
    → New, STRONGER signal enters workspace → wins biased competition
    → New signal captures thalamo-cortical loop → old signal loses
    → = "Something else grabs your attention"
    → Explains: why distraction sometimes helps (new task, conversation)
    → Maps to: §5.2 biased competition, §3.6 mechanism ③

  ③ OSCILLATORY SHIFT:
    → Brain state changes → thalamic mode shifts
    → Hypnagogic: fully waking → tonic mode stabilizes → hallucination stops
    → Falling asleep: tonic → burst → waking hallucination stops (but
      dreaming may begin)
    → Maps to: §3.6 mechanism ④ (gamma → alpha)

  ④ CHEMICAL/PHARMACOLOGICAL CHANGE:
    → Psychedelic: drug metabolized → 5-HT2A occupancy drops →
      thalamic filter restores → flooding stops
    → Schizophrenic: antipsychotic → D2 blockade → aberrant salience
      reduced → signal weakens
    → Fever: temperature normalizes → metabolic disruption resolves
    → NOT a natural exit — external intervention

  ⚠️ TYPE-SPECIFIC EXIT:
    → Hypnagogic: ③ oscillatory shift (sleep→wake transition completes)
    → Psychedelic: ④ drug wears off + ② distraction
    → Charles Bonnet: ② displacement + ① decay (episodes self-limit)
    → Schizophrenic: ④ medication (without medication, often chronic)
    → Fever: ④ temperature normalization
    → = Different entry mechanisms → different exit mechanisms

  🟡 Exit mechanism taxonomy = framework synthesis
  🟢 Individual mechanisms established (§3.6 decay)
```

# §4 — 6 TYPES OF HALLUCINATION: UNIFIED ANALYSIS

## §4.1 Type-by-Type Deep Analysis

```
⭐ TYPE ① — SCHIZOPHRENIC HALLUCINATION:

  MECHANISM:
    → Howes & Kapur 2009: mesolimbic dopamine EXCESS
    → Kapur 2003: "aberrant salience" — dopamine assigns motivational
      significance to INTERNAL representations stimulus-independently
    → Internal signals (inner speech, memories, spontaneous firing)
      get tagged as NOVEL and IMPORTANT
    → = Prediction-delta system misfires:
      VTA fires without genuine prediction-delta → PFC engages →
      internal signal treated as novel external input
    → Mesocortical dopamine DEFICIT → PFC hypofunction →
      source monitoring (mPFC) impaired → cannot tag as internal

  WHY AUDITORY DOMINANT — COROLLARY DISCHARGE MODEL:
    → Inner speech = most common form of self-generated mental content
    → Normal: speech production generates EFFERENCE COPY (corollary
      discharge) sent to auditory cortex → SUPPRESSES self-generated
      sound → N1 ERP suppression during talking (Ford et al. 2001)
    → Schizophrenia: N1 suppression REDUCED or ABSENT →
      auditory cortex does NOT recognize signal as self-generated
    → Result: inner speech not tagged as self-produced →
      MISATTRIBUTED to external source → "voices"
    → = NOT hallucinating "new" content but MISATTRIBUTING self-generated
      speech to external source
    → Brain regions: superior temporal gyrus (hyperactivated during AVH),
      Broca's area (speech generation, source of corollary discharge),
      fronto-temporal DISCONNECTION = anatomical substrate

  THALAMOCORTICAL DYSCONNECTIVITY (Woodward 2012, Anticevic 2014):
    → REDUCED PFC-thalamus connectivity (esp. mediodorsal nucleus)
    → INCREASED thalamus-sensorimotor/temporal connectivity
    → = Bidirectional pattern: PFC underconnected, sensory overconnected
    → Thalamic hyperconnectivity to superior temporal gyrus =
      directly implicates thalamo-temporal circuit in AVH
    → Replicated across 5 cohorts, n=415+405 (Cheng 2015)
    → = Connects to 3-Role thalamus: Role 3 (routing) MISWIRED

  THALAMIC CHANGES:
    → Thalamic volume reduction (esp. mediodorsal nucleus)
    → Altered thalamocortical connectivity
    → = Routing (Role 3) compromised → content misrouted

  SOURCE MONITORING:
    → BROKEN — "autonoetic agnosia" (failure to recognize own agency
      in generating the content)
    → mPFC dysfunction correlated with hallucination severity
    → Patients genuinely CANNOT distinguish internal from external

  FRAMEWORK MAPPING:
    → Signal: high (dopamine-amplified salience)
    → Anchor: verbal (voices have words → strong verbal anchor)
    → Attention: captured (salience = bottom-up push §5.3)
    → CP affected: primarily SOURCE MONITORING, not checkpoint failure
    → Exit: ④ medication (D2 blockade) — without medication, chronic

  🟢 Howes & Kapur 2009; Kapur 2003; Allen et al. 2008
```

```
⭐ TYPE ② — PSYCHEDELIC HALLUCINATION:

  MECHANISM:
    → Psilocybin, LSD, DMT: agonists at 5-HT2A receptors
    → 5-HT2A receptors concentrated in:
      (a) Cortical layer V pyramidal neurons → enhanced excitability
      (b) Thalamic relay neurons → filter disruption
    → Vollenweider's thalamic filter model:
      Normal: TRN + cortical feedback filter sensory input → relevant only
      Psychedelic: 5-HT2A reduces TRN inhibition → gate opens too wide
      → SENSORY FLOODING: both internal noise AND amplified external
        signals flood the workspace simultaneously

  CSTC LOOP DISRUPTION (Vollenweider & Preller 2020):
    → Cortico-Striato-Thalamo-Cortical (CSTC) feedback loops normally
      maintain gating. 5-HT2A disrupts at cortical entry point (layer V)
    → Layer V pyramidal neurons: 5-HT2A activation → enhanced
      spontaneous glutamate release (Aghajanian & Marek 1999)
    → This glutamate surge disrupts normal corticothalamic feedback
    → Result: thalamic filter receives aberrant cortical commands →
      gating fails → sensory/cognitive flooding

  WHY VISUAL DOMINANT:
    → V1 has HIGHEST density of 5-HT2A receptors in cortex
    → Visual system has uniquely well-characterized retinotopic/
      hierarchical architecture (V1→V2→V4→IT) → disinhibition produces
      spatially-organized percepts (not random noise)
    → GEOMETRIC HALLUCINATIONS (Klüver form constants):
      Spirals, tunnels, lattices, cobwebs = 4 basic forms
      Bressloff, Cowan et al. 2001 (Phil Trans R Soc B):
        V1 has retino-cortical log-polar mapping +
        anisotropic lateral connections →
        Turing-like symmetry-breaking instability →
        striped activity patterns in cortex →
        mapped through inverse retino-cortical transform →
        REPRODUCE all 4 Klüver form constants
      = Geometric hallucinations = V1 ARCHITECTURE being directly revealed

```
⭐ TYPE ④ — CHARLES BONNET SYNDROME:

  MECHANISM — DEAFFERENTATION + RELEASE:
    → Vision loss (macular degeneration, glaucoma, etc.) →

      RELEASED from constraint → generates its own content
    → PHANTOM LIMB ANALOGY (Baffour-Awuah et al. 2024):
      Amputation → central sensitization → phantom sensation
      Vision loss → cortical hyperexcitability → phantom images

## §6.2 Source Monitoring — The Mechanism Behind the Distinction

```
⭐ JOHNSON ET AL. 1993: HOW THE BRAIN DISTINGUISHES INTERNAL FROM EXTERNAL

  SOURCE MONITORING FRAMEWORK (SMF):
    → There is NO hardwired "tag" marking whether experience is
      internal or external
    → Attribution is a JUDGMENT made from QUALITIES of the experience:

    ┌────────────────────────┬──────────────────┬──────────────────┐
    │ Quality                │ External (real)  │ Internal (imagined)│
    ├────────────────────────┼──────────────────┼──────────────────┤
    │ Perceptual detail      │ HIGH (rich)      │ LOW (pale)        │
    │ Spatial context        │ SPECIFIC (where)  │ VAGUE             │
    │ Temporal context       │ SPECIFIC (when)   │ VAGUE             │
    │ Emotional intensity    │ Variable          │ Usually lower     │
    │ Cognitive operations   │ FEW (automatic)   │ MANY ("I made it")│
    │ Cross-modal coherence  │ YES (see+hear+...)│ NO (one modality) │
    └────────────────────────┴──────────────────┴──────────────────┘

    → Brain evaluates these qualities → makes PROBABILISTIC judgment
    → Distributions OVERLAP → misattribution always possible
    → This is a HEURISTIC, not a foolproof system

  WHY HALLUCINATION PASSES THE "EXTERNAL" THRESHOLD:
    → Hallucinatory content has HIGH perceptual detail
      (unlike imagination's "pale" quality)
    → Hallucinatory content has LOW cognitive operation traces
      (unlike imagination's "I produced this" traces)
    → These qualities match the "external" profile
    → Brain's heuristic judgment: "this must be external" → WRONG

  WHY IMAGINATION DOES NOT:
    → Imagined content is PALE, low detail
    → Imagined content carries cognitive operation traces
      ("I am deliberately producing this image")
    → These qualities match the "internal" profile
    → Brain's heuristic judgment: "this is internal" → CORRECT

  ⭐ THE GRADIENT BETWEEN THEM:
    → "Vivid imagination" = higher than normal perceptual detail
      BUT still carries cognitive operation traces → classified internal
    → "Mild hallucination" = high perceptual detail
      WITH reduced cognitive traces → classified external → "real"
    → = There IS a gradient, and the boundary is a probabilistic
      threshold, not a hard line
    → People with naturally vivid imagery sometimes have difficulty
      distinguishing (high hallucination-proneness)

  CONNECTION TO PREDICTIVE CODING:
    → Source monitoring can be reframed as:
      Imagination: brain generates content WITH the metacognitive
        prediction "I am generating this" → prediction matches
      Hallucination: brain generates content WITHOUT the metacognitive
        prediction → prediction error: "where did this come from?"
        → Default attribution: "must be external"
    → = Source monitoring FAILURE = missing metacognitive prediction

  🟢 Johnson, Hashtroudi & Lindsay 1993
  🟡 Predictive coding reframing of source monitoring = synthesis
```

# §7 — PREDICTIVE CODING CONNECTION

## §7.1 Hallucination = Strong Prior + Weak Evidence

```
⭐ THE PREDICTIVE CODING ACCOUNT — DIRECT FRAMEWORK CONNECTION:

  PREDICTIVE CODING (Friston, Clark):
    Perception = precision-weighted integration of:
      TOP-DOWN predictions (priors, expectations)
      BOTTOM-UP sensory evidence (data, error signals)
    → Normal: balanced → accurate perception
    → Prior too strong: hallucination (top-down dominates)
    → Evidence too weak/absent: also hallucination (priors unopposed)

  FRAMEWORK CONNECTION (§7.3, §8.2 — prediction-delta):
    → Framework already has prediction-delta:
      Actual outcome ≠ predicted outcome → delta signal → VTA fires
    → Normal: delta → learning → update predictions
    → Hallucination in predictive coding terms:
      → Schizophrenia: dopamine excess → ABERRANT precision on prediction
        errors → noise coded as reliable signal → false learning
      → Charles Bonnet: NO sensory data → priors fill void
      → Psychedelics: altered precision weighting → priors and sensory
        both amplified → chaotic integration
      → Sensory deprivation: sensory data removed → priors dominate

  CORLETT ET AL. 2019 — "HALLUCINATIONS AND STRONG PRIORS":
    → High-precision priors → override sensory evidence
    → Superior temporal cortex + insula + hippocampal-frontostriatal circuits
    → = WHERE prior-driven templates override sensory input

  POWERS ET AL. 2017 — CONDITIONED HALLUCINATIONS (Science):
    → Paradigm: tone + checkerboard paired, then tone at threshold/absent
    → Voice-hearers (clinical AND non-clinical) reported illusory tones
      far more often on no-tone trials
    → HGF modeling: prior-weighting parameter ν (nu) ELEVATED in
      voice-hearers (F1,55=13.96, p=4.45×10⁻⁴)
    → = Hallucination-prone individuals OVERWEIGHT learned associations
    → = Continuum: same mechanism in clinical and non-clinical

  STERZER ET AL. 2018 — HIERARCHICAL ACCOUNT:
    → Aberrant prediction errors at LOW (sensory) levels → hallucinations
    → Aberrant prediction errors at HIGH (conceptual) levels → delusions
    → Dopamine modulates PRECISION assigned to prediction errors
    → = Schizophrenia: both levels affected (positive symptoms = both)

  ⭐ FRAMEWORK IMPLICATION:
    → Prediction-delta (§7.3): VTA fires when actual ≠ predicted
    → In hallucination: precision weighting is ABNORMAL
    → This is not a NEW mechanism — it's prediction-delta with
      WRONG precision settings
    → = Framework's prediction-delta concept ALREADY contains the
      seed of hallucination explanation, just needs precision weighting

  🟢 Corlett 2019; Powers 2017; Sterzer 2018
  🟡 Framework prediction-delta connection = synthesis
```

## Q2: Tại sao ảo giác "thật" hơn tưởng tượng?

```
⭐ ANSWER: SOURCE MONITORING CRITERIA — 3 FACTORS:

  ① PERCEPTUAL DETAIL:
    → Imagination: PALE, low sensory detail
    → Hallucination: VIVID, rich sensory detail
    → Brain's heuristic: "rich detail = external source"
    → Hallucination crosses the detail threshold → tagged "external"

  ② COGNITIVE OPERATION TRACES:
    → Imagination: carries "I produced this" marker
      (Johnson 1993: record of deliberate cognitive effort)
    → Hallucination: NO such marker (signal arrived involuntarily)
    → Brain's heuristic: "no effort trace = external source"
    → Hallucination has no cognitive trace → tagged "external"

  ③ INVOLUNTARINESS:
    → Imagination: can START and STOP voluntarily
    → Hallucination: cannot stop → "if I can't control it, it's not me"
    → Brain's heuristic: "uncontrollable = external agent"
    → = Attribution of agency to external source

  ⭐ ALL 3 FACTORS ALIGN IN HALLUCINATION:
    → High detail + no effort trace + involuntary = "this is REAL"
    → All 3 point to "external source" conclusion
    → Brain's probabilistic judgment: confident external attribution
    → = WHY hallucination feels MORE real than imagination

  ⭐ EXCEPTION — CHARLES BONNET:
    → High detail ✅ + no effort trace ✅ + involuntary ✅
    → But: source monitoring INTACT → patient reasons:
      "I know I'm losing vision. These images make no sense in context.
       Therefore: my brain is generating them."
    → = TOP-DOWN cognitive override of bottom-up source cues
    → = Metacognition corrects the heuristic
    → Requires: intact mPFC + knowledge of own condition

  🟢 Johnson et al. 1993 (source monitoring criteria)
  🟡 Application to hallucination × imagination = synthesis
```
