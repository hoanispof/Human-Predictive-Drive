```
    → §2.1 says anchor determines stability/chainability — but does not say
      chunk density determines RICHNESS of what appears on stage
    → Landmark levels describe AMOUNT of knowing, not DETAIL of knowing

  THE GAP: THRESHOLD ≠ RESOLUTION
    Clarity = chunk density → lowers threshold = about ENTRY
    But: same threshold crossing, same "vague knowing" level,
    DIFFERENT detail level depending on chunk density = about RESOLUTION

    Both at "clearer knowing" level (can label: "red wine").
    But expert's conscious CONTENT has 10-100× more detail.

    These 2 factors are RELATED but SEPARABLE:

    ① High chunk density + low anchor quality:
       Expert craftsman who KNOWS something is wrong but CAN'T EXPLAIN why
       = Rich conscious experience (many details noticed)
       + Poor articulation (no verbal anchors for the details)
       = Tacit knowledge (Polanyi 1958)

    ② Low chunk density + high anchor quality:
       Student who memorized wine vocabulary but CAN'T TASTE differences
       = Can chain logic ("wine should have notes of X because Y")
       + But conscious EXPERIENCE of tasting = same as novice
       = "Knowing about" ≠ "knowing from experience"

    ③ High chunk density + high anchor quality:
       Master sommelier = rich experience + can articulate
       = Highest resolution + highest knowing level

    ④ Low chunk density + low anchor quality:
       Novice's first wine tasting
       = Minimal detail + vague knowing
       = Lowest resolution

    → Chunk density = richness of CONTENT that appears on stage
    → Anchor quality = ability to HOLD, RE-FIRE, and CHAIN that content
    → Both contribute to "resolution" but through DIFFERENT mechanisms

  ⚠️ IS RESOLUTION A NEW PARAMETER?
    No. Chunk density affects existing parameters:
      → Signal: more chunks activated → stronger aggregate signal
      → Anchor: more anchor types available → better stability
      → Attention: more structure for PFC to grab → better hold
    = PRECONDITION that modulates all 3, not independent 4th parameter
    = Like temperature in chemistry: affects all reactions, not a reactant

  ENRICHMENT RECOMMENDATION:
    → Extend Feeling.md §2.2b concept:
      Current: "clarity = chunk density → lowers threshold" (ENTRY only)
      Extended: "clarity = chunk density → lowers threshold AND
                enriches post-threshold conscious content" (ENTRY + RESOLUTION)
    → This is a SMALL EXTENSION, not a paradigm shift

  🟢 Expert perception: Chase & Simon 1973 (chess chunks); Gobet & Simon 1998 (~50,000 templates)
  🟢 Perceptual learning changes perception: Kellman & Garrigan 2009
```

### §2.4 ENCODING: Outside Consciousness Scope (Correctly)

```
⭐ ENCODING = Long-term retention after leaving consciousness stage

  THIS IS A MEMORY PROCESS, NOT A CONSCIOUSNESS PROCESS.

  WHY ENCODING ≠ DURATION:
    → Long duration + no encoding: daydreaming, highway hypnosis
      (pattern on stage for minutes but not stored)
    → Short duration + strong encoding: flashbulb memories
      (brief but intensely emotional → encoded permanently)
    → CORRELATION exists: longer duration → more hippocampal exposure
      → more likely to encode. But NOT identical processes.

  ENCODING DEPENDS ON:
    → Hippocampal consolidation (requires activation during consciousness)
    → Emotional tagging (amygdala → hippocampal flagging)
    → Repetition (repeated firing → LTP → stabilization)
    → Sleep consolidation (hippocampus → cortex replay)
    → Duration on stage (more time → more hippocampal exposure)

  WHY THE FRAMEWORK CORRECTLY EXCLUDES ENCODING:
    → Framework scope = consciousness (= "knowing" = access consciousness)
    → "Knowing" = WHETHER you know RIGHT NOW
    → Encoding = whether you will REMEMBER LATER
    → Two different questions, two different processes
    → Encoding is in Chunk.md (anchor decay, Ebbinghaus)

  HOWEVER — ENCODING INTERACTS WITH CONSCIOUSNESS:
    → Consciousness provides the WINDOW for encoding (must be on stage)
    → Emotional consciousness enhances encoding (amygdala → hippocampus)
    → Deliberate rehearsal (PFC hold = consciousness + encoding)
    → These interactions are CROSS-DOMAIN, not consciousness-internal

  ASSESSMENT: ✅ CORRECTLY EXCLUDED from consciousness model.
  Encoding belongs in Chunk.md / memory system, not here.

  🟢 Hippocampal consolidation: Squire 1992
  🟢 Emotional encoding: McGaugh 2004 (amygdala modulation)
  🟢 Flashbulb memories: Brown & Kulik 1977 (brief + emotional → permanent)
  🟢 Duration ≠ encoding: Craik & Lockhart 1972 (depth of processing > duration)
```

### §3.1 Expert vs Novice Cases — Stress-Testing Resolution

```
⭐ CASE ANALYSIS: SAME STIMULUS, DIFFERENT RESOLUTION

  ─────────────────────────────────────────────────────────────────
  CASE 1: WINE SOMMELIER vs NOVICE
  ─────────────────────────────────────────────────────────────────

  Stimulus: Same glass of red wine

  NOVICE:
    → Entry: threshold crossed (can smell wine) ✅
    → Resolution: "smells like red wine" (1-2 features detected)
    → Duration: ~5-10 sec attention, then moves on
    → Why low resolution:
      Few olfactory chunks → few features extracted from same input
      Few verbal anchors → can't label subtleties even if detected
      No prediction model for wine → no prediction-delta from subtleties

  EXPERT:
    → Entry: threshold crossed (same as novice) ✅
    → Resolution: "I detect blackberry, cedar, hint of tobacco, slightly
      tannic, likely 2018 Bordeaux Left Bank" (20-50 features)
    → Duration: 30+ sec sustained attention (trained to hold)
    → Why high resolution:
      ~10,000 olfactory chunks (Bende & Nordin 1997) → many features
      extracted from SAME chemical input
      Rich verbal anchors → each feature labeled and chainable
      Rich prediction model → prediction-delta from subtle deviations
      → "This tannin feels unexpected for this grape" = delta → attention

  MECHANISM — HOW CHUNK DENSITY → RESOLUTION:
    → Same molecules hit same olfactory epithelium
    → Same raw signal reaches olfactory bulb
    → Expert's chunks DIFFERENTIATE where novice's don't:
      Novice: "grape + alcohol + wood" = 3 chunks activated
      Expert: "cassis + cedar + tobacco + vanilla + tannin level +
              acidity + finish length + body" = 30+ chunks activated
    → Each chunk activation = a feature APPEARING on consciousness stage
    → More chunks = more features on stage = higher resolution

  3-PARAMETER MAPPING:
    Signal: Expert generates STRONGER aggregate signal (many chunks fire)
    Anchor: Expert has RICHER anchors (verbal for each feature)
    Attention: Expert's attention MORE EFFECTIVE (trained to hold wine)
    → All 3 parameters HIGHER for expert → higher resolution = EXPECTED

  ─────────────────────────────────────────────────────────────────
  CASE 2: CHESS MASTER vs BEGINNER
  ─────────────────────────────────────────────────────────────────

  Stimulus: Same chess board position (meaningful game position)

  BEGINNER: Sees individual pieces. "White has a bishop there, black
    has a knight here." Remembers ~4 pieces after 5-sec exposure.

  MASTER: Sees CONFIGURATIONS. "Sicilian Defense, Najdorf variation,
    kingside attack developing, weak d5 square."
    Remembers ~20-25 pieces after 5-sec exposure.

  KEY: De Groot 1946, Chase & Simon 1973:
    → RANDOM positions: master and beginner remember EQUALLY (~4 pieces)
    → MEANINGFUL positions: master 4-5× better
    → = Master's advantage is CHUNKS, not superior memory hardware

  MECHANISM:
    → Master has ~50,000 chess chunks (Gobet & Simon 1998)
    → Each chunk = group of related pieces + positional meaning
    → Same board → master activates ~5-6 large chunks (each holding 4-5 pieces)
    → Beginner activates ~4 individual-piece chunks
    → = Same raw input, different chunk activation, different resolution

  CONSCIOUS EXPERIENCE:
    → Beginner: "I see pieces" (feature-level perception)
    → Master: "I see plans, threats, structures" (configuration-level)
    → The NATURE of conscious content qualitatively differs
    → Not just "more detail" — DIFFERENT LEVEL of organization

  🟢 Chase & Simon 1973: chunk theory of expertise
  🟢 Gobet & Simon 1998: ~50,000 chess templates
  🟢 De Groot 1946/1965: equal recall for random positions

  ─────────────────────────────────────────────────────────────────
  CASE 3: PAINTING — ART EXPERT vs CASUAL VIEWER

    Resolution: ~30-50 features, each labeled, each chainable.

  MECHANISM (same as wine/chess):
    → Expert's visual chunks for painting techniques: ~thousands
    → Each chunk = a learned pattern (brushwork type, composition rule,

    → Same visual input → different chunk activation → different resolution
```

### §3.2 Chunk Density: Precondition, Not Parameter

```
⭐ WHY CHUNK DENSITY IS NOT A 4TH PARAMETER:

  TEST: Is chunk density SEPARABLE from existing parameters?

  ① Can signal be strong WITHOUT chunk density?
     → YES: loud noise, intense pain, bright flash
     → Signal strength has sources INDEPENDENT of chunks:
       Hardware signal (body-input), attention capture, novelty
     → Chunks CONTRIBUTE to signal but don't define it

  ② Can anchor quality be high WITHOUT chunk density?
     → YES: memorized verbal anchor for unfamiliar domain
     → "E=mc²" = verbal anchor with zero physics understanding
     → Anchor is a retrieval PATH — can exist without deep content

  ③ Can attention be effective WITHOUT chunk density?
     → YES: attention to novel stimulus (no prior chunks)
     → Novel object: PFC directs attention, holds briefly
     → Attention is PFC DIRECTION — doesn't require existing chunks

  THEREFORE:
    → Chunk density CONTRIBUTES to all 3 parameters but is not IDENTICAL
      to any of them
    → Chunk density is a PRECONDITION — sets the range for all 3 parameters
    → Like temperature in chemistry: affects reaction rates, not a reactant

  ANALOGY:
    → 3 parameters = 3 dimensions of a space
    → Chunk density = the SCALE of measurement in that space
    → Expert operates in the same 3D space, just at higher magnitudes
    → Adding chunk density as 4th parameter = OVER-SPECIFYING
      (it would double-count what's already in signal + anchor + attention)

  ⚠️ COUNTER-ARGUMENT CHECK:
    Could chunk density be a truly INDEPENDENT dimension?
    → If chunk density affected consciousness IN A WAY NOT CAPTURED
      by signal, anchor, or attention → then yes, it would be independent
    → But: every effect of chunk density I can identify maps to one of the 3:
      More features detected → stronger signal
      More labels available → better anchor
      More structure for PFC → better attention effectiveness
    → No residual effect found → not independent

  🟡 Precondition vs parameter distinction = framework synthesis
  🟢 Individual effects established: Chase & Simon 1973 (chunks → perception),
     Feeling.md §2.2b (clarity → threshold)
```

### §4.1 Einstein + Compass vs Normal Child

```
⭐ THE EINSTEIN CASE — PREDICTION-DELTA AS QUALITY MULTIPLIER

  FACT CHECK (from user's question):
    → Einstein autobiographical notes (Schilpp 1949):
      "A wonder of this kind I experienced as a child of 4 or 5 years,
       when my father showed me a compass. That this needle behaved
       in such a determined way did not at all fit into the kind of
       occurrences that could find a place in the unconscious world
       of concepts."
    → ✅ CONFIRMED: Einstein described the compass as formative

  EINSTEIN (age ~4-5):
    → Had developing prediction model for physical objects:
      Objects move when pushed. Objects fall when dropped.
      Things stay still unless you do something to them.
    → Compass needle: MOVES WITHOUT BEING TOUCHED
    → Prediction-delta: ENORMOUS (violates core physics model)
    → VTA fires maximally → dopamine surge → PFC FULLY engages
    → Signal amplified + attention captured + emotional anchor created
    → Duration: EXTENDED (PFC holds → "what is this?")
    → Encoding: PERMANENT (emotional intensity + prediction model violation)
    → Resolution: every detail noticed (needle movement, case, cardinal points)

  NORMAL CHILD (same age, same compass):
    → Has SAME basic prediction model for objects
    → Sees compass needle move → prediction-delta present
    → BUT: several possible differences:

    POSSIBILITY A — SAME DELTA, DIFFERENT SUSTAINED INTEREST:
      → Both children experience same initial delta
      → Normal child: delta fires → brief curiosity → parent explains
        "it points north" → accepted → delta resolved → attention moves on
      → Einstein: delta fires → curiosity → BUT explanation INSUFFICIENT
        → "How can an invisible force move a physical object?"
        → Delta NOT resolved → sustained drive → repeated re-engagement

    POSSIBILITY B — DIFFERENT META-CHUNKS:
      → Einstein may have had richer causal-physics meta-chunks
```

### §5.3 Three Types of Forgetting

```
⭐ "QUÊN" KHÔNG PHẢI 1 LOẠI — CÓ 3 LOẠI:

  │                        │ DETAIL decays        │                      │
  ├────────────────────────┼──────────────────────┼──────────────────────┤
  │ ② "Không biết đã quên" │ Both meta-chunk AND  │ 99% of people you    │
  │   Don't-know-you-forgot│ content chunk decayed│ walked past          │

  │                        │ → cannot be aware    │                      │

  MECHANISM ANALYSIS:

  TYPE ①: META-CHUNK > CONTENT ANCHOR
    → Meta-chunk = "I experienced X" (verbal anchor: "I met someone")
    → Content = specific details (name, face, conversation)
    → Meta-anchor: verbal + contextual (event) = strong
    → Content anchor: contextual only (weak without rehearsal)
    → Differential decay: meta survives, content decays
    → Result: awareness of gap ("I KNOW I can't remember")
    → = Meta-consciousness about own memory (Level 3 in 4-Level model)

  TYPE ②: NO ANCHOR AT ALL
    → Pattern was on stage briefly → no anchor created → exits → decays
    → No meta-chunk either → no trace → no awareness of loss
    → THIS IS THE DEFAULT for most daily conscious experiences
    → = Normal, adaptive: PFC 4±1 slots cannot accumulate everything
    → If everything were remembered → hippocampal overload → pathological
      (hyperthymesia: remembering everything → often debilitating)

  TYPE ③: DORMANT META-CHUNK
    → Meta-chunk exists but NOT currently active (compiled, below threshold)
    → External cue provides activation → meta-chunk crosses threshold
    → Suddenly AWARE of the forgotten content (or aware you forgot it)
    → = Cue-dependent recall (Tulving & Thomson 1973: encoding specificity)
    → The memory wasn't "lost" — the retrieval PATH was dormant

  FRAMEWORK MAPPING:
    Type ① = meta-chunk (§8.3) + differential anchor decay (Chunk.md §5)
    Type ② = no anchor → noise (§4.3)
    Type ③ = compiled meta-chunk + external trigger → re-entry

  🟢 Encoding specificity: Tulving & Thomson 1973
  🟢 Ebbinghaus forgetting curve: Ebbinghaus 1885
  🟢 Hyperthymesia: Parker, Cahill & McGaugh 2006
  🟡 3-type forgetting taxonomy = framework synthesis (new framing)
```

### §5.4 Must-Forget Design: Why Forgetting Is Adaptive

```
⭐ FORGETTING = DESIGN FEATURE, NOT BUG

  WHY THE BRAIN MUST FORGET MOST CONSCIOUS EXPERIENCES:

  ① CAPACITY CONSTRAINT (PFC):
     → PFC holds ~4±1 items simultaneously (Cowan 2001)
     → Every second: new patterns compete for these slots
     → If PFC held everything → no room for new processing
     → Forgetting = FREEING SLOTS for current processing

  ② HIPPOCAMPAL SELECTIVITY:
     → Hippocampus receives inputs from ALL conscious processing
     → But consolidates SELECTIVELY (not everything → LTM)
     → Selection criteria: emotional significance, novelty, repetition
     → = Filtering mechanism: only "important" patterns encoded

  ③ CORTICAL EFFICIENCY:
     → If every experience were stored as separate memory →
       retrieval would be SLOW (search through everything)
     → Instead: similar experiences MERGED into generalized chunks
     → Generalization > individual memory for prediction
     → = Compression: lose details, keep patterns

  ④ SLEEP CONSOLIDATION:
     → Hippocampus replays day's experiences during sleep
     → SELECTIVE replay: emotionally tagged, repeated, novel
     → Cortex integrates replayed patterns into existing network
     → Non-replayed patterns: gradually DECAY
     → = Nightly cleanup: consolidate important, release trivial

  PATHOLOGICAL WHEN BROKEN:
    → PTSD: emotional tagging TOO STRONG → traumatic memory
      over-consolidated → cannot forget → intrusive re-entry
    → Hyperthymesia (HSAM): remember EVERYTHING → cognitive load,
      difficulty with abstraction (Parker et al. 2006)
    → = Evidence that forgetting IS adaptive (its absence = pathology)

  THIS CONNECTS TO PATTERN LIFECYCLE (§5.1):
    → Most patterns: Entry → Brief hold → Exit → Forgotten (Type ②)
    → Some patterns: Entry → Hold → Encoding → Remembered
    → Selection between these fates = hippocampal + emotional filtering
    → The lifecycle is DESIGNED for most patterns to be temporary

  🟢 Memory consolidation: Diekelmann & Born 2010 (sleep)
  🟢 Hippocampal selectivity: Eichenbaum 2000
  🟢 Adaptive forgetting: Anderson & Hanslmayr 2014
  🟢 Hyperthymesia: Parker, Cahill & McGaugh 2006
```

### §5.5 Meta-Consciousness of Forgetting

```
⭐ "BIẾT LÀ ĐÃ QUÊN" = META-CONSCIOUSNESS (Level 3)

  WHAT META-CONSCIOUSNESS OF FORGETTING REQUIRES:
    → Meta-chunk: "I experienced X" (must exist)
    → Content retrieval failure: cannot recall details
    → AWARENESS of the failure: knows the retrieval failed
    → This awareness = meta-consciousness = Level 3 (4-Level model)

  MECHANISM:
    → PFC attempts retrieval → sends activation to meta-chunk
    → Meta-chunk fires → "I was at the restaurant"
    → PFC follows retrieval path to content → weak signal → fails
    → ACC detects conflict: "retrieval attempted, result = nothing"
    → dlPFC reports: "I know there's something but can't get it"
    → = Tip-of-the-tongue phenomenon (Brown & McNeill 1966)

  LEVELS OF META-CONSCIOUSNESS ABOUT MEMORY:
    Level 1: "I remember X" (recall successful — not meta, just recall)
    Level 2: "I can't remember X" (retrieval failed — awareness of failure)
    Level 3: "I know I used to know X" (meta-chunk about prior knowing)
    Level 4: "I notice I forget things more often now" (meta-meta-awareness)

  WHEN DOES META-CONSCIOUSNESS OF FORGETTING DEVELOP?
    → Requires: theory of own memory (meta-cognitive development)
    → Children ~5-7: begin developing metamemory (Flavell 1979)
    → Before that: forget without KNOWING they forgot (Type ② only)
    → With age: Type ① and ③ increase as meta-cognitive chunks develop
    → = Meta-consciousness of forgetting is a DEVELOPMENTAL ACHIEVEMENT

  FRAMEWORK MAPPING:
    → Meta-chunks (§8.3): meta-chunk about prior experience = prerequisite
    → ACC conflict detection (§5.2): detects retrieval failure
    → PFC report (§1.1): can articulate "I forgot" (verbal anchor for gap)
    → 4-Level model: Level 3 = meta-awareness of own cognitive states
    → All existing framework concepts. No new mechanism needed.

  🟢 Tip-of-tongue: Brown & McNeill 1966
  🟢 Metamemory development: Flavell 1979
  🟡 Meta-consciousness of forgetting = framework synthesis
```

## §6 — TEMPORAL MODEL × 5-CHECKPOINT MAPPING

```
⭐ 2 MODELS: SPATIAL (WHERE) vs TEMPORAL (WHEN)

  5-CHECKPOINT = WHERE in the brain (spatial):
    CP1: TRN gate
    CP2: Tonic mode (thalamic firing state)
    CP3: High-order nuclei (CM-Pf activation)
    CP4: Cortical recurrence (V1↔V4 feedback loops)
    CP5: Global broadcast (PFC + parietal + temporal ignition)

  TEMPORAL STAGES = WHEN in the process (temporal):
    Entry: threshold crossing
    Resolution: content detail level
    Duration: time on stage
    (Encoding: post-consciousness memory — outside scope)

  MAPPING:

  ┌──────────────┬──────────────────────────────────────────────────┐
  │ Temporal      │ Which checkpoints involved?                      │
  │ Stage         │                                                  │
  ├──────────────┼──────────────────────────────────────────────────┤
  │ ENTRY         │ CP1 (TRN opens gate) +                           │
  │               │ CP2 (tonic mode allows relay) +                  │
  │               │ CP3 (CM-Pf signals readiness) +                  │
  │               │ CP5 (ignition threshold crossed)                 │
  │               │ = ALL checkpoints involved in Entry              │

  │               │ = Quality of what appears after Entry             │
  ├──────────────┼──────────────────────────────────────────────────┤
  │ DURATION      │ CP3 (CM-Pf sustain → primary duration) +         │

  │               │ = How long pattern stays after Entry              │
  └──────────────┴──────────────────────────────────────────────────┘

  KEY INSIGHT:
    → Entry involves ALL 5 checkpoints (all must pass for consciousness)
    → Resolution involves CP4-CP5 + chunk density (content quality)
    → Duration involves CP3 + CP5 (sustain mechanisms)

    → Both models describe the SAME process from different angles
    → A checkpoint FAILURE maps to a specific temporal failure:
      CP1 blocked → Entry fails (pattern never reaches stage)
      CP2 burst mode → Entry fails (system not in conscious-compatible state)
      CP3 CM-Pf damage → Entry weakened + Duration shortened
      CP4 disrupted → Resolution degraded (fewer features recurrent)
```
