```

  (Domain.md §2.1 — precision note. This file = deep analysis.)

    FEEDBACK:
      → Absolute: fall = fall, burn = burn, correct = correct.
      → Unconditional: non-negotiable, no exceptions, no shifts.
      → Consistent: same input → same output, always and everywhere.

    STABILITY: ≈ 0 change (on human-observable timescales).

    → Patterns compiled from physics = SHELF-LIFE ≈ ∞


  ② BODY HARDWARE — temporary system, changes very slowly (~100K years):

    PROPERTIES:
      → PFC bandwidth 4±1 dimensions
      → Opioid/dopamine/cortisol reward-pain system
      → Senses: eyes, ears, skin, proprioception, interoception
      → Needs: food, sleep, temperature regulation, social connection, novelty
      → = THE OPERATING SYSTEM on which patterns run

    POSITION IN THE HIERARCHY:
      → Biology RUNS ON physics (atoms, chemistry, thermodynamics)
      → Biology DIES → returns to physics (dust, decomposition)
      → Biology CAN be hijacked: chemicals, injury, disease
      → = Temporary system — not permanent like physics
      → = But STABLE ENOUGH for one human lifetime (~80 years << 100K year evolution)

    CHANGES THROUGH:
      → Natural selection (100K+ years): only through reproduction + survival pressure
      → Epigenetics (1–3 generations): minor, reversible
      → NOT through: will, culture, education, technology (within a single lifetime)

    → Patterns compiled from body hardware = SHELF-LIFE ≈ ∞ within one lifetime

      → Market, law, culture, norms, trends, language, technology

      → Emerges from interactions among millions/billions of bodies

      → Collective RUNS ON bodies (needs bodies to exist)
      → Body runs on physics

      → Each layer: its own emergent properties, reducible but NOT PREDICTED from the layer below

    CHANGES THROUGH:
      → Participants change behavior → collective shifts
      → New technology → collective reorganizes
      → Information flow → norms evolve
      → External shocks (war, pandemic, discovery) → rapid shift

      → Small scale (startup culture, friend group) → shift FASTER

      Mathematics collective (peer review, proofs) ≈ very stable

      → WHY the difference? = §3 (dependency ratio)

    → Patterns compiled from collective = SHELF-LIFE LIMITED

  ⭐ STACKED HIERARCHY — NOT SEPARATE:

    Physics → enables Body → enables Collective
    ↑ base       ↑ runs on physics   ↑ runs on many bodies

    BUT FEEDBACK FLOWS IN BOTH DIRECTIONS:
      → Physics → Body: gravity pulls, heat burns
      → Collective → Body: fires you, praises you, punishes, rewards
      → Body → Collective: revolt, innovate, leave, comply
```

```
      Body (always present) → Social (vivid) → Physics (requires alertness to perceive clearly)

      → Body signals = DIRECT (no modeling needed, no cortical processing required)

      → Eyes closed: STILL see parents, friends (compiled chunks in memory)

      → Requires OPEN EYES + alertness → only then perceives physical space clearly

      → Because: physics model = CONSTRUCTED by cortex from sensory input
        → Requires ACTIVE processing → expensive → degrades when resources are low
      → Body signals REMAIN STRONG when tired (pain still hurts, heat still burns)
        → Because: interoception = direct, no model construction needed

    WHY INVERTED — Domain.md §1 ALREADY EXPLAINS:
      "Humans do NOT contact domain directly. Body touches → feedback → brain knows."
      → Body = THE INTERFACE. Closest to the interface = most vivid.
      → Physics = furthest from the interface (known ONLY through body reflection) = dimmest.
      → Social = intermediate (compiled from body interactions, vivid in memory)

    → = Ontology: Physics > Body > Collective (fundamental → emergent)
    → = Experience: Body > Social > Physics (direct → constructed)
    → = NOT A CONTRADICTION: most fundamental ≠ most directly accessible.
      Like quarks being more fundamental than tables,
      but tables feel "more real" in experience.

    🟢 Depersonalization/derealization when sleep-deprived (documented neuroscience)
    🟢 Social pain same pathways as physical pain (Eisenberger et al. 2003)
    🟢 Interoception primary, exteroception requires cortical model (Craig 2002)
    🟡 "Inverted hierarchy" as named concept = framework synthesis

    → Pattern from physics: "falling = pain" — body compiles, holds permanently. ✓
    → Pattern from collective: "university degree = good job" — body compiles THE SAME WAY. ✓

    → Physics DOESN'T SHIFT → pattern stays correct → body is NEVER disconfirmed
```
