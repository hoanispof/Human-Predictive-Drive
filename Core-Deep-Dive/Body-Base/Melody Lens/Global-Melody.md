```
    CEO set "company melody" (values, process, culture)

      → Company melody = FIXED (rank, command, uniform)

    → Resonance = khi cả 2 SPM match thành công → "à, hiểu nhau!"

      → Bố mẹ lệch: "con phải giỏi" vs "con phải vui" → tranh cách nuôi
      → Công ty: cùng muốn lợi nhuận → chia KHÔNG đều → conflict
      → (Conflict-Dynamics.md: Overlap × Scarcity × Commitment)

```

```
    → = "Trong văn hóa NÀY, hành vi NÀY ok → hành vi KIA không ok"
    → Compiled chunks từ shared exposure → "normal" per-culture


  VÍ DỤ cultural KEYs:

    Việt Nam (key: Family + Hierarchy + Education):
      → Notes OK: tôn trọng lớn tuổi, gia đình ưu tiên, học giỏi
      → Notes KHÔNG OK: nói trái ý bề trên, bỏ học
      → Melody TRONG key → harmony → status ok
      → Melody NGOÀI key → dissonance → social signal

    Mỹ (key: Individual + Innovation + Competition):
      → Notes OK: tự lập, dám khác, thắng
      → Notes KHÔNG OK: phụ thuộc, copy
      → Melody TRONG key → harmony
      → Melody NGOÀI key → dissonance

    Nhật (key: Group Harmony + Precision + Respect):
      → Notes OK: hòa hợp nhóm, chính xác
      → Notes KHÔNG OK: nổi bật cá nhân
      → Melody NGOÀI key → dissonance CỰC MẠNH


  CULTURE SHOCK = KEY CHANGE đột ngột:
    → VN → Mỹ: key Family → key Individual
    → Personal melody VN chơi trong key Mỹ = patterns LỆCH → body signal
    → Chunk-Gap + Chunk-Miss: patterns cũ không match, patterns mới chưa có
    → Thời gian adapt: tháng → năm (compile new patterns dần)
    → = KHÔNG phải "yếu đuối" → melody ĐANG ở KEY KHÁC

  KEY CHANGE TRONG CÙNG NƯỚC:
    → Nông thôn → thành phố: key khác (tốc độ, cạnh tranh)
    → THPT → đại học: key khác (tự do hơn → autonomy jump)
    → Đi làm → nghỉ hưu: key khác (Chunk-Miss: vai trò quen ABSENT)
    → = MỖI CONTEXT SWITCH = mini key change → cần compile mới
```
