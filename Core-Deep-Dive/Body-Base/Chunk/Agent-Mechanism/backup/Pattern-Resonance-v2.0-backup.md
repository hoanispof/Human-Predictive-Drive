## §0 — THESIS + DEFINITION

### §0.1 — Definition

**Pattern-Resonance** = emergent mutual phenomenon arising khi 2+ agents' Self-Pattern-Match mechanisms co-fire, VÀ cả 2 fill gap CỦA MÌNH qua interaction với nhau. Reflects **actual alignment** between agents — but alignment KHÔNG cần cùng cảm xúc hay cùng gap. **Retrospectively knowable only** — no real-time direct access. Serves as **feedback calibration source** for SPM library refinement.

## §5 — 4 ĐIỀU KIỆN CẦN

Pattern-Resonance KHÔNG emerge tự phát. Cần 4 conditions ĐỒNG THỜI.

### §5.1 — Condition 1: Both sides running SPM

Cả 2 agents phải có SPM mechanism active firing toward nhau.

```
  FAIL NẾU:
    → 1 bên không có SPM capability (infant < 18 tháng, côn trùng, vật vô tri)
    → 1 bên suppress SPM (dehumanization, hostile schema)
    → 1 bên absent (parasocial fan → celebrity: celebrity không biết fan tồn tại)
    → Cross-species: dog không run human SPM templates → architecture blocked

  PARTIAL:
    → Adult → toddler (18-24 tháng): SPM emerging → partial PR, tăng dần
    → Human → AI: AI match patterns KHÔNG PHẢI qua SPM → PR giả (§8.2)
```

### §5.2 — Condition 2: Feedback channel available

Phải có kênh feedback để mỗi bên check predictions against reality.

```
  CHANNEL QUALITY GRADIENT:
    Face-to-face: richest (facial, vocal, body, verbal, immediate)
    Video call: reduced body
    Phone: voice only
    Text: delayed, verbal only
    One-way: media consumption → NO return feedback
    None: imagined interaction → NO PR possible

  ⭐ Feedback = CONSTITUTIVE, not just helpful
    Không có feedback → rich SPM investment → parasocial distortion (not PR)
```

### §5.3 — Condition 3: Pattern match — NOW with F1/F2 dimension

```
⭐ v2.0 REFINEMENT — Condition 3 có 2 CHẤT LƯỢNG:

  F1 MATCH (body-level patterns match):
    Cả 2 fire similar affective/somatic response cho shared stimulus
    → F1 body confirm → DEEP PR
    Ví dụ: music fans cùng xúc động, dancers cùng feel rhythm

  F2 MATCH (logic-level patterns match):
    Cả 2 predict đúng hành vi nhau qua logic/schema
    → F2 accurate → FUNCTIONAL PR (không phải deep)
    Ví dụ: mathematicians cùng hiểu proof, business partners predict nhau

  DOMAIN-SPECIFIC: match ở 1 domain, mismatch ở domain khác = bình thường
    Mathematicians F2 match ở math, không nhất thiết F1 match ở cảm xúc
    = PR emerge PER DOMAIN, không cần full personality match
```

### §5.4 — Condition 4: Sufficient interaction time

```
  TIME SCALES:
    Minimum: seconds sustained co-firing (shared laugh requires SIMULTANEOUS)
    Moderate: minutes (brief conversation → rapport)
    Deep: hours to days
    Compiled: years (family, long-term partners)

  EXCEPTION — prior compiled PR:
    Long-term partners re-encounter → re-activate compiled PR nhanh
    "Old friends gặp lại 10 năm vẫn hiểu" = compiled library enables rapid re-firing
```

### §5.5 — All 4 necessary

Thiếu BẤT KỲ 1 condition → PR KHÔNG emerge:
- Condition 1 fail → no emergence possible
- Condition 2 fail → parasocial distortion
- Condition 3 fail → discomfort, distance, failed connection
- Condition 4 fail → fleeting impression, not PR

4 conditions = **necessary**. Whether **sufficient** = less certain (some combinations may still fail).

## §6 — RETROSPECTIVE DETECTION

### §6.1 — Why not real-time

```
  SELF CÓ:                           SELF KHÔNG CÓ:
  → Own SPM process                  → Target's internal state
  → Observable target cues           → Target's SPM toward self
  → Own response to target           → Whether templates ACTUALLY match
  → Context + situation              → Ground truth of alignment

  → PR = actual internal alignment → self CANNOT directly observe
  → Self runs SPM forward (predict) → verification requires BACKWARD inference
```

### §6.2 — 3 inference methods

```
  ① IMMEDIATE OBSERVABLE (trong interaction):
    Target verbal: "đúng rồi, chính xác" vs polite nod
    Target facial/body: genuine smile vs polite, relaxation vs tension
    Target elaboration: build on → deepening vs change subject → stalling

  ② CONVERSATIONAL DYNAMICS:
    Deepening: topics personal hơn, vulnerability tăng, tangents explored
    Stalling: topics cạn, awkward silences, forced transitions
    Mutual extending vs mutual contracting

  ③ POST-INTERACTION OUTCOMES:
    Short-term: muốn gặp lại? Hành vi consistent với prediction?
    Long-term: relationship progression hay regression?
    Pattern over time: consistently deepening vs consistently failing
```

### §6.3 — Accuracy limits

```
  4 PROBLEMS:

  ① Target's actual state remains OPAQUE:
    Self chỉ thấy behavior → behavior match prediction nhưng lý do KHÁC
    "Họ đồng ý" → có thể vì polite, không phải vì genuine match

  ② Self's memory is RECONSTRUCTIVE:
    Memory reshape qua confirmation bias → "nhớ" khác thực tế

  ③ Multiple VALID interpretations:
    Same outcome data → supports nhiều PR inferences

  ④ Self's SPM of SELF is limited:
    Alexithymic self → cannot assess own reactions accurately
    "Tôi feel connection" vs "tôi project connection" → khó phân biệt
```

### §6.4 — Approximate detection = sufficient

Dù imperfect, **approximate retrospective detection** đủ:
- Trends over many interactions smooth noise
- Strong patterns emerge clearly
- Library refinement proceeds with imperfect but directional feedback
- Self-awareness training improves accuracy

### §6.5 — Mutual verification

**Higher-fidelity**: cả 2 phía reflect + share ("How did that land for you?") → approximate ground truth. Therapy + deep friendships rely on explicit meta-conversation.

## §7 — FEEDBACK CALIBRATION LOOP

PR's primary architectural function: **feedback calibration** cho SPM library.

### §7.1 — Loop architecture

```
  [Self-Pattern state t₀]
          ↓
  [SPM fires forward prediction]
          ↓
  [Interaction proceeds]
          ↓
  [Outcome observed]
          ↓
  [PR inferred retrospectively]
          ↓
  [Library update]:
    → Confirmed chunks strengthened (Hebbian)
    → Misfired chunks weakened
    → New target-specific chunks created
    → Retrieval heuristics refined per-target
    → Schema links adjusted
          ↓
  [Updated SPM library → next prediction BETTER]
```

### §7.2 — 4 library update types

```
  ① CHUNK STRENGTHENING:
    Prediction matched → chunks reinforced → future retrieval priority ↑
    "Friend prefers practical gifts" → confirmed → strengthened

  ② CHUNK WEAKENING:
    Prediction failed → chunks reduced weight
    "Colleague frown = upset" → actually tired → weakened

  ③ NEW CHUNK CREATION:
    Target unexpected behavior → new chunk formed
    "Didn't know she loved jazz" → new "friend's music: jazz" chunk

  ④ RETRIEVAL HEURISTIC REFINEMENT:
    Meta-learning: WHICH chunks for WHICH target
    "For this colleague, prioritize verbal-cognitive — affective cues subtle"
```

### §7.3 — Time scales

```
  Immediate:    mid-conversation micro-adjustments (seconds)
  Short-term:   post-interaction reflection (hours-days)
  Medium-term:  relationship trajectory observation (weeks-months)
  Long-term:    deep relationship compilation (years)
  Cross-target: lessons transfer to similar targets
```

### §7.4 — Bird & Cook 2013: loop break

```
⭐ DECISIVE EVIDENCE — alexithymia breaks feedback loop:

  🟢 Bird & Cook 2013:
    Alexithymia → poor self-reading → thin SPM templates

  LOOP BREAK MECHANISM:
    Alexithymic self runs SPM with thin chunks
      → Prediction noisy (thin templates)
      → Interaction unfolds
      → Self CANNOT accurately read OWN reaction to outcome
      → Feedback signal GARBLED
      → Library update noisy or FAILS
      → Next SPM still uses thin chunks
      → Empathy deficit PERSISTS over time

  → Empathy KHÔNG improve through ordinary experience trong alexithymic
  → Vì feedback loop BROKEN

  INTERVENTION TARGET:
    Fix self-awareness (upstream self-chunks) → loop begins working
    → PR inference improves → SPM library refines → empathy improves
    = Bird & Cook 2013 insight operationalized trong framework

  🟢 CRITICAL: this is WHY alexithymia is persistent without targeted intervention
```

### §7.5 — Training methods

```
  5 METHODS accelerate calibration:

  ① Post-interaction reflection:
    "What did I predict? What happened? What surprised me?"

  ② Explicit hypothesis tracking:
    Note predictions BEFORE interaction → check against outcomes

  ③ Feedback seeking:
    Ask target directly ("How did that land?") → compare → update

  ④ Journaling social experiences:
    Written reflection strengthens chunks + reveals systematic biases

  ⑤ Therapy as structured calibration:
    Therapist facilitates self-awareness → co-explore predictions + outcomes
```

### §7.6 — 5 loop failure modes

```
  ① Alexithymia (§7.4): cannot interpret feedback → no update
  ② Parasocial (no feedback source): library drifts without reality check
  ③ Schema suppression: ignore disconfirming feedback → dehumanization persists
  ④ Confirmation bias: interpret feedback to fit prediction → no genuine update
  ⑤ Trauma distortion: past trauma → systematic mis-reading → stuck patterns
```

## §9 — OBLIGATION-TRAPPED NON-RESONANT INTERACTION

### §9.1 — Scenario

2 agents với PR Baseline THẤP (§4). PR không emerge tự nhiên. Bình thường → drift apart. NHƯNG: obligation KHOÁ cả 2 trong tương tác liên tục. Condition 3 genuinely fail + CANNOT EXIT.

**Common settings:**
- Parent ↔ child: con phụ thuộc + parent L2 drive + social obligation
- Marriage/partnership: con chung, tài chính chung, áp lực xã hội
- Workplace: employee phải tương tác hàng ngày với sếp/đồng nghiệp
- Extended family: obligation văn hóa — sống cùng ông bà, anh chị em
- Military/institutional: forced proximity + authority chain

### §9.2 — 4-step mechanism

```
STEP 1 — NATURAL PR FAILURE:
  SPM fires → patterns DON'T match → body: giảm tương tác
  NHƯNG obligation blocks exit → DISSONANCE: drive disengage + forced engage

STEP 2 — SUPPRESS NATURAL PATTERNS:
  PFC INHIBIT natural response → tránh conflict
  Autonomy-Hardware §5 — 2-layer dissonance:
    Layer 1: efference mismatch (natural → forced)
    Layer 2: vmPFC meta-learning "in THIS relationship, I CANNOT be me"
    → DRN passive coping (🟢 Maier & Seligman 2016)
  Body feel: "gượng" (forced), "giả" (fake), "mệt" (tired)

STEP 3 — AMPLIFY/FABRICATE PATTERNS:
  Select chunks KHÔNG PHẢI dominant CỦA MÌNH → cố match target:
    "Ít nói → với sếp phải hoạt bát"
    "Thích explore → với bố phải tỏ ra ngoan"
  = PFC-HEAVY, NOT compiled, deliberate MỖI LẦN
  = KHÔNG BAO GIỜ compile (patterns không genuinely của mình → no Hebbian)
  = FRESH PFC effort MỖI LẦN → không giảm cost

STEP 4 — HOPE FOR PARTIAL PR:
  Target nhận ra effort → partial PR, reward NHỎ → sustain tạm
  Target KHÔNG nhận ra → effort wasted → frustration compound
  Target phản ứng NGƯỢC ("giả tạo") → effort BACKFIRE → worse
```

### §9.3 — Cost analysis — CUMULATIVE

```
  (a) AUTONOMY COST:
    Mỗi suppress = efference mismatch → body: "tôi KHÔNG ĐANG LÀ TÔI"
    Repeated → vmPFC: "in relationship NÀY, tôi không thể là tôi"
    → DRN → passive coping IN THIS RELATIONSHIP
    🟢 Deci & Ryan 2000: autonomy need frustration → reduced wellbeing

  (b) CORTISOL ACCUMULATION:
    Mỗi forced interaction = mild cortisol Role ② Holding
    Repeated → baseline SHIFT upward
    Chronic → possible Role ⑤ Silent: cortisol HIGH nhưng self-signal atrophy
    = "Quen rồi, không sao" — nhưng body deteriorate

  (c) PFC DRAIN:
    Fabricating patterns = deliberate PFC MỖI LẦN (no compile)
    = "Small talk với người không click" mệt DISPROPORTIONATE

  (d) SPM LIBRARY DISTORTION:
    Library build trên PERFORMED patterns → "with this person, I am [performed]"
    Nếu obligation lifts → "tôi không biết cách LÀ THẬT với họ"
```

### §9.4 — "Functional avoidant" variant

```
🟡 BÊN NGOÀI: regularly interacts, fulfills obligations, appears normal
   BÊN TRONG: natural patterns SUPPRESSED, no genuine self present
   = Behavioral presence, experiential absence

  DISTINGUISHED FROM:
    Path A monitor: genuinely INVESTS SPM → costly but REAL
    Path B withdraw: openly DISENGAGES → others see
    Functional avoidant: APPEARS engage → actually PERFORMING → invisible

  KHÓ DETECT:
    Obligation provides behavioral framework → behavior adequate
    Self CÓ THỂ tự lừa: "tôi ổn" (Cortisol Role ⑤ parallel)
    Chỉ detect khi structural change FORCE reveal (ly hôn, con ra riêng, nghỉ việc)

  🟡 "Functional avoidant" = framework synthesis
```

### §9.5 — Vicious cycle

```
  PR Baseline THẤP → suppress/amplify required
  → Cortisol + PFC drain → PFC capacity DECREASES
  → Interactions WORSE (leaks, irritation)
  → Other party frustrated → THEIR patterns hostile hơn
  → Baseline drops FURTHER → ACCELERATING spiral

  COMPOUNDED: suppress → no genuine SPM → library KHÔNG refine
  → Cannot LEARN this person → more errors → more suppress → more cost
```

### §9.6 — Parent-child specific

```
🟡 CASE QUAN TRỌNG NHẤT (child CANNOT exit + developmental impact):

  Baseline THẤP → child suppress MỖI NGÀY
  → vmPFC meta-learning: "my prediction doesn't matter" → learned helplessness RISK
  Parent CŨNG frustrated → inconsistency
  → Baseline ⑦ ẢNH HƯỞNG parent consistency ① — 2 biến KHÔNG độc lập

  Baseline CAO → positive cycle → parent MORE consistent → healthy development
  Baseline THẤP → negative cycle → parent LESS consistent → compound risk

  🟢 Thomas & Chess 1977: poor fit → behavioral problems
  🟢 Baumrind 1966: authoritative (aware of fit) > authoritarian (ignore fit)
```

### §9.7 — 5 resolution pathways

  (a) FIND DOMAIN OF OVERLAP:
    2 agents may not match globally nhưng match IN SPECIFIC DOMAIN

    Understand difference INTELLECTUALLY even without affective PR
    "Tôi biết bố coi trọng X dù tôi coi trọng Y" → friction GIẢM
    Không SÂU nhưng FUNCTIONAL

  (c) REDUCE OBLIGATION WHERE POSSIBLE:
    Adult child → structural independence → choose frequency
    Marriage → renegotiate expectations
    Paradoxically: less forced contact → better contact

  (d) EXPLICIT META-COMMUNICATION:
    "Chúng ta dường như khó connect ở [domain]" — NAME the gap
    Cả hai biết → suppress cost GIẢM (không cần pretend)

    Mismatch ở expression modality (verbal ↔ somatic):
    → Bên verbal HỌC đọc body signal. Bên somatic HỌC articulate.
    → Slow nhưng genuine improvement (EXPAND, không suppress)

## §10 — COMMUNICATION OUTCOMES

PR presence/absence correlates với observable communication outcomes.

### §10.1 — Positive signals (PR present)

- Conversation deepening (topics personal hơn, vulnerability tăng)
- Mutual investment (both contribute, build on each other)
- Time extension (kéo dài hơn planned)
- Follow-up desire (both muốn gặp lại)
- Remembered positively
- Subsequent behavior aligned with prediction
- Relationship progression over time

### §10.2 — Negative signals (PR absent/failed)

- Conversation stalling (topics cạn, awkward silences)
- One-sided effort (1 talks, other withdraws)
- Time contraction (kết thúc sớm)
- Avoidance of next meeting
- Remembered as strained
- Subsequent behavior contradicts prediction
- Relationship regression or termination

### §10.3 — Caveats

- Social norms mask PR signals (polite conversation continues despite PR absence)
- Power dynamics affect behavior independently of PR
- Individual differences in expression (introverts vs extroverts express PR differently)
- Context matters (work vs party vs therapy)
- One-sided detection error (self feels PR that other doesn't)

### §10.4 — Directional causation + trajectory

PR → outcome (not outcome → PR). Outcomes REVEAL whether PR was present.

```
  Sustained deep PR    → relationship deepens + stabilizes
  Sustained shallow PR → maintains but doesn't deepen
  Declining PR         → relationship decays
  Never-emerging PR    → relationship fails early

  If consistently fails despite effort → PR not emerging (conditions not met)
  If naturally progresses without strain → PR present
```

## §11 — INDIVIDUAL + CULTURAL VARIATION

### §11.1 — Individual variation sources

**Self-chunk library richness:**
- Alexithymia spectrum → thin affective chunks → reduced F1 PR
- High self-awareness → rich chunks → high-quality PR potential

**Pattern-Type bias:**
- Somatic-dominant → strong F1 affective + somatic PR
- Verbal-dominant → strong F2 verbal-cognitive PR
- Composite-capable → rich PR across modalities

**Attachment history:**
- Secure → flexible PR engagement, robust to conflict
- Anxious → intense PR pursuit, distress at perceived absence
- Avoidant → limited PR engagement, protective distance
- Disorganized → inconsistent PR patterns

**Training level:**
- Trained (therapists, meditators) → higher-quality PR even with strangers
- Untrained → rely on default patterns

### §11.2 — Twin variation = maximum hardware case

```
  Identical twins = ~100% hardware overlap → F1 match MAXIMUM
  Fraternal twins = ~50% → F1 match HIGH
  Siblings = ~50% + different developmental timing → MODERATE
  Self-selected friends → potentially HIGHER than sibling ở specific domain
    (self-selection = optimize for domain overlap)

  🟢 Bouchard 1990: twins apart → immediate bond = hardware alone significant
  🟢 Segal 2012: identical > fraternal closeness
```

### §11.3 — Cross-cultural variation

**Emotional vocabulary:**
- Rich vocabulary → rich affective chunks → rich F1 PR
- Sparse vocabulary → coarser F1 (may have rich other Pattern-Types)

**Body awareness traditions:**
- Strong body practices (yoga, martial arts) → rich somatic PR
- Screen-mediated + verbal-dominant → reduced somatic PR

**Communication norms:**
- High-context cultures → rich implicit PR detection (non-verbal emphasis)
- Low-context cultures → explicit verbal PR confirmation

**Framework caution**: cross-cultural research ON PR is thin (framework-specific construct). Observations above inferred from broader cross-cultural psychology.
