# Self-Pattern-Modeling — Solo Forward Simulation Mechanism

> **Convention:** Full name = **Self-Pattern-Modeling**. Abbreviated = **Self-Pattern-Modeling**.
> Renamed from "Self-Pattern-Match" because "Match" = passive comparison,
> "Modeling" = active simulation. Self-Pattern-Modeling is forward simulation
> (build model + run + predict), not pattern-matching.

> You see your close friend crying. Your body feels a brief sadness —
> even though nothing happened to you.
> That is Compiled: body SIMULATES the other person's state — automatic, nearly unconscious.
>
> Then you think: "they're sad because of a breakup — they probably want someone to listen."
> That is Fresh: PFC CHAINS a draft prediction about what comes next — deliberate, effortful.
>
> Same mechanism. Same chunk library. But 2 DIFFERENT outputs.
> "Feeling" and "Logic"? Just LABELS — Einstein "felt" mathematics because math was COMPILED.
> A new therapist "thinks through" emotions because new cases are still FRESH.
> The real axis: COMPILED ←→ FRESH. Content doesn't determine it.
>
> And ALL of it depends on Per-Agent Valence — how does the body evaluate THIS agent?
> Positive valence: Compiled = empathy. Negative valence: Compiled = REVERSED
> (pleasure when the enemy suffers).
> Same mechanism, same 6 steps — different output because different valence.

### §1.3 — One-Paragraph Summary

Self-Pattern-Modeling is the mechanism by which you use yourself as a template to model others — APPLICATION-1 on Simulation-Engine (Other × Present × Simulate). Not direct mind-reading, not a hardware mirror module — it is simulation-based inference using self-chunks as building blocks. Self-Pattern-Modeling runs 2 parallel functions: Compiled body-level simulation (near-automatic, cost ≈ 0), and Fresh PFC draft prediction (deliberate, costly). Context determines WHICH chunks get retrieved (unconsciously selected). Self-Pattern-Modeling is 1 mechanism across 3 dimensions (Processing Level × Valence Direction × Output Intensity), not 4 separate skills.


### §15.2 — Sadism vs Dehumanization
