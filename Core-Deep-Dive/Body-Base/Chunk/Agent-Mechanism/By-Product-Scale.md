```

    → RISK: survivorship bias (only remember successes).

  LEVEL 2 — TRUST BYPASS:
    Followers bypass direct verification → trust node's judgment.

    Trust compiled at cultural/institutional level → individual receives PRE-COMPILED.

    → Collective-Arc-Dynamics §5: "true but unnecessary" → shelf-life LIMITED.
    → Pre-compiled chunks CAN expire (degree inflation, market shift).
    → Individual DOESN'T KNOW expired until body-feedback mismatch.
    → FASTEST but MOST FRAGILE (entire generation can compile wrong).

    │ 3 Inst. │ FAST     │ LOW          │ Institutional       │

    → = Same trade-off as personal→role→symbolic maps (Coordination-Node §4).
```
