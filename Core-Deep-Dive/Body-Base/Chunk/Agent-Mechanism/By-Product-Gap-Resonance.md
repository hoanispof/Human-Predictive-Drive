## §1 — DEFINITION: RESONANCE CORE + BY-PRODUCT MATCH

### §1.1 — 3 principles of "100% individual"

```
🟡 FOUNDATION — everything INSIDE each entity is INDIVIDUAL:

  ① EMOTIONS = 100% INDIVIDUAL:
    Body-feedback = f(hardware × chunk library × current state × context)
    These 4 variables are NEVER identical between 2 entities → output is never identical.
    "Both enjoy football": the defender resolves NE, the striker gets VTA-dopamine,
    the goalkeeper gets opioid.
    "Happy" = a shared PFC label for DIFFERENT body-feedback patterns.
    🟢 Bouchard 1990: identical twins → nearly alike but NEVER identical.

  ② GAP = 100% INDIVIDUAL:
    Gap = f(surrounding chunks) — Gap-Direction §1.
    Individual chunks → individual gap. Same classroom: A is curious, B corrects errors, C is bored.

  ③ DRIVE = 100% INDIVIDUAL:
    Drive = f(gap + body-feedback + current state).
    Mother's drive to "protect" ≠ child's drive to "explore" → DIFFERENT but they fit together.

  → Each entity = 1 ENTIRELY SEPARATE WORLD. Applies CROSS-SPECIES.
```

```

## §3 — 2 RESONANCE SCENARIOS + TEMPORAL DYNAMICS

### §3.1 — Scenario ① Direct (compiled, cost ≈ 0)

```
⭐ BY-PRODUCT MATCH NATURAL — NO PFC EFFORT REQUIRED:

  2 SUB-TYPES:

  ①a — HARDWARE BY-PRODUCT (from ATTRIBUTES / EXISTENCE):
    Entity B EXISTS → A perceives → matches A's hardware gap → A receives reward.
    B DOES NOT NEED TO DO ANYTHING. Unidirectional per person.
    HABITUATES (Weber-Fechner): perception adjusts → normalizes.
    EXCEPTION: evolution-protected bonds (mother-infant) → habituation EXTREMELY SLOW.
    Examples: husband → beautiful wife, mother → newborn (oxytocin), person → cute dog.

  ①b — COMPILED Self-Pattern-Modeling BY-PRODUCT (from Compiled Mutual Self-Pattern-Modeling):
    A's Compiled Self-Pattern-Modeling fires → body simulates B's state → responds appropriately.
    → B FEELS understood → B's Self-Pattern-Modeling fires → responds → FEEDBACK LOOP.
    ANTI-HABITUATES: Hebbian — each use → stronger → more automatic.
    → Cost DECREASES, accuracy INCREASES, reward INCREASES → POSITIVE SPIRAL.
    Example: "old couple who understand each other with half a sentence" = maximum Compiled ①b.
    = DEEPEST + MOST SUSTAINABLE form of resonance.

  ①a + ①b RELATIONSHIP:
    ①a INITIATES + PROVIDES SAFETY BUFFER for ①b (beauty → proximity → Self-Pattern-Modeling begins).
    ①b REPLACES ①a as ①a habituates.
    HEALTHY = ①b grows LARGE ENOUGH to replace declining ①a.
    "The fire is gone" = ①a done + ①b underdeveloped.
    "Mature love" = ①b dominant, ①a = bonus.

  🟢 Feldman 2007: interpersonal synchrony.
  🟢 Coan & Sbarra 2015: social co-regulation.
  🟢 Hedonic adaptation: established psychology.
```

### §3.2 — Scenario ② Fresh Processing Mediated (cost > 0)

```
⭐ PFC INVESTS COST TO CREATE CONDITIONS FOR MATCH:

  Fresh processing ITSELF ≠ resonance. PFC = COST to create CONDITIONS.
  Resonance = what EMERGES when match SUCCEEDS.

  3-COST MODEL (Inter-Body-Mechanism §4):
    ① PFC draft: processing load of novel chains
    ② Suppress: efference mismatch cost of overriding compiled response
    ③ Uncertainty: cortisol holding cost
    TOTAL = ① + ② + ③ (independent, cumulative)

  3 SUB-CASES:

  ②a — GENUINE EXPLORATION (Fresh Type A: Hold only):
    PFC holds new pattern. Body OPEN. Cost: ① only, declining.
    CAN compile → becomes ①. = HEALTHY path.

  ②b — FORCED EFFORT (Fresh Type B: Hold + Suppress):
    PFC holds new + suppresses old. Body CONFLICT.
    3 OUTCOMES (PFC-Operations §3):
      A: Genuine shift → compiled ①. B: Compiled suppress → flat. C: PFC fail → revert.
    = DANGEROUS — can create compiled suppress.

  ②c — STRATEGIC (Tool-mode mimicking Agent-mode):
    PFC mimics Agent-mode. Goal = extract outcome.
    WHEN GOAL ENDS → dissolves. = NEVER compiles into ①.
