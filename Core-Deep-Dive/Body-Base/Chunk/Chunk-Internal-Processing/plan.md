## §0 — North Star

**Goal**: Formalize **how internal processing transforms existing chunks into novel chunks** — without direct external shared input, seeking label post-formation (or never labeled). Age-independent mechanism.

**Primary insight from N+4 prep**: Internal novel formation is the mechanism that drives:
- Logical connection between chunks (PFC hold A + B, check if they link)
- Aha moments + tacit knowledge (Polanyi)
- Feeling/intuition gradient (felt sense)
- Chunk decay without external anchor reinforcement
- Adult "gần đúng / mâu thuẫn / concept mới chưa có từ" → drives language evolution (gap 2, H12)

**Distinction vs F3**: F3 focuses on HOW external sources install chunks. F4 focuses on HOW internal mechanism transforms existing chunks. Same case (Einstein reading Newton → aha relativity) can appear in both folders with different analytical focus.

**Scope**:
1. Chunk-chunk logical connection (Thread 2, H2 test — 4th connection type)
2. Feeling vs intuition gradient (Thread 3, H5)
3. Chain + anchor decay dynamics (Thread 8, H6)
4. Right-wrong vague (Thread 10)
5. Insight + tacit knowledge (aha, Polanyi, gap 2 formalized)

**Primary test hypotheses**: H2, H5, H6, H12 tentative

**Estimated scope**: ~5100-6900 lines across 7 files + plan. 2 sessions (N+8, N+9).
