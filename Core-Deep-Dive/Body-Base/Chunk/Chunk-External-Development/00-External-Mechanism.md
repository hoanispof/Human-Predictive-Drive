## §1 — 5 External Install Mechanisms

### §1.0 — Core principle

```
EXTERNAL INSTALL = process by which chunk template EXISTS OUTSIDE individual
                   → gets COMPILED INSIDE individual's neural substrate

⭐ KEY DISTINCTION:
  F1 (substrate): raw input → pattern → chunk (bottom-up, no template needed)
  F3 (external):  external template → individual compile (template exists outside)
  F4 (internal):  existing chunks → novel combination (no external template)

SAME MECHANISM AT ANY AGE:
  → 9mo infant joint attention = 40yo adult shared pointing = SAME mechanism
  → Different: number of existing chunks (library size)
  → = F3 scope covers MECHANISM, not age brackets
```

### §1.1 — Mechanism 1: Co-attention / Joint Attention

```
DEFINITION:
  2+ người CÙNG CHÚ Ý vào 1 object/event CÙNG LÚC
  → Shared reference point → chunk installs in BOTH minds

MECHANISM (neural):
  ① Person A perceives object X (bottom-up: sensor → cortex → chunk fire)
  ② Person B perceives SAME object X (same pathway)
  ③ ADDITIONALLY: A perceives B looking at X (social: gaze direction detection)
     → STS (superior temporal sulcus) tracks gaze direction  🟢 Pelphrey 2005
  ④ A's brain: object X + B's attention toward X = CONVERGENCE
     → "Chúng ta đang nhìn CÙNG THỨ" = shared reference chunk compiles
  ⑤ A and B now hold OVERLAPPING chunks about X
     → NOT identical (each processes through own chunk library)
     → But OVERLAPPING enough for communication about X

DEVELOPMENTAL TIMELINE:
  ~6-8 weeks:    mutual gaze (E12 social smile — gaze detection hardware online)
  ~4-6 months:   gaze following (baby follows adult's gaze direction)
  ~9-12 months:  ⭐ TRIADIC joint attention (baby + adult + object)
                 = Tomasello 1995: "9-month revolution"
                 = First reliable external chunk install via shared attention
  ~12-18 months: proto-declarative pointing (baby DIRECTS adult attention)
                 = Baby actively CREATES co-attention episodes
  Adult:         "nhìn kìa!" = same mechanism, instant, automatic

🟢 Tomasello 1995, 2003 — joint attention as foundation of cultural learning
🟢 Mundy & Newell 2007 — joint attention development review
🟢 Pelphrey et al. 2005 — STS gaze direction processing

⭐ IMPLICATION:
  → Co-attention = SIMPLEST external install mechanism
  → No language needed — just shared perception
  → = Proto-foundation of ALL cultural transmission
  → Group anchor (Logic-Planning §2.2) BEGINS here:
    "Cùng nhìn → cùng biết → cùng nói về nó" = shared chunk
```

### §1.2 — Mechanism 2: Imitation / Motor Mirror

```
DEFINITION:
  Observe action of other person → own motor system fires SIMILARLY
  → Motor chunk compiles FROM OBSERVATION, not from own execution

MECHANISM (neural):
  ① Observer sees person B perform action (visual cortex → IT → action recognition)
  ② Motor cortex PARTIALLY activates (same areas as if observer did action)
     → Mirror neuron system: premotor + parietal  🟢 Rizzolatti & Craighero 2004
  ③ If observer IMITATES (actually performs action):
     → Motor execution + visual feedback + proprioceptive feedback = MULTI-MODAL compile
     → Chunk compiles FASTER than observation alone (more modalities)
  ④ If observer only watches:
     → Motor chunk compiles PARTIALLY (weaker, visual + partial motor only)
     → = "Xem YouTube nấu ăn ≠ nấu ăn" (observation < practice)

DEVELOPMENTAL TIMELINE:
  Neonatal:      facial imitation (Meltzoff & Moore 1977 — debated, recent challenges)
  ~4-6 months:   action imitation (reaching, grasping patterns)
  ~9-12 months:  deferred imitation (see action → reproduce LATER)
                 = Evidence: chunk COMPILED from observation, STORED, retrieved later
  ~18-24 months: imitative learning (understand GOAL of action, not just motor copy)
                 = Tomasello 1999: "emulation" vs "imitation" distinction
  Adult:         skill learning via demonstration, apprenticeship, video tutorial

🟢 Rizzolatti & Craighero 2004 — mirror neurons review
🟡 Meltzoff & Moore 1977 — neonatal imitation (challenged by Oostenbroek 2016)
🟢 Tomasello 1999 — cultural learning via imitation

LINK TO F1 + FEELING:
  → F1 05-Motor-Output-Arc: motor chunk compile via own execution (bottom-up)
  → F3 Imitation: motor chunk compile via OBSERVATION (external input)
  → CÙNG compile mechanism (multi-modal), KHÁC input source
  → = F3 extends F1 motor arc into social domain
  → Imitation = WHY apprenticeship works (watch master → partial compile → practice → full compile)
```

### §1.3 — Mechanism 3: Social Referencing

```
DEFINITION:
  Observe OTHER PERSON'S EMOTIONAL RESPONSE to object/event
  → Learn VALENCE from others (good/bad/dangerous/safe)
  → Install valence tag on own chunk WITHOUT own direct experience

MECHANISM (neural):
  ① Baby encounters novel object X (e.g., crawling toward cliff — visual cliff)
  ② Baby looks at caregiver's face (social referencing check)
  ③ Caregiver shows FEAR face → baby perceives emotional expression
     → Mirror neurons + amygdala fire → baby's own fear system PARTIALLY activates
     → Chunk for X now tagged with THREAT valence (from caregiver, not from own fall)
  OR
  ③' Caregiver shows ENCOURAGING face → baby perceives positive expression
     → Baby proceeds → X tagged with SAFE/POSITIVE valence

DEVELOPMENTAL TIMELINE:
  ~10-12 months: reliable social referencing (Sorce et al. 1985 visual cliff)
  ~12-18 months: referencing extends to novel objects, strangers, situations
  ~2-3 years:    verbal social referencing ("đừng chạm vào, nóng đấy!")
  Adult:         "Mọi người nói restaurant đó dở" → valence install WITHOUT trying

🟢 Sorce et al. 1985 — visual cliff social referencing
🟢 Feinman 1992 — social referencing review

⭐ CRITICAL IMPLICATION FOR FRAMEWORK:
  → Social referencing = mechanism for EXTERNAL VALENCE TRANSMISSION (H7 support)
  → Valence CAN BE INSTALLED WITHOUT OWN EXPERIENCE
  → = Phần lớn "sợ" và "thích" ở adult = SOCIALLY TRANSMITTED, not personally experienced
  → = "Sợ rắn" nhưng chưa bao giờ gặp rắn = social referencing from childhood
  → = Prejudice: "người X nguy hiểm" = social referencing valence install

  → NT7 connection: caregiver's emotional response DURING social referencing
    is a compile-context factor. Caregiver overreacting (Level 3-4) installs
    DISPROPORTIONATE threat valence on chunks that might be neutral/novelty.

  → ⭐ REPAIR: Social referencing works BOTH directions:
    → Install fear: caregiver shows fear → child learns fear
    → Install safety: caregiver shows calm → child learns safe
    → Adult: new social group shows calm toward previously-feared object → valence UPDATE
    → = One mechanism for therapy: therapist = new social reference source
```

### §1.4 — Mechanism 4: Label Installation (Direction B)

```
DEFINITION:
  Receive LABEL from external source (spoken word, written text, sign, symbol)
  → Label ATTACHES to existing or forming chunk
  → Chunk becomes retrievable, chainable, communicable

MECHANISM (neural):
  ① External source produces label: "đây là CON CHÓ" (verbal)
  ② Baby perceives: auditory pattern (phonemes → word pattern "con chó")
  ③ SIMULTANEOUSLY: baby perceives object (visual: furry, 4 legs, barking)
  ④ MULTI-MODAL BINDING: auditory label pattern + visual object pattern
     → 4 concurrent binding mechanisms fire (F1 NT3)
     → Label-object link COMPILES as chunk
  ⑤ After sufficient repetition: hear "con chó" alone → visual cortex fires
     → Label = RETRIEVAL PATH to chunk (F1 NT6 verdict)

  = Direction B (Logic-Planning.md §7.4):
    Label-first → practice/experience → body compile chunk → feel match label

DEVELOPMENTAL TIMELINE:
  ~4.5 months:   name recognition (Mandel Jusczyk & Pisoni 1995)
  ~6-9 months:   word comprehension begins (Bergelson & Swingley 2012)
  ~12-18 months: first words produced (H11: production lags ~6-12mo)
  ~18-24 months: vocabulary spurt (~50 words → rapid acceleration)
  Adult:         read new term → understand → use = Direction B pipeline

VERBAL AS HANDLE (F1 NT6 recap — không re-derive):
  → Label ≠ chunk content. Label = HANDLE for retrieval + compression
  → Removing label ≠ removing chunk. Chunk persists, just harder to retrieve
  → 5 handle systems exist: verbal, gestural, visual, olfactory, contextual
  → Verbal = most versatile (broadest coverage + shareability)

🟢 Mandel Jusczyk & Pisoni 1995 — infant name recognition
🟢 Bergelson & Swingley 2012 — 6-month word comprehension
🟢 Tomasello 2003 — usage-based language acquisition

⭐ LABEL INSTALL = MOST STUDIED EXTERNAL MECHANISM:
  → Entire field of language acquisition = studying Direction B
  → Logic-Planning.md §2: 3-tier anchor system shows label progression
    Individual → Group → Global
  → F3 adds: the MECHANISM of label attachment (multi-modal binding)
  → Logic-Planning already covers: WHAT labels look like, how they're USED
  → F3 covers: HOW labels GET ATTACHED to chunks (compile mechanism)
```

### §1.5 — Mechanism 5: Cultural Transmission

```
DEFINITION:
  Chunks propagate ACROSS GENERATIONS without requiring each individual
  to compile from raw experience. Knowledge ACCUMULATES.

MECHANISM:
  ① Generation N compiles chunks from experience (F1 + F4)
  ② Generation N EXTERNALIZES: tells stories, creates rituals, writes books
     → Chunks encoded in EXTERNAL FORMAT (verbal, visual, behavioral)
  ③ Generation N+1 RECEIVES:
     → Listen to stories → label install (Mechanism 4)
     → Observe rituals → imitation (Mechanism 2)
     → Read books → label install via visual-symbolic format
  ④ Generation N+1 now has chunks WITHOUT own raw experience
  ⑤ Generation N+1 extends: adds OWN chunks → externalizes → N+2 receives
     → = CUMULATIVE CULTURE (Tomasello 1999)

RATCHET EFFECT:
  → Each generation ADDS to existing knowledge (not start from zero)
  → = "Standing on shoulders of giants"
  → Enabled by: writing (long-term external storage) + institutions (organized transmission)
  → = Why human chunk library GROWS exponentially while individual brain capacity is FIXED
  🟢 Tomasello 1999 — cultural learning + ratchet effect

3 CHANNELS OF CULTURAL TRANSMISSION:

  ① VERTICAL (parent → child):
     → Strongest: exposure from birth, high repetition, emotional bond
     → NT7 applies: compile context during vertical transmission = critical
     → Language L1 = primarily vertical transmission

  ② HORIZONTAL (peer → peer):
     → Moderate: playground, social group, team, community
     → Group anchors (Logic-Planning §2.2) = primarily horizontal
     → Jargon, slang, team culture = horizontal transmission

  ③ OBLIQUE (non-parent adult → child, institution → individual):
     → Education system, media, religious institutions, internet
     → Global anchors (Logic-Planning §2.3) = primarily oblique
     → Formal knowledge = oblique transmission via books/school

⭐ CULTURAL TRANSMISSION = WHY HUMAN COGNITION EXPONENTIALLY EXCEEDS
   INDIVIDUAL BRAIN CAPACITY:
  → Individual brain: ~3-5 WM slots, finite lifetime
  → Cultural library: millions of books, thousands of years
  → Individual ACCESSES cultural library via Direction B (label install)
  → AI ERA: AI = instant access to ENTIRE cultural library
  → = "AI hỗ trợ Logic cực mạnh" (Logic-Planning.md §6)
```

## §3 — Non-Verbal External Anchors

### §3.1 — Ritual as external chunk install

```
RITUAL = behavioral sequence SHARED BY GROUP, repeated, anchoring chunks

Mechanism:
  ① Group performs sequence together (co-attention + imitation)
  ② Repetition → chunks compile (mechanism 1 from Chunk.md §2)
  ③ Emotional peak often present (ceremony = heightened state)
  ④ Multi-modal: visual + auditory + motor + social + often somatic
  → DEEP compile: ritual chunks are among MOST persistent

Examples:
  • Infant: peek-a-boo (earliest ritual — F1 E30)
  • Child: birthday party song, bedtime routine
  • Adult: wedding ceremony, funeral, religious service, team standup
  • Cultural: harvest festival, new year, coming-of-age
  • Modern: team standup, karaoke night, national anthem at games

⭐ RITUAL = NON-VERBAL EQUIVALENT OF LANGUAGE FOR EXTERNAL INSTALL:
  → Language: verbal label → chunk retrieval path (symbolic)
  → Ritual: behavioral sequence → chunk retrieval path (somatic + social)
  → BOTH serve as anchors, DIFFERENT modality bias:
    Language: auditory-symbolic dominant
    Ritual: motor-social-emotional dominant
  → Ritual PREDATES language (proto-cultural, possibly >500,000 years)
    🟡 Rappaport 1999, Alcorta & Sosis 2005

MODERN RITUAL DECLINE:
  → Fewer shared rituals → fewer non-verbal external anchors
  → Compensated by: social media (new ritual forms), gaming communities, fandoms
  → "Meaning crisis" (Vervaeke): partially = loss of ritual anchor infrastructure
  → Framework view: meaning ≠ mystical — meaning = anchored chunks with deep compile
  → Restore meaning = restore deep-compile experiences (not necessarily traditional ritual)
```

### §3.2 — Shared sensory events

```
SHARED SENSORY = 2+ people experience SAME sensory event → shared chunks

Examples:
  • Sấm: cả nhóm nghe → shared chunk "tiếng sấm hôm đó"
  • Hoàng hôn: cùng nhìn → shared chunk "cảnh đẹp hôm đó"
  • Mùi: vào quán phở → shared olfactory chunk (mùi phở = Group anchor)
  • Tai nạn: cùng chứng kiến → shared emotional chunk (flashbulb)
  • Concert: cùng nghe → shared auditory-social-emotional chunk

Mechanism: Co-attention (§1.1) + multi-modal binding (F1 NT3)
  → No language needed initially
  → Language LATER: "nhớ hôm đó không?" = verbal retrieval of shared chunk
  → = Group anchor formation (Logic-Planning §2.2)

⭐ SHARED SENSORY IS THE OLDEST EXTERNAL MECHANISM:
  → Before language, before ritual: just BEING TOGETHER + experiencing SAME event
  → "Hai người cùng thấy hổ → chỉ tay → chạy" (Logic-Planning §2.2)
  → Modern: watching movie together, attending concert, road trip
  → = Foundation of GROUP BONDING (shared chunks = connection substrate)
```

### §3.3 — Gesture + Image + Artifact

```
GESTURE: body movement as external chunk anchor

  → Iconic gesture (miming) = imitation aid (mechanism 2)

  → Icon/emoji: compressed visual label (Global anchor — Logic-Planning §2.3)

ARTIFACT (TOOL, OBJECT):
  → Object AS anchor: "cái ly đó" = object triggers memory chunk
  → Tool AS extended mind: paper extends WM (Clark & Chalmers 1998)
```

```
H4 SKETCH (formal verdict in 01-External-Synthesis.md):
  → Abstract: pattern-extraction from concrete experience + label compress
    → Modality: cross-modal / somatic dominant (feel pattern across examples)
  → Metaphysical: cultural label + PFC top-down simulation + ritual depth
    → Modality: visual-heavy top-down (imagine unseen) + social-emotional
  → BOTH require external label system for MANIPULATION
  → DIFFERENCE: abstract has extractable pattern, metaphysical does not
  → H4 = modality encoding difference → 🟡 SUPPORTED with refinement:
    Not just modality difference, but also GROUNDING difference
```

```
     → "Chó nguy hiểm" = verbal label TELLS you valence (without experience)
     → "Pizza ngon lắm" = verbal label installs positive expectation
     → = Language carries VALENCE INFORMATION inside labels
     → Classifiers encode valence: "thằng X" (negative) vs "anh X" (neutral/positive)
       (Vietnamese classifier system — 01-Natural-Language-Architecture.md §2)

  ③ CULTURAL NORM VALENCE:
     → Culture defines: "stealing = bad", "sharing = good"
     → Installed via: vertical transmission (parent teaches) + oblique (school)
     → = Moral valence chunks = CULTURALLY COMPILED, not individually discovered
     → Cross-cultural variation: same behavior, different valence in different cultures

  ④ MEDIA / NARRATIVE VALENCE:
     → Story villain → character chunk tagged NEGATIVE
     → Story hero → character chunk tagged POSITIVE
     → News: "country X dangerous" → country-chunk tagged NEGATIVE
     → = Modern mass external valence transmission (scale unprecedented)

H7 SKETCH (formal verdict in 01-External-Synthesis.md):
  → Valence = chunk × schema interaction
  → SAME chunk CAN HAVE different valence in different schema contexts:
    "Dao" in cooking schema → positive (useful tool)
    "Dao" in threat schema → negative (weapon)
    "Dao" in surgery schema → positive (saving life)
  → External world provides INITIAL valence tag (via 4 sources above)
  → But schema context MODULATES valence at retrieval time
  → H7 = valence not chunk-intrinsic → 🟡 SUPPORTED

  ⭐ IMPLICATION:
  → Valence CAN BE CHANGED by changing schema context
  → = One mechanism for therapy: re-contextualize threatening chunk
  → = One mechanism for prejudice reduction: new schema for "other group"
  → = NT7 3 update paths applicable: re-associate / novelty hijack / therapy
```

## §7 — Education = Batch Direction B Installation

```
⭐ EDUCATION = ORGANIZED, SYSTEMATIC BATCH INSTALLATION OF CHUNKS via Direction B

MECHANISM:
  ① Curriculum SELECTS which chunks to install (domain knowledge)
  ② Teacher PRESENTS labels + examples (label installation — §1.4)
  ③ Student PRACTICES (repetition compile — Chunk.md §2 mechanism 1)
  ④ Student EXPERIENCES (multi-modal compile — mechanism 3)
  ⑤ Assessment VERIFIES: does student's chunk match intended chunk?
  ⑥ Repeat across years → LARGE chunk library installed

WHY EDUCATION WORKS (framework view):
  → Education = batch Direction B (label-first → experience → compile)
  → Individual WITHOUT education: must compile ALL chunks from raw experience (slow)
  → Individual WITH education: receives LIBRARY of pre-labeled chunks (fast)
  → = Cumulative culture (§1.5) applied SYSTEMATICALLY

WHY EDUCATION SOMETIMES FAILS:
  → ① Label without experience = SHALLOW chunk (verbal only, 1 modality)
     → Student recites "F = ma" but never FELT force → 1-modality chunk → weak
     → FIX: lab, hands-on, project-based learning = add modalities
  → ② Threat compile context (NT7) = chunk tagged with aversion
     → Teacher shaming student for wrong answer → "math = shame" chunk
     → FIX: Level 1 compile context (curiosity + warmth) → "math = interesting"
  → ③ No Imagine-Final = no motivation (no feeling target)
     → Student doesn't FEEL why this knowledge matters → no PFC engagement
     → FIX: connect to student's existing Imagine-Final (relevance)
  → ④ Mismatch with existing chunks = confusion (not error)
     → New label conflicts with existing chunk → ACC conflict → "sai sai"
     → FIX: explicit reframe (acknowledge old chunk, install new one)

AI ERA EDUCATION:
  → AI = 24/7 Direction B installer (explain, demonstrate, assess)
  → AI CANNOT provide: Direction A experience (feel-first, body-based)
  → Optimal: AI handles Direction B (install labels) + human provides Direction A (feel)
  → = "Con người cần FEEL đúng → AI sẽ giúp PLAN đúng" applied to education
```
