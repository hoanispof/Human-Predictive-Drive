### §2.1 — Core hypothesis questions (to Language/02-Grammar-Architecture)

**Q6.1** — Grammar categories = chunk category taxonomy?
- Moderate claim: grammar provides ONE scaffolding (many possible). Each language = distinct scaffolding. Chunks underneath universal.

**Q6.2** — Broca handles grammatical hierarchy → chunk-chain scaffolding?
- Modality §2.0 committed: Broca = sequence processing (including but not limited to grammar)
- Drill formalize: Broca sequential processing = neural substrate for chunk-chain scaffolding. Grammar = special case (verbal chunk chain). Music, action, ritual = other cases.

### §2.2 — Vietnamese classifier case study (to 01-Vocabulary-Taxonomy)

**Q6.3** — Vietnamese classifier system forces animacy + shape + social classification?

- **con** → animate (con người, con chó, con kiến, con sông)
- **cái** → inanimate tool/object (cái bàn, cái bút)
- **chiếc** → individual-of-pair (chiếc giày, chiếc xe)
- **quyển/cuốn** → bound object (quyển sách)
- **ngài/vị** → high-status person
- **tên/thằng** → negative-valence person
- **ông/bà/anh/chị** → relational-age kin-mapped

**Claim**: Vietnamese force classification per noun invocation → scaffolding animacy + shape + social hierarchy as chunk categories.

**Counter**: English "the" for all. Still functions. → Classifier không strictly necessary, chỉ là 1 taxonomy option.

**Resolution**: Each language forces attention to DIFFERENT distinctions. English forces tense, Turkish forces evidentiality, Vietnamese forces animacy + shape + social. Each language = biased scaffolding along different dimensions.

**Sub-question 6.3.1** — Vietnamese speaker VTC binary Object-Agent (Object-Agent.md §2) reinforced by classifier use? Prediction: Vietnamese Object-Agent chunk taxonomy faster/more explicit than English.

### §2.4 — Construction Grammar (to 02-Grammar-Architecture)

**Q6.6** — Construction Grammar (Goldberg 1995, 2006) — constructions = compiled chunks at higher level?

**Construction examples**:
- "The X-er the Y-er" (càng X càng Y)
- Ditransitive "give [someone] [something]"
- Resultative "She painted the wall blue"

**Integration**:
- Construction = schema of chunks (not single chunk)
- Learning language = learning construction inventory
- Compatible với Schema.md §2 two-path compile

**Q6.7** — Construction frequency → chunk strength?
- Tomasello usage-based: high-frequency constructions compile first, then generalized
- Matches Chunk.md §2 repetition compile

### §2.5 — Universal Grammar reconciliation (to 02-Grammar-Architecture)

**Q6.8** — Chomsky UG reconcile với usage-based?

**Framework position**:
- Hardware for sequence processing (Broca) = innate (partial UG claim)
- Specific grammar rules = learned scaffolding (Construction Grammar claim)
- "Poverty of stimulus" weakened by corpus studies (Tomasello)
- → **Moderate**: Neural substrate innate, specific grammars learned

### §2.6 — Framework embedding (to 02-Grammar-Architecture + 06-Synthesis)

**Q6.9** — Grammar-scaffolding extends Modality §2.0:
- Grammar = one structured way to implement LABEL + GHÉP + SEQUENCE
- Grammar categories → chunk label taxonomy (NOT ontology)
- Grammar scaffolds chunk-CHAIN formation, not chunk CREATION
- Different languages = different attentional biases
- **Chunk exists pre-grammar** (trẻ 0-2 tuổi) — grammar enhancer, not substrate

**Q6.10** — Gaps where grammar doesn't reach (to 05-Language-Gaps-Evolution):

- Each specialized anchor system optimizes different dimensions

## §4 — NEW: H12 Language Evolution Driver (to 05-Language-Gaps-Evolution)

**H12 tentative** (N+4 prep insight from user):
> "Adults feel 'gần đúng / hơi mâu thuẫn / có vẻ có khái niệm gì đó mới' but find no existing word. This drives language evolution — new words emerge every few generations to fill these gaps."

**Mechanism**:
1. Internal novel chunk forms (F4 territory)
2. PFC searches verbal library → no fitting word
3. Chunk persists via alternative anchors (felt sense, description, metaphor)
4. If chunk benefits community + re-emerges across speakers → cultural pressure to coin
5. New word emerges (often via metaphor, borrowing, or compound)
6. Language updates

**Research support**:
- Kirby iterated learning (2002) — grammar evolution via cultural transmission
- Meillet 1912 — grammaticalization paths
- Sapir-Whorf weak version — language reflects conceptual needs
- Lakoff & Johnson metaphor theory — new concepts via metaphor extension
- Recent examples: "gaslight" (verb sense ~2016), "doomscroll" (2020), "cancel culture"

**Framework integration**:
- H12 connects F3 Language to F4 Internal (post-verbal bypass mechanism)
- Language is not static artifact — it's continually refined by internal processing gap 2
- Bidirectional: language scaffolds chunks (F3 installs) AND chunks evolve language (F4 feedback)

**05-Language-Gaps-Evolution.md** will formalize this with falsifiable predictions.
