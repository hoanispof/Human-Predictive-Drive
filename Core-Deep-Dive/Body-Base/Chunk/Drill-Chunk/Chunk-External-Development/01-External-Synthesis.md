### §1.1 — H3: Grammar = Most Optimized External Anchor

```
HYPOTHESIS (from plan §4):
  "Grammar-scaffolding partial-embedded in Modality §2.0 — drill formalizes
   as most-optimized external anchor system"

🟡 VERDICT: SUPPORTED WITH REFINEMENT

  ✅ Grammar IS the most optimized external anchor for GENERAL-PURPOSE cognition
  ✅ ~100,000 years cultural-evolutionary refinement → unique combination of:
     Speed + Capacity + Shareability + Recursion + Abstraction + Inner Speech
  ✅ Grammar categories (noun/verb/adj/classifier) = chunk category TAXONOMY
     → Vietnamese classifiers force animacy + shape + social classification
     → English tense forces temporal marking
     → Turkish evidentiality forces source marking
     → Each language = biased scaffolding along different dimensions

  REFINEMENT (from N+5 pivot + F3 analysis):
  ⚠️ Grammar is NOT the ONLY chain enabler:
     → Math notation enables precise quantitative chains
     → Music notation enables temporal-affective chains
     → Programming enables executable chains
     → Ritual enables somatic-social chains
     → Visual/diagram enables spatial chains
  ⚠️ Grammar is NOT superior in ALL dimensions:
     → Precision: Math > Language
     → Executability: Code > Language
     → Emotional depth: Music/Ritual > Language
     → Spatial: Visual > Language
  
  → H3 refined: Grammar = MOST VERSATILE (broadest coverage),
    not most precise or most powerful in any single dimension.
  → "Jack of all trades, master of none — but better than master of one"
  → This explains WHY language is the DEFAULT communication system:
    Breadth > specialization for GENERAL cognition

CONFIDENCE: 🟡 (framework synthesis, consistent with evidence, novel framing)

EVIDENCE BASE:
  🟢 Goldberg 1995 — Construction Grammar
  🟢 Tomasello 2003 — Usage-based acquisition
  🟢 Slobin 1996 — Thinking for Speaking
  🟡 Fedorenko 2011 — Language vs MD networks (language ≠ thought)
  🟢 Logic-Planning.md §3 — 5 format comparison

FALSIFIABLE PREDICTIONS:
  P-H3-1: Removing grammar scaffolding (agrammatic aphasia) should
           reduce chain-forming ability but NOT eliminate it (other
           anchor systems compensate partially).
  P-H3-2: Bilingual speakers should show scaffold-dependent differences
           in chunk access patterns for each language's biased dimensions.
  P-H3-3: Constructed languages (Esperanto, Lojban) with different
           grammatical biases should produce different chunk access patterns.
```

### §1.2 — H4: Abstract vs Metaphysical = Modality Encoding Difference

```
HYPOTHESIS (from plan §4):
  "Abstract vs Metaphysical = modality encoding difference
   (visual structural vs somatic field)"

🟡 VERDICT: PARTIALLY SUPPORTED — REFINED to include GROUNDING difference

  User's original:
    Metaphysical = visual-heavy (imagine unseen)
    Abstract = somatic/cross-modal (feel unseeable)

  F3 analysis (00-External-Mechanism.md §5):
    ✅ Modality difference EXISTS:
       Abstract: cross-modal / somatic dominant (pattern EXTRACTED from experience)
       Metaphysical: visual-heavy top-down (unseen, IMAGINED via PFC simulation)
    
    REFINEMENT — GROUNDING difference is MORE fundamental:
       Abstract: experience-GROUNDED (pattern extracted from real cases)
         → "Fairness" = extracted from many fair/unfair experiences
         → TESTABLE against new experiences (can verify "is this fair?")
       Metaphysical: culturally-GROUNDED (no pattern from experience)
         → "Soul" = received from culture, simulated by PFC
         → NOT testable against experience (cannot verify "does soul exist?")
    
    → H4 refined: Abstract vs Metaphysical differ in BOTH:
      (a) Modality bias (somatic-cross vs visual-top-down) — user's insight
      (b) Grounding source (experience-extractable vs culture-only) — F3 addition
      → (b) is more fundamental than (a)
      → (a) FOLLOWS from (b): experience → somatic feel; culture → imagine visual

CONFIDENCE: 🟡 (plausible, consistent, needs more evidence)

EVIDENCE BASE:
  🟡 Barsalou 1999 — Perceptual symbols (abstract concepts grounded in sensory)
  🟡 Lakoff & Johnson 1980 — Metaphors we live by (abstract via concrete metaphor)
  🟢 Kosslyn 2005 — Mental imagery in visual cortex (metaphysical = top-down)
  🟡 Boyer 2001 — Religion explained (minimally counterintuitive concepts)

FALSIFIABLE PREDICTIONS:
  P-H4-1: Abstract concept processing should show more distributed cortical
           activation (cross-modal). Metaphysical should show more visual +
           default mode network activation.
  P-H4-2: Cultures without specific metaphysical concepts should lack those
           chunks entirely (not form them independently).
  P-H4-3: Abstract concepts should transfer cross-culturally more easily than
           metaphysical concepts (grounding in shared experience vs culture-specific).
```

### §1.3 — H7: Valence = Chunk × Schema Interaction

```
HYPOTHESIS (from plan §4):
  "Valence emerges from chunk × schema interaction (not chunk-intrinsic).
   Same chunk + different schema → different valence"

🟡 VERDICT: SUPPORTED

  ✅ Valence is NOT chunk-intrinsic:
     "Knife" in cooking schema → positive (useful)
     "Knife" in threat schema → negative (weapon)
     "Knife" in surgery schema → positive (saving life)
     → Same chunk, 3 different valences

  ✅ External valence installation via 4 sources (00-External-Mechanism §6):
     Social referencing, verbal labels, cultural norms, media/narrative
     → All install valence AS SCHEMA CONTEXT, not as chunk property

  ✅ Valence CAN BE CHANGED by changing schema context:
     → Therapy: re-contextualize threatening chunk → valence shifts
     → Exposure: repeated safe encounter → schema overrides old valence
     → NT7 3 update paths: re-associate / novelty hijack / therapy

  ✅ Valence.md multi-channel model CONSISTENT:
     → L0 threat, L1 interoceptive, novelty (evaluative), meaning/schema (evaluative)...
     → Valence = INTEGRATED signal from multiple channels (not 1 source)
     → Schema context ACTIVATES different channels → different valence

  ⚠️ NUANCE: Some valence IS partially chunk-intrinsic:
     → Pain chunks have NEGATIVE valence almost universally (body-level)
     → Sweetness chunks have POSITIVE valence (evolutionary reward)
     → = L0-L1 valence = chunk-intrinsic (body hardware, Direct-State)
     → = Evaluative valence = schema-dependent (context modulates)
     → H7 holds for evaluative valence. L0-L1 valence has intrinsic component.

CONFIDENCE: 🟡 (strong framework synthesis, consistent with Valence.md)

EVIDENCE BASE:
  🟢 Damasio 1994 — Somatic markers (context-dependent valuation)
  🟢 Barrett 2017 — Constructed emotion (valence = constructed, not intrinsic)
  🟡 Valence.md §1-§6 — Framework multi-channel model

FALSIFIABLE PREDICTIONS:
  P-H7-1: Same chunk primed in different schema contexts should show different
           amygdala / insula activation patterns.
  P-H7-2: Therapeutic re-contextualization should measurably reduce threat-schema
           amygdala response while leaving neutral-schema response unchanged.
  P-H7-3: Cross-cultural valence differences should correlate with schema
           differences (not chunk differences) for culturally-variable items.
```

### §1.4 — H12: Language Evolution Driven by Gap 2

```
HYPOTHESIS (from plan §4 + N+4 prep):
  "Post-verbal bypass drives language evolution. Adults feel concept without
   existing word → cultural pressure to coin new word → language evolves."

🟡 VERDICT: SUPPORTED (preliminary — full treatment in F4)

  ✅ MECHANISM plausible:
     Internal chunk forms (F4 Direction A) → PFC searches label library
     → No fitting label → chunk persists via felt sense / description / metaphor
     → If shared need → community coins new word
     → Language vocabulary GROWS

  ✅ EVIDENCE from recent word emergence:
     "Gaslight" (verb sense ~2016): manipulation experience EXISTED before word
     "Doomscroll" (~2020): behavior EXISTED before word
     "Cancel culture" (~2019): social phenomenon EXISTED before label
     "Ghosting" (~2015): dating behavior EXISTED before word
     "Phở" (entered English): food experience EXISTED, English lacked label
     → Each case: chunk EXISTED → word COINED to share chunk

  ✅ CONSISTENT with framework:
     → Somatic-Articulation Loop: individual → articulate → label
     → Gap 2 = WHEN articulation FAILS (no existing label fits)
     → Gap 2 pressure → new word → language evolves
     → = "Language GROWS because internal processing OUTPACES vocabulary"

  ⚠️ LIMITATION: hard to test DIRECTLY at population level
     → Language evolution is slow and multi-causal
     → New words also emerge from: borrowing, technology, marketing, slang
     → Gap 2 is ONE driver, not the only driver

  → Full formalization with predictions in F4 05-Insight-Tacit-Upgrade.md

CONFIDENCE: 🟡 (plausible, internally consistent, hard to test directly)

EVIDENCE BASE:
  🟢 Kirby 2002 — Iterated learning (language evolution mechanism)
  🟢 Lakoff & Johnson 1980 — Metaphor as word creation strategy
  🟡 Meillet 1912 — Grammaticalization paths
  🟡 Recent neologisms as case evidence (not systematic)
```
