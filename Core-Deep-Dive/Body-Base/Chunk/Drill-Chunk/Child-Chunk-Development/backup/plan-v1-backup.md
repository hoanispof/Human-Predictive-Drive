## §0 — North Star

**Goal**: Formalize **how biological substrate (hardware + modality) compiles raw sensory input into chunks**, using pre-verbal infant development (0-24mo) as cleanest test case where chunks form WITHOUT language contamination.

**Why this folder matters**:
- Foundation for all other folders — chunks must form before language can install on top (F3) or before internal processing can transform them (F4)
- Pre-verbal period is clean room for testing "chunk genesis mechanism" — no verbal confound
- Developmental data already exists in `Feel-Example-Draft.md` §1-§3 (~30 cases) — goldmine not yet integrated into Chunk-Analysis
- Answers user's N+4 prep academic question: receptive-productive gap, infant name recognition age, minimum experience for first chunks

**Folder scope**: ~4800-6400 lines across 6 drill files + plan. 1 session target (N+4), possibly split into N+4a + N+4b if batch too large.

### §2.1 — Thread 7 (primary): Chunk genesis flow

**User's formulation** (verbatim, preserved from Feeling session directions):
> "Sẽ phải tạo 1 file về quá trình tiến triển chunk — toàn bộ chuỗi quá trình mà chunk sinh ra rồi được label rồi được củng cố (tự nhắc lại, hoặc giao tiếp với mọi người), rồi ghép nối tạo chunk mới, đặt anchor label mới"

**Sub-questions (primary home F1)**:

- **Q7.1** — Raw sensation → proto-chunk → chunk → labeled chunk: threshold mỗi stage là gì?
  - Framework stance: Chunk.md §2 mentions 4 compile mechanisms but doesn't define thresholds
  - Drill: operationalize threshold with LTP mechanism + hippocampal replay + sleep consolidation
  - Data: Feel-Example §1 crude distress E1 → §2 differentiated cry E11 shows threshold emergence

- **Q7.2** — "Proto-chunk" có phải khái niệm thực, hay chunk là discrete?
  - Gradient view: proto-chunk = stable pattern not yet compiled to meta-chunk
  - Discrete view: chunk either exists or not, no intermediate
  - Framework implication: H1 (chunk substrate) matters differently under each view
  - Drill verdict needed

- **Q7.3** — 4 compile mechanisms (repetition / emotional peak / multi-modal / sleep) — integrate vào genesis flow?
  - All 4 mechanisms work for infants
  - Question: does temporal order matter (multi-modal before repetition, or vice versa)?
  - Sleep critical for infant — they sleep 16+ hours/day → sleep compile is primary mode 0-3mo

- **Q7.4** — First-time chunk formation: minimum experience cần thiết?
  - Single exposure chunk (flashbulb memory) vs repeated exposure chunk (skill)
  - Threshold likely depends on valence strength + modality count + contextual uniqueness
  - Data: E2 pain cry (single strong exposure) vs E11 cry discrimination (accumulated experience)

- **Q7.5** — Consolidation: self-repetition vs communication verification — khác biệt?
  - Pre-verbal: no communication verification available → only self-repetition + caregiver contingent response
  - Post-verbal: communication verification becomes available as consolidation pathway
  - Bridge to F3 (external social verification)

- **Q7.6** — Combination (chunk A + B → C): mechanism cụ thể?
  - Framework: compile meta-chunk via repeated co-activation
  - Additional: binding via modality overlap, temporal co-occurrence, emotional co-tag
  - Developmental: when does combination emerge? Infant face-voice binding at 2-4mo

- **Q7.7** — Labeling: verbal auto-attach hay manual PFC attach?
  - Pre-verbal: no verbal label possible → chunk exists without verbal label
  - Verbal emergence 12-24mo: first labels attach via caregiver naming + imitation
  - Bridge to F3 Language

- **Q7.8** — Re-labeling: chunk đã có label, khi nào rename/extend?
  - Post-verbal question mostly, defer detail to F3/F4
  - Note F1 position: relabeling NOT relevant for pre-verbal substrate drill

- **Q7.9** — Implicit vs explicit chunk — thật sự khác hay chỉ khác labeling stage?
  - Likely labeling stage difference, not ontological difference
  - Implicit chunk = compiled pattern without verbal label
  - Explicit chunk = same compiled pattern + verbal label
  - Drill validate

- **Q7.10** — Developmental perspective: trẻ build chunk thế nào khi chưa có verbal?
  - Primary F1 question — entire sub-folder partially answers this
  - Data: Feel-Example §1-§2 cases
