### §2.2 — Signal profile analysis

```
WHAT MAKES FEELING CLEAR vs VAGUE?

3 variables determine position on gradient:

  VARIABLE 1 — SIGNAL COUNT:
    Few (1-2) signals → CLEAR (easy for PFC to isolate source)
    Many (5-20+) signals → VAGUE (PFC cannot distinguish individual sources)
    → = Cocktail party effect but for body signals
    → 1 voice in quiet room = CLEAR
    → 20 voices in party = VAGUE (overall "murmur" but can't hear words)

  VARIABLE 2 — SIGNAL STRENGTH:
    Strong signals → CLEAR (PFC threshold easily exceeded)
    Weak signals → VAGUE (individual signals below PFC threshold)
    → But: many weak signals CAN converge → exceed threshold → become clearer
    → = H5 core claim

  VARIABLE 3 — LABEL AVAILABILITY:
    Label available → CLEARER (verbal tag = Feel-Location anchoring)
    No label → VAGUER (stuck at L4, cannot articulate)
    → NT6 verbal-as-handle: label = retrieval path, not content
    → Having label does NOT change the signal — changes ACCESS to signal
    → = Somatic-Articulation Loop: transition implicit → explicit via label


⭐ THE 3 VARIABLES INTERACT:

  Clear feeling = FEW signals × STRONG × LABELED
    → "My foot hurts" = 1 signal × strong nociception × labeled "pain"
    
  Vague intuition = MANY signals × WEAK × UNLABELED
    → "Something feels off" = 12 signals × each weak × no label fits

  Middle ground = SOME signals × MODERATE × PARTIALLY LABELED
    → "I feel a bit uneasy" = 3-4 signals × moderate × partial label "uneasy"
```

## §3 — H5 Test: "Vague" = Multi-Weak-Signal Convergence

### §3.1 — H5 claim formalization

```
H5 CLAIM:
  "Vague" (vague feeling / intuition / gut feeling) = 
  MULTI-WEAK-SIGNAL CONVERGENCE:
  
  Many chunks fire WEAKLY at the same time → individual signals NOT STRONG ENOUGH
  for PFC to detect → but AGGREGATE of many weak signals → coherent body signal
  → PFC detects AS "something is here" → VAGUE but DIRECTIONALLY correct.

  = "Each piece of evidence is weak → but 10 weak pieces pointing the same
    direction = strong"
  = Statistical convergence at body level
```

### §3.2 — Mechanism drill

```
🟡 FRAMEWORK SYNTHESIS — multi-weak-signal convergence mechanism:

STEP 1 — CHUNK LIBRARY ACTIVATION:
  PFC hold current situation/question in WM
  → Spreading activation (🟢 Collins & Loftus 1975) propagates
  → Many chunks fire at DIFFERENT LEVELS:
     - Some chunks fire STRONGLY (high overlap with current situation)
     - Some chunks fire WEAKLY (partial overlap)
     - Some chunks fire VERY WEAKLY (distant association)
  
  → Clear feeling case: 1-3 strong fires → PFC CLEARLY SEES source
  → Vague intuition case: 10-20 weak fires → PFC CANNOT see each source

STEP 2 — BODY INTEGRATION:
  Body does NOT process each chunk individually.
  Body INTEGRATES all signals into 1 state:
  
  → Think of body as ANALOG COMPUTER:
    → Digital (PFC): process individual items sequentially
    → Analog (body): process ALL inputs simultaneously → 1 integrated output
    → 🟡 Framework analogy, consistent with distributed processing
  
  → Many weak signals all pointing SAME DIRECTION:
    → Body state SHIFTS in that direction
    → = "Many small ripples in the same direction → 1 large wave"
    → PFC detect the aggregate wave, NOT individual ripples
    
  → Many weak signals pointing DIFFERENT DIRECTIONS:
    → Body state NOISY → no clear direction
    → PFC detect: "confused, mixed" → not useful
    → = "Many ripples in opposite directions → chaos"

STEP 3 — PFC DETECTION:
  PFC has THRESHOLD for detection:
  → Signal above threshold → "something is here" (detected, vague)
  → Signal well above threshold → "clearly" (detected, clear)
  → Signal below threshold → "nothing" (not detected)
  
  → Converging weak signals CAN cross threshold → vague detection
  → = "Something feels off" = body signals crossed threshold but barely
  → = PFC knows SOMETHING but not WHAT specifically

STEP 4 — DIRECTIONAL ACCURACY:
  ⭐ KEY H5 PREDICTION:
  If signals converge (same direction) → vague BUT DIRECTIONALLY CORRECT
  → Expert: many chunks fire in domain → converge → ACCURATE gut feeling
  → Beginner: few chunks → random directions → INACCURATE gut feeling
  
  → VAGUENESS ≠ WRONGNESS
  → Vagueness = low resolution. Accuracy = correctness of direction.
  → Expert intuition: LOW resolution + HIGH accuracy
  → Beginner "intuition": LOW resolution + LOW accuracy
  → Clear feeling: HIGH resolution + HIGH accuracy (or clear error)
```

### §3.3 — Expert vs Beginner intuition

```

  Firefighter case:
    → Experienced firefighter enters burning building
    → "Something feels wrong" → orders evacuation
    → Floor collapses 30 seconds later
    → Post-hoc: "I didn't analyze — I just KNEW"
    
  Mechanism (framework interpretation):
    → 20 years firefighting = THOUSANDS of fire-related chunks compiled
    → Current situation activates MANY chunks weakly
    → Most chunks point toward "unusual pattern" (floor too hot, fire too quiet)
    → NO SINGLE SIGNAL strong enough for articulation
    → BUT: aggregate signal crosses threshold → "something wrong"
    → Direction: DANGER (accurate, because chunks are relevant)
    → = Multi-weak-signal convergence with HIGH accuracy
    → = H5 in action

🟢 ESTABLISHED — Kahneman & Klein 2009 agreement conditions:

  When is intuition reliable?
    ① Environment must be REGULAR (patterns exist to learn)
    ② Person must have PROLONGED PRACTICE (chunks compiled)
    → = Framework translation: reliable intuition REQUIRES deep chunk library
    → = H5 prediction confirmed by meta-analysis consensus

  When is intuition unreliable?
    → In IRREGULAR environments (stock market, political prediction)
    → With SHORT experience
    → = Framework: insufficient chunks → random convergence → unreliable
    → = H5 predicts EXACTLY this failure mode


🟡 FRAMEWORK APPLICATION — Expert vs Beginner:

  ┌──────────────────────────────────────────────────────────────┐
  │                 EXPERT                    BEGINNER           │
  ├──────────────────────────────────────────────────────────────┤
  │ Chunks:       Thousands (deep, domain)    Few (shallow)     │

  ⚠️ DANGER ZONE — INTERMEDIATE:
    → Some chunks, not enough for reliable convergence
```

## §6 — Practical Implications

### §6.1 — When to trust feeling vs verify

```
⭐ FRAMEWORK DECISION GUIDE:

TRUST FEELING WHEN:
  → Clear body signal (position ①-② on gradient)
  → Domain expertise deep (many compiled chunks)
  → Environment regular (patterns exist)
  → Signal consistent over time (not just momentary)
  → No known trauma contamination in this domain
  → = Klein/Kahneman conditions MET

VERIFY WHEN:
  → Vague signal (position ④-⑥ on gradient)
  → Domain expertise shallow
  → Environment irregular or unfamiliar
  → Signal inconsistent or fluctuating
  → Known trauma in related domain
  → High stakes (cost of error is high)

BOTH (FEEL + VERIFY):
  → Most real-world decisions fall HERE
  → Feel first → verify with logic/data/AI → integrate
  → = Direction A + Direction B combined
  → = "Humans need to FEEL correctly → AI will help PLAN correctly"

  ⚠️ NEVER:
  → Dismiss vague feelings as "just feelings" — they may be valid convergence
  → Trust vague feelings blindly — they may be noise or bias
  → Skip body check in favor of pure logic — body has information PFC doesn't
  → Skip logic check in favor of pure feeling — body can be wrong
```

### §6.2 — Feeling literacy training implications

```
⭐ GRADIENT → TRAINING PROGRAM:

STAGE 1 — DETECT (position ⑥→⑤):
  → Learn to NOTICE vague body signals instead of dismissing
  → Practice: "Wait. Do I feel ANYTHING about this?" before deciding
  → = Gendlin Step 1-2 (clearing space + noticing)
  → Tool: body scan, pause before response

STAGE 2 — DIRECTION (position ⑤→④):
  → Learn to identify DIRECTION of vague signal: positive? negative?
  → Practice: "This vague feeling — is it pulling me toward or away?"
  → = Gendlin Step 2-3 (felt sense + handle)
  → Tool: journaling, dialogue with trusted person or AI

STAGE 3 — ARTICULATION (position ④→③):
  → Learn to find WORDS for intuitive signals
  → Practice: Somatic-Articulation Loop (try labels → body confirm → refine)
  → = Gendlin Step 3-4 (handle + resonating)
  → Tool: writing, talking, Focusing practice

STAGE 4 — DISCRIMINATION (position ③→②):
  → Learn to distinguish TYPES of body signals
  → Practice: "Is this genuine concern or trauma response?"
  → = Requires understanding own chunk library + trauma history
  → Tool: therapy, self-reflection, feedback from trusted others

STAGE 5 — INTEGRATION (position ②→①):
  → Signal becomes CLEAR + LABELED + TRUSTED
  → Practice: accumulated expertise + feeling literacy + verified track record
  → = Expert-level intuition (Klein 1998 level)
  → Tool: deliberate practice + reflection + real-world feedback loops

  → Stages 1-5 are YEARS of practice, not a workshop
  → BUT: Stage 1-2 alone already improves decision quality significantly
  → = "Something feels off" → NOTICE instead of dismiss → better outcomes
```
