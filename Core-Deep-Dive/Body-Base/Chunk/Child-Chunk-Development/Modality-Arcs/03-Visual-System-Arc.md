# 03 — Visual System Developmental Arc

> **File 03 of F1 Child-Chunk-Development sub-folder.** First of six body-input arc files (visual, auditory, motor, interoceptive, social, verbal). This file traces visual system chunk compile mechanism from birth to ~24 months via **Connect → Read → Control phases**, classifying observable events with **PFC-Inference Ladder** (plan §0.7) and applying the **event-inference methodology rule** from [01-PFC-Hardware-Reframe.md](01-PFC-Hardware-Reframe.md) §6.

---

## §0 — Abstract

🟢🟡 **Claim**: Visual chunks compile over 0-24 months in a gradient trajectory beginning from a sparse prenatal starting state (CONSPEC face template innate, no learned object/person chunks) and reaching rich multi-modal-bound identity chunks by ~18-24 months. The trajectory is **not a PFC maturation story** — PFC hardware is online from birth per [01-PFC-Hardware-Reframe.md](01-PFC-Hardware-Reframe.md) §3 — but a **chunks accumulation story** with observable events at progressively higher PFC-Inference Ladder levels.

The visual system is analyzed along three phases:

- **Connect** (§2): hardware readiness for visual input — retina, V1-V5, optic tract, acuity and color development.
- **Read** (§3): input detection chunks — face templates, direct gaze detection, mother face recognition, object file, object permanence, feature integration.
- **Control** (§4): output chunks — smooth pursuit, visually guided reaching, gaze following, joint attention, object tracking through occlusion.

Events analyzed with PFC-Inference Ladder:

| Event | Age | Ladder | Research |
|---|---|---|---|
| Face preference | Birth (2-5d) | L0-L2 | Fantz 1961, Morton & Johnson 1991 |
| Direct gaze preference | 2-5 days | L2 🟡 | Farroni et al. 2002 |
| Social smile (multi-modal) | 6-8 weeks | L2-L3 🟢 | Feel-Example E12 |
| Mother face recognition | 5 months | L2-L3 🟢 | Feel-Example E15 |
| Gaze following | 6 months | L3 🟢 | Feel-Example E14, Tomasello |
| Object permanence VoE | 4-6 months | L2 🟢 | Baillargeon 1987, 1991 |
| Joint attention (triadic) | 9-12 months | L4 🟢 | Feel-Example E18, Tomasello 1995 |
| Mirror self-recognition | 18-24 months | L4 🟡 | Amsterdam 1972 |

**Two nút thắt addressed**:

1. **Nút Thắt 1** — object file emergence (§3.8 + §5) provides one of the visual system's test cases for proto-chunk gradient vs discrete verdict.
2. **Nút Thắt 3** — multi-modal binding visual component (§6) covers the visual side of the social smile E12 / mother recognition E15 / joint attention E18 binding mechanism. Full verdict deferred to [07-Social-Recognition-Arc.md](07-Social-Recognition-Arc.md) §6.

**What this file does NOT claim** (per §6 methodology rule):

- ❌ "PFC matures at X age for visual task Y"
- ❌ "Visual cortex wakes up at X age"
- ❌ Direct PFC measurement claims for any visual event

**What this file DOES claim**:

- ✅ Event X is observable at age Y per research citation
- ✅ Event X plausibly requires chunks A, B, C to have compiled (framework inference 🟡)
- ✅ Ladder classification per event with plausibility qualifier
- ✅ Baseline state at t=0 from [02-Womb-to-Birth-Baseline.md](02-Womb-to-Birth-Baseline.md) §9.1 as starting point

### §1.2 — Why visual is instructive for chunk substrate

Three reasons visual arc is particularly instructive for F1's central question (how chunks compile):

1. **Gradient emergence visible across many events**: face preference → direct gaze → mother recognition → joint attention is a clear ascending chunks-accumulation trajectory across 0-12 months. Each step has observable markers. Few body-input systems have this many discrete milestones in such a short window.

2. **Multi-modal binding rich**: visual chunks bind with auditory (social smile E12 = face + voice), with interoceptive (mother recognition E15 = face + comfort history), with motor (reaching, gaze following). Visual is the **central hub for early multi-modal binding** — touching Nút Thắt 3 at multiple angles.

3. **Object file emergence tests proto-chunk gradient**: the "is it one chunk or gradient?" question (Nút Thắt 1) has a clean test case in object file formation between 4-6 months. Behaviorally, babies show intermediate states (partial tracking, partial recognition, partial permanence) — not discrete "object chunk present / absent" switches.

## §2 — Connect Phase (Hardware Readiness)

> "Connect" = hardware capable of transducing visual input into neural signal. This phase establishes **what the visual system can physically do at birth** — per the baseline state in 02 §9.1. Not a chunks question — a substrate question.

### §2.1 — Visual cortex active from prenatal

🟢 **Evidence**: V1 (primary visual cortex) synaptogenesis begins in utero and reaches newborn levels by term birth. V1 is among the **earliest cortical areas to reach adult-like synaptic density** — peaking ~3-4 months postnatal per Huttenlocher 1979.

V2-V5 (extrastriate visual areas including MT for motion, V4 for color, fusiform for face) develop in overlapping windows:
- V1 (primary): peak synaptogenesis ~3-4 mo
- V2-V4: peak ~6-8 mo
- Fusiform face area (FFA): functionally responsive from ~2-4 mo per fMRI studies on awake infants (Deen et al. 2017)

🟡 **Framework note**: Just as PFC was mis-read as "offline in infants" per the old view rejected in 01, visual cortex readiness is likewise not a maturation story — the hardware is online early, and what changes across development is the **content** (chunks compiled in the hardware), not the hardware being "switched on".

### §2.2 — Retinal development

🟢 Retinal photoreceptors differentiate by term birth. Rod-dominant periphery functional from day 1. Cone-rich macula (foveal region) continues maturing postnatally:
- Cone density at fovea: increases ~5x from birth to 4 months (Hendrickson & Yuodelis 1984)
- Foveal cone outer segment length: doubles 0-6mo
- Adult macular structure reached ~4 years

**Implication for F1**: Newborn visual acuity is limited not by cortex but by retinal hardware — specifically, the immature fovea. This is a **substrate limitation** on what visual chunks can compile. Fine-detail face features (micro-expressions, individual eye shape) below newborn acuity threshold → cannot be in chunks yet.

### §2.3 — Acuity trajectory

🟢 **Empirical trajectory** (Dobson & Teller 1978, Mayer & Dobson 1982, Teller 1997):

| Age | Snellen acuity equivalent |
|---|---|
| Birth | ~20/400 (can see high-contrast ~30 cm — "arm's length when nursing") |
| 1 month | ~20/300 |
| 3 months | ~20/100 |
| 6 months | ~20/40 |
| 12 months | ~20/30 (near adult) |
| 3-5 years | ~20/20 (adult baseline) |

**Critical framework point**: at birth, ~20/400 acuity is still **sufficient for face-scale patterns at feeding distance** (~30 cm = optimal visual focal distance for newborn). This is why CONSPEC face preference (Fantz 1961) works from day 1 despite "limited" acuity — the acuity is adequate for the task at the task's distance.

### §2.4 — Color vision development

🟢 Cone trichromacy present from birth but weak:
- Red-green discrimination: demonstrable from ~1-2 months (Teller et al. 1978)
- Full trichromatic color vision: established ~2-3 months
- Color category formation (blue vs green category boundary): ~4-6 months (Franklin & Davies 2004)

🟡 **Framework note**: Color is a modality dimension that enters chunks **gradually** over first 4-6 months. Before full color, early face chunks are luminance-pattern-based (CONSPEC is luminance-pattern detection, not color-based). This matters for the object file test in §3.8 — 4-month-old's ability to discriminate "two balls different colors" requires color chunks compiled, not just object file chunks.

### §2.5 — Depth perception (stereopsis)

🟢 Stereoscopic depth perception **emerges suddenly** around 3-4 months (Fox et al. 1980, Held et al. 1980). Before this, depth cues are monocular (occlusion, motion parallax). After this, random-dot stereograms perceivable.

**Why sudden**: stereopsis requires binocular vergence (eyes converging to same point) + cortical cell tuning for binocular disparity. Both develop in parallel, and when they cross threshold together, stereopsis emerges in a step change. This is **one of the few developmental events in F1 that IS a discrete threshold** — contrast with face recognition and object chunks which are gradient.

### §2.6 — Connect phase summary for visual system

🟢🟡 **What is available at birth for visual chunks to compile on**:

| Component | State at birth | Developmental trajectory |
|---|---|---|
| V1-V2 synapses | Near adult density | Peaks 3-4mo, prunes through childhood |
| Retinal rods | Functional | Stable |
| Retinal cones (fovea) | Sparse, immature | Densifies over 0-4mo |
| Acuity | ~20/400 | Improves to near-adult by 12mo |
| Color | Weak trichromacy | Full by 2-3mo |
| Stereopsis | Absent | Emerges 3-4mo |
| Fusiform face area | Functionally responsive | Specializes 2-8mo |

**Bottleneck**: Not cortex, not hardware — the bottleneck for visual chunks is **input quality + exposure history**, not substrate readiness. This is the same pattern the PFC reframe identified for PFC: hardware ready, content missing. Visual chunks are fast to compile because visual input is rich and frequent from day 1.

## §3 — Read Phase (Input Detection Chunks)

> "Read" = chunks that detect, categorize, and identify visual input. Each event in this section follows the §6 methodology template: observable event → chunks plausibly required → ladder level → plausibility. Events are arranged roughly chronologically.

### §3.1 — Face preference from birth (CONSPEC)

🟢 **Event**: Newborns (2-5 days old) look longer at face-like patterns (two dots above one dot in oval) than at equally complex non-face patterns (scrambled or inverted).

🟢 **Research**: 
- Fantz 1961 — original demonstration using schematic face vs control
- Morton & Johnson 1991 — CONSPEC/CONLERN two-system model (CONSPEC = innate subcortical face-template detector, CONLERN = cortical learning-based face processing emerging 2-3mo)
- Johnson et al. 1991 — newborn preference for face-like pattern across many replication studies

🟡 **Chunks required for event**:
- **Innate face template** (CONSPEC) — NOT a compiled chunk. Hardwired pattern detector operating below cortical level (Morton & Johnson propose subcortical route via superior colliculus + pulvinar).
- No learned face chunks required at this age (no experience yet).

**PFC-Inference Ladder**: **L0** with caveat — the preference itself is reflex-like (innate template drives attention), but the *attention* (longer looking) is subject to modulation by CONLERN pattern-match over first weeks. Classification: L0 at birth → transitions to L2 (pattern-match fire) as CONLERN engages by ~2-3 months.

**🟢 Plausibility**: High for event, framework-standard for chunks claim.

**Implication for F1**: The framework distinguishes **innate templates** (CONSPEC) from **compiled chunks** (CONLERN-built face chunks). Both are "face detection capability" from a behavioral standpoint, but architecturally different. The reframe from [01-PFC-Hardware-Reframe.md](01-PFC-Hardware-Reframe.md) §5 (hardware vs content 7-component distinction) applies: CONSPEC = hardware innate detector, CONLERN = content compiled from exposure.

### §3.2 — Direct gaze preference 2-5 days (Farroni 2002) ⭐

🟢 **Event**: Newborns (mean age 2-5 days) look longer at photographs of faces with **direct eye gaze** (eyes looking at the viewer) than identical faces with **averted eye gaze** (eyes looking 20° sideways).

🟢 **Research**: 
- Farroni et al. 2002 — original study, 17 newborns, direct gaze = longer looking time
- Replicated multiple times (Batki et al. 2000, Farroni et al. 2004)
- Extended: 4-month-olds show enhanced neural response (ERP N290) to direct gaze faces

🟡 **Chunks required**:
- CONSPEC face template firing (already covered §3.1)
- **Direct-gaze subcomponent detector** — either innate (plausible given 2-5 day timing) or compiled in utero from... what exactly? Womb is dark, fetus sees no eyes.
- **Framework claim**: 🟡 Most plausible account is **innate direct-gaze detector tied to CONSPEC** — the template contains not just "two dots above one" but specifically "two dots pointed at viewer". Evolutionary argument: direct gaze is the strongest social signal, worth pre-wiring.

**PFC-Inference Ladder**: **L2** 🟡. This is one of the earliest observable L2 events — pattern-match fires (looking time modulated by stimulus category), no action involvement required. Critical: 2-5 days is earlier than any plausible post-birth learning window for this specific pattern. If CONSPEC has a direct-gaze subcomponent, L2 behavior is possible from birth.

**🟢 Plausibility**: Event high confidence (replicated). Chunks-required claim 🟡 (innate vs compiled distinction speculative).

**Why this event is load-bearing for reframe**: Direct gaze preference at 2-5 days is a **falsification challenge for the old "PFC offline" view**. If PFC were non-functional, an L2 pattern-match event could still happen (since L2 can run on subcortical + early cortex), BUT the old view typically conflated PFC maturity with **any** "higher cognitive" function. Farroni 2002 shows discrimination based on subtle visual features at 2-5 days — too early for any learning-based account, too nuanced for pure brainstem reflex. The event-inference methodology handles this cleanly: observable event (🟢) → chunks required (innate template 🟡) → L2 classification → no PFC timeline claim needed.

### §3.3 — High-contrast pattern detection

🟢 **Event**: Newborns visually track high-contrast moving patterns (dark-light edges, checkerboards) preferentially over uniform or low-contrast fields.

🟢 **Research**: Fantz 1963, Salapatek 1975, Banks & Salapatek 1981 — well-replicated.

🟡 **Chunks required**: None learned. Retinal ganglion cell tuning + V1 orientation-selective columns are innately organized for edge detection. This is substrate, not chunks.

**PFC-Inference Ladder**: **L0** (reflex-level attention).

**Implication**: High-contrast preference is a **scaffolding mechanism for chunks compilation** — by preferentially looking at high-contrast regions, newborns oversample the most information-rich parts of the visual field (e.g., the hairline-face boundary, the eyes, mother's mouth while speaking). This accelerates chunks compile by concentrating exposure on high-information regions.

### §3.4 — Mother face recognition at 5 months (E15) ⭐ multi-modal

🟢 **Event**: Baby ~5 months old, upon seeing mother enter room after brief absence, shows: eye brightening, kicking, smile, vocalization, reach toward. Reaction is **mother-specific** — distinguishable from reaction to stranger, sibling, father, etc. (Feel-Example E15, [../../Body-Base/Feeling/Feel-Example-Draft.md](../../Body-Base/Feeling/Feel-Example-Draft.md#e15) line 927.)

🟢 **Research**:
- Bushnell et al. 1989 — 4-day-olds prefer mother's face over stranger's (earliest detection)
- Pascalis et al. 1995 — confirmed 4-day-olds can discriminate mother face from similar stranger
- de Haan & Nelson 1999 — 6-month-olds show ERP (N290) differentiation for mother face
- Feel-Example E15 framing as **identity chunk** (multi-modal bound)

🟡 **Chunks required**:
- **Mother-specific visual face chunk** — compiled from ~months of visual exposure. Note: 4-day discrimination (Bushnell 1989) suggests compilation begins immediately after birth, but the *richer* response at 5 months (E15) suggests the chunk is multi-modal and has accumulated additional dimensions (voice binding, smell binding, comfort history binding).
- **Binding chunks** — linking visual face pattern to voice chunks (from prenatal, see 02 §4.1), to smell chunks (compiled postnatally), to warmth/comfort chunks (compiled during feeding).
- **Associative chunk**: "mother visual → positive state expected" (anchored to prior comfort experience).

**PFC-Inference Ladder**: **L2-L3** 🟢🟡. L2 because pattern-match fires (visual recognition → face lights up = recognition marker, not action). Borders L3 because motor component emerges (reach, kick, smile output) — the smile is multi-modal output, arguably a crude coordinated response. Feel-Example frames this as the **first specific-identity chunk** (vs generic face detection).

**🟢 Plausibility**: Event well-documented. Framework claim about identity chunk vs generic face: Feel-Example § justifies the distinction via the "eye brightening + kicking + reach" pattern which is **mother-specific**, not generic-face-triggered.

**Why this event is load-bearing**: Mother face recognition shows the **compiled chunk** story in action. CONSPEC handled birth-day face detection (§3.1). By 5 months, mother face is a richly-bound identity chunk integrating:
- Visual face pattern (fusiform face area)
- Voice pattern (from prenatal auditory chunks)
- Smell chunks (from nursing)
- Comfort/warmth association (from feeding and soothing)
- Relative stability (mother mostly present during early weeks)

This is where **Nút Thắt 3 (multi-modal binding mechanism)** takes a visible concrete form. Full mechanism drill deferred to [07-Social-Recognition-Arc.md](07-Social-Recognition-Arc.md) §6, but the visual component is primary anchoring modality.

### §3.5 — Feature integration emergence (Treisman 1980)

🟡 **Event**: Between ~3-6 months, babies begin integrating separate visual features (color, shape, motion) into unified object representations. Before this, features seem to be detected but not bound together into single "objects".

🟡 **Research**: 
- Treisman 1980 Feature Integration Theory (adult work)
- Mareschal & Johnson 2003 — object file formation infant studies
- Tremoulet et al. 2000 — feature binding in 4-month-olds using habituation paradigm

🟡 **Chunks required**:
- Feature detection chunks (color chunks, shape chunks, motion chunks) — compiled over first months
- **Binding chunks** — associating co-occurring features at same spatial location
- No PFC-level "integration" needed — binding plausibly happens in early visual cortex + parietal regions

**PFC-Inference Ladder**: Difficult to classify cleanly — feature integration is a **substrate capability**, not a discrete observable event per se. Habituation studies show gradient emergence over 3-6 months. Best classified as **compile mechanism phase** rather than single event.

**🟡 Plausibility**: Event itself 🟡 (inferred from habituation paradigms, not directly observable). Chunks-required claim 🟡.

**Framework note**: Feature integration is a **substrate for object file chunks** (§3.8 next). It's a necessary precondition. The gradient emergence is evidence for Nút Thắt 1 (proto-chunk gradient view) — integration doesn't snap on at a discrete moment, it strengthens over weeks.

### §3.6 — Color discrimination chunks

🟢 **Event**: 4-month-olds discriminate colors into adult-like categories (blue vs green boundary similar to adults despite continuous physical color space).

🟢 **Research**: Bornstein, Kessen & Weiskopf 1976 — classic paper demonstrating infant color categorical perception. Franklin & Davies 2004 — replications + developmental trajectory.

🟡 **Chunks required**: Color chunks compiled from visual exposure + possibly categorical boundaries influenced by innate cone tuning + possibly language-influenced (though this is controversial — Franklin et al. 2008 show pre-linguistic infants already categorize).

**PFC-Inference Ladder**: **L2** 🟡. Pattern-match on color category fires (habituation recovers when color crosses category boundary). No action involvement. Classic L2 event.

### §3.7 — Object file chunks emerging (4-6 months)

🟢 **Event**: 4-6 month olds show "object file" behavior in looking-time experiments — when an object is hidden then replaced with a physically different object (different color, shape), infants look longer as if surprised, but if same object, look less. This indicates **object-level representation** (not just retinal image).

🟢 **Research**:
- Kahneman, Treisman & Gibbs 1992 — object file theory (adult work)
- Xu & Carey 1996 — infant object individuation studies
- Baillargeon 1995 — object tracking through occlusion

🟡 **Chunks required**:
- Individual object features chunks (color, shape, size)
- **Object file chunk** — binding of features into a single "this object" representation that persists through momentary occlusion
- Spatiotemporal continuity chunks (object doesn't teleport, doesn't change identity at random)

**PFC-Inference Ladder**: **L2** 🟡🟢. Pattern-match on mismatch event (object changes to different object) → longer looking time is the L2 marker. No action involvement required.

### §3.8 — Object permanence (Baillargeon VoE 4-6mo → stable 8-9mo) ⭐ Nút Thắt 1

🟢 **Event**: Using **Violation of Expectation (VoE)** paradigm, 4-6 month-olds show longer looking times when physical impossibilities involving occluded objects are shown (e.g., object seems to pass through solid barrier, object appears to occupy two locations).

🟢 **Research**:
- Baillargeon 1987 — "drawbridge" study with 4-month-olds
- Baillargeon & DeVos 1991 — object permanence evidence from 3.5 months
- Spelke et al. 1992 — core knowledge object principles
- Hespos & Baillargeon 2001 — 2.5-month-olds showing object-related surprise

Contrast with classic **Piaget A-not-B task** (Piaget 1954) where 8-9-month-olds fail to reach for hidden object → Piaget concluded "no object permanence". Baillargeon's looking-time method reveals permanence knowledge 4+ months earlier than action-based Piaget method. This is the classic **knowing vs acting asymmetry** — mirrors the receptive-productive gap (Nút Thắt 5) at a different level.

🟡 **Chunks required**:
- Object file chunks (§3.7)
- **Persistence chunks** — "object exists when not visible"
- **Trajectory chunks** — "object moves continuously in space"
- **Solidity chunks** — "solid object does not pass through solid object"
- All these need to be compiled with stability high enough to generate prediction + mismatch detection

**PFC-Inference Ladder**: **L2** 🟢. This is a textbook L2 event: prediction chunk fires, reality mismatches, looking time increases (PFC pattern-match attention signal). No action involvement. The L2→L3 transition for object permanence is the **reaching side** — 8-9 month olds can *act* on the permanence knowledge (reach into location where object was hidden) which adds L3 action chunks on top of the L2 pattern-match.

**⭐ Nút Thắt 1 test case**: Object permanence chunks do not emerge "all at once". Baillargeon's work shows **gradient emergence**:
- 2.5 months: basic surprise at impossible trajectory (Hespos & Baillargeon 2001)
- 3.5-4 months: solid barrier impossibility (Baillargeon 1987)
- 4-6 months: more complex permanence scenarios
- 8-9 months: action-based permanence (reaching into hidden location)
- 12+ months: more elaborate permanence reasoning

**F1 drill verdict contribution**: Object permanence chunks compile in **gradient** — not discrete. Each sub-chunk (solidity, trajectory, continuity, identity) compiles semi-independently with different maturation rates. This supports **Nút Thắt 1 gradient view** (over discrete view). Full Nút Thắt 1 verdict in [04-Auditory-System-Arc.md](04-Auditory-System-Arc.md) §6 (phoneme discrimination provides even cleaner gradient evidence) + [10-F1-Synthesis.md](10-F1-Synthesis.md).

### §3.9 — Face categorization and individuation 6-9 months

🟢 **Event**: Between 6-9 months, babies show perceptual narrowing for faces — they can discriminate own-species and own-race faces better than other-species or other-race faces, a trajectory opposite to the early unconstrained face detection.

🟢 **Research**: Pascalis, de Haan & Nelson 2002 — 6-month-olds discriminate monkey faces, 9-month-olds do not (perceptual narrowing parallel to phoneme narrowing in 04 §3.5). Kelly et al. 2007 — other-race effect emerges similarly.

🟡 **Chunks required**: Own-race face chunks heavily compiled from exposure → categorical boundaries sharpen for familiar types → unfamiliar types fall outside sharpening boundary.

**PFC-Inference Ladder**: **L2** 🟢. Pattern-match strength differs by familiarity.

**Cross-reference to Nút Thắt 1**: This event shows the **same gradient pattern** as auditory phoneme narrowing (04 §3.5). The parallelism is suggestive — chunks compilation via exposure follows similar shape across sensory modalities. Strong indirect support for gradient view.

## §4 — Control Phase (Output Chunks)

> "Control" = chunks producing visual motor output (eye movements, head turns, coordinated gaze) or guiding body motor output using visual input. These are action-involved events, higher on the ladder.

### §4.1 — Smooth pursuit eye movements (6-8 weeks)

🟢 **Event**: Infants track moving targets smoothly (without saccadic jumps) from ~6-8 weeks. Before this, tracking is saccadic (jerky stepwise).

🟢 **Research**: Aslin 1981 review, Shea & Aslin 1990, Rosander & von Hofsten 2000.

🟡 **Chunks required**: Motor prediction chunks (where target will be next) + smooth-motor output chunks (oculomotor). Both compile over first weeks.

**PFC-Inference Ladder**: **L1-L2** 🟡. Response is reactive (L1) but with prediction component (L2 emerging). Transitional.

### §4.2 — Head turn toward visual target (3-4 months)

🟢 **Event**: Infants turn head toward visually interesting targets by 3-4 months (integrated neck motor + visual attention chunk).

🟡 **Chunks required**: Attention chunks (object is salient) + head motor chunks (neck control matured — see [05-Motor-Output-Arc.md](05-Motor-Output-Arc.md) §3.2) + binding chunks (visual salience → head motor response).

**PFC-Inference Ladder**: **L2-L3** 🟡. Coordinated output, pattern-match on salient visual triggers head motor.

### §4.3 — Visually guided reaching (4-5 months)

🟢 **Event**: Infants reach for visually perceived objects with reasonable accuracy at 4-5 months. Before this, reaching is random swipes; visual target not yet integrated with arm motor.

🟢 **Research**: von Hofsten 1979, 1984 — classic infant reaching studies.

🟡 **Chunks required**:
- Visual object location chunks
- Arm motor chunks (see [05-Motor-Output-Arc.md](05-Motor-Output-Arc.md) §2.4)
- **Visuomotor binding chunks** — linking visual target coordinates to motor coordinates (coordinate transformation)

**PFC-Inference Ladder**: **L3** 🟡. Partial coordinated action, not yet precise, improving over months.

**Multi-modal binding note**: Visually guided reaching is the **first visuomotor chunk** binding visual input with motor output. Critical milestone for Nút Thắt 3 in motor dimension. Full treatment in [05-Motor-Output-Arc.md](05-Motor-Output-Arc.md) §2.4.

### §4.4 — Gaze following (6 months, E14) ⭐

🟢 **Event**: Baby ~6 months old. Mother looks at baby, then turns head to look at a specific object (a door, a toy). Baby follows mother's gaze and looks at the same object. ([Feel-Example E14](../../Body-Base/Feeling/Feel-Example-Draft.md#e14) line 876.)

🟢 **Research**: Butterworth 1991, Corkum & Moore 1995, Tomasello 1995, Carpenter et al. 1998. Consistent cross-cultural finding: 6-9 months is the earliest reliable gaze following age.

🟡 **Chunks required**:
- "Mother gaze direction" chunk (direction of face turn)
- Spatial mapping chunk (from direction of face to location in 3D space)
- Output chunk: "redirect own gaze to follow" (saccadic gaze shift executed)
- Possibly an **expectation chunk**: "if mother looks there, something interesting there" (curiosity-driven)

**PFC-Inference Ladder**: **L3** 🟢. Coordinated output (gaze shift), pattern-match on mother's direction, action response. Not yet full triadic (no checking back to mother — that's E18).

**Why 6 months and not earlier**: Chunks for spatial direction mapping need several months of co-occurrence experience — looking at mother's face + objects in mother's gaze direction + both visible simultaneously → binding. Also requires motor chunks for controlled gaze shift.

### §4.5 — Object tracking through occlusion (8-9 months)

🟢 **Event**: 8-9 month-olds actively search for objects hidden behind occluders (Piaget A-not-B task; also various search paradigms).

🟡 **Chunks required**: Object permanence chunks (from §3.8) + search motor chunks + "where the object was last seen" spatial chunk + inhibition chunks (preventing perseverative reaching to old location).

**PFC-Inference Ladder**: **L3-L4** 🟢. Coordinated action using permanence knowledge. The **A-not-B error** itself (reaching to wrong previous location) is a beautiful framework case:
- Baby knows object is hidden (L2 permanence)
- Baby reaches for it (L3 action)
- Baby reaches to PREVIOUS correct location, not current one (perseverative error)
- Error shows inhibition chunks not yet stable

Piaget read this as "no permanence" (wrong — behavioral act-failure). Modern view: permanence chunks present (VoE from 4 months), but **inhibition chunks to override prepotent motor response not yet strong enough**. This is a classic chunks-state vs "PFC maturity" question — and chunks-state wins the explanation.

### §4.6 — Joint attention triadic (9-12 months, E18) ⭐

🟢 **Event**: Baby ~10 months. Mother points to an object ("nhìn kìa, con chó!"). Baby looks along mother's pointing direction, sees object, then **looks back at mother**, then back at object. The gaze triangle me-mother-object is **triadic joint attention** — the defining moment. ([Feel-Example E18](../../Body-Base/Feeling/Feel-Example-Draft.md#e18) line 1092.)

🟢 **Research**: Butterworth 1991, Tomasello 1995, Tomasello 1999 "Cultural Origins of Human Cognition", Carpenter, Nagell & Tomasello 1998 — longitudinal study showing typical emergence 9-12 months.

🟡 **Chunks required** (this is the most demanding visual-social chunk set in first year):
- **Point gesture interpretation chunk** — understanding that pointing hand indicates direction, not the hand itself as target. Wittgenstein "dog sees the finger, not what it points at" observation — human infants **learn** to see past the finger.
- **Gaze direction + point direction coordination chunk** — mother's eyes + mother's hand often point same direction, creating strong co-occurrence for binding.
- **Shared attention schema chunk** — "mother and I can look at the same object simultaneously and mother notices I'm looking too"
- **Reference chunk** — "this object is the topic of mother's attention"
- **Checking-back chunk** — the me→object→mother→object gaze loop is what distinguishes joint attention from mere gaze following

**PFC-Inference Ladder**: **L4** 🟢. Full coordinated execution: multiple chunks (gaze, point, shared reference, checking-back) integrated into reliable, goal-directed action.

**Why joint attention is the capstone of first-year visual arc**: It integrates:
- Visual face/gaze chunks (§3.2, §4.4)
- Point-gesture chunks (see [05-Motor-Output-Arc.md](05-Motor-Output-Arc.md) §2.8)
- Social reference chunks (see [07-Social-Recognition-Arc.md](07-Social-Recognition-Arc.md))
- Early theory-of-mind pre-requisites (mother has a "view" of an object)
- Communicative intent schema (both agents orient to same thing with purpose)

This is a **major multi-modal multi-body-input integration event** — visual + social + motor + pre-verbal + attention. Feel-Example E18 frames it as the foundation for subsequent language explosion (18-24 months). Tomasello's thesis: without joint attention, no language. Joint attention is the **referential scaffold** on which word-object mapping can occur.

### §4.7 — Mirror self-recognition (18-24 months, Amsterdam 1972)

🟢 **Event**: 18-24 month-olds, when a red dot is secretly applied to their forehead and they look in a mirror, reach to touch **their own forehead** (not the mirror). Before ~18 months, they reach for the mirror image as if it were another baby.

🟢 **Research**: Amsterdam 1972 — classic "rouge test". Gallup 1970 — chimpanzee parallel. Lewis & Brooks-Gunn 1979 — developmental trajectory.

🟡 **Chunks required**:
- Body schema chunks (self-proprioception over body)
- Visual self-chunk — "this face reflected is me"
- **Self-other distinction chunk at visual level** — critical prerequisite
- Cross-modal binding: visual face ↔ proprioceptive body

**PFC-Inference Ladder**: **L4** 🟡. Coordinated execution, involving self-other distinction. The chunks-required claim is 🟡 — alternative interpretations exist (some argue it's not evidence of "self-concept" in adult sense, just body schema).

**Framework implication**: Mirror self-recognition is one of F1's latest events (approaching end of 24-month window). It demonstrates that **visual self chunks** (seeing oneself as a distinct entity) compile **after** years of multi-modal experience and are among the most complex visual chunks in the first two years. This is the substrate on which Agent/Self-Pattern-Match (H9, drilled N+2) subsequently operates at higher levels.

## §6 — Multi-Modal Binding (Visual Component) — Nút Thắt 3 contribution

> Visual chunks bind with chunks from other modalities to form multi-modal chunks. This section covers the **visual side** of multi-modal binding. Full Nút Thắt 3 verdict in [07-Social-Recognition-Arc.md](07-Social-Recognition-Arc.md) §6 which uses social smile (E12) as cleanest multi-modal exemplar.

### §6.1 — Visual + auditory binding (social smile E12)

🟢 **Event**: Social smile emerges at 6-8 weeks. Baby smiles in response to face + voice combination (ineffective to face alone for many weeks, ineffective to voice alone). ([Feel-Example E12](../../Body-Base/Feeling/Feel-Example-Draft.md#e12) line 758.)

🟡 **Framework claim**: Social smile is the **first documented large-scale multi-modal binding event** in F1 arc — visual (face) + auditory (voice) + interoceptive (calm state baseline) → unified smile output chunk.

**Critical evidence from blind infants**: Blind infants also develop social smile at approximately the same age (~8 weeks) but using **auditory + tactile** cues instead of visual. This confirms that the multi-modal binding chunk is not "face-driven hardware" but **learned binding of whatever input channels are available**. (Feel-Example E12 §Notes line 775-779.)

**Why 6-8 weeks specifically**: Not a PFC maturation timeline. It's the time required for:
- Visual face chunks to compile (V1 CONSPEC → V2 individuation by 2-4 weeks)
- Auditory voice chunks strengthened (already seeded prenatal per [02](02-Womb-to-Birth-Baseline.md) §4.1)
- Co-occurrence exposure (face + voice happen together during feeding, many repetitions)
- Binding chunk compile (repetition-driven compile mechanism)

🟡 **Ladder**: L2-L3 for the bind event, L3 for the smile output (Feel-Example classification as Resonance Stage 1).

### §6.2 — Visual + interoceptive binding (mother recognition E15)

🟢 **Event**: Mother recognition at 5 months (§3.4 V12) binds visual face with:
- Auditory voice (from prenatal)
- Smell/olfactory chunks (compiled from feeding)
- Warmth/body comfort chunks
- Comfort history chunks (feeding satiation, soothing)

🟡 **Framework claim**: This is the **most modality-rich chunk** in F1 first year (6+ modalities). It takes 5 months to compile despite beginning from day 1 because each modality-pair binding needs repetition + the cross-modality binding needs repeated co-occurrence of all modalities simultaneously.

Compile mechanism candidates (plan §0.7 + [Schema/Chunk.md](../../Body-Base/Schema/Chunk.md) §2):
- **Repetition (LTP)** — primary mechanism. Hundreds of feeding/soothing events provide massive repetition.
- **Emotional peak** — feeding reward is positive valence peak, tagging the chunk.
- **Multi-modal binding** — co-occurrence across modalities.
- **Sleep consolidation** — chunks reorganize overnight (Diekelmann & Born 2010).

### §6.3 — Visual + motor binding (visuomotor reaching)

🟢 **Event**: Visually guided reaching at 4-5 months (§4.3 V10) binds visual target chunks with arm motor chunks. First **voluntary visuomotor chunk**.

🟡 **Framework claim**: Visuomotor binding is substrate for virtually all subsequent tool-use and object-manipulation chunks. It's the **platform on which hand-object chunks** compile in second year. See [05-Motor-Output-Arc.md](05-Motor-Output-Arc.md) §2.4 for full treatment.

### §6.4 — Visual + social + motor + attention binding (joint attention E18)

🟢 **Event**: Joint attention triadic at 9-12 months (§4.6 V17) is the **peak multi-modal multi-body-input chunk** of F1 first year. Integrates visual + auditory + motor + social + attention + pre-theory-of-mind.

🟡 **Framework implication**: Joint attention chunks are **binding** — they link chunks from every body-input system except interoceptive. This is why Tomasello argues joint attention is the keystone for subsequent language and cultural learning: once a baby can coordinate attention across agents + objects, word-object referencing (language grounding) becomes possible.

### §6.5 — Mechanism candidates for binding (partial)

🟡 **Framework question**: What is the neural mechanism for cross-modal binding? Candidates:

- **Temporal co-occurrence binding** (Hebb-style) — features firing together at same time wire together. Plausible for short-latency binding (visual face + voice heard simultaneously).
- **Multi-sensory convergence neurons** — cells in superior temporal sulcus, superior colliculus, and fusiform area respond to multi-modal input. Found in infants and adults. Stein & Meredith 1993.
- **Sleep-consolidated associative binding** — overnight memory reorganization + associative linking (Tononi & Cirelli 2014).
- **Emotional tag binding** — high-valence states create strong associative links across modalities that happened together during the state (Phelps & LeDoux 2005).

F1 does NOT commit to a mechanism winner here — [07-Social-Recognition-Arc.md](07-Social-Recognition-Arc.md) §6 (Nút Thắt 3 primary home) will drill the verdict using social smile as cleanest case. This file's contribution is **compiling the visual-side events that any mechanism must explain**.

## §7 — Compile Mechanisms Mapping for Visual Chunks

> Plan §0.7 + [Chunk.md](../../Body-Base/Schema/Chunk.md) §2 identify 4 chunk compile mechanisms. This section maps each mechanism onto the visual chunks trajectory.

### §7.1 — Repetition (LTP-based compile)

**Dominant for**: Face chunks stabilization (CONSPEC → individual faces → mother face), object file chunks, color chunks, gaze following chunks.

**Evidence**: Every baby gets thousands of face views per day. Repetition drives LTP stabilization (Bliss & Lømo 1973 seminal LTP work). Chunks compile in proportion to exposure frequency × co-occurrence rate.

**Framework claim**: This is the **primary compile mechanism for visual chunks** because visual input is dense, high-rate, and persistently available from day 1.

### §7.2 — Emotional peak tagging

**Dominant for**: Mother face chunk (+ feeding reward valence), stranger anxiety (+ fear valence), joint attention (+ shared-experience reward).

**Evidence**: Emotional peaks from feeding, soothing, distress create strong associative tags on co-occurring visual inputs. Amygdala + hippocampus consolidation (Phelps & LeDoux 2005, McGaugh 2000).

**Framework claim**: Emotional tagging accelerates chunk compile beyond what pure repetition would predict. Mother face compiles faster than father face partly because mother is more associated with feeding (strong positive valence peak).

### §7.3 — Multi-modal binding

**Dominant for**: Social smile E12, mother recognition E15, joint attention E18, visuomotor reaching.

Covered in §6. The binding mechanism is itself a compile mechanism — it creates cross-modal chunks that could not form from single-modality repetition alone.

### §7.4 — Sleep consolidation

**Dominant for**: Overnight visual chunk reorganization. Infants sleep ~16 hours/day in first months; sleep is dense in REM in early infancy (nearly 50% REM vs adult 20%, Roffwarg et al. 1966). REM has been implicated in visual system consolidation specifically.

**Evidence**: Gomez et al. 2006 — infant word learning benefits from nap. Walker 2017 on sleep + memory consolidation. Infant visual perceptual learning benefits from sleep (Friedrich et al. 2015).

**Framework claim**: Sleep is the **hidden compile engine** for visual chunks. Daily exposure provides raw material; sleep consolidates + binds + organizes. This is particularly relevant for multi-modal binding since cross-modality associations form during offline replay (Diekelmann & Born 2010).

### §7.5 — Summary table

| Chunk type | Primary compile mechanism | Supporting mechanisms |
|---|---|---|
| CONSPEC face template | Innate (not compiled) | — |
| Individual face chunks | Repetition | Emotional tagging |
| Mother face chunk | Multi-modal binding + emotional peak | Repetition, sleep |
| Object file chunks | Repetition | Multi-modal binding (when applicable) |
| Object permanence | Repetition + prediction-error driven | Sleep consolidation |
| Gaze following | Repetition + imitation + social reward | Emotional tagging |
| Joint attention | Multi-modal binding | Emotional tagging + social reward |

## §8 — Cross-Body-Input Connections

> Visual chunks don't exist in isolation. This section maps visual ↔ other body-input binds for F1 cross-referencing.

### §8.1 — Visual ↔ Auditory

**Primary binds**:
- Face ↔ voice (social smile E12, §6.1)
- Mother face ↔ mother voice (E15)
- Object ↔ object sound (ball bouncing + thud sound)
- Face ↔ speech streaming (name recognition, see [04-Auditory-System-Arc.md](04-Auditory-System-Arc.md) §3.6 V2-Aud integration)

### §8.2 — Visual ↔ Motor

**Primary binds**:
- Visually guided reaching (§4.3, V10) → foundation
- Gaze following (§4.4, V13) → coordinated eye motor
- Joint attention (§4.6, V17) → hand + gaze + look coordination
- Imitation (gestures, facial expressions, actions)

See [05-Motor-Output-Arc.md](05-Motor-Output-Arc.md) §6.1 for motor-side treatment.

### §8.3 — Visual ↔ Interoceptive

**Primary binds**:
- Mother face ↔ comfort/warmth state (E15)
- Stranger face ↔ fear/distress state (E16)
- Peek-a-boo face return ↔ positive arousal (E18-related anticipation)

These are **emotionally tagged visual chunks**. See [06b-Interoceptive-Other-States.md](06b-Interoceptive-Other-States.md) for full treatment + [07-Social-Recognition-Arc.md](07-Social-Recognition-Arc.md) §4 for E15 binding drill.

### §8.4 — Visual ↔ Social

Virtually every visual social milestone (CONSPEC, direct gaze, mother recognition, social smile, gaze following, joint attention) is a **visual-social chunk bind**. Social arc in [07-Social-Recognition-Arc.md](07-Social-Recognition-Arc.md) is heavily dependent on visual chunks as anchoring modality.

**Framework observation**: Vision is the **primary anchoring modality for social chunks in typical development**. Blind infants achieve social milestones using alternative anchoring (auditory + tactile) but slightly delayed (Fraiberg 1977).

### §8.5 — Visual ↔ Verbal (preview, full in 08)

**Visual chunks as receptive grounding for words**:
- Word-object mapping requires stable object file chunks (compile by 4-6 months, §3.7)
- Word-face mapping requires individual face chunks (compile by 5 months, §3.4)
- Common noun comprehension (Bergelson & Swingley 2012) at 6-9 months requires both visual object chunks + auditory word chunks + cross-modal binding
- Full treatment in [08-Verbal-Production-Arc.md](08-Verbal-Production-Arc.md) §3

**Receptive-productive gap** (Nút Thắt 5 preview): Baby comprehends "mẹ" by 6-9 months (visual face chunk + auditory word chunk + bind) but does not produce "mẹ" until ~12-18 months. The comprehension side is built largely on visual chunks. This is one of the "knowing vs acting" asymmetries across F1 — covered in Nút Thắt 5, primary home [08-Verbal-Production-Arc.md](08-Verbal-Production-Arc.md).

### §9.1 — Nút Thắt 1 — Proto-chunk gradient vs discrete (partial contribution)

**This file's contribution**: Object file chunks (§3.7-§3.8) and object permanence chunks provide a **visual system test case** for the gradient vs discrete question.

**Evidence for gradient**:
- Baillargeon 2.5-month surprise at impossible trajectory (very basic)
- 3.5-month solidity (specific property)
- 4-6 month more complex scenarios
- 8-9 month action-based permanence
- 12+ month elaborate reasoning

These are not a single discrete "object permanence present/absent" switch. They are **incrementally elaborated sub-chunks** (solidity, trajectory, continuity, identity), each strengthening at its own rate per exposure + compile dynamics.

**Visual arc verdict preview**: Gradient view supported by the visual data. Each sub-property of "object permanence" compiles semi-independently. This is consistent with **early-LTP → late-LTP gradient** at the biological level (Bliss & Lømo 1973, Abraham & Williams 2008 late-LTP review).

## §10 — Open Questions

🟡 **Open questions** this file surfaces or leaves for downstream files:

1. **Does CONSPEC contain a direct-gaze subcomponent, or is direct gaze detection compiled in first 2-5 days?** The timing is too early for pure post-birth learning but the evolutionary argument for innate pre-wiring is a framework claim, not direct anatomical evidence. Open — needs better developmental neuroscience.

2. **What is the minimum exposure count for a compiled visual chunk?** Mother face distinguishable at 4 days (Bushnell 1989) — how many discrete feeding-face-view events in 4 days? Probably 50-100 episodes × minutes of face contact. Can we quantify "compile threshold" more precisely?

3. **Why does stereopsis emerge discretely (3-4 months) while other visual chunks emerge gradient?** The substrate difference (binocular cell tuning + vergence) may explain. This is a case where the **gradient vs discrete distinction (Nút Thắt 1)** has mixed evidence within the visual system itself — most chunks gradient, but stereopsis appears discrete. Interesting.

4. **Is visuomotor reaching (4-5mo) best classified as L2 or L3?** The reaching **attempt** is L3 (action component) but the accuracy is poor for weeks after first emergence. Classification depends on how "coordinated" the execution needs to be. This may indicate ladder levels have internal graduation.

5. **Visual chunks for causality** (one object launches another — Michotte perception) — when do these compile? Leslie 1984, Oakes 1994 show ~7-9 months. Where in visual arc does this fit? Partially addressed in [09-Event-Chunks-Inference-Matrix.md](09-Event-Chunks-Inference-Matrix.md).

6. **Mirror self-recognition cross-cultural variation**: passes earlier in Western WEIRD samples than in some non-WEIRD samples (Broesch et al. 2011). Framework implication: visual self chunk timing is **culturally modulated**, not universal biological schedule. What varies — the self-concept chunks or just the mirror test-taking behavior?

7. **Autism spectrum gaze-following and joint attention delay**: known early marker (Klin et al. 2002). Framework question: which chunks are delayed compiling — visual chunks, binding chunks, social reference chunks, or motivational/reward chunks for attending to faces? Open — downstream clinical implication of F1 framework.
