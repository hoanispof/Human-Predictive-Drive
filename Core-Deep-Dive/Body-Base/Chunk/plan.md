## §1 — Architectural Commitment: 4-Folder Structure

### §1.1 — Axis split rationale

**Original linear 10-file plan** (pre-N+4) split by thread topic và drill phase (Foundation → Core → Meta). Problem:
- Thread topics overlapped across drill files (connection mechanism shows up in 03, 04, 05, 06)
- Age-based thinking implicit (child vs adult) but framework threads cross-cut ages
- External mechanism vs internal mechanism never cleanly separated

**Session N+4 prep conversation** identified cleaner axis: **mechanism type**, not topic or age:

1. **Substrate mechanism** — How biology (hardware + modality) compiles raw input into chunks. Applies all ages but child pre-verbal = cleanest test case.

2. **External-shared mechanism** — How externally-provided input (shared with others) installs chunks. Includes infant joint attention (9mo), mother-child imitation, shared sensory events (chó sủa / sấm), adult language, ritual, cultural transmission. Age-independent — same mechanism at 9mo and 40yo.

3. **Internal-novel mechanism** — How chunks form inside head without external shared label, seeking label post-formation (or never labeled). Aha moments, tacit knowledge, intuition, felt sense. Age-independent — child discovery and adult insight use same mechanism.

4. **Language (special case of #2)** — Most optimized external anchor system after ~100,000 years cultural-evolutionary refinement. Deserves deep sub-folder inside External mechanism folder (not top-level — it's ONE instance of external shared, albeit the most dominant for general-purpose cognition).

### §1.3 — Folder boundary definitions

Subtle cases handled explicitly:

**F1 vs F3**: substrate mechanism vs external-shared mechanism
- F1 = HOW biology compiles raw modality input into chunks (bottom-up, regardless of social context)
- F3 = HOW external shared source provides chunk content/label (top-down from environment)
- Example: trẻ 4.5mo recognize own name
  - F1 covers: auditory pattern recognition substrate (phoneme → word pattern)
  - F3 covers: social response mechanism (turn to speaker, name-as-social-reference)
  - Primary home: F3 (social context dominant), cross-ref F1 (substrate enables it)

**F3 vs F4**: external input vs internal processing
- F3 = chunks formed BECAUSE of external shared source (observable, transmittable)
- F4 = chunks formed INTERNALLY regardless of input source, focus on mechanism transforming existing chunks into new ones
- Example: Einstein reading Newton → decades thinking → aha relativity
  - F3 covers: how reading Newton installed Newtonian chunks (cultural transmission)
  - F4 covers: internal PFC chain + dissonance + insight emergence mechanism
  - Both use same case, different focus

**Child vs Adult**: NOT an axis. Each folder applies to both, developmental arc appears as TIMELINE within folder where relevant. "Child" trong F1 name means "cleanest test case for substrate mechanism," not "only children."
