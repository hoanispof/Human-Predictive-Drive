## §6 — COMPILATION PER SCALE

```

  LEVEL 1 — DIRECT VERIFICATION:
    Every interaction = body-feedback → by-product match confirmed/denied
    → "Gặp bạn → vui" = DIRECT body-feedback confirm match

    → → EXPONENTIAL scaling: 1→5→25→125... (proxy chain)
    → Compilation: compile "trust node" as SHORT chunk → stop verifying
    → FAST but FRAGILE (node wrong → many people wrong simultaneously)
    → RISK: node fills WRONG gaps → collective follows wrong direction
    → = "Tin vì NGƯỜI TÔI TIN tin"
    
  LEVEL 3 — INSTITUTIONAL COMPILATION:
    Trust compiled at cultural/institutional level → individual receives PRE-COMPILED
    → "Bằng ĐH → việc tốt" = cultural chunk compiled over generations

    → → Pre-compiled chunks CAN expire (degree inflation, market shift)

    → RISK: collective arc shifts → compiled chunks OUTDATED → mass mismatch
    → = "Tin vì MỌI NGƯỜI TIN"

  ⭐ COMPILATION SPEED × ACCURACY TRADE-OFF:
    Level 1: SLOW + ACCURATE (direct verify)
    Level 2: MEDIUM + MODERATE (trust bypass, some verify)
```
