## III — PROCESSING: Compiled vs Fresh (Reframe F1/F2)

```
TRỤC THẬT: COMPILATION LEVEL (không phải Feeling vs Logic content)

COMPILED (= "F1-like"):
  → Automatic, body-feedback direct, cost ≈ 0
  → Hebbian reinforced: càng dùng càng fast
  → "Cảm thấy" (not "nghĩ ra")
  → Einstein + toán quen = compiled
  → Bạn thân gặp nhau = compiled
  → Chef nếm → biết thiếu muối = compiled

FRESH PFC DRAFT (= "F2-like"):
  → Deliberate, PFC chain, cost > 0
  → Mỗi lần = fresh effort (chưa compile)
  → "Nghĩ ra" (not "cảm thấy")
  → Einstein + bài toán MỚI = fresh draft
  → Gặp stranger đoán cảm xúc = fresh draft
  → Therapist + case lạ = fresh draft

TRANSITION:
  Fresh → lặp lại + verify OK → Hebbian → compile → thành Compiled
  Compiled → disrupted (context mới, error) → phải Fresh lại
  = Learning = fresh → compiled. Unlearning = compiled → fresh.

IMPLICATION:
  "Logic" = observer label cho "PFC đang draft"
  "Feeling/Intuition" = observer label cho "compiled automatic"
  "Logic vs Intuition" = "shareable compiled vs non-shareable compiled"
  Bên trong body: chỉ có compiled ←→ fresh spectrum
  Con người = 100% feeling-driven ở tầng nền (PFC serve body-base)
```

## IV — 3 NGUỒN COST (Independent)

```
Total interaction cost = ① + ② + ③

① PFC DRAFT COST (metabolic):
  f(chain_length × novelty_degree)
  Compiled path: ≈ 0. Novel chain: HIGH.
  "Trời mưa → dùng ô" = 0. Einstein bài mới = HIGH.

② SUPPRESS COST (efference mismatch):
  f(intensity_of_suppressed_response × duration)
  Override compiled reaction → body resist → dissonance.
  "Kiềm phóng khoáng nói với bố" = moderate.
  "Introvert suppress mỗi ngày" = chronic.

③ UNCERTAINTY COST (cortisol holding):
  f(number_of_options × time_held × stakes)
  Multiple options, none clear → must hold open → cortisol.
  "Nên đổi việc?" held months = chronic.

SUSTAINABILITY = f(Total cost per interaction × frequency ÷ reward):
  Bạn thân: ① ≈ 0, ② ≈ 0, ③ ≈ 0 → MAX sustainable
  Business F2: ① moderate, ② ≈ 0, ③ low → sustainable if infrequent
  Forced suppress daily: ① moderate, ② HIGH, ③ moderate → burnout
```
