## §2 — SPM → RESONANCE CONNECTOR MODEL

### §2.1 — HOW SPM tăng Resonance probability

```
⭐ SPM = TOOL tăng xác suất Resonance qua 4 SKILLS:

  SKILL 1 — PREDICT:
    SPM core function: predict agent's state + behavior
    → Accurate prediction → better action selection
    → Better actions → by-product match PROBABILITY tăng
    → = SPM accuracy → Resonance probability
    
    SPM axes affect:
      ① Pattern-type → predict ĐÚNG LOẠI response
      ② Depth → predict NHIỀU aspects
      ③ Similarity → predict CHÍNH XÁC (template fit)
      ④ Feedback → CALIBRATE prediction real-time

  SKILL 2 — MIMIC/CLONE (session hôm nay):
    F1 output → unconscious mimicry (Chartrand & Bargh 1999)
    → State overlap tăng (enhancer c)
    → Gap direction gần nhau → by-product match easier
    → Trust + affiliation tăng (mimic → liked more)
    → = SPM CREATES conditions for Resonance via behavioral sync

  SKILL 3 — CALIBRATE:
    Feedback loop (Resonance §8, Bird & Cook 2013):
    → SPM predict → observe outcome → adjust model → predict better
    → Over time: model CONVERGE toward target reality
    → = SPM self-improves → Resonance quality increases over time
    → Needs FEEDBACK (not just output) → WHY Stream 2 has higher bar

  SKILL 4 — COMPILE (→ Entity-Compiled):
    Repeated SPM on same target → Hebbian compile → Entity-Compiled
    → Compiled SPM = F1 (cost ≈ 0) → sustainable
    → Entity-Compiled deep → Resonance AUTOMATIC (auto by-product match)
    → = SPM's END STATE: Resonance without effort
```
