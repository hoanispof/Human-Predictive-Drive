### §1.2 — Resonance Core redefined

```
⭐ RESONANCE CORE:

  2 entities TỒN TẠI → có ACTIONS → actions tương tác với DOMAIN
  → Mỗi bên fill gap CỦA MÌNH qua actions
  → Output (by-product) = sản phẩm phụ CỦA action
  → Khi by-product CỦA A match gap direction CỦA B → B reward
  → Khi MUTUAL (cả 2 chiều) → Resonance

  DOMAIN = môi trường tương tác (social, vật lý, abstract)
  → Resonance xảy ra TRONG domain, KHÔNG trong body
  → Body chỉ RECEIVE kết quả → reward hoặc không

  LIÊN KẾT Conflict-Dynamics:
  → Conflict formula: OVERLAP × SCARCITY × COMMITMENT
  → Khi 2 actions target CÙNG resource + resource SCARCE → CONFLICT (anti-match)
  → Khi 2 actions target KHÁC resource hoặc COMPLEMENTARY → RESONANCE
  → Resonance và Conflict = 2 outcomes TRÊN CÙNG 1 CƠ CHẾ (by-product interaction)

  STREAM 1 NUANCE:
  → Stream 1 = by-product từ EXISTENCE (không cần action)
  → Vợ xinh TỒN TẠI → by-product match aesthetic gap CỦA chồng
  → Bé TỒN TẠI → by-product match caregiving gap CỦA mẹ
  → "Action" ở Stream 1 = rất rộng: tồn tại, hiện diện, có thuộc tính
```

## §3 — ENHANCERS (tăng xác suất, KHÔNG PHẢI Resonance)

### §3.1 — 5 Enhancers

```
  (a) HARDWARE OVERLAP:
      Hardware similar → body-feedback pattern gần → gap direction gần
      → By-product match PROBABILITY tăng (vì outputs come from similar bodies)
      🟢 Bouchard 1990: twins → immediate resonance
      ⚠️ KHÔNG = Resonance. Hardware overlap + competing direction = rivalry

  (b) CHUNK/SPM OVERLAP:
      Shared chunks → SPM predict accurate → action match xác suất cao
      → Stream 2 quality tăng
      ⚠️ CHỈ áp dụng Stream 2 (Stream 1 không cần SPM)
      Ma trận 2D (Drill-Entity-Compiled §10b) = enhancer (b) + trục 4

  (c) STATE OVERLAP:
      Body-feedback hiện tại tương thích → gap direction tạm thời gần
      → Dễ by-product match TẠI THỜI ĐIỂM này
      State = current activation pattern (NOT trait — see §4)
      Clone mechanism tăng state overlap (see §5)

  (d) EXPRESSION MODALITY MATCH:
      Cùng channel (verbal↔verbal, somatic↔somatic)
      → Feedback loop quality tăng (read nhau rõ hơn)
      → Stream 2 calibrate nhanh hơn

  (e) EXPOSURE SUFFICIENT:
      Đủ tương tác để pattern emerge → by-product match lặp lại
      → From episode → pattern → Resonance
```

### §3.2 — Tại sao enhancers QUAN TRỌNG dù không phải Resonance

```
  Enhancers ↑↑ → Resonance probability ↑↑ → tìm thấy Resonance DỄ HƠN
  Enhancers ↓↓ → Resonance vẫn POSSIBLE nhưng RARE
  = "Duyên" = khi enhancers align + Resonance emerges
  = "Không duyên" = enhancers thấp + Resonance không emerge

  NHƯNG: enhancers CAO ≠ guarantee Resonance
  → Twins compete (high enhancers + anti-match direction) = NO Resonance
  → Strangers complementary (low enhancers + compatible direction) = Resonance

  → TRỤC 4 (direction) > enhancers cho Resonance outcome
```

## §4 — STATE vs TRAIT: OBSERVATION LENSES, KHÔNG PHẢI THAM SỐ

### §4.1 — Mainstream definitions

```
  MAINSTREAM:
  🟢 STATE (Spielberger 1966): temporary condition — mood, arousal at time t
  🟢 TRAIT (Allport 1937, Costa & McCrae 1992): stable disposition across time

  NHƯNG — CRITICAL:
  🟢 Fleeson 2001 (JPSP, landmark):
     "Traits as DENSITY DISTRIBUTIONS of states"
     → Trait = CENTRAL TENDENCY of states over time
     → Trait KHÔNG PHẢI neural region riêng
     → Trait = STATISTICAL PATTERN of repeated states
     → = CÙNG substrate, KHÁC temporal resolution

  🟢 Wright & Mischel 1987:
     Personality = "if...then..." situation-behavior profiles
     → NOT fixed trait → CONDITIONAL patterns

  🟢 Spielberger STAI (State-Trait Anxiety Inventory):
     CÙNG construct (anxiety) đo AS BOTH state AND trait
     → 2 CÁCH ĐO cùng 1 thứ, không phải 2 thứ khác nhau
```

### §4.2 — Framework mapping

```
⭐ STATE/TRAIT = 2 OBSERVATION LENSES trên CÙNG 1 SUBSTRATE:

  "STATE" = CURRENT ACTIVATION PATTERN:
    = Chunks đang fire TẠI THỜI ĐIỂM NÀY
    = Body-feedback hiện tại (tùy context, health, interaction)
    = SNAPSHOT at time t
    = BFM: current chunk dynamics (Shift/Miss/Gap)

  "TRAIT" = BACKGROUND PATTERN (BP §1):
    = Accumulated link density qua 15-20 năm
    = THẤM VÀO mọi chunk mới → BIAS activation
    = PFC KHÔNG observe trực tiếp (BP §5)
    = BP L62: "Personality = PFC label cho Background Pattern"

  CÙNG SUBSTRATE — KHÁC TEMPORAL SCALE:
    State = what's firing NOW (activation)
    Trait/BP = what TENDS to fire (accumulated bias)
    = Fleeson 2001 exactly: trait = density distribution of states
```

### §4.3 — Tại sao State converge, Trait không

```
  STATE CONVERGE vì:
    → Current activation = RESPONSIVE to current input
    → Partner present = DAILY input → body-feedback shift
    → Clone mechanism (§5) → body adopt partner's patterns
    → Shallow compiled, sensory-proximal, interaction-responsive
    → Cơ chế: F1 fire → simulate partner → body shift → repeat → compile

  TRAIT/BP KHÔNG CONVERGE vì:
    → Link density TÍCH LŨY qua 15-20 năm × nghìn micro-events
    → Phân tán qua TOÀN BỘ chunk network
    → PFC không observe → SPM cannot target
    → Clone mechanism shift ACTIVATION (state), KHÔNG shift LINK DENSITY (trait)
    → Partner = 1 input source / tổng 20 năm inputs → shift NHỎ

  🟢 Anderson 2003: emotional convergence over time
  🟢 Gonzaga 2010: NO personality convergence in couples
  🟢 Roels 2016: immune system 50% less variation in couples (state-level)
```

### §4.4 — Kết luận cho framework

```

  THAY VÀO ĐÓ dùng framework language:
```

## §5 — CLONE MECHANISM: SPM F1 OUTPUT SPECTRUM

### §5.1 — Research confirmed

```
  🟢 Chartrand & Bargh 1999 (JPSP, ~4,000+ citations):
     Chameleon effect: VÔ THỨC mimic posture, gesture, speech
     Khi bị mimic → đánh giá interaction TÍCH CỰC hơn
     Mechanism: perception-behavior link — THẤY → TỰ ĐỘNG fire motor

  🟢 Karremans & Verwijmeren 2008 (PSPB):
     Mimic NHIỀU HƠN khi thấy người kia HẤP DẪN
     Trong relationship → mimic ÍT HƠN (self-regulatory gate)

  🟢 Lakin et al. 2003 (J. Nonverbal Behavior):
     BIDIRECTIONAL: mimicry → affiliation, AND affiliation → mimicry
     Evolution: ban đầu = communication → evolved thành bonding
```

### §5.2 — 3-Layer mechanism

```
  CLONE = 3 LAYERS cộng dồn:

  LAYER 1 — HARDWARE FOUNDATION (evolution):
    Perception-behavior link (Chartrand & Bargh):
      THẤY hành vi → motor cortex fire TƯƠNG TỰ (vô thức)
    = HARDWARE default → everyone có, vary by degree

  LAYER 2 — DEVELOPMENTAL COMPILATION (sơ sinh → trưởng thành):
    Child-Dev 07-Social-Recognition-Arc: E29 Delayed Imitation
    Framework: imitation = "active learning of action chunks via Resonance"
    NOT hardware mirror module → learned social mechanism
    → Bé sơ sinh: lè lưỡi theo mẹ (earliest compile)
    → 4-6 tháng: action imitation (motor chunk from observation)
    → 9-12 tháng: deferred imitation (chunk STORED → reproduce later)
    → 18-24 tháng: goal-understanding (hiểu MỤC ĐÍCH, không chỉ copy motor)
    → 20+ năm practice → clone skill = DEEPLY COMPILED (F1, cost ≈ 0)

  LAYER 3 — CONTEXTUAL MODULATION (trust × valence × state):
    Trust CAO (hấp dẫn, yêu, ngưỡng mộ) → clone AMPLIFIED
    Trust THẤP (stranger, enemy) → clone SUPPRESSED
    Có partner → clone với người mới GIẢM (🟢 Karremans 2008)
    = Per-Agent Valence GATE clone behavior
    = CƠ CHẾ GIỐNG SPM valence gate (SPM §0.3 claim 6)
```
