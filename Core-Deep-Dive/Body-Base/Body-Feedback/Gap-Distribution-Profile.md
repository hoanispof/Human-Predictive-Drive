### §1.2 — A distribution, not a single point

```
⭐ EACH PERSON = 1 DISTRIBUTION, NOT 1 POINT:

  WRONG: "This person is in the body-domain" (oversimplification)
  RIGHT: "This person has gaps DISTRIBUTED with center of mass in the body-domain"

  Why:
    → EVERYONE has Tier 1 gaps (hunger, thirst, heat, cold) — sensory domain
    → EVERYONE has social gaps (connection, status) — social domain
    → Differences = RELATIVE, not ABSOLUTE
    → A CEO whose center is in the abstract domain STILL enjoys good food
      (sensory gap still present)
    → A blue-collar worker whose center is in the body domain STILL has some
      curiosity (small abstract gap)

  Distribution has a "CENTER OF MASS":
    → Where gaps are DENSEST
    → Where body-need fires MOST STRONGLY and MOST FREQUENTLY
    → Where activities concentrate MOST
    → = Center of mass = proxy for "where this person is on the spectrum"

  Distribution has "TAILS":
    → A body-domain person still has an abstract tail (mild curiosity)
    → An abstract-domain person still has a sensory tail (still eats, sleeps)
    → Tails = secondary gaps — lower priority, but STILL FIRE

  Distribution CHANGES OVER TIME:
    → Chunks accumulate → new gaps form → distribution SHIFTS
    → Chunks decay (disuse) → old gaps weaken → distribution NARROWS
    → Major life events → distribution CAN shift dramatically
```

    → See good food → a few encounters → gap "want better food"

    → Requires understanding social dynamics → compiled → gap about social position

    → = COMMON — but depth varies by cultural install

    → Requires years of domain chunks — "don't know it exists = no gap" applies FULLY
    → Only when enough chunks → gap FORMS → reward loop BEGINS
    → Domain-Mapping-Drive.md §5: "need to reach threshold in a matching domain"
    → = MINORITY — requires exposure + hardware fit + time

  COLLECTIVE: many deep chunks + collective perspective → GAP NEEDS HIGH THRESHOLD
    → Requires understanding collective systems + own domain deeply + broad perspective
    → Gap: "what is the collective MISSING?" → Collective-Body §2.5 detection mechanism
    → = VERY RARE — few people compile enough to SEE collective-level gaps
