## §1 — DISTINCTIONS (Feeling vs Related Concepts)

### 1.1 Feeling vs Emotion (Narrow)

```
🟡 MOST IMPORTANT DISTINCTION:

  MAINSTREAM "EMOTION":
    → Limbic valence (joy, sadness, anger, fear, disgust, surprise)
    → Ekman basic emotions (1970s-2000s)
    → Culturally universal expressions (Ekman claim, contested)
    → Discrete categories
    → Primarily felt/expressed emotions in social contexts

  FRAMEWORK "FEELING":
    → Much broader
    → Includes emotion narrow AS A SUBSET
    → Also includes: body states, social reading, cognitive,
      predictive, valence, meta-feelings, etc.
    → Continuous multi-dimensional
    → Not limited to socially expressed

  RELATIONSHIP:
    Emotion narrow ⊂ Feeling

  → Every "emotion" is a "feeling"
  → Not every "feeling" is an "emotion" narrow


  EKMAN BASIC EMOTIONS:
    → Anger, Disgust, Fear, Happiness, Sadness, Surprise (core 6)
    → Later added: contempt, etc.
    → Universal facial expressions (controversial, Barrett contests)
    → Biologically rooted affective programs

  → Framework: these are RECOGNIZABLE feeling categories
    but not the totality of feeling


  BARRETT'S CONSTRUCTIONIST VIEW:
    → Emotions = constructed from core affect + context + concept
    → Not discrete biological programs
    → Valence + arousal as building blocks
    → Categories are cultural

  → Framework: compatible with Feel-Labeling insight
    Raw body state (Feel-Consciousification) is universal,
    labels (Feel-Labeling) are cultural


  WHY THE DISTINCTION MATTERS:
    → "Feeling" in everyday speech often synonymous with "emotion"
    → Reader assumes narrow meaning
    → Framework must explicitly broaden
    → Entire Feeling.md §1.2 dedicated to this disclaimer


  TERMINOLOGY CONVENTION:
    → Framework uses "feeling" for broad unified system
    → "Emotion" reserved for specific subset (limbic valence)
    → "Affect" for technical psychology contexts (§1.7)
```

### 1.2 Feeling vs Sensation

```
🟡 SENSATION = SINGLE-MODALITY, FEELING = MULTI-MODAL:

  SENSATION:
    → Single sensory modality
    → Touch, sound, sight, smell, taste, proprioception
    → Relatively localized in processing
    → Can be very precise

  FEELING:
    → Multi-modal integration
    → Across many sensory streams + interoception + schema
    → Integrated into unified sense
    → Precision variable

  EXAMPLES:
    Sensation: "The smell of coffee" (olfactory alone)
    Feeling:   "Warmth and comfort when you smell coffee in the morning"
               (olfaction + memory schema + valence + body state)

    Sensation: "Touch on arm"
    Feeling:   "Comfort from this person's touch"
               (touch + trust schema + oxytocin + regulation)

    Sensation: "Cold temperature"
    Feeling:   "Bone-chilled, lonely winter feeling"
               (temperature + memory + mood + schema)


  OVERLAP ZONES:
    → Pain: can be pure sensation (sharp cut) or feeling
      (chronic back pain with emotional weight)
    → Pleasure: sensation (taste of chocolate) or feeling
      (birthday cake with family)
    → Discomfort: sensation (tight shoe) or feeling
      (uneasiness in a place)

  → Difference of DEGREE of integration


  FRAMEWORK POSITION:
    → Sensation = often Feel-RawSignals raw signal
    → Feeling = at Feel-Integration + beyond
    → Same body can produce both simultaneously
    → Pain is common bridge between them
```

### 1.3 Feeling vs Thought

```
🟡 THOUGHT vs FEELING — DIFFERENT BUT INTERACTING:

  THOUGHT:
    → PFC symbolic processing
    → Language-mediated (mostly)
    → Explicit, step-by-step often
    → Can be deliberately controlled
    → Happens in cortex primarily

  FEELING:
    → Body-based sensing
    → Pre-symbolic often
    → Implicit, holistic often
    → Cannot be directly controlled (only accessed or suppressed)
    → Distributed throughout body + brain


  INTERACTION:
    → Thought can TRIGGER feeling (schema firing)
    → Feeling can SHAPE thought (mood affects cognition)
    → Not independent, but distinct processes


  MAINSTREAM CONFUSION:
    → "I think I'm sad" — is this thought or feeling?
    → Framework: the feeling is body-based; "I think" is Feel-Labeling/Feel-Explanation
    → The statement is a LABEL of feeling, not the feeling itself


  KAHNEMAN'S SYSTEM 1 / SYSTEM 2:
    → System 1: fast, automatic, feeling-based
    → System 2: slow, deliberate, thought-based
    → Complementary, not adversarial
    → Most decisions use both


  DESCARTES' ERROR (Damasio 1994):
    → Descartes separated thinking from feeling ("I think, therefore I am")
    → This Cartesian split is MISGUIDED
    → Thinking and feeling are integrated in real cognition
    → VMPFC patients: thinking intact, feeling broken → cannot decide
    → Shows they're not independent in practice

  → Framework: follows Damasio's integration view


  NOT CLAIMING:
    ✗ Thought is bad, feeling is good
    ✗ Feelings are always right
    ✗ Thought is unbiological
    ✗ Feeling replaces thought

  CLAIMING:
    ✓ Both needed
    ✓ Body-first temporal usually
    ✓ Integration is real and important
    ✓ Wisdom = using both well
```

### 1.4 Feeling vs Intuition

```
🟡 INTUITION = SUBSET OF FEELING (compiled, fast, expert):

  MAINSTREAM "INTUITION":
    → Fast knowing without explicit reasoning
    → Often domain-specific
    → Sometimes accurate, sometimes not
    → "I just know"

  FRAMEWORK "INTUITION":
    → Compiled feeling firing fast
    → Subset of feeling
    → Specifically: pattern-matching that bypasses conscious analysis
    → Expert intuition = well-compiled pattern library

  RELATIONSHIP:
    Intuition ⊂ Feeling


  EXAMPLES:
    "Feeling that the answer is 42" = cognitive intuition
      → feeling + cognitive source
    "Feeling this person is lying" = social intuition
      → feeling + mirror source
    "Feeling the code has a bug here" = expert intuition
      → feeling + compiled expertise


  KAHNEMAN + KLEIN SYNTHESIS (2009):
    → Kahneman: intuitions often biased, unreliable
    → Klein: expert intuitions reliable in domain
    → Jointly: intuitions reliable WHEN environment is predictable
      AND practitioner has feedback-rich experience
    → Framework: intuitions = compiled feelings, quality depends on compilation


  WHEN TO TRUST INTUITION:
    ✓ In expertise domain with feedback learning
    ✓ For pattern-rich decisions
    ✓ When stakes match accuracy level
    ✗ In novel domains
    ✗ For subtle statistical judgments
    ✗ When biased patterns have compiled


  TERMINOLOGY:
    → "Intuition" is common word
    → "Compiled feeling" is framework technical term
    → Same phenomenon, different emphasis
```

### 1.5 Feeling vs Gut Feeling

```
🟡 GUT FEELING = POPULAR TERM FOR VISCERAL INTUITION:

  POPULAR "GUT FEELING":
    → Visceral sensation guiding decision
    → Often about people, situations, risks
    → Trusted when accurate, dismissed when wrong
    → Cultural expression across many languages

  FRAMEWORK "GUT FEELING":
    → Type of feeling heavy on visceral interoception
    → Often mirror-based (reading others)
    → Or predictive (imagine-final preview)
    → Or schema-triggered (pattern recognition)

  RELATIONSHIP:
    Gut feeling ⊂ Feeling


  ENTERIC NERVOUS SYSTEM ("second brain"):
    → 500 million neurons in gut
    → Vagus nerve afferents to brain
    → Gut microbiome affects mood
    → Body signals from gut reach insula

  → "Gut feeling" has literal physiological basis
  → Not just metaphor


  GAVIN DE BECKER "GIFT OF FEAR" (1997) 🟡:
    → Argues gut feelings about danger are often correct
    → Should be trusted especially in safety contexts
    → Critics: selection bias, cherry-picked cases
    → Framework: compatible with L0 threat channel fast firing


  VS INTELLECTUAL "KNOWING":
    → Gut feeling: pre-verbal, body-based
    → Intellectual: post-verbal, cognitive
    → Both valid data sources
    → Wisdom integrates


  TERMINOLOGY:
    → "Gut feeling" is everyday language
    → Framework can use it for visceral-heavy feelings
    → Don't overload into technical definition
```

### 1.6 Feeling vs Mood

```
🟡 MOOD = SUSTAINED FEELING STATE:

  MOOD:
    → Lasting hours to days
    → Diffuse, background
    → Shapes cognition and perception
    → Less object-directed

  FEELING:
    → Can be acute (seconds-minutes) or sustained
    → Can be focused on specific object
    → More variable intensity

  RELATIONSHIP:
    Mood = extended, diffuse feeling state
    → Mood is one TIMESCALE of feeling


  EXAMPLES:
    Acute feeling: "Happy when receiving good news"
    Mood:          "In a good mood today"

    Acute feeling: "Angry when a driver cuts me off"
    Mood:          "Been irritable lately"

    Acute feeling: "Sad upon hearing bad news"
    Mood:          "Sad for weeks without a clear reason"


  CLINICAL DISTINCTION:
    → Depression = sustained negative mood
    → Anxiety disorder = sustained fear/worry mood
    → Mood disorders ≠ feeling disorders
    → Feelings happen, moods persist


  NEURAL BASIS:
    → Mood: neuromodulator baselines (serotonin, dopamine, cortisol)
    → Feeling: acute firing + body state changes
    → Mood shapes feeling responses
    → Feeling can shift mood over time


  FRAMEWORK:
    → Mood = chronic feeling state (like chronic body-dissonance)
    → Important for mental health understanding
    → Requires different interventions than acute feelings
```

### 1.7 Feeling vs Affect

```
🟡 AFFECT = TECHNICAL PSYCHOLOGY TERM:

  AFFECT (technical):
    → Psychology/psychiatry technical term
    → Includes feelings, emotions, moods
    → Often described in valence + arousal dimensions
    → Observable component of emotional state
    → Core Affect (Russell 1980) = valence × arousal

  FEELING (framework):
    → Broader everyday meaning
    → Includes technical "affect" phenomena
    → Plus body states, cognitive, predictive, etc.
    → Less technical, more inclusive

  RELATIONSHIP:
    → Framework "feeling" is broader than technical "affect"
    → Affect is core component framework uses
    → Not contradictory, different scopes


  CORE AFFECT (Russell):
    → 2-dimensional: valence (+/-) × arousal (high/low)
    → Circular plot of emotional states
    → Cultural variation in labeling
    → Basis for Barrett's constructionist theory

  → Framework: valence + arousal are subset of feeling dimensions
  → Framework adds: multi-source channels, 7-layer structure


  AFFECTIVE NEUROSCIENCE (Panksepp):
    → Different lineage from cognitive psychology
    → 7 primary affective systems
    → Subcortical origins
    → Cross-species

  → Framework: compatible with Panksepp
  → "Affect" in Panksepp's sense includes more than valence+arousal


  TERMINOLOGY:
    → Academic/clinical contexts: use "affect"
    → Everyday + framework: use "feeling"
    → Sometimes interchangeable
    → Context determines precision
```

## §2 — RESEARCH STREAM: Damasio

```
🟢 ANTONIO DAMASIO — SOMATIC MARKER HYPOTHESIS:

  BACKGROUND:
    → Neurologist, USC
    → Studied brain-damaged patients
    → Published "Descartes' Error" (1994)
    → Follow-ups: "The Feeling of What Happens" (1999),
      "Looking for Spinoza" (2003), "Self Comes to Mind" (2010)

  CORE CLAIM:
    Feelings are NECESSARY for decision-making, not optional.

  EVIDENCE:
    → Phineas Gage (1848 railroad accident)
       - Retained logic + memory + language
       - Lost feeling-based decision making
       - "Personality changed" → poor life choices
    → Modern VMPFC patients (ventromedial prefrontal cortex damage)
       - Same pattern: logic intact, feeling impaired
       - Iowa Gambling Task: cannot feel "bad deck" warning
       - Normal subjects develop skin conductance response (SCR)
         before consciously knowing which deck is bad
       - VMPFC patients: no anticipatory SCR, cannot learn
       - → Feeling-based learning critical

  SOMATIC MARKER HYPOTHESIS:
    → Decisions are felt before consciously known
    → Body generates marker (good/bad feeling)
    → Marker guides decision
    → Without marker: paralyzed choice

  COMPATIBLE WITH:
    → Framework Imagine-Final preview feelings
    → Framework body-first temporal order
    → Framework feeling literacy importance

  "DESCARTES' ERROR":
    → Descartes: "I think, therefore I am"
    → Damasio: mind cannot function without body
    → Emotion + reason integrated in real cognition
    → Not "animal vs human" dualism
    → Whole body-brain system as cognitive unit

  CORE + EXTENDED SELF (Damasio):
    → Proto-self: body state monitoring (unconscious)
    → Core self: awareness of being affected
    → Autobiographical self: narrative self
    → Hierarchical, built on body foundation


  FRAMEWORK DRAWS FROM DAMASIO:
    → Body-first temporal order (foundation)
    → Feeling necessary for decisions (§10 application)
    → Somatic marker = preview feeling subtype
    → Integration of emotion + cognition


  CONFIDENCE:
    → 🟢 Core patient evidence robust
    → 🟢 Iowa Gambling Task replicated
    → 🟡 Philosophical extensions (consciousness theory) less solid
    → 🟢 Decision role of feeling established

  Key references:
    Damasio A. (1994). Descartes' Error: Emotion, Reason,
      and the Human Brain. Putnam.
    Damasio A. (1999). The Feeling of What Happens. Harcourt.
    Damasio A. (2003). Looking for Spinoza. Harcourt.
    Damasio A. (2010). Self Comes to Mind. Pantheon.
    Bechara et al. (1997). Deciding advantageously before
      knowing the advantageous strategy. Science.
```

## §3 — RESEARCH STREAM: Gendlin

```
🟢 EUGENE GENDLIN — FELT SENSE & FOCUSING:

  BACKGROUND:
    → Philosopher + psychotherapist
    → Worked with Carl Rogers at University of Chicago
    → Developed Focusing methodology
    → "Focusing" published 1978
    → Founded International Focusing Institute
    → "Philosophy of the Implicit" theoretical framework

  CORE INSIGHT:
    There is body-based knowing BEFORE words exist to express it.
    This is the "felt sense".

  FELT SENSE:
    → Pre-verbal body knowledge
    → Holistic (not list of sensations)
    → Specific but not yet articulated
    → Experiential, not intellectual
    → "That thing I feel about this situation"

  KEY OBSERVATION (from therapy research):
    → Gendlin studied what distinguished clients who improved
      from those who didn't
    → Answer: those who paused and CHECKED with body sense
      ("Hmm, wait... it's more like...")
    → Those who rationalized quickly did NOT improve
    → Across modalities, this pattern held
    → → Accessing felt sense predicts therapeutic success

  FOCUSING METHOD (6 steps):
    ① Clearing a space
    ② Felt sense
    ③ Handle (word/image that fits)
    ④ Resonating (check word against body)
    ⑤ Asking (what's this about?)
    ⑥ Receiving (welcome what comes)

  THINKING AT THE EDGE (TAE):
    → Gendlin's method for accessing pre-conceptual understanding
    → Use body sense to generate new concepts
    → Relevant to framework development


  PHILOSOPHY OF THE IMPLICIT:
    → "...." (blank in speech) = place of felt sense
    → Body can sense more than words express
    → Implicit knowledge is structured, not chaotic
    → Experience has "more" waiting to be said

  COMPATIBLE WITH:
    → Framework Feel-Consciousification (pre-labeling)
    → Somatic-Articulation-Loop.md (body → words process)
    → Framework skepticism about Feel-Labeling/Feel-Explanation labels
    → Literacy training rationale


  INFLUENCE:
    → Therapy generally (not just "Focusing" therapists)
    → Influenced AEDP, Hakomi, sensorimotor therapy
    → Writing methods (Peter Elbow influenced by Gendlin)
    → Philosophy (continental phenomenology influences)


  FRAMEWORK DRAWS FROM GENDLIN:
    → Felt sense as direct evidence body-first is real
    → Feel-Consciousification described using Gendlin's observations
    → Focusing method in literacy training (Feeling-Accuracy §7.2)
    → "Pre-verbal but structured" insight


  CONFIDENCE:
    → 🟢 Clinical effectiveness evidence robust
    → 🟢 Phenomenological accuracy (matches self-report)
    → 🟡 Specific neural correlates not fully mapped
    → 🟢 Across-modality prediction solid

  Key references:
    Gendlin E. (1978). Focusing. Bantam.
    Gendlin E. (1996). Focusing-Oriented Psychotherapy. Guilford.
    Gendlin E. (1997). A Process Model. Northwestern UP (philosophy).
    Hendricks M. (2001). Focusing-oriented research review.
    Purton C. (2004). Person-Centred Therapy: The Focusing-Oriented Approach.
```

## §4 — RESEARCH STREAM: Craig & Interoception

```
🟢 A.D. (BUD) CRAIG — INTEROCEPTION & INSULA:

  BACKGROUND:
    → Neuroanatomist, Barrow Neurological Institute
    → Mapped interoceptive afferent pathways
    → "How do you feel?" (2002) landmark paper
    → "How Do You Feel? An Interoceptive Moment with Your
       Neurobiological Self" book (2015)

  CORE CONTRIBUTION:
    Mapping the neural pathway from body to feeling.

  INTEROCEPTION:
    → Sensing internal body state
    → Includes: heart rate, respiration, temperature, pain,
      organ states, muscle effort, hunger, thirst, etc.
    → Afferent pathway distinct from exteroception
    → Integration hub: anterior insula

  PATHWAY:
    Body → Lamina I spinothalamic → Thalamus → Posterior insula →
    Mid insula → Anterior insula → Feeling

  INSULA FUNCTIONS:
    → Posterior insula: primary interoceptive sensory
    → Mid insula: integration, meta-representation
    → Anterior insula: subjective feeling awareness
    → Right anterior insula: feeling consciously accessible

  EVIDENCE:
    → Stroke damage in anterior insula → feeling disruption
    → fMRI: insula activates during emotional awareness
    → Insula activates during interoceptive accuracy tasks
    → Meditation training → improved insula function (Farb 2013)

  SPECIES COMPARISON:
    → Craig proposed anterior insula unique to primates with
      advanced interoception
    → Theory: humans have specialized feeling awareness
    → Von Economo neurons (VENs) concentrated in anterior
      cingulate + insula — large spindle neurons
    → Speed + efficiency of feeling integration


  FRAMEWORK DRAWS FROM CRAIG:
    → Interoception as foundation of feeling
    → Insula as integration hub (Feel-Integration neural substrate)
    → Pathway from body to feeling made explicit
    → Physical basis for "body-first" claim


  INTEROCEPTIVE ACCURACY MEASUREMENT:
    → Heartbeat detection task
    → Garfinkel 2015: 3 dimensions (accuracy, sensibility, awareness)
    → Individual differences reliable
    → Correlates with emotional regulation + mental health


  COMPATIBLE WITH:
    → Framework Feel-RawSignals/Feel-Integration architecture
    → Damasio somatic marker (insula as substrate)
    → Feeling literacy as trainable skill
    → Individual variation in feeling access


  CONFIDENCE:
    → 🟢 Anatomical pathways well-mapped
    → 🟢 Insula role in interoception solid
    → 🟡 Primate uniqueness of anterior insula (some contest)
    → 🟢 Measurement methods validated

  Key references:
    Craig A.D. (2002). How do you feel? Interoception: the sense
      of the physiological condition of the body. Nat Rev Neurosci.
    Craig A.D. (2009). How do you feel — now? The anterior insula
      and human awareness. Nat Rev Neurosci.
    Craig A.D. (2015). How Do You Feel? An Interoceptive Moment
      with Your Neurobiological Self. Princeton UP.
    Garfinkel S. et al. (2015). Knowing your own heart:
      distinguishing interoceptive accuracy from interoceptive
      awareness. Biological Psychology.
```

## §5 — RESEARCH STREAM: Panksepp

```
🟢 JAAK PANKSEPP — AFFECTIVE NEUROSCIENCE:

  BACKGROUND:
    → Estonian-American neuroscientist
    → Coined "affective neuroscience" (1992)
    → Studied cross-species affective systems
    → "Affective Neuroscience" (1998) field-founding book
    → "The Archaeology of Mind" (2012) for lay + professional

  CORE CLAIM:
    Mammals share primary affective systems rooted in subcortical
    brain areas, cross-species conserved.

  7 PRIMARY AFFECTIVE SYSTEMS:
    ① SEEKING — exploration, desire, curiosity
       (VTA dopamine pathways, most broad)
    ② FEAR — freeze, escape
       (amygdala circuits)
    ③ RAGE — anger, aggression
       (medial amygdala, hypothalamus, PAG)
    ④ LUST — sexual desire
       (hypothalamic nuclei, sex hormones)
    ⑤ CARE — nurturing, maternal
       (oxytocin, cingulate, BNST)
    ⑥ PANIC/GRIEF — separation distress, social loss
       (ACC, septal, PAG)
    ⑦ PLAY — rough-and-tumble, joy
       (dorsomedial thalamus, PAG)

  EVIDENCE:
    → Cross-species electrical stimulation produces
      behavioral patterns characteristic of each system
    → Pharmacological manipulation specifically affects systems
    → Humans self-report matching emotions when corresponding
      circuits activated
    → Subcortical origin — not PFC-created

  CROSS-SPECIES CONSERVATION:
    → Rats have all 7 systems
    → Mammals broadly similar
    → Implies ancient origin
    → Framework: explains evolutionary sufficiency

  PLAY SYSTEM (Panksepp signature):
    → Rats "laugh" (50kHz ultrasound)
    → Play deprivation harms development
    → Parallel to human play/joy
    → Revolutionary for animal affect science


  SUBCORTICAL ORIGIN INSIGHT:
    → Feelings come from SUBCORTICAL structures primarily
    → PFC modulates, doesn't generate
    → Damage to cortex doesn't eliminate feelings
    → Damage to subcortex does
    → Supports framework body-first claim


  COMPATIBLE WITH:
    → Framework L2 novelty (SEEKING)
    → Framework L0 threat (FEAR)
    → Framework agent input (CARE, PANIC/GRIEF)
    → Framework evolutionary framing (ancient systems)


  FRAMEWORK DRAWS FROM PANKSEPP:
    → Feelings have ancient biological basis
    → Not cultural constructions alone
    → Cross-species continuity
    → Subcortical origin of feeling


  CONTROVERSIES:
    → Barrett constructionist view contests discrete systems
    → Panksepp + Barrett debated publicly
    → Framework: compatible with both (systems exist, labels cultural)
    → Integration possible, not zero-sum


  CONFIDENCE:
    → 🟢 Cross-species neural circuits robust
    → 🟢 Subcortical origin well-evidenced
    → 🟡 Exact number of systems (7) somewhat arbitrary
    → 🟢 Human-animal parallel defensible

  Key references:
    Panksepp J. (1998). Affective Neuroscience. Oxford UP.
    Panksepp J. & Biven L. (2012). The Archaeology of Mind. Norton.
    Panksepp J. (2005). Affective consciousness: Core emotional
      feelings in animals and humans. Consciousness and Cognition.
```

```
    → Distinct from "I can recite this"

    ① Novice: explicit rules, slow analysis
    ② Beginner: some pattern recognition, still conscious
    ③ Competent: patterns compiled, feelings emerging
    ④ Proficient: feelings guide, verification secondary
    ⑤ Expert: feelings dominate, explanation often post-hoc

  → Parallel to Dreyfus skill acquisition model 🟢


  LITERACY MATTERS FOR LEARNING:

     → Pre-verbal confusion signal

     → → Move on without real learning

  ② NOTICING "DO UNDERSTAND" FEELING
     → The click sensation
     → Body confirms integration
     → Feel-Consciousification/Feel-Observation signal

      → "Feelings off-topic"
```
