- Feeling literacy = ability to READ the compiled layer

**What it involves** (from Feeling-Accuracy.md §5):

2. Location — where in body
3. Vocabulary — richness of labels
4. Attribution — mapping feeling to cause
5. Discrimination — distinguishing similar feelings
6. Calibration — knowing accuracy per layer

**Modern relevance** (from Feeling.md §5): Ancestral feelings were simple + reliable for ancestral environments. Modern mismatches require INTERPRETER bridge — literacy is that bridge.

**Medium decisions** (weeks-months):

**Large decisions** (months-years):
