## §1 — ĐỊNH NGHĨA FEELING

### 1.1 Formal Definition

```
⭐ FEELING — ĐỊNH NGHĨA CHÍNH THỨC:

  FEELING = WHAT PFC SEES khi nó observe body-chunk interaction.

  Mở rộng:
    Feeling = signal phản hồi đã được INTEGRATED về state của body systems
              (L0-L3 body-base + agent input + mirror + schema + imagine-final
              + valence + cognitive prediction) → OBSERVABLE ở PFC level.

  Nói ngắn gọn: Feeling là CÁCH body tự làm cho chính mình trở thành
  noticeable với PFC.


  PHÂN TÍCH THEO TỪNG PHẦN:

  ① "signal phản hồi" — feedback, không phải input mới
     Feeling KHÔNG tạo ra thông tin mới. Feeling REPORT về trạng thái
     body đã tính toán. Body là source, feeling là messenger.

  ② "đã được INTEGRATED" — unified, không phải list
     Feeling KHÔNG phải danh sách sensation. Feeling = ONE unified
     sense về "tôi đang ở trạng thái nào". Multi-source → single stream.

  ③ "state của body systems" — multi-source
     Body có NHIỀU systems đang fire đồng thời (L0-L3, mirror, schema,
     imagine-final, valence, cognitive). Feeling tích hợp TẤT CẢ.

  ④ "OBSERVABLE ở PFC level" — PFC-accessible
     Feeling phải ĐỦ mạnh để PFC nhận biết. Body tính toán vô thức
     liên tục — chỉ 1 phần nhỏ nổi lên thành feeling.
     "Observable" không có nghĩa "có từ sẵn" — có nghĩa PFC CÓ THỂ
     tương tác được (observe, locate, label, explain — dù imperfect).


  CÔNG THỨC ĐƠN GIẢN:
    Body fire + Chunk activate + Integrated + PFC-noticeable = FEELING
```

### 1.2 Feeling ≠ Emotion Narrow — Disclaimer

```
⚠️ DISCLAIMER QUAN TRỌNG:

  Khi nghe từ "feeling", mainstream thường hiểu EMOTION NARROW:
  vui, buồn, giận, sợ, yêu, ghét.

  TRONG FRAMEWORK NÀY, "feeling" RỘNG HƠN NHIỀU:

  Emotion narrow = 1 SUBSET của feeling (limbic valence)
  Feeling = tất cả body state changes mà PFC có thể observe

  7 LOẠI FEELING (đều là CÙNG HỆ THỐNG):

    ① Body-physical: "đói", "mệt", "đau bụng", "lạnh", "chân tê"
    ② Emotional (narrow sense): "vui", "buồn", "giận", "sợ"
    ③ Social reading: "người kia không tin", "không khí tense"
    ④ Cognitive/prediction: "bài này đúng", "có gì đó sai", "bế tắc"
    ⑤ Predictive/intuition: "có gì đó không ổn", "không nên tin người này"
    ⑥ Valence: "thích cái này", "ghét cái tên", "yêu em"
    ⑦ Schema-triggered: "bình yên khi về nhà", "rùng mình khi nhớ nơi đó"

  TẤT CẢ 7 LOẠI = CÙNG MECHANISM: integrated body signal → PFC observe.
  Framework dùng "FEELING" vì rộng nhất, inclusive nhất.
```

### 1.3 Ba Đặc Tính Critical

```
🟡 3 ĐẶC TÍNH DEFINING FEELING:

  ① MULTI-SOURCE (Đa Nguồn)
     Feeling KHÔNG đến từ 1 system duy nhất.
     Feeling = convergence của nhiều body systems fire đồng thời.

     Ví dụ "cảm thấy yêu em":
       → L0 safe + L1 relax + L2 novelty-reward + L3 meaning
       → agent input thỏa mãn + mirror positive + imagine-final tích cực
       → valence compiled positive + schema relationship active
       = 9+ channels convergence → PFC compress → "yêu em" (2 từ)

  ② INTEGRATED (Đã Tích Hợp)
     Feeling KHÔNG phải LIST of signals. Feeling = UNIFIED sense.
     Body gửi PFC: "toàn bộ trạng thái tôi là VẦY."
     Không phải: "channel 1, channel 2, channel 3..."
     Integration happens ở insula + ACC + VMPFC (🟢 Craig 2009).

  ③ OBSERVABLE (PFC Có Thể Tương Tác)
     Threshold phụ thuộc 2 chiều: MAGNITUDE × CLARITY.
     Body tính toán 24/7 — hầu hết KHÔNG trở thành feeling.

     MAGNITUDE = cường độ signal (body-feedback generate):
       → Cao: đau, sợ, phấn khích → PFC khó bỏ qua
       → Thấp: lấn cấn nhẹ, vi sai nhỏ → dễ lọt

     CLARITY = chunk density cho phép PFC phân biệt signal nói gì:
       → f(chunks tích lũy trong domain + self-signal interoception)
       → Cao: expert detect dù signal nhẹ ("nét vẽ kỳ lạ" ở chợ đồ cũ)
       → Thấp: novice cần signal mạnh mới notice, hoặc bỏ qua hoàn toàn

     → Clarity CAO = hạ magnitude threshold → expert "thấy nhiều hơn"
     → Cùng input → expert thấy rõ, novice bỏ qua
     → Từ "sao sao ấy" (mạnh+mờ) đến "rõ ràng" (mạnh+rõ hoặc nhẹ+rõ).
```

### 2.3 Layer 4-7: PFC Processing Domain

```

```

```
  Feeling = INTERFACE giữa body computation và PFC awareness.
  Body = tính toán. Feeling = kết quả hiện lên. PFC = observe + label + explain.


⭐ 4 HỆ QUẢ:

  ① Feeling KHÔNG TẠO RA thông tin mới
     → Body tính toán XONG → feeling REPORT → PFC observe
     → Temporal order: Body compute → Feeling emerge → PFC observe
     → 🟢 Damasio 1994, 1999: somatic markers PRECEDE conscious decision

  ② Feeling KHÔNG THỂ SAI ở raw level (Layer 1-3)
     → Body truth IS body truth. SAI xảy ra ở PFC: label wrong, explain wrong

     → Expert: large, domain-real library → accurate feeling
     → Beginner: small library → unreliable feeling

  ④ Chunk system và feeling system = MỘT hệ thống
     → Chunk operations produce feeling layers (§9 Bridge)
     → Feeling feeds back into chunk operations (§9 Bridge)
     → = Same elephant, 2 observation angles
```

### 3.2 Body-First Invariant

```

    → 🟢 Damasio VMPFC patients: PFC intact, feeling broken → cannot decide

    → 🟢 Bechara 1997: skin conductance changes BEFORE conscious "bad deck"
    → 🟢 Libet 1983: brain activity (~350ms) BEFORE conscious awareness
    → 🟢 LeDoux 1996: amygdala pathway (~12ms) BYPASS cortex entirely

  TIMELINE:
    T=0ms      Body systems fire
    T=50ms     Integration begins (insula + ACC)
    T=100ms    Integrated state forms
```
