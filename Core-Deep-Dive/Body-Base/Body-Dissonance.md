## 1. Bộ 3 Signal — Tại Sao Dissonance Đặc Biệt

```
⭐ 3 BODY SIGNAL = 3 trạng thái của melody:

  Body-Satisfaction:  "Đủ rồi, dừng"       = Harmony ĐẠT
  Body-Reward:        "Hay, tiếp tục"       = Harmony đang BUILD
  Body-Dissonance:    "Chưa ổn, thay đổi"   = Harmony BỊ PHÁ

  Tại sao Dissonance có SPECTRUM RỘNG NHẤT:

    Body-Satisfaction: PFC KHÔNG nghe → chỉ biết "hết muốn" → KHÔNG CẦN phân loại
    Body-Reward: chỉ cần "tiếp" → gradient đơn giản (dễ chịu → sướng → cực khoái)
    Body-Dissonance: CẦN phân biệt LOẠI mismatch → vì MỖI LOẠI cần RESPONSE KHÁC

    → "Bứt rứt" → cần: tìm hiểu thêm, explore
    → "Đói" → cần: ăn
    → "Sợ" → cần: chạy hoặc đóng băng
    → "Tức giận" → cần: bảo vệ boundary
    → "Đau" → cần: tránh source of damage
    → "Chán" → cần: đổi activity, tìm novelty

    = Body KHÔNG thể chỉ nói "khó chịu" — phải nói "khó chịu KIỂU GÌ"
    = Vì survival PHỤTHUỘC vào response ĐÚNG loại mismatch
    = Phản ứng sai loại = chết (sợ mà không chạy, đau mà không tránh)

  → = Body có HÀNG CHỤC loại dissonance vì body cần HÀNG CHỤC loại response
  → = Đây là lý do "cảm xúc" phong phú — không phải để "thú vị"
      mà để PHÂN LOẠI mismatch cho survival


  ⭐ SIGNAL = DIRECTION × MAGNITUDE (áp dụng cho CẢ 3 signal):

    3 signal = 3 DIRECTION: Satisfaction (DỪNG), Reward (TIẾP), Dissonance (ĐỔI)
    Mỗi signal có MAGNITUDE: micro → nhẹ → trung → mạnh → cực → SHUTDOWN

    SHUTDOWN = circuit breaker — không thuộc signal NÀO riêng:
      → Body: "delta QUÁ LỚN → CẮT ĐIỆN — bất kể hướng nào"
      → Reward CỰC (vui quá, fan gặp idol) → NGẤT (vagal syncope)
      → Dissonance CỰC (sợ quá, bị hổ vồ) → NGẤT / FREEZE / amygdala hijack
      → Satisfaction CỰC (kiệt sức) → NGỦ THIẾP (repair shutdown)
      → = Giống cầu dao: quá tải → NGẮT — không care dòng đi đâu

    File này deep dive DISSONANCE direction — spectrum RỘNG NHẤT trong 3.
    (Chi tiết Direction × Magnitude model: Core-v7.5-Draft.md §5 FEEDBACK)

```

```
  NGUỒN 1 — INFORMATION MISMATCH (prediction error):

    Micro:    "Có gì đó hơi khác" → lấn cấn
              VD: vào phòng, đồ đạc hơi lệch → chưa biết gì, hơi lạ

              VD: bạn thân phản bội → "không thể tin được" → worldview lung lay

    Cực:     "Có thể CHẾT" → panic / freeze

    Nhẹ:     "Bị hiểu sai / bỏ qua" → bực / buồn nhẹ

    Trung:   "Bị phê bình / mất mặt" → xấu hổ / tức

              VD: 3 tháng học, chưa thấy kết quả → "mình có đang đi đúng không?"

  → = CÙNG cảm giác "khó chịu" nhưng KHÁC NGUỒN → cần response KHÁC:

  → = Không biết nguồn → "khó chịu mà không biết vì sao" → response SAI → kéo dài
```

## 3. Tại Sao Cần Nhiều Loại — Evolutionary Logic

```
🟡 Body invest NHIỀU hormone + pathway cho dissonance vì:

  SURVIVAL = TRÁNH CHẾT > TÌM SƯỚNG

    → Body-Reward sai (tưởng sướng mà không sướng) → mất 1 cơ hội → OK, sống tiếp
    → Body-Dissonance sai (tưởng an toàn mà nguy hiểm) → CHẾT
    → = Evolution ưu tiên DISSONANCE accuracy hơn REWARD accuracy

    → 🟢 Negativity bias (Baumeister 2001): bad information processed
      MORE THOROUGHLY than good information → evolutionary advantage
    → 🟢 Loss aversion (Kahneman & Tversky 1979): losing $100 hurts MORE
      than gaining $100 feels good → ~2x ratio

  PHÂN LOẠI ĐÚNG = RESPONSE ĐÚNG = SỐNG:
    → Sợ + chạy = đúng khi gặp thú dữ
    → Sợ + đối đầu = sai → chết
    → Tức giận + đối đầu = đúng khi bị xâm phạm
    → Tức giận + chạy = sai → mất territory
    → = MỖI loại dissonance có RESPONSE TỐI ƯU riêng
    → = Body CẦN phân biệt → nên có NHIỀU loại

  BODY INVEST cho dissonance:
    → Nhiều hormone: NE, adrenaline, cortisol, CRH, substance P,...
    → Nhiều pathway: amygdala, insula, ACC, sympathetic nervous system
    → Nhiều receptor: pain (A-delta, C), chemoreceptor, thermoreceptor
    → = "Phòng cháy" phức tạp hơn "phòng vui" — vì fire KHÔNG được sai
```

## 4. Dissonance × 3 Bộ Tham Số Khác

```

  × BODY-BASE LAYERS:
    L0 Alive: dissonance CỰC MẠNH (panic, extreme pain) — suppress all
    L1 Survival: dissonance MẠNH (đói, mệt, đau) — suppress L2+L3
    L2 Quality: dissonance TRUNG BÌNH (khó chịu body, mất connection)

  × MELODY ARC:

  × IMAGINE FINAL:
```

```
    → Dissonance VỪA ĐỦ + imagine final + recovery = TỐI ƯU
```
