# PFC-Operations — Hold + Suppress trên Compiled/Fresh Spectrum

> **PFC có 2 operations trên Compiled/Fresh spectrum:**
> **HOLD = amplify pattern mới. SUPPRESS = block pattern cũ.**
>
> **2 operations KHÁC CĂN BẢN: khác cost, khác outcome, khác sustainability.**
> **Hold → CAN compile → sustainable. Suppress → CANNOT compile "not" → unsustainable.**
>
> **Compiled chunks có QUALITY DIMENSION:**
> **Genuine-compiled (body confirmed) ≠ Schema-compiled (PFC-driven) ≠ Threat-compiled (bị ép).**
> **Cùng "compiled" nhưng KHÁC tag, KHÁC SPM capacity, KHÁC suốt đời.**
>
> **PFC budget = FINITE. SHARED. Mệt ở work → SPM cho con YẾU → không phải "không yêu con."**

## §2 — 2 OPERATIONS: HOLD + SUPPRESS

### §2.1 — HOLD = PFC Amplify

```
⭐ HOLD = PFC sustain attention + working memory trên 1 pattern:

  CƠ CHẾ:
    PFC → top-down signal → amplify activation CỦA target pattern.
    = "Tập trung vào cái này" / "Giữ trong đầu."

  COST: ① PFC draft (metabolic — glucose consumption).

  ĐẶC TÍNH:
    → Active process — cần PFC LIÊN TỤC maintain.
    → Khi PFC release → pattern DROPS (nếu chưa compiled).
    → Lặp đủ → Hebbian compile → pattern self-sustain → PFC release OK.
    → Learning trajectory: Hold → compile → automatic.

  VÍ DỤ:
    → Học bài mới: PFC hold problem trong working memory.
    → Tập thói quen: PFC hold "dậy sớm" intention.
    → Mẹ observe con teenage: PFC hold "con đang thay đổi, lắng nghe."

  Brain region: dlPFC, FEF.
  🟢 Baddeley 2003: working memory = PFC sustained attention.
  🟢 Miller & Cohen 2001: PFC = top-down attention amplification.
  🟢 Curtis & D'Esposito 2003: PFC sustained firing = working memory.
```

### §2.2 — SUPPRESS = PFC Inhibit

```
⭐ SUPPRESS = PFC actively BLOCK pattern đang tự fire:

  CƠ CHẾ:
    Pattern đã compiled → triggered bởi context → SELF-FIRE (automatic).
    PFC → inhibitory signal → block output route.
    = "Đừng nghĩ về cái này" / "Kìm lại" / "Ép xuống."

  COST: ② Suppress (efference mismatch — "tôi KHÔNG ĐANG LÀ TÔI").

  ĐẶC TÍNH:
    → Active process — cần PFC LIÊN TỤC inhibit.
    → Pattern vẫn EXIST (Hebbian: wired = wired) — chỉ BLOCK output.
    → Pattern underground → fire khi PFC yếu (stress, fatigue).
    → Efference copy mismatch: body expect → PFC block → dissonance.

  VÍ DỤ:
    → Thích chơi nhưng mẹ bắt học → suppress play-urge.
    → Giận nhưng phải giữ bình tĩnh → suppress anger.
    → CEO muốn chơi với con nhưng deadline → suppress family urge.

  Brain region: rIFG, vlPFC.
  🟢 Anderson & Green 2001: Think/No-Think paradigm.
  🟢 Aron 2007: right inferior frontal gyrus = inhibitory control hub.
  🟢 Wegner 1987: ironic process theory — suppress → rebound.
```

### §2.3 — HOLD ≠ SUPPRESS: Phân biệt nền tảng

```
⭐ 2 OPERATIONS KHÁC CĂN BẢN:

  ┌──────────────┬────────────────────┬────────────────────┐
  │              │ HOLD               │ SUPPRESS            │
  ├──────────────┼────────────────────┼────────────────────┤
  │ Direction    │ BOOST new          │ BLOCK existing      │
  │ Action       │ Amplify activation │ Inhibit output      │
  │ Target       │ New/weak pattern   │ Strong/compiled     │
  │ Cost source  │ ① PFC draft       │ ② Efference mismatch│
  │ Body feels   │ "Effort" (neutral) │ "Not being me"      │
  │ Compile?     │ CAN compile new    │ Cannot compile "not"│
  │ Sustainable  │ Yes (→compile→free)│ No (cost persists)  │
  │ Brain region │ dlPFC, FEF         │ rIFG, vlPFC         │
  └──────────────┴────────────────────┴────────────────────┘

  HOLD → CAN lead to compilation (positive trajectory).
  SUPPRESS → CANNOT compile "absence" (only block, not build).
  → Suppress alone = ALWAYS temporary (PFC budget = finite).
  → Suppress MUST pair with Hold (replacement) to sustain.

  🟡 Hold/Suppress as 2 structurally distinct PFC operations = framework synthesis.
  🟢 Individual components established (Miller 2001, Aron 2007).
```

## §4 — 3 OUTCOMES CỦA HOLD + SUPPRESS

> Khi cần thay đổi (tổ hợp ②), 3 outcomes có thể xảy ra.

### §4.1 — Outcome A: Genuine Shift

```
⭐ HOLD + SUPPRESS → NEW COMPILED → OLD FADES:

  "Thích ngủ nướng → dần quen dậy sớm → cơ thể khỏe hơn."

  CƠ CHẾ:
    ① PFC HOLD new pattern.
    ② Pattern cũ compiled nhưng KHÔNG CỰC MẠNH.
    ③ Lặp lại → Hebbian compile pattern mới.
    ④ Pattern mới cho BODY REWARD (genuine) → body CONFIRM.
    ⑤ Pattern cũ KHÔNG fire → Hebbian reverse → decay.
    ⑥ KẾT QUẢ: new compiled THAY THẾ old → PFC release → automatic.

  4 ĐIỀU KIỆN ĐỒNG THỜI:
    → Hardware ĐỦ KHẢ NĂNG shift.
    → Pattern cũ KHÔNG quá deep (chưa BP level).
    → Pattern mới cho BODY REWARD (genuine, not forced).
    → Internal drive (bản thân muốn, vmPFC suppress DRN).

  = "Thật sự thay đổi" — body chuyển từ resist → accept → prefer.
  🟢 Lally et al. 2010: habit formation 18-254 days (average 66).
```

### §4.2 — Outcome B: Compiled Suppress → Flat State

```
⭐ HOLD + SUPPRESS → COMPILE "HOW TO SUPPRESS" → BURIED + FLAT:

  "Quên cảm xúc thích thể thao, không hứng thú gì cả."

  CƠ CHẾ:
    ① PFC HOLD new pattern ("học toán theo mẹ").
    ② Pattern cũ ("thích thể thao") compiled MẠNH (multi-modal, years).
    ③ Lặp lại → COMPILE CÁI GÌ?
       → KHÔNG compile "yêu toán" (body KHÔNG confirm).
       → COMPILE "cách suppress thể thao" = META-PATTERN:
         [thấy thể thao] → [ép xuống] → [ignore] → [turn off].
       → = Compile THE SUPPRESS OPERATION ITSELF.
    ④ Pattern cũ KHÔNG BỊ DELETE — bị BURIED:
       → Link density vẫn cao → vẫn bias processing.
       → Chỉ bị BLOCK output route (compiled suppress blocks it).
    ⑤ KẾT QUẢ:
       → Reward system dampened → flat affect.
       → Không thích cũ (suppress) VÀ không thích mới (no genuine).
       → = "Chịu đựng" KHÔNG PHẢI "thay đổi."

  ⚠️ DANGEROUS: looks "adapted" from outside.
    → Parent: "đã ngoan, đã chịu học" → THỰC TẾ: reward system ĐÓNG.
    → Long-term: anhedonia risk, depression trajectory.

  🟢 Maier & Seligman 1967/2016: repeated suppress → passivity.
  🟡 "Compiled suppress" as meta-pattern = framework concept.
```

### §4.3 — Outcome C: PFC Failure → Revert + Conflict

```
⭐ HOLD + SUPPRESS → PFC DEPLETED → PATTERN BREAK THROUGH:

  "PFC bỏ cuộc, cãi nhau với mẹ, vùng vằng chạy ra ngoài."

  CƠ CHẾ:
    ① PFC HOLD + SUPPRESS → DOUBLE drain.
    ② Pattern cũ compiled CỰC MẠNH (deep + years).
    ③ PFC metabolic budget = FINITE → glucose depletes.
    ④ Pattern cũ BREAK THROUGH: emotional burst.
       → Efference mismatch GIẢI PHÓNG đột ngột.
       → = WHY suppressed emotions come out as BURSTS.
    ⑤ THÊM: negative valence COMPILE vào context.
       → "Học toán" gắn conflict → lần sau suppress CÒN KHÓ HƠN.
       → NEGATIVE SPIRAL: ép → burst → conflict → ép harder.

  🟢 Gailliot & Baumeister 2007: glucose depletion → self-control fails.
  🟢 Muraven & Baumeister 2000: ego depletion.
```

### §4.4 — Phân biệt 3 trạng thái

```
⭐ 3 TRẠNG THÁI KHÁC CĂN BẢN:

  ┌────────────────┬───────────────────────────────────────┐
  │ GENUINE CHANGE │ New compiled. Old DECAYED.             │
  │ (Outcome A)    │ Body PREFER new. No suppress needed.   │
  │                │ Cost: 0. Sustainable. Healthy.          │
  ├────────────────┼───────────────────────────────────────┤
  │ COMPILED       │ Suppress compiled. Old BURIED.         │
  │ SUPPRESS       │ Body NEUTRAL (flat). Auto-suppress.    │
  │ (Outcome B)    │ Cost: low. Reward system dampened.     │
  │                │ "Functional but joyless."               │
  ├────────────────┼───────────────────────────────────────┤
  │ ACTIVE         │ PFC still fighting. Old still FIRING.  │
  │ SUPPRESS       │ Body DISTRESSED (efference mismatch).  │
  │ (pre-outcome)  │ Cost: HIGH. Unsustainable.             │
  │                │ Will become A, B, or C.                 │
  └────────────────┴───────────────────────────────────────┘

  ⚠️ Nhầm lẫn phổ biến:
    Compiled suppress LOOKS LIKE genuine change from outside.
    Phân biệt: body response to trigger (not verbal report).
    Genuine: trigger → body neutral/positive (truly moved on).
    Compiled suppress: trigger → body spike THEN auto-suppress → flat.
    → Milliseconds genuine response TRƯỚC suppress kicks in = diagnostic window.

  🟡 3-state distinction = framework synthesis.
  🟢 Components: Maier 2016, Wegner 1987, Gailliot 2007.
```

## §6 — B vs C DETERMINANT: 7 FACTORS

> Khi Hold + Suppress xảy ra, CÁI GÌ quyết định Outcome B vs C?

```
⭐ 7 FACTORS:

  FACTOR 1 — PATTERN CŨ: Depth × Density (BP 2D model)
    Density CAO → xu hướng B (compile "suppress everywhere").
    Depth CAO + Density THẤP → B HOẶC C (tùy factors khác).

  FACTOR 2 — PATTERN MỚI: Compiled Quality
    Genuine (curiosity) → Hold cost THẤP → xu hướng A (shift thật).
    Schema (obligation) → Hold cost TRUNG BÌNH → xu hướng B.
    Threat (bị ép) → Hold cost CAO → xu hướng C hoặc B nếu sustained.

  FACTOR 3 — PFC BUDGET (individual variation)
    🟢 PFC volume + connectivity = genetically variable.
    Budget lớn → có thể B (đủ sức compiled suppress).
    Budget nhỏ → xu hướng C (depleted sớm → burst).
    Budget GIẢM nếu cortisol chronic (🟢 McEwen 2007).

  FACTOR 4 — EXTERNAL vs INTERNAL DRIVE (vmPFC/DRN)
    Internal (vmPFC robust) → xu hướng A nếu genuine, B nếu schema.
    External (DRN default) → xu hướng C ("vô ích" → bỏ cuộc sớm).
    🟢 Maier & Seligman 2016: controllability = vmPFC LEARNED.

  FACTOR 5 — DURATION + CONSISTENCY
    Short suppress (days-weeks) → active suppress → C nếu fail.
    Long + consistent (months-years) → Hebbian compile suppress → B.
    Inconsistent (on-off) → C liên tục ("burst → suppress → burst").

  FACTOR 6 — SLEEP QUALITY
    Sleep tốt → emotional decoupling → dễ A hơn.
    Sleep kém → PFC budget KHÔNG recover → C dễ hơn.
    🟢 Walker 2017: REM emotional decoupling.

  FACTOR 7 — SOCIAL ENVIRONMENT
    Có agent hỗ trợ → cortisol giảm → PFC budget tăng → A dễ hơn.
    Hostile/alone → PFC drain faster → B hoặc C.
    🟢 Coan & Sbarra 2015: Social Baseline Theory.


  TÓM TẮT:
  ┌──────────────────────────────────────────────────────────────┐
  │ B (compiled suppress) XÁC SUẤT CAO KHI:                     │
  │   Pattern cũ density CAO + Pattern mới schema/obligation     │
  │   + Duration DÀI + consistent + PFC budget đủ               │
  │   = "Quen chịu đựng, functional but flat."                  │
  ├──────────────────────────────────────────────────────────────┤
  │ C (PFC thất bại) XÁC SUẤT CAO KHI:                          │
  │   Pattern cũ depth CAO + Pattern mới threat                  │
  │   + PFC budget nhỏ/damaged + External force + Sleep kém      │
  │   = "Bùng nổ, mất kiểm soát."                               │
```
