# PFC-Operations — Hold and Suppress on the Compiled/Fresh Spectrum

> **PFC has 2 operations on the Compiled/Fresh spectrum:**
> **HOLD = amplify a new pattern. SUPPRESS = block an old one.**
>
> **The 2 operations are FUNDAMENTALLY different: different cost, different outcome, different sustainability.**
> **Hold → CAN compile → sustainable. Suppress → CANNOT compile "absence" → unsustainable.**
>
> **Compiled chunks have a QUALITY DIMENSION:**
> **Genuine-compiled (body confirmed) ≠ Schema-compiled (PFC-driven) ≠ Threat-compiled (forced).**
> **Same "compiled" status — but DIFFERENT NET link pattern, DIFFERENT Self-Pattern-Modeling capacity, DIFFERENT across a lifetime.**
>
> **PFC budget = FINITE. SHARED. Exhausted at work → Self-Pattern-Modeling for children WEAKENS → NOT "not loving them."**

## §2 — 2 OPERATIONS: HOLD + SUPPRESS

### §2.1 — HOLD = PFC Amplify

```
⭐ HOLD = PFC sustain attention + working memory on 1 pattern:

  MECHANISM:
    PFC → top-down signal → amplify activation OF target pattern.
    = "Focus on this" / "Keep in mind."

  COST: ① PFC draft (processing load — serial bottleneck + catecholamine demand).

  PROPERTIES:
    → Active process — PFC must CONTINUOUSLY maintain.
    → When PFC releases → pattern DROPS (if not yet compiled).
    → Repeat enough → Hebbian compile → pattern self-sustains → PFC can release.
    → Learning trajectory: Hold → compile → automatic.

  EXAMPLES:
    → Learning something new: PFC holds the problem in working memory.
    → Building a habit: PFC holds the "wake early" intention.
    → Parent observing a teenager: PFC holds "child is changing, listen carefully."

  Brain region: dlPFC, FEF.
  🟢 Baddeley 2003: working memory = PFC sustained attention.
  🟢 Miller & Cohen 2001: PFC = top-down attention amplification.
  🟢 Curtis & D'Esposito 2003: PFC sustained firing = working memory.
```

### §2.2 — SUPPRESS = PFC Inhibit

```
⭐ SUPPRESS = PFC actively BLOCKS a pattern that is self-firing:

  MECHANISM:
    Pattern already compiled → triggered by context → SELF-FIRES (automatic).
    PFC → inhibitory signal → blocks output route.
    = "Don't think about this" / "Hold it back" / "Push it down."

  COST: ② Suppress (efference mismatch — "I am NOT BEING MYSELF").

  PROPERTIES:
    → Active process — PFC must CONTINUOUSLY inhibit.
    → Pattern still EXISTS (Hebbian: wired = wired) — only OUTPUT is blocked.
    → Pattern underground → fires when PFC is weak (stress, fatigue).
    → Efference copy mismatch: body expects → PFC blocks → dissonance.

  EXAMPLES:
    → Wanting to play but forced to study → suppress play-urge.
    → Angry but must stay calm → suppress anger.
    → CEO wants to be with family but deadline looms → suppress family urge.

  Brain region: rIFG, vlPFC.
  🟢 Anderson & Green 2001: Think/No-Think paradigm.
  🟢 Aron 2007: right inferior frontal gyrus = inhibitory control hub.
  🟢 Wegner 1987: ironic process theory — suppress → rebound.
```

### §2.3 — HOLD ≠ SUPPRESS: Fundamental Distinction

```
⭐ 2 OPERATIONS FUNDAMENTALLY DIFFERENT:

  ┌──────────────┬────────────────────┬────────────────────────┐
  │              │ HOLD               │ SUPPRESS                │
  ├──────────────┼────────────────────┼────────────────────────┤
  │ Direction    │ BOOST new          │ BLOCK existing          │
  │ Action       │ Amplify activation │ Inhibit output          │
  │ Target       │ New/weak pattern   │ Strong/compiled         │
  │ Cost source  │ ① PFC draft       │ ② Efference mismatch    │
  │ Body feels   │ "Effort" (neutral) │ "Not being myself"      │
  │ Compile?     │ CAN compile new    │ Cannot compile "not"    │
  │ Sustainable  │ Yes (→compile→free)│ No (cost persists)      │
  │ Brain region │ dlPFC, FEF         │ rIFG, vlPFC             │
  └──────────────┴────────────────────┴────────────────────────┘

  HOLD → CAN lead to compilation (positive trajectory).
  SUPPRESS → CANNOT compile "absence" (only block, not build).
  → Suppress alone = ALWAYS temporary (PFC budget = finite).
  → Suppress MUST pair with Hold (replacement) to sustain.

  🟡 Hold/Suppress as 2 structurally distinct PFC operations = framework synthesis.
  🟢 Individual components established (Miller 2001, Aron 2007).
```

### §2.4 — Hold/Suppress × Dissonance Type: Differential Effectiveness

```
🟡 PFC OPERATIONS × 2 DISSONANCE TYPES (Dissonance-Signal-Architecture.md v1.0 §7.2):

  ┌─────────────────┬──────────────────────────┬────────────────────────────┐
  │ PFC Operation   │ On Evaluative Dissonance │ On Direct-State Dissonance │
  ├─────────────────┼──────────────────────────┼────────────────────────────┤
  │ HOLD            │ ✅ EFFECTIVE              │ ⚠️ PARTIAL                 │

  │                 │ term (🟢 Wegner 1994)    │ budget exhausts            │

## §4 — 3 OUTCOMES OF HOLD + SUPPRESS

> When change is needed (combination ②), 3 outcomes are possible.

### §4.1 — Outcome A: Genuine Shift

```
⭐ HOLD + SUPPRESS → NEW COMPILED → OLD FADES:

  "Liked sleeping in → gradually adapted to early rising → body feels healthier."

  MECHANISM:
    ① PFC HOLDS new pattern.
    ② Old compiled pattern is NOT EXTREMELY strong.
    ③ Repeat → Hebbian compiles new pattern.
    ④ New pattern gives GENUINE BODY REWARD → body CONFIRMS.
    ⑤ Old pattern STOPS firing → Hebbian reverse → decays.
    ⑥ RESULT: new compiled REPLACES old → PFC releases → automatic.

  4 CONDITIONS REQUIRED SIMULTANEOUSLY:
    → Hardware CAPABLE of shift.
    → Old pattern NOT too deep (not yet at Background-Pattern level).
    → New pattern gives GENUINE BODY REWARD (not forced).
    → Internal drive (person genuinely wants it, vmPFC suppresses DRN).

  = "True change" — body moves from resisting → accepting → preferring.
  🟢 Lally et al. 2010: habit formation 18-254 days (average 66).
```

### §4.2 — Outcome B: Compiled Suppress → Flat State

```
⭐ HOLD + SUPPRESS → COMPILE "HOW TO SUPPRESS" → BURIED + FLAT:

  "Forgot the feeling of loving sports — now indifferent to everything."

  MECHANISM:
    ① PFC HOLDS new pattern ("study math for mom").
    ② Old pattern ("loved sports") STRONGLY compiled (multi-modal, years).
    ③ Repeat → WHAT COMPILES?
       → NOT "love of math" (body DOES NOT confirm).
       → COMPILES "how to suppress sports" = META-PATTERN:
         [see sports] → [push down] → [ignore] → [shut off].
       → = Compiling THE SUPPRESS OPERATION ITSELF.
    ④ Old pattern NOT DELETED — BURIED:
       → Link density still high → still biases processing.
       → Only OUTPUT ROUTE is BLOCKED (compiled suppress blocks it).
    ⑤ RESULT:
       → Reward system dampened → flat affect.
       → Doesn't enjoy the old (suppressed) AND doesn't enjoy the new (no genuine reward).
       → = "Enduring" NOT "changing."

  ⚠️ DANGEROUS: looks "adapted" from outside.
    → Parent: "they've settled down, they're studying now" → REALITY: reward system CLOSED.
    → Long-term: anhedonia risk, depression trajectory.

  🟢 Maier & Seligman 1967/2016: repeated suppress → passivity.
  🟡 "Compiled suppress" as meta-pattern = framework concept.
```

### §4.3 — Outcome C: PFC Failure → Revert + Conflict

```
⭐ HOLD + SUPPRESS → PFC DEPLETED → PATTERN BREAKS THROUGH:

  "PFC gave up, fought with mom, stormed out."

  MECHANISM:
    ① PFC HOLDS + SUPPRESSES → DOUBLE drain.
    ② Old compiled pattern EXTREMELY STRONG (deep + years).
    ③ PFC processing capacity = FINITE → load exceeds capacity.
       (Serial bottleneck saturated + catecholamine decline + allocation shift)
    ④ Old pattern BREAKS THROUGH: emotional burst.
       → Efference mismatch SUDDENLY RELEASED.
       → = WHY suppressed emotions come out as BURSTS.
    ⑤ ADDITIONALLY: negative valence COMPILES into context.
       → "Studying math" is now linked to conflict → suppress even HARDER next time.
       → NEGATIVE SPIRAL: pressure → burst → conflict → pressure harder.

  🟢 Kurzban 2013: subjective effort = opportunity cost signal → PFC reallocates.
  🟢 Arnsten 2009: prolonged PFC use → catecholamine decline → sustained firing degrades.
```

### §4.4 — Distinguishing the 3 States

```
⭐ 3 STATES FUNDAMENTALLY DIFFERENT:

  ┌────────────────┬───────────────────────────────────────┐
  │ GENUINE CHANGE │ New compiled. Old DECAYED.             │
  │ (Outcome A)    │ Body PREFERS new. No suppress needed.  │
  │                │ Cost: 0. Sustainable. Healthy.          │
  ├────────────────┼───────────────────────────────────────┤
  │ COMPILED       │ Suppress compiled. Old BURIED.         │
  │ SUPPRESS       │ Body NEUTRAL (flat). Auto-suppress.    │
  │ (Outcome B)    │ Cost: low. Reward system dampened.     │
  │                │ "Functional but joyless."               │
  ├────────────────┼───────────────────────────────────────┤
  │ ACTIVE         │ PFC still fighting. Old still FIRING.  │
  │ SUPPRESS       │ Body DISTRESSED (efference mismatch).  │
  │ (pre-outcome)  │ Cost: HIGH. Unsustainable.             │
  │                │ Will become A, B, or C.                 │
  └────────────────┴───────────────────────────────────────┘

  ⚠️ Common confusion:
    Compiled suppress LOOKS LIKE genuine change from outside.
    Distinguish by body response to trigger (not verbal report).
    Genuine: trigger → body neutral/positive (truly moved on).
    Compiled suppress: trigger → body spike THEN auto-suppress → flat.
    → Milliseconds genuine response BEFORE suppress kicks in = diagnostic window.

  🟡 3-state distinction = framework synthesis.
  🟢 Components: Maier 2016, Wegner 1987, Kurzban 2013.
```

### §5.2 — Compile-Time Direction Lock

```
⭐ LINK PATTERN FORMS AT COMPILE — STRONGLY PERSISTENT, NOT PERMANENT:

  Chunks compile at the MOMENT cortisol fires.
  Cortisol AMPLIFIES link formation in the direction of body state at that moment.
  Positive emotional links (Type 1) OR negative emotional links (Type 1) form.
  = COMPILE-TIME variable — after compiling, NET link pattern is strongly persistent.

  Student A (chose freely, from genuine interest):
    → Moderate cortisol + dopamine + opioid preview.
    → Math chunks compile with positive Type 1 links.
    → 20 years later: "I love math — use it freely."

  Student B (forced, under threat):
    → Moderate cortisol (SAME LEVEL) + NE + adrenaline.
    → Math chunks compile with negative Type 1 links.
    → 20 years later: "I'm good at math but HATE opening a math textbook."

  → SAME content. SAME cortisol level. DIFFERENT direction.
  → DIFFERENT NET link pattern → DIFFERENT across an entire lifetime.
  → = "Skilled but hate it" = threat-compiled chunks (very common).

  ⭐ 'Compile-time lock' = STRONG PERSISTENCE, not permanence (Drill v4.0 §4):
    No binary tag at neural level. Direction = NET of Type 1 links.
    Links CAN shift via competitive re-linking (Chunk.md v3.1 §4.3).
    But: high emotional weight at compile (factor ③) → link VERY strong → LOOKS permanent.
    7-factor link strength explains why pattern persists without needing a "tag."

  🟢 Cortisol direction × chunk compile: Cortisol-Baseline v2.4 §7.2-§7.3.
  🟡 "Compile-time lock" as strongly persistent direction = framework synthesis (Drill v4.0 §4.5).
```

### §5.3 — Quality × Self-Pattern-Modeling Capacity

```
⭐ COMPILED QUALITY DETERMINES Self-Pattern-Modeling QUALITY:

  GENUINE-COMPILED about an entity:
    → Body TRULY experienced it → multi-sensory chunks.
    → Self-Pattern-Modeling model RICH: knows MANY dimensions → CAN expand → predict novel.
    → E.g.: Mom genuinely loves cooking soup → notices child prefers "chewy" texture
      → tries new variations = creative Self-Pattern-Modeling.

  SCHEMA-COMPILED about an entity:
    → PFC logic "knows child likes soup" → single-channel chunk.
    → Self-Pattern-Modeling model NARROW: knows ONLY WHAT IS KNOWN → CANNOT expand.
    → E.g.: Mom cooks because she knows child likes it → makes only 1-2 known dishes.

  THREAT-COMPILED about an entity:
    → Negative-linked chunks → Self-Pattern-Modeling bias NEGATIVE.
    → Self-Pattern-Modeling predicts THREAT from entity → defensive → misses positive.
    → E.g.: Student forced to study math → Self-Pattern-Modeling predicts "math = uncomfortable"
      → even a fun math puzzle → body still avoidance.

  🟡 Quality × Self-Pattern-Modeling capacity = framework synthesis.
     Consistent with: expert intuition rich (genuine) vs
     rote knowledge narrow (schema) vs trauma avoidance (threat).
```

## §6 — B vs C DETERMINANT: 7 FACTORS

> When Hold + Suppress occurs, WHAT determines Outcome B vs C?

```
⭐ 7 FACTORS:

  FACTOR 1 — OLD PATTERN: Depth × Density (Background-Pattern 2D model)
    High Density → tends toward B (compile "suppress everywhere").
    High Depth + Low Density → B OR C (depends on other factors).

  FACTOR 2 — NEW PATTERN: Compiled Quality
    Genuine (curiosity) → Hold cost LOW → tends toward A (real shift).
    Schema (obligation) → Hold cost MODERATE → tends toward B.
    Threat (forced) → Hold cost HIGH → tends toward C or B if sustained.

  FACTOR 3 — PFC BUDGET (individual variation)
    🟢 PFC volume + connectivity = genetically variable.
    Large budget → can achieve B (enough capacity for compiled suppress).
    Small budget → tends toward C (depleted early → burst).
    Budget DECREASES with chronic cortisol (🟢 McEwen 2007).

  FACTOR 4 — EXTERNAL vs INTERNAL DRIVE (vmPFC/DRN)
    Internal (vmPFC robust) → tends toward A if genuine, B if schema.
    External (DRN default) → tends toward C ("futile" → gives up early).
    🟢 Maier & Seligman 2016: controllability = vmPFC LEARNED.

  FACTOR 5 — DURATION + CONSISTENCY
    Short suppress (days-weeks) → active suppress → C if fails.
    Long + consistent (months-years) → Hebbian compiles suppress → B.
    Inconsistent (on-off) → continuous C ("burst → suppress → burst").

  FACTOR 6 — SLEEP QUALITY
    Good sleep → emotional decoupling → easier path to A.
    Poor sleep → PFC budget DOES NOT recover → easier path to C.
    🟢 Walker 2017: REM emotional decoupling.

  FACTOR 7 — SOCIAL ENVIRONMENT
    Supportive agent present → cortisol lower → PFC budget increases → easier path to A.
    Hostile/alone → PFC drains faster → B or C.
    🟢 Coan & Sbarra 2015: Social Baseline Theory.


  SUMMARY:
  ┌──────────────────────────────────────────────────────────────┐

### §8.3 — PFC Cost Mechanism + Ego Depletion: Framework Position

```
⭐ GLUCOSE MODEL FALSIFIED — FRAMEWORK USES PROCESSING LOAD:

  EGO DEPLETION DEBATE:
    🟢 Baumeister 1998: self-control = limited resource.
    ❌ Gailliot & Baumeister 2007: glucose depletion → self-control fails.
       → FALSIFIED: Hagger et al. 2016 (23 labs, N=2,141, d=0.04).
       → FALSIFIED: Dang et al. 2021 (N=3,531, near-zero effect).
       → Physiologically implausible: brain = ~5.6mg glucose/min,
         homeostasis maintains blood glucose, 10-min task CANNOT deplete.
    🟢 Inzlicht, Schmeichel & Macrae 2014: motivational SHIFT, not depletion.

  PFC COST = PROCESSING LOAD (3 established mechanisms replace glucose):

    MECHANISM A — SERIAL BOTTLENECK (architectural):
      PFC uses shared representations (overlapping neural populations).
      → Shared representations = CANNOT parallel process.
      → 2 tasks sharing ANY representation → interference if simultaneous.
      → PFC MUST process SERIALLY — hardware constraint, PERMANENT.
      → Compiled paths → basal ganglia, cerebellum → CAN parallel → cost ≈ 0.
      → Fresh paths → PFC serial bottleneck → cost > 0.
      🟢 Musslick & Cohen 2021 (Trends in Cognitive Sciences).
      🟢 Petri, Musslick et al. 2021 (Nature Physics).

    MECHANISM B — CATECHOLAMINE DEMAND (neuromodulatory):
      PFC sustained firing REQUIRES dopamine (D1) + norepinephrine (α2A).
      → Inverted-U: too little OR too much → PFC impaired.
      → Prolonged use → catecholamine levels DECLINE → performance degrades.
      → Sleep RESTORES catecholamine levels → "budget recharges."
      → Stress → excess NE → PFC goes OFFLINE (Arnsten 2009: already in framework).
      🟢 Arnsten 2009 (Nature Reviews Neuroscience).
      🟢 Westbrook & Braver 2016 (Neuron — "Dopamine Does Double Duty").

    MECHANISM C — ALLOCATION COMPETITION (computational):
      PFC = shared hardware for ALL cognitive control tasks.
      → Using PFC for task A = UNAVAILABLE for task B.
      → Subjective "effort" = brain's signal: "current allocation may not be optimal."
      → dACC computes: EVC = (payoff × probability) − cost of control.
      → Sufficient incentive CAN restore "depleted" performance → not true exhaustion.
      🟢 Kurzban, Duckworth & Kable 2013 (Behavioral and Brain Sciences).
      🟢 Shenhav, Botvinick & Cohen 2017 (Annual Review of Neuroscience).

  FRAMEWORK POSITION (strengthened by new models):
    ① PFC processing capacity = FINITE → 🟢 established (A+B+C above).
    ② Hold + Suppress cost processing load → 🟢 established.
    ③ PFC capacity varies per person (hardware) → 🟢 established.
    ④ Chronic cortisol → PFC structural damage → 🟢 established.

    FRAMEWORK ADDS:
    → Compiled = ZERO ongoing PFC cost (bypasses serial bottleneck).
    → Active suppress = DOES cost PFC (occupies bottleneck + catecholamine drain).
    → COMPILATION LEVEL determines cost, not "willpower."
    → Someone with compiled habits → LESS processing load.
    → Someone with active suppress → MORE processing load.
    → = "Ego depletion debate misses: compilation level matters."
    → Processing load model explains WHY motivation modulates "depletion":
      incentive → dACC recalculates EVC → reallocate PFC → performance restored.

  🟡 Framework position on ego depletion = synthesis.
  🟢 3 mechanisms (A+B+C) = individually established neuroscience.
```
