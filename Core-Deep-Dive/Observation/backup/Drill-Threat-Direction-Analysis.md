# Drill — Threat Direction Analysis: Threat Does NOT Always Equal Avoidance

> **The brain is not a computer.**
> **It does not store binary labels on data.**
> **There is no "approach tag" or "avoidance tag" in neural architecture.**
>
> **What exists:**
> **Content chunks (math, code, gardening) = NEUTRAL. Like tools in a drawer.**
> **Emotional chunks (entities, experiences, body-states) = CARRY valence.**
> **Links between them (Type 1: neurons co-fired → co-wired).**
> **Spreading activation: content fires → links propagate → emotional chunks fire.**
> **NET valence = sum of all emotional contributions → approach or avoidance.**
>
> **What we OBSERVE: "this domain usually → avoidance."**
> **What we CALL it: "avoidance-tagged."**
> **What we SHOULD call it: observation parameter for a NET pattern.**
>
> **And the second insight: "bounded" vs "unbounded" is ALSO an observation parameter.**
> **P(resolution) = a continuous spectrum, not 2 categories.**
> **Same behavior under threat: succeed → "persistent." Fail → "stubborn."**
> **The label is retrospective. The mechanism is identical.**

## §4 — THE CORE: NO TAG EXISTS — ONLY NEURAL GEOGRAPHY + LINKS + NET

```
⭐⭐⭐ v3.1 DEEPEST INSIGHT: NO TAG, AND NO "2 TYPES OF CHUNKS" EITHER.


§4.1 — All chunks = same substrate. No types. Only neural geography.

  v3.0 distinguished "neutral chunks" vs "emotional chunks" as 2 categories.
  v3.1 CORRECTS: there are NOT 2 types. All chunks = same substrate (Hebb).

  The difference = NEURAL GEOGRAPHY:

  ┌─────────────────────────────────────────────────────────────────┐
  │                                                                 │
  │  MATH PROCESSING (intraparietal sulcus, IPS):                   │
  │    Zone B parietal — physically FAR from amygdala (Zone C)      │
  │    Activation pattern: relatively LOCAL (3-4 regions)           │
  │    Connections to limbic: FEW, INDIRECT                         │
  │    → Naturally LESS emotional coloring                          │
  │    → Can be used across MANY contexts without triggering emotion │
  │    → 🟢 IPS math specialization (Cantlon et al. 2006, Dehaene) │
  │                                                                 │
  │  ENTITY PROCESSING (temporal pole + amygdala + mPFC + insula):  │
  │    Zone B + C — OVERLAPPING with amygdala/limbic                │
  │    Activation pattern: DISTRIBUTED (6+ regions)                 │
  │    Connections to limbic: DIRECT, INHERENT                      │
  │    → Naturally HIGH emotional coloring                          │
  │    → Face recognition → amygdala fires in 12ms (subcortical)    │
  │    → 🟢 Social cognition network overlaps amygdala              │
  │                                                                 │
  │  NOT 2 TYPES — a GRADIENT:                                      │
  │    IPS (far) ←────── proximity to amygdala ──────→ FFA (close) │
  │    Less emotional ←────── emotional coloring ──────→ More       │
  │    Math, syntax ←─────── examples ───────→ Faces, entities      │
  │                                                                 │
  │  Motor cortex, Broca's area, V1 = various positions on gradient │
  │  ALL = same substrate (Hebb). Difference = LOCATION.            │
  │                                                                 │
  └─────────────────────────────────────────────────────────────────┘

  WHY THIS MATTERS:
    → "Math is neutral" = NOT because math chunks ARE a different type
    → "Math is neutral" = because IPS is far from amygdala
    → "Mother is emotional" = NOT because entity chunks ARE emotional
    → "Mother is emotional" = because entity processing overlaps limbic
    → Same substrate, different geography → different emotional coloring

  ⭐ WHY v3.0's "2 categories" WAS MISLEADING:
    → Presented neutral/emotional as an INTRINSIC PROPERTY of the chunk
    → But it's a GEOGRAPHIC PROPERTY of where in the brain it compiles
    → A chunk compiled in IPS = far from amygdala = usually neutral
    → A chunk compiled in temporal pole = overlapping amygdala = usually emotional
    → "Usually" because amygdala modulation CAN bridge the gap (§4.2)


§4.2 — Amygdala modulation (Richter-Levin): how emotion BRIDGES distance

  ⭐ HOW DOES MATH BECOME "SCARY" IF IPS IS FAR FROM AMYGDALA?

  Richter-Levin emotional tagging hypothesis:
    → Amygdala activation MODULATES synaptic plasticity in OTHER brain regions
    → When amygdala is strongly active → it ENHANCES LTP at whatever synapse
      is currently firing, even in DISTANT regions
    → 🟢 Richter-Levin 2004; Roozendaal 2009

  MECHANISM:
    → Normal math: IPS fires, amygdala quiet → no bridge → neutral
    → Teacher screaming during math: IPS fires, amygdala FIRES STRONGLY
      → Richter-Levin: amygdala modulation reaches IPS
      → ENHANCES links between IPS neurons (math) and limbic neurons (fear)
      → = Bridge CREATED by amygdala modulation

  AFTER: Math alone, no teacher:
    → IPS fires (math) → enhanced links propagate → amygdala fires
    → Body-state = fear → avoidance behavior
    → = "Math is scary" — IPS content is still neutral,
      but LINKS to amygdala were artificially strengthened by trauma modulation

  WHY ENTITIES DON'T NEED THIS BRIDGE:
    → Entity processing ALREADY overlaps amygdala (same region)
    → No modulation needed — connection is ANATOMICAL
    → = Why gặp mẹ ALWAYS triggers emotion, but math usually doesn't
    → = Neural geography explains the NATURAL difference

  WHY TRAUMA CAN MAKE ANYTHING SCARY:
    → Amygdala modulation is POWERFUL enough to bridge ANY distance
    → Strong enough trauma → even IPS (math), motor cortex (driving),
      Broca's (speaking) can become emotionally linked
    → = "Any domain can become a source of trauma"
    → = But it requires STRONG or PROLONGED amygdala activation
    → = This is why mild pressure ≠ avoidance, but prolonged abuse → avoidance

  GRADIENT OF VULNERABILITY:
    → Chunks processed CLOSE to amygdala: low threshold for emotional linking
      (entities, social contexts, faces → emotional easily)
    → Chunks processed FAR from amygdala: high threshold for emotional linking
      (math, syntax, motor skills → emotional only under strong/prolonged threat)
    → = Explains why bố-mẹ is "always emotional" but toán is "usually fine"


§4.3 — Type 1 links + amygdala modulation = the complete mechanism

  Chunk.md §3.2 — Type 1 = Shared Contamination:
    "2 chunks SHARE neurons → fire into each other"
    "UNCONSCIOUS, automatic, no PFC needed"
    "🟢 Spreading activation (Collins & Loftus 1975)"

  HOW LINKS FORM BETWEEN DISTANT REGIONS:
    → Normally: IPS (math) and amygdala (fear) DON'T co-fire → NO link
    → Under threat: teacher screams → amygdala fires STRONGLY
      → Richter-Levin modulation → ENHANCES synaptic plasticity at IPS
      → IPS neurons (math) and limbic neurons (fear) NOW co-fire
      → Hebb: co-fire = co-wire → Type 1 link FORMS
      → = Amygdala modulation is the MECHANISM that creates Type 1 links
        between anatomically distant regions

  EXAMPLE — child learns math under threat:

    Compile event: teacher screaming while student solves formula.

    What happens neurally:
      A: [quadratic formula] fires in IPS (parietal, far from amygdala)
      B: [teacher angry voice] fires in auditory + amygdala
      C: [shame feeling] fires in insula + amygdala
      D: [heart↑, cortisol↑] fires in autonomic + amygdala
      → Amygdala STRONGLY active → modulates plasticity at IPS
      → Links form: A ←→ B, A ←→ C, A ←→ D (bridges that normally don't exist)

    LATER, when [quadratic formula] fires:
      A fires (IPS) → links propagate (amygdala-enhanced):
        → B (teacher screaming) fires
        → C (shame) fires
        → D (cortisol↑) fires
      → Body-state = negative → avoidance behavior

    → "Avoidance tag on math" = OBSERVATION of this bridge pattern
    → IPS content is geographically neutral — the LINKS bridge the gap
    → Without trauma: these links would NEVER have formed (too far)

  COMPARE — SAME content, different teacher:

    Enthusiastic teacher, student discovers elegant solution.

    What happens neurally:
      A fires (IPS) — same content
      E: [teacher excited] fires in auditory + social brain
      F: [eureka] fires in VTA (dopamine) + opioid hotspots
      G: [curiosity] fires in DMN + anterior insula
      → Amygdala NOT strongly active → modulation MINIMAL at IPS
      → But: VTA/opioid fire → DIFFERENT modulation → positive links form
      → Links: A ←→ E, A ←→ F, A ←→ G

    Later: A fires → E, F, G fire → positive NET → approach

    → SAME IPS content → DIFFERENT links → DIFFERENT NET → DIFFERENT behavior
    → NO tag changed — only links (and the modulation that created them) differ


§4.4 — Entity-Valence = same mechanism (neural geography explains why entities are ALWAYS emotional)

  Entity-Valence (Valence-Propagation.md, Entity-Valence-Dynamics):
    Mother = mixed (cooking:positive, scolding:negative)
    Dog that bit me = negative
    Friend = positive

  WHY ENTITIES ARE ALWAYS EMOTIONAL — neural geography:
    → Entity processing (FFA, temporal pole, mPFC) OVERLAPS amygdala
    → Face recognition → amygdala fires in 12ms (subcortical shortcut)
    → No modulation NEEDED — connection is ANATOMICAL
    → = Entity chunks naturally have emotional coloring (proximity)
    → = This is NOT a "type of chunk" — it's WHERE it processes

  WHY MATH IS USUALLY NEUTRAL — same geography:
    → Math processing (IPS) is FAR from amygdala
    → No anatomical overlap → no automatic emotional coloring
    → Only gets emotional when amygdala modulation BRIDGES the gap (§4.2)

  "Math avoidance" = Entity-Valence of [toxic teacher]
    + context-valence of [school environment]
    + body-state memory of [shame episodes]
    — propagated to [math content] via amygdala-modulated Type 1 links

  THE FRAMEWORK ALREADY UNDERSTANDS ENTITY-VALENCE.
  §4 extends it: ALL "approach/avoidance" = the SAME mechanism
  (chunks + links + proximity + modulation + NET valence).

  ⭐ EXAMPLES — ALL explained by geography + links, no "tag":

    Knife cut hand → [knife visual, temporal] CLOSE to amygdala
      + [nociception] fires simultaneously → link forms EASILY
      = Proximity = low threshold for emotional linking

    Dog bit me → [dog] Entity-Compiled, processing OVERLAPS amygdala
      = Entity-Valence (anatomical overlap = inherent emotional coloring)

    Mom scolds me → [mom] Entity-Compiled: multi-channel, overlaps limbic
      = [scolding channel] negative, [hugging channel] positive
      = Context determines which channel fires → NET determines behavior

    Fell off bike → [biking motor chunks, motor cortex] MODERATE distance
      + [fall pain, amygdala + nociception] → link forms under trauma modulation
      = New positive biking → competing links → probability shift

    Nearly drowned → [water visual] + [drowning, amygdala] → link EXTREME
      = §4.2 factor ③ emotional weight so high → link fires almost always
      = LOOKS like permanent tag. Actually: extremely strong modulated link.

    Bullied at school → [school context] + [shame, amygdala + insula]
      = Context-valence. Different school → context mismatch → links weaker

    Math under toxic teacher → [IPS, far] + [amygdala, strongly active]
      = Richter-Levin bridge → link that NORMALLY wouldn't exist
      = Requires PROLONGED or INTENSE amygdala activation to bridge distance
      = Why mild pressure ≠ math avoidance, but chronic abuse → math avoidance


§4.5 — 7-factor link strength explains why patterns LOOK permanent

  Chunk.md §4.2: link strength = f(7 factors):
    ① REPETITION — abused daily for years → link VERY STRONG
    ② RECENCY — recently triggered → link BOOSTED
    ③ EMOTIONAL WEIGHT — 1 near-drowning outweighs 100 pleasant swims
    ④ MULTI-MODAL — voice + face + pain + shame → more neurons = stronger
    ⑤ CONTEXT MATCH — similar classroom → old links BOOSTED
    ⑥ EMOTIONAL STATE MATCH — currently anxious → fear links BOOSTED
    ⑦ ANCHOR STRENGTH — teacher still in life → link MAINTAINED

  WHY "TAG" SEEMS PERMANENT:
    → Trauma links: ③ high + ① high + ④ high = EXTREMELY STRONG
    → Fire nearly every time content chunk fires
    → OBSERVATION: "always avoidance" → "must be a permanent tag"
    → MECHANISM: link is very strong, not tag is fixed

  WHY IT CAN CHANGE:
    → Competitive re-linking (§4.3): new positive links COMPETE
    → P(new) gradually increases, P(old) decreases
    → "Never delete, only probability shift" — old links persist but weaken
    → Exposure therapy = systematically creating competitive links
    → Reconsolidation (Nader 2000) = faster re-linking within window

## §5 — TWO TIME-SCALES

```
⭐⭐ THREAT CREATES 2 DIFFERENT QUESTIONS AT 2 TIME-SCALES:

  TIME-SCALE 1 — IMMEDIATE: "What do I do RIGHT NOW?" (§2-§3)
    Threat detected → body assesses 6 variables → fight/flight/freeze/shutdown
    → Cortisol = amplifier for whichever direction selected
    → = Response to tiger, fire, scolding parent

  TIME-SCALE 2 — FUTURE: "What happens when these chunks activate LATER?"
    During threat: content chunks compile + emotional chunks compile
    + Type 1 links form between them (co-fire = co-wire)
    + Link strength = f(7 factors, especially ③ emotional weight)

    Later, when content chunk fires:
    → Spreading activation → linked emotional chunks fire
    → NET current valence = sum of all emotional contributions
    → NET positive → approach. NET negative → avoidance.
    → CONTEXT matters: §4.2 factors ⑤ ⑥ = context/emotion match

  WHY THIS DISTINCTION MATTERS:
    → "Compile-time lock" is HALF TRUE:
      TRUE: links formed at compile are permanent (never delete)
      FALSE: behavioral direction is permanent
      (because NET = old links + new links + context → NET can change)

    → Same chunk, different context → different links fire → different NET
    → New positive experiences → new competing links → probability shift
```

## §6.5 — RESOLUTION PROBABILITY SPECTRUM

```
⭐⭐ INSIGHT: "BOUNDED vs UNBOUNDED" IS ALSO AN OBSERVATION PARAMETER

  Cases A-H treated bounded/unbounded as 2 CATEGORIES.
  But examining complex real-world cases reveals: it is a CONTINUOUS SPECTRUM.

  Just as "approach/avoidance tag" turned out to be an observation parameter
  for NET link pattern (§4.7), "bounded/unbounded" is ALSO an observation
  parameter — for the RESOLUTION PROBABILITY of the threat.

  = CONSISTENT WITH FRAMEWORK PHILOSOPHY:
    Schema = observation parameter → ✅
    Drives = observation parameter → ✅
    Approach/Avoidance = observation parameter → ✅ (this drill §4)
    Bounded/Unbounded = observation parameter → ✅ (THIS SECTION)


⭐ CORE CONCEPT: P(resolution)

  P(resolution) = probability that this threat can be resolved
    = f(domain structure, individual resources, time horizon, competitors, luck)

  CONTINUOUS SPECTRUM:
    P(resolution) ≈ 1.0 ──────────────────────────── P(resolution) ≈ 0.0
    "chó đuổi,                                       "công nhân chăm chỉ
     chạy thoát"                                      muốn giàu = mua vé số"

  Examples across the spectrum:
    P(resolution) ≈ 0.95: Chó đuổi → chạy (bounded, immediate, clear endpoint)
    P(resolution) ≈ 0.70: Học đại học → có nghề tốt (sustained effort, open domain)
    P(resolution) ≈ 0.40: Khởi nghiệp restaurant (possible, high failure rate)
    P(resolution) ≈ 0.15: Khởi nghiệp thương mại điện tử (winner-takes-all, few can win)
    P(resolution) ≈ 0.01: Trở thành cầu thủ bóng đá chuyên nghiệp (extreme selection)
    P(resolution) ≈ 0.001: Công nhân → giàu by hard work alone (structurally near-impossible)

  "Bounded" = observation parameter for HIGH P(resolution)
  "Unbounded" = observation parameter for LOW P(resolution)
  Most real threats = SOMEWHERE IN BETWEEN.

  Cases A-H from §6 = POINTS on this spectrum:
    Case A (unbounded parental pressure): P(resolution) ≈ 0.05
      (no direction, no endpoint → P(resolution) structurally very low)
    Case B (bounded parental pressure): P(resolution) ≈ 0.70
      (direction + endpoint + opioid → P(resolution) high and self-resolving)
    Case C (mixed): P(resolution) ≈ 0.3-0.6
      (depends on ratio of positive/negative episodes)
    Case D (burnout): P(resolution) shifted from ~0.6 → ~0.1 over time
      (self-degrading — started bounded, became unbounded)
    Case H (self-created threat): P(resolution) depends on whether 4 conditions met
      (all 4 met → P(resolution) high; missing any → P(resolution) drops sharply)


═══════════════════════════════════════════════════════
DIMENSION 1 — DOMAIN STRUCTURE LIMITS P(resolution)
═══════════════════════════════════════════════════════

  Not all domains allow everyone to succeed.
  P(resolution) is not purely an individual variable —
  the STRUCTURE of the domain imposes an upper bound.

  TYPE A — OPEN DOMAIN:
    "Học lập trình → có việc làm"
    → Many positions, many employers, skill is transferable
    → P(resolution) for each individual ≈ INDEPENDENT of others
    → One person's success ≠ another's failure
    → = High aggregate P(resolution) → most people CAN resolve → bounded-like

  TYPE B — WINNER-TAKES-ALL:
    "Xây dựng nền tảng thương mại điện tử tốt nhất → thống lĩnh thị trường"
    → Domain structure: only 1-2 can win (Shopee vs Lazada vs Tiki)
    → EACH company perceives bounded threat:
      - Has direction ✅ (build the best platform)
      - Has endpoint ✅ (market dominance)
      - Has resource chunks ✅ (funding, talent, technology)
      - Meets all 4 Self-Created Threat conditions ✅
    → But STRUCTURAL aggregate P(resolution): most MUST fail
    → = PERCEIVED bounded ≠ ACTUAL bounded for most participants
    → = P_perceived ≈ 0.5 per company, P_actual ≈ 0.15 for most

  TYPE C — POSITIONAL (relative ranking):
    "Đứng đầu lớp" / "Được thăng chức" / "Vô địch giải đấu"
    → Success defined RELATIVE to others → zero-sum
    → P(resolution) depends not just on own effort but on competitors'
    → Cannot be determined by introspection alone
    → = P(resolution) is STRUCTURAL (what the domain allows), not just INDIVIDUAL

  ⭐ IMPLICATION FOR SELF-CREATED THREAT (§6 Case H):
    The 4 conditions (chain to body-base, Imagine-Final matches reality,
    sufficient resource chunks, bounded) can ALL be met from the individual's
    perspective — and STILL fail because domain structure doesn't allow it.
    → Condition ② "Imagine-Final matches reality" = domain verification REQUIRED
    → But in winner-takes-all: reality = "only 1-2 can win" → most Imagine-Finals
      are STRUCTURALLY misaligned even if individually well-formed
    → = Why domain structure analysis MATTERS before committing to a threat

  🟢 Winner-takes-all markets (Rosen 1981, Frank & Cook 1995)
  🟡 Domain structure × P(resolution) = framework synthesis


═══════════════════════════════════════════════════════
DIMENSION 2 — TIME DEGRADES P(resolution)
═══════════════════════════════════════════════════════

  P(resolution) is NOT static — it can SHIFT over time.

  PATTERN A — SELF-RESOLVING (P(resolution) INCREASES over time):
    t=0:   P(resolution)=0.4 "challenging but possible"
    t=6m:  P(resolution)=0.6 (resource chunks accumulate, positive links form)
    t=12m: P(resolution)=0.8 (approaching mastery, endpoint visible)
    → = Case B bounded: each milestone → opioid → positive links → P(resolution) INCREASES
    → = Bounded threat that BECOMES MORE bounded over time
    → = THE PRODUCTIVE PATTERN

  PATTERN B — SELF-DEGRADING (P(resolution) DECREASES over time):
    t=0:   P(resolution)=0.5 "looks achievable"
    t=6m:  P(resolution)=0.4 (harder than expected, negative links accumulate)
    t=12m: P(resolution)=0.25 (resources depleting, competitors advancing)
    t=18m: P(resolution)=0.10 (TIPPING POINT — §6.1 Case 2)
    t=24m: P(resolution)=0.05 (structurally near-impossible)
    → = Bounded threat that BECOMES UNBOUNDED over time
    → = Imagine-Final Case 2 (DESTRUCTIVE SPIRAL) — now with explicit mechanism
    → = The "kiên trì → cố chấp" transition: same behavior, but P(resolution) crossed threshold

  PATTERN C — STATIC LOW (P(resolution) NEVER improves meaningfully):
    t=any: P(resolution) ≈ 0.01 (structurally near-impossible)
    → "Công nhân chăm chỉ muốn giàu" → no realistic path
    → No amount of effort significantly changes P(resolution)
    → = Genuinely unbounded from the start
    → Effort expended → only negative links accumulate (no mastery opioid)

  ⭐ WHY TIME DEGRADES P(resolution) — the accumulation mechanism:
    Thời gian để resolve threat càng dài:
      → Cortisol cost tích lũy (Neural Wear, chronic damage)
      → Negative links tích lũy (each difficulty → more negative links compile)
      → Positive links KHÔNG tích lũy nếu chưa resolve (no opioid fires yet)
      → Agency perception GIẢM DẦN (repeated effort without result → helplessness)
      → NET ratio shifts: negative/positive → INCREASINGLY NEGATIVE
      → = Avoidance probability INCREASES with time
      → = Bỏ cuộc giữa chừng = avoidance WINS over approach
      → = "Chưa chắc approach mãi sẽ thành công" — CORRECT,
          because P(resolution) may have dropped below resolution threshold

  ⭐ THE SUNK COST TRAP (Pattern B at tipping point):

    As time passes: P(resolution) DROPS but accumulated investment RISES.
    At tipping point, two COMPETING threats emerge:

      Threat-CONTINUE: "tiếp tục → tốn thêm resource, cortisol, time"
        → NET of continuing = negative (more cost, low P(resolution))
      Threat-QUIT: "bỏ cuộc → mất TẤT CẢ đã đầu tư"
        → NET of quitting = ALSO negative (loss of sunk investment)

    Which wins? Threat-QUIT, almost always:
      → Loss aversion ~2× (Kahneman & Tversky 1979):
        loss of $100 invested hurts ~2× more than gaining $100 new
      → = Threat-QUIT feels ~2× stronger than Threat-CONTINUE
      → = Person CONTINUES — not because P(resolution) is high, but because quitting HURTS MORE
      → = APPROACH behavior powered by AVOIDANCE motivation
      → = "Cố chấp" = approach driven by fear of loss, not by genuine resolution prospect

    ESCAPE FROM THE TRAP:
      → Requires PFC override: "this investment is gone regardless of future action"
      → = Rational economic reasoning (sunk cost = irrelevant to future decisions)
      → But body doesn't "reason" — body FEELS the loss → avoidance fires
      → = Why sunk cost trap is so universal: mechanism-level, not reasoning-level

  🟢 Sunk cost fallacy (Arkes & Blumer 1985)
  🟢 Loss aversion ~2× (Kahneman & Tversky 1979)
  🟡 P(resolution) dynamic change over time = framework synthesis


═══════════════════════════════════════════════════════
DIMENSION 3 — PERCEIVED vs ACTUAL P(resolution)
═══════════════════════════════════════════════════════

  PFC ESTIMATES P(resolution) — but the estimate can be WRONG.
  The gap between perceived and actual P(resolution) is a critical variable.

  ① P_perceived ≈ P_actual → CALIBRATED:
    → Accurate assessment of position on the spectrum
    → Makes rational approach/avoidance decisions
    → Self-Created-Threat §6 Calibration: this is the TARGET STATE
    → = RARE — requires domain expertise + honest self-assessment
    → = Imagine-Final-Evaluation.md: Domain Fit × Hardware Fit verified
    → When calibrated: "dừng lại" is rational, not failure

  ② P_perceived >> P_actual → OVERCONFIDENT:
    → Believes bounded when actually unbounded
    → = Imagine-Final MISALIGNED (§6.1 Case 2) — now with explicit P(resolution) mechanism
    → = "Kiên trì" that becomes "cố chấp" — same behavior, P(resolution) was too low
    → Examples:
      - Startup #10 entering saturated market: P_perceived=0.5, P_actual=0.05
      - Aging athlete continuing competition: P_perceived=0.4, P_actual=0.02
      - Person pursuing career mismatch: P_perceived=0.6, P_actual=0.1
    → MECHANISM:
      - PFC selectively processes CONFIRMING evidence (confirmation bias)
      - Sunk cost amplifies commitment (stopping feels like bigger loss)
      - Self-Created-Threat condition ② violated (Imagine-Final ≠ reality)
        BUT the person CANNOT SEE the violation from inside
      - NET of available chunks BIASES PFC toward high P(resolution) estimate
        (only success stories visible, survivor bias)

  ③ P_perceived << P_actual → UNDERCONFIDENT:
    → Believes unbounded when actually bounded
    → = Learned helplessness (Seligman 1967)
    → = Threat-compiled chunks make everything seem more threatening than it is
    → Examples:
      - Student with strong ability but toxic teacher history:
        P_actual=0.7, P_perceived=0.15
      - Programmer with impostor syndrome:
        P_actual=0.6, P_perceived=0.1
      - Child of abusive parents who believes they're worthless:
        P_actual=varies, P_perceived ≈ 0
    → MECHANISM:
      - Strong negative links from past threat experiences →
        NET is negative when evaluating domain →
        PFC underestimates P(resolution) → avoidance of BOUNDED threats
      - = Self-Created-Threat condition ③ PERCEIVED as violated
        (perceives "no resource chunks" when they actually HAVE them)
      - = Threat.md §4 Imposed Adult threat → internalized → fires without authority

  🟢 Confirmation bias (Nickerson 1998)
  🟢 Learned helplessness (Seligman 1967)
  🟢 Survivorship bias (Wald 1943, applied: Taleb)
  🟢 Dunning-Kruger (overconfidence without metacognition) (Kruger & Dunning 1999)
  🟡 Perceived vs actual P(resolution) → 3 states = framework synthesis


═══════════════════════════════════════════════════════
THE "KIÊN TRÌ vs CỐ CHẤP" PARADOX
═══════════════════════════════════════════════════════

  Same behavior. Same threat. Same approach pattern.
  One succeeds → "kiên trì" (persistence). One fails → "cố chấp" (stubbornness).

  What's ACTUALLY different:
    → P_actual (which NEITHER person KNEW in advance)
    → "Kiên trì" = P_actual was sufficient → approach resolved threat → success
    → "Cố chấp" = P_actual was insufficient → approach failed → waste
    → THE LABEL IS RETROSPECTIVE — assigned by observer AFTER outcome known
    → From INSIDE, both people experienced IDENTICAL threat + approach

  From the BODY's perspective:
    → Both people fired the same mechanism:
      threat detected → resource chunks sufficient → approach selected
    → Both experienced same cortisol amplification, same drive
    → Both had similar P_perceived (high enough to justify approach)
    → The ONLY difference was the domain's actual resolution probability
    → = Body did its job correctly in BOTH cases
    → = The "error" (if any) was in P_perceived calibration, not in approach mechanism

  ⭐ FRAMEWORK IMPLICATION:

    The label "bounded" or "unbounded" is ITSELF an observation parameter:
    → Assigned by the OBSERVER (often retrospectively), not intrinsic to the threat
    → What EXISTS: P(resolution) as a continuous, dynamic, uncertain variable
    → What we OBSERVE: "bounded" or "unbounded" → label assigned after the fact
    → = PERFECTLY CONSISTENT with framework philosophy:
      Schema = observation parameter, Tag = observation parameter, Bounded = observation parameter

  ⚠️ PRACTICAL IMPLICATION — 3 QUESTIONS, NOT 1 BINARY:

    OLD: "Is this threat bounded or unbounded?" → binary, misleading
    NEW: 3 questions that matter:

    ① "Where is P(resolution) RIGHT NOW, and is it increasing or decreasing?"
      → Continuous + dynamic → much more informative than binary label
      → Pattern A (self-resolving) vs Pattern B (self-degrading) vs Pattern C (static)

    ② "Is my P_perceived calibrated to P_actual?"
      → Calibration check → requires domain feedback, not just introspection
      → Overconfident → risk of cố chấp trap
      → Underconfident → risk of learned helplessness

    ③ "Does the DOMAIN STRUCTURE allow resolution at all?"
      → Structural check → open domain vs winner-takes-all vs positional
      → If winner-takes-all: P(resolution) is structurally capped regardless of effort
      → = Most important question, most often SKIPPED

  🟡 "Kiên trì vs cố chấp" paradox = framework analysis
  🟡 3 practical questions = framework synthesis
  🟡 Bounded/unbounded as observation parameter = framework extension
  🔴 Exact P(resolution) values per scenario = illustrative only (not measurable)
  🔴 Method to calibrate P_perceived → P_actual = undefined (domain-specific)
