## §1 — 2 CORE INSIGHTS TO PROPAGATE

```
⭐ INSIGHT 1 — NO TAG AT NEURAL LEVEL (Drill v3.1 §4 + §7):

  OLD MODEL:
    → Threat → chunks get "avoidance tag" (binary, permanent)
    → Cortisol = direction selector
    → "Compile-time lock" = permanent

  NEW MODEL (v3.1):
    → ALL chunks = same Hebbian substrate (no types, no tags)
    → Neural geography = proximity to amygdala → emotional gradient
    → Amygdala modulation (Richter-Levin) bridges distance when active
    → Type 1 links form between co-firing regions (co-fired = co-wired)
    → Link strength = f(7 factors) — explains persistence + change
    → Spreading activation → NET valence = sum of all contributions → direction
    → "Approach/avoidance tag" = observation parameter for NET pattern
    → Cortisol = amplifier (BOTH directions), not direction selector
    → Direction CAN change (competitive re-linking, Nader 2000)


⭐ INSIGHT 2 — BOUNDED/UNBOUNDED = ALSO OBSERVATION PARAMETER (Drill v4.0 §6.5):

  OLD MODEL:
    → Bounded = threat has direction + endpoint + reward → 2 categories
    → Unbounded = no direction, no endpoint → binary classification

  NEW MODEL (v4.0):
    → P(resolution) = continuous variable from 0 to 1
    → P(resolution) = f(domain structure, individual resources, time, competitors, luck)
    → P(resolution) changes over time:
      - Self-resolving: P(resolution) increases → productive
      - Self-degrading: P(resolution) decreases → tipping point → trapped
      - Static low: P(resolution) never improves → genuinely unbounded
    → P_perceived ≠ P_actual:
      - Overconfident → "cố chấp" trap (Imagine-Final Case 2)
      - Underconfident → learned helplessness (Seligman)
      - Calibrated → rational decisions (rare)
    → "Bounded/unbounded" = observation parameter for P(resolution)
    → The LABEL is often retrospective ("kiên trì" vs "cố chấp")
    → Domain structure caps P(resolution) regardless of individual effort (winner-takes-all)
    → Sunk cost trap = two competing threats → approach by avoidance


  WHAT STAYS THE SAME (both insights):
    → Threat-compiled content usually → negative NET → avoidance OBSERVED
    → Education conclusions: imposed threat → negative links → avoidance = STILL TRUE
    → 3 Compiled Quality categories remain USEFUL as observation labels
    → Bounded pressure → CAN be productive (now: when P(resolution) sufficiently high)
    → The PATTERN described is correct — MECHANISM explanation refined + deepened
```
