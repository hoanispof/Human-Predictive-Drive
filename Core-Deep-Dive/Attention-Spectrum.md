## 1. Attention Spectrum Là Gì

```
KHÔNG phải "có ADHD" hoặc "không có ADHD" (binary):
MÀ LÀ: dải LIÊN TỤC (spectrum) từ detail → big picture

  2R ←──── 4R ────── 7R ────── ADHD
  Detail   Standard   Big       Extreme
  mọi thứ  chunk vừa  picture   chỉ chunk
  chunk nhỏ           chunk lớn  cực lớn

  → Mỗi người NẰM ở 1 VỊ TRÍ trên spectrum
  → Vị trí = HARDWARE (DRD4 receptor variant) + epigenetics + experience
  → KHÔNG ai ở "đầu tuyệt đối" → ai cũng NẰM GIỮA ở mức nào đó
  → = "Kiểu chú ý" KHÔNG phải "chất lượng chú ý"
  → = 2R KHÔNG "tốt hơn" 7R → KHÁC, KHÔNG hơn/kém

  ⚠️ QUAN TRỌNG:
    "ADHD" trong DSM = chẩn đoán y khoa (có criteria cụ thể)
    "ADHD" trong spectrum = ĐẦU CỰC ĐOAN (threshold cực cao)
    → Framework KHÔNG chẩn đoán ADHD → framework MÔ TẢ spectrum
    → Ai nghi ngờ ADHD → GẶP BÁC SĨ (không dùng framework thay)
```

## 5. Hyperfocus — Advantage Đặc Biệt Của 7R/ADHD

```
🟡 Hyperfocus = threshold CAO → khi vượt → TOÀN BỘ attention DỒN:

  Cơ chế:
    → 4R: nhận chunk nhỏ từ MỌI hướng → attention PHÂN TÁN (một chút)
      → Focus vào task → VẪN nhận signal nhẹ từ chỗ khác → CÓ THỂ distract
    → 7R/ADHD: CHỈ nhận chunk lớn → chunk nhỏ từ chỗ khác = BLOCKED
      → Focus vào task có chunk lớn → noise HOÀN TOÀN bị filter
      → = "Noise cancelling TỰ NHIÊN" → focus THUẦN KHIẾT

  Khi nào hyperfocus xảy ra:
    ① Domain CHO chunk lớn liên tục (game, code, art, sport, music)
    ② Chunks tích lũy ĐỦ trong domain (base knowledge)
    ③ Body-base CÓ connection (thích domain → opioid khi process)
    → ①+②+③ = hyperfocus LOCK → giờ → nhiều giờ → quên ăn ngủ

  Tại sao NGUY HIỂM:
    → 7R/ADHD hyperfocus = schema override body signal MẠNH
    → Vì: body signal = chunk NHỎ → threshold CHẶN
    → "Đói" = body signal NHẸ → 7R KHÔNG nhận → quên ăn
    → "Mệt" = body signal NHẸ → 7R KHÔNG nhận → quên ngủ
    → = "Game 12 giờ quên ăn" = KHÔNG phải "nghiện" → là "body signal bị filter"
    → = Cùng cơ chế "noise cancelling" → block noise VÀ block body signal

  Tại sao ADVANTAGE:
    → 4R focus 4 giờ: good work → nhưng CÓ distractions → output 80%
    → 7R focus 4 giờ: EXCELLENT work → noise blocked → output 100%
    → ADHD focus 4 giờ: PEAK work → EVERYTHING blocked → output 120%
    → = Khi đúng domain → 7R/ADHD VƯỢT 4R vì focus QUALITY cao hơn

  → Kết luận: ADHD ≠ "attention deficit"
  → = "Attention ALLOCATION deficit" — attention CÓ nhưng PHÂN BỔ extreme:
    Chunk nhỏ: ZERO attention (trông giống "thiếu")
    Chunk lớn: ALL attention (hyperfocus)
    → = Tên ĐÚNG HƠN: "Variable Attention" hoặc "Threshold Attention"
```
