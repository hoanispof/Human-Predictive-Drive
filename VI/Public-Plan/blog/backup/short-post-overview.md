# Short Posts: Framework Overview

> Each short post = AUDITION. The blog = FULL PERFORMANCE.
> If the audition fails (no engagement) → framing needs work.
> If the audition passes → traffic to blog → deeper engagement.
>
> KEY DIFFERENCE vs neuro short posts:
> Neuro posts = 1 specific claim (X ≠ Y), question-hook → deep-dive blog.
> Overview posts = framework overview, substance description → overview blog.
>
> Structure (7-component, from plan-overview-blog.md §4.1):
>   ① WHAT (1-2 câu) → ② CONTAINS (2-3 câu) → ③ POSITION (1 câu) →
>   ④ MECHANISM TEASE (2-3 câu, varies per platform) → ⑤ EPISTEMIC (1 câu) →
>   ⑥ LINKS → ⑦ CLOSE (1-2 câu)
>
> Na ná approach: same core message (①②③⑤⑥⑦ identical), only ④ MECHANISM TEASE
> and FORMAT adapted per platform. Details: plan-overview-distribution.md §2.

**Title:** Body evaluates first, prefrontal cortex observes second — a framework connecting neuroscience, psychology, and evolutionary biology

**Body:**

[Epistemic status: hypothesis inviting falsification. Individual findings are established science; proposed connections are new and unvalidated.]

A framework connecting neuroscience, psychology, and evolutionary biology into a single architecture — mapping how the brain's core systems produce behavior, from dopamine signaling to social dynamics.

Fresh processing without a compiled base is expensive *and* unreliable. But compiled doesn't mean correct — untested assumptions feel identical to verified expertise. Domain feedback is the only arbiter.

If the wanting/liking distinction (Berridge) holds at mechanism level, current RL reward signals may conflate two fundamentally different systems.
