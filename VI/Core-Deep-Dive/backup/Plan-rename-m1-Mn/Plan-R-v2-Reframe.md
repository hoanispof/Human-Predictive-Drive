### §1.2 — Kiến trúc mới

```
RESONANCE DECLINE — 2 Forces + 1 Fuel

  2 FORCES (observable, specific mechanism — ĐO ĐƯỢC):

    ① COMPILED-SUPPRESS — tấn công NGUỒN
       Schema → PFC suppress gap riêng → compiled thành thói quen
       Observable: flat affect, lost drives, "không biết mình muốn gì"
       Fix: nhận ra + giải phóng drive riêng
       Research: Muise & Goss 2024, Gottman Type 2
       ĐẶC BIỆT: Accelerate ② + drain ③. LEVERAGE POINT.

    ② REWARD-HABITUATED — hardware adaptation
       VTA Weber-Fechner → same stimulus → less fire
       Observable: "vẫn tốt nhưng bớt exciting"
       Fix: absence/reunion, sensory novelty (context change)
       Research: Bao & Lyubomirsky 2013

  1 FUEL (calibration parameter — threshold-based, KHÔNG đo chính xác):

    NOVELTY ≥ THRESHOLD?
       Bản chất: Nhiên liệu cho ①b growth (compiled by-product anti-habituates)
       Probabilistic: quá nhiều input variables (partner growth, interaction
         patterns, domain breadth, gap landscape, growth rate...)
       Observable: CHỈ output ("đủ mới" hay "hết mới"), KHÔNG inputs riêng lẻ
       Threshold: khác nhau mỗi người, mỗi giai đoạn
       Dưới ngưỡng: ①b plateau + ①a dominates → decline
       Trên ngưỡng: resonance sustained → OK
       Fix: general direction (more novelty) — specifics case-dependent
       Absorbs cũ M3 + M4 (2 lenses cùng 1 phenomenon):
         - Prediction-delta = 0 (cũ M3) = lens prediction
         - Entity-Compiled saturated (cũ M4) = lens knowledge
       Research: Baumeister 1999 (passion=rate of change),
                 Aron 1996/2000 (self-expansion),
                 Tsapelas 2009 (boredom→satisfaction 9-year lag)

  CONTEXTUAL FACTOR:

    GAP DRIFT
       Gap landscapes diverge → direction match probability ↓
       Khác Novelty: ngay cả CÓ novelty, nếu hướng gap xa → không match
       Observable: "chúng tôi không còn chung sở thích/giá trị nữa"
       Affected by: proximity paradox (gần→②↑+①risk↑, xa→match↓)


  INTERACTION MODEL:
    ① accelerate ② + drain ③ (suppress → ít grow → habituation nhanh + novelty cạn)
    Fix ① = LEVERAGE POINT → counter cả ② và ③
    ②+③ independent of each other but compound
    Gap drift = condition change, ảnh hưởng cả 3 gián tiếp


  SATIATION PROFILE MAPPING (Gap-Body-Need §3.2):
    ① Compiled-Suppress → attacks BOTH Tonic AND Generative
    ② Reward-Habituated → Tonic slowly declining (Weber-Fechner)
    Novelty threshold  → Generative alive/dead
    "Chán nhau" = Tonic intact + Generative dead
```

### §1.3 — Phân biệt Force vs Fuel

```
  Tương tự:
    ① = ai đó đổ cát vào động cơ (sabotage → fixable)
    ② = linh kiện mòn tự nhiên (wear → manageable)
    Novelty = hết xăng (fuel depleted → need refuel, cannot measure tank precisely)
    Gap drift = đường thay đổi (route changed → even full tank → wrong direction)

  Tại sao phân biệt QUAN TRỌNG:
    ① và ② = CAN be detected and specifically addressed
    Novelty = CANNOT be precisely measured, only threshold observed
    → Therapist CAN diagnose "bạn đang suppress" (①) or "cùng routine quá" (②)
    → Therapist CANNOT prescribe "bạn cần chính xác X lượng novelty"
    → Therapist CAN observe "novelty stream đủ hay thiếu?" → general direction
```
