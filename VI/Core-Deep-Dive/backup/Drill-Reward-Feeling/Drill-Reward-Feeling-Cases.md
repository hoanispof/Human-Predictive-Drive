### §2.1 — Insight Reward Signal (Kounios & Beeman 2009; Tik et al. 2020)

```
🟢 KOUNIOS & BEEMAN 2009 (Current Directions in Psychological Science):

  METHOD: EEG + fMRI during problem solving (insight vs non-insight)
  
  FINDING:
    → Gamma burst in RIGHT anterior superior temporal gyrus (aSTG)
      at moment of insight
    → ~100ms AFTER: reward signal in ORBITOFRONTAL CORTEX
      (same region: food, social reward, drugs, orgasm)
    → Reward signal ONLY in participants with HIGH reward sensitivity
  
  IMPLICATION:
    → Insight reward = AUTOMATIC (too fast for conscious appraisal)
    → Same reward REGION as sensory pleasures → same opioid system
    → BUT: right aSTG = UNIQUE to insight → additional signal


🟢 TIK, SOREQ, KOUNIOS et al. 2020 (NeuroImage):

  FINDING:
    → Insightful solving activates: VTA, substantia nigra, NAcc,
      thalamus, hippocampus
    → = SAME dopaminergic circuitry as food, sex, drugs
    → Too fast for conscious post-solution appraisal → automatic
    → "Creative cognition is INTRINSICALLY MOTIVATING due to
       this reward signal"
    → Insight-related NAcc reward → drives RISKY decision-making after

  IMPLICATION:
    → Confirms: insight uses SAME reward hardware
    → Explains: why breakthrough → "nothing else will compare" (Wiles)


🟢 ULTRA-HIGH-FIELD fMRI 2018 (Tik et al., PMC6055807):

  FINDING:
    → "Aha!" = qualitatively DISTINCT from pride
    → Not just "happy I solved it" — DIFFERENT kind of positive signal
    → Subcortical (bilateral thalamus, hippocampus, dopaminergic midbrain)

  IMPLICATION:
    → Supports: coherence reward (Profile ②) ≠ social reward (Profile ③)
    → "Not pride" = not about others knowing → about PATTERN itself
```

### §2.7 — Reward Prediction Error (Schultz 1997)

```
🟢 SCHULTZ, DAYAN & MONTAGUE 1997 (Science):

  METHOD: Single-neuron recording in monkey VTA/dopaminergic neurons.
  
  FINDING:
    → VTA fires when reward > expected (positive prediction error)
    → VTA SILENT when reward = expected (fully predicted)
    → VTA DEPRESSES when reward < expected (negative prediction error)
    → = Reward signal = DELTA, not absolute value

  FRAMEWORK MAPPING:
    → H10 P3 (VTA prediction delta) = THIS mechanism
    → Type A habituation = VTA habituation (Schultz confirmed):
      lần 1 chocolate = big delta → lần 50 = zero delta
    → Type B likely BYPASSES VTA (posterior insula path) →
      explains why B does NOT undergo same habituation pattern
    → "Boring but still good" = VTA habituated, receptor intact (A)
    → "NOT boring AND still good" = no VTA needed, receptor persists (B)

  IMPLICATION:
    → Hedonic treadmill = VTA habituation applied to life events
    → = Primarily Type A phenomenon
    → Type B = hedonic FLOOR (resistant to treadmill)
    → "Rich or poor, a hug still feels the same" — B is democratic
```
