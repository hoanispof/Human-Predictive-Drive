## §1 — COMPILED/FRESH SPECTRUM

### §1.1 — Spectrum Definition

```
⭐ TRỤC THẬT BÊN TRONG BODY: COMPILATION LEVEL

  COMPILED ─────────────────────────────────────── FRESH
  (automatic)                                    (PFC draft)
  body-direct                                    PFC-mediated
  cost ≈ 0                                      cost > 0
  "cảm thấy biết"                               "phải nghĩ ra"
  tức thời                                      cần thời gian
  Hebbian reinforced                            mỗi lần = effort mới

  ĐẶC ĐIỂM COMPILED:
    → Fire automatically when triggered
    → Body-feedback direct (confirm/reject immediate)
    → Near-zero PFC cost (path đã tạo sẵn)
    → "Biết" mà KHÔNG CẦN articulate WHY
    → VD: Chef nếm → biết thiếu muối. Einstein → "thấy" lời giải quen.
    → VD: Mẹ hiểu con → decades compiled → "biết con đang buồn" ngay.

  ĐẶC ĐIỂM FRESH:
    → PFC phải DRAFT novel path
    → Body-feedback DELAYED (chưa có path → chưa biết body nói gì)
    → Cost > 0 (glucose, neurotransmitters)
    → "Nghĩ" = conscious effort, CAN articulate steps
    → VD: Babysitter ngày đầu → phải PFC observe + guess child's state.

  SPECTRUM, KHÔNG PHẢI BINARY:
    → Mỗi chunk nằm SOMEWHERE trên spectrum
    → "Bán-compiled": steps 1-3 automatic, step 4 needs fresh draft
    → Expert "partially stuck" = compiled phần quen, fresh phần mới

  🟢 Expert intuition = compiled (Klein 1998, Chase & Simon 1973)
  🟢 Dual process: System 1 ≈ compiled, System 2 ≈ fresh (Kahneman 2011)
  🟡 Compilation level (not content) determines processing = framework
```

### §1.2 — Transition: Fresh → Compiled và Compiled → Fresh

```
⭐ FRESH → COMPILED (Learning):
  Mechanism: Repeat + domain verify → Hebbian strengthen → automatic
  = "Logic → feeling" (cho PERSON ĐÓ, ở DOMAIN ĐÓ)
  Timeline: simple (hours-days) → complex (years-decades)
  🟢 Lally et al. 2010: habit formation average 66 days, range 18-254

⭐ COMPILED → FRESH (Unlearning):
  Mechanism: Disrupted (new context, error, betrayal) → phải FRESH lại
  Cost: HIGH (3-cost model: PFC draft + suppress old + uncertainty)
  = "Thay đổi" = structurally costly, not "lazy"
  = WHY therapy takes time — re-compiling = building NEW paths
```

## §2 — 2 PFC OPERATIONS: HOLD + SUPPRESS

### §2.1 — HOLD = PFC Amplify

```
⭐ HOLD = PFC sustain attention + working memory trên 1 pattern:

  CƠ CHẾ:
    PFC → top-down signal → amplify activation CỦA target pattern
    = "Tập trung vào cái này" / "Giữ trong đầu"

  COST: ① PFC draft (metabolic — glucose consumption)

  ĐẶC TÍNH:
    → Active process — cần PFC LIÊN TỤC maintain
    → Khi PFC release → pattern DROPS (nếu chưa compiled)
    → Lặp đủ → Hebbian compile → pattern self-sustain → PFC release OK
    → Learning trajectory: Hold → compile → automatic

  VD:
    → Học bài mới: PFC hold problem trong working memory
    → Tập thói quen: PFC hold "dậy sớm" intention
    → Tìm hiểu người mới: PFC hold "chú ý lắng nghe"
    → Mẹ observe con teenage: PFC hold "con đang thay đổi, lắng nghe"

  🟢 Baddeley 2003: working memory = PFC sustained attention
  🟢 Miller & Cohen 2001: PFC = top-down attention amplification
  🟢 Curtis & D'Esposito 2003: PFC sustained firing = working memory
```

### §2.2 — SUPPRESS = PFC Inhibit

```
⭐ SUPPRESS = PFC actively BLOCK pattern đang tự fire:

  CƠ CHẾ:
    Pattern đã compiled → triggered bởi context → SELF-FIRE (automatic)
    PFC → inhibitory signal → block output route
    = "Đừng nghĩ về cái này" / "Kìm lại" / "Ép xuống"

  COST: ② Suppress (efference mismatch — "tôi KHÔNG ĐANG LÀ TÔI")

  ĐẶC TÍNH:
    → Active process — cần PFC LIÊN TỤC inhibit
    → Pattern vẫn EXIST (Hebbian: wired = wired) — chỉ BLOCK output
    → Pattern underground → fire khi PFC yếu (stress, fatigue)
    → Efference copy mismatch: body expect → PFC block → dissonance
    → = "Ép bản thân" = body CONFLICT (expected ≠ actual output)

  VD:
    → Thích chơi nhưng mẹ bắt học → suppress play-urge
    → Giận nhưng phải giữ bình tĩnh → suppress anger
    → Sợ chó nhưng cố "không sợ" → suppress fear
    → CEO muốn chơi với con nhưng deadline → suppress family urge

  🟢 Anderson & Green 2001: Think/No-Think paradigm
  🟢 Aron 2007: right inferior frontal gyrus = inhibitory control hub
  🟢 Wegner 1987: ironic process theory — suppress → rebound
```

### §2.3 — HOLD ≠ SUPPRESS

```
⭐ PHÂN BIỆT NỀN TẢNG:

  ┌──────────────┬────────────────────┬────────────────────┐
  │              │ HOLD               │ SUPPRESS            │
  ├──────────────┼────────────────────┼────────────────────┤
  │ Direction    │ BOOST new          │ BLOCK existing      │
  │ Action       │ Amplify activation │ Inhibit output      │
  │ Target       │ New/weak pattern   │ Strong/compiled     │
  │ Cost source  │ ① PFC draft       │ ② Efference mismatch│
  │ Body feels   │ "Effort" (neutral) │ "Not being me"      │
  │ Compile?     │ CAN compile new    │ Cannot compile "not"│
  │ Sustainable  │ Yes (→compile→free)│ No (cost persists)  │
  │ Brain region │ dlPFC, FEF         │ rIFG, vlPFC         │
  └──────────────┴────────────────────┴────────────────────┘

  HOLD → CAN lead to compilation (positive trajectory)
  SUPPRESS → CANNOT compile "absence" (only block, not build)
  → Suppress alone = ALWAYS temporary (PFC budget = finite)
  → Suppress MUST pair with Hold (replacement) to sustain
```

## §5 — COMPILED SUPPRESS = FRAMEWORK CONCEPT

### §5.1 — Definition

```
⭐ COMPILED SUPPRESS = meta-pattern đã compiled:

  NORMAL suppress: PFC actively block → costly → temporary
  COMPILED suppress: suppress pattern ITSELF has been Hebbian compiled
    → [trigger] → [suppress response] fires AUTOMATICALLY
    → Cost ≈ 0 (compiled = automatic)
    → LOOKS like "adapted" (no visible struggle)
    → NHƯNG: underlying pattern STILL EXISTS (buried, not deleted)

  Compiled suppress × therapy:
    → Therapy goal: DETECT compiled suppress → UNCOMPILE it
    → Reactivate buried → feel again → process → genuine change
    → = WHY therapy can feel WORSE before better
    → = BP §10: years, not weeks (for high density patterns)
```
