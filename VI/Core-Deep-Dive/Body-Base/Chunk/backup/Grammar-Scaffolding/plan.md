## §2 — Thread 6 Decomposed (Sub-questions)

Copy + expand from master plan §2.6.

### §2.1 — Core hypothesis questions

**Q6.1** — Grammar categories (noun/verb/adjective/classifier/case/tense/evidentiality/gender) = chunk category taxonomy?

- Evidence path: word-level grammar forces speaker to classify each chunk per grammar rules
- Counter-evidence: grammar varies widely across languages — if it's chunk taxonomy, why so variable?
- Resolution: **Moderate claim** — grammar provides ONE scaffolding (many possible). Each language = distinct scaffolding. Chunks underneath universal.

**Q6.2** — Broca handles grammatical hierarchy → implication cho chunk-chain scaffolding?

- Evidence: Maess 2001 (musical syntax Broca), Fadiga action sequences, Petitto sign language Broca
- Modality §2.0 đã commit: Broca = sequence processing (including but not limited to grammar)
- Drill formalize: Broca's sequential processing = neural substrate cho chunk-chain scaffolding. Grammar = special case of sequence (verbal chunk chain). Music, action, ritual = other cases.

### §2.2 — Vietnamese classifier case study

**Q6.3** — Vietnamese classifier system forces animacy + shape + social classification each time speaker names entity?

- **con** → animate (con người, con chó, con kiến, con sông)
- **cái** → inanimate tool/object (cái bàn, cái bút)
- **chiếc** → individual-of-pair (chiếc giày, chiếc xe)
- **quyển/cuốn** → bound object (quyển sách)
- **ngài/vị** → high-status person
- **tên/thằng** → negative-valence person
- **ông/bà/anh/chị** → relational-age kin-mapped

**Claim**: Vietnamese force classification per noun invocation → scaffolding animacy + shape + social hierarchy as chunk categories.

**Counter**: English uses "the" for all. Still functions. → classifier không strictly necessary, chỉ là 1 taxonomy option.

**Resolution**: Each language forces attention to DIFFERENT distinctions. English forces tense (past/present/future chia mỗi động từ). Turkish forces evidentiality (nguồn tin). Vietnamese forces animacy + shape + social. → Each language = biased scaffolding along different dimensions.

**Sub-question 6.3.1** — Does Vietnamese speaker's VTC binary Object-Agent (Object-Agent.md §2) get reinforced by constant classifier use? Prediction: Vietnamese speakers' Object-Agent chunk taxonomy is faster/more explicit than English speakers?

- Research to cite: Imai & Mazuka (Japanese classifier cognition effects)
- Note: Research limited, mostly on Japanese/Mandarin. Vietnamese-specific research sparse.

### §2.3 — Cross-language evidence

**Q6.4** — Linguistic Relativity moderate — evidence from Boroditsky et al?

Specific studies to reference:
- **Boroditsky 2001** — Mandarin vs English time metaphors (horizontal vs vertical)
- **Boroditsky German/Spanish key/bridge experiment** — grammatical gender → adjective attribution
- **Winawer 2007** — Russian blue (синий/голубой) vs English blue → perceptual discrimination
- **Everett (Pirahã)** — no exact number words → no exact counting ability

**Q6.5** — Thinking for Speaking (Slobin)?

Claim: Language forces speaker to attention to distinctions during speech planning. This selectively activates certain chunks during cognition.

Example: English speaker describing event automatically allocates processing to tense (past/present). Turkish speaker allocates to evidentiality (first-hand vs hearsay).

→ Grammar biases chunk ATTENTION at speech-planning time, not chunk existence.

**Moderate claim**: Language bias chunk access pattern, not chunk ontology.

### §2.4 — Construction Grammar integration

**Q6.6** — Construction Grammar (Goldberg 1995, 2006) — constructions = compiled chunks at higher level?

Construction examples:
- "The X-er the Y-er" (càng X càng Y)
- Ditransitive "give [someone] [something]"
- Resultative "She painted the wall blue"

Claim: Each construction is form-meaning pair = compiled chunk. Speaker learns inventory of constructions, not abstract rules.

**Integration với framework**:
- Construction = schema of chunks (not single chunk)
- Learning language = learning construction inventory
- "Ngữ pháp" ≠ abstract rule set, ≠ universal grammar deep structure → = compiled construction chunks
- **Compatible với framework's Schema.md §2** two-path compile: construction compile qua 2 paths same as any schema

**Q6.7** — Construction frequency → chunk strength?

Cite Tomasello usage-based acquisition: high-frequency constructions compile first, then generalized. Matches Chunk.md §2 repetition compile.

### §2.5 — Counter-hypothesis: Universal Grammar

**Q6.8** — Chomsky Universal Grammar (UG) — grammar innate, not learned scaffolding. Reconcile?

UG claim: Children acquire grammar too fast and with too little input — must be innate. Language-specific grammars derive from universal parameters.

Usage-based/Construction Grammar counter: Children learn from exposure, construction by construction. No universal parameters needed.

**Framework position** (drill resolution):
- Hardware for sequence processing (Broca) = innate (partial UG claim)
- Specific grammar rules = learned scaffolding (Construction Grammar claim)
- "Poverty of stimulus" argument (UG) weakened by corpus studies showing children have more input than Chomsky assumed
- → **Moderate position**: Neural substrate innate, specific grammars learned. Framework lean Construction Grammar + usage-based.

### §2.6 — Framework embedding

**Q6.9** — Grammar-scaffolding hypothesis extends Modality §2.0 how specifically?

Current Modality §2.0 claims:
- Verbal = commentator (narration, not thinking)
- Broca handles sequence + hierarchy
- 4 functions: LABEL, TRANSFER, GHÉP, SEQUENCE
- "Thinking ≠ Language" (Fedorenko)

Grammar-scaffolding extends:
- Grammar = one structured way to implement LABEL + GHÉP + SEQUENCE
- Grammar categories → chunk label taxonomy (NOT chunk ontology)
- Grammar scaffolds chunk-CHAIN formation, not chunk CREATION
- Different languages = different scaffoldings → different attentional biases → different chunk access patterns
- **Chunk itself exists pre-grammar** (trẻ 0-2 tuổi) — grammar is enhancer, not substrate

**Q6.10** — Gaps where grammar doesn't reach?

- Pre-verbal children (0-2) — chunks exist without grammar scaffolding
- Adult vague "right-wrong lờ mờ" — PFC sensing without ability to articulate → chunks firing without verbal anchor → pre-grammatical processing in adult
- Expert intuition (Klein 1998) — compiled chunks fire faster than verbal articulation
- Meditation states — thinking without narration

→ **Grammar covers ~90% of adult conscious chunk-chain operations, but not all**. Framework needs pre-verbal + post-verbal-bypass accounting.
