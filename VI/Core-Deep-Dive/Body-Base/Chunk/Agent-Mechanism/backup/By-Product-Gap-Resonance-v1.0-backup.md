**By-Product-Gap-Resonance** (viết tắt: Resonance, Gap-Resonance) = emergent mutual phenomenon khi 2+ entities' interactions produce sustained by-product match — mỗi bên fill gap CỦA MÌNH qua actions mà output (by-product) match gap direction CỦA BÊN KIA.

```
⭐ RESONANCE = MUTUAL BY-PRODUCT GAP FILL:

  ╔══════════════════════════════════════════════════════════════════════╗
  ║  Entity B fill gap CỦA B → output = by-product.                    ║
  ║  Khi by-product match gap direction CỦA A → A reward.              ║
  ║                                                                     ║
  ║  RESONANCE: A by-product match B AND B by-product match A           ║
  ║    = MUTUAL by-product gap fill → both receive reward               ║
  ║    = Emerge Ở GIAO ĐIỂM, không ở BÊN TRONG ai                     ║
  ║                                                                     ║
  ║  KHÔNG PHẢI resonance:                                              ║
  ║    One-way: chỉ 1 bên receive → parasocial, service                ║
  ║    Anti-match: by-products CONFLICT gap direction → friction        ║
  ║    Tool-use: 1 chiều reward, entity kia = công cụ                  ║
  ╚══════════════════════════════════════════════════════════════════════╝
```

### §1.3 — By-Product Match: THE mechanism

```

🟡 "By-product match" as resonance mechanism = framework synthesis.
```

### §6.5 — Anti-match (worse than no-match)

```
🟡 ANTI-MATCH = BY-PRODUCTS CONFLICT GAP DIRECTION:

  NO-MATCH (neutral): B's output doesn't match A's gap → no reward, no harm
  ANTI-MATCH (negative): B's output ACTIVELY CONFLICTS A's gap direction
    = B's by-products make A's gap WORSE

  VD: CEO đổi mới ↔ nhân viên thích an nhàn:
    CEO by-product = disruption, new demands → HARMS nhân viên's stability gap
    Nhân viên by-product = resist change → BLOCKS CEO's innovation gap
    = MUTUAL ANTI-MATCH: each person's outputs WORSEN the other

  ANTI-MATCH ≠ "không resonance":
    Không resonance = neutral (no reward, no harm)
    Anti-match = NEGATIVE (active friction, energy cost)
    = WORSE than strangers (strangers = neutral, anti-match = costly)

  SELF-SELECTION protects:
    People gravitate AWAY from anti-match naturally
    Only FORCED proximity (work, family, institutional) sustains anti-match
    → Trục ⑤ Constraint = factor quyết định anti-match có kéo dài không
```
