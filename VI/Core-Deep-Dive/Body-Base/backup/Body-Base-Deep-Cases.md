## §0 — METHODOLOGY: Why deep case analysis

### §0.1 The channel identification problem

**Problem statement**: body-base channels are architectural, not verbal.
Many channels operate silently — individual cannot name what they are
missing (channel-lexithymia, parallel to clinical alexithymia). This
creates fundamental data-gathering problem:

- If individual cannot verbalize deficit → media cannot report it
- If media cannot report it → researchers have no documented data
- If researchers have no data → framework cannot describe channel
- Framework cannot describe channel → society cannot preserve it

Result: **channels most silently starved are channels least visible
to analysis**. Framework work requires bypassing this visibility
barrier through data types that don't depend on subject's verbal
self-report.

### §0.2 Extreme single case methodology — what it reveals và its limits

**Body-Base-Example.md approach**: find individuals với ONE channel
dramatically dominant, others starved. Each case = two data points:
what over-fed channel provides + what starved channels would have
provided. Used extreme case methodology for 50 cases.

**What it reveals well**:
- Channels với clear starvation symptoms (burnout, crash, depression)
- Channels với recognizable pattern (food deficit = hunger)
- Compound patterns visible from outside (B1 workaholic = widely
  recognized)

**What it misses**:
- Silent channels không có vocabulary to name
- Channels operating в background without acute symptoms
- Multi-channel compound effects where no single channel dominates
- Variables invisible to both subject and observer
- Historical cases where psychological vocabulary didn't exist yet

### §0.3 Contrast pair methodology — what it reveals và its limits

**Logic**: two individuals sharing surface similarity (same passion,
same lifestyle, same label) but with radically different outcomes
(one thrives, one crashes). The difference = candidate variable to
investigate.

**What it reveals well**:
- Variables invisible to single-case analysis
- Conditions under which same pursuit sustainable vs pathological
- Features that distinguish healthy compound from risky

**What it misses**:
- Variables that differ но aren't discoverable without hypothesis
- Individual variation confounds (the difference might be trait-
  level, not architecture-level)
- Cases where both individuals fail silently in different ways
- Historical contexts too different for clean comparison

### §0.4 Multi-angle triangulation — this file's approach

**Principle**: use MULTIPLE imperfect data types on SAME case.
Convergence across independent data types strengthens claims;
divergence reveals uncertainty.

**The triangulation logic**:
- Each data type has blind spot
- Blind spots differ across data types
- If 5 independent data types point toward same variable, confidence
  elevated
- If data types disagree, mark as tentative
- If only one data type supports claim, flag as single-angle evidence

This is standard scientific methodology in fields where controlled
experiment impossible (historical analysis, clinical case studies,
anthropology, natural ethology).

### §0.5 Ten data types to triangulate

Each deep case should include as many of these as practical:

1. **Retrospective recovery reports**: individual post-crisis sometimes
   articulates what they couldn't during crisis. AA testimony, post-
   therapy reflection, memoirs written after recovery. Past channel-
   lexithymia through hindsight.

2. **Third-party biographical observation**: biographers, partners,
   therapists, family members often see what subject cannot. Long-term
   relationships reveal patterns subject can't perceive. Triangulate
   multiple independent biographers when possible.

3. **Behavioral data over verbal reports**: what person CHOOSES when
   choice available, what they RETURN to when free, what they DO when
   alternative offered. More reliable than what they SAY. Actions
   bypass verbal self-report limitations.

4. **Trajectory / longitudinal data**: watching person over years
   reveals patterns invisible in snapshot. Life-course studies, diary
   evolution, correspondence changes. Documents when architecture
   shifts occurred.

5. **Cross-cultural comparison**: channel that's silent in one culture
   may be explicit in another. If Vietnamese culture explicitly names
   what American culture leaves unnamed, comparison reveals architectural
   feature independent of vocabulary.

6. **Physiological signals**: HRV, sleep metrics, cortisol, inflammation,
   cardiac markers don't depend on verbal report. Body speaks when
   person doesn't. Rarely available для historical cases, но increasingly
   documented for contemporary.

7. **Post-mortem reconstruction**: suicide notes, final letters, death
   circumstances, autopsy reports, estate documents. Final state often
   most revealing. Example: McCandless's note "happiness only real when
   shared."

8. **Contrast pair analysis**: compare với individual similar on most
   dimensions but different outcome. Isolates candidate variables.

9. **Extreme single-case patterns**: cross-reference với Body-Base-
   Example.md case patterns để identify recurring architectural signatures.

10. **Reconstruction from consequences**: what individual reported
    feeling even without naming why (phenomenology), what healed them
    (validates deficit backward), what they avoided when given choice.

**Practical note**: no case will have all 10. Minimum viable deep case
has 3-4 independent data types converging. Richer cases have 5-7+.

### §0.6 Epistemic humility — tentative conclusions OK

**Framework stance**: acknowledging uncertainty is scientifically
superior to false certainty. Each deep case analysis should explicitly
state:

- **What the data supports confidently** (multiple convergent sources)
- **What the data suggests tentatively** (partial triangulation)
- **What remains uncertain** (single-angle evidence, or no data)
- **What would strengthen claim** (what additional data type would help)

"I don't know" is valid conclusion when data insufficient.

### §0.7 Channel lexithymia в framework — making invisible visible

**Observation**: body-base channels most silently starved are those
without cultural vocabulary. Different eras/cultures have different
vocabularies, so different channels are "nameable":

- Modern Western culture explicitly names: work, achievement, status,
  romantic love, individual freedom
- Silent или partially silent: nature exposure, living pattern,
  embedded community, sustained mastery, meaning inheritance
- Some cultures name more: Japanese shokunin (craft dedication),
  Vietnamese hiếu (filial piety), Greek eudaimonia (flourishing),
  Confucian ren (human-heartedness)

Framework hypothesis: channels that appear in some cultures' vocabulary
but not others are often silently starved в non-naming cultures.
Vocabulary presence does não guarantee channel health; vocabulary
absence suggests channel vulnerability to silent starvation.

### §0.8 Relationship với Body-Base-Example.md

**Complementary, not replacement**:
- Body-Base-Example.md: 50 single-extreme cases, surface vocabulary-
  accessible, extensive pattern coverage
- Body-Base-Deep-Cases.md: 12-15 hard cases, multi-angle triangulation,
  cases where single-angle methodology insufficient

**Cross-reference strategy**: each deep case references related
Body-Base-Example.md cases; each Example case references relevant
Deep Case where exists. Mutual reinforcement architecture.

**Combined framework contribution**: both files feed §14-equivalent
refinements back to Body-Base.md Phase C. Deep Cases add:
- Uncertainty-honest claims
- Multi-angle-triangulated variables
- Contrast-pair isolated features
- Cases where Body-Base-Example.md methodology insufficient

### §0.9 Case selection criteria

Deep case qualifies if:

1. **Extensively documented** (multiple independent sources available
   for triangulation)
2. **Architectural interest** — case reveals something Body-Base-
   Example.md methodology missed or treated superficially
3. **Data-rich trajectory** — not just snapshot, longitudinal evidence
4. **Contrast pair available** OR cross-cultural comparator exists
5. **Channel-lexithymia relevance** — case where subject did not fully
   verbalize deficit but data allows reconstruction

Cases excluded:
- Well-covered by Body-Base-Example.md methodology (would just
  duplicate)
- Insufficient documentation for multi-angle analysis
- Private/personal cases (NO PERSONAL INFO rule)
- Contemporary cases where outcome not yet determined

### §0.10 Framework caution — triangulation is not proof

Even multi-angle convergent data does NOT prove channel architecture.
It creates strong candidates. Framework should hold even triangulated
findings с humility — future evidence may revise. Multi-angle is
better than single-angle; neither is laboratory-controlled experiment.

### §1.2 Data type availability checklist

For each case during fill, verify which data types are available:

- [ ] Retrospective reports (subject's own words, post-crisis если applicable)
- [ ] Third-party biographical (2+ independent sources ideal)
- [ ] Behavioral data (documented choices, preferences, returns)
- [ ] Trajectory / longitudinal (multi-year documentation)

- [ ] Phenomenological reports (what subject reported feeling, even without naming)
- [ ] Healing / recovery data (what reversed pattern, if recovered)

**Minimum viable deep case**: 3-4 data types available.
**Rich deep case**: 5-7+ data types available.

### §1.3 Confidence marker logic

Per candidate variable:

- **🟢 Confident**: 3+ independent data angles converge. Convergent
  evidence across types reduces blind spot concern. Strong candidate.

- **🟡 Tentative**: 2 data angles support, others silent or non-
  contradicting. Plausible but single-type blind spot still possible.

- **🔴 Single-angle**: only one data type supports. Worth noting
  but low confidence. Flagged as needing additional evidence.

### §1.4 Case numbering

Use **DC-series** (DC1-DC12+) to distinguish from Body-Base-Example.md
B-series and Feel-Example-Draft E-series. "DC" = Deep Case.

### §1.5 Voice và style guide

- **Analytical**, not narrative
- Separate data angles clearly (each in its own block)
- Explicit uncertainty ("we cannot determine from available data...")
- Multiple candidate variables rather than single causal claim
- Cross-reference dense (other cases, framework sections)
- Honest about methodology limits

## §2 — DC1: Nikola Tesla (với Leonardo contrast) 🟢

**Priority**: TIER 1 — user-flagged priority case

### Why this is a deep case

Nikola Tesla had extreme novelty + meta-inquiry + creative mastery
channels — by surface framework measures, should have thrived như
B25 contemplative or B30 Jiro craftsman (mastery + meaning compound).
Yet trajectory shows progressive decline ending в poverty, isolation,
eccentricity, bird-feeding compulsions, death alone in hotel room
1943. This is framework-puzzling — how does someone với so many
"positive" channels crash?

Single-angle analysis insufficient because:
- Tesla himself had limited self-insight (channel-lexithymia evident)
- Biographers disagree about mechanisms
- Autism/OCD retrospective diagnoses contested
- Trait-level vs architecture-level variables confounded

Contrast pair с **Leonardo da Vinci** helps isolate variables —
similar polymath profile (arguably greater breadth than Tesla),
similar celibacy, similar novelty pursuit, similar "visionary
leaving projects unfinished" pattern, but very different outcome
(sustained thriving until natural death at 67, supported legacy
continued through students).

### Surface description

**Tesla** (1856-1943): Serbian-American inventor born Smiljan, Austrian
Empire (now Croatia). Son of Orthodox priest, family of 5 children.
Engineering education Graz Polytechnic, Prague University (interrupted
multiple times). Worked for Edison Paris office 1882, emigrated to
US 1884. **Peak productive years 1884-1905**: AC polyphase system,
induction motor, Tesla coil, wireless power experiments, Colorado
Springs research 1899-1900, Wardenclyffe Tower 1901-1917.

**Decline years 1905-1943**: Wardenclyffe financial catastrophe 1905,
increasingly grand unrealized projects, loss of laboratories,
progressive poverty, hotel living, eccentric behaviors intensifying.
Died January 7 1943 в Hotel New Yorker room 3327, alone, weight
~142 lbs (skeletal), heart failure. Maid discovered body days after
death. Estate: debts to hotel. Papers seized by US government's
Office of Alien Property same day (WWII era concerns about potential
super-weapon designs).

**Leonardo** (1452-1519): Italian polymath born Vinci (Tuscany).
Illegitimate son of notary. Apprenticed to Verrocchio in Florence
workshop. Court employment Sforza Milan 1482-1499, Borgia brief
1502, various Italian cities. Final years (1516-1519) at French court
of Francis I, honored pensioner. Died natural causes 67 tuổi at
Château du Clos Lucé, surrounded by students и apprentices, legacy
protected by student Francesco Melzi who preserved notebooks and
carried Leonardo's knowledge forward.

Both polymaths, celibate, visionary, many projects left unfinished.
Very different outcomes. Let's triangulate what differed.

### Data types available

- [x] **Retrospective reports**: Tesla's "My Inventions" autobiography
      (1919), extensive interview record 1890s-1943, patent filings,
      letters to backers (Morgan, Westinghouse, Astor). Tesla's self-
      descriptions limited emotionally но rich technically
- [x] **Third-party biographical**: 5+ major independent biographies
      (Cheney 1981, O'Neill 1944, Seifer 1996, Carlson 2013, Hunt
      1964, Jonnes 2003). Triangulable. Carlson is most academically
      rigorous, Seifer most comprehensive, Cheney most accessible
- [x] **Behavioral data**: celibacy documented, hotel-only living,
      refused Nobel Prize (1915 rumor, though disputed), germaphobia,
      dietary rituals, pigeon-feeding compulsion, obsessive counting
      of 3s, extensive evening walks solo
- [x] **Trajectory**: 87 năm of life documented in detail, productive
      peak mapped (1888 induction motor through 1901 Wardenclyffe),
      decline turning points identifiable (1895 lab fire, 1904
      Wardenclyffe financial crisis, 1917 tower demolition)
- [x] **Contrast pair**: Leonardo da Vinci (primary), Thomas Edison
      (secondary — see DC8 slot)
- [x] **Cross-cultural**: Serbian Orthodox Christian cultural
      background transplanted to American Gilded Age capitalist
      individualism — cultural mismatch potential
- [⚠️] **Physiological signals**: limited records, но 1895 mental
      breakdown post-fire documented (months of recovery), final
      weight и autopsy basic
- [x] **Post-mortem reconstruction**: hotel room state, pigeon-care
      final connections, papers seized by FBI, minimal estate
- [x] **Phenomenological**: scattered reports of loneliness, visions,
      ideas coming "from space", late-life self-descriptions
- [-] **Recovery data**: no recovery — continuous decline

**Data richness**: 8-9 data types available. Most richly documented
framework case.

### DATA ANGLE 1: Retrospective reports (Tesla's own words)

**Primary sources**: "My Inventions" autobiography serial 1919, long-
form interviews (Kenneth Swezey 1920s, John O'Neill 1930s, Liberty
magazine 1935 "A Machine to End War"), letters to financial backers.

**What Tesla articulated rõ ràng**:
- Ideas, visions, technical concepts — extremely articulate
- Memory for mathematical detail ("Every step I had taken I was
  able to traverse в mind, I was in absolute command of my vision")
- Philosophical-spiritual speculations (universe as mathematical
  unity, futuristic predictions 100+ năm ahead)
- Confidence в own work historic significance

**What Tesla didn't articulate** (channel-lexithymia evident):
- Emotional state rarely discussed
- Relational needs never framed as need
- Loneliness acknowledged but not как deficit — "I have not time
  для such things"
- Body signals ignored: "I do not care to live long" (while
  simultaneously working compulsively)
- Explicit self-reflection on why isolation increasing missing

**Revealing quotes** (each implies framework variable):
- "Of all things, I liked books best." (reading as substitute
  community — parallel B29 autodidact hermit)
- "I do not think there is any thrill that can go through the
  human heart like that felt by the inventor as he sees some
  creation of the brain unfolding to success" (flow state
  through invention — §4.3.2)
- "Be alone, that is the secret of invention; be alone, that is
  when ideas are born" (rationalizing isolation as productivity
  requirement — possible cause-effect confusion)
- "The scientists of today think deeply instead of clearly. One
  must be sane to think clearly, but one can think deeply and
  be quite insane" (self-awareness of mental health variance
  but not framed clearly)
- "I have energy to spare. What I need is more ideas." (denial
  of body signals)

**Channel-lexithymia assessment**: HIGH. Tesla had extensive vocabulary
for ideas, visions, technical concepts but minimal vocabulary for
relational, bodily, emotional needs. This is itself diagnostic —
the channels не verbalized are the ones we hypothesize starved.

### DATA ANGLE 2: Third-party biographical observation

**Triangulated across 5+ independent biographers**:

**Cheney (1981) "Man Out of Time"** observes:
- Tesla's increasing solitariness post-1900, hotel lifestyle
- Financial naïveté (signed away royalty rights to Westinghouse
  1890s for $216,600 when they should have been hundreds of
  millions)
- Refusal of conventional human relationships while forming
  parasocial ones (pigeons, select nephews)

**Carlson (2013) "Tesla: Inventor of the Electrical Age"**
(most academically rigorous) observes:
- Pattern of ideals > execution: Tesla visioned ultimate, mid-project
  engineering practicalities would frustrate him
- Increasing disconnection from commercial reality as markets
  changed around him
- Loss of relationships with key supporters (Westinghouse, Morgan,
  Astor) due to Tesla's inability to modify vision for commercial
  constraints

**Seifer (1996) "Wizard"** documents:
- Personal relationships extensively — almost all patron/
  professional, rarely personal
- Letters reveal increasing "grand schemes" disconnected from
  practical execution
- Late-life pigeon care compulsion, hotel manager accommodations,
  growing eccentricity

**O'Neill (1944)** — knew Tesla personally last years:
- Reports daily walks, pigeon feeding, visits to hotel lobby
- Tesla introduced O'Neill to "White Pigeon" (specific bird
  he loved)
- Documented direct quote: "I loved that pigeon as a man loves
  a woman, and she loved me"

**Convergent third-party findings**:
- Tesla progressively isolated from all relational contexts
- Had capacity для deep bonds (demonstrated via pigeons, few
  specific people) but applied it in channels não reciprocal
- Financial naïveté was architectural, not occasional — pattern
  of signing away interests, over-investing, refusing practical
  compromises
- Commercial-era shift (industrial America 1910s-1930s) left him
  behind as the "individual inventor" era ended

### DATA ANGLE 4: Trajectory / longitudinal

**Life phases identified**:

**Phase 1 — Formation (1856-1884, ages 0-28)**:
- Orthodox priest's son, educated household
- Catholic school then Polytechnic Graz (gambling episode
  interrupted), Charles University Prague (father's death
  interrupted)
- Budapest Central Telephone Exchange, then Paris Continental
  Edison
- Formation in community context — school, family, professional
  teams
- Channels fed: mastery (education), community (schools, family,
  professional), autonomy developing

**Phase 2 — Rising star (1884-1892, ages 28-36)**:
- Arrived NYC 1884, Edison Machine Works briefly
- Own laboratory 1886, Tesla Electric Light Company
- AC polyphase system 1888 (landmark achievement)
- Westinghouse licensing deal 1888 — financial peak
- Peak public recognition through 1893 World's Columbian
  Exposition Chicago (AC demonstrated on world stage)
- Channels fed: mastery peak, achievement, community (engineers,
  backers, public), financial security
- **Sustained engagement successful at this phase**

**Phase 3 — Peak → first crisis (1892-1895)**:
- Laboratory fire March 1895 destroyed 4 năm work
- Tesla documented mental breakdown following fire, months
  of recovery
- Lost ~\$50,000 equipment + all research notes
- Key moment: first major body-base crisis. Recovered,
  but architecture revealing fragility

**Phase 4 — Second peak + overreach (1896-1905)**:
- Colorado Springs experiments 1899-1900
- Wardenclyffe Tower 1901 (ambitious wireless power project)
- J.P. Morgan funding, then withdrawn
- Wardenclyffe financial failure 1904-1905
- Channels starting to starve: financial insecurity beginning,
  community fragmenting, ambitions outpacing execution capacity

**Phase 5 — Slow decline (1905-1920)**:
- Laboratories lost to creditors
- No permanent workspace
- Hotel living begins (Waldorf-Astoria 1905-1922)
- Grand projects remain unrealized (death rays, wireless power,
  earthquake machines)
- Still producing ideas но diminishing execution
- Communities fragmenting: Westinghouse deceased 1914, Morgan 1913
- Commercial era shift: individual-inventor model replaced by
  corporate labs (GE, Bell, etc. — Tesla architecturally excluded)

**Phase 6 — Accelerating decline (1920-1943)**:
- 1922 moves to cheaper hotels (St. Regis, Hotel Pennsylvania,
  finally Hotel New Yorker)
- Increasing eccentricity, pigeon-feeding compulsion
- Financial dependency on Westinghouse charity, Yugoslavian
  government stipend
- Public perception shifts from "mad genius" to "crazy old man"
- 1931 Time magazine cover restored some dignity but too late
- Progressive physical decline parallel to mental/relational
- Last interview 1943 still optimistic about projects
- Death January 7, 1943, alone, heart failure

**Trajectory reveals**:
- Tesla's architecture was stable during 1884-1895 phase (community,
  mastery, novelty, achievement all fed)
- Crash was gradual, multi-year, not acute
- Key trigger points: 1895 fire (first test), 1905 Wardenclyffe
  failure (crucial inflection), post-1914 patron deaths (community
  accelerating loss)
- NOT born isolated — became isolated progressively as community
  structures eroded
- **This is crucial**: Tesla was NOT innately isolated individual
  — his architecture could have sustained in different context

### DATA ANGLE 5: Contrast pair (Leonardo da Vinci)

**Surface similarities** (to control for):
- Both polymath (science + engineering + art for Leonardo, physics
  + engineering + speculation for Tesla)
- Both visionary ideas ahead of time
- Both celibate (Tesla declared, Leonardo probably homosexual or
  celibate by inclination, not documented heterosexual)
- Both left many projects unfinished
- Both obsessive by temperament
- Both struggled с commercial execution (Leonardo: tardy
  commissions; Tesla: failed ventures)
- Both fed by novelty + mastery + meta-inquiry + flow

**Outcome divergence**:
- Tesla: died 86, alone, poor, skeletal, hotel, pigeons as
  final relational contact
- Leonardo: died 67, surrounded by students, held royal pension,
  legacy protected by successor (Melzi), notebooks preserved

**Variables isolated through difference**:

**Variable A — Patronage system**:

Leonardo operated through **patronage**:
- Medici family Florence (early career)
- Ludovico Sforza Milan 1482-1499 (17 năm stability)
- Cesare Borgia brief military engineer 1502
- Various Florentine commissions
- Pope Leo X Rome brief
- **Francis I France 1516-1519** — provided Château du Clos Lucé
  residence, royal pension, freedom to work without commercial
  pressure, respect. Francis reportedly held Leonardo in his arms
  as Leonardo died

Tesla operated through **entrepreneurship**:
- Edison Machine Works (briefly) — hostile
- Westinghouse licensing 1888 (commercial contract, contested
  royalties)
- J.P. Morgan (patron-like, withdrew)
- Astor family briefly
- Then no stable financial support
- Commercial era required adapting vision to market demand —
  Tesla constitutionally unable

**Framework finding**: patronage provides **channel architecture**:
stable financial security + freedom from market adaptation + embedded
in court community + meaning through serving patron's larger
project. Commercial entrepreneurship provides financial pressure +
market-adaptation requirement + isolated execution responsibility.
Same creative work, radically different supporting architecture.

**Variable B — Workshop and students**:

Leonardo had **workshop tradition**:
- Apprenticed trong Verrocchio workshop (learned by community-
  embedded practice)
- Own workshop throughout career với apprentices (Salaì Giacomo,
  later Melzi, various others)
- Daily community of practice — master-apprentice relationship
  sustained
- **Melzi preserved notebooks, carried Leonardo's legacy forward
  for decades** — transmission loop operational

Tesla had **solo laboratory model**:
- Some staff в rising phase, но temporary assistants, не disciples
- No sustained apprentice relationships
- No successor preserved his work — papers scattered, many lost,
  some seized by government
- Nephew Sava Kosanović tried to preserve some, but minimal
  lineage
- **Transmission loop broken** — what Tesla knew, Tesla took
  with him

**Framework finding**: mastery + community + transmission compound
(B27, B30, B44 pattern) was present for Leonardo, absent for Tesla.
This is the single most robust sustainable architecture в framework
findings. Leonardo had it, Tesla didn't.

**Variable C — Social community**:

Leonardo embedded trong Italian Renaissance humanist network:
- Regular correspondence với scholars
- Participation trong court life (Sforza, French)
- Collaborations и rivalries (Michelangelo, others)
- Recognized as cultural figure during life
- Respect and honors during lifetime

Tesla embedded early (1884-1895 NYC engineering community), then
progressively isolated:
- Early decades: engineering community, patron dinners, public
  lectures
- Post-1905: withdrawn to hotels, selective visits, reduced
  communication
- Post-1920: minimal sustained contact

**Framework finding**: sustained community presence decades matters
as much or more than community intensity in peak years. Leonardo
maintained, Tesla eroded.

**Variable D — Financial architecture**:

Leonardo: patron pension stable (especially final French period)
— freedom from commercial pressure, allowed experimentation, didn't
require project-completion для survival

Tesla: commercial revenue required, project-completion-or-starve
model. Wardenclyffe failure meant financial ruin, not just career
setback.

**Framework finding**: financial structure shapes channel
architecture. Patron/pension models provide **channel-feeding
infrastructure** (security + freedom + community) that commercial
models don't. Not a body-base channel itself but enables body-base
channels to operate.

**Variable E — Aging architecture**:

Leonardo aged into respect — court figure, advisor to king,
honored elder. Mastery increased с age, compatible with aging.

Tesla aged into irrelevance — commercial era moved past him,
public perception shifted from genius to oddball, physical decline
had no replacement role.

**Framework finding**: different life-course architectures have
different aging compatibility. Craft mastery (B27, B30) and
contemplative (B25) age well because wisdom accumulates. Commercial
novelty inventor ages poorly because innovation is youth-biased в
some eras. Leonardo aged as master-honored; Tesla aged as
forgotten-inventor.

**Variable F — Both lost projects but differently**:

Both left massive unfinished work. But Leonardo's notebooks were
PRESERVED (Melzi), allowing posthumous recognition и influence
(published Leonardo's scientific insights centuries later).
Tesla's papers were SEIZED/SCATTERED/LOST. Posthumous recognition
took longer and was more diffuse.

**Framework finding**: transmission loop isn't just в lifetime —
it extends posthumously. Student/legacy management determines
whether unfinished work lives on or disappears.

**Variable G — Not all variables favor Leonardo**:

Honest caveat: Leonardo also had failures, patron conflicts
(fled Milan 1499 during French invasion), unfinished commissions
(famously poor at finishing), tardiness reputation. He wasn't
archetype of integrated perfect thriving. But his architectural
supports (patronage, workshop, community, aging into honor) buffered
his difficulties. Tesla had similar difficulties but no architectural
buffers.

### DATA ANGLE 6: Cross-cultural comparison

Tesla's cultural transplant context:

**Serbian Orthodox heritage** (culture of origin):
- Family-lineage oriented — strong ties across generations
- Orthodox Christian tradition (rich meaning channel structure)
- Rural/small-town embedded communities
- Collective identity importance
- Father Priest Milutin — community center figure
- Tesla's family expected participation, return, lineage
  continuation

**American Gilded Age context** (culture of destination):
- Individualist capitalism
- Commercial innovation required
- Celebrity inventor as role model (Edison)
- Rapid urbanization, broken traditional ties
- Immigrant isolation — far from family support
- Religious vacuum compared to Serbia context

**Cultural mismatch variable**: Tesla's architecture was formed
in Serbian collectivist Orthodox context but operated в American
individualist commercial context. He brought internal structure
designed для one architecture into opposite architecture.

Evidence:
- Tesla maintained Orthodox Christianity privately (never converted,
  occasional letters home reference)
- Never returned to Serbia despite family requests
- Tesla family legacy continued in Serbia while Tesla isolated in
  America — both halves partial
- Serbian government provided some financial support late life
  (stipend from Yugoslavia) — cultural obligation activated

**Framework finding**: cross-cultural transplant removes **collective
architecture infrastructure** that home culture provided. Individual
may not recognize what they're losing because American ideology
frames individualism as freedom, not loss. B14 Vietnamese extended
family architecture — Tesla was culturally from that type, transplanted
to opposite. **This is invisible variable often missed trong
individual-focused analysis**.

existed, direction was pathological, но feeding was attempted.

### CONFIDENT CLAIMS

**C1. Community/transmission architecture absence was primary**:
Tesla's channel architecture lacked the sustained community +
transmission loop that B27 craftsman, B30 Jiro, B28 chess master,
B44 open source all demonstrate as framework's most robust sustainable
pattern. This was the single biggest architectural difference от
Leonardo and from framework's healthy compound signature.

**C2. Commercial era mismatch amplified isolation**: Tesla operated
в 1880s-1900s era when individual inventor model was giving way
to corporate R&D labs. His architecture was designed for earlier
era when lone inventor could sustain. By 1920s, his architecture
was structurally obsolete. Not personal failing — historical
context shift.

**C3. Progressive loss trajectory, не innate**: crucial for
framework. Tesla was NOT "born broken." He had functional
architecture в 1884-1895 phase. Architecture progressively eroded
through financial disasters, patron deaths, commercial era shift,
cultural isolation. This is framework-important — individuals
can have architecture destroyed by environmental changes they
cannot adapt to.

**C4. Channel-lexithymia evident throughout**: Tesla could not
verbalize relational/body/emotional needs despite being extraordinarily
articulate about ideas. The channels not verbalized were the channels
starved. Framework implication: verbal self-report alone would
have missed this entirely. Multi-angle analysis necessary.

**C5. Substitution via parasocial bonds was real feed**: pigeon
relationships activated agent channel partially (B35 parasocial
finding applied). Not sufficient to prevent decline, but evidence
that Tesla's relational capacity was intact — direction was
maladaptive.

## §4 — DC3: Digital nomad vs ancestral tribal nomad (differential analysis) 🟢

**Priority**: TIER 1 — user-flagged, methodological test case

### Why this is a deep case

Not a single-person case but **structural differential analysis**:
modern "digital nomad" phenomenon vs ancestral "tribal nomad"
pattern. Both travel constantly, both engage với nature, both
experience novelty — surface similarity very strong. Yet modern
digital nomad shows documented burnout pattern (B6 Body-Base-
Example.md), while ancestral nomadic peoples thrived for millennia
sustaining multi-generational culture, raising children, reaching
elders. Isolating the variables через systematic comparison is
framework-critical.

This case serves dual function:
1. **Variable isolation**: which architectural features separate
   sustainable nomadism từ pathological?
2. **Methodology test**: if differential analysis works anywhere,
   this case should demonstrate it. Cleanest surface-similarity
   pair in framework.

**Framework stakes**: if nomadism itself is body-base-compatible
(ancestral evidence), then modern nomad burnout reveals what
specific features were stripped — which becomes framework's
clearest list of **what modern "freedom" architectures are
missing** от ancestral counterpart.

### Surface description

**Digital nomad** (contemporary pattern, emerging 2007+ with
remote work technology maturation):
- Individual (often solo, sometimes couple) quits settled job
- Travels continuously across countries, staying weeks-months
  в various locations
- Works remotely online (programming, design, writing, consulting,
  teaching, etc.)
- Uses modern infrastructure: wifi, banks, visa systems, airports,
  AirBnB, co-working spaces, digital nomad hubs (Bali, Chiang
  Mai, Lisbon, Mexico City, Medellín, etc.)
- Community: online forums (Nomad List founded 2014), Facebook
  groups, co-working spaces, temporary meetups
- Documented pattern: **burnout around năm 7-12** per community
  literature (Rolf Potts "Vagabonding" 2003; blog confessionals;
  academic research Reichenberger 2018, Thompson 2019)

**Ancestral tribal nomads** (documented anthropologically):
- **Mongols** (steppe pastoral, documented Marco Polo 13th century
  through contemporary research)
- **San / !Kung Bushmen** (Kalahari Southern Africa, Richard Lee
  research 1960s-present — "The Dobe !Kung" 1979, "Man the
  Hunter" 1968)
- **Hadza** (Tanzania, hunter-gatherers, Hawkes/Hill/O'Connell
  research)
- **Inuit** (Arctic, traditional nomadic seal/caribou following)
- **Bedouin** (Middle East pastoral)
- **Saami/Lapp** (Northern Europe reindeer herding, Ingold
  research)

### DATA ANGLE 1: Retrospective reports (both sides)

**Digital nomad phenomenology** (from burnout blogs, forums,
books):

Common themes trong burnout reports:
- "I've seen enough mountains" (novelty adaptation complete)
- "Everyone I care about is far away"
- "I don't know what I'm building" (meaning instability)
- "I can't make deep friends на road"
- "The places start to blur together"
- "I miss having people who know me"
- "I'm tired of being a stranger everywhere"
- "I realized I was alone даже когда surrounded by other nomads"

**Key quote (Rolf Potts "Vagabonding" later reflections)**:
movement without purpose eventually becomes its own burden. The
initial euphoria của escape fades when displaced from embedded
context.

**Academic sample (Reichenberger 2018)**:
interviews revealed "quest for holistic freedom" becoming unstable
"need for return" after sustained years. Subjects reported
difficulty articulating what they were missing — classic channel-
lexithymia indicator.

**Tribal nomad phenomenology** (from anthropological interviews):

Common themes:
- Travel framed as following (game, seasons, water, ancestors)
  not fleeing
- "Home" concept не tied to place but to people và routes
- Satisfaction of knowing landscape deeply
- Pride in passing knowledge to children
- Security of knowing community over lifetime
- Fear of separation from band, not от places
- Death near family as good outcome, death isolated as tragedy

**Key observation (Lee Marshall !Kung studies)**: !Kung members
showed less anxiety about uncertainty than modern urban populations
despite "harder" life, attributed to embedded social support
system making individual adversity shared burden.

**Framework signal**: phenomenological reports от both sides
describe same activities (movement, nature) với radically different
accompanying emotional architecture. Digital nomads experienced
loneliness/ungroundedness; tribal nomads described connection/
belonging. Activity alone doesn't determine phenomenology.

### DATA ANGLE 2: Third-party / ethnographic observation

**Ethnographic findings on tribal nomads** (triangulated across
multiple independent researchers, multiple cultures):

**Richard Lee on !Kung (Kalahari Bushmen 1963-present)**:
- Bands of 15-60 people, related through complex kinship
- Daily movement of 0-20km based on resource needs
- Sharing ethic extremely strong (Woodburn's "egalitarian
  society" concept)
- Children raised by entire band, multi-adult care
- Elders transmitted knowledge, held ritual roles
- Conflict exists but managed via separation (band fissure)
- Life satisfaction reportedly high

**Marshall Thomas on Bushmen ("The Harmless People" 1959, "The
Old Way" 2006)**: multi-decade longitudinal observation:
- Social organization sustains despite continuous physical
  movement
- "Home" is the band, not location
- No homelessness concept possible because bands always include
  all members
- Life meaningful through embedded role, not individual
  achievement

**Hawkes/Hill/O'Connell on Hadza (Tanzania)**:
- Grandmother hypothesis validated here — older women
  significant food providers for grandchildren
- Multi-generational band supports maternal care (allomothering)
- Sustained demographic stability
- Sexual division of labor но both genders valued

**Ingold on Saami (Lapland reindeer herders)**:
- Animal husbandry tradition spans centuries
- Family bands, siida organization
- Landscape knowledge transmitted generations
- "The perception of the environment" — nomadic peoples
  develop deep place-knowledge через sustained familiarity
  with travel routes

**Anderson on Arctic Siberian Evenki / Yakut**:
- Reindeer-following pattern
- Clan-based social organization
- Shamanism провides meaning framework
- Resisted Soviet sedentarization — pattern sustained by
  internal preference against state pressure

**Mongolian pastoral nomads (Goldstein, various)**:
- Family ger (yurt) units
- Seasonal migration routes traveled по multi-generational
  knowledge
- Extended kinship networks сустайнинг
- Recent disruption from urbanization revealing what's lost

**Convergent pattern across all ethnographies**:
- Group unit is always family/kinship-based
- Always multi-generational (elders + adults + children)
- Always collective decision-making
- Always embedded cosmology/meaning structure
- Always defined roles for each member
- Always knowledge transmission operational
- Movement is functional / seasonal / ancestral, not free-form
- **Never "individual freedom seeking"**

**Digital nomad third-party observation** (Reichenberger,
Thompson, Willment, Jarrahi & Sutherland research):

- Solo or couple (rarely family)
- Community: temporary co-working spaces, online forums,
  surface connections
- Age-segregated: primarily 25-40 demographic
- No elder presence
- No children typically (or children left behind)
- No defined role — identity self-constructed
- No inherited meaning — individual meaning pursuit
- No skill transmission — work для distant employers
- Movement is expressive / experiential
- **"Individual freedom seeking" is defining framing**

**Architectural inversion**: almost every variable reversed от
ancestral nomad pattern. Same physical activity, systematically
inverted social architecture.

### DATA ANGLE 3: Behavioral data

**What digital nomads carry và do**:

- Small backpack (40L-70L) with electronics, clothes, minimal
  possessions
- Primary tools: laptop, phone, passport, credit cards
- Accommodations: AirBnB, hostels, hotels — temporary, impersonal
- Work: remote for employers 1000km+ away
- Meals: cafes, delivery, restaurants — rarely home-cooked
- Social: app-based meetups, co-working lunch, occasional
  events
- Movement decisions: based on cost, visa, wifi, community
  hub presence
- Technology: always online, heavy app use

**What tribal nomads carry và do**:

- Minimal personal possessions (varies by culture but generally
  what can be carried)
- Primary tools: hunting/gathering gear, practical implements,
  ceremonial objects, children, elders (!Kung carry very frail
  members in slings)
- "Accommodations": structures constructed from local materials
  (ger, wigwam, wind-break, etc.) built/dismantled by tribe
- Work: subsistence activity integrated with daily life
- Meals: prepared together, shared ceremonially, gathered/
  hunted by known band members
- Social: constant band presence, shared hearth, ongoing
  conversation, ritual
- Movement decisions: tribal consensus, seasonal, ancestor
  routes, environmental signals
- Technology: tools made from immediate environment, transmitted
  knowledge

**Behavioral signature**:

Digital nomad movement is **detached**: from others, from
production, from place, from community, from ancestors. They
move through contexts без carrying context.

Tribal nomad movement is **embedded**: bringing entire context
(people, relationships, roles, meaning, knowledge) with them.
Place changes, context doesn't.

**Framework: movement itself is neutral. Context-with-movement
vs context-abandoning-movement are architecturally different
activities despite identical surface.**

### DATA ANGLE 4: Trajectory / longitudinal

**Digital nomad burnout timeline** (aggregated từ community
literature):

- **Năm 0-1**: Honeymoon phase. Euphoria of escape, novelty
  maximum, content production high, social media posts
  celebratory
- **Năm 1-3**: Peak sustained engagement. Routine established,
  work stable, travel competent, real explorations happening
- **Năm 3-7**: Subtle shifts. Places starting to feel similar,
  "same bars different countries" phenomenon, loneliness
  beginning sometimes acknowledged. Some pivot to partial
  settling (base + travel)
- **Năm 7-12**: Burnout manifest for most. "Travel fatigue,"
  depression, meaning questioning, return-home fantasies,
  sometimes crash into crisis. Common articulation: "I don't
  know what I'm doing anymore"
- **Năm 12+**: Subset sustains (usually those who integrated
  travel with partner/family/specific purpose); majority have
  settled or substantially modified pattern
- **Return pattern**: those who return to settled life typically
  report rapid healing — "felt at home within months" —
  suggesting channels starved were reversible không permanent

**Tribal nomad trajectory** (multi-generational observation):

- **Individual lifespan**: birth in band, childhood integrated,
  adult contribution, elder respect, death surrounded by
  community
- **Generational continuity**: each generation replaces previous,
  cultural knowledge transmitted, patterns persist
- **Cultural lifespan**: thousands of năm (Aboriginal Australians
  60K+, other cultures multi-millennia)
- **No burnout pattern at individual level**: individuals die,
  get sick, experience hardship — but não "burn out từ nomadism"
- **Cultural collapse only from external disruption**: colonialism,
  sedentarization pressure, disease introduction, land removal
  — NOT from nomadism itself being unsustainable

**Framework signal**: single generation digital nomad crashes
в 7-12 năm; tribal nomads sustain thousands of năm. This is not
small difference. Something structurally distinct is at work.

### DATA ANGLE 5: Contrast pair analysis — variables isolated

**This IS the contrast pair**. Systematic comparison reveals
multiple variable differences. The key methodological finding:
**multiple variables differ simultaneously**, not one clean
isolate. This is important framework finding itself — cannot
blame "one variable" for digital nomad burnout.

**The 10 systematic variables** (refined от outline):

**V1. Family/band/kinship unit structure**
- Digital nomad: solo (~70%), couple (~25%), small family
  (~5%)
- Tribal nomad: entire band 15-60 people, multi-generation,
  kinship-bound
- **Variable**: collective embeddedness, kinship architecture

**V2. Sustained relational continuity**
- Digital nomad: always new people, weeks to months
- Tribal nomad: same people decades, lifetime
- **Variable**: depth of agent channel через continuous bond
  over time (§4.2.3 depth dimension)

**V3. Collective meaning / shared cosmology**
- Digital nomad: "finding myself" / individual pursuit meaning
- Tribal nomad: shared cosmology, mythology, ancestor
  relationships, ritual framework
- **Variable**: inherited vs constructed meaning structure
  (§4.3.3)

**V4. Defined role / function / identity**
- Digital nomad: no cultural role — identity self-constructed,
  effortful maintenance
- Tribal nomad: defined role (hunter, gatherer, healer, elder,
  mother, shaman) from birth through life
- **Variable**: role-based identity stability, cultural
  identity scaffolding

**V5. Elder presence**
- Digital nomad: peer-aged cohort, no elders embedded
- Tribal nomad: multi-generation bands включают elders as
  knowledge-holders, ritual leaders, child-care
- **Variable**: elder wisdom access, multi-generational
  perspective

**V6. Child presence**
- Digital nomad: childless or children left (nuclear family
  separated)
- Tribal nomad: children integral band members, raised
  throughout travel
- **Variable**: generational connection, meaning through
  lineage, care-giving compound

**V7. Skill/craft transmission**
- Digital nomad: work для distant employers, skills not
  transmitted locally
- Tribal nomad: continuous master-apprentice transmission
  (hunting, tracking, herbcraft, navigation, crafts)
- **Variable**: mastery channel transmission loop (§4.3.4)

**V8. Autonomous vs dependent material production**
- Digital nomad: fully dependent on modern infrastructure
  (wifi, supply chains, banks, visa systems, technology)
- Tribal nomad: self-sufficient within band (food production,
  clothing, tools, shelter from local resources)
- **Variable**: material autonomy, meaningful productive work

**V9. Landscape knowledge depth**
- Digital nomad: GPS-dependent, no sustained landscape
  relationship, constant new terrain
- Tribal nomad: deep landscape knowledge across generations
  (water sources, game migrations, seasonal patterns, sacred
  sites, plant locations)
- **Variable**: living pattern channel через embedded
  place-knowledge (§4.2.4) — nature relationship sustained
  despite movement

**V10. Purpose of movement**
- Digital nomad: expressive (freedom, novelty, individual
  experience, lifestyle branding)
- Tribal nomad: functional (game, water, seasons, ancestor
  routes)
- **Variable**: instrumental vs expressive movement,
  meaningful necessity vs discretionary choice

### DATA ANGLE 6: Cross-cultural comparison across tribal cultures

**Finding**: all ancestral nomadic cultures share ~8-10 variables
regardless of specific context (climate, food source, geography).
This is important — suggests architecture, not accident.

**Universal tribal nomad features** (observed across San, Hadza,
Mongols, Inuit, Saami, Bedouin, Maasai, Roma, Tibetan, Aboriginal):

- ✓ Family/kinship band structure (always)
- ✓ Multi-generational (always — elders and children integrated)
- ✓ Collective decision-making (consensus or traditional authority)
- ✓ Defined roles (always — cultural vocabulary defines position)
- ✓ Shared meaning framework (always — mythology, cosmology,
  ancestors)
- ✓ Knowledge transmission (always — apprenticeship operational)
- ✓ Landscape relationship (always — deep familiarity with
  routes/environment)
- ✓ Material self-sufficiency (mostly — some trade networks but
  core independence)

**Culture-specific variables** (differ across tribes):
- Specific cosmology content
- Specific ritual forms
- Specific social stratification (some more egalitarian,
  others more hierarchical)
- Specific kinship structure (patrilineal, matrilineal, etc.)
- Specific material culture

**Framework signal**: the universal features are exactly the
features digital nomad lifestyle STRIPS OUT. Cross-cultural
evidence makes it clear these aren't cultural accidents —
they're **architectural requirements** for sustainable nomadism
from body-base perspective.

### CANDIDATE VARIABLES (multi-angle synthesized)

**🟢 Confident variables** (3+ data angles converge):

**V1. Collective embeddedness absent — solo vs band**
- Angles: behavioral, trajectory, phenomenological, contrast,
  cross-cultural
- Strongest single variable. Digital nomad solo structure
  removes the band-unit that ALL ancestral nomad cultures
  required.

**V2. Sustained relational continuity absent — new people vs
same people decades**
- Angles: behavioral, trajectory, phenomenological, cross-
  cultural
- Agent channel depth requires continuous bonds over years.
  Movement neutralized trong tribal case because people
  moved together. Digital nomad movement destroys continuity.

**V3. Multi-generational integration absent — peer cohort vs
elders+adults+children**
- Angles: behavioral, cross-cultural, ethnographic
- Age segregation is modern industrial feature; tribal nomads
  always integrated ages. Loss of elder knowledge + loss of
  caring for children both affect multiple channels.

**V4. Inherited meaning framework absent — constructed vs
cosmology+ritual+ancestors**
- Angles: phenomenological, ethnographic, trajectory
- Digital nomad must continuously construct meaning individually;
  tribal nomad inherits meaning structurally. Constructed
  meaning is architecturally more fragile, more effortful.

**V5. Defined cultural role absent — self-construction vs
inherited position**
- Angles: ethnographic, behavioral, phenomenological
- Identity scaffolding через cultural role is structural;
  self-constructed identity requires continuous maintenance
  with inherent instability.

**V6. Knowledge/skill transmission absent — remote work vs
master-apprentice**
- Angles: behavioral, cross-cultural, framework-pattern
- Mastery channel transmission loop (framework's strongest
  sustainability variable) is completely absent trong digital
  nomad pattern, operational trong tribal pattern.

**🟡 Tentative variables** (2 data angles):

**V7. Landscape knowledge depth absent**
- Angles: behavioral, ethnographic
- Living pattern channel feeding through sustained place-
  knowledge not available when constantly moving through
  new places без returning.

**V8. Purpose of movement (functional vs expressive)**
- Angles: phenomenological, cross-cultural
- Functional movement (serving band needs) feels different
  phenomenologically от expressive movement (serving
  individual experience). Meaning implication.

**🔴 Low-confidence variables**:

**V9. Material autonomy missing** — not clearly central
- Digital nomads function well with modern infrastructure;
  the dependency is не experienced as deprivation.
- Framework: this variable may not be directly body-base
  relevant, just contextual.

**V10. Specific novelty adaptation** — dopaminergic saturation
- Some evidence of novelty ceiling from extended travel
- But tribal nomads also had continuous novelty; difference
  is novelty-with-context vs novelty-in-isolation
- Framework: not the variable itself, compound issue

### CONFIDENT CLAIMS

**C1. Multi-variable failure, not single-variable**: digital
nomad burnout is compound failure across 6-8 architectural
features simultaneously. Framework warning: "fix the loneliness"
or "build community" are single-variable interventions that
won't solve multi-variable architectural inversion.

**C2. Convergent finding — "collective/embedded/continuity"
features**: all the 6 confident variables cluster around the
same theme: tribal nomadism preserved **collective embedded
continuous context** through movement; digital nomadism strips
this context. Same physical activity, opposite context architecture.

**C3. "Movement" is not itself a body-base channel**: movement
is an activity context. What matters is what OTHER channels
get fed during movement. When channels fed (community, meaning,
transmission, elders, children, role, knowledge, rhythm), movement
sustainable. When channels stripped, movement starves body-base.

**C4. Mastery + community + transmission compound validated
от ancestral evidence**: framework's strongest positive finding
(B27, B30, B44) receives ANCESTRAL empirical support. The most
sustainable human body-base architecture from 200,000 năm
Homo sapiens history is structurally same as Vietnamese craft
village, Jiro Ono shokunin, open source community. This is не
coincidence — it IS the evolved architecture.

**C5. Reversibility confirmed**: digital nomad recovery через
settling is rapid, suggesting architecture damage is not
permanent. Framework: body-base channels restore when inputs
available. Good news для interventions.

**C6. Context can be invisibly architectural**: digital nomads
often don't recognize what they've stripped because modern
individualist ideology frames the stripping as "freedom."
Framework: architectural starvation can be ideologically masked
as freedom, achievement, empowerment. This is channel-lexithymia
at cultural scale.

---

### TENTATIVE CLAIMS

**T1. Individual variation matters**: some digital nomads sustain
longer (typically those traveling with partner/family or embedded
в specific purpose). Architecture fix can be partial if one or
two missing variables restored. Not binary.

**T2. Novelty adaptation vs novelty integrated**: both tribal
and digital nomads experience novelty. Tribal adapted to novelty
within embedded context; digital experiences novelty in isolation.
Difference may not be novelty itself but its architectural
context.

**T3. Not all tribal architectures identical**: some tribal
nomads had more hierarchical, others egalitarian; some more
patrilineal, others matrilineal. Framework: universal
architectural features are 6-8 listed, specific implementations
vary culturally.

---

### REMAINS UNCERTAIN

- Which variable is most critical? All 6-8 cluster together
  normally — hard to isolate one from the others.
- Minimum viable "fix"? Can a digital nomad who travels với
  partner + child + inherited meaning + defined role avoid
  burnout? Probably closer. Insufficient data.
- Cultural variation effect on burnout: do digital nomads
  từ different home cultures experience different patterns?
  Limited data.
- Tribal nomads in contact with modernity: when partial
  modernization begins, do they show partial burnout? Some
  evidence (Mongolia urbanizing) но insufficient.

### FRAMEWORK IMPLICATIONS

**FI1. Collective architecture is framework-level variable**:
Body-Base.md §13 should explicitly model **collective body-
base architectures** alongside individual. This case adds
evidence to B14 Vietnamese extended family и B40 Rothschild
dynasty finding — body-base channels can operate at collective
scale.

**FI2. Ancestral pattern as reference**: framework can cite
ancestral evidence because evolved calibration matches ancestral
conditions (2M năm hominid + 200K năm Homo sapiens). Modern
patterns deviating từ ancestral should face burden of proof
about sustainability, не assumed sustainable until evidence
otherwise.

**FI3. "Freedom" as architectural starvation**: modern
individualist freedom ideology can mask body-base starvation.
Framework critical for AI era: distinguishing real freedom
(options within sustained architecture) từ pseudo-freedom
(stripped architecture framed as liberation).

**FI4. Multi-variable interventions needed**: single-variable
"solutions" (build community! find meaning! define your
role!) fail because variables are compound. Intervention must
address architecture, not symptom.

**FI5. Differential analysis methodology validated**: this
case demonstrates contrast pair can isolate multiple variables
when surface similarity strong. Methodology worth using more.

**FI6. Context vs activity distinction**: framework refinement —
activities (movement, work, sex, eating) are not themselves
body-base channels. What matters is what channels are fed
during activity. Same activity different context = different
body-base outcome.

**FI7. Cross-generational transmission as architectural feature**:
multi-generational bands provide elder wisdom + child care
simultaneously. Age-segregated modern life strips both. Framework
should explicitly include multi-generational integration as
body-base architectural feature.

**FI8. Contemporary reconstruction possible?**: if architectural
features can be deliberately reconstructed in modern context
(digital nomad with family + purpose + tribe + role), then
modernization is not destiny. Parallel B44 open source reconstruction
finding.
