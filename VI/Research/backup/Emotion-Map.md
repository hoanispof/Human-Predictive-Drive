#### 4.1 Yêu (Romantic Love) — Multi-Channel State, Không Phải 1 Cảm Xúc

```
GIAI ĐOẠN 1 — Attraction (rạo rực):
  Channels: Dopamine (wanting) + NE (arousal) + Testosterone/Estrogen
  Brain:    VTA + Nucleus accumbens + body arousal circuits
  Schema:   L1-L2 (surface, fast, person-specific)
  Feel:     "Muốn gần, muốn biết thêm, tim đập nhanh"
  ≈ curiosity + anticipation + body arousal

GIAI ĐOẠN 2 — Infatuation (say đắm):
  Channels: Dopamine HIGH + Opioid (khi gần) + Oxytocin (bonding)
            + Cortisol TĂNG (anxiety: "họ có yêu mình không?")
            + Serotonin GIẢM (→ obsessive thinking, giống OCD!)
  Brain:    Reward circuits MAX + Amygdala (uncertainty) + DMN (think about them)
  Schema:   L3 form ("be with them" = goal)
  Feel:     Overwhelming, khó tập trung, nghĩ về họ liên tục
  → Multi-channel BURST → đây là tại sao "yêu" feels overwhelming

GIAI ĐOẠN 3 — Attachment (yêu lâu dài):
  Channels: Oxytocin + Opioid dominant, Dopamine GIẢM
  Brain:    DMN + Insula (calm presence)
  Schema:   L4-L5 ("người này là một phần cuộc đời tôi")
  Feel:     "Bình an khi bên nhau" thay vì "rạo rực"

⚠️ "Hết yêu" thường = chuyển từ giai đoạn 2 → 3
   mà NHẦM TƯỞNG là hết yêu (vì dopamine giảm = hết rạo rực).
   Thực ra: đổi CHANNEL DOMINANT, không phải hết.
   Dopamine (wanting) giảm, Opioid + Oxytocin (having) tăng.
```

## 7. Patterns Quan Trọng

### Pattern 1 — Lo âu vs Hy vọng = CÙNG CƠ CHẾ, NGƯỢC DẤU

```
Lo âu:   predict PE ÂM trong tương lai  → Amygdala + cortisol
Hy vọng: predict PE DƯƠNG trong tương lai → PFC + dopamine
→ Cùng "prediction about future PE", khác dấu
→ Giống pattern defense (16-20) vs pursuit (21-25) trong 50 schema examples
```

### Pattern 2 — Shame vs Guilt = CÙNG CƠ CHẾ, KHÁC DEPTH

```
Guilt: "Tôi LÀM sai" → L3-L4 (hành vi, sửa được)  → ACC dominant
Shame: "Tôi SAI"      → L4-L5 (identity, khó sửa) → Insula dominant
→ Guilt = schema về behavior bị violated → dẫn tới "sửa hành vi"
→ Shame = schema về SELF bị violated → dẫn tới "thu mình/trốn"
```

### Pattern 3 — Xúc động vs Kính sợ (Awe) = KHÁC CƠ CHẾ

```
Xúc động: Deep schema CONFIRMED → warm, opioid, tears
Awe:      Deep schema OVERWHELMED → schema EXPANDED, chills
          DMN bị SUPPRESS (self tạm mất) → "hòa vào vũ trụ"
→ Xúc động = "thế giới đúng như tôi tin" (confirm)
→ Awe = "thế giới LỚN HƠN tôi tin" (expand)
```

### Pattern 4 — Chán = PE Decay Rate observable

```
Chán xảy ra khi PE ≈ 0 → dopamine thấp → "không có gì mới"
→ Đây chính xác là ③ PE Decay Rate trong framework!
→ Chán = signal rằng chunk hiện tại hết PE
```

### Pattern 5 — Tuyệt vọng = Schema collapse endpoint

```
Tuyệt vọng ≠ buồn nhiều
Tuyệt vọng = KHÔNG CÒN predict PE dương NÀO trong tương lai
→ Dopamine system COLLAPSED (không phải giảm, mà COLLAPSED)
→ PFC reduced activity (không plan vì "plan để làm gì?")
→ Trong framework: TẤT CẢ chunks tương lai đều predicted PE ≈ 0 hoặc âm
→ Đây là endpoint của depression trajectory
```

### Pattern 6 — Cô đơn dùng circuits của PHYSICAL PAIN

```
Eisenberger et al.: Social rejection activates CÙNG circuits
với physical pain (dorsal ACC, anterior Insula)
→ Não KHÔNG phân biệt "bị đánh" và "bị bỏ rơi"
→ Opioid withdrawal (mất connection) = literal pain
→ Đây là tại sao cô đơn CÓ THỂ GIẾT NGƯỜI (mortality risk +26%)
```

### Pattern 7 — "Hết yêu" = đổi channel, không phải mất

```
Giai đoạn 2 (infatuation): Dopamine HIGH → rạo rực, obsess
Giai đoạn 3 (attachment):  Oxytocin + Opioid → bình an, ấm áp

Chuyển 2→3: dopamine giảm → "hết rạo rực" → tưởng "hết yêu"
Thực tế: channel dominant ĐỔI, không phải tắt
→ Serotonin cũng normalized (hết obsessive thinking)
→ "Yêu bình thường" vs "yêu say đắm" = khác channel, cùng attachment
```

## 8. Emotion Spectrums — Gradient, Không Phải Discrete

Hầu hết cảm xúc KHÔNG phải discrete — chúng là **điểm trên spectrum**.

### 8.1 Thương Mến Spectrum

```
Mến ──→ Quý ──→ Thương ──→ Yêu
(L2)    (L3)    (L4)       (L4-L5 + multi-channel)
opioid↑  +sero   +oxy       +dopamine +NE +body

→ Gradient intensity + depth + số channels involved
→ Mến = 1 channel nhẹ, Yêu = multi-channel burst
```

### 8.2 Anticipation Spectrum

```
Hứng thú ──→ Háo hức ──→ Bồn chồn ──→ Lo âu
(PE+ nhẹ)    (PE+ mạnh)   (PE± unclear)  (PE– predicted)
dopamine↑    +NE          +cortisol       cortisol dominant

→ Gradient từ positive certain → negative certain
→ Bồn chồn = transition point (PE chưa rõ dấu)
```

### 8.3 Negative Surprise Spectrum

```
Ngạc nhiên ──→ Thất vọng ──→ Hối hận
(PE lệch)      (PE+ ko đến)   (PE+ đáng ra có)

→ Ngạc nhiên: PE sai hướng (có thể + hoặc –)
→ Thất vọng: PE+ expected nhưng không đến
→ Hối hận: PE+ CÓ THỂ có nếu CHỌN KHÁC → thêm element counterfactual
```

### 8.4 Social Negative Spectrum

```
Ngượng ──→ Tội lỗi ──→ Xấu hổ
(L2, nhẹ)   (L3-L4)     (L4-L5, identity)

→ Gradient depth: situation → behavior → identity
→ Ngượng = "tình huống này awkward" (thoáng qua)
→ Guilt = "tôi làm sai" (sửa được)
→ Shame = "tôi sai" (identity-level)
```

### 8.5 Loss Spectrum

```
Nhớ nhung ──→ Buồn ──→ Cô đơn ──→ Tuyệt vọng
(1 người)     (general)  (no connection) (no future PE)

→ Gradient scope: specific → general → total
→ Nhớ nhung: 1 connection missing
→ Cô đơn: ALL connection missing
→ Tuyệt vọng: all connection + all future PE missing
```

## 10. 6 Chiều Định Nghĩa Không Gian Cảm Xúc

Mỗi cảm xúc = 1 điểm trong không gian 6 chiều:

```
1. KÊNH DOMINANT:  Dopamine / Opioid / Oxytocin / Cortisol / mixed
2. PE DẤU:         Dương (+) / Âm (–) / Hỗn hợp (±) / Zero (0)
3. THỜI GIAN:      Quá khứ / Hiện tại / Tương lai / Toàn bộ
4. SCHEMA DEPTH:   Surface (L1-L2) / Middle (L3-L4) / Deep (L4-L5)
5. CERTAINTY:      Confident / Uncertain / Unknown
6. INTENSITY:      Nhẹ / Vừa / Mạnh / Overwhelming

→ 6 chiều tạo ra KHÔNG GIAN CẢM XÚC LIÊN TỤC
→ Ngôn ngữ chỉ CẮT không gian này thành discrete labels
→ Mỗi ngôn ngữ cắt KHÁC NHAU (tiếng Việt có "thương"
   nhưng English không có exact equivalent)
→ "Schadenfreude" (Đức) = PE dương + khi người khác PE âm
   → tồn tại trong không gian nhưng tiếng Việt không có 1 từ cho nó
```

```
Q1. Emotion regulation = PFC inhibit emotion output,
    hay PFC reframe schema (cognitive reappraisal)?
    → Cả hai: inhibit = suppress (tạm, tốn effort),
      reframe = update schema (lâu dài, nhưng chậm hơn)
    → 2 strategies khác nhau, khác cost/benefit

Q2. "Emotional intelligence" = gì trong framework?
    → Khả năng NHẬN DIỆN readout (Insula sensitivity)?
    → Khả năng PFC LABEL chính xác?
    → Khả năng CHỌN response phù hợp?
    → Hay cả 3?

Q3. Mixed emotions (xúc động = vui+buồn, nostalgia = warm+loss):
    → 2 PE signals chạy SONG SONG trên 2 channels?
    → Hay 1 PE signal phức hợp?
    → Dường như song song: xúc động vừa khóc vừa cười
       = opioid (warm) + loss signal cùng lúc

Q4. Emotion contagion (thấy người khác khóc → mình muốn khóc):
    → Mirror neuron system + Insula?
    → Framework giải thích: chunk của NGƯỜI KHÁC
       trigger PE ở schema "tôi có thể ở vị trí đó"?
    → Hay đơn giản hơn: somatic mimicry → emotional state follows?

    → PFC over-inhibit? → Dissociation
    → Hay Amygdala shutdown (learned helplessness)?
    → Hay cả hai tùy trường hợp?
    → Trong framework: PE system bị "tắt" hay bị "chặn đầu ra"?

    → Trong framework: verbal encoding = 1 modality
       → không có từ ≠ không có cảm xúc
       → nhưng không có từ → khó ARTICULATE → khó PFC process

Q7. Xúc động cho biết L5 của bạn là gì:
    → "Bạn xúc động vì cái gì" = diagnostic tool cho deep schema
    → Có thể dùng systematic: show 20 video → track xúc động
       → map L5 profile?
    → Potential ứng dụng cho therapy: discover hidden L5 beliefs
```
