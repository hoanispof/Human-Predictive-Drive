```

  Education-System.md = HOW: stages, roles, assessment, bridge
  File NÀY = WHAT: dạy CÁI GÌ, khi nào, thứ tự nào

  Tại sao tách:
    → "Dạy gì" THAY ĐỔI per era (content = variable)
    → "Cách dạy" ÍT THAY ĐỔI hơn (mechanism = more stable)
    → = Tách → update content KHÔNG cần rewrite system design
    → = Modular: era đổi → update file NÀY + Era-Analysis → giữ System + Hardware


4 NGUYÊN TẮC CURRICULUM:

  ① FOUNDATION TRƯỚC, SPECIALIZATION SAU (NL2)
     → Foundation domains = MỌI era, MỌI hardware cần
     → Specialization = emerge per-hardware + per-interest, Stage 3+
     → = "Tất cả biết đọc, biết tính, biết giao tiếp → RỒI MỚI chuyên sâu"
     → ≠ "Specialization sớm" (nguy hiểm: build trên foundation yếu)

  ② MECHANISM-BASED DELIVERY (NL1)
     → MỌI content → deliver qua đúng brain mechanism
     → Multi-modal, experiential, practice+sleep, novelty flow
     → = Content KHÔNG quyết định method → method quyết định bởi MECHANISM
     → VD: "Dạy toán" → method = depends on mechanism, NOT on "toán"

  ③ ADAPTABILITY SONG SONG VỚI CONTENT (NL7)
     → Meta-learning ≠ "dạy sau khi dạy content"
     → Meta-learning = SONG SONG → "trong khi học toán, cũng đang học CÁCH HỌC"
     → = Mỗi content domain = vehicle for BOTH domain knowledge + meta-skill

  ④ ALIGN VỚI DEVELOPMENTAL TIMELINE (Education-Principles §2.8)
     → Đúng skill + đúng window = compile dễ + sâu
     → Sai timing = vẫn compile được nhưng tốn effort hơn
     → = Curriculum timing align với PFC development + sensitive periods
```

## 1. TAXONOMY KNOWLEDGE DOMAINS

```
3 TIERS — từ durable đến flexible:

  ┌──────────────────────────────────────────────────────────────┐
  │ TIER 1: FOUNDATION DOMAINS (6 domains)                       │
  │   Cross-era, mọi hardware cần, DURABLE (decades)            │
  │   = "Bất kỳ era nào, bất kỳ ai, đều CẦN"                  │
  ├──────────────────────────────────────────────────────────────┤
  │ TIER 2: ERA-SPECIFIC DOMAINS (4-5 domains)                   │
  │   2025+ specific, TIME-BOUND (update 3-5 năm)               │
  │   = "Era NÀY cần THÊM → era sau CÓ THỂ khác"              │
  ├──────────────────────────────────────────────────────────────┤
  │ TIER 3: PER-HARDWARE DOMAINS (varies per individual)         │
  │   Emerge per-hardware + per-interest, Stage 3+               │
  │   = "NGƯỜI NÀY mạnh CÁI GÌ → đi sâu CÁI ĐÓ"             │
  └──────────────────────────────────────────────────────────────┘


TIER 1 — 6 FOUNDATION DOMAINS (cross-era, durable) 🟡:

  ① LITERACY / COMMUNICATION
     → Đọc, viết, nói, nghe, biểu đạt ý tưởng
     → Tại sao FOUNDATION: mọi domain khác CẦN literacy để ACCESS
     → Dù AI đọc/viết hộ → "HIỂU ý nghĩa" = human skill
     → 🟢 Language processing = core brain function

  ② NUMERACY / LOGIC
     → Toán, logic, pattern recognition, reasoning
     → Tại sao FOUNDATION: logical thinking = SUBSTRATE cho mọi analysis
     → Dù calculator/AI → "HIỂU logic" = khác "nhấn nút"
     → 🟢 Pattern recognition = core brain function

  ③ SOMATIC / PHYSICAL
     → Vận động, coordination, body awareness, hands-on skills
     → Tại sao FOUNDATION: body-base = processing channel (Education-Principles §2.5)
     → Physical health = prerequisite cho brain function
     → ≠ "Giờ thể dục" → = body AS learning channel
     → 🟢 Motor development, interoception = core brain functions

  ④ SOCIAL / EMOTIONAL
     → Empathy, collaboration, emotion regulation, conflict resolution
     → Tại sao FOUNDATION: human = social animal → social skills = SURVIVAL
     → AI era: human connection = DIFFERENTIATOR (AI không thay thế)
     → 🟢 Social brain, mirror neurons, attachment = core

  ⑤ META-LEARNING
     → Learn-how-to-learn, critical thinking, self-assessment, error analysis
     → Tại sao FOUNDATION: era speed → specific knowledge = obsolete nhanh
     → Meta-learning = "kỹ năng TỰ LEARN bất kỳ cái mới" → ERA'S MOST VALUABLE
     → NL7: adaptability > specific knowledge → meta-learning = vehicle cho NL7
     → 🟡 Framework emphasis — research supports "learning to learn" value

  ⑥ CREATIVE / EXPRESSIVE
     → Nghệ thuật, sáng tạo, biểu đạt cá nhân, aesthetic sense
     → Tại sao FOUNDATION: creativity = human differentiator vs AI
     → Channel cho Imagine-Final emerge (NL4)
     → Window cho hardware tendencies lộ qua expression
     → 🟡 Creativity as foundation = framework position (debatable)


TIER 2 — ERA-SPECIFIC DOMAINS (2025+, time-bound) 🟡→🔴:

  ⑦ AI LITERACY
     → Dùng AI tools, đánh giá AI output, biết AI limits
     → = "Biết dùng calculator" thời đại mới nhưng PHỨC TẠP hơn nhiều
     → ⚠️ AI tools thay đổi nhanh → teach PRINCIPLES of AI, not specific tools

  ⑧ INFORMATION CURATION
     → Lọc, đánh giá, synthesize information từ unlimited sources
     → = "Biết hỏi đúng" + "biết source nào đáng tin" + "biết tổng hợp"
     → Era-Analysis §2: "biết hỏi" > "biết" → curation = core era-skill

  ⑨ SYSTEMS THINKING
     → Hiểu complexity, interconnections, feedback loops
     → = Thế giới phức tạp hơn → cần HIỂU hệ thống, không chỉ parts
     → Convergence: link cross-domain (ref: backup 00_Overview §2.4)

  ⑩ CROSS-CULTURAL / GLOBAL AWARENESS
     → Collaboration across cultures, languages, perspectives
     → = Connected world → work WITH diverse people

  ⑪ DIGITAL WELLBEING
     → Manage screen time, attention, information overload, cortisol
     → Era-Analysis §2: information overload = new cortisol source
     → = "DÙNG technology" + "QUẢN LÝ technology impact on brain"

  ⚠️ Tier 2 = ERA-BOUND → khi era đổi → review + update


TIER 3 — PER-HARDWARE DOMAINS (per individual):

  → Emerge based on hardware tendencies (Hardware-Calibration.md)
  → Identified through observation Stage 2 → clearer Stage 3
  → Specialization Stage 3+ → deep in hardware-matched domain
  → VD: somatic-dominant → sports science, surgery, craft, performance
  → VD: verbal-dominant → literature, law, communication, research
  → VD: high-VTA → innovation, entrepreneurship, exploration
  → = KHÔNG prescribe → OBSERVE + SUPPORT per-individual direction
  → = Tier 3 = CÁI KHÔNG THỂ plan trước → emerge per person
```

```
     → Stage 4: foundation maintain + DEEP specialization

     → "Hiểu TOÁN ở L3" > "biết NHIỀU MÔN ở L1"
     → Depth = compile mạnh, transfer được, durable
     → Volume without depth = "biết tên nhiều thứ, không HIỂU thứ nào"

  🟡 Foundation curriculum framework = derived from principles + developmental research
     Specific depth targets per stage = framework estimate, not proven benchmarks
```

## 3. ERA-SPECIFIC CURRICULUM (2025+, TIME-BOUND)

```
⚠️ SECTION NÀY CÓ HẠN SỬ DỤNG → update khi era shifts
   Era context → xem Era-Analysis-2025.md cho chi tiết

MỤC ĐÍCH:
  → Tier 2 domains (§1) → chi tiết hóa: DẠY GÌ CỤ THỂ, KHI NÀO
  → = Cái THÊM vào foundation cho era 2025+


⑦ AI LITERACY — dạy gì, khi nào:

  Stage 2 (6-12): EXPOSURE + BASICS
    → Biết AI là gì (ở level phù hợp tuổi): "máy tính có thể học"
    → Dùng AI tools cơ bản (với supervision): "hỏi AI, đọc trả lời"
    → Biết AI CÓ THỂ SAI: "AI không phải lúc nào cũng đúng"
    → ⚠️ Age-appropriate: KHÔNG cần dạy "LLM architecture" cho 8 tuổi
    → Depth target: L1 (recognize AI, biết dùng cơ bản)

  Stage 3 (12-18): CRITICAL USE
    → Đánh giá AI output: bias, hallucination, limitations
    → Dùng AI cho research + learning (per-pace, per-interest)
    → Biết WHEN to use vs WHEN NOT to use AI
    → Ethics: AI + privacy, AI + fairness, AI + jobs
    → Human-AI collaboration skills bắt đầu
    → Depth target: L2→L3 (explain AI limits + apply effectively)

  Stage 4 (18+): PROFESSIONAL USE
    → AI trong professional domain (per-specialization)
    → Advanced human-AI collaboration
    → AI as thinking partner, not just tool
    → Depth target: L3→L4 (apply + create with AI)


⑧ INFORMATION CURATION — dạy gì, khi nào:

  Stage 2: BASICS
    → "Không phải mọi thứ trên internet đều đúng"
    → Source evaluation cơ bản: "ai nói? tại sao?"
    → Compare sources: "2 bài khác nhau → cái nào đáng tin hơn?"
    → Depth target: L1

  Stage 3: CRITICAL CURATION
    → Advanced source evaluation: bias, methodology, evidence
    → Synthesize: tổng hợp từ NHIỀU sources → own understanding
    → "Learn to ask RIGHT questions" = core skill
    → Depth target: L2→L3

  Stage 4: PROFESSIONAL CURATION
    → Domain-specific information evaluation
    → Research methodology, evidence hierarchy
    → Depth target: L3→L4


⑨ SYSTEMS THINKING:

  Stage 2: SEEDS
    → Cause and effect: "nếu X → thì Y"
    → Simple systems: "ecosystem, water cycle" = begin
    → Depth target: L1

  Stage 3: DEVELOP
    → Feedback loops, interconnections, unintended consequences
    → Apply to real-world: economics, environment, social systems
    → Cross-domain connections (convergence begins)
    → Depth target: L2→L3

  Stage 4: COMPLEX SYSTEMS
    → Model complex systems, predict behavior, design interventions
    → Depth target: L3→L4


⑩ CROSS-CULTURAL / GLOBAL AWARENESS:

  Stage 2: EXPOSURE
    → "Thế giới có nhiều cultures khác nhau → đều có giá trị"
    → Basic: languages, customs, food, celebrations
    → Depth target: L1

  Stage 3: UNDERSTANDING
    → Cultural perspectives, biases, collaboration challenges
    → Communication across cultures (practical skill)
    → Depth target: L2→L3

  Stage 4: PROFESSIONAL
    → Global collaboration, cross-cultural leadership
    → Depth target: L3


⑪ DIGITAL WELLBEING:

  Stage 2: BASICS
    → Screen time awareness: "bao nhiêu là đủ?"
    → Online safety basics
    → "Sau khi dùng screen → body CẢM THẤY thế nào?" (body-base awareness)
    → Depth target: L1

  Stage 3: SELF-MANAGEMENT
    → Attention management: "tôi bị distract bởi cái gì?"
    → Social media literacy: comparison, addiction patterns, FOMO
    → Self-regulation: "tôi CẦN offline khi nào?"
    → Cortisol management: recognize information overload signals
    → Depth target: L2→L3

  Stage 4: LIFESTYLE DESIGN
    → Conscious tech use: technology serves ME, not reverse
    → Professional digital boundaries
    → Depth target: L3


  🔴 Era-specific curriculum = prediction-based → WILL need updates
     "AI literacy" of 2025 ≠ "AI literacy" of 2030
     = Teach PRINCIPLES of using tools → not specific tools
```

## 4. SEQUENCING BY STAGE

```
MỤC ĐÍCH:
  → Tổng hợp: foundation (§2) + era-specific (§3) → flow per stage
  → = "Ở MỖI stage, balance giữa tiers THẾ NÀO?"

  ┌──────────┬─────────────┬─────────────┬─────────────┐
  │ Stage    │ Tier 1      │ Tier 2      │ Tier 3      │
  │          │ Foundation  │ Era-specific│ Per-hardware │
  ├──────────┼─────────────┼─────────────┼─────────────┤
  │ 2 (6-12) │ ████████    │ ██          │ ░           │
  │          │ 70-80%      │ 15-20%      │ 5-10%       │
  │          │ DOMINANT    │ exposure    │ seeds       │
  ├──────────┼─────────────┼─────────────┼─────────────┤
  │ 3 (12-18)│ █████       │ ████        │ ███         │
  │          │ 40-50%      │ 25-30%      │ 20-30%      │
  │          │ maintain+   │ develop     │ BEGIN       │
  │          │ deepen      │             │ specialize  │
  ├──────────┼─────────────┼─────────────┼─────────────┤
  │ 4 (18+)  │ ███         │ ███         │ ██████      │
  │          │ 20-30%      │ 20-25%      │ 45-55%      │
  │          │ maintain    │ maintain    │ DOMINANT    │
  └──────────┴─────────────┴─────────────┴─────────────┘


STAGE 2 (6-12) — FOUNDATION HEAVY:

  → Foundation 70-80%: literacy, numeracy, somatic, social, creative, meta-seeds
    → = Core business của Stage 2 → CHẮC ở đây → tất cả sau build TRÊN
  → Era-specific 15-20%: exposure level → AI basics, digital safety, global awareness
    → = "Biết AI tồn tại, biết internet có sai" → không cần SÂU
  → Per-hardware 5-10%: observe tendencies → cho EXPLORE per interest
    → = "Trẻ A hứng thú science → cho thêm science exposure"
    → ≠ Specialize → = Seeds cho direction Stage 3

  PHƯƠNG CHÂM: "Rộng + vừa sâu — xây foundation CHẮC"


STAGE 3 (12-18) — TRANSITION:

  → Foundation 40-50%: MAINTAIN + DEEPEN (không bỏ, nhưng tỉ lệ giảm)
    → Literacy: từ "đọc hiểu" → "phân tích, lập luận" (L2→L3)
    → Numeracy: từ "concrete" → "abstract reasoning" (L2→L3)
    → Meta-learning: từ seeds → DEVELOP thực sự (L1→L2)
  → Era-specific 25-30%: DEVELOP
    → AI literacy: critical use, evaluate output
    → Information curation: synthesize, evaluate sources
    → Systems thinking: feedback loops, cross-domain
  → Per-hardware 20-30%: BEGIN SPECIALIZATION
    → Hardware tendencies đã VISIBLE → cho chọn depth domains
    → Elective system: tự chọn 2-3 domains → go deeper
    → = "T-shape begins": foundation base + depth columns emerge

  PHƯƠNG CHÂM: "Foundation giữ + era-skills develop + direction emerge"


STAGE 4 (18+) — SPECIALIZATION DOMINANT:

  → Foundation 20-30%: MAINTAIN (không bỏ, duy trì health)
    → Literacy: maintain + expand per need
    → Social: maintain + leadership skills
    → Meta-learning: TỰ DIRECTED → lifelong (L2→L3→L4)
  → Era-specific 20-25%: MAINTAIN + PROFESSIONAL APPLICATION
    → AI: professional domain-specific use
    → Curation: research-level evaluation
  → Per-hardware 45-55%: DEEP SPECIALIZATION
    → Domain match hardware → GO DEEP (L3→L4)
    → Cross-domain integration = convergence
    → Contribution to society begins

  PHƯƠNG CHÂM: "Deep in 1-2 + maintain rest + contribute"


  🟡 Percentages = APPROXIMATE → per-individual varies
     Per-hardware learner: Tier 3 CÓ THỂ cao hơn sớm hơn (nếu direction rõ)
     Late bloomer: Tier 1 CÓ THỂ extend vào Stage 3 → BÌNH THƯỜNG
```

## 5. CÁI CẦN GIẢM / BỎ TỪ CURRICULA HIỆN TẠI

```
⚠️ CONTROVERSIAL nhưng CẦN THIẾT

  Thêm Tier 2 + Tier 3 = THÊM content → nhưng time = LIMITED
  → Nếu chỉ THÊM mà không BỎ → overload → cortisol → ngược lại (NL6)
  → = PHẢI giảm/bỏ CÁI GÌ ĐÓ để CÓ CHỖ cho cái mới
  → ⚠️ "BỎ" ≠ "xóa sổ" → = "GIẢM TỈ TRỌNG" hoặc "ĐỔI CÁCH DẠY"


CÁI GIẢM:

  ① ROTE MEMORIZATION OF FACTS
     → AI lookup facts trong giây → memorize facts = giá trị GIẢM
     → ≠ "Không cần nhớ gì" → = "nhớ FOUNDATION, lookup DETAILS"
     → VD: nhớ "F=ma" CẦN → nhớ bảng tuần hoàn 118 nguyên tố = AI lookup
     → = GIẢM drill/memorize → TĂNG understand/apply

  ② EXCESSIVE STANDARDIZED TESTING
     → Test culture = optimize surface (L1) → miss depth (L2-L4)
     → Time dành cho test prep → CÓ THỂ dùng cho project, depth, meta-learning
     → ≠ "Bỏ hết thi" → = "GIẢM high-stakes test + TĂNG formative assessment"
     → (ref: Education-System §6)

  ③ CONTENT ONLY RELEVANT CHO FORMAT CŨ
     → Kỹ năng CHỈ hữu ích trong system hiện tại, KHÔNG transferable
     → VD: "thuộc lòng bài thơ để đọc trước lớp" → test memory, NOT understanding
     → = Giữ bài thơ → ĐỔI "thuộc lòng" → "phân tích ý nghĩa" (depth)

  ④ PASSIVE LECTURE TIME
     → Lecture = weakest learning mechanism (Education-Principles §2.1)
     → KHÔNG bỏ hết → GIẢM tỉ lệ → TĂNG experiential, project, discussion
     → Balance: lecture 20-30% (intro + framework) + active 70-80% (practice + apply)


CÁI KHÔNG BỎ (dù có thể bị áp lực "bỏ"):

  ✓ Foundation domains — KHÔNG BAO GIỜ bỏ
    → "AI đọc hộ → bỏ dạy đọc?" → KHÔNG → "đọc" = HIỂU ý nghĩa, không chỉ decode
    → "AI tính hộ → bỏ dạy toán?" → KHÔNG → "toán" = logical thinking, không chỉ arithmetic

  ✓ Physical education — KHÔNG bỏ
    → Body-base = processing channel → somatic domain = brain development
    → ⚠️ Trend giảm PE → NGUY HIỂM cho brain development (NL1: multi-modal)

  ✓ Arts/Creative — KHÔNG bỏ
    → Creativity = human differentiator → ERA NÀY CẦN HƠN
    → ⚠️ Trend: "arts = không thực tế" → NGƯỢC LẠI: arts = hardware expression channel

  ✓ Social interaction — KHÔNG bỏ (dù online learning tăng)
    → Social brain = CẦN real human interaction → AI/online ≠ substitute
    → ⚠️ Post-COVID: social skill decline observable → CẦN TĂNG, không giảm


  TỔNG KẾT:
```
