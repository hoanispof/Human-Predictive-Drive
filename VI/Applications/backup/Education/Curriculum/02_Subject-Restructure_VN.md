## 2. Framework Phân Tích — Đánh Giá Mỗi Môn Thế Nào

🟡

```
MỖI MÔN HIỆN TẠI ĐƯỢC ĐÁNH GIÁ QUA 6 CÂU HỎI:

  ❶ NỀN hay CHUYÊN SÂU?
     → Content thuộc Tầng nào? (Nền = ai cũng cần. Chuyên sâu = per profile)
     → Nếu GỘP cả 2 → phải TÁCH

  ❷ DEPTH hiện tại ở stage nào?
     → Chủ yếu Stage 1 (nhớ đúng)? Hay có Stage 2+ (hiểu tại sao)?

  ❸ GIỮ gì, BỎ gì, CHUYỂN TẦNG gì?
     → Giữ: content AI CŨNG CẦN (nền tảng thật)
     → Bỏ: content không ai dùng (redundant)
     → Chuyển tầng: content tốt nhưng SAI TẦNG (ép tất cả vs cho ai cần)

  ❹ GỘP với môn nào? (shared foundation chunks)
     → Cùng foundation chunk → gộp = convergence TỰ NHIÊN

  ❺ CONVERGENCE potential?
     → Môn này share chunks gì với môn khác?

  ❻ PE profile?
     → Môn này cho PE ở profile nào? Mismatch ở profile nào?

→ Mỗi section dưới đây = 1 môn qua 6 câu hỏi này.
```

## 3. Ngữ Văn → Communication

### 3.1 Phân Tích Hiện Trạng

🟡

```
HIỆN TẠI (GDPT 2018):
  Tiểu học: Tiếng Việt — 7-12 tiết/tuần (MÔN NẶNG NHẤT ở TH)
  THCS: Ngữ văn — 4 tiết/tuần
  THPT: Ngữ văn — 3 tiết/tuần (bắt buộc)
  → Tổng: ~1,750 tiết (TH) + ~560 tiết (THCS) + ~315 tiết (THPT) ≈ 2,625 tiết

NỘI DUNG HIỆN TẠI:
  ① Đọc hiểu văn bản (văn học + thông tin)
  ② Viết (văn bản nghị luận, miêu tả, biểu cảm, thuyết minh)
  ③ Nói và nghe (có trong chương trình mới, nhưng thi KHÔNG đánh giá)
  ④ Kiến thức ngôn ngữ (ngữ pháp, từ vựng, phong cách ngôn ngữ)
  ⑤ Kiến thức văn học (lý luận văn học, lịch sử văn học)

ĐÁNH GIÁ QUA 6 CÂU HỎI:

  ❶ NỀN hay CHUYÊN SÂU?
     → Đọc hiểu + Viết rõ ràng + Nói/Nghe = NỀN TẢNG (Nhóm 1)
     → Phân tích văn học nâng cao + Lý luận văn học = CHUYÊN SÂU (Tầng 3)
     → HIỆN TẠI: GỘP cả 2 → ép TẤT CẢ phân tích văn học sâu

  ❷ DEPTH?
     → Đọc: Stage 1-2 (đọc hiểu cơ bản tốt, phản biện yếu)
     → Viết: Stage 1 (viết theo khuôn, "đáp án mẫu")
     → Nói/Nghe: Stage 0-1 (gần không practice)
     → Văn học: Stage 1 (nhớ tác phẩm, tác giả, "ý nghĩa chuẩn")

  ❸ GIỮ / BỎ / CHUYỂN TẦNG:
     GIỮ (Tầng Nền):
       ✅ Đọc hiểu văn bản thông tin (tin tức, hợp đồng, hướng dẫn)
       ✅ Viết rõ ràng (email, báo cáo, argument)
       ✅ Nói và nghe (TĂNG MẠNH — hiện gần không có)
       ✅ Đọc văn học như MIRROR cảm xúc (link Nhóm 4 emotional)
       ✅ Đọc phản biện (THÊM MỚI — hiện không dạy)
       ✅ Viết thuyết phục (THÊM MỚI — argument essay)
     CHUYỂN TẦNG (→ Tầng 3 Chuyên sâu):
       ↗ Phân tích văn học nâng cao (cho ai chọn Văn)
       ↗ Lý luận văn học, lịch sử văn học (cho ai chọn Văn)
       ↗ Viết sáng tạo nâng cao (cho ai match)
     BỎ/GIẢM:
       ✂ "Phân tích theo đáp án chuẩn" → thay bằng "phân tích có lý, chấp nhận đa ý kiến"
       ✂ Nhớ tên tác giả, năm sáng tác (Stage 1 thuần)
       ✂ Viết theo "dàn ý mẫu" cứng

  ❹ GỘP:
     → GỘP với Ngoại ngữ → thành "COMMUNICATION"
     → Shared foundation: đọc hiểu, viết, nói/nghe, phản biện
     → Tiếng Việt + Tiếng Anh = 2 ngôn ngữ, CÙNG SKILLS

  ❺ CONVERGENCE:
     → Đọc phản biện = link Nhóm 3 (scientific thinking)
     → Viết argument = link Nhóm 2 (logic)
     → Văn học = link Nhóm 4 (emotional literacy)
     → HIGH convergence potential nếu dạy đúng

  ❻ PE PROFILE:
     → Internal-deep: PE từ phân tích sâu, "tại sao tác giả viết thế?"
     → Internal-explore: PE từ creative writing, variety format
     → External: PE từ structure rõ (template viết, rubric rõ)
     → Drifter: PE thấp nếu "bài văn dài" — format ngắn, visual, practical
```

### 3.2 Đề Xuất Tái Cấu Trúc

🔴

```
TRƯỚC:                              SAU:
  Tiếng Việt (TH) 7-12t/t            COMMUNICATION (TH) ~6-8t/t
  Ngữ văn (THCS) 4t/t                  → Tiếng Việt core
  Ngữ văn (THPT) 3t/t                  → Ngoại ngữ (immersion blend)
  Ngoại ngữ (TH) 4t/t                  → Đọc phản biện
  Ngoại ngữ (THCS) 3t/t                → Viết thuyết phục
  Ngoại ngữ (THPT) 3t/t                → Debate/trình bày
  ─────────────────                     → Văn học như emotional mirror
  Tổng: ~10-15 tiết/tuần             COMMUNICATION (THCS) ~6t/t
                                      COMMUNICATION (THPT) ~4t/t
                                      ─────────────────────────
                                      Tổng: ~6-8 tiết/tuần (GIẢM)
                                      + Văn học nâng cao → Tầng 3

THAY ĐỔI CỐT LÕI:
  ① "Phân tích đúng đáp án" → "Phân tích có logic, đa ý kiến"
  ② "Viết theo khuôn" → "Viết argument + practical"
  ③ "Nói/nghe = 0" → "Debate + trình bày = core skill"
  ④ "Ngoại ngữ riêng" → "Gộp vào Communication, immersion-based"
  ⑤ "Văn học = bắt buộc sâu" → "Văn học nền (emotional) + sâu = Tầng 3"

TIẾT KIỆM: ~4-7 tiết/tuần → dùng cho MÔN MỚI (Emotional, Financial,...)
```

## 4. Toán → Numeracy & Logic + Toán Chuyên Sâu

### 4.1 Phân Tích Hiện Trạng

🟡

```
HIỆN TẠI (GDPT 2018):
  Tiểu học: Toán — 3-5 tiết/tuần
  THCS: Toán — 4 tiết/tuần
  THPT: Toán — 3 tiết/tuần (bắt buộc)
  → Tổng: ~875 tiết (TH) + ~560 tiết (THCS) + ~315 tiết (THPT) ≈ 1,750 tiết

NỘI DUNG HIỆN TẠI (progressive):
  TH: Số học, hình học cơ bản, đo lường, thống kê sơ bộ
  THCS: Đại số, hình học, thống kê-xác suất, hàm số
  THPT: Đại số + Giải tích, Hình học không gian, Thống kê-Xác suất,
        Tổ hợp, Dãy số, Giới hạn, Đạo hàm, Tích phân, Số phức

ĐÁNH GIÁ QUA 6 CÂU HỎI:

  ❶ NỀN hay CHUYÊN SÂU?
     NỀN TẢNG (Nhóm 2 — ai cũng cần):
       → Số học 4 phép tính, phân số, %, tỉ lệ
       → Ước lượng (Fermi estimation)
       → Thống kê cơ bản (trung bình, median, đọc biểu đồ)
       → Xác suất cơ bản ("80% khả năng = ?")
       → Hình học thực tế (đo đạc, diện tích thực tế)
       → Logic: suy luận if-then, phát hiện ngụy biện
       → Decision-making: pro/con, opportunity cost
     CHUYÊN SÂU (Tầng 3 — cho ai cần):
       → Đại số nâng cao (phương trình bậc cao, hệ phương trình phức tạp)
       → Giải tích (giới hạn, đạo hàm, tích phân)
       → Hình học không gian, chứng minh hình học
       → Tổ hợp, dãy số, số phức
       → Lượng giác nâng cao
     → HIỆN TẠI: ~40% nội dung = Tầng 3, ÉP TẤT CẢ

  ❷ DEPTH?
     → Tính toán: Stage 1-2 (tính đúng, một phần hiểu tại sao)
     → Chứng minh: Stage 2-3 ở top students, Stage 1 ở đa số
     → Ước lượng: KHÔNG DẠY (Stage 0)
     → Thống kê: Stage 1 (công thức, rất muộn — lớp 10+)
     → Logic đời sống: KHÔNG DẠY (Stage 0)
     → NGHỊCH LÝ: dạy Toán NHIỀU nhưng numeracy thực tế THIẾU

  ❸ GIỮ / BỎ / CHUYỂN TẦNG:
     GIỮ (Tầng Nền):
       ✅ Số học cơ bản (TH) — đã tốt, giữ nguyên
       ✅ Phân số, %, tỉ lệ — thực tế, cần thiết
       ✅ Hình học thực tế (đo, diện tích, thể tích)
     THÊM MỚI vào Nền:
       ✅ Ước lượng (Fermi) — KHÔNG CÓ hiện tại
       ✅ Thống kê thực tế (đọc biểu đồ phản biện, sample size)
       ✅ Xác suất đời sống ("xác suất mưa 80% nghĩa là gì?")
       ✅ Logic + ngụy biện (phân tích quảng cáo, tin tức)
       ✅ Financial math cơ bản (lãi kép — link Nhóm 5)
       ✅ Decision-making (opportunity cost, trade-off)
     CHUYỂN TẦNG (→ Tầng 3):
       ↗ Giải tích (đạo hàm, tích phân) — CHỈ cho STEM path
       ↗ Hình học không gian + chứng minh phức tạp
       ↗ Số phức, tổ hợp nâng cao, dãy số
       ↗ Lượng giác nâng cao
       ↗ Đại số nâng cao (hệ PT phức tạp)
     BỎ/GIẢM:
       ✂ Bài tập drill lặp lại cùng dạng (threshold ↑, PE = 0)
       ✂ "Mẹo giải nhanh" (trick, không build chunks)

  ❹ GỘP:
     → TÁCH thay vì gộp:
       Numeracy & Logic = NỀN (trong Communication-Numeracy block)
       Toán Chuyên sâu = TẦNG 3 (elective cho STEM path)
     → Logic component = link với Scientific Thinking + Communication (debate)

  ❺ CONVERGENCE:
     → Modeling = shared với KHTN (Nhóm 3)
     → Thống kê = shared với Financial (Nhóm 5), Health (Nhóm 6)
     → Logic = shared với Scientific Thinking, Media Literacy
     → TOÁN = MÔN CÓ CONVERGENCE POTENTIAL CAO NHẤT nếu dạy đúng

  ❻ PE PROFILE:
     → Internal-deep: PE từ "tại sao công thức này đúng?" (proof, insight)
     → Internal-explore: PE từ puzzle, challenge, surprise answer
     → External: PE từ "tính đúng", procedure clear, step-by-step
     → Drifter: Toán = MÔN GÂY GHÉT NHẤT (failure loop → PE âm → "tôi dốt Toán")
       → Schema "dốt Toán" hình thành SỚM (lớp 3-4) → RẤT KHÓ undo
```
