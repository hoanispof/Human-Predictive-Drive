## 1. Insight Nền: Mọi Bài Test Đều Đo Predictive-Chunks

🟡 **Giả thuyết trung tâm:**

```
Mọi bài test tâm lý học đã tồn tại đang đo MỘT HOẶC VÀI
khía cạnh của predictive-chunk configuration — chỉ là:
  (a) KHÔNG BIẾT mình đang đo chunks
  (b) Đo tách rời từng domain, không có unified lens
  (c) Thiếu nhiều tham số quan trọng của framework

IQ      = chunk depth (sub 1: quality) ở domain abstract/logic
EQ      = chunk depth ở domain cảm xúc/social
Big Five = 5 proxy cho 5 tham số chunks khác nhau
MBTI    = chunk source (I/E) + processing style (thô)
Grit    = sustained chunk investment (deepen persistence)

→ Framework KHÔNG phát minh ra việc đo prediction depth.
→ Framework THÊM GIÁ TRỊ ở 3 chỗ:
  1. UNIFIED LENS: gộp tất cả tests thành 1 hệ thống
  2. ĐO CÁI CHƯA AI ĐO: source per domain, depth composite (4 sub-params),
     compliance per reference group, convergence (chunk topology)
  3. TƯƠNG TÁC: các tham số tương tác — test hiện tại đo tách rời

⚠️ THAY ĐỔI TỪ v5.0:
  v5.0: "chưa đo source, sync, threshold per domain"
  v5.5: "chưa đo source per domain, depth COMPOSITE (4 sub-params),
         convergence (cross-domain chunk overlap), compliance PER reference group"
  → Sync KHÔNG CÒN là dimension đo riêng — merge vào depth composite
    (connectivity = sub-parameter #2 của depth).

⚠️ THAY ĐỔI TỪ v5.5 → v6.0:
  1. PE Sensitivity TÁCH 4 sub-mechanisms (Core.md §5.3):
     ① Amplitude (volume tín hiệu)
     ② Precision weighting (PE đáng tin cỡ nào)
     ③ Habituation rate (chán nhanh/chậm)
     ④ Generalization scope (chán rộng/hẹp) — MỚI QUAN TRỌNG
     → ④ giải thích tại sao cùng Novelty + Threshold mà expression khác
     → Test hiện tại GỘP cả 4 → cần TÁCH để đo đúng

  2. Novelty Reframe:
     → Novelty ≠ "thích cái mới khác domain" (broad)
     → Novelty = MỌI chunk mới, KỂ CẢ deepen cùng domain
     → Ảnh hưởng: NFC đo Novelty ĐÚNG HƠN Sensation Seeking
       (SS thiên broad novelty, NFC bao gồm cả deep novelty)

  3. Cortisol 5-level gradient (Core.md §6.4):
     ①thấp ②sweet-spot ③khó-chịu ④suy-yếu ⑤sụp-đổ
     → Mức ③ = tiger parenting zone: chunks hình thành + negative affect
     → Ảnh hưởng: Grit cần phân biệt thêm (mức ② vs ③)

  4. Cortisol × Novelty vai trò bổ sung:
     → Cortisol = đánh thức PFC, Novelty = hướng dẫn PFC
     → Depth > Stage 2 ĐÒI HỎI novelty (cortisol alone = ceiling)
     → Ảnh hưởng: đo depth cần KÈM đo motivation source

  5. Spatial reasoning ≠ Spatial navigation:
     → IQ test đo spatial REASONING (PFC + parietal = pattern matching)
     → Navigation dùng spatial MAPPING (hippocampus = cognitive map)
     → Cùng "spatial" trên IQ test nhưng KHÁC hệ thống não
     → Ảnh hưởng: IQ spatial score KHÔNG predict navigation ability
```

## 2. Map Chi Tiết: 18 Bài Test Phổ Biến

### 2.1 IQ Tests (WAIS, Raven's Progressive Matrices)

```
🟢 ĐO GÌ:
  Pattern recognition, spatial reasoning, verbal reasoning,
  working memory, processing speed.

🟡 QUA v5.5 LENS:
  IQ = DEPTH SUB-1 (chunk quality) + SPEED ở domain ABSTRACT/LOGIC/PATTERN.
  IQ cao → chunks dài + chính xác ở domain trừu tượng.

  v5.5 MAPPING MỚI — depth composite:
    Sub 1 (quality): ★★★★ — IQ đo trực tiếp prediction accuracy
    Sub 2 (connectivity): ★★ — working memory = giữ nhiều chunks SYNC
    Sub 3 (cluster formation): ★ — không đo cấu trúc clusters
    Sub 4 (convergence): · — không đo cross-domain overlap

  NHƯNG KHÔNG ĐO:
    ❌ Depth ở domain KHÁC (social, aesthetic, kinesthetic)
    ❌ Source (internal vs external) — IQ cao vẫn có thể ở cạnh phải Mô Hình Vuông
    ❌ Depth sub 3-4 (cluster + convergence)
    ❌ Threshold, kênh gốc, PE Sensitivity

  GIÁ TRỊ: IQ ≈ Capacity (tham số nền §5). TRẦN, không phải actual depth per domain.
  IQ 140 + chưa train = chunks VẪN NÔNG. IQ 100 + 20 năm train = có thể SÂU HƠN.
  🟢 Deary (2012): IQ heritability ~50-80%. Flynn effect: training CÓ ảnh hưởng.

  ⚠️ v6.0 — SPATIAL REASONING ≠ SPATIAL NAVIGATION:
    IQ "spatial" đo: mental rotation, block design, matrix reasoning, visual puzzles
    = PATTERN RECOGNITION (PFC + parietal cortex)

    Navigation spatial: biết hướng, cognitive map, tự tìm đường
    = COGNITIVE MAP (hippocampus place cells + grid cells)
    🟢 Marchette et al. (2011): 2 nhóm people — "spatial" (hippocampus) vs
       "verbal/sequential" (caudate + PFC) — KHÁC chiến lược, khác vùng não

    → Người IQ spatial CAO + navigation KÉM = hoàn toàn bình thường
    → Cùng label "spatial" nhưng khác domain, khác chunks, khác brain system
    → IQ spatial score KHÔNG predict navigation ability
    → = Ví dụ cho vấn đề lớn hơn: 1 ĐIỂM test KHÔNG represent domain phức tạp
```

### 2.2 EQ — Emotional Intelligence (Mayer-Salovey-Caruso / Goleman / Bar-On)

```
🟢 ĐO GÌ: Perceiving, using, understanding, managing emotions.
  3 mô hình: MSCEIT (khả năng), Goleman (hỗn hợp), Bar-On EQ-i (tự đánh giá).

🟡 QUA v5.5 LENS:
  EQ = DEPTH COMPOSITE ở domain CẢM XÚC / SOCIAL.
    Perceiving = sub 1 (chunk quality: predict trạng thái cảm xúc CHÍNH XÁC)
    Understanding = sub 2 (connectivity: predict CHUỖI cảm xúc liên kết)
    Managing = cluster formation: chunks cảm xúc tổ chức thành hệ thống điều chỉnh

  EQ ≠ Oxytocin channel: Novelty-primary vẫn EQ cao nếu đã deepen domain social.
  Oxytocin-primary có thể EQ thấp nếu chưa deepen ("cảm" mà không "hiểu").

  NHƯNG KHÔNG ĐO:
    ❌ Source: EQ từ sách (external) vs tự trải nghiệm (internal) → KHÁC
    ❌ Convergence: chunks social có overlap với chunks domain khác?
    ❌ Kênh gốc, Threshold

  GIÁ TRỊ: EQ = depth composite ở domain social/emotional.
  "IQ vs EQ" debate → dissolve: depth ở domain khác nhau.
```

### 2.3 Big Five / OCEAN (NEO-PI-R, BFI)

```
🟢 ĐO GÌ: 5 nhân tố tính cách ổn định, heritability 40-60% (Bouchard, 1994).

🟡 MAP TỪNG NHÂN TỐ → v5.5:

  OPENNESS → EXPLORE TENDENCY + Novelty channel proxy
    O cao = thích tạo chunk MỚI (explore), tolerance for unknown.
    → Tương quan: PE Sensitivity (cao → explore → O cao), Novelty channel.
    ⚠️ O cao + Opioid = explore TRẢI NGHIỆM (khác Novelty = explore Ý TƯỞNG).

  CONSCIENTIOUSNESS → CHUNK ORGANIZATION (depth sub 2+3)
    C cao = chunks CÓ CẤU TRÚC, connectivity cao, cluster formation tốt.
    C thấp = chunks lộn xộn, connectivity thấp, cluster yếu.
    ⚠️ v5.5: C = proxy cho depth sub 2-3 (connectivity + cluster formation).
    v5.0 nói C = "sync proxy" → v5.5: sync = emerge từ depth → C đo DEPTH COMPONENTS.
    C KHÔNG phân biệt source: cạnh phải (external organized) vs trên (internal organized).

  EXTRAVERSION → SOCIAL CHUNK ACTIVATION + NE level
    E cao = chunks social kích hoạt mạnh + thường xuyên.
    E thấp ≠ chunks social nông (có thể sâu nhưng kích hoạt ít).

  AGREEABLENESS → EXTERNAL ALIGNMENT TENDENCY
    A cao = thiên sync chunks với NGƯỜI KHÁC → cạnh phải Mô Hình Vuông.
    A thấp = giữ chunks bất kể người khác → cạnh trái hoặc trên.
    ⚠️ v5.5: A cao ≈ compliance score CAO (chunk overlap cao với majority).
    NHƯNG A không đo compliance PER reference group (v5.5 = hàm 2 biến).

  NEUROTICISM → CORTISOL SENSITIVITY + THREAT PREDICTION
    N cao = cortisol reactivity cao → threat prediction mạnh.
    N cao + capacity cao = CÓ THỂ dùng anxiety làm drive (inverted-U §6.4).
    N cao + capacity thấp = cortisol-driven behavior.

  TỔNG HỢP v5.5:
    O ≈ PE Sensitivity + Novelty channel (proxy)
    C ≈ Depth sub 2-3: connectivity + cluster formation (proxy)
    E ≈ Social chunk activation + NE (proxy)
    A ≈ External alignment / compliance tendency (proxy)
    N ≈ Cortisol sensitivity (proxy)

  NHƯNG KHÔNG ĐO:
    ❌ Depth per domain (general trait, không per domain)
    ❌ Source TRỰC TIẾP (C không phân biệt vị trí trên Mô Hình Vuông)
    ❌ Convergence (cross-domain chunk overlap)
    ❌ Threshold trực tiếp
    ❌ Compliance PER reference group (A = general tendency)

  GIÁ TRỊ: BẢN ĐỒ PHỦ CAO NHẤT hiện có, nhưng KHÔNG ĐỦ SÂU.
```

### 2.5 Attachment Styles (Ainsworth / Bartholomew)

```
🟢 4 style: Secure, Anxious-Preoccupied, Dismissive-Avoidant, Fearful-Avoidant.
  🟢 Heritability ~25-40%, phần lớn từ môi trường sớm (Fraley, 2002).

🟡 QUA v5.5 LENS — vị trí trên Mô Hình Vuông ở DOMAIN QUAN HỆ GẮN BÓ:

  Secure = depth composite CAO + source ổn định
    → Ở giữa-trên Mô Hình Vuông: chunks social sâu, accurate, tổ chức tốt.
    Schema: "người khác đáng tin" + "mình có giá trị."

  Anxious = depth NÔNG + external source + connectivity thấp
    → Cạnh phải-dưới: cần validation liên tục = external dependency.
    Schema: "không chắc" → cortisol → cling.

  Dismissive = chunks social BỊ SUPPRESS + internal default
    → Cạnh trái: "không cần người" = tắt social domain, không phải không có.
    Schema: "người khác không đáng tin" → tắt oxytocin pathway.

  Fearful = chunks social MÂU THUẪN → connectivity CỰC THẤP
    → Cạnh dưới: schema xung đột → chunks không tổ chức được.

  GIÁ TRỊ: CAO. Attachment = chunk config ở domain QUAN HỆ GẮN BÓ cụ thể.
  Phù hợp v5.5: per domain, có schema, có source, có depth composite.
  ⚠️ Chỉ đo 1 DOMAIN. Dismissive ở gắn bó CÓ THỂ Secure ở friendship.
```

### 2.6 Grit Scale (Duckworth)

```
🟢 "Perseverance and passion for long-term goals."
  🟢 Duckworth et al. (2007): Grit predict thành công hơn IQ ở nhiều bối cảnh.

🟡 QUA v5.5 LENS:
  Grit = KHẢ NĂNG DUY TRÌ chunk investment (DEEPEN) bất chấp cost.
  = f(Threshold hợp lý, PE Sensitivity thấp, Schema "nỗ lực = đền đáp")

  ⚠️ v5.5 — Dopamine inverted-U (§6.6) giải thích grit KHÁC nhau:
    Dopamine tối ưu → tonic đủ → deepen BỀN = grit TỰ NHIÊN
    Dopamine hơi cao → phasic > tonic → explore THẮNG deepen = grit THẤP
    → Grit scale KHÔNG phân biệt: grit thật (dopamine) vs grit giả (cortisol-driven)

  NHƯNG KHÔNG ĐO:
    ❌ TẠI SAO grit (threshold? schema? dopamine inverted-U position?)
    ❌ Grit PER DOMAIN
    ❌ Cortisol-driven grit ("không dám bỏ") vs Dopamine-driven grit ("còn PE")

  ⚠️ v6.0 — CORTISOL 5-LEVEL GRADIENT × GRIT:
    Grit ở cortisol mức ②: sustainable, có novelty, TỰ NGUYỆN tiếp tục
    Grit ở cortisol mức ③: hoàn thành nhưng GHÉT, dừng ngay khi hết ép
      → Tiger parenting grit: chunks hình thành + negative affect
      → Score Grit Scale CAO nhưng outcome KHÁC (burnout sau khi thoát áp lực)
    Grit ở cortisol mức ④: không còn grit — PFC suy yếu → learned helplessness

    → Grit Scale cần THÊM: "Sau khi xong, bạn có muốn tiếp không?"
      CÓ = mức ② grit (thật). KHÔNG = mức ③ grit (ép). KHÔNG DÁM BỎ = cortisol.
    → Ref: "tự nguyện tiếp tục" proxy — Core.md §6.4 (Cortisol × Novelty)
```

## 5. Chunk Topology × Psychometrics — Đo Convergence


## 7. Habituation Blindness × Test Reliability

🟡 Section MỚI v5.5. Core.md §6.7 mô tả Habituation Blindness (PE=0 ≠ Value=0). Section này phân tích ảnh hưởng lên psychometrics.

```
ẢNH HƯỞNG LÊN TEST RESULTS:
  1. Self-report bias: người habituated ĐÁNH GIÁ THẤP giá trị chunks đang có.
     VD: SDT Competence satisfaction → score thấp KHÔNG vì thiếu competence
         mà vì PE=0 cho competence đã có (predicted → invisible).
     → Self-report underestimate giá trị thực của stable chunks.

     Big Five đo general traits → ỔN ĐỊNH theo thời gian (🟢 test-retest cao).
     NHƯNG: "ổn định" có thể = habituated (không CẢM NHẬN thay đổi).
     Chunks THỰC TẾ có thể đã thay đổi nhưng self-report CHƯA UPDATE.

  3. Attachment Style stagnation:
     "Secure attachment" score ổn định → có thể = habituated.
     Chỉ khi MISMATCH xảy ra (partner mới, stress) → re-test → score THAY ĐỔI.
     → Test-retest reliability CAO ở attachment có thể PHÓNG ĐẠI stability.

🔴 GIẢ THUYẾT: "Test Habituation Effect"
  Self-report tests đo chunks ĐÃ habituated = UNDERCOUNT.
  Ability-based tests (IQ, MSCEIT) đo ACTUAL performance = ACCURATE hơn.
  → Ability-based > self-report cho đo DEPTH THỰC TẾ.
  → Self-report đo PERCEIVED depth (bị habituation bias).

  Practical implication:
    Battery tối ưu nên THIÊN VỀ ability-based tests (performance measure)
    thay vì self-report (perception measure — bị habituation blind).
    Kết hợp: ability-based (actual) + self-report (perceived) → GAP giữa 2
    = habituation blindness indicator.
```
