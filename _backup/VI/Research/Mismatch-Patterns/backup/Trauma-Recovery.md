## 1. Trauma = Gì Ở Mức Schema

```
TRAUMA ≠ "sự kiện xấu":
  → Sự kiện xấu XẢY RA → nhưng KHÔNG phải ai cũng bị trauma
  → Trauma = CÁCH SCHEMA GHI LẠI sự kiện → GÂY ĐAU KHI FIRE LẠI

  2 DẠNG TRAUMA (cả 2 đều compile schema mạnh):

  DẠNG 1 — TRAUMA SỰ KIỆN (intense, 1 lần hoặc vài lần):
    Ví dụ: bố đánh (1 lần mạnh)
    → Schema compile: [bố] + [đau] + [sợ] + [giọng quát] + [cái roi]
    → Chunks = MỌI sensory input LÚC ĐÓ → compiled CÙNG LÚC
    → = "Gói" gồm NHIỀU chunks fire ĐỒNG BỘ
    → = "Nghe giọng quát → TOÀN BỘ gói fire → body đau + sợ"
    → Dù 20 năm sau, dù KHÔNG phải bố → giọng quát = trigger → gói fire

  DẠNG 2 — TRAUMA TÍCH LŨY (nhẹ, lặp × hàng nghìn lần):
    Phổ biến HƠN dạng 1 — nhưng KHÓ NHẬN RA hơn nhiều:

    🟢 ACE Study (Adverse Childhood Experiences) — 5 dạng phổ biến nhất:

    ① Bị bỏ bê / lơ là (neglect) — PHỔ BIẾN NHẤT toàn cầu:
       → KHÔNG bị đánh → bố mẹ chỉ KHÔNG QUAN TÂM
       → "Mẹ ơi xem con vẽ!" → "Mẹ bận" → lặp × 1000 lần
       → MỖI lần = nhẹ (không ai gọi "trauma")
       → 1000 lần = schema compile SÂU: [tôi không đáng được chú ý]
       → = Micro-trauma × repetition = MACRO schema
       → KHÓ NHẬN RA vì: "bố mẹ KHÔNG LÀM GÌ SAI" (không đánh, không chửi)
       → Chỉ "KHÔNG LÀM GÌ" → nhưng "không làm gì" = thiếu Connection

    ② Bố mẹ ly hôn / gia đình tan vỡ:
       → Trẻ: "bố mẹ bỏ NHAU → có phải lỗi TÔI không?"
       → Schema: [gia đình = không bền] + [yêu = có thể mất bất kỳ lúc nào]
       → → Lớn lên: khó COMMIT relationship (sợ mất lại)
       → → HOẶC: bám quá chặt (sợ bị bỏ → kiểm soát)

    ③ Bị bắt nạt ở trường:
       → "Mày béo / xấu / ngu" × hàng ngày × hàng năm
       → MỖI lần = nhẹ → nhưng × 1000 = schema [tôi không xứng] compiled SÂU
       → → Lớn lên: thiếu tự tin, rụt rè, status schema thu hẹp

    ④ Bị quát mắng / chửi liên tục:
       → Không đánh → nhưng lời nói GÂY ĐAU SCHEMA
       → (đau schema KHÔNG có endorphin buffer → KÉO DÀI hơn đau vật lý)
       → "Mày vô dụng" × 1000 lần = schema [tôi vô dụng] compiled CỰC SÂU

    ⑤ Bị ép thành tích / so sánh:
       → "Con nhà người ta giỏi hơn con"
       → Schema: [tôi chỉ ĐÁNG YÊU khi GIỎI] = conditional love
       → → Lớn lên: perfectionist + sợ fail + self-worth gắn với performance

    → DẠNG 2 phổ biến hơn DẠNG 1 rất nhiều
    → DẠNG 2 khó nhận ra hơn (vì "không ai đánh tôi" → "tôi không có trauma"?)
    → Nhưng: micro-trauma × repetition CÓ THỂ compile SÂU BẰNG macro-trauma

  Tại sao compile MẠNH đến vậy:

    DẠNG 1 (sự kiện): 🟢 Emotional intensity × single trial learning:
      Bình thường: cần 10-100 lần lặp → compile
      Trauma sự kiện: 1 lần ĐỦ (vì emotional CỰC MẠNH)
      → Amygdala fire CỰC MẠNH → tag "CỰC KỲ QUAN TRỌNG"

      → = KHÔNG "nhớ 1 lần cụ thể" → nhưng "FEEL luôn luôn"
      → = "Tôi không biết TẠI SAO tôi thiếu tự tin → chỉ LUÔN thiếu"

      → = Schema compiled từ TỔNG HỢP, không từ 1 event
```

### 1.1 Chunks trong trauma — phần lớn TỐT

```
🟡 Insight quan trọng:

  Bị ép học toán suốt tiểu học (bố cầm roi):
    → Hàng NGHÌN chunks toán: cộng, trừ, nhân, chia, bảng cửu chương,...
    → = 99% chunks ĐÚNG + HỮU ÍCH (kiến thức toán = đúng thực tế)
    → Chỉ VÀI chunks TRIGGER: giọng bố, cái roi, bàn học cũ, mùi sách cũ
    → = 1% chunks trigger → "ô nhiễm" 99% chunks tốt

  Giống: 1 giọt mực trong ly nước
    → Nước VẪN OK (chunks toán vẫn đúng)
    → Nhưng TRÔNG bẩn (body FEEL khó chịu khi dùng chunks toán)
    → Vì: chunks trigger GẮN CHUNG → fire thì gói TOÀN BỘ fire

  → Vấn đề KHÔNG phải "chunks xấu"
  → Vấn đề = VÀI chunks trigger GẮN CHUNG VỚI chunks tốt
  → = "Chữa" = TÁCH trigger khỏi chunks tốt → KHÔNG phải xóa tất cả
```

### 1.2 Trigger = CONTEXT-DEPENDENT

```
  Trauma KHÔNG fire MỌI LÚC → chỉ fire khi GẶP TRIGGER:

    Ở nhà + gặp bố: trigger [giọng bố, cái roi, bàn học] → fire → SỢ
    Ở trường + gặp bạn: trigger KHÔNG CÓ → trauma IM → bình thường
    Đi làm + gặp đồng nghiệp: trigger KHÔNG CÓ → trauma IM
    Đi làm + dùng toán: chunk [bảng cửu chương] → GẦN trigger → HƠI khó chịu
    Đi làm + sếp quát: chunk [giọng quát] → GIỐNG trigger → fire MẠNH → sợ/khóc

  → Trauma = DORMANT khi không có trigger (Schema-Atlas-v2 §5: dormant state)
  → Trauma = ACTIVE khi gặp trigger chunk
  → Trigger càng GẦN trauma chunks gốc → fire càng MẠNH
  → Trigger xa (chỉ hơi giống) → fire NHẸ ("hơi khó chịu không hiểu vì sao")
  → Trigger chính xác (giọng bố thật) → fire CỰC MẠNH (full trauma response)
```

## 4. Hướng A — Re-Associate Chunks Trigger

```
CƠ CHẾ:
  → Chunk trigger [giọng bố] GẮN schema [bố = đau]
  → Tiếp xúc bố TRONG context AN TOÀN + TÍCH CỰC + NHIỀU LẦN
  → Mỗi lần: chunk [giọng bố] + context MỚI [bố = ok]
  → 2 schemas CẠNH TRANH: [bố = đau] (cũ) vs [bố = ok] (mới)
  → Schema mới REINFORCE liên tục → mạnh DẦN → cũ yếu DẦN
  → = KHÔNG xóa cũ → THÊM mới MẠNH HƠN → cũ bị override

  🟢 = Exposure therapy principle (Foa & Kozak, 1986) ✅

ĐIỀU KIỆN (bắt buộc):
  ✅ Context PHẢI an toàn THẬT (body CẢM THẤY, không chỉ BIẾT)
  ✅ Nguồn trauma PHẢI thay đổi THẬT (bố VẪN đánh → re-associate FAIL)
  ✅ Lặp NHIỀU lần (schema cũ compiled SÂU → cần nhiều lần override)
  ✅ Thời gian: tháng → NĂM

TIMELINE ví dụ (bố đã thay đổi + gặp thường xuyên):
  Lần 1:   gặp bố → body TENSE (schema cũ fire) → nhưng bố HIỀN → "hmm"
  Lần 5:   gặp bố → body HƠI tense → bố VẪN hiền → "có vẻ ok"
  Lần 20:  gặp bố → body GẦN bình thường → "bố đã đổi thật"
  Lần 50+: gặp bố → body OK → schema mới ĐÃ mạnh hơn cũ
  → = CHẬM nhưng BỀN VỮNG

⚠️ NẾU bố KHÔNG thay đổi:
  → Re-associate FAIL (reality VẪN = trauma → confirm schema cũ → MẠNH HƠN)
  → = "Cố gắng tha thứ nhưng bố VẪN VẬY" → schema cũ REINFORCE thêm
  → = Trường hợp này: cần HƯỚNG KHÁC (B, C, D) → KHÔNG ép re-associate
```

## 5. Hướng B — Tách Emotional Chunk Khỏi Memory

```
MỤC TIÊU: "nhớ BỐ ĐÁNH (memory) nhưng KHÔNG ĐAU nữa (emotion tách)"

🟢 EMDR (Eye Movement Desensitization and Reprocessing):
  → Shapiro (1989), nhiều RCTs confirm effective cho PTSD ✅

  Cách hoạt động:
    → Nhớ trauma + ĐỒNG THỜI mắt theo dõi ngón tay qua lại
    → Bilateral movement → não reprocess memory
    → Sau vài sessions: NHỚ trauma nhưng emotional charge GIẢM đáng kể

  🟡 Framework giải thích:
    → Trauma = nhiều chunks fire ĐỒNG BỘ (memory + emotion + body response)
    → Bilateral movement → DISRUPT đồng bộ tạm thời
    → Chunks fire KHÔNG đồng bộ → TÁCH → re-compile RIÊNG
    → Memory chunks: compile riêng (NHỚ nhưng neutral)
    → Emotional chunks: yếu dần (không được reinforce cùng memory)
    → ⚠️ Mechanism chính xác VẪN TRANH CÃI trong academia

🟢 Somatic Experiencing (Peter Levine):
  → Focus: body RESPONSE bị "đóng băng"

  Cơ chế:
    → Trauma = body response BỊ FREEZE (fight/flight KHÔNG hoàn thành)
    → Ví dụ: muốn CHẠY nhưng bị giữ → body freeze impulse "chạy"
    → Therapy: guide body CẢM NHẬN impulse → từ từ RELEASE
    → Body: "ồ, ĐÃ CHẠY xong" (dù trong đầu) → response COMPLETE
    → = Schema trauma GIẢI PHÓNG body response bị kẹt

  🟡 Framework giải thích:
    → Trauma schema chứa chunk [body response chưa hoàn thành]
    → Chunk này fire MỖI LẦN gặp trigger → body TENSE (chuẩn bị response chưa xong)
    → Complete response → chunk [đã xong] compile → override chunk [chưa xong]
    → = Body "kết thúc câu chuyện" → schema SETTLE
```

## 6. Hướng C — Giảm Cường Độ Fire

```
KHÔNG tách, KHÔNG re-associate → chỉ GIẢM volume:

🟢 SSRI (thuốc tăng serotonin):
  → Serotonin tăng → certainty bias tăng → schema fire NHẸ hơn
  → + Amygdala BỚT nhạy → trigger detect GIẢM
  → = "Volume giảm" → trauma VẪN CÒN nhưng "chịu được"
  → ⚠️ Dừng thuốc → CÓ THỂ quay lại (schema chưa đổi)
  → = "Thuốc giảm đau" hữu ích khi trauma QUÁ NẶNG → cần giảm TRƯỚC → rồi therapy

🟢 Beta-blockers (propranolol):
  → Block NE receptor → body KHÔNG respond fight/flight
  → Uống → recall trauma → body KHÔNG đáp → "nhớ mà không sợ"
  → 🟢 Kindt et al. (2009): propranolol + recall → fear GIẢM đáng kể
  → = Tách emotional response QUA chemistry
  → ⚠️ Cần đúng timing + chuyên gia hướng dẫn

🟢 Exercise:
  → BDNF tăng → neuroplasticity → dễ re-associate hơn
  → Endorphin → body state tốt → trauma fire ÍT
  → Cortisol regulated → repair TỐT hơn
  → = TẠO ĐIỀU KIỆN tốt cho hướng A, B, D

🟢 Sleep (ĐẶC BIỆT REM):
  → 🟢 Walker (2009): REM sleep → brain reprocess emotional memories
  → Emotional charge GIẢM DẦN qua NHIỀU đêm REM
  → = "Thời gian chữa lành" thật ra = "NHIỀU ĐÊM REM reprocess"
  → ⚠️ PTSD → ngủ KÉM → REM thiếu → reprocess KHÔNG đủ
  → = "Thời gian KHÔNG chữa lành" = vì thiếu REM
  → = Fix sleep = bước 1 quan trọng nhất cho mọi hướng chữa
```
