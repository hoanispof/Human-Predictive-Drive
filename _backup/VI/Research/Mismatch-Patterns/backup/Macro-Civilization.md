## 1. Mapping Cá Nhân → Văn Minh

### 1.1 Tại Sao Scale Được?

🟡 Văn minh = tập hợp cá nhân + cấu trúc tổ chức. Cấu trúc tổ chức (luật, kinh tế, văn hóa) hoạt động như TẦNG -1 cho cá nhân — và cũng CẦN PE tập thể để duy trì.

🟢 Peter Turchin (2003, 2023): đế quốc sụp khi mất "asabiyyah" (social cohesion + collective drive) = PE tập thể giảm.
🟢 Jared Diamond (2005) — Collapse: văn minh sụp khi không adapt = prediction tập thể bị khóa.

## 2. PEM: Prediction Environment Mismatch

```

  Supply ngắn HẠN CHẾ (phải effort)      Supply ngắn VÔ HẠN (MXH, AI, tiêu dùng)
  Threshold THẤP (ít stimulation)        Threshold TĂNG liên tục (dopamine flooding)
  Prediction xa CÓ SẴN (tôn giáo...)    Prediction xa SỤP (secularization)
  Oxytocin DỒI DÀO (cộng đồng nhỏ)     Oxytocin THIẾU (cô lập, MXH ≠ thật)
  Cortisol NGẮN HẠN (nguy hiểm rõ)      Cortisol MÃN TÍNH (bất ổn trừu tượng)
  ECP structure RÕ RÀNG                  ECP structure SỤP + internal CHƯA XÂY
  Chunks shared (cộng đồng nhỏ)          Chunks phân mảnh (specialist society)

v5.5 — 2 inverted-U tập thể:
  Cortisol inverted-U: savanna = ascending limb (acute, controllable)
                       hiện đại = descending limb (chronic, uncontrollable)
  Dopamine inverted-U: savanna = tối ưu (explore VÀ exploit cân bằng)
                       hiện đại = hơi cao → quá cao (flooding → threshold tăng → addiction)

  Encoding Modality bias:

    Savanna: loop chủ yếu CỤ THỂ (đói → kiếm ăn → no → đóng)
    Hiện đại: loop chủ yếu TRỪU TƯỢNG ("thành công?", "đủ tốt?", "tương lai?")

    → = Điều kiện tối ưu cho prediction suppression tập thể (Depression-Predictive-Model.md §2)

## 4. Habituation Blindness Quy Mô Xã Hội

### 4.1 "Prosperity Blindness"

🟡 Core.md §6.7: PE=0 ≠ Value=0. Dopamine phasic chỉ fire cho PE. Scale lên xã hội:

```
THỂ CHẾ DÂN CHỦ:
  Thế hệ 1 (lập): PE CỰC MẠNH (freedom = prediction error lớn)
  Thế hệ 2 (hưởng): PE vừa (dân chủ đã predicted → PE giảm)
  Thế hệ 3+ (thừa kế): PE ≈ 0 (dân chủ = default → invisible)
  → "Democracy is the worst form of government, except for all the others"
    = BIẾT value nhưng KHÔNG CẢM (PE=0)

HÒA BÌNH:
  Chiến tranh → hòa bình: PE CỰC MẠNH (relief + hope)
  20 năm hòa bình: PE ≈ 0 (predicted)
  → "Hòa bình nhàm chán" = Habituation Blindness
  → Mất hòa bình → PE ÂM CỰC MẠNH (prediction break)
  → "Chỉ trân trọng khi mất" = KHÔNG PHẢI thờ ơ — là CƠ CHẾ dopamine

THỊNH VƯỢNG KINH TẾ:
  Từ nghèo → trung lưu: PE MẠNH (predictive improvement)
  Đã trung lưu: PE ≈ 0 (predicted → invisible)
  🟢 Easterlin Paradox: GDP tăng nhưng happiness KHÔNG tăng sau threshold
  → Framework: GDP tăng → predicted → PE=0 → "GDP tăng = vô nghĩa"
  → NHƯNG GDP GIẢM → PE âm CỰC MẠNH (prediction break)
```

### 4.2 Hệ Quả Chính Trị

```
🔴 PROSPERITY BLINDNESS → CHÍNH TRỊ:

  Dân KHÔNG CẢM value thể chế hiện tại (habituated)
  → "Thay đổi" (novelty) = PE > 0 → HẤP DẪN
  → "Giữ nguyên" (predicted) = PE = 0 → NHÀM CHÁN
  → Populist THẮNG vì HỨA THAY ĐỔI (PE dương) → dân CHỌN PE

  NGUY CƠ:
    Thay đổi phá thể chế → prediction break → PE âm CỰC MẠNH
    → "Tại sao người ta vote phá hoại chính mình?"
    → = Habituation Blindness: KHÔNG CẢM value → chọn novelty → MẤT value → tiếc

  v5.5 — Chunk Topology lens:
    Thể chế = shared foundation chunks (luật, quyền, quy trình)
    Habituated → INVISIBLE → dễ bị thay/phá
    Phá thể chế = phá shared foundation → CASCADE SỤP mọi domain
    → GIỐNG mất đức tin (Religion.md §9) ở quy mô xã hội
```

### 4.3 Counteract: Institutional PE Maintenance

```
🔴 GIỐNG ritual chống habituation (Religion.md §6):

  Thể chế CẦN "ritual" để duy trì PE > 0:
    Bầu cử = ritual dân chủ (PE từ participation)
    Ngày lễ quốc gia = ritual quốc gia (PE từ shared celebration)
    Civic education = "tại sao thể chế này" (rebuild PE từ understanding)

  NHƯNG: ritual thể chế ĐANG GIẢM:
    Bầu cử = "cả hai tệ" → PE ≈ 0
    Ngày lễ = "nghỉ ngơi" thay vì "celebration" → PE giảm
    Civic education = minimal → generation mới không biết "tại sao"

  → Thể chế habituate + ritual giảm → vulnerability = mất dù có value
  → Giải pháp: thiết kế "institutional ritual" mới (§14)
```

### 5.3 Nghịch Lý Cortisol-Dopamine: "Nghỉ Hưu Tập Thể"

🔴 **Cortisol ∝ thực thi. Dopamine ∝ sáng tạo. Chúng CỘNG TÁC, không đối lập.**

```
INVERTED-U Ở QUY MÔ VĂN MINH:
  Cortisol moderate tập thể → amplify dopamine → thực thi + sáng tạo CỘNG TÁC
  Cortisol quá thấp tập thể → dopamine mất amplifier → sáng tạo CŨNG sụp

  🟢 Rohwedder & Willis (2010): nghỉ hưu → cognitive decline
  🟢 Fratiglioni et al. (2004): hoạt động duy trì → giảm dementia risk
  → Thực thi KHÔNG CHỈ tạo output — thực thi DUY TRÌ hệ thống sinh ý tưởng

QUỸ ĐẠO LỊCH SỬ QUA 2 INVERTED-U:
  Săn bắn:     Cortisol >> Dopamine (gần như toàn thực thi)
  Nông nghiệp: C > D  (bớt thực thi → nghệ thuật, triết học)
  Công nghiệp: C ≈ D  (ĐỈNH inverted-U — innovation cực đại)
