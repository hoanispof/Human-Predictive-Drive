## Research Summary (~55-65 citations verified)

### Cluster A — Core Neuroscience

```
🟢 ESTABLISHED:
  LeDoux 1996, 2000 (Annual Rev Neurosci 23) — fear conditioning, amygdala 2 pathways
    Amygdala = implicit emotional memory (automatic, durable, extinction-resistant)
    Hippocampus = explicit episodic memory (contextual, declarative, malleable)
    Hippocampal lesion → no contextual fear. Amygdala lesion → no cued fear.
  
  Gilbertson 2002 (Nature Neurosci 5:1242) — twin study
    Smaller hippocampus = PRE-EXISTING vulnerability, not just consequence
    Unexposed co-twin of PTSD veteran ALSO had smaller hippocampus
  
  Smith 2005 (Hippocampus 15:798) — meta-analysis 13 studies
    Left hippocampus 6.9-7.2% smaller, right 6.6-7.0% smaller in PTSD
  
  Bremner 1995 (Am J Psychiatry 152:973) — first MRI demonstration
    8% right hippocampal reduction in 26 combat veterans
  
  Kim & Diamond 2002 (Nature Rev Neurosci 3:453) — temporal dynamics model
    NE surge → enhances amygdala LTP (long-term potentiation)
    High cortisol GR → impairs hippocampal LTP, facilitates LTD
    = Amygdala encoding enhanced + hippocampus suppressed simultaneously
  
  Brewin 1996 (Psych Review 103:670) + Brewin 2010 (Psych Review 117:210)
    Dual Representation Theory (DRT → revised R-DRT)
    S-reps (Sensation-based) = dorsal stream, sensory without context
    C-reps (Contextualized) = ventral stream, full episodic context
    Flashback = S-rep fires without C-rep → "re-experiencing" not "remembering"
  
  Yehuda 1990, 2001, 2004 (Am J Psychiatry) — HPA axis paradox
    PTSD = LOWER cortisol baseline (opposite chronic stress!)
    Enhanced negative feedback (GR upregulated)
    Elevated CRH but system overshoots → low cortisol stable state
    Low cortisol → prevents normal extinction of fear memories
```

### Cluster B — Flashback + Extinction + Dissociation

```
🟢 ESTABLISHED:
  Rauch et al. 1996 (Arch Gen Psychiatry 53:380) — PET symptom provocation
    Amygdala + insula hyperactivation, PFC hypoactivation
    Broca's area deactivation = "speechless terror"
  
  Ehlers & Clark 2000 (Behav Res Ther 38:319) — cognitive model
    Traumatic memories encoded fragmented, data-driven mode
    Lack temporal context → "here and now" quality of flashbacks
  
  Bouton 2004 (Learning & Memory 11:485) — extinction ≠ erasure
    New inhibitory learning competes with (not replaces) fear memory
    4 return phenomena: spontaneous recovery, renewal, reinstatement, rapid reacquisition
  
  Milad et al. 2009 (Biol Psychiatry 66:1072) — PTSD extinction recall
    PTSD patients: intact extinction learning, IMPAIRED extinction recall
    Reduced vmPFC + hippocampal activation during recall
  
  Lanius et al. 2010 (Am J Psychiatry 167:640) — dissociative subtype
    Standard PTSD = undermodulation (amygdala↑, PFC↓)
    Dissociative = overmodulation (PFC hyperactive, amygdala suppressed)
    13-30% prevalence. More common in early/chronic trauma.
    DSM-5 formally recognized dissociative specifier
  
  Shin 2006 — PTSD neuroimaging
    PFC activity↓ + Amygdala activity↑ confirmed
  Vyas 2002 — chronic stress → amygdala dendrites GROW
```

### Cluster C — Sleep + Treatment + C-PTSD + Comorbidity

```
🟢 ESTABLISHED:
  Walker & van der Helm 2009 (Psych Bulletin 135:731) — REM "overnight therapy"
    REM = reactivate emotional memories in low-NE environment → valence decoupled
    PTSD: elevated NE during sleep → disrupts emotional decoupling
    Nightmares = failed emotional processing
  
  Germain 2013 (Am J Psychiatry 170:372) — sleep as hallmark
    PTSD: more Stage 1, less SWS, greater REM density
    Sleep disruption actively interferes with exposure therapy outcomes
  
  Foa & Kozak 1986 (Psych Bulletin 99:20) — emotional processing theory
    Fear structure activated + corrective information → new non-fear structure
  
  Craske et al. 2014 (Behav Res Ther 58:10) — inhibitory learning model
    Extinction = new inhibitory memory, NOT erasure
    Fear reduction DURING session NOT necessary for good outcome
    Expectancy violation > habituation
  
  van den Hout & Engelhard 2012 (J Exp Psychopathol 3:724) — EMDR mechanism
    Working memory taxation = active ingredient
    Bilateral stimulation NOT specifically necessary
    Any demanding dual task produces comparable effects
  
  Nader, Schafe & LeDoux 2000 (Nature 406:722) — reconsolidation
    Recalled memory → labile → protein synthesis required to restabilize
    Window ~6 hours. Requires prediction error.
  
  Schiller et al. 2010 (Nature 463:49) — human reconsolidation
    Extinction within reconsolidation window → no fear return at 1 year
    Replication mixed (PMC7115860 reanalysis)
  
  Herman 1992 (J Traumatic Stress 5:377) — Complex PTSD concept
  ICD-11 (2022) — C-PTSD formal recognition (code 6B41)
    DSO: affect dysregulation + negative self-concept + relational difficulties
  
  Kessler et al. 2005 (Arch Gen Psychiatry 62:593) — NCS-R prevalence
    US lifetime PTSD = 6.8% (women 10.4%, men 5.0%)
  Koenen et al. 2017 (Psych Medicine 47:2260) — WHO global
    Global lifetime = 3.9%. Only ~20-30% trauma-exposed develop PTSD.
  
  Khantzian 1985 (Am J Psychiatry 142:1259) — self-medication hypothesis
    PTSD × SUD co-occurrence 30-50%
    Substance choice fits emotional state
  
  Yehuda et al. 2016 (Biol Psychiatry 80:372) — intergenerational
    FKBP5 methylation changes in Holocaust offspring
    Prenatal pathway = strongest human evidence
    Small samples, controversy, replication inconsistent

🔴 HYPOTHESIS (framework):
  Chunk context-tag 4 metadata types (formalization)
  2 chunk types (contextual vs context-free)
  Treatment unification via context-tag re-attachment
  Body-override-PFC as temporal sequence (not override per se)
```
