## 2. PHÂN TÍCH TÁC ĐỘNG

### Trust.md v1.0 — TOP IMPACTS cho Education-Mechanism:

**① Education-Mechanism: Trust là trung tâm arc design**
- "Love the teacher → learn the lesson" (CT §7.3) ĐÃ CÓ nhưng thiếu Trust.md backing
- Trust = Amplifier for VALUE stream → arc design PHẢI build trust BEFORE content
- 4 Compile Pathways (CT §6): pathway ② (Trust+Moderate) = phổ biến nhất → cần Trust.md depth
- Critical Asymmetry: PFC→Entity-Valence SLOW, Entity-Valence→PFC FAST → teacher relationship = INVESTMENT

**② Connection-Education: Trust = FOUNDATION cho tất cả 5 pillars**
- Pillar ① Understanding others: requires trust to practice Self-Pattern-Modeling
- Pillar ② Building connections: trust = prerequisite for sustained connection
- Pillar ③ Selecting relationships: trust calibration = core skill (Default→Calibrated)
- Pillar ④ Sustaining: 5-source trust maintenance = sustainability mechanism
- Pillar ⑤ Groups: Coordination-Node trust cascade, position vs person trust

**③ Compile-Type-Learning: Trust Compile = 1 of 3 types**
- Trust.md formalizes WHAT trust is, HOW it forms → deepens Trust Compile analysis
- Trust ≠ Valence distinction → clarifies "trust a teacher but not like them" cases
- 3 sub-dimensions → different trust dimensions compile at different speeds

**④ Expansion-Saturation-Crisis + Arms-Race: Institutional trust collapse**
- Education system = trust infrastructure
- Credential inflation = trust signal degradation
- "Học làm gì?" = collective trust chain GÃY

### Self-Observation.md v1.0 — TOP IMPACTS cho Education-Mechanism:

**① Education-Mechanism: Self-Observation = metacognitive feedback**
- Arc design must include Self-Observation development (not just content delivery)
- Self-Observation Mức determines what metacognitive strategies work per student
- Teacher as External Scaffold (Pathway 2) per Mức level

**② Connection-Education: Self-Observation = KEYSTONE cho social skills**
- Self-Observation §7.1: fail → cascade Self-Pattern-Modeling, Dual Check
- Pillar ① requires Self-Observation as prerequisite (Bird & Cook 2013)
- Pillar ② requires self-awareness for relationship quality
- Per-age Self-Observation capacity limits social skill training design

**③ Compile-Type-Learning: Self-Observation for learning quality**
- Expertise Compile needs Self-Observation (PFC-directed → need body feedback)
- Body-check during learning = Self-Observation Mức 3+ application
- Compiled Quality × Self-Observation = how learning was done affects what can be observed
