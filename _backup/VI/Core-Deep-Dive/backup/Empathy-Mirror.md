## 4. Vulnerability Cue Detection — Tại Sao Baby/Pet Trigger Mạnh

```
🟡 Mirror strength KHÔNG đồng đều — có GRADIENT rõ:

  GRADIENT (mạnh → yếu):
    Em bé ruột (gene carrier + vulnerability CỰC CAO) → mirror MẠNH NHẤT
    Em bé quen (vulnerability cao + chunks compiled) → rất mạnh
    Em bé LẠ (vulnerability cao + NO chunks) → VẪN mạnh (baby schema)
    Pet nhỏ (vulnerability + expressiveness) → mạnh
    Trẻ nhỏ quen → mạnh
    Người trưởng thành yếu ớt (bệnh, già, xa nhà) → trung bình
    Pet lớn → trung bình
    Người trưởng thành khỏe mạnh → yếu
    Cây cối → rất yếu (có sống nhưng gần zero biểu cảm)
    Đồ vật → zero (trừ khi nhân cách hóa)


  4 YẾU TỐ QUYẾT ĐỊNH mirror strength:

  ① LIVING BEING — phải có sự sống (hoặc perceived có):
     → Living → mirror CÓ
     → Non-living → mirror KHÔNG CÓ (trừ khi project)
     → = Prerequisite: phải là sinh vật TRƯỚC khi mirror fire

  ② VULNERABILITY CUES — càng yếu ớt → mirror CÀNG MẠNH:
     → 🟢 Baby Schema / Kindchenschema (Lorenz 1943):
       Mắt to, mặt tròn, mũi nhỏ, body nhỏ, đầu to so với thân
       → Trigger nurturing response ở NGƯỜI LỚN nói chung 🟢
       → Nucleus accumbens fire khi nhìn infant faces — kể cả non-parents
         (Glocker 2009) 🟢
     → Tại sao?: vulnerability = POTENTIAL DEFICIT LỚN
       → Body auto-prioritize: "sinh vật này CÓ THỂ chết nếu không ai giúp"
       → Mirror MẠNH = body BẮT BUỘC phải chú ý
       → = Evolutionary: gene survival CẦN adult chăm infant → hard-wire
     → Mở rộng: pet nhỏ CÓ baby schema features (mắt to, body nhỏ, phụ thuộc)
       → = Tại sao mèo con/chó con trigger MẠNH hơn mèo/chó trưởng thành

  ③ EXPRESSIVENESS — càng biểu hiện rõ state → mirror CÀNG MẠNH:
     → Mặt người: CỰC expressive → mirror MẠNH
     → Mặt chó: khá expressive → mirror tốt
     → Mặt mèo: ít expressive hơn → mirror OK
     → Cá: gần zero biểu cảm → mirror rất yếu
     → Cây: zero biểu cảm → mirror gần zero
     → = Mirror CẦN INPUT để fire — ít input = ít mirror
     → 🟢 Facial expressions = primary trigger cho empathic response
       (Dimberg 2000: facial EMG mimicry = automatic, 300ms) 🟢

  ④ SIMILARITY / FAMILIARITY — càng giống/quen → mirror CÀNG MẠNH:
     → Cùng loài > khác loài (species overlap, Personal-Melody §12)
     → Quen > lạ (chunks compiled → simulate CHÍNH XÁC → mirror MẠNH)
     → Schema mapping: "ngang tuổi con mình" → map vào "con" slot → AMPLIFY
     → = "Coi nó như con" = body dùng CÙNG mirror circuit, target KHÁC
     → 🟢 In-group empathy bias: empathy mạnh hơn cho in-group
       (Xu 2009: ACC response mạnh hơn khi thấy in-group member đau) 🟢


  CÁC YẾU TỐ NHÂN với nhau (KHÔNG cộng):
    Baby ruột = living ✅ × vulnerability CỰC CAO × expressiveness CAO × familiarity CỰC CAO
    → = Mirror MẠNH NHẤT → drive chăm sóc MẠNH NHẤT
    Em bé lạ = living ✅ × vulnerability CAO × expressiveness CAO × familiarity THẤP
    → = VẪN mạnh (3/4 yếu tố cao) → giải thích sao người lớn thấy em bé lạ → cười
    Cá cảnh = living ✅ × vulnerability THẤP × expressiveness GẦN ZERO × familiarity THẤP
    → = Mirror rất yếu → "nuôi cá đẹp" chủ yếu Aesthetic, ít Connection

```

```
    → "Thấy chiến tranh trên TV + tôi KHÔNG LÀM ĐƯỢC GÌ"

  → = Perceived ability là "VAN ĐIỀU KHIỂN" giữa mirror dissonance và drive
  → = Ability cao → van MỞ → drive → action → resolve
  → = Ability thấp → van ĐÓNG → cắt input → defense
```

## 11. Honest Assessment

```
  ESTABLISHED (🟢):

       anterior insula + ACC fire khi QUAN SÁT người khác đau

    🟢 Interpersonal synchrony (Feldman 2007): physiology SYNC khi interact
    🟢 Theory of Mind circuits (Frith & Frith 2006): mPFC + TPJ
    🟢 In-group empathy bias (Xu 2009): ACC response mạnh hơn cho in-group
    🟢 Facial mimicry (Dimberg 2000): automatic, 300ms, pre-conscious
    🟢 Social buffering (Kiyokawa 2004): presence giảm stress response
    🟢 Cross-species empathy behavior (Church 1959, Masserman 1964):
       chuột, khỉ biểu hiện empathic response
    🟢 mPFC dùng own schema để simulate others (Frith & Frith 2006)
```
