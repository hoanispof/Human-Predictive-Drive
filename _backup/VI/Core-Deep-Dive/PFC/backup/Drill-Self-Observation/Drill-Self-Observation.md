## Round 1 — Self-Observation LÀ GÌ?

### Insight 1: Self-Observation = Feeling khi attention hướng SELF

**Feeling** = INTERFACE (output channel): "WHAT PFC sees when observing body-chunk interaction." Bao gồm observe BẤT KỲ body-chunk interaction — kể cả khi target là domain (giải toán, nếm thức ăn).

**Self-Observation** = Feeling khi TARGET of observation = SELF (body-state, emotional state, cognitive state).

Phân biệt theo **attention direction**:
- Feeling hướng DOMAIN: PFC observe body reaction để ĐÁNH GIÁ domain input. "Bài toán này khó → tôi frustrated → đổi hướng." Target = toán.
- Feeling hướng SELF: PFC observe body state để HIỂU MÌNH. "Tôi frustrated → tại sao? → à tôi mệt, không phải toán khó." Target = self.

### Insight 2: Self-Observation ≠ Body-Knowing Inward (nhưng overlap)

**Body-Knowing Inward** (Body-Knowing.md §2): COMPILED recognition — "tôi biết tôi buồn" (automatic, cost≈0). Requires Hebbian-reinforced patterns.

**Self-Observation** bao gồm thêm:
- **Fresh**: "cái gì đang xảy ra trong người tôi?" (effortful, cost>0, đang tìm hiểu)
- **Pre-Body-Knowing**: "có gì đó nhưng không biết gì" (vague, hunch, nagging)
- **Meta**: "tôi đang observe cái gì? Observation này có tin được không?"

→ Body-Knowing Inward ⊂ Self-Observation. Self-Observation rộng hơn — bao gồm cả phần chưa compiled.

### Insight 3: Formal definition

```
Self-Observation = f(Interoception × PFC_Encoding × Attention_Direction=SELF)

  Interoception = anterior insula readout (Simulation-Engine Component 1)
               = direct body-state access (UNIQUE cho self-target)
  
  PFC_Encoding  = signal vượt threshold + PFC encode được
               = Feel-Consciousification → Feel-Observation (Feeling 7-layer)
  
  Attention_Direction = target = SELF (body-state, emotional state, thought pattern)
```

Nếu thiếu bất kỳ component nào:
- Interoception bị hỏng → Alexithymia (Body-Knowing.md §6 Case D)
- PFC Encoding fail → Body fires nhưng PFC không biết (Case A: amygdala hijack)
- Attention ở domain → Feeling hướng ngoài, không phải Self-Observation

### Insight 4: Self-Observation trên Compiled/Fresh spectrum

| Compiled Self-Observation | Fresh Self-Observation |
|--------------------------|----------------------|
| "Tôi biết tôi đói" (instant) | "Cái gì đang xảy ra trong người?" (searching) |
| Cost ≈ 0, ~95% hàng ngày | Cost > 0, ~5% exception |
| = Body-Knowing Inward | = PFC-directed attention to body |
| Accuracy: high nếu domain-checked | Accuracy: variable, confabulation risk |

## Round 2 — Trigger Principle v2: Khi Nào Self-Observation Kích Hoạt?

### Insight 5: Self-Observation = f(Prediction-Delta × Solution Availability)

**Nguyên lý cốt lõi:** Self-Observation KHÔNG chạy liên tục. Nó kích hoạt khi có PREDICTION-DELTA (expected ≠ actual).

Cơ chế (Body-Feedback-Mechanism.md §3): VTA detect expected ≠ actual → delta fire → signal vượt PFC threshold → PFC notice → Self-Observation triggered.

**3 DẠNG DELTA (v2 — refined):**

| Dạng | Mô tả | Ví dụ | Valence |
|------|-------|-------|---------|
| **① Negative delta** (dissonance) | Actual TỆ HƠN expected | Đau, thất bại, mất mát, khó chịu | Avoidance — "thoát khỏi" |
| **② Micro-negative delta** (sankhara-dukkha) | Subtle "background hum" — actual KHÔNG SAI nhưng "thiếu gì đó" | "Cuộc sống ổn nhưng sao trống trải." Người thành đạt nhưng vô nghĩa | Avoidance nhẹ — thường bị DISMISS |
| **③ Positive delta** (aspiration) | Thấy cái GÌ ĐÓ hấp dẫn mà mình CHƯA CÓ | Thích Nhất Hạnh thấy ảnh Phật mỉm cười. "Tôi muốn được bình an như thế" | Approach — "hướng tới" |

**Tại sao 3 dạng?**

Framework Gap-Direction.md v2.0 §1: Gap = shape of hole in chunk network. Gap có DIRECTION (hướng), không chỉ có valence.

- **Negative delta** = gap CÓ SẴN bị vi phạm (expected → actual khác → dissonance)
- **Micro-negative delta** = gap TÍCH TỤ từ Miss MỜ (Body-Feedback-Mechanism §3.2ⓑ — delta/ngày quá nhỏ → compound)
- **Positive delta** = gap MỚI ĐƯỢC TẠO bởi external input (thấy cái mới → chunks install → gap emerge: current_self ≠ desired_self)

Cả 3 đều là prediction-delta. Cả 3 đều có thể trigger Self-Observation. Nhưng:
- ① phổ biến nhất (74% meditators bắt đầu vì stress/mental health — PMC9728621)
- ② dễ bị dismiss nhất (Somatic-Articulation-Loop Stage 3 Reaction A)
- ③ ít được framework address (hiện tại Trigger Principle nghiêng về avoidance)

**Nếu body-base SMOOTH (predicted = actual):**
```
Prediction-delta = 0
→ Signal DƯỚI threshold
→ PFC KHÔNG BIẾT
→ Self-Observation = ZERO
→ Không cần, không có, không biết
```

**Nếu prediction-delta > 0 VÀ solution SẴN CÓ (compiled):**
```
Delta vượt threshold → PFC notice
→ Compiled action schema fire → act → done
→ Self-Observation = Mức 2 (nhẹ, nhanh)
→ Ví dụ: buồn đái → gọi mẹ
```

**Nếu prediction-delta > 0 VÀ solution CHƯA CÓ:**
```
Delta vượt threshold → PFC notice
→ KHÔNG CÓ compiled action → PFC phải HOLD open
→ HOLD lâu = observe SÂU hơn (buộc phải, vì chưa có answer)
→ Self-Observation escalate: Mức 3 → 4 → 5
→ Ví dụ: bồn chồn thi → phân tích → thử → adjust
```

**Nếu prediction-delta RẤT LỚN VÀ solution KHÔNG RÕ:**
```
Delta rất lớn + không có hướng giải quyết
→ PFC HOLD rất lâu + uncertainty cost cao
→ Self-Observation bị ĐẨY đến mức existential
→ Mức 5+: questioning cả framework sống
→ Ví dụ: "học để làm gì?" / "tôi đã sống sai hướng?"
```

### Insight 6: Oxy — ví dụ hoàn hảo cho Trigger Principle

```
Hít thở hàng ngày:
  Predicted: oxy bình thường → Actual: oxy bình thường
  Delta = 0 → PFC KHÔNG BIẾT → Self-Observation = ZERO
  → Trẻ con KHÔNG THỂ biết mình đang thở oxy

Nghẹt mũi khi ốm:
  Predicted: thở bình thường → Actual: khó thở
  Delta = NEGATIVE → PFC threshold CROSSED
  → Self-Observation TRIGGERED: "tôi đang khó thở"

Về quê sau ốm/sau ô nhiễm:
  Predicted: thở khó (compiled từ trải nghiệm gần đây)
  Actual: thở dễ, sảng khoái → Delta = POSITIVE
  → PFC threshold CROSSED
  → Self-Observation TRIGGERED: "ôi, thở trong lành quá!"
  → Lần ĐẦU TIÊN "thấy" giá trị oxy
```

Body-base smooth = invisible. Chỉ khi smooth bị BREAK (qua đau khổ/thiếu thốn rồi quay lại) thì PFC mới "thấy."

### Insight 7: Dissonance escalation — buồn đái → phòng thi → "học để làm gì?"

| Dissonance | Mức kích hoạt | Ví dụ | Solution | Self-Observation depth |
|-----------|--------------|-------|----------|----------------------|
| ZERO | 0 | Thở oxy. Học suôn sẻ. Sinh viên thuận lợi | Không cần | ZERO — PFC không biết |
| NHẸ + RÕ | 2 | Buồn đái. Quần ướt khó chịu | Dễ: gọi mẹ (compiled) | Nông — recognize state → act |
| TRUNG BÌNH + RÕ | 3-4 | "Hay buồn đái khi lo." Ảnh hưởng công việc | Cần phân tích pattern | Trung bình — so sánh nhiều episodes |
| TRUNG BÌNH + MỜ | 4 | Bồn chồn thi → làm bài kém | Cần đối chiếu, thử strategies | Sâu — controlled comparison |
| MẠNH + MỜ | 4-5 | Học mãi không giỏi + mẹ mắng | Questioning mục đích | Rất sâu — existential |
| RẤT MẠNH + MỜ | 5+ | Ra trường khó kiếm việc | Restructure cả worldview | Sâu nhất — "tôi là ai?" |

### Insight 8: Solution difficulty ĐẨY Self-Observation depth

Observation sâu dần KHÔNG phải vì người muốn observe sâu — mà vì PFC bị BUỘC phải observe sâu khi không có compiled solution:

```
Dissonance trigger → PFC tìm compiled solution
  ├→ CÓ solution sẵn → act → done → Self-Observation DỪNG (Mức 2)
  └→ KHÔNG CÓ solution → PFC HOLD open (PFC-Operations.md §2)
       → HOLD = buộc continue observe (chưa có answer nên chưa thể act)
       → Observe lâu hơn = observe SÂU hơn
       → Sâu hơn = phát hiện thêm patterns → Mức 3, 4
       → Vẫn chưa giải quyết → questioning causes → Mức 5
```

PFC-Operations.md §2: HOLD = PFC amplify pattern mới/yếu. Cost = ① PFC draft. Lâu quá → ③ uncertainty cost (cortisol).

→ Self-Observation depth = f(dissonance_strength × solution_difficulty × HOLD_duration).

### Insight 9: Smooth body-base = Self-Observation KHÔNG CẦN THIẾT (và đây KHÔNG phải vấn đề)

Ví dụ từ discussion:

**Học bài suôn sẻ:**
- Trẻ thích học, nỗ lực thêm là được, bố mẹ hỗ trợ
- Predicted (học → tiến bộ) = Actual (tiến bộ) → Delta ≈ 0
- Self-Observation = Mức 2 tối đa ("biết mình hiểu / chưa hiểu bài")
- → "Cứ thế mà học thôi" — và HOÀN TOÀN ỔN

**Chỉ khi smooth BỊ BREAK:**
- Ra trường → khó kiếm việc → Predicted ≠ Actual → Delta LỚN
- Self-Observation Mức 4-5 BỊ KÍCH HOẠT (không phải "trưởng thành" — là dissonance mạnh hơn)
- "Học để làm gì?" = PFC buộc phải observe sâu vì compiled framework sống bị challenge

**Implication:** Nhiều người sống suôn sẻ → KHÔNG phát triển Self-Observation sâu. KHÔNG phải vì "nông cạn" — vì body-base KHÔNG CÓ dissonance đủ mạnh để trigger. Nếu cuộc sống thật sự smooth, Self-Observation sâu KHÔNG CẦN THIẾT.

Vấn đề CHỈ xảy ra khi:
- (a) Dissonance CÓ nhưng bị DISMISS (Somatic-Articulation-Loop Stage 3 Reaction A)
- (b) Dissonance SẼ ĐẾN nhưng chưa tới (thị trường lao động sẽ đổi, sinh viên chưa biết)
- (c) Compiled Suppress che giấu dissonance (PFC-Operations §8: "flat" nhưng body responsive)

### Insight 10: Miss MỜ — delta nhỏ tích tụ vượt threshold

Body-Feedback-Mechanism.md §3.2ⓑ: Chunk-Miss MỜ = delta mỗi ngày quá nhỏ → dưới threshold → PFC không detect → nhưng delta TÍCH TỤ.

```
Ngày 1: thở hơi ô nhiễm → delta = 0.1 → dưới threshold → không biết
Ngày 2: thở hơi ô nhiễm → delta tích = 0.2 → vẫn dưới
...
Ngày 30: delta tích = 3.0 → "sao dạo này uể oải quá"
→ PFC CROSS threshold → Self-Observation TRIGGERED
→ Nhưng PFC không biết NGUYÊN NHÂN (vì mỗi ngày delta quá nhỏ)
→ = "chán mà không biết vì sao" (Boredom.md core mechanism)
```

Oxy example tương tự: người sống ở thành phố ô nhiễm KHÔNG BIẾT không khí xấu (delta/ngày quá nhỏ). Chỉ khi về quê (delta ĐỘT NGỘT lớn) thì mới "thấy."

→ Miss MỜ tích tụ = Self-Observation BỊ DELAY. Body biết trước PFC rất lâu.

### Insight 10b: 2 Pathways kích hoạt Self-Observation

Self-Observation CÓ 2 PATHWAY kích hoạt, không chỉ 1:

**PATHWAY 1 — INTERNAL (bottom-up, tự trigger):**
```
Prediction-delta vượt threshold → PFC tự notice → Self-Observation kích hoạt
= "Tự mình nhận ra"
= Cần delta > threshold (hoặc delta tích tụ qua Miss MỜ)
= DEFAULT mode cho hầu hết Self-Observation hàng ngày
```

**PATHWAY 2 — EXTERNAL (top-down, scaffolded):**
```
Người khác DIRECT attention → amplify signal → Self-Observation kích hoạt
= "Ai đó giúp mình nhận ra"
= KHÔNG CẦN delta tự vượt threshold (external amplify signal)
= Cần external agent: caregiver, teacher, therapist, friend, AI, culture
```

**Ví dụ Pathway 2:**
- **Mẹ:** "Con buồn đái hả?" → label body state cho trẻ → trẻ compile: cảm giác này = "buồn đái" + action = "gọi mẹ." Scaffold CẢ HAI component (recognition + plan)
- **Bố (phòng thi):** Con thi kém → khóc → "con kém" (Feel-Explanation sai). Bố KHÔNG accept surface → hỏi: "phòng thi có lo không? thiếu ngủ? bài khó?" → Bố DIRECT attention con đến body-state → offer candidate explanations (= Somatic-Articulation-Loop Stage 4) → con body-confirm → tìm ra root cause → bố gợi ý solution
- **Therapist:** Patient nói "tôi ổn" → therapist: "nhưng tôi thấy bạn hay chạm tay lên ngực khi nói về X" → AMPLIFY signal mà patient tự mình miss
- **Bạn thân:** "Ơ dạo này mày lạ quá" → external observation mà PFC tự mình KHÔNG detect (vì compiled suppress hoặc delta quá nhỏ)
- **AI:** Prompt questions → direct attention → scaffold Mức 4-5 (pattern recognition, meta-reflection)

**Per-level scaffold:**

| Mức SO | External Scaffold | Agent |
|--------|-------------------|-------|
| **Mức 2** | Label body state: "con buồn đái hả?" | Mẹ/Bố |
| **Mức 3** | Prompt predict: "nếu uống nhiều nước thì sao?" | Mẹ/Bố/Giáo viên |
| **Mức 3-4** | Direct attention deeper: "phòng thi có lo không?" + offer candidates | Bố tốt / Giáo viên giỏi |
| **Mức 4** | Point out pattern: "con có thấy mỗi khi thi con hay lo không?" | Mentor / Therapist |
| **Mức 5** | Prompt meta-reflection: "explanation 'tôi kém' có đúng không? Body nói gì?" | Therapist / AI |
| **Mức 5-6** | Direct attention training: "observe thôi, đừng label" | Meditation teacher |

**Bố trong ví dụ phòng thi = làm CHÍNH XÁC những gì therapist làm:**

```
Somatic-Articulation-Loop mapping:
  Stage 3 (PFC notice): Bố HỎI → direct attention → con notice body-state
  Stage 4 (Catalyst):   Bố offer candidates: "lo? thiếu ngủ? bài khó?"
  Stage 5 (Body-confirm): Con: "ờ đúng, con lo lắm!" (body vote)
  Stage 6 (Sharpening):  Bố + con iterate → narrow down root cause
  Stage 7 (Felt shift):  Con: "à, không phải tôi kém, tôi bị lo!" (reframe)
```

**Implication quan trọng:**

Pathway 2 giải thích tại sao:
- **Therapy tồn tại**: scaffold Self-Observation cho người tự không thể (compiled suppress, alexithymia, untrained)
- **Good parenting quan trọng**: bố hỏi "lo không?" = scaffold Mức 4 cho con. Bad parenting: "sao con ngu thế?" = confirm Feel-Explanation sai → compile wrong self-model
- **Friendship sâu giúp**: bạn thân = external observer detect signal mà PFC miss
- **AI có giá trị**: prompt questions → amplify signals → scaffold Mức 4-5
- **Meditation teacher cần thiết**: "observe breath" = scaffold proactive mode

## Round 3 — 2-Axis Model: Development × Activation

### Insight 11: Gradient là 2 trục, không phải 1

Drill v1.0 mô tả gradient 0→6 như developmental stages tuyến tính. Reframe:

**Trục 1 — DEVELOPMENT CAPACITY (tăng theo tuổi + training):**
Khả năng TỐI ĐA mà người đó CÓ THỂ Self-Observe.

| Capacity | Emerge khoảng | Cần gì |
|----------|--------------|--------|
| Mức 0-1 | Prenatal → 6 tháng | Hardware (brainstem, basic insula) |
| Mức 2 | ~12-24 tháng | Compiled chunks trong domain đó |
| Mức 3 | ~3-5 tuổi | Simulation-Engine online |
| Mức 4 | ~8-15 tuổi (if trained) | Meta-chunks + episodic memory across instances |
| Mức 5 | ~15+ tuổi (if trained) | Introspective training (meditation, therapy, journaling) |
| Mức 6 | Decades experience | Domain feedback + intellectual humility |

**Trục 2 — ACTIVATION LEVEL (tăng theo dissonance × solution difficulty):**
Mức Self-Observation THỰC TẾ kích hoạt trong moment.

```
Activation = f(prediction_delta × solution_difficulty)
  delta = 0 → activation = 0 (dù capacity = 6)
  delta > 0 + solution compiled → activation = 2
  delta > 0 + no solution → activation escalate
  delta rất lớn + no solution → activation = capacity maximum
```

**Tương tác 2 trục:**

```
Người lớn capacity Mức 5:
  Hàng ngày smooth → activation Mức 2 (biết đói, biết mệt — compiled)
  Deadline stress → activation Mức 3-4 (analyze why stressed)
  Life crisis → activation Mức 5 ("tôi đang rationalize, không phải reflect")

Trẻ 3 tuổi capacity Mức 3:
  Dù dissonance RẤT MẠNH → activation TỐI ĐA = 3 (KHÔNG THỂ meta-observe)
  Vì chưa có meta-chunks → hardware ceiling
```

→ **Development capacity = ceiling. Activation = actual usage. Dissonance = fuel.**

### Insight 12: Observation + Planning INTERLEAVE ở mỗi mức

Mỗi mức Self-Observation đi kèm planning, nhưng tỷ lệ thay đổi:

**Mức 2 (nhẹ):** Observe 30% / Plan 70%
- "Buồn đái" (observe) → "gọi mẹ" (plan) → done
- Observation nhanh, plan compiled, cycle ngắn

**Mức 3-4 (trung bình):** Observe 60% / Plan 40%
- "Bồn chồn thi" (observe) → so sánh kinh nghiệm (observe thêm) → identify pattern (observe sâu) → thử strategy (plan) → observe kết quả → adjust
- Observation chiếm phần lớn, plan cần nhiều iteration

**Mức 5 (sâu):** Observe 80% / Plan 20%
- "Học để làm gì?" (observe) → questioning framework sống (observe sâu) → chưa có answer (HOLD) → tiếp tục observe → plan CHƯA RÕ
- Observation rất dài, plan khó/không rõ ràng

## §11 — Open Questions (GAPs)

### GAP-SO-1: Self-Pattern-Modeling applied to SELF
SPM = model OTHER using self-template. Mức 4 dùng cùng cơ chế model SELF across time. Đây là SPM, Self-Observation, hay concept riêng?

### GAP-SO-2: Meta-observation = body-mediated cycle
Cần explicit hóa: meta-observation = another body-mediated cycle, NOT PFC-observe-PFC. Implications cho consciousness, meditation, therapy.

### GAP-SO-3: Observation ≠ Regulation (Teen paradox proof)
Self-Observation (dlPFC) KHÔNG TỰ ĐỘNG dẫn đến regulation. Cần vmPFC-body connection. Teen = CAN observe, CAN'T regulate. Framework chưa phân biệt rõ.

### GAP-SO-4: Self-Observation = interoception conscious switch
Missing link: unconscious interoception → conscious Self-Observation. Feel-Consciousification là switch point nhưng chưa analyzed dưới lăng kính Self-Observation.

### GAP-SO-5: Education scaffold Mức 4-6
Framework scaffold Mức 2-3 tốt (caregiver labeling, prediction). Không cover Mức 4-6 (pattern observation, meta-observation, calibration).

### GAP-SO-6: Self-Observation × AI era
AI làm catalyst cho SAL (§5). Nhưng AI KHÔNG improve interoception (Component 1). AI contribute ở đâu trên gradient?
- Mức 4: CÓ (AI point out patterns)
- Mức 5: PARTIALLY (AI prompt meta-reflection)
- Mức 2: KHÔNG (cần body training)

### GAP-SO-7: Self-Observation accuracy paradox
Direct interoceptive access (advantage) + PFC Lawyer (disadvantage). Net: Self-Observation accuracy < Domain observation TRONG MỌI TRƯỜNG HỢP? (Domain có external check, self không có → therapy/friendship = external observer provides reality check)

### GAP-SO-8: Attention hardware × Self-Observation training
Detail-oriented: natural advantage. ADHD: natural disadvantage. Training strategies khác nhau per hardware? ADHD cần amplify signal (quiet environment, journaling) thay vì lower threshold (meditation — có thể counter-productive nếu bored)?

### GAP-SO-9: Miss MỜ accumulation → delayed Self-Observation
Delta nhỏ tích tụ → cross threshold sau weeks/months. Framework nói Miss MỜ (Body-Feedback-Mechanism §3.2ⓑ) nhưng không connect rõ với Self-Observation delay. Oxygen example = perfect illustration.

### GAP-SO-10: Compiled Suppress detection
Compiled Suppress = person DOESN'T KNOW they're suppressing. Self-Observation literally CANNOT detect. Cần external observer (therapist, close friend). Chicken-and-egg: Self-Observation needed to detect suppress, but suppress blocks Self-Observation.

### GAP-SO-11: Positive delta (aspiration) chưa được framework address đầy đủ
Trigger Principle hiện nghiêng về negative delta (dissonance/avoidance). Positive delta (approach/aspiration) = approach-tagged generative gap (Gap-Direction.md). Nhưng cơ chế "thấy cái đẹp → tạo gap mới → drive" chưa connected rõ với Self-Observation activation. Thích Nhất Hạnh case = positive delta trigger meditation entry.

### GAP-SO-12: External scaffold quality × Self-Observation outcome
Bố hỏi "lo không?" = good scaffold (direct attention to body-state). Bố nói "sao con ngu thế?" = bad scaffold (confirm Feel-Explanation sai → compile wrong self-model). Framework chưa phân tích systematically: scaffold CHẤT LƯỢNG ảnh hưởng Self-Observation development thế nào? Compiled quality (Genuine/Schema/Threat) apply cho scaffold?

### GAP-SO-13: Suffering vs aspiration entry → meditation sustainability
Research: 74% start from suffering, nhưng suffering motivation NEGATIVELY associated with continued practice. Aspiration/spiritual motivation → bền hơn. Framework implication: avoidance-driven Self-Observation training → dừng khi pain giảm. Approach-driven → continue vì intrinsically rewarding. Cần explicit analyze trong training recommendations.

### GAP-SO-14: Self-Observation depth ≠ Life quality (non-linear)
Mức 2 = stable quality of life. Mức 4 = potentially better OR worse. Mức 6 = BEST or WORST. Domain feedback = factor quyết định ↑ hay ↓. Framework chưa map non-linear relationship này. Implication cho education: "dạy Self-Observation sâu" KHÔNG tự động = "cuộc sống tốt hơn." Cần dạy KÈMM domain feedback + survivorship bias awareness.

### GAP-SO-15: Survivorship bias trong Self-Observation narratives
Framework celebrate cases thành công (Einstein, Đức Phật) → implicit bias "Self-Observation sâu = thành công." Thực tế: cùng mechanism → hàng triệu failure cases invisible. Framework cần explicit acknowledge survivorship bias khi discuss Self-Observation benefits.

### GAP-SO-16: Cùng mechanism cho genuine insight VÀ self-referencing trap
General-purpose reward fire cho BẤT KỲ gap fill (Inter-Body-Mechanism §1). Conspiracy theorist nhận opioid giống Einstein. Differentiator = domain feedback, KHÔNG phải Self-Observation depth. Framework cần explicit warning: "Self-Observation sâu + no domain feedback = DANGEROUS (self-referencing amplified)."
