### §1.2 — Core hypothesis (H9)

**H9 unified**: Agent-reading = **Self-Pattern-Match** (solo forward simulation mechanism). PFC retrieve chunks from self repertoire matching target agent, project/apply, read output as prediction. Operates multi-modal (affective / somatic / visual-symbolic / verbal-cognitive / composite). Quality gradients theo 3+1 axes. Schema override (Mode 1/Mode 2) có thể replace entirely. **Pattern-Resonance** = emergent mutual phenomenon xuất hiện khi 2+ agents' Self-Pattern-Match co-fire successfully — retrospectively knowable via communication outcome feedback, serves as calibration data for Self-Pattern-Match refinement.

### §1.3 — 3-concept split (critical architectural decision)

| Concept | Definition | Epistemic | Direction |
|---|---|---|---|
| **Self-Pattern** | Total self chunks currently firing/active | State (direct access) | N/A |
| **Self-Pattern-Match** | Selection + application of self chunks to analyze target agent | Mechanism (direct access) | Forward (self → target) |
| **Pattern-Resonance** | Actual mutual co-firing between self + agent | Ground truth (retrospective only) | Bidirectional (mutual) |

Split này SHARPER hơn Theme E current definition (which conflates solo mechanism với mutual phenomenon).

### §1.4 — Collective-Agent insight (include vào Agent.md §11)

Collective-Agent phenomenon (tin Đảng/Bác Hồ/bộ đội, humanity trust, belonging to nation) **KHÔNG phải mechanism riêng**. Nó là **schema-linked Agent processing** — khi encounter agent (known/unknown), mình link với 1+ schemas, schemas cung cấp valence + prediction template + behavioral script + identity feeling. Schema scale: individual → group → collective → abstract divine. **Same mechanism, different scale**. Cover vào Agent.md §11 "Schema-Linked Agent Processing" (~200-300L).

## §3 — Key Architectural Decisions (session conversation N+2)

Captured from conversation for traceability:

"Agent" không phải storage category riêng, chỉ là entity CẦN modeling overlay để predict behavior.

### §3.2 — Agent function NOT 2-layer (retracted)
Tôi initial proposed 2-layer split (affective empathy + cognitive simulation). User corrected via gift-candy + mathematicians/fans/study group examples. **1 mechanism, multiple pattern-types**. Affective = fire affective pattern. Cognitive = fire cognitive pattern. Same mechanism (Self-Pattern-Match), different pattern-type input.

### §3.3 — Pattern-Resonance is MUTUAL, not solo (retracted Theme E usage)
Theme E current definition conflates solo simulator với mutual phenomenon. User's 3-concept split sharpens:
- Self-Pattern-Match = solo forward mechanism
- Pattern-Resonance = emergent mutual co-firing, retrospective knowability only

This is a **framework update recommendation** (logged §17 of Agent.md + Theme E file).

### §3.4 — Collective-Agent = Schema-linked, same mechanism
User's tin Đảng/Bác Hồ/bộ đội insights → schema-linked Agent processing. When encounter agent, link to schemas; schemas provide valence + prediction + script + identity. Same mechanism at different scale. Cover in Agent.md §11. NO separate Collective-Agent.md file needed (can be future extension if scope grows).

### §3.5 — Chunks fallback when Self-Pattern-Match fails
Mouse case: Self-Pattern-Match threshold fail (too small, no readable cues, no self-template mapping). Fallback to **Layer 1+2 mechanical prediction** (pure chunks + logic, no simulation). Still successful prediction (hun khói → chạy, pho mát → bẫy). Proves Self-Pattern-Match OPTIONAL, not requirement for behavior prediction.

### §3.6 — Mother-10-year suppression = Logic-Feeling §7.1 sub-case D
Same-target-over-time example (mẹ day 0 → year 1 → year 10) demonstrates Self-Pattern-Match progression: unverified → compiled → suppressed (by negative schema overlay "ghét"). Mapping to Logic-Feeling §7.1 Dehumanization-lite DANGER zone.

### §3.7 — Retrospective calibration loop
User's key insight: Pattern-Resonance NOT knowable real-time. Only inferred post-hoc from communication outcome (progression vs reduction). Serves as **calibration feedback** to Self-Pattern-Match library. Enables framework interpretation of Bird & Cook 2013 alexithymia-empathy link: poor self-reading → poor Self-Pattern-Match → poor Pattern-Resonance emergence → refined labeling label: "empathy deficit".
