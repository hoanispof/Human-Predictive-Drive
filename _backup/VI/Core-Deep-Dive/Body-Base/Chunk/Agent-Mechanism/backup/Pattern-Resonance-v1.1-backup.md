## §0 — Abstract + Definition

### §0.1 — Definition

**Pattern-Resonance** = emergent mutual phenomenon arising khi 2+ agents' Self-Pattern-Match mechanisms co-fire successfully matching patterns. Reflects **actual alignment** between agents, not merely one agent's prediction. **Retrospectively knowable only** — no real-time direct access. Serves as **feedback calibration source** for Self-Pattern-Match library refinement.

### §0.2 — One-paragraph summary

Pattern-Resonance là **hiện tượng xuất hiện** khi 2+ người cùng fire Self-Pattern-Match toward each other thành công — templates của cả hai bên match được, co-firing patterns tương đồng, feedback channels hoạt động. Phenomenon không phải cơ chế solo nào — nó là **kết quả thực tế** của mutual engagement. Self không thể biết Pattern-Resonance chính xác ngay lúc đó — chỉ có thể infer sau khi nhìn lại cuộc giao tiếp + outcomes thực sự về sau. Khi detected retrospectively, Pattern-Resonance observations serve as **calibration data** để refine Self-Pattern-Match library của mình.

### §0.3 — Central claims

1. **Emergent, not mechanism**: Pattern-Resonance is a PHENOMENON that arises when conditions met, not a dedicated cognitive mechanism
2. **Mutual**: Requires 2+ agents, not solo
3. **Retrospective epistemic**: Cannot be known real-time, only inferred post-hoc
4. **Feedback source**: Observations about PR serve as library calibration data for SPM
5. **Ground truth**: PR reflects actual alignment, distinct from self's forward prediction
6. **Communication outcome indicator**: PR presence/absence correlates with conversation progression/reduction
7. **Cross-scale**: PR can emerge in dyads, small groups, larger collectives (with different quality characteristics)
8. **Bird & Cook architectural**: Alexithymia breaks PR emergence even when SPM attempted (self can't interpret feedback)

### §0.4 — Relationship với Self-Pattern-Match

| | Self-Pattern-Match | Pattern-Resonance |
|---|---|---|
| **Type** | Solo cognitive mechanism | Emergent mutual phenomenon |
| **Direction** | Forward (self → target prediction) | Bidirectional (mutual co-firing) |
| **Locus** | Inside one agent | Between 2+ agents |
| **Access** | Direct (self knows what it's running) | Indirect (inferred retrospectively) |
| **Time frame** | Immediate (runs in milliseconds-seconds) | Emerges over interaction, detected after |
| **Requires** | Self-chunks + target cues | Both sides' SPM + feedback channel + matching patterns |
| **Produces** | Prediction (hypothesis) | Ground truth about alignment |
| **Feedback role** | Receives feedback to refine | Provides feedback for SPM calibration |

→ SPM is the **mechanism** self runs; Pattern-Resonance is the **phenomenon** that emerges when 2+ SPMs co-fire well. Not the same thing, not replaceable.

## §2 — Requirement conditions

Pattern-Resonance does NOT emerge spontaneously. It requires specific conditions to be met simultaneously. This section formalizes the 4 conditions.

### §2.1 — Condition 1: Both sides running Self-Pattern-Match

**Requirement**: Both (or all) agents must have active Self-Pattern-Match mechanism firing toward each other (or toward shared target).

**Implications**:
- Cannot emerge if one side doesn't have SPM capability (infant below Stage 2, insect, inanimate object)
- Cannot emerge if one side actively suppresses SPM (dehumanization, hostile schema)
- Cannot emerge if one side is absent (parasocial with celebrity, prayer to non-responsive deity)

**Example failures**:
- **Parasocial fan → celebrity**: Celebrity doesn't know fan exists → no SPM from celebrity side → no PR possible, regardless of fan's rich SPM investment
- **Human → dog**: Dog has sensory awareness but cannot run SPM templates as humans do. Dog fires its own internal patterns, not matching human templates. Cross-species PR is architecturally blocked at this condition.
- **Adult → infant (under 18 months)**: Pre-SPM infant cannot run forward simulation toward adult. Adult can fire SPM on infant but PR cannot emerge until infant's SPM comes online.

**Partial PR possible**:
- Adult → toddler (18-24 months): toddler's SPM emerging; partial PR possible, increasing with age
- Adult → cognitively impaired person: variable, depends on SPM capacity of the impaired individual

### §2.2 — Condition 2: Feedback channel available

**Requirement**: There must be a feedback channel through which each side's predictions can be checked against the other's actual state.

**Channel types**:
- **Real-time face-to-face**: richest (facial, vocal, body, verbal, immediate)
- **Real-time limited**: phone (voice only), video call (reduced body)
- **Delayed**: text messages, email, letters (delayed with loss)
- **One-way**: media consumption (no return feedback)
- **None**: imagined interaction, prayer to non-responding deity

**Quality gradient**: Richer feedback → higher-quality PR emergence + detection. Zero feedback → no PR possible, only one-sided SPM.

**Implication**: Feedback is not merely helpful — it's **constitutive** of PR. Without feedback loop, rich SPM investment cannot produce PR; it produces parasocial distortion instead.

**Example**:
- Long-distance relationship via text: reduced PR quality compared to face-to-face, though still real. Participants often report "missing something" despite verbal exchange being full.
- Meeting someone intensively for a day: rich feedback enables rapid PR emergence even without prior history.

### §2.3 — Condition 3: Matching patterns

**Requirement**: Both sides must actually fire similar patterns — affective, cognitive, somatic, visual-symbolic, verbal, or composite. Not same exact chunks, but patterns matching enough to resonate.

**What counts as "matching"**:
- **Similar affective response** to shared stimulus (music fans experiencing same song emotionally)
- **Similar cognitive approach** to shared problem (mathematicians working through proof)
- **Similar somatic state** in shared activity (dancers moving together, musicians playing ensemble)
- **Similar narrative understanding** (people who "get" same joke, share cultural reference)
- **Similar body state reading** of situation (partner recognizing tiredness, both feeling weight)

**What doesn't match**:
- Two people at same event but different emotional response (one enjoys, other hates)
- Same problem but different reasoning styles (one analytic, other intuitive)
- Same experience but divergent interpretations (trauma survivor vs non-survivor hearing story)

**Critical**: Matching can be **domain-specific**. Mathematicians match in math domain (visual-symbolic), not necessarily in personal emotional domain. Fans match in music-affective, not necessarily in life philosophy. PR can emerge in specific domains without requiring full personality match.

### §2.4 — Condition 4: Sufficient interaction time

**Requirement**: Co-firing must persist long enough to establish. Instantaneous flash of recognition doesn't constitute PR — at minimum, multiple exchanges or sustained shared state.

**Time scales**:
- **Minimum**: seconds of sustained co-firing (e.g., shared laugh at joke requires both laughing simultaneously, not sequentially)
- **Moderate**: minutes of sustained interaction (brief conversation establishing rapport)
- **Deep**: hours to days (building deep relationship)
- **Compiled**: years (family, long-term partners, close friends)

**Threshold note**: Framework doesn't commit to precise minimum duration. Different PR depths correspond to different time scales. Deep resonance unlikely to emerge in brief exchanges.

**Exception — prior compiled PR**: With long-term partners, brief re-encounter can re-activate compiled PR quickly. Old married couple can re-establish intimate PR in seconds after time apart because compiled library enables rapid re-firing.

### §2.5 — All 4 conditions necessary

If **any one** condition fails, PR cannot emerge:

- Failed Condition 1 (SPM absent) → no emergence possible
- Failed Condition 2 (no feedback) → parasocial-like distortion
- Failed Condition 3 (pattern mismatch) → discomfort, distance, failed connection
- Failed Condition 4 (too brief) → fleeting impression, not PR

**Multiple conditions can fail simultaneously**: cross-species sympathy fails Condition 1 (dog doesn't SPM human) AND Condition 3 (dog's internal patterns differ from human projection). Both independently prevent PR.

**Framework claim**: These 4 conditions are **necessary**. Whether they are **sufficient** (i.e., meeting all 4 guarantees PR) is less certain. Some combinations may still fail due to factors not yet formalized (chemistry, individual quirks, emotional state).

## §3 — Retrospective detection mechanism

Critical epistemic point: Pattern-Resonance **cannot be known directly in real-time**. Self can only infer PR **retrospectively** through indirect signals. This section details how.

### §3.1 — Why not real-time

Self has direct access to:
- Own Self-Pattern-Match process (what self is running)
- Observable target cues (face, voice, body, words)
- Own affective/cognitive response to target
- Context + situation

Self does NOT have direct access to:
- Target's internal state (only inferred via SPM)
- Target's actual Self-Pattern-Match toward self (what they're running)
- Whether templates match (only inferred via outcome)

→ **PR is about actual internal alignment** which self cannot directly observe. Self runs SPM forward (prediction), but verification requires observing outcomes and inferring backward.

### §3.2 — Signals for retrospective inference

**Primary signal types**:

**Immediate observable response**:
- Target's verbal acknowledgment ("yes, that's exactly right")
- Target's facial + body response (genuine smile vs polite smile, relaxation vs tension)
- Target's verbal elaboration (building on, deepening vs changing subject)
- Target's physical approach or distance

**Conversational dynamics**:
- Conversation deepening: topics become more personal, vulnerability increases, tangents explored
- Conversation stalling: topics dry up, awkward silences, forced transitions
- Mutual extending: both building on each other's contributions
- Mutual contracting: one or both withdrawing, shorter responses

**Post-interaction outcomes**:
- Target wants to meet again (positive PR signal)
- Target avoids or delays next meeting (negative PR signal)
- Target behaves consistently with self's prediction in subsequent days
- Target behaves unexpectedly, contradicting prediction

**Long-term outcomes**:
- Relationship progression (deepening, investment, trust)
- Relationship regression (distance, formal, loss of intimacy)
- Pattern matches self's sense of connection
- Pattern contradicts self's sense

### §3.3 — Inference process

```
Self completes interaction with predictions (from SPM) + memory of observable signals
     ↓
Outcomes unfold (immediate + short-term + long-term)
     ↓
Self reviews (deliberately or not): did my predictions match observations?
     ↓
Self infers: PR was present (predictions matched) or absent (predictions failed)
     ↓
Specific chunks identified as confirmed or disconfirmed
     ↓
SPM library update (see §4)
```

**Speed of inference varies**:
- **Immediate**: within the conversation itself ("I think we're connecting")
- **Post-conversation**: within minutes after ("that went better than expected")
- **Short-term**: hours to days ("looking back, I think I misread them")
- **Long-term**: weeks to months ("turns out my initial read was wrong")

**Deliberate vs automatic**:
- Automatic inference: unconscious library update without reflection
- Deliberate inference: conscious review, metacognitive assessment

**Training effect**: Deliberate post-interaction reflection + feedback seeking **accelerates** library refinement. This is the mechanism behind many empathy training programs.

### §3.4 — Accuracy limits of retrospective inference

Self cannot perfectly verify PR even retrospectively:

**Problem 1 — Target's actual state remains opaque**:
- Self only sees target's behavior, not target's actual SPM or internal state
- Target's behavior might match prediction for different reasons than self assumes
- "They agreed so my prediction was right" might be wrong (they were being polite)

**Problem 2 — Self's memory is reconstructive**:
- Memory of interaction reshapes over time (consistency bias, confirmation bias)
- Self may "remember" interaction differently from what happened
- Past predictions get re-interpreted in light of subsequent outcomes

**Problem 3 — Multiple valid interpretations**:
- Same outcome data supports multiple PR inferences
- "Relationship continued" doesn't definitively mean PR was present
- "Relationship ended" doesn't definitively mean PR was absent

**Problem 4 — Self's SPM of self is limited**:
- Alexithymic self cannot accurately assess own reactions
- Distinguishing "I felt connection" from "I projected connection" is hard
- Self-deception can mask PR failures

### §3.5 — Approximate detection is still useful

Despite accuracy limits, **approximate retrospective detection** is sufficient for framework purposes:

- Trends over many interactions smooth out noise
- Strong patterns emerge clearly (consistently deepening vs consistently failing relationships)
- Library refinement proceeds with imperfect but directional feedback
- Self-awareness training improves accuracy over time

**Framework claim**: PR detection is **imperfect but trainable + functional**. Not precise scientific measurement, but sufficient for lived social cognition + relationship repair + empathy development.

### §3.6 — Mutual verification (partial solution)

**Higher-fidelity detection**: When both parties reflect + share ("What did you think of our conversation?" "How did that land for you?"), mutual verification approximates ground truth.

**Therapy + deep friendships** rely on this: explicit meta-conversation about the conversation itself enables higher-quality PR inference than solo reflection alone.

**Limits**: Other party may not reflect accurately, may not share honestly, may lack vocabulary. Mutual verification is better than solo but still imperfect.

## §4 — Feedback calibration loop

Pattern-Resonance's **primary architectural function** in the framework is to provide **feedback calibration** for Self-Pattern-Match library. This section details the loop mechanism.

### §4.1 — Loop architecture

```
[Self-Pattern state at t₀]
        ↓
[SPM fires forward prediction]
        ↓
[Interaction proceeds]
        ↓
[Outcome observed]
        ↓
[Pattern-Resonance inferred retrospectively]
        ↓
[Library update]:
  - Confirmed chunks strengthened
  - Misfired chunks weakened
  - New target-specific chunks added
  - Retrieval heuristics refined
  - Schema link adjusted
        ↓
[Updated SPM library ready for next prediction]
        ↓
[Self-Pattern state at t₁ (improved)]
```

### §4.2 — Types of library updates from PR observation

**Chunk strengthening**:
- If prediction matched outcome → chunks used in that prediction get reinforcement
- Example: "I predicted friend would appreciate this gift → she genuinely loved it → 'friend prefers practical gifts' chunk strengthened"
- Future retrieval prioritizes these chunks for similar situations

**Chunk weakening**:
- If prediction failed → chunks used get reduced weight
- Example: "I assumed colleague was upset → turns out they were just tired → 'colleague's frown = upset' chunk weakened"
- Future retrieval deprioritizes these chunks

**New chunk creation**:
- Target exhibits unexpected behavior → new chunk formed
- Example: "I didn't know she loved jazz → now I have 'friend's musical taste: jazz' chunk"
- Library expands with target-specific details

**Retrieval heuristic refinement**:
- Meta-learning about WHICH chunks to retrieve for WHICH targets
- "For this colleague, prioritize verbal-cognitive Pattern-Type; their affective cues are subtle"
- Customizes SPM behavior per target

**Schema link adjustment**:
- Target's behavior contradicts initial schema → schema link updated
- Example: "I categorized new teammate as 'aggressive type' but they're actually 'analytical'" → schema shift
- Affects future predictions for this target

### §4.3 — Time scales of feedback

**Immediate feedback** (within interaction):
- Mid-conversation micro-adjustments
- "She seems confused by that analogy → switch to different framing"
- Library updates happen in real-time during exchanges

**Short-term feedback** (hours to days after):
- Post-interaction reflection
- "Looking back, I think I misread that moment"
- Deliberate library updates from explicit review

**Medium-term feedback** (weeks to months):
- Relationship trajectory observation
- "Over the past month, our interactions have become easier — my reads are improving"
- Cumulative library refinement

**Long-term feedback** (years):
- Deep relationship compilation
- "After 20 years of marriage, I know my partner's moods from their gait"
- Massively refined libraries for long-term targets

**Cross-target transfer**:
- Lessons from one target transfer to similar targets
- "I learned from friend X that introverts need processing time; I apply this to friend Y"
- Generalized calibration beyond specific relationship

### §4.4 — Bird & Cook 2013 loop break

Critical framework point: **Alexithymia breaks the feedback loop**.

**Mechanism of loop break**:
```
Alexithymic self runs SPM with thin chunks
     ↓
Prediction is noisy (thin templates)
     ↓
Interaction unfolds
     ↓
Self cannot accurately read own reaction to outcome
(alexithymia = poor self-awareness)
     ↓
Feedback signal is garbled
     ↓
Library update is noisy or fails
     ↓
Next SPM still uses thin chunks
     ↓
Empathy deficit persists over time
```

→ **Empathy doesn't improve through ordinary experience** in alexithymic individuals because the feedback loop is broken. This is why alexithymia is persistent without targeted intervention.

**Intervention target**: Fix self-awareness (upstream self-chunks) → loop begins working → PR inference improves → SPM library refines → empathy improves. This is the Bird & Cook 2013 insight operationalized.

### §4.5 — Training the loop explicitly

Deliberate practice accelerates calibration:

**Post-interaction reflection**:
- "What did I predict? What actually happened? What surprised me?"
- Write down + review

**Explicit hypothesis tracking**:
- Note predictions before interactions
- Check against outcomes
- Identify patterns in errors

**Feedback seeking**:
- Ask target directly ("How did that land?", "What were you thinking?")
- Compare their report to self's prediction
- Update library with explicit delta

**Journaling social experiences**:
- Written reflection strengthens chunks
- Retrospective pattern recognition across multiple interactions
- Identifies systematic biases

**Therapy as structured calibration**:
- Therapist facilitates self-awareness development
- Co-explore predictions + outcomes
- Build new chunks in safe environment

### §4.6 — Loop failures

**Loop failure mode 1 — Alexithymia** (§4.4): cannot interpret feedback → no update
**Loop failure mode 2 — Parasocial** (no feedback source): library updates without reality check → drift
**Loop failure mode 3 — Schema suppression**: ignore feedback that contradicts schema → dehumanization persistence
**Loop failure mode 4 — Confirmation bias**: interpret feedback to fit pre-existing predictions → no genuine update
**Loop failure mode 5 — Trauma distortion**: past trauma causes systematic mis-reading of feedback → stuck patterns

## §5 — Quality levels

Pattern-Resonance is not binary. Spectrum of quality emerges from how fully requirements met. This section maps the spectrum.

### §5.1 — Deep resonance 🟢

**Characteristics**:
- Multi-modal matching (composite Pattern-Type)
- Extended duration + high feedback richness
- Both sides consciously recognize the resonance
- Ongoing compiled library for this specific dyad/group
- Rapid re-activation after separation

**Examples**:
- **Long-term family** (parents + children after years of attunement)
- **Close friendships** (decades of shared history + mutual reading)
- **Creative partnerships** (co-authors, musical duos, creative teams deeply aligned)
- **Romantic relationships** at depth (compiled intimate reading)
- **Deep therapeutic alliance** (therapist-client in long treatment)

**Subjective experience**: "We really understand each other", "cùng tần số", "don't need to explain", "one word is enough", "I know what you're thinking"

**Framework mechanism**: Library compiled across many interactions, 4 axes all high, composite Pattern-Type firing rapidly, feedback loop continuously refining.

### §5.2 — Moderate connection 🟡

**Characteristics**:
- Single or dual Pattern-Type matching
- Medium duration or moderate feedback
- One or both sides recognize some connection but not deep
- Domain-specific or context-specific

**Examples**:
- **Study groups**: cognitive Pattern-Type matching on shared problem
- **Professional colleagues** (same field, shared technical understanding)
- **Shared fandoms**: affective Pattern-Type matching on shared object of enthusiasm
- **Casual friends**: occasional rich moments, mostly moderate
- **Sports teammates** (somatic + cognitive in game context)

**Subjective experience**: "We get along well in this context", "we share [specific interest]", "good working together", "fellow traveler"

**Framework mechanism**: One or two axes high (depth in domain), others moderate. Pattern-Type limited to specific modality. Feedback adequate but not saturated.

### §5.3 — Surface match 🟡

**Characteristics**:
- Brief co-firing, often affective-only
- Short duration, cue-driven
- Both sides aware of shared context but shallow
- Often anchored to external shared stimulus

**Examples**:
- **Strangers at shared event**: fellow fans at concert, soldiers marching together, ritual co-participants
- **Brief social encounters**: friendly interactions with acquaintances
- **Shared crisis response**: strangers united by external threat
- **Brief moments of empathy**: passing stranger's visible grief

**Subjective experience**: "Moment of connection", "shared experience", "felt like one of us"

**Framework mechanism**: Affective Pattern-Type fires on shared stimulus, producing co-activation without deeper chunk matching. Durkheim's collective effervescence operates here.

### §5.4 — Failed / One-sided 🔴

**Characteristics**:
- Only one side runs Self-Pattern-Match
- No matching patterns despite effort from one side
- Feedback unavailable, misread, or absent
- No genuine resonance

**Examples**:
- **Parasocial relationships** (fan + celebrity)
- **One-sided crush** (feelings not reciprocated)
- **Cross-species pet "bonds"** (human rich SPM, animal different internal patterns)
- **Failed friendship attempts** (one side engages, other doesn't)
- **Dehumanization** (one side suppresses SPM entirely)

**Subjective experience** (from engaged side): "I feel close but something's off", "they don't really know me", "connection isn't mutual", "I'm invested but they're not"

**Framework mechanism**: Condition 1 or Condition 3 failed. Self's SPM runs robustly but no PR emergence possible. Feedback loop broken → library drift → potential distortion.

### §5.5 — Quality dynamics over time

Quality can shift:

**Deep → Moderate**: Long-term relationships can decline due to neglect, distance, changing circumstances
**Moderate → Deep**: Relationships deepen through sustained engagement + compilation
**Surface → Moderate**: Initial encounter can develop into deeper understanding with continued interaction
**Any → Failed**: Betrayal, dehumanization, schema override can collapse PR entirely
**Failed → Moderate**: Reconciliation, schema renovation can restore PR (slowly, with effort)

**Framework insight**: Quality level is not fixed. Active maintenance required for deep resonance (time, attention, feedback, reflection). Neglect leads to decay.

## §6 — Cases: Genuine Pattern-Resonance

Detailed analysis of PR cases that work. Each case shows how conditions are met + what Pattern-Type dominates + what makes it distinctive.

### §6.1 — Mathematicians + physicists sharing proof

**Scenario**: Two mathematicians working through a proof together, or one explaining to another.

**Pattern-Type**: Visual-symbolic dominant (spatial + notational patterns), verbal-cognitive secondary (explanation language)

**Depth**: Deep in domain (both have extensive math chunks), thin in personal (don't need to know each other's lives)

**Similarity**: Moderate-high (similar reasoning styles, shared training in discipline)

**Feedback**: Rich real-time (gestures, writing on board, verbal checking, diagrammatic correction)

**Emergence**: When both fire similar visual-symbolic patterns while following proof, Pattern-Resonance emerges in math domain. "I see where you're going with that" not metaphorical — literal co-firing of same structural patterns.

**Subjective experience**: Feeling of shared understanding, "thinking together", "seeing the same thing"

**Key insight**: Deep PR possible **without personal depth**. Shared domain chunks + domain-specific Pattern-Type + real-time feedback = sufficient.

### §6.2 — Study group detecting shared impasse

**Scenario**: Students working on hard problem together, multiple people stuck.

**Pattern-Type**: Verbal-cognitive + visual-symbolic (depending on problem)

**Depth**: Shared problem chunks (same reading, same attempted approaches)

**Similarity**: Similar skill level, similar reasoning at this learning stage

**Feedback**: Rich discussion, visible attempts + errors

**Emergence**: When group hits shared impasse, members fire similar stuck-at-same-place patterns. "I know where you're stuck — I'm stuck there too" = co-firing recognition.

**Subjective experience**: "We're all confused together", "this is genuinely hard not just me", relief + solidarity

**Key insight**: PR emerges from shared problem state, not from understanding each other personally. The common external stimulus (problem) is the anchor.

### §6.3 — Music fans at concert

**Scenario**: Fans at live concert experiencing song they all love.

**Pattern-Type**: Affective dominant (emotional response to music), somatic secondary (body moving to rhythm)

**Depth**: Shared musical chunks (same favorite songs, same emotional associations with music)

**Similarity**: Similar emotional response patterns (self-selected by fan community)

**Feedback**: Via shared stimulus (same song triggers same affective response in both), visible reactions of others

**Emergence**: When everyone's affective patterns fire on same musical moment, mass co-firing produces collective Pattern-Resonance. Durkheim's collective effervescence in framework terms.

**Subjective experience**: "Cùng gu", "cùng tần số", "we're all feeling this together", sense of belonging

**Key insight**: Shared external stimulus (music) enables PR without direct interaction between participants. Each fan is firing their own SPM toward the music, but their SPM outputs resonate with each other's because of shared firing pattern.

### §6.4 — Family member reading subtle state

**Scenario**: Close family member detecting another's subtle mood change before others notice.

**Pattern-Type**: Composite (multi-modal — facial, vocal, body, contextual history)

**Depth**: Deep (years of accumulated chunks about this specific person)

**Similarity**: High (family resemblance + shared life context)

**Feedback**: Rich, compiled over many interactions

**Emergence**: Compiled SPM library fires rapidly on subtle cues. Family member's prediction matches actual state. When verified ("yes, I am worried about work"), PR confirmed in depth.

**Subjective experience**: "No one else noticed but my sister knew right away", "mẹ biết ngay khi tôi buồn"

**Key insight**: Deep personal PR requires long-term compilation. Cannot be shortcutted.

### §6.5 — Partners in crisis moment

**Scenario**: Romantic partners navigating difficult situation (loss, illness, conflict).

**Pattern-Type**: Composite (all available Pattern-Types fire simultaneously)

**Depth**: Compiled relationship history

**Similarity**: High (chosen partner, shared life)

**Feedback**: Intense real-time (high stakes amplify attention)

**Emergence**: Crisis brings full composite SPM firing on both sides. Match produces deep PR, failure produces relationship strain.

**Subjective experience**: "He just knows what I need", "we got through it together", or conversely "we couldn't connect when it mattered"

**Key insight**: High-stakes moments stress-test PR quality. Compiled libraries + active feedback enable crisis PR; thin libraries or broken feedback fail under pressure.

### §6.6 — Therapist-client deep session

**Scenario**: Trained therapist with client in therapeutic alliance, moment of insight.

**Pattern-Type**: Verbal-cognitive + affective composite

**Depth**: Therapist has broad SPM library from training + case experience; deep in this client after sessions

**Similarity**: Bridged by therapist's training to simulate across personality differences

**Feedback**: Rich verbal + emotional, structured therapeutic setting

**Emergence**: When therapist's read matches client's genuine state + client recognizes this → PR established → therapeutic alliance strengthens → breakthrough possible.

**Subjective experience** (client): "She really gets me", "I feel heard for the first time"

**Key insight**: Trained SPM can establish PR across similarity gaps that untrained SPM cannot bridge.

### §6.7 — Artists + craftspeople in shared work

**Scenario**: Expert craftspeople or artists working on shared project.

**Pattern-Type**: Visual-symbolic + somatic (embodied expertise)

**Depth**: Deep in craft domain (both have extensive technical chunks)

**Similarity**: Similar expert body + skill

**Feedback**: Through the work itself (collaborative piece evolves)

**Emergence**: Shared craft understanding enables wordless coordination. "Flow" state in team sports, jazz improvisation, ensemble work.

**Subjective experience**: "Felt like we were one", "anticipated each other's moves", "cùng một nhịp"

**Key insight**: Somatic + visual-symbolic Pattern-Types enable PR without verbal explicit exchange. Body-level resonance in action.

## §7 — Cases: Failed Pattern-Resonance

Detailed analysis of Pattern-Resonance failures. Each case shows which condition fails + consequences. §7.8-§7.9 mở rộng: Pattern-Resonance Baseline (tại sao Condition 3 dễ/khó) + obligation-trapped interaction (khi không thể exit dù Pattern-Resonance fail).

### §7.1 — Parasocial celebrity relationship

**Scenario**: Fan develops deep "relationship" with celebrity via media.

**Failed condition**: Condition 2 (feedback channel) — celebrity doesn't know fan exists, provides no feedback to fan's SPM.

**What happens**:
- Fan fires rich Self-Pattern-Match on celebrity via consumed media
- Fan's library accumulates chunks about celebrity's personality, preferences, apparent emotional patterns
- **No correction signal** — celebrity never responds to fan specifically
- Library drifts, confabulates details, fills gaps with projection
- Fan's predictions become increasingly disconnected from celebrity's actual self
- Fan may experience "real" emotional investment but it's one-sided

**Consequences**:
- **Mild**: harmless admiration, some distortion, doesn't interfere with life
- **Moderate**: investment proportional to real-relationship absence (compensatory)
- **Severe**: clinical distress when reality intrudes (scandal, death, unexpected behavior), delusional belief in relationship, stalking behavior
- **AI era extension**: parasocial with AI characters, chatbots, virtual influencers — same mechanism, more accessible

**Recovery**: Restoring real-world feedback-providing sources. Cognitive insight alone insufficient — need actual agent interactions with real feedback to refill feedback axis.

### §7.2 — Cross-species sympathy (dog, cat)

**Scenario**: Human "understanding" pet's emotional state.

**Failed condition**: Condition 1 (both sides running SPM) + Condition 3 (matching patterns).

**What happens**:
- Human fires affective SPM on pet's distress cues (cúp đuôi, ẳng ẳng, cúi đầu chậm)
- Human's projection produces own distress template — "dog is sad"
- **Dog fires its own internal patterns**, not matching human's human-shaped templates
- Dog's actual experience of distress is species-specific, not what human projects
- Human experiences sympathy, but it's not PR — it's one-sided affective projection

**Not pathological** usually: normal cross-species empathy, often protective, encourages care for pets.

**Becomes problematic when**:
- Projection leads to welfare neglect (denying species-typical needs because "my pet wouldn't want that")
- Anthropomorphism extreme (treating pet as human substitute for social connection)
- Decisions based on projected preferences that violate species biology (dietary, behavioral, medical)

**Framework insight**: Cross-species sympathy is **legitimate affective SPM firing**, not PR. Useful for motivating care but doesn't establish mutual alignment. Best practice: combine sympathy with species-specific knowledge (Layer 1+2 mechanical chunks about species).

### §7.3 — Dehumanization (hostile schema suppression)

**Scenario**: Combat soldier viewing enemy, ethnic conflict participant viewing out-group, corporate figure viewing exploitable customers.

**Failed condition**: Condition 1 (one side actively suppresses SPM toward target).

**What happens**:
- Hostile schema fires, blocking SPM invocation
- SPM library intact but not accessed for this target
- Target treated as unit/threat/resource, not agent
- No simulation of target's experience
- PR architecturally impossible

**Framework mechanism**: Layer 4 schema override actively suppresses Layer 3 SPM. Not mechanism absence — mechanism blocked.

**Research evidence**: Grossman "On Killing" 🟢 (combat dehumanization), intergroup conflict research 🟢, Meat Paradox (Loughnan & Bastian 2014 🟢).

**Mother-10-year case parallel**: Mild dehumanization-lite within family when negative schema accumulates ("ghét mẹ"). Same mechanism at lower intensity.

**Reversibility**: Schema change enables SPM re-invocation. Truth and reconciliation, reconciliation therapy, exposure to humanizing narratives can revise schema. Agent.md §16 Prediction P6.

**Framework caution**: Dehumanization is dangerous precisely because the mechanism is **sticky** once schema-linked. Requires deliberate + sustained intervention to reverse.

### §7.4 — Mutual mismatch (classroom example)

**Scenario**: Student A studying hard; Student B chatting. Both in same class.

**Failed condition**: Condition 3 (matching patterns) — A and B fire different internal patterns about the situation.

**What happens**:
- Student A's SPM on B: "B is slacking, disrespecting the class, should study like me"
- Student B's SPM on A: "A is tense, no fun, should relax"
- Each perceives the other correctly in behavior but **interpret through own values**
- No matching patterns (A's study-focus vs B's social-focus)
- No PR emerges — both feel "we don't get each other"

**Neither is "wrong"**: Both are running SPM accurately based on their own chunks. But chunks differ, so projections differ.

**Framework insight**: Mismatch PR failures are the most common interpersonal friction source. Not about villainy or ignorance — about genuinely different compiled Pattern-Type biases.

**Potential resolution**: Explicit conversation about values + expectations enables **cognitive understanding** (Theory theory side) even without affective PR. "I understand you value X even if I don't" without necessarily feeling alignment.

### §7.5 — State-gated absence (tôi buồn + others vui)

**Scenario**: Self in low mood + observing happy group.

**Failed condition**: Complex — Condition 1 partially fails (SPM gated by self-state).

**What happens**:
- Self's current affective state (buồn) gates chunk retrieval
- SPM toward happy others doesn't fully fire — own state can't match
- Either: withdraw (not engaging), or: forced smile without resonance, or: active discomfort
- No PR emerges because self's internal state blocks matching

**Framework mechanism**: Current state is itself a chunk retrieval context. Incompatible state gates SPM.

**Two sub-outcomes**:
- **Non-engagement**: Self stays apart, no interaction, no PR attempted
- **Cognitive without affective** (Case 3 in old outline): Self knows others are happy but cannot feel it → dissonance → active negative feeling

The cognitive-without-affective variant is **decisive** evidence for framework (§10 detailed).

### §7.6 — Psychopathy cognitive-affective split

**Scenario**: Individual with psychopathic traits reading target's behavior.

**Failed condition**: Condition 3 (partial) — only cognitive patterns match, affective do not.

**What happens**:
- Cognitive SPM fires normally: can predict target's actions, thoughts, decisions
- Affective SPM deficient: does not fire empathic distress contagion, no mirror reward
- Can manipulate target effectively without restraint
- No full PR — only cognitive-side resonance

**Framework caution**: Research on psychopathy as clear cognitive-affective dissociation is contested. Framework acknowledges descriptive case, mechanism details uncertain (🟡 moderate, not 🟢 strong).

**Behavior implication**: Enables high-functioning manipulation. "Reading" people successfully without feeling inhibition from their distress.

### §7.7 — AI interactions (emerging case)

**Scenario**: Human interacting with sophisticated conversational AI.

**Failed condition**: Condition 1 (AI does not have SPM of human in human-SPM sense) — but cues present may trigger human to invoke SPM as if there were.

**What happens**:
- Human invokes SPM toward AI despite cognitive awareness AI is non-agent
- AI provides responses that appear to match human's prediction
- Human feels "AI understands me"
- But AI is not actually running SPM toward human — AI matches patterns via different mechanism
- No genuine PR, only one-sided human SPM + pattern-matched AI output

**Emerging concern**: As AI becomes more sophisticated, one-sided SPM becomes more convincing. Human cannot easily distinguish "AI that matches my pattern successfully" from "AI that has PR with me". Framework predicts growing distortion risk (Agent.md §16 Prediction P10).

**Speculative boundary**: If future AI develops genuine SPM-like mechanism, PR might become possible in human-AI dyads. Framework doesn't commit to prediction, flags as open question.

### §7.8 — Pattern-Resonance Baseline: "Hợp Tính" Defined

**Concept**: §7.1-§7.7 phân tích từng case cụ thể khi Pattern-Resonance fail. Nhưng TRONG Condition 3 (matching patterns), có 1 biến số CHƯA ĐƯỢC DEFINE: mức dễ/khó mà patterns match TỰ NHIÊN giữa 2 agents cụ thể. Đây chính là "hợp tính" — thuật ngữ thường dùng nhưng chưa bao giờ được giải thích CƠ CHẾ trong framework.

**Definition**:

> **Pattern-Resonance Baseline** = mức độ Pattern-Resonance emerge TỰ NHIÊN giữa 2 agents cụ thể, TRƯỚC KHI effort hoặc adjustment nào can thiệp.

- Baseline CAO ("hợp tính"): Self-Pattern-Match self-template match target by default → patterns co-fire dễ → Pattern-Resonance emerge với ÍT EFFORT
- Baseline THẤP ("khác tính"): Self-template fail on target → patterns don't match → Pattern-Resonance khó emerge hoặc KHÔNG emerge
- Baseline = PROBABILITY SPECTRUM liên tục, không phải binary

**3 thành phần:**

**(a) Hardware parameter overlap:**
- Novelty drive: cả hai thích explore → cùng fire khi gặp mới → match
- Cortisol sensitivity: cùng ngưỡng chịu stress → cùng respond ở mức tương đương
- Connection drive ❶: cùng mức cần gần → cùng rhythm tiếp cận
- Processing tempo: cùng tốc độ suy nghĩ/phản ứng → tương tác mượt
- Nhiều overlap → Self-Pattern-Match §3.3 template match accuracy CAO by default (self-template ≈ target)

**(b) Expression modality match (Logic-Feeling §6):**
- Verbal-dominant ↔ verbal-dominant → hiểu ngôn ngữ nhau
- Somatic-dominant ↔ somatic-dominant → đọc body signal nhau
- Mismatch: verbal person + somatic person → "sao không nói?" / "nói gì khi body đã nói rồi"
- Matching modality → feedback channel RICHER → Pattern-Resonance detection chính xác hơn

**(c) Domain chunk overlap:**
- Shared interests, experiences, knowledge → chunk libraries OVERLAP
- Einstein + Grossmann: math chunks overlap → Pattern-Resonance ở math domain (§6.1)
- Parent + child cùng thích nấu ăn → Pattern-Resonance ở domain đó
- Component NÀY dynamic NHẤT — chunks thay đổi liên tục theo trải nghiệm

**5 thuộc tính quan trọng:**

**① RELATIONAL — không phải property CỦA 1 NGƯỜI:**

Cùng 1 đứa trẻ: Pattern-Resonance Baseline CAO với mẹ, THẤP với bố. Không nói được "đứa trẻ này khó" — chỉ nói được "cặp này có baseline THẤP." Tương tự: cùng 1 người, baseline CAO với bạn A, THẤP với bạn B. Đây là property của CẶP, không phải cá nhân.

**② DYNAMIC — thay đổi theo thời gian:**

2 bạn thân thời đại học: chunks overlap CAO (cùng phase, cùng lo lắng, cùng explore). 10 năm sau: A có con, B độc thân → chunks DIVERGED → baseline GIẢM → "sao hồi xưa hợp mà giờ không nói gì được." 20 năm sau: cả hai qua nhiều mất mát, thành bại → chunks overlap LẠI ở wisdom level → "lâu rồi gặp lại mà thấy vẫn hiểu nhau." = "Grow apart" VÀ "reconnect" đều giải thích được qua baseline shift.

**③ DOMAIN-SPECIFIC:**

Pattern-Resonance Baseline có thể CAO ở 1 domain, THẤP ở domain khác. Hợp ở domain CẢM XÚC (cả hai emotionally expressive) nhưng KHÁC ở domain TƯ DUY (one analytical, one intuitive). = Tại sao 2 người "hợp" trong 1 context nhưng "lệch" trong context khác. Và Pattern-Resonance §6 (genuine cases) đã show: Pattern-Resonance emerge PER DOMAIN, không đòi toàn bộ personality match.

**④ PARTIALLY GENETIC nhưng KHÔNG DETERMINISTIC:**

Con chia sẻ ~50% hardware từ mỗi parent → SOME baseline similarity. Nhưng WHICH 50%? → lottery. + Epigenetics + developmental timing + environmental shaping → không predict được trước sinh — chỉ observe qua tương tác thực tế.

**⑤ BIDIRECTIONAL nhưng KHÔNG SYMMETRIC hoàn hảo:**

Nếu A baseline thấp với B → B cũng likely baseline thấp với A (pattern mismatch là mutual). NHƯNG: A có thể giỏi adjust template hơn B (SPM skill khác nhau). A compensate → feels effortful cho A, natural cho B. = Asymmetric EFFORT dù symmetric BASELINE.

**Self-Pattern-Match mechanism — TẠI SAO baseline matters:**

```
Pattern-Resonance Baseline CAO (hợp tính):

  Self-Pattern-Match §3.3 Template Match: self-template ≈ target by DEFAULT
    → Projection accurate WITHOUT extra calibration
    → SPM predict → result MATCHES → reward signal
    → vmPFC: "controllable!" → active coping (Autonomy-Hardware §2)
    → Body: "SPM WORKS" → invest more → library BUILDS sâu
    → = SPM naturally REWARDING → MUỐN engage
    → = Feedback loop (§4) ACCELERATE: predict đúng → refine → predict đúng hơn
    → = POSITIVE CASCADE

Pattern-Resonance Baseline THẤP (khác tính):

  Self-Pattern-Match §3.3: self-template ≠ target
    → Projection FAILS → needs heavy PFC adjustment mỗi lần
    → SPM predict → result DOESN'T match → error signal
    → vmPFC: unclear controllability
    → Body: "SPM FAILS" → cost high, reward low
    → = SPM naturally EXHAUSTING → TRÁNH engage
    → = Feedback loop (§4) STALL: predict sai → thiếu data → cannot refine
    → = NEGATIVE CASCADE (hoặc stagnation)
```

**Bidirectional cascade (beyond individual SPM):**

- A match B tự nhiên → B CŨNG dễ match lại → POSITIVE cycle:
  A predict B đúng → B feel understood → B engage more → more feedback → A refine → predict đúng hơn. CẢ HAI feel "hợp" → invest thêm → library builds → Pattern-Resonance deepens.

- A KHÔNG match B → B CŨNG không match lại → NEGATIVE cycle:
  A predict B sai → B feel misunderstood → B engagement GIẢM → ít feedback cho A → A worse. CẢ HAI feel "lệch" → invest ít → library stagnant → Pattern-Resonance stalled.

**Parent-child đặc biệt (developmental impact):**

Pattern-Resonance Baseline giữa parent-child quan trọng VƯỢT khỏi relationship quality, vì ảnh hưởng DEVELOPMENTAL TRAJECTORY:

- Baseline CAO → parent respond CONTINGENTLY (hiểu con tự nhiên) → con's self-chunks build CHÍNH XÁC (Self-Pattern-Match §7 Stage 1-2) → SPM develop mạnh
- Baseline THẤP → parent respond NON-CONTINGENTLY (không hiểu con) → con's self-chunks CONFUSED → SPM develop lệch
- = Pattern-Resonance Baseline parent-child KHÔNG CHỈ ảnh hưởng relationship → ảnh hưởng NỀN TẢNG phát triển cognitive-social
- Chi tiết: Love-Analysis §7.3c biến ⑦, Connection §7.1 (L2 strong + F1 weak)

**Existing framework mentions (now clarified):**
- Self-Pattern-Match §3.5: "hợp tính" as intensity variable → now DEFINED as Pattern-Resonance Baseline
- Self-Pattern-Match §3.5: "không hợp tính" = zero simulation intensity → now EXPLAINED as Baseline ≈ 0
- Connection §3.4: "bạn thân hợp tính" = deep connection → now MECHANISM clear (high baseline → easy Pattern-Resonance → deep connection)

**Research alignment:**
- 🟢 Thomas & Chess 1977: "Goodness of Fit" — child temperament × parent expectations → outcome depends on FIT, not child alone. Framework maps: fit = Pattern-Resonance Baseline
- 🟢 Byrne 1971: Similarity-attraction hypothesis — behavioral similarity → friendship formation. Framework maps: similarity → high baseline → Pattern-Resonance emerge → friendship

### §7.9 — Obligation-Trapped Non-Resonant Interaction

**Scenario**: 2 agents với Pattern-Resonance Baseline THẤP (§7.8). Pattern-Resonance không emerge tự nhiên. Bình thường, cả 2 sẽ giảm tương tác → drift apart. NHƯNG: obligation (structural, social, economic) KHOÁ cả 2 trong tương tác liên tục.

**Failed condition**: Condition 3 (matching patterns) — patterns genuinely không match.

**Additional constraint UNIQUE cho case này**: CANNOT EXIT. Khác với §7.1-§7.7 — trong mọi case trước, agent CÓ THỂ drift apart, giảm contact, hoặc disengage. Ở đây, obligation LOCKS continued interaction bất chấp Pattern-Resonance failure.

**Common settings:**
- **Parent ↔ child**: Con phụ thuộc (không thể rời). Parent có L2 drive + social obligation (không thể bỏ)
- **Marriage/partnership có structural binding**: con chung, tài chính chung, áp lực xã hội
- **Workplace hierarchical**: employee không thể dễ dàng nghỉ việc, phải tương tác hàng ngày với sếp/đồng nghiệp
- **Extended family co-habitation**: obligation văn hóa — sống cùng ông bà, anh chị em
- **Military/institutional**: forced proximity + authority chain

**What happens — 4-step mechanism:**

```
STEP 1 — NATURAL PATTERN-RESONANCE FAILURE:

  SPM fires → self-template project lên target → patterns DON'T match
  Body default response: giảm tương tác → drift apart
  = Mechanism bình thường khi Condition 3 fail (§7.4 mutual mismatch)
  
  NHƯNG: obligation blocks exit
  → Body CẦN giảm contact → KHÔNG ĐƯỢC giảm
  → = DISSONANCE: drive to disengage + forced to engage


STEP 2 — SUPPRESS NATURAL PATTERNS:

  Natural response patterns fire → nhưng sẽ KHÔNG match target
  PFC INHIBIT natural response → tránh conflict/friction
  
  Autonomy-Hardware §5 — 2-layer dissonance:
    Layer 1: immediate prediction override (natural → forced)
      → Body predict "tôi sẽ react thế NÀY" → nhưng PHẢI react thế KHÁC
      → = Efference mismatch: motor prediction ≠ actual action
    Layer 2: meta-learning "my natural pattern doesn't matter here"
      → vmPFC domain-specific learning: "in THIS relationship, I CANNOT be me"
      → DRN tendency (Maier & Seligman 2016) → passive coping IN THIS DOMAIN
  
  Body FEEL: "gượng" (forced), "giả" (fake), "mệt" (tired)
  → "Gượng" = Somatic-Articulation-Loop: body VERBALIZE CHÍNH XÁC mechanism
    (suppress + override = body knows → language captures accurately)


STEP 3 — AMPLIFY/FABRICATE PATTERNS:

  Suppress xong → CHƯA ĐỦ. Phải TẠO patterns có thể match target:
  
  Select chunks KHÔNG phải dominant CỦA MÌNH → cố match target:
    "Bình thường tôi ít nói → nhưng với sếp phải hoạt bát"
    "Bình thường tôi thích explore → với bố phải tỏ ra ngoan"
    "Bình thường tôi direct → với mẹ chồng phải vòng vo"
  
  = PFC-HEAVY processing (NOT compiled → deliberate MỖI LẦN)
  = KHÁC với SPM compiled thông thường:
    SPM compiled: vô thức, nhanh, ít cost (§3 six-step gần automatic)
    SPM fabricated: có ý thức, chậm, PFC drain (effortful mỗi episode)
  = KHÔNG BAO GIỜ compile thành tự động:
    Vì patterns này KHÔNG PHẢI genuinely của mình
    → Body KHÔNG reinforce → KHÔNG Hebbian strengthening
    → = FRESH PFC effort MỖI LẦN — không giảm cost theo thời gian


STEP 4 — HOPE FOR PARTIAL PATTERN-RESONANCE:

  Sau suppress + amplify/fabricate → KỲ VỌNG Pattern-Resonance emerge:
  
  Nếu target NHẬN RA effort:
    → Partial Pattern-Resonance (appreciation, "cũng cố rồi")
    → Reward NHỎ → sustain effort thêm 1 thời gian
    → Nhưng reward < cost → unsustainable long-term
    
  Nếu target KHÔNG nhận ra:
    → Effort WASTED → zero Pattern-Resonance → frustration compound
    → "Tôi cố hết sức mà bố vẫn không hiểu"
    → = Worst cost/reward ratio: maximum effort, zero return
    
  Nếu target PHẢN ỨNG NGƯỢC (khó chịu vì "giả tạo"):
    → Target's SPM detect inauthenticity → valence GIẢM
    → = Effort BACKFIRE — suppress + fabricate → Pattern-Resonance WORSE
    → "Càng cố càng xa" — vicious feedback
```

**Cost analysis — CUMULATIVE qua thời gian:**

**(a) Autonomy cost (Autonomy-Hardware §5):**

Mỗi lần suppress = efference mismatch → body detects "tôi KHÔNG ĐANG LÀ TÔI." Repeated → vmPFC meta-learning: "trong relationship NÀY, tôi không thể là tôi." → DRN domain-specific tendency → passive coping IN THIS RELATIONSHIP (Autonomy-Hardware §2.2: vmPFC learns PER DOMAIN). = "Tôi chịu được, nhưng tôi không sống" — autonomy SURRENDERED ở domain này.

**(b) Cortisol accumulation (Cortisol-Baseline §2.3, §7.7):**

Mỗi forced interaction = mild cortisol (Cortisol Role ② Holding: gap unresolved — "tôi vẫn chưa ĐƯỢC là tôi"). Repeated → baseline SHIFT upward. ## §8 — Communication outcome correlation

Pattern-Resonance presence/absence correlates with observable communication outcomes. This section formalizes the correlation.

### §8.1 — Positive correlation signals

**PR present** tends to correlate with:
- Conversation deepening (topics become more personal, vulnerability increases)
- Mutual investment (both sides contribute, build on each other)
- Time extension (conversation lasts longer than planned)
- Follow-up desire (both want to meet again)
- Remembered positively later
- Subsequent behavior aligned with prediction
- Relationship progression over time

### §8.2 — Negative correlation signals

**PR absent or failed** tends to correlate with:
- Conversation stalling (topics dry up, awkward silences)
- One-sided effort (one talks, other withdraws)
- Time contraction (conversation ends early, excuses made)
- Avoidance of next meeting
- Remembered as strained or unsatisfying
- Subsequent behavior contradicts prediction
- Relationship regression or termination

### §8.3 — Not perfectly predictive

Caveats:
- **Social norms** mask PR signals (polite conversation continues despite PR absence)
- **Power dynamics** affect behavior independently of PR (subordinate defers regardless)
- **Individual differences** in expression (introverts vs extroverts express PR differently)
- **Context** matters (PR at work vs PR at party vs PR in therapy)
- **One-sided detection error**: person may feel strong PR that other doesn't, leading to misread signals

### §8.4 — Framework claim

Over **many interactions**, PR presence/absence correlates with communication outcomes at statistical level. Single interaction ambiguous; patterns over time clearer.

**Practical implication**: If relationship consistently fails to deepen despite sustained effort, likely PR not emerging (conditions not met). If relationship naturally progresses without strain, likely PR present.

### §8.5 — Directional causation

**Causation direction**: PR → outcome, not outcome → PR

- PR presence (mutual SPM match) → both engaged + verified → outcomes emerge
- PR absence → one or both disengage → outcomes fail

Not: outcomes create PR. Outcomes reveal whether PR was present. This is why framework treats outcomes as evidence for retrospective PR inference.

### §8.6 — Long-term relationship trajectory

**Sustained deep PR** → relationship deepens + stabilizes + invests
**Sustained shallow PR** → relationship maintains but doesn't deepen
**Declining PR over time** → relationship decays, may end
**Never-emerging PR** → relationship fails early

**Implication for relationship repair**: Address conditions preventing PR emergence (similarity adjustment via better understanding, feedback restoration, pattern-matching through shared experiences). Not just behavior change — underlying PR conditions.

## §9 — State-gated cases (4-case taxonomy)

Preserved from old `Pattern-Resonance.md` outline §4 — 4 key cases that illustrate state-gating and decisive framework evidence.

### §9.1 — Case 1: Aligned positive ("Tôi vui + thấy mọi người vui → càng vui hơn")

**Scenario**: Self is in positive state, observes others in positive state, feels even more positive.

**Framework mechanism**:
- Self's current affective state: positive
- Self's SPM fires on others' positive cues
- Chunks retrieved: positive affective patterns (matching self-state)
- Template projection: "they feel what I feel"
- Output read: confirmed positive
- **Matching patterns present** → PR emerges → mutual reinforcement
- Feedback loop amplifies (both enjoying increases mutual enjoyment)

**Subjective experience**: Mood lift, social contagion, collective celebration, "cùng vui"

**Evidence**: Emotional contagion research 🟢, collective effervescence (Durkheim) 🟢

**Case significance**: Simplest case — aligned states produce PR easily. This is why parties, celebrations, mutual joyful moments feel powerful.

### §9.2 — Case 2: State-gated mismatch ("Tôi buồn + thấy mọi người vui → không quan tâm")

**Scenario**: Self is in negative state, observes others in positive state, experiences neutral detachment rather than pain.

**Framework mechanism**:
- Self's current affective state: negative (sadness)
- Self's state gates chunk retrieval — sad-state chunks prioritized
- SPM fires weakly on others' positive cues (templates don't match current state)
- No full simulation fires (state incompatibility)
- Self withdraws or disengages — self-protection
- **No PR emerges, but no active distress either**

**Subjective experience**: "Not in the mood", "can't connect", withdrawal without pain

**Framework note**: This is **protective disengagement**. Self's state-gating prevents forced SPM on incompatible state, avoiding overload.

**Clinical significance**: Distinguishing this from depression withdrawal. Healthy state-gating allows temporary disengagement; depression may involve persistent failure to re-engage even when state improves.

### §9.3 — Case 3: Cognitive without affective ("Tôi buồn + biết họ vui + không tận hưởng được + khó chịu") — DECISIVE

**Scenario**: Self is in negative state, consciously recognizes others are positive, cannot feel enjoyment, experiences dissonance as negative feeling.

**Framework mechanism**:
- Self's current affective state: negative
- **Cognitive SPM fires**: verbal-cognitive template recognizes "others are happy"
- **Affective SPM fails to fire**: own sad state blocks affective matching
- Recognition without resonance → **dissonance**
- Dissonance itself becomes negative valence
- Result: self feels worse, not neutral — active discomfort

**Subjective experience**: "I know I should feel happy for them but I can't", envy, cynicism, depressive anhedonia, burnout

**Why this case is DECISIVE for framework**:

This case dissociates cognitive from affective SPM cleanly. If SPM were single monolithic mechanism, couldn't have "know but can't feel" experience. The dissociation shows:

1. Cognitive and affective Pattern-Types can fire independently
2. Self-awareness of the dissociation produces additional negative feeling
3. This explains depression anhedonia, envy, burnout, cynicism — phenomena hard to explain with simpler models

**Research alignment**:
- Depression anhedonia research 🟢
- Envy psychology 🟢
- Burnout research 🟢 (empathy fatigue often involves cognitive-without-affective)

**Framework commit**: This case is one of the **most decisive** for Pattern-Type independence claim (Agent.md §16 Prediction P3).

### §9.4 — Case 4: Mutual mismatch ("A-B classroom example")

**Scenario**: Two people in same situation, fire SPM toward each other, but each's SPM produces different interpretations based on their own values/patterns.

**Framework mechanism**:
- Both A and B fire SPM toward each other
- Both retrieve own-chunk templates
- Templates differ because chunks differ (A values studying, B values socializing)
- A's projection: "B is wrong to chat in class"
- B's projection: "A is too tense, no fun"
- Both SPM "accurate" within own frame but incompatible
- **No PR emerges** despite both trying

**Subjective experience**: Both feel "we don't understand each other" or "we're different types"

**Key insight**: PR failure is **not always about one party's deficiency**. Mutual mismatch is common and normal. Neither A nor B is wrong; they're genuinely different in values/patterns.

**Resolution pathways**:
- **Accept difference**: learn to coexist without forcing alignment
- **Cognitive bridging**: understand each other's perspective intellectually even without affective PR
- **Find domain of match**: discover where their patterns do align (shared interests, values)
- **Accept relationship limits**: not every pair can achieve deep PR

**Framework insight**: Society often assumes everyone should be able to "connect" with everyone. Framework says: natural limits exist. Mutual mismatch is architectural, not moral failure.

## §10 — Cognitive-affective split

Building on §9 Case 3, this section details the cognitive-affective split phenomenon and its framework implications.

### §10.1 — The phenomenon

**Cognitive-affective split** = Self can **know** target's state without **feeling** it. Cognitive SPM fires (verbal-cognitive or visual-symbolic), affective SPM does not (or is suppressed/inhibited).

**Observational signature**:
- "I know intellectually they're suffering but I don't feel it"
- "I understand what's happening but can't connect emotionally"
- "My head gets it but my heart doesn't"
- "I recognize but don't resonate"

### §10.2 — Conditions producing the split

**Cause 1 — Current state incompatibility (temporary)**:
- Self is in mood incompatible with target's
- Example: tired person at celebration — cognitively knows it's celebration, affectively can't join

**Cause 2 — Chronic state dissociation (depression)**:
- Depressed state persistently blocks affective resonance
- Cognitive capacity intact ("I know my friend is happy")
- Affective engagement absent ("I can't feel happy for them")
- Contributes to depression's isolating quality

**Cause 3 — Burnout (empathy fatigue)**:
- Prolonged high-affective SPM engagement depletes resources
- Protective down-regulation of affective SPM
- Cognitive SPM remains functional
- "I know they need my empathy but I have nothing left"

**Cause 4 — Trauma avoidance**:
- Trauma makes affective engagement painful
- Protective suppression of affective SPM for specific triggers or broadly
- Cognitive recognition preserved

**Cause 5 — Alexithymia**:
- Poor self-chunks for emotions → poor affective SPM substrate
- May still have cognitive chunks intact
- Classical Bird & Cook 2013 pattern

**Cause 6 — Psychopathy (debated)**:
- Constitutional weak affective SPM
- Cognitive SPM intact
- Enables manipulation without affective restraint
- Framework marks 🟡 (contested)

### §10.3 — Experience of the split from inside

**Immediate experience**:
- "I feel broken"
- "Why can't I just feel this?"
- "I'm here but not here"
- "Going through the motions"

**Self-directed consequences**:
- Shame (if feeling split is culturally judged)
- Self-alienation
- Distrust of own emotional capacity
- Despair ("I've lost the ability to feel")

**Relationship consequences**:
- Partners notice diminished affective engagement
- Relationships strain
- Isolation can result
- Vicious cycle (isolation → less calibration → worse state)

### §10.4 — Distinguishing causes matters for intervention

Different causes require different interventions:

- **Temporary state incompatibility** → time, mood regulation, activity shift
- **Depression** → medical/psychological depression treatment
- **Burnout** → rest, reduced affective engagement, self-care
- **Trauma avoidance** → trauma therapy, graded exposure, processing
- **Alexithymia** → self-chunk building, Focusing, body-based practices
- **Psychopathy** → limited intervention options, often managed not cured

Misidentifying the cause → wrong intervention → worse outcomes. Framework implication: cognitive-affective split is not one phenomenon but several with overlap.

### §10.5 — Why cognitive-affective split matters for framework

1. **Dissociates Pattern-Types**: Evidence against monolithic SPM claim
2. **Explains painful emotions**: Envy, burnout, depression anhedonia become mechanism-based, not mysterious
3. **Clinical relevance**: Framework maps clinical phenomena to specific mechanism failures

## §11 — Individual + cultural variation

Brief section on how PR quality varies across individuals and cultures.

### §11.1 — Individual variation sources

**Self-chunk library richness**:
- Alexithymia spectrum → thin affective chunks → reduced affective PR
- High self-awareness → rich chunks → high-quality PR potential

**Pattern-Type bias**:
- Somatic-dominant individuals → strong affective + somatic PR
- Verbal-dominant individuals → strong verbal-cognitive PR
- Visual-dominant individuals → strong visual-symbolic PR
- Composite-capable individuals → rich PR across modalities

**Attachment history**:
- Secure attachment → flexible PR engagement, robust to conflict
- Anxious attachment → intense PR pursuit, distress at perceived absence
- Avoidant attachment → limited PR engagement, protective distance
- Disorganized attachment → inconsistent PR patterns

**Trauma history**:
- Traumatized individuals may have systematic PR distortions (threat projection, avoidance patterns)
- Recovery enables restored PR capacity

**Training level**:
- Trained individuals (therapists, meditators, deep listeners) achieve higher-quality PR even with strangers
- Untrained individuals rely on default patterns

### §11.2 — Cross-cultural variation

**Emotional vocabulary**:
- Cultures with rich emotional vocabulary → rich affective chunks → rich affective PR
- Cultures with sparse vocabulary → coarser affective PR (but may have rich other Pattern-Types)

**Body awareness traditions**:
- Cultures with strong body practices (yoga, martial arts, meditation traditions) → rich somatic PR
- Screen-mediated + verbal-dominant cultures → reduced somatic

**Communication norms**:
- High-context cultures → rich implicit PR detection (non-verbal emphasis)
- Low-context cultures → explicit verbal PR confirmation (verbal-cognitive emphasis)

**Collective vs individualist orientation**:
- Collective cultures → PR more embedded in group context, easier large-scale PR
- Individualist cultures → PR more dyadic, more explicit negotiation

**Framework caution**: Cross-cultural research on PR itself is thin (framework-specific construct). Observations above are inferences from broader cross-cultural psychology, not direct PR measurement.

### §11.3 — Developmental variation

Already covered in Agent.md §13 developmental timeline. Briefly:
- PR capability emerges Stage 2 (18-24 months, with SPM)
- PR depth increases through Stage 4-6 (calibration + expansion)
- Adult PR varies based on individual life history + training
- Late-life PR can remain strong or decline depending on health + engagement
