## Background — Why This Project Exists

Sau khi Feeling/ folder Phase 2 complete (Feel-Example-Draft.md filled với
124 examples including §9b drive chunks), user raised meta-question về
body-base needs:

> "Chúng ta có thực sự cần thiên nhiên không? Có 2 giả thiết... nature 
> như nhu cầu thiết yếu explore vô thức, fire pattern theo domain real... 
> hoặc 2 triệu năm tiến hóa với receptors cho natural signals"

Discussion revealed:

1. **Missing foundation file**: Body-Base folder có 5 files nhưng KHÔNG có
   central "Body-Base.md" describing whole system architecture. Current
   coverage scattered:
   - Theory: Why-Body-Knows.md (META — why body knows)
   - Signals: Body-Dissonance.md (14+ dissonance types)
   - Inputs: Calibration-Inputs-Analysis.md (DRAFT with nature, movement,
     living pattern, agent inputs)
   - Applications: Body-Personal-Optimization.md, Body-Parenting-
     Optimization.md
   - Downstream: Feeling/ subfolder (7-layer feeling system)
   - **Gap**: no entry point, no systems overview, no channel
     enumeration with full depth

2. **Slow-decay needs concept** (NEW framework concept from user):
   - Acute replenishment needs (food, water, sleep): miss → crisis
   - Slow-decay needs (nature, movement, social, meaning): miss → diffuse
     unease, không crisis, silent deficiency
   - Pattern fire + storage + decay model for slow-decay needs
   - Silent deficiency as real phenomenon, requires specific detection
     methodology

3. **Methodology breakthrough**: extreme passion cases for signal
   isolation:
   - Instead of average-population studies (too noisy)
   - Use extreme passion cases where ONE channel dramatically dominant
   - Reveals what over-fed channel provides + what starved channels
     would have provided
   - Complementary to survival extremes already documented in
     Calibration-Inputs-Analysis.md §4 (Onoda, Proenneke, etc.)
   - Together: complete picture of channel architecture

4. **"Living Pattern Input"** — novel framework contribution already
   identified trong Calibration-Inputs-Analysis.md but not widely known
   in mainstream. Distinct from both physical environment và agent
   input. "Watching life move" as separate channel (aquarium effect,
   bird watching, people watching).

This project materializes these insights into formal framework files.
