## §2 — EVIDENCE CHO NATURE INPUT NEED

### Research foundation

```
🟢 ESTABLISHED RESEARCH:

  1. BIOPHILIA HYPOTHESIS (E.O. Wilson 1984):
     → Con người có xu hướng BẨM SINH tìm kiếm kết nối với thiên nhiên
     → 2 triệu năm evolution TRONG tự nhiên → body calibrated
     → 4 bức tường = mới ~10,000 năm → body CHƯA calibrate cho nó

  2. ATTENTION RESTORATION THEORY (Kaplan & Kaplan 1989):
     → Môi trường nhân tạo → directed attention FATIGUE
     → PFC mệt → "đần đầu"
     → Thiên nhiên → "soft fascination" → attention PHỤC HỒI mà KHÔNG tốn PFC
     → Cây, mây, gió = ENGAGES attention nhẹ, không đòi PFC process
     → PFC được NGHỈ trong khi body vẫn nhận input

  3. STRESS REDUCTION THEORY (Ulrich 1991):
     → Nhìn thiên nhiên (qua cửa sổ) → cortisol GIẢM đo được
     → Bệnh nhân phòng có view nhìn cây → hồi phục NHANH HƠN
       phòng nhìn tường gạch (classic study)

  4. SHINRIN-YOKU / Forest Bathing (Li 2010, Nhật Bản):
     → Đi trong rừng → NK cells (immune) TĂNG
     → Cortisol GIẢM, blood pressure GIẢM
     → Cơ chế: PHYTONCIDES — hóa chất cây TIẾT ra
     → Người hít vào → body respond ở mức biochemistry
     → = KHÔNG CHỈ do O2 hay mát — CÓ chemical signals thật

  5. FRACTAL PATTERNS (Taylor 2006, Hagerhall 2004):
     → Thiên nhiên đầy fractal patterns (cành cây, mây, sóng, lá)
     → Visual system người PREFER fractal complexity D ≈ 1.3-1.5
     → Tường phẳng = D ≈ 1.0 (quá đơn giản) → KHÔNG engage
     → Cây, bờ biển, đồng cỏ = "SWEET SPOT"
     → Body THÍCH nhìn cây VÌ visual pattern ở optimal complexity

  6. NATURE DEFICIT DISORDER (Richard Louv 2005):
     → Trẻ ÍT tiếp xúc thiên nhiên → behavioral + health issues
     → Not clinical diagnosis, but research support

  7. KUO & TAYLOR (2004):
     → Trẻ ADHD → hoạt động trong THIÊN NHIÊN → triệu chứng GIẢM đo được
     → So với indoor → ít hiệu quả hơn

  8. WELLS & EVANS (2003):
     → Trẻ sống gần thiên nhiên → STRESS RESILIENCE cao hơn

  9. DADVAND et al. (2015, Barcelona):
     → 2,593 trẻ 7-10 tuổi
     → Nhiều greenspace xung quanh trường → COGNITIVE DEVELOPMENT tốt hơn
     → Working memory + attention tests measurable improvement

  10. FABER TAYLOR & KUO (2009):
      → Trẻ chơi trong môi trường GREEN → concentration TĂNG sau đó
      → Trẻ chơi indoor hoặc concrete → KHÔNG có effect
```

## §3 — EVIDENCE CHO LIVING PATTERN INPUT (most novel)

### Distinction from physical environment

```
🟡 LIVING PATTERN INPUT ≠ Physical environment ≠ Agent input:

  PHYSICAL ENVIRONMENT:
    → Sun, wind, air, temperature
    → Direct body feed (skin, lung, thermoregulation)
    → Doesn't require "seeing"

  AGENT INPUT:
    → Requires social presence/interaction
    → Involves modeling, mirror, reciprocity
    → Social Baseline Theory

  LIVING PATTERN INPUT (proposed distinct):
    → Observing life moving in domain
    → Does NOT require interaction
    → Does NOT require modeling (not same as agent processing)
    → Does NOT require physical contact (just visual observation)
    → Body signal: "domain is alive, healthy, functional"
    → Confirms brain's baseline expectation: "should have life around"


  FRAMEWORK HYPOTHESIS:

  Brain = PREDICTION MACHINE.
  Brain predicts based on "domain hoạt động bình thường".
  "Bình thường" theo 2M năm = domain CÓ sự sống đa dạng.

  Input "thấy châu chấu nhảy, chim bay, lá đung đưa":
    → = Confirm: "domain healthy, nothing unusual"
    → = Prediction match baseline
    → = Body relax (no prediction error)

  Input "4 bức tường tĩnh, không có gì sống":
    → = Unusual pattern (2M năm baseline nói "should have life")
    → = Body: "something missing, domain feels wrong"
    → = Diffuse dissonance, PFC không gọi tên được


  PARALLEL với Social Baseline Theory:
    SBT: brain default giả định CÓ người gần → alone = deviation
    Living Baseline (proposed): brain default giả định CÓ sinh vật xung quanh
                                → sterile environment = deviation


🟢 EVIDENCE HINTS:

  1. Aquarium effect (Cracknell 2016):
     → Nhìn cá bơi → blood pressure giảm, mood improve
     → NOT agent modeling (cá = minimal agent-like qualities)
     → NOT physical nature (qua kính, không contact)
     → CHỈ "watching life move" → body relax
     → = DIRECT evidence cho living pattern input as distinct

  2. Bird watching → mental health benefits (multiple studies)
     → Passive observation, not interaction
     → Therapeutic effect measurable

  3. Urban parks với wildlife → better stress reduction
     → vs parks without visible wildlife
     → Suggests living presence matters beyond physical nature

  4. People-watching as calming activity (common observation)
     → Passive observation of others moving
     → Not agent interaction (strangers, no communication)
     → Just "seeing life operate" = mild reward
```
