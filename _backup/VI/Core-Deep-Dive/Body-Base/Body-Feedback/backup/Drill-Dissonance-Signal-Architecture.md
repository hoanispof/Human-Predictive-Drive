## §4 — EVALUATIVE GATES DIRECT-STATE: DISSONANCE VERSION

### §4.1 — Mechanism (parallel RSA §3)

```
⭐ RSA §3: "EVALUATIVE GATES DIRECT-STATE" — ÁP DỤNG CHO CẢ 2 DIRECTIONS:

  RSA §3: Evaluative assessment MODULATES Direct-State reward signal
  → posterior insula (raw) → anterior insula (integration) → ACC → OFC

  CÙNG mechanism cho DISSONANCE:
  → Direct-State Dissonance arrives at posterior insula (RAW, unmodulated)
  → Evaluative assessment fires in parallel (context: WHO/WHAT/WHERE)
  → Anterior insula integrates BOTH → EXPERIENCED dissonance signal
  → = Evaluative MODULATES Direct-State Dissonance

  5-STEP NEURAL PROCESS (Craig 2002, applies to ALL body signals):

  Step 1: Direct-State Dissonance → posterior insula (~0-50ms)
    → Raw, unmodulated first-order pain/damage representation
    → SAME signal regardless of WHO causes it

  Step 2: Evaluative assessment fires in parallel (~50-200ms)
    → Context chunks: WHO caused this? WHY? WHAT does it mean?
    → Social schemas: trusted person? enemy? accident?

  Step 3: Anterior insula integrates (~100-300ms)
    → Direct-State signal propagates anteriorly
    → Evaluative context arrives top-down
    → Integration = EXPERIENCED dissonance
    → 🟢 Craig 2009: anterior insula = "seat of subjective awareness"

  Step 4: ACC conflict check (~200-400ms)
    → Direct-State negative + Evaluative negative → AMPLIFY (no conflict)
    → Direct-State negative + Evaluative positive → SUPPRESS (conflict)

  Step 5: OFC/vmPFC value assignment (~300-500ms)
    → Final computation integrating all signals
```

```
    Visual: large moving object → amygdala fast path (🟢 LeDoux 1996)
    Startle reflex: brainstem → freeze/startle
    = Direct-State Dissonance (hardware alarm, NO evaluation)
    Infant CŨNG react (Moro reflex)

  PHASE 2 (~12-200ms): Evaluative assessment
    Compiled chunks: "hổ = nguy hiểm = chết" (requires learning)
    Cortical evaluation: threat level, escape routes
    = Evaluative Dissonance (compiled pattern → threat prediction)
    Infant WITHOUT tiger schema: LESS Phase 2

  PHASE 3 (~200ms+): Evaluative AMPLIFIES Direct-State
    Evaluative "nguy hiểm thật" → AMPLIFY Direct-State alarm
    Cortisol, adrenaline, NE cascade
    = Evaluative Gates Direct-State (amplify mode)

  3 PROFILES:
    Trẻ nhỏ gặp hổ: mostly Phase 1 (Direct-State startle)
    Người lớn gặp hổ: Phase 1 + 2 + 3 (compound)
    Huấn luyện viên hổ: Phase 1 + Phase 2 = "hổ quen, không nguy"
      → Evaluative SUPPRESS Direct-State → LESS fear (trained handler)
```

## §10 — ASYMMETRIC TRANSITION SPEED: EVOLUTION'S SURVIVAL-FIRST DESIGN

### §10.1 — Reward → Dissonance: NHANH

```
⭐ TRANSITION TỪ REWARD SANG DISSONANCE = MILLISECONDS-TO-SECONDS:

  DEACTIVATION SPEED:
    Opioid deactivation: seconds — reward CAN stop almost instantly
    NE + adrenaline spike: ~500ms (Threat.md §1.1)
    Amygdala fast path: ~12ms (🟢 LeDoux 1996) — BYPASS PFC entirely
    ACC conflict detection: ~200-400ms

  MECHANISM (Threat.md §1.1 — Schema = TRIGGER, Hormone = SUSTAIN):
    New chunk arrives (e.g., "gián", "lừa dối", "hổ")
    → Schema fire: 0ms (compiled pattern match NGAY)
    → Chunk-Shift fires → valence FLIP → ACC detect conflict
    → NE spike: ~500ms (energy tức thì)
    → Adrenaline: 1-2s (fight/flight)
    → Opioid system: deactivation parallel (receptor internalization)
    → Cortisol: 5-20 min (SUSTAIN alert — arrives SAU nhưng stays LÂU)

  VÍ DỤ:
    Ăn ngon (opioid) → gián → buồn nôn < 1 second
    Người yêu ôm → tin nhắn lừa dối → uncomfortable NGAY
    Đang cười → nhớ "HÔM NAY THI!" → sợ < 1 second
```

### §10.2 — Dissonance → Reward: CHẬM

```
⭐ TRANSITION TỪ DISSONANCE SANG REWARD = MINUTES-TO-HOURS:

  CLEARANCE SPEED:
    Cortisol half-life: ~60-90 phút — dù threat BIẾN MẤT, vẫn trong máu
    NE clearance: minutes (reuptake + MAO degradation)
    Adrenaline clearance: ~2-3 phút nhưng somatic effects linger
    Amygdala "cool down": requires PFC inhibition (sustained effort)

  MECHANISM:
    Opioid RE-activation: cần H10 5 preconditions VERIFY LẠI → chậm
    P1 (Schema pending): threat schema vẫn active → blocks reward
    P2 (Chunks): threat chunks vẫn firing → prevent reward evaluation
    P5 (Chunk tag): cortisol tag = AVOIDANCE → opioid BLOCKED
    PFC phải RE-ASSESS: "thật sự an toàn chưa?" → costs time + budget
    Cortisol = SUSTAINER: arrives SAU, stays LÂU (Cortisol-Baseline.md §1)

  VÍ DỤ:
    Hổ đi rồi → tim vẫn đập nhanh 10-20 phút → cortisol hạ ~1-2 giờ
    Suýt tai nạn xe → run tay ~30 phút dù đã an toàn
    Cãi nhau xong → "bình tĩnh lại" takes hours, not minutes
    Sau buổi thuyết trình căng → relief CHẬM dù đã xong
```

### §10.3 — Tại sao evolution thiết kế asymmetric

```
🟡 SURVIVAL COST ANALYSIS:

  ┌──────────────────────┬──────────────────────┬──────────────────┐
  │ Error Type           │ Consequence          │ Evolutionary Cost│
  ├──────────────────────┼──────────────────────┼──────────────────┤
  │ MISS THREAT          │ CHẾT                 │ Gene bị loại     │
  │ (false negative)     │                      │ COST = MAXIMUM   │
  ├──────────────────────┼──────────────────────┼──────────────────┤
  │ FALSE ALARM threat   │ Chạy bừa, tốn NL    │ Sống, ăn sau     │
  │ (false positive)     │                      │ cost = LOW       │
  ├──────────────────────┼──────────────────────┼──────────────────┤
  │ MISS REWARD          │ Bỏ lỡ bữa ăn       │ Sống, ăn sau     │
  │ (false negative)     │                      │ cost = LOW       │
  ├──────────────────────┼──────────────────────┼──────────────────┤
  │ FALSE ALARM reward   │ Ăn nhầm độc /       │ Có thể CHẾT      │
  │ (false positive)     │ approach predator    │ COST = HIGH      │
  └──────────────────────┴──────────────────────┴──────────────────┘

  → Natural selection STRONGLY favors:
    Threat detection: NHANH, nhạy, thà nhầm còn hơn chậm
    Threat clearance: CHẬM, giữ cảnh giác lâu hơn cần thiết
    Reward activation: CHẬM, kỹ, verify H10 trước khi "an toàn"
    Reward deactivation: NHANH, rút reward ngay khi dấu hiệu nguy

  → "Better safe than sorry" = BUILT INTO HARDWARE

  🟢 Baumeister et al. 2001: "Bad is stronger than good"
    Negative events ảnh hưởng mạnh hơn positive cùng magnitude
    Một trong những findings robust nhất tâm lý học
  🟢 Kahneman & Tversky 1979: Loss aversion
    Mất 100 đồng "đau" gấp ~2x so với "sướng" được 100 đồng
  🟢 Rozin & Royzman 2001: Negativity dominance
    1 giọt "dirty" làm ô nhiễm toàn bộ (gián trong đĩa)
    1 giọt "clean" KHÔNG làm sạch dirty → ASYMMETRIC
```

### §10.4 — Asymmetry chủ yếu ở EVALUATIVE component

```
🟡 ASYMMETRY × EVALUATIVE/DIRECT-STATE:

  DIRECT-STATE: tương đối SYMMETRIC
    Pain onset ≈ touch onset (cùng hardware speed, cùng receptor pathway)
    Pain offset ≈ touch offset (khi stimulus removed)
    Hardware không "giữ" signal lâu hơn stimulus

  EVALUATIVE: STRONGLY ASYMMETRIC
    Evaluative threat detection: NHANH
      → amygdala ~12ms, CAN bypass PFC entirely
      → compiled threat templates fire WITHOUT cortical verification
    Evaluative reward evaluation: CHẬM
      → requires H10 5 preconditions, PFC involvement
    Evaluative threat clearance: CHẬM
      → cortisol sustain 60-90 min half-life
      → PFC must actively re-assess + inhibit amygdala
    Evaluative reward deactivation: NHANH
      → opioid off in seconds upon threat detection

  → Asymmetry resides in EVALUATIVE layer, NOT Direct-State
  → Compilable Architecture cho phép:
    FAST compiled threat detection (amygdala = compiled threat templates)
    SLOW compiled reward verification (H10 = thorough evaluation)
  → = Evolution optimize Evaluative system cho SURVIVAL-FIRST
  → = WHY "calming down" takes time (biochemical constraint, not weakness)
  → = WHY 1 negative event can ruin an entire positive experience
    (reward→dissonance: instant. dissonance→reward: hours)
```
