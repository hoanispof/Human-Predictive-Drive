| Signal direction | Intensity range | Context examples |
|---|---|---|
| **Dissonance nhẹ** | Micro-Mild (L3 pattern, 5-30% body resource) | Mini daily dissonance, chunk linking, learning struggle, flow-edge, social comparison |

**Framework confirmation**: This distinction is ALREADY committed in framework via **3 states of knowledge** (Somatic-Articulation-Loop §1: Implicit → Transitional felt sense → Explicit) + **7-layer feeling model** (Feeling.md / Theme A §2.4: Raw → Integration → Noticeable → Observe → Locate → Label → Explain) + **Self-signal interoception keystone** (Body-Input-Enumeration §4.9: reading capacity as prerequisite).

**Key insight**: body-feedback (raw, L1-L2) vs feeling (PFC observation, L3-L7) is **NOT binary** — it's a **gradient trainable over developmental time**. User's gradient examples ARE the training trajectory:

| Developmental stage | State | Example | Framework layer |
|---|---|---|---|
| Infant (pre-chunks) | Implicit auto-react | Đái → ướt / đau → khóc | Layer 1-2 only (body) |
| Early childhood | Post-hoc recognition | Vừa đái → biết mình vừa đái | Layer 3-4 emerge |
| Child (language forming) | Locate + label | "Đau ở chân này" / "con sắp đái" | Layer 5-6 compile |
| Adult (rich chunks) | Explain + attribute | "Buồn vì người yêu" / "stress vì sếp" | Layer 6-7 full |
| Expert (mindfulness training) | Refined felt sense | Gendlin felt sense articulation | Trained Layer 3-5 access |

**Critical claim**: Body-feedback is **foundation**. Feeling layer **built on top** via chunks accumulation. Without self-signal interoception keystone (§4.9), feeling layer **cannot develop** regardless of chunks quantity → alexithymia (Bird & Cook 2013, ~10% population).
