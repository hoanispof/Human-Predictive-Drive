## 2. Contentment vs Boredom — Smooth ≠ Chán

```
⭐ SMOOTH MELODY → BODY REWARD → CONTENT. Không mâu thuẫn.
   "Chán" KHÔNG BAO GIỜ xảy ra khi melody THẬT SỰ smooth.

  CONTENTMENT (IDLE mode — bãi biển):
    Body-base layers:
      L1: ấm, thoải mái ✅
      L2 Experience: sensory RICH
        → Visual: trời, biển, mây → liên tục thay đổi NHẸ
        → Auditory: sóng biển → pattern đều nhưng mỗi sóng HƠI khác
        → Somatic: gió, nắng, cát, nhiệt → kích thích nhẹ LIÊN TỤC
      L2 Connection: relax → ok ✅
      Motor: body thả lỏng, cơ bắp relax ✅
    Cortisol: rất thấp → IDLE mode
    → = TẤT CẢ MET → body reward → melody THẬT SỰ smooth → content
    → = Có thể nằm HÀNG GIỜ mà KHÔNG chán

  BOREDOM (LAZY mode — ngồi nhà):
    Body-base layers:
      L1: ok ✅
      L2 Experience: sensory NGHÈO
        → Visual: cùng 4 bức tường → KHÔNG thay đổi
        → Auditory: im lặng hoặc tiếng nền quen → KHÔNG input
        → Somatic: cùng ghế, cùng nhiệt, không gió → KHÔNG kích thích
      L2 Connection: có thể thiếu (một mình) ❌ nhẹ
      Motor: ít vận động → body ứ ❌ nhẹ
    Cortisol: HƠI CAO hơn idle (vì L2 + motor unmet nhẹ) → LAZY mode
    → = KHÔNG thật sự smooth → body "khó chịu nhẹ" → PFC: "chán"

  MAP VÀO CORTISOL MODES (Core §3):
    IDLE:   cortisol rất thấp → TẤT CẢ met → content         = BÃI BIỂN
    LAZY:   cortisol hơi trên idle → CÓ GÌ ĐÓ unmet nhẹ       = "CHÁN"
    ACTIVE: cortisol vừa → đang làm, engaged                   = LÀM VIỆC

  → = "Chán" = LAZY mode = cortisol HƠI CAO hơn comfortable
  → = KHÔNG phải "smooth mà chán" → mà "KHÔNG THẬT SỰ smooth"


  ⭐ INSIGHT QUAN TRỌNG — "sóng biển KHÔNG MỚI nhưng KHÔNG chán":

    Sóng biển: mỗi sóng TƯƠNG TỰ → KHÔNG phải "cái mới"
    NHƯNG: mỗi sóng kích thích auditory + visual + somatic NHẸ
    → = L2 Experience ĐƯỢC FEED liên tục (dù không "mới")
    → = "Mới" KHÔNG quan trọng → "FEED ĐỦ channel" mới quan trọng

    Tường nhà: cũng TƯƠNG TỰ nhau → NHƯNG KHÔNG feed gì cả
    → = Visual channel BỊ ĐÓI (cùng input tĩnh, không thay đổi)

    → = Mainstream có thể nói: "bãi biển CÓ micro-novelty (mỗi sóng hơi khác)"
    → = Framework nói: "bãi biển FEED sensory channels (dù pattern tương tự)"
    → = CÓ THỂ cùng 1 thứ, khác góc nhìn
    → = NHƯNG framework CỤ THỂ hơn: chỉ ra CHANNEL NÀO đang được feed / bị đói
    → = Và giải thích tại sao "có novelty MÀ VẪN chán" (mismatch — §3 ④)
```
