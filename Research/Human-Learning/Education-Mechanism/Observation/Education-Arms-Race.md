> **When 1 family hires a private tutor → their child gains an advantage.**
> **When ALL families hire a private tutor → NOBODY has an advantage, but EVERYONE still pays.**
>
> Like an audience standing on tiptoe: 1 person tips up → sees better.
> Everyone tips up → no one sees better, but everyone's feet hurt.
>
> Education arms race = prisoner's dilemma at social scale.
> Individually rational. Collectively catastrophic.
> Fertility down 28% from status externalities in education alone (AER 2024).
> No country has ever successfully STOPPED a race once it started.

## §1 — Mechanism: Why the Race Self-Accelerates

```
🟢 EDUCATION = "POSITIONAL GOOD" — VALUE DEPENDS ON RELATIVE POSITION:

  Fred Hirsch (1976, Social Limits to Growth):
    → "The value of spending on education at a given level DECREASES
       when MORE PEOPLE reach that level"
    → = A bachelor's degree once WAS ENOUGH to get a good job
    → Now: BA = minimum entry → need a master's → need a prestigious university
    → = Value is NOT absolute → it's RELATIVE to everyone else


🟢 PRISONER'S DILEMMA — GAME THEORY STRUCTURE:

  ┌────────────────────────────────────────────────────────────────┐
  │                      FAMILY B                                  │
  │                 No extra tuition    Extra tuition              │
  │  ┌──────────────────────────────┬──────────────────────────┐   │
  │  │ A: No extra  │ BOTH OK       │ A LOSES, B WINS          │   │
  │  │ tuition      │ (collective   │ (worst outcome for A)    │   │
  │  │              │  optimal)     │                          │   │
  │  ├──────────────┼───────────────┼──────────────────────────┤   │
  │  │ A: Extra     │ A WINS        │ TIE — but BOTH PAY       │   │
  │  │ tuition      │ B LOSES       │ (Nash equilibrium)       │   │
  │  └──────────────┴───────────────┴──────────────────────────┘   │
  └────────────────────────────────────────────────────────────────┘

  → DOMINANT STRATEGY: extra tuition REGARDLESS of what others do
  → = Nash equilibrium: ALL do extra tuition → NOBODY gains advantage
    → but EVERYONE still pays
  → = Collectively optimal: NOBODY does extra tuition → SAME outcome → FREE
  → = BUT: collectively optimal = UNSTABLE (someone will "defect")


🟢 CREDENTIAL INFLATION — RED QUEEN EFFECT:

  Data (US):
    → Analysis of 26 MILLION job postings: employers INFLATE degree requirements
    → 6.2 million workers LOSE opportunities due to credential inflation
    → 19% of secretaries have a BA → but 65% of job postings REQUIRE a BA
    → 39% of IT support have a BA → but 70% of postings REQUIRE it
    → 50%+ graduates START careers in jobs that DON'T REQUIRE a degree
    → 40 fields now REQUIRE a master's → previously only needed a bachelor's

  → = "Running faster just to stay in the same place" (Lewis Carroll)
  → = Credentials RISE → requirements RISE accordingly → value FALLS → need HIGHER credentials

  ⭐ FRAMEWORK CONNECTION:
    → Credential inflation is not just an economic phenomenon
    → It RESHAPES societal Imagine-Final: the "success marker" keeps moving
    → The body of society chases a moving target → never reaches satisfaction
    → = Prediction-delta never reached (goal post keeps moving)
    → = Societal hedonic treadmill version of Brickman 1978 (individual)

    🟡 GAP-DISTRIBUTION-PROFILE LENS (Gap-Distribution-Profile.md v1.1):
    → Credential inflation = Gap-Distribution-Profile HOMOGENIZATION:
      All graduates share the same profile: narrow abstract center, installed-dominant, rigid
    → Arms race FORCES 1 gap landscape onto ENTIRE SOCIETY → diversity collapse
    → Millions with the same profile → value of that profile → 0
    → = Credential inflation MECHANISM through the Gap-Distribution-Profile lens


🟢 SIGNALING THEORY — Michael Spence (Nobel 1973):

  → Education CAN primarily be a SIGNAL of ability
    → NOT actually building real ability (human capital)
  → Employers CANNOT directly measure ability → use credentials as proxy
  → When EVERYONE has a degree → signal INFLATES → need HIGHER degree
  → = Arms race PRODUCES signals → NOT capabilities
  → 🟢 Fed Cleveland (2025): still found "substantial signaling component"


🟢 MORE SPENDING ≠ MORE OUTCOMES (beyond threshold):

  → 90% of studies: more spending = better outcomes → BUT:
  → +$1,000/student → +0.15 SD test scores → diminishing returns
  → Effect STRONG in countries spending BELOW median → NEGLIGIBLE in high-spending countries
  → High-income countries: NO relationship between PISA scores and spending
  → = Arms race spending BEYOND threshold = WASTE
  → = Consistent with game theory: spending more on a positional good = zero-sum
```
