### §1.1 — Literacy / Communication

```
🟢 LANGUAGE + COMMUNICATION:

  THE TREE:
    Root   → Read, write, speak, listen (decode + encode)
    Trunk  → Express ideas, argue, persuade
    Branches → Literature, journalism, public speaking, poetry,
               academic writing, storytelling, debate, translation...
    Leaves → Infinite — every domain has its own language

  WHY FOUNDATION:
    → Every other domain NEEDS literacy to ACCESS it
    → Cannot read → cannot learn (any era)
    → Even if AI reads/writes for you → "UNDERSTAND meaning" = human skill
    → Language processing = core brain function 🟢

  PREREQUISITE:
    → Foundation chunks from 0–6 (Child-Dev: language exposure)
    → Phonemic awareness → decoding → fluency → comprehension → expression
    → = LONGEST and EARLIEST prerequisite chain

  TIMING:
    → Start earliest (sensitive period 0–6 for language)
    → Continues lifelong — depth can always increase
    → AI era: literacy does NOT decrease in value — FORMAT changes
      (typing vs handwriting, prompting vs commanding)
```

### §1.2 — Numeracy / Logic

```
🟢 MATH + LOGIC + PATTERN RECOGNITION:

  THE TREE:
    Root   → Counting, addition/subtraction/multiplication/division,
             comparison, measurement
    Trunk  → Logic reasoning, pattern recognition, problem-solving
    Branches → Algebra, geometry, statistics, probability,
               programming logic, data analysis, financial literacy...
    Leaves → Infinite — applied math is everywhere

  WHY FOUNDATION:
    → Logical thinking = SUBSTRATE for all analysis
    → Pattern recognition = how the brain UNDERSTANDS the world 🟢
    → Even if AI does the calculating → "UNDERSTAND the logic" ≠ "pressing a button"
    → Basic numeracy = prerequisite for science, technology, economics

  PREREQUISITE:
    → Concrete → abstract (PFC development timeline)
    → Counting → operations → logic → abstract reasoning
    → ⚠️ Abstract math too early (PFC not yet ready) = avoidance tag common
      → Many "hate math" = wrong timing, NOT lower capability

  TIMING:
    → Concrete math: early (3–6, through physical experience)
    → Abstract math: after PFC is sufficiently developed (~10–12+)
    → AI era: "knowing how to ask the right math question" > "calculating fast"
```

```

  THE TREE:
    Root   → Self-awareness ("this is hard/easy for me")
    Trunk  → Learning strategies, self-assessment, error analysis
    Branches → Research methodology, critical thinking, self-directed learning,
               knowledge management, transfer across domains...
    Leaves → Infinite — meta-learning = engine for ALL other domains

  WHY FOUNDATION:
    → Era speed → specific knowledge becomes obsolete quickly
    → Meta-learning = "the skill to LEARN anything new on your own"
    → = Adaptability > specific knowledge (Education-Mechanism §0.3)
    → Meta-learning chunks = chunks ABOUT HOW TO COMPILE chunks
      → Person with meta-chunks → era shifts → CAN compile new chunks independently
      → Person without → era shifts → stuck
    → Simulation-Engine: meta-learning = training 1 shared engine
      → improves ALL applications (Self-Pattern-Modeling, Imagine-Final, Self-Observation)
      → skilled meta-learner = STRONGER Simulation-Engine (Simulation-Engine.md v1.1)
    → Dissonance-Signal-Architecture: recognize WHICH TYPE of dissonance during learning
      → Evaluative (compiled, workable by reframe) vs Direct-State (hardware, needs rest)
      → = meta-skill: knowing WHEN to push, WHEN to stop
      (Dissonance-Signal-Architecture.md v1.0)
    → Trust Calibration: meta-skill of EVALUATING SOURCE RELIABILITY
      → Trust DEFAULT = per-entity (trust the PERSON, no per-domain distinction)
      → Trust CALIBRATED = per-domain (trust THIS PERSON on THIS domain — PFC-intensive)
      → Default→Calibrated = LEARNING PATH that needs to be taught
        (4 phases: Installed→Default→Calibrate→Override)
      → Foundation domains: trust in established knowledge = HIGH by default
        (math rules, physics laws)
      → Era-specific domains: trust in new knowledge = LOWER, needs calibration
        (AI claims, health trends)
      → = Meta-learning MUST TEACH: evaluate RELIABILITY, not just evaluate content
      (Trust.md v1.0 §5 — Default→Calibrated 4 Phases)
    → Self-Observation: formal mechanism for "self-awareness" at the Root
      → "This is hard/easy for me" = Self-Observation Level 2–3 (compiled → detect)
      → Meta-learning = Self-Observation applied to the learning process
      → Tool Not Virtue: Level 2 SUFFICIENT for routine learning,
        Level 3+ for novel domains
      → ⚠️ Domain feedback = THE ONLY DIFFERENTIATOR — same mechanism for
        Einstein AND a conspiracy theorist (Self-Observation.md v1.0 §9.4)
      → = Quality meta-learning = Self-Observation + domain feedback
      (Self-Observation.md v1.0 §9 — Tool Not Virtue)

  PREREQUISITE:
    → PFC needs to be sufficiently developed for meta-cognition
      (~10–12+ for explicit meta-learning)
    → PFC Budget: meta-cognition = HIGH PFC operation → finite resource
      → cannot meta-learn + domain-learn + social AT FULL CAPACITY simultaneously
      (PFC-Operations.md v1.1 §9)
    → Other Foundation domains = "vehicle" for meta-learning to emerge
      (while learning math → also learning HOW TO LEARN math)
    → = Meta-learning is NOT TAUGHT SEPARATELY → EMERGES in parallel with domain learning

  TIMING:
    → Seeds: 6–8 (implicit — "this is hard/easy for me")
    → Explicit: 10–12+ (PFC sufficiently developed for reflection)
    → Self-directed: 15–18+ (PFC ~70–90%)
    → Lifelong: THE ONLY domain that continues developing throughout life
    → = Meta-learning STARTS LATEST but LASTS LONGEST

  🟡 Meta-learning as foundation = framework emphasis
  🟢 "Learning to learn" validated: Hattie 2009, OECD Learning Compass 2030
```
