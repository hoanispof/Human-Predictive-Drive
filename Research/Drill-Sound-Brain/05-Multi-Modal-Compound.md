# Multi-Modal Compound — Audio + Visual + Action Sync Hierarchy

> **Listening to good music = pleasant.**
> **Watching a good music video = even more pleasant.**
> **Playing Beat Saber = INTENSELY pleasant.**
>
> **Why does the SAME piece of music → MORE reward when you ADD modalities?**
>
> **Not just additive. Potentially SUPER-ADDITIVE — IF coherent.**
> **3 modalities × COHERENT sync = compound reward (sometimes > sum of parts).**
> **3 modalities × INCOHERENT = compound dissonance.**
>
> **Coherence = CONDITION. Not automatic.**
> **More modalities ≠ more reward. More modalities × SYNC = more reward.**
>
> **This file: 3-tier hierarchy + coherence condition + game analysis.**

## §0 — Position + Thesis

```
⭐ THESIS:

  Multi-modal experience = HIERARCHY of compound reward:

    Tier 1: Sound alone    (1-3 modalities) → Evaluative baseline
    Tier 2: + Visual       (3-4 modalities) → enhanced multi-modal binding
    Tier 3: + Action       (4-5 modalities) → compound peak (efference copy added)

  KEY PRINCIPLE:
    ✅ More modalities × COHERENT sync = compound reward (potentially super-additive)
    ❌ More modalities × INCOHERENT = compound dissonance
    ⚠️ v1.1: "Multiplicative" overstated. Multisensory integration CAN BE super-additive
       (response > sum of parts) but also additive or sub-additive depending on conditions
       (eLife 2024). "Potentially super-additive" = more accurate than "multiplicative."
    = Modality.md §4: Reinforcement (same direction → STRONGER) vs Conflict (opposing directions → crash)

  Sound WITHIN compound:
    → Oscillates Tonic (background baseline) ↔ Cyclic (foreground peaks)
    → Visual + Action ADD modalities on top of sound substrate
    → Compound = sound-as-foundation + visual/action-as-layers

  Film vs Game = DIFFERENT reward types, not hierarchy:
    → Film: more Simulation-Engine (reflective, DMN) → Evaluative-dominant
    → Game: more efference copy (action, compound) → Evaluative + Direct-State + efference
    → Neither "better" — different neural profiles
```

## §1 — 3-Tier Hierarchy Model

```
🟡 3 TIERS — additive modalities, compound reward:


  ═══ TIER 1: SOUND ALONE (1-3 modalities) ═══

    Active modalities:
      ① Auditory: pitch, rhythm, harmony, timbre
      ② Motor (automatic): beat → SMA/premotor fire passively (Grahn 2007)
      ③ Emotional: valence response (amygdala, insula)

    Reward source:
      → Evaluative Reward (opioid — Body-Feedback-Precondition required)
      → Prediction match across auditory patterns → opioid
      → = 03-Sound-Reward-Pipeline §1 full walkthrough

    Reward level: BASELINE
    Examples: Spotify listening, background music, radio


  ═══ TIER 2: SOUND + VISUAL (3-4 modalities) ═══

    Active modalities:
      ① Auditory (same as Tier 1)
      ② Motor (automatic — same as Tier 1)
      ③ Emotional (same as Tier 1 — AMPLIFIED by visual)
      ④ Visual: images, movement, color, spatial patterns

    Reward source:
      → Evaluative Reward (same as Tier 1)
      → + Multi-modal BINDING reward:
        Visual pattern SYNC with auditory pattern → gamma binding (Singer 1995)
        → ADDITIONAL prediction match channels
        → Each sync = prediction confirmed = micro-opioid

    Reward level: BASELINE + BINDING BONUS
    Examples: Music video, film with score, live concert (watching performer)

    ⭐ TIER 2 BINDING MECHANISM:
      → Modality.md §2: 5 binding mechanisms simultaneous:
        Gamma oscillation 40Hz + multisensory neurons + convergence zones
        + amygdala affective tagging + DMN scaffolding
      → Music video: beat drop SYNC with visual cut → binding FIRE → reward
      → Film score: tension music + threatening visual → binding AMPLIFY emotion
      → = Prediction match ACROSS modalities = more channels = more reward


  ═══ TIER 3: SOUND + VISUAL + ACTION (4-5 modalities) ═══

    Active modalities:
      ① Auditory (same)
      ② Motor: now ACTIVE (self-action, not just passive entrainment)
      ③ Emotional (same — FURTHER amplified)
      ④ Visual (same)
      ⑤ Somatic: haptic feedback, body position, proprioception

    Reward sources:
      → Evaluative Reward (same as Tier 1-2)
      → + Multi-modal binding reward (same as Tier 2)
      → ⭐ + EFFERENCE COPY reward (§3 — NEW, unique to Tier 3)
      → ⭐ + Direct-State Reward (movement → endorphin)
      → = COMPOUND: 4 reward sources SIMULTANEOUS

    Reward level: BASELINE + BINDING + EFFERENCE + DIRECT-STATE = PEAK
    Examples: Rhythm game, dance, instrument playing, VR music game


  COMPARISON:
  ┌─────────┬──────────┬────────────┬──────────┬──────────┬──────────┐
  │ Tier    │ Modality │ Evaluative │ Binding  │Efference │ Direct-  │
  │         │ count    │ (opioid)   │ (sync)   │ (copy)   │ State    │
  ├─────────┼──────────┼────────────┼──────────┼──────────┼──────────┤
  │ 1: Sound│ 1-3      │ ✅          │ ❌        │ ❌        │ ❌        │
  │ 2:+Vis  │ 3-4      │ ✅          │ ✅        │ ❌        │ ❌        │
  │ 3:+Act  │ 4-5      │ ✅          │ ✅        │ ✅        │ ✅        │
  └─────────┴──────────┴────────────┴──────────┴──────────┴──────────┘
```

## §2 — Coherence = Condition, Not Automatic

```
⭐ MORE MODALITIES ≠ MORE REWARD. COHERENCE = GATE.

  Modality.md §4 — 3 Interaction Types:

  ① REINFORCEMENT (same direction → chunk STRENGTHENED):
    Music beat drop + visual flash + haptic pulse = ALL "PEAK NOW"
    → 3 modalities all say "PEAK!" → body: COMPOUND reward ✅
    → = Multiplicative positive

  ② CONFLICT (opposing directions → dissonance):
    Happy music + sad visual + action fail = 3 OPPOSING DIRECTIONS
    → Body: "CONTRADICTION!" → compound DISSONANCE ❌
    → = Multiplicative NEGATIVE — WORSE than single-modality dissonance

  ③ COMPENSATION (1 modality weak → others compensate):
    Music ok + visual poor + action smooth = 1 weak, 2 strong
    → Body: partial match → ATTENUATED reward (not compound)


  COHERENT SYNC = ALL modalities ON THE SAME PREDICTION PATTERN:

    ✅ COHERENT:
      Beat Saber: audio beat (120 BPM) → visual note approach (timed) → slice action (on beat)
      → 3 modalities PREDICT THE SAME moment → MATCH simultaneously → compound ✅

    ❌ INCOHERENT:
      Game with upbeat music + dying scene + frustrating controls
      → Audio: "fun!" + Visual: "dead!" + Action: "fail!" → 3 OPPOSING directions → compound ❌

    ⚠️ PARTIAL:
      Film with decent score but poor acting
      → Audio: match + Visual: partial → attenuated, not compound

  GAME DESIGN IMPLICATION:
    → "Music mismatch gameplay emotion → break immersion"
      (01-Sound-Brain-Neuroscience §3, Musical-Notation-Architecture §8.2 Q4)
    → Adaptive music (dynamic soundtrack) SYNC gameplay state → coherence maintained
    → Static music + changing gameplay → RISK incoherence

  ⭐ RULE: Compound reward = f(modality count × coherence degree).
    High modality × high coherence = PEAK.
    High modality × low coherence = WORSE than low modality.
    → Game designers intuitively know this: music must match gameplay.
```

## §3 — Efference Copy × Multi-Modal (Tier 3 Mechanism)

```
🟡 EFFERENCE COPY = TIER 3 EXCLUSIVE REWARD SOURCE:

  Autonomy-Hardware.md §1:
    → Self-action → brain generates SENSORY PREDICTION before action executes
    → Action executes → sensory feedback ARRIVES
    → Match (predicted ≈ actual) → micro-opioid → reward
    → = "Self-action = better prediction = more reward"

  APPLIED TO GAME / INSTRUMENT:

    RHYTHM GAME (Beat Saber, Osu, Guitar Hero):
      ① SEE note approaching (visual prediction: "hit at T=0.5s")
      ② HEAR beat approaching (auditory prediction: "beat at T=0.5s")
      ③ PLAN action (motor preparation: "swing at T=0.5s")
      ④ EXECUTE swing → efference copy PREDICTS sensory feedback
      ⑤ FEEDBACK: haptic vibration + visual explosion + score +1
      ⑥ Match: predicted ≈ actual → micro-opioid → "HIT!"
      → = 3 predictions (visual + auditory + motor) CONFIRMED SIMULTANEOUSLY
      → = TRIPLE prediction match → COMPOUND micro-opioid

    PLAYING INSTRUMENT:
      ① INTEND chord (motor plan)
      ② FINGERS move (efference copy predicts: "will hear Am chord")
      ③ HEAR Am chord → match → micro-opioid → "Right!"
      ④ + If playing WITH others: output MATCH band → social binding
      → = Efference copy + Evaluative (music quality) + social

    DANCING:
      ① HEAR beat → predict next beat → PLAN movement
      ② EXECUTE movement → body feedback → match beat
      ③ FEEL body move → endorphin (Direct-State)
      ④ SEE others dance in sync → social sync reward
      → = Efference + Direct-State + Evaluative + social

  ⭐ WHY TIER 3 = PEAK:
    → Tier 1: predictions in 1-3 modalities → reward FROM prediction match
    → Tier 2: predictions in 3-4 modalities → reward FROM prediction + binding
    → Tier 3: predictions in 4-5 modalities → reward FROM prediction + binding
       + ⭐ EFFERENCE COPY = ADDITIONAL prediction PER ACTION
       + ⭐ Direct-State = ADDITIONAL reward FROM movement itself
    → = Tier 3 has BOTH more prediction channels AND more reward types
    → = Multiplicative, not additive

  🟢 Dunbar 2012: active > passive for endorphin
  🟢 Grahn & Brett 2007: motor active even during passive listening
  🟡 Compound from efference × multi-modal = framework prediction
```

## §4 — Game Design = Peak Compound Territory

```
🟡 RHYTHM GAMES = DESIGNED OPTIMAL FOR MULTI-MODAL COMPOUND:

  BEAT SABER ANALYSIS (5-modality sync):

    ┌───────────┬──────────────┬─────────────────────┬──────────────────┐
    │ Modality  │ Input        │ Prediction          │ Reward on match  │
    ├───────────┼──────────────┼─────────────────────┼──────────────────┤
    │ Auditory  │ Beat/melody  │ "Next beat at 500ms"│ Evaluative       │
    │ Visual    │ Notes approach│ "Red block, left"   │ Binding (sync)   │
    │ Motor     │ Arm swing    │ Efference copy      │ Efference opioid │
    │ Somatic   │ Haptic buzz  │ "Will feel vibrate" │ Direct-State     │
    │ Emotional │ Score + combo│ "Streak continues"  │ Evaluative (meta)│
    └───────────┴──────────────┴─────────────────────┴──────────────────┘

    EVERY beat = 5 predictions confirmed simultaneously
    PERFECT streak = sustained compound → "flow state"
    BREAK streak = 5 predictions VIOLATED → compound dissonance → "ugh!"

  WHY RHYTHM GAMES ARE ADDICTIVE:
    → 5-modality sync = peak compound reward territory
    → Immediate feedback (ms precision) → tight prediction loop
    → Difficulty curve = Goldilocks maintained (Body-Feedback-Precondition Precondition-4 Match-Range)
    → Score = progress marker → approach tag reinforced
    → Social (leaderboard/multiplayer) = additional Evaluative layer
    → = MULTIPLE framework mechanisms converge → compound appeal

  NON-RHYTHM GAMES — VARIES:
    RPG + soundtrack: Tier 2 primarily (audio+visual, action less sync with music)
    FPS + adaptive music: Tier 2-3 (music ADAPTS to action → coherence maintained)
    Puzzle + ambient: Tier 1-2 (music = background, action = independent)
    → Genre determines which tier applies

  DIFFICULTY × COMPOUND:
    → Too easy: prediction match TOO certain → prediction-delta = 0 → boredom
    → Too hard: prediction miss TOO frequent → compound dissonance → frustration
    → OPTIMAL: Goldilocks → sustained compound → flow
    → = Witek 2014 groove inverted-U applied to game difficulty
    → 🟢 Csíkszentmihályi: flow = skill ≈ challenge (same principle)

  🟡 Game compound analysis = framework application
  🟢 Flow state (Csíkszentmihályi 1990): skill ≈ challenge
  🟢 Witek 2014: groove inverted-U (applied to game difficulty)
```

## §5 — Film = Different Reward Type (NOT Less)

```
🟡 FILM ≠ "LESS REWARD" — DIFFERENT REWARD PROFILE:

  FILM (Tier 2 — audio + visual, passive):

    WHAT FILM HAS:
      ✅ Evaluative Reward (music + visual aesthetics + narrative coherence)
      ✅ Multi-modal binding (score SYNC story → compound Evaluative)
      ✅ Simulation-Engine STRONG activation:
        DMN fires (Taruffi 2017) → self-referential processing
        Character simulation → (Other, Future, Construct)
        Emotional resonance via narrative → DEEP Evaluative

    WHAT FILM LACKS:
      ❌ Efference copy (passive → no self-action → no prediction-from-action)
      ❌ Direct-State from movement (sitting still)
      ❌ Agency (cannot change outcome → no autonomy reward)

    FILM REWARD PROFILE:
      → Heavy Evaluative (narrative + aesthetic + emotional)
      → Strong Simulation-Engine (DMN → reflection → meaning)
      → Weak Direct-State (body mostly static)
      → = "Deep but passive" reward

  GAME (Tier 3 — audio + visual + action, active):

    → Evaluative + Binding + Efference + Direct-State (see §1)
    → Simulation-Engine: VARIES (some games trigger DMN, others suppress)
    → = "Broad compound" reward

  COMPARISON:

    ┌──────────────────┬──────────────┬──────────────┐

    │ Direct-State     │ ☆            │ ★★★          │
    │ Simulation-Engine│ ★★★★★        │ ★★★          │
    │ DMN activation   │ ★★★★★        │ ★★ (varies)  │
    │ Reflection/depth │ ★★★★★        │ ★★           │
    │ Compound breadth │ ★★           │ ★★★★★        │
    └──────────────────┴──────────────┴──────────────┘

    Film = DEEP Evaluative + Simulation-Engine
    Game = BROAD Compound + Efference + Direct-State
    NEITHER "better" — different neural profiles

  "CRYING DURING MOVIE" = peak Evaluative:
    → Narrative coherence → body EVALUATES → DEEP opioid
    → Simulation-Engine fires: (Other, All-times, Construct+Observe)
    → = Evaluative Reward at E₂-E₃ complexity (Reward-Signal-Architecture §2)

    → 5-modality sync → compound → sustained → "lost in game"
```

## §6 — VR / Immersive = Tier 3+ (Frontier)

```
🔴 VR = EXTENDED TIER 3, MOSTLY SPECULATIVE:

  VR ADDS:
    → Full-body tracking = MORE efference copy channels
    → Spatial audio (3D) = auditory prediction RICHER
    → Haptic feedback suit = somatic modality EXPANDED
    → Head tracking = vestibular modality ADDED (6th modality territory)

  FRAMEWORK PREDICTS:
    → More modalities × coherent = EVEN MORE compound reward
    → BUT: coherence requirement STRICTER:
      Vestibular input MUST match visual → else MOTION SICKNESS
      Motion sickness = vestibular-visual INCOHERENCE → compound DISSONANCE
      = Modality §4 type ② CONFLICT at hardware level

  VR MUSIC GAMES (Beat Saber VR):
    → Already Tier 3 + spatial + head tracking
    → Some of the HIGHEST engagement metrics in gaming
    → = Framework prediction: more modalities + maintained coherence = peak

  VR RISK:
    → Uncanny valley (visual fidelity ≈ real but NOT real → dissonance)
    → Latency > 20ms → efference copy MISMATCH → nausea
    → = Coherence tolerance TIGHTER in VR than flat screen
    → = "More modalities = more to go wrong"

  🔴 VR territory = mostly speculative
  🟢 Motion sickness = vestibular-visual conflict = established
  🟡 "More modalities × coherence" principle = framework prediction
```

## §7 — Sound as Background: User Hypothesis Refined

```
🟡 USER HYPOTHESIS + FRAMEWORK REFINEMENT:

  ORIGINAL HYPOTHESIS:
    "Sound runs in the background, visual + action activate supplementary channels,
    when playing together → synchronization creates coherence across multiple modalities
    → higher pattern quality → stronger body-reward."

  FRAMEWORK ASSESSMENT: CORRECT in principle — needs 3 refinements:


  REFINEMENT 1 — Sound OSCILLATES Tonic ↔ Cyclic:

    Sound is NOT just "background" (pure Tonic):
      → Verse/build sections = Tonic channel (background, continuous, invisible)
      → Drop/peak/chorus = Cyclic channel (foreground, VTA FIRE, attention shift)
      → = Sound oscillates Tonic ↔ Cyclic WITHIN 1 session

    02-Sound-Background-Pattern §5:
      → Tonic = "music always present" → Background-Pattern substrate
      → Cyclic = "music peak" → prediction-delta → VTA fire → attention

    IN GAME CONTEXT:
      → Background music = Tonic (baseline, sets mood)
      → Combat music / boss theme = Cyclic (peak, sync with action peak)
      → Victory jingle = Cyclic (reward confirmation)
      → = DESIGNED oscillation for optimal engagement


  REFINEMENT 2 — Visual + Action ≠ just "supplementary":

    Visual + Action = INDEPENDENT prediction channels, not supplements:
      → Each modality generates ITS OWN predictions
      → Each match = ITS OWN reward source
      → Combined = COMPOUND (potentially super-additive), not merely additive

    Visual predictions:
      → "Block approaching from left" → predict position → match → reward
    Action predictions:
      → "Swing arm to left" → efference copy → match feedback → reward
    Sound predictions:
      → "Beat at T+500ms" → match → reward

    All 3 INDEPENDENT → but when SYNCHRONIZED → compound > sum


  REFINEMENT 3 — Coherence = NECESSARY condition:

    User implied: "playing together → synchronization → reward"
    Framework adds: "→ IF coherent"
    → Incoherent multi-modal = WORSE than single-modal
    → Coherence = game design responsibility (not automatic)


  ⭐ REFINED HYPOTHESIS:

    Sound provides DUAL function:
      Tonic = emotional SUBSTRATE (mood, baseline, invisible)
      Cyclic = temporal ANCHOR (peaks, drops, sync points)

    Visual provides:
      SPATIAL prediction channels → binding with audio temporal

    Action provides:
      EFFERENCE COPY = additional prediction → additional reward
      + DIRECT-STATE from movement

    ALL 3 × COHERENT SYNC:
      = COMPOUND reward (potentially super-additive)
      = Peak engagement territory (flow state)

    CONDITION:
      Coherence across modalities = REQUIRED
      Incoherence = compound DISSONANCE (worse than single-modal)

  🟡 User hypothesis refined = framework application
```
