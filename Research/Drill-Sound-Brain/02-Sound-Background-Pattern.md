## §1 — Music Chunks = Inherently Multi-Modal

```
🟡 MUSIC CHUNKS ENCODE ACROSS 3+ MODALITIES SIMULTANEOUSLY:

  When listening to a piece of music, THE BRAIN FIRES:

    AUDITORY (primary):
      → Pitch patterns, chord progressions, melody contour
      → Auditory cortex (Heschl's gyrus, superior temporal gyrus)
      → = Modality ① (Modality.md §1)

    MOTOR (automatic — Grahn & Brett 2007 🟢):
      → Rhythm → SMA + premotor + cerebellum + basal ganglia fire
      → ⭐ Active EVEN WHEN NOT MOVING — beat perception = fundamentally motor
      → = Modality ④ (Modality.md §1)
      → 01-Sound-Brain-Neuroscience §6

    EMOTIONAL (valence tagging):
      → Amygdala + insula + limbic fire → emotional response
      → Happy/sad/tense/resolved — valence tag at compile time
      → = Modality ⑤ (Modality.md §1)

    SOMATIC (conditional — when body responds):
      → Chills, goosebumps, tears, body sway
      → Interoception detects body state change (Trost 2017)
      → = Modality ③ (Modality.md §1)

  MODALITY COUNT:
    Passive listening minimum: auditory + motor + emotional = 3 modalities
    Active engagement: + somatic + visual (if present) = 4-5 modalities
    Active music-making: + motor output (efference copy) = 5-6 modalities

  COMPARE:
    Reading text (verbal only) = 1 modality → SHALLOW chunk
    Listening to a lecture = 1-2 modalities (auditory + verbal) → MODERATE chunk
    Listening to music = 3+ modalities → DEEP chunk
    Playing music = 5-6 modalities → EXTREMELY DEEP chunk

  ⭐ INSIGHT 1: Music chunks = INHERENTLY DEEP (3+ modalities minimum).
    Per Modality.md §3: chunk depth = modality count.
    Music chunks are NATURALLY deeper than verbal-only chunks — no extra effort needed.
    = Why music memories are more durable than verbal memories.

  🟢 Grahn & Brett 2007: motor activation during passive listening
  🟢 Trost 2017: rhythmic entrainment → interoception (insula)
  🟡 Music chunks = inherently multi-modal = framework application
```

```
      → Content diverse (artists constantly changing)

    T=10 years: Pattern = BACKGROUND → PFC habituates → invisible
    T=15 years: Background-Pattern COMPLETE → biases all processing

  ⭐ INSIGHT 2: Genre = sustained context consistency → ideal substrate
    for Background-Pattern formation. Diverse content WITHIN genre

    DIFFERENT genre = DIFFERENT context → DIFFERENT Background-Pattern.

  🟡 Genre as context consistency = framework application
```

## §4 — 2D Model: Low Depth + HIGH Density (Alzheimer Validated)

```
⭐ MUSIC BACKGROUND-PATTERN OCCUPIES THE "INVISIBLE BUT PERVASIVE" QUADRANT:

  Background-Pattern.md §2.2 — 4 Quadrants:

                    Link Density
                    LOW              HIGH
                 ┌────────────────┬──────────────────┐
    Compile HIGH │ ISOLATED EVENT │ CHRONIC DEEP     │
    Depth        │                │                  │
                 ├────────────────┼──────────────────┤
            LOW  │ SURFACE RULE   │ ★ MUSIC          │
                 │                │ BACKGROUND       │
                 │                │ PATTERN          │
                 └────────────────┴──────────────────┘

  WHY Low Depth + HIGH Density:

    LOW DEPTH:
      → Each individual piece of music: AVERAGE emotional weight
      → Not a trauma event (no high emotional peak per-chunk)
      → Each music chunk = moderate compile depth

    HIGH DENSITY:
      → Thousands of pieces × years × daily exposure
      → Each chunk links into the same genre context
      → Sleep gist extraction + consolidation × thousands of nights
      → = THOUSANDS of inter-linked chunks distributed across the neocortex

  ⭐ ALZHEIMER — MUSICAL MEMORY PRESERVED (with mechanism nuance v1.2):

    🟢 Alzheimer destroys chunk substrate → reveals the pattern:
      → Recent memories lost FIRST
      → Musical memories lost LAST (among the last to degrade)

    JACOBSEN 2015 MECHANISM (Brain):
      → Musical memory regions identified: caudal anterior cingulate (cACC),
        ventral pre-supplementary motor area (ventral pre-SMA),
        anterior temporal cortex, frontal polar cortex, insular cortex
      → These regions show MINIMAL cortical atrophy + preserved glucose-metabolism
      → = ANATOMICAL SPARING: regions happen to be LESS AFFECTED by AD
      → ⚠️ NOT necessarily "distributed = resistant" — specific regions happen to be spared

    FRAMEWORK INTERPRETATION (🟡 — consistent but not what Jacobsen found):
      → Music chunks = distributed across MANY brain regions (multi-modal, §1)
      → Music chunks = HIGH link density (thousands of inter-linked)
      → Multi-modal → more brain regions → more chances some survive atrophy
      → = 2D model "Low Depth + HIGH Density = RESISTANT" is PLAUSIBLE
      → BUT Jacobsen's actual finding = regional vulnerability patterns, not link density

    ⚠️ TERMINOLOGY NOTE: §1 says "chunks INHERENTLY DEEP (3+ modalities)"
       vs §4 here says "LOW Depth." Different "depth":
       §1 depth = modality count (richness of encoding)
       §4 depth = COMPILE depth (emotional weight per individual event)
       Music = HIGH modality-depth + LOW emotional-compile-depth per song
       = Rich encoding per chunk BUT no single chunk is traumatic/life-changing

  ⭐ INSIGHT 4: Music Background-Pattern = lower-right quadrant (INVISIBLE BUT PERVASIVE).
    Musical memory preservation in AD = CONFIRMED by clinical data.
    Mechanism: Jacobsen 2015 = anatomical sparing of musical memory regions.
    Framework adds: multi-modal + high link density may CONTRIBUTE to resistance.
    Both explanations compatible — not competing.

  🟢 Jacobsen et al. 2015 (Brain): musical memory regions spared in AD
  🟢 Alzheimer music preservation: multiple clinical sources
  🟡 2D model (Low compile-depth + HIGH link density) = framework interpretation
  ⚠️ v1.2: Mechanism nuance added. "Depth" terminology clarified.
```

## §5 — 3 Satiation Types × Music Listening

```
🟡 MUSIC LISTENING ENCOMPASSES ALL 3 SATIATION TYPES (Gap-Body-Need v1.0 §2):

  ① CYCLIC CHANNELS — "Listen to a single track"
    → Listen → satisfied → stop → want to listen again LATER
    → Oscillating: on/off/on
    → Background-Pattern contribution: MODERATE (fires then stops)
    → Example: listen to favorite song → "that's enough" → hours later → want to hear it again
    → Easy to redirect to an alternative (different song, different genre)

  ② TONIC CHANNELS — "Always have background music"
    → Continuous: always play music while working, driving, cooking
    → VTA habituated → PFC INVISIBLE (Background-Pattern §5 reason ①)
    → Background-Pattern contribution: HIGHEST
    → ⭐ TONIC = CORE SUBSTRATE for music Background-Pattern

    Why Tonic = STRONGEST:
      → Continuous firing → link density grows CONTINUOUSLY (no break)
      → VTA habituates → pattern INVISIBLE → PFC stops inquiry
      → = Background-Pattern §9.2: "Tonic becomes background" exactly
      → When music stops → body: "something is MISSING" → Firing-Chunk-Miss
      → "Feel empty without music" = Background-Pattern EXPOSED

  ③ GENERATIVE CHANNELS — "Exploring new music"
    → Novelty-dependent: seeking new songs, new artists, new sub-genres
    → Self-renewing: fill → new gap → fill → new gap
    → Background-Pattern contribution: CONDITIONAL (only while novelty continues)
    → If novelty stops (genre exhausted) → Generative DIES → boredom

    "Bored with this genre" = Generative channels EXHAUSTED:
      → All patterns in genre already predicted → prediction-delta = 0
      → Boredom formula (Boredom.md §1): dissonance + Imagine-Final dim
      → Fix: inject novelty (new sub-genre) or shift genre entirely

  PER-LISTENER PROFILE:
    → Tonic-dominant listener: Background-Pattern STRONG + INVISIBLE + stable
    → Cyclic-dominant listener: Background-Pattern MODERATE + oscillating
    → Generative-dominant listener: Background-Pattern CONDITIONAL + novelty-dependent
    → Every listener = MIX of 3 types, differing only in ratio

  ⭐ INSIGHT 5 + 6:
    Insight 5: 3 satiation types apply to music → per-listener profile
    Insight 6: Tonic listeners (always have background music) = STRONGEST music Background-Pattern
    because continuous firing → PFC invisible → link density EXTREMELY HIGH

  🟡 3 Satiation Types × music = framework application
```
