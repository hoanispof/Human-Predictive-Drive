## §1 — ADHD: REFRAME

### §1.1 — "Attention Deficit" = Misnomer

```
TÊN SAI — "Attention Deficit Hyperactivity Disorder":

  "Deficit" → KHÔNG thiếu attention
    → CÓ attention, nhưng REGULATE khác
    → Bằng chứng: hyperfocus 6-12 giờ liên tục = KHÔNG "thiếu"

  "Hyperactivity" → KHÔNG phải đặc điểm universal
    → Chỉ 1/3 presentations (Combined type)
    → Inattentive type: KHÔNG hyperactive observable
    → Hyperactivity GIẢM DẦN theo tuổi

  Framework reframe:
    ADHD = ATTENTION REGULATION TUNING ở extreme trên spectrum liên tục
    = Threshold system calibrated khác → filter input khác → behavior khác
```

### §1.2 — DSM-5: 3 Presentations

```
① Predominantly Inattentive (ADHD-PI):
  → Khó sustain attention cho tasks "nhỏ" (threshold không vượt)
  → Mind wandering (DMN interference)
  → "Dreamy", "in their own world"
  → THƯỜNG bị miss diagnosis (đặc biệt ở nữ)

② Predominantly Hyperactive-Impulsive (ADHD-HI):
  → Body output TRƯỚC PFC process (inhibition under-fueled)
  → Fidgeting, restlessness (body-feedback seeking)
  → Impulsive decisions (PFC ⑱ Inhibit weak)
  → ÍT phổ biến nhất standalone

③ Combined (ADHD-C):
  → CẢ inattention + hyperactivity-impulsivity
  → PHỔ BIẾN NHẤT (~50-75% cases)
  → MOST visible → most diagnosed
```

### §1.3 — Prevalence

```
Trẻ em: ~5-7% toàn cầu (Polanczyk et al. 2007 meta-analysis, 🟢)
Người lớn: ~2.5-4% (Faraone et al. 2006, 🟢)

KHÔNG "hết" khi trưởng thành:
  → ~50% persistence rate (Faraone et al. 2006 meta-analysis)
  → ~70% vẫn impaired dù không đủ full criteria (residual symptoms)
  → Hyperactivity giảm > Inattention giảm
  → Nhiều người KHÔNG BIẾT mình có ADHD cho tới trưởng thành

Giới tính:
  → Trẻ em: nam:nữ = 3:1 (diagnosis bias — nam visible hơn)
  → Người lớn: nam:nữ ≈ 1.5:1 (nữ under-diagnosed hồi nhỏ, diagnosed muộn)
  → Nữ thường Inattentive type → ít disruptive → bị miss
```

### §1.4 — Framework framing tổng quát

```
ADHD qua lens v7.8:

  Hardware: Neurons INTACT (khác Parkinson — neurons CHẾT)
  Software: Cycle prediction-action-feedback HOẠT ĐỘNG
  Tuning: Parameters ở EXTREME trên spectrum liên tục:
    → DAT density ↑ (dopamine clear nhanh)
    → COMT Val/Val tendency (PFC workspace clear nhanh)
    → DRD4-7R tendency (chunk threshold cao)
    → NE sensitivity pattern khác

  = CÙNG KIẾN TRÚC, TUNING KHÁC
  = Không "hỏng" — calibrated cho MÔI TRƯỜNG KHÁC
    (environment cần constant novelty, high-stakes, immediate feedback)
```

## §2 — NEUROCHEMISTRY: DAT + DOPAMINE CLEARANCE

### §2.1 — DAT: Dopamine Transporter

```
DAT (Dopamine Transporter) = protein BƠM NGƯỢC dopamine:

  Neuron fire dopamine vào synapse (khe giữa 2 neuron)
  → Dopamine ở synapse = SIGNAL (post-synaptic neuron nhận)
  → DAT bơm dopamine NGƯỢC VỀ neuron gốc = KẾT THÚC signal

  ADHD: DAT DENSITY CAO hơn bình thường
    → Bơm ngược NHANH hơn
    → Dopamine ở synapse NGẮN hơn
    → Signal NGẮN hơn
    → Post-synaptic neuron nhận ÍT tín hiệu hơn

  🟢 Spencer et al. 2007 (PET imaging, altropane):
    "Significantly increased DAT binding in right caudate
     in adults with ADHD vs matched controls"
    → TRỰC TIẾP đo được: ADHD = nhiều DAT hơn = clear nhanh hơn
```

### §2.2 — Tại sao CLEARANCE nhanh ≠ Thiếu dopamine

```
PHÂN BIỆT QUAN TRỌNG:

  ❌ "ADHD = thiếu dopamine" (pop misconception)
  ✅ "ADHD = dopamine signal NGẮN" (mechanism đúng)

  Neuron SẢN XUẤT dopamine BÌNH THƯỜNG
  Neuron FIRE dopamine BÌNH THƯỜNG
  Nhưng DAT DỌN quá nhanh → signal kết thúc sớm

  TƯƠNG TỰ: Không phải "ít nước" — mà "ống thoát nước QUÁ RỘNG"
  Nước (dopamine) vẫn đổ vào bình thường
  Nhưng thoát đi quá nhanh → không đủ thời gian thấm
```

### §2.3 — PFC đặc biệt chịu ảnh hưởng

```
TẠI SAO PFC bị ảnh hưởng NHIỀU NHẤT:

  ① DAT density cao ở striatum → signal motor ngắn
  ② COMT enzyme CHỈ CÓ Ở PFC → clear thêm tại PFC
     (PFC-Hardware.md §2: COMT = PFC-specific clearance)
  ③ Double clearance: DAT + COMT → PFC dopamine signal NGẮN NHẤT

  Hệ quả:
    → PFC = vùng não cần dopamine LIÊN TỤC để sustain function
    → Signal ngắn → PFC "tắt" nhanh → attention drops
    → Signal ngắn → PFC working memory "clear" sớm → quên mid-task

  🟢 Arnsten 2009, 2011 (Yale):
    "PFC is especially sensitive to its neurochemical environment;
     relatively small changes in norepinephrine and dopamine
     can produce significant changes in its function"
    → PFC = CRITICAL target — nhỏ thay đổi catecholamine = lớn thay đổi function
```

### §2.4 — NE (Norepinephrine) song song

```
ADHD KHÔNG CHỈ dopamine — NE cũng bị ảnh hưởng:

  NE tại PFC (Arnsten 2009):
    → Moderate NE + α2A receptor = ENHANCE PFC function (signal clarity ↑)
    → High NE + α1 receptor = DISCONNECT PFC (circuit breaker)
    → INVERTED-U dose-response

  ADHD: NE regulation CŨNG khác
    → Baseline NE có thể sub-optimal cho α2A engagement
    → PFC networks kém connected hơn
    → = Double hit: ÍT dopamine sustain + ÍT NE enhance

  🟢 Arnsten 2009:
    "Blockade of α2 receptors in monkey PFC recreates the symptoms of ADHD:
     impaired working memory, increased impulsivity, locomotor hyperactivity"
    → TRỰC TIẾP: block NE enhance pathway = ADHD symptoms
```

### §2.5 — Đặt trong spectrum (Attention-Spectrum.md link)

```
Attention-Spectrum.md 4-factor model (reference):

  Factor 1: COMT clear speed
    → Val/Val = fast clear → IMPROVISER (drift toward ADHD extreme)
    → Met/Met = slow clear → SPECIALIST

  Factor 2: DRD4 chunk threshold
    → 7R = HIGH threshold → only BIG chunks detect → "bored easily" + hyperfocus
    → 4R = LOW threshold → many small chunks detect → detail-oriented

  Factor 3: NE sensitivity (α2/α1 balance)
    → α2 dominant = PFC ENHANCE (focused, calm)
    → α1 dominant = PFC DISCONNECT (overwhelmed, reactive)

  Factor 4: Cortisol state (moment-to-moment)
    → Low cortisol = PFC clean workspace
    → High cortisol = PFC noisy, impaired

  ADHD = EXTREME COMBINATION:
    Val/Val (fast COMT) + 7R (high threshold) + α2 sub-optimal + DAT high
    = "Perfect storm" cho attention regulation difficulty
    = SPECTRUM, không phải binary ON/OFF
```

### §3.2 — Key distinctions

```

  Nicotine: ACQUIRED (bắt đầu khi first use, escalate with tolerance)
  Parkinson: ACQUIRED (onset 50+, progress over decades)
  ADHD: DEVELOPMENTAL (from birth, STABLE — compensation changes, not condition)

④ IDENTITY:
  Nicotine: "Tôi nghiện thuốc" = identity chunk CÓ THỂ re-compile
  Parkinson: "Tôi bị bệnh" = progressive loss identity
  ADHD: "Tôi là người ADHD" = identity trait (reframe, not remove)
```

## §4 — EXECUTIVE FUNCTION BEYOND ATTENTION

### §4.1 — Barkley's Model (1997)

```
🟢 Barkley 1997 (Psychological Bulletin):
  "Behavioral inhibition, sustained attention, and executive functions:
   constructing a unifying theory of ADHD"

  Core claim: BEHAVIORAL INHIBITION = primary deficit
  → Inhibition impaired → 4 executive functions impaired:

  ┌─────────────────────────────────────────────────────────────┐
  │           BEHAVIORAL INHIBITION (primary)                    │
  │                        ↓                                     │
  │  ┌──────────────┬──────────────┬──────────────┬──────────┐  │
  │  │ Working      │ Self-Reg     │ Internalized │ Reconsti-│  │
  │  │ Memory       │ Affect/      │ Speech       │ tution   │  │
  │  │ (nonverbal)  │ Motivation/  │              │ (analysis│  │
  │  │              │ Arousal      │              │ +synth.) │  │
  │  └──────────────┴──────────────┴──────────────┴──────────┘  │
  │                        ↓                                     │
  │              MOTOR CONTROL/OUTPUT                             │
  └─────────────────────────────────────────────────────────────┘

  Framework mapping:
    Behavioral inhibition = PFC Function ⑱ Inhibit (vlPFC)
    Working memory = PFC §2 HOLD (4±1 dimensions)
    Self-reg affect = PFC Function ⑧ Emotional regulation
    Internalized speech = PFC metacognition ④
    Reconstitution = PFC analysis + creativity functions
```

### §4.2 — Working Memory (PFC HOLD)

```
PFC-Hold-Dimensions.md: ~4±1 dimensions simultaneously

  ADHD impact:
    → Dopamine signal ngắn → HOLD time ngắn hơn
    → "Hold 4 items" → effectively 2-3 under sub-optimal dopamine
    → Mid-task: "vừa muốn gì nhỉ?" = working memory DROPPED
    → NOT hardware limit (phải chăng neurons still capable)
    → = FUEL limit (dopamine clears → hold dissolves)

  Observable:
    → Quên mid-sentence
    → Start task → forget original goal
    → Need external reminders (lists, alarms, notes)
    → NHƯNG: under hyperfocus, working memory FINE (dopamine sustained)
```

### §4.3 — Inhibition (PFC ⑱)

```
PFC Function ⑱ Inhibit = STOP signal before action:
  → PFC evaluates impulse → SENDS STOP signal → prevent motor output

  ADHD: Inhibit function UNDER-FUELED
    → Impulse → PFC SHOULD stop → nhưng dopamine signal ngắn → stop signal WEAK
    → Body acts BEFORE PFC finishes evaluation
    → = "Impulsive" behavior

  KHÔNG phải "thiếu self-control" (morality framing):
    → = vlPFC literally KHÔNG CÓ ĐỦ FUEL để fire inhibit signal đúng lúc
    → Latency: inhibit signal arrives 50-100ms SLOWER (measurable via Stop-Signal Task)
    → NOT "choosing not to stop" → "stop signal arrives TOO LATE"
```

### §4.4 — Planning + Sequencing

```
Multi-step tasks require:
  ① HOLD full sequence in working memory
  ② INHIBIT premature execution of later steps
  ③ MONITOR progress (track which step current)
  ④ SWITCH attention between steps

  ADHD: ALL 4 under-fueled simultaneously:
    → ① Sequence drops from working memory mid-way
    → ② Later steps execute before earlier ones complete
    → ③ Lose track of progress
    → ④ Switch TO distractors easy, switch BACK difficult

  Framework: PFC-Configuration.md Mode ① Normal
    → ADHD Mode ① = UNSTABLE
    → Drifts toward Config ② (if interest high) hoặc DMN (if interest low)
    → CANNOT maintain Mode ① steadily for mundane tasks
```

### §4.5 — Task Switching: HYPER vs HYPO

```
PARADOX:
  → CAN'T sustain attention on one task (HYPO sustained)
  → CAN'T stop switching to new stimuli (HYPER switching)
  → BUT: once in hyperfocus, CAN'T switch AWAY (HYPO switching out)

  Framework explanation:
    Task switching = dopamine-dependent PFC function
    → Small tasks: dopamine signal insufficient → PFC DOESN'T LOCK ON → jumps
    → Novel stimulus: dopamine spike (VTA delta) → PFC locks MOMENTARILY → jumps again
    → Big task (gap-direction matched): dopamine SUSTAINED → PFC LOCKS → can't release

  = SAME mechanism (dopamine-gated attention locking)
    Different TRIGGERS produce opposite-looking behaviors
```

### §5.1 — Not "Immaturity" — Hardware Regulation Difference

```

  "Emotion dysregulation in attention deficit hyperactivity disorder"

  Key findings:
```

### §5.2 — RSD: Rejection Sensitive Dysphoria

```

    ① Social input → prediction about others' evaluation (SPM F2 logic)
```

## §8 — DMN INTERFERENCE HYPOTHESIS

### §8.1 — Default Mode Network vs Task-Positive Network

```
🟢 Sonuga-Barke & Castellanos 2007:
  "Default Mode Interference Hypothesis"

  Normal brain:
    → DMN (Default Mode Network): active during rest, mind-wandering, self-reflection
    → TPN (Task-Positive Network): active during focused external tasks
    → DMN ↔ TPN: ANTICORRELATED (one ON = other OFF, clean switch)

  ADHD brain:
    → DMN-TPN anticorrelation REDUCED
    → DMN "leaks" into task mode → attention lapses
    → TPN cannot fully suppress DMN → mind wandering during tasks

  🟢 Castellanos et al. 2008 (Biological Psychiatry):
    Children with ADHD: "less likely to enter or maintain brain states
    characterized by anticorrelations between DMN and task-relevant networks"
```

### §8.2 — Framework interpretation

```
🟡 Framework synthesis:

  PFC-Configuration.md 6 modes:
    → Mode ① Normal: DMN suppressed, TPN active (external focus)
    → Mode ② Flow: dlPFC split — task ON, self OFF (deep focus)
    → Rest/Default: DMN active, TPN suppressed (internal processing)

  ADHD: Mode ① UNSTABLE
    → PFC dopamine too short to MAINTAIN mode ① suppression of DMN
    → DMN "bleeds through" periodically (micro-lapses)
    → Observable: "spacing out mid-sentence", "eyes glazing"

  WHY dopamine relates to mode-switching:
    → PFC Configuration requires SUSTAINED catecholamine state
    → Dopamine signal short → PFC configuration DRIFTS
    → DMN has LOWER catecholamine requirement (default state = low-energy)
    → = PFC naturally DRIFTS toward DMN when fuel runs out

  Methylphenidate CONNECTION:
    → Block DAT → dopamine stays longer → PFC can maintain Mode ① longer
    → = DMN-TPN anticorrelation IMPROVES on medication
    → 🟢 Liddle et al. 2011: methylphenidate modulates DMN in ADHD
```

## §10 — MEDICATION MECHANISM QUA FRAMEWORK

### §10.1 — Stimulants: Methylphenidate

```
🟢 Volkow et al. 2001, 2012 (Brookhaven/NIDA, PET imaging):

  Mechanism:
    → Methylphenidate (Ritalin, Concerta) BLOCKS DAT
    → DAT blocked → dopamine STAYS in synapse longer
    → Signal duration ↑ → PFC receives sustained signal
    → >50% DAT occupancy at therapeutic dose

  Framework interpretation:
    → NOT "adding dopamine" → "SLOWING REMOVAL"
    → Normal production + blocked removal = LONGER SIGNAL
    → = Threshold EASIER to exceed (signal accumulates)
    → PFC stays fueled → executive functions ONLINE longer
    → Attention-Spectrum.md §5: "giữ dopamine lại"

  Paradox explained:
    → STIMULANT calms ADHD person:
    → PFC UNDER-FUELED → can't regulate behavior → restless, impulsive
    → Add fuel (block DAT) → PFC CAN regulate → calmer, focused
    → = "Stimulant" stimulates PFC specifically → PFC regulates everything else DOWN

  Signal amplification model (Volkow 2012):
    → Methylphenidate AMPLIFIES existing dopamine signals
    → Does NOT create artificial signals (khác nicotine which FORCES fire)
    → = Context-dependent: task-related dopamine amplified → task attention improved
    → = Methylphenidate WITHOUT task = less effective (cần task để có signal to amplify)
```

### §10.2 — Stimulants: Amphetamine

```
🟢 Mechanism (distinct from methylphenidate):

  Amphetamine (Adderall, Vyvanse):
    ① BLOCKS DAT (same as methylphenidate)
    ② REVERSES DAT direction → PUSHES dopamine OUT of neuron into synapse
    ③ Also reverses VMAT2 → releases dopamine from vesicles inside neuron
    → = Double mechanism: block removal + actively release MORE

  Framework comparison:
    → Methylphenidate: "đóng cống thoát" → nước (dopamine) tự đọng lại
    → Amphetamine: "đóng cống thoát" + "bơm thêm nước vào" → nhiều hơn

  Clinical implications:
    → Amphetamine = STRONGER effect (more dopamine available)
    → Also MORE side effects (heart rate, appetite, sleep)
    → Also HIGHER abuse potential (can produce euphoria at high doses)
    → Vyvanse = prodrug (lisdexamfetamine → d-amphetamine slowly in body)
      = Slower onset → less "rush" → lower abuse potential

  Framework note:
    → Amphetamine mechanism CLOSER to "forced fire" than methylphenidate
    → At therapeutic doses: PFC fuel → regulation → calm
    → At abuse doses: mesolimbic flood → reward → hijack risk
    → = Dose determines framework category (PFC fuel vs reward hijack)
```

### §10.3 — Non-stimulants: Atomoxetine

```
🟢 Mechanism:

  Atomoxetine (Strattera):
    → Selective NET (norepinephrine transporter) inhibitor
    → Blocks NE reuptake → NE stays in synapse longer → α2A activation
    → AT PFC SPECIFICALLY: NET also clears dopamine (PFC has minimal DAT)
    → = Blocking NET at PFC → BOTH NE AND DA stay longer at PFC

  🟢 Bymaster et al. 2002 (Neuropsychopharmacology):
    "Atomoxetine increases extracellular levels of norepinephrine
     AND dopamine in prefrontal cortex"
    → NE: +300% in PFC (without affecting NAcc)
    → DA: increases selectively in PFC (not striatum)

  Framework:
    → Arnsten model: NE via α2A = ENHANCE PFC network connectivity
    → + DA increase at PFC = FUEL for executive functions
    → BUT: no effect on mesolimbic pathway → NO euphoria, NO abuse potential
    → = SELECTIVE PFC enhancement without reward system stimulation

  Trade-offs:
    → Slower onset (2-4 weeks vs hours for stimulants)
    → Less effective overall (~40-60% response vs ~70-80% stimulants)
    → No abuse potential → better for: anxiety comorbid, tic comorbid, substance history
```

### §10.4 — Non-stimulants: Guanfacine

```
🟢 Mechanism:

  Guanfacine (Intuniv):
    → Selective α2A-adrenoceptor AGONIST (mimic NE at α2A specifically)
    → Directly strengthens PFC network connectivity
    → Opens potassium channels → inhibits cAMP → enhances neural efficiency

  Framework:
    → Arnsten: α2A stimulation = "enhance signals in PFC networks"
    → Guanfacine BYPASSES NE production entirely → directly activate target
    → = Not "more fuel" but "tune receiver to work better with existing fuel"

  Clinical niche:
    → Often combined with stimulants (additive: fuel + tuning)
    → Helps specifically with: emotional regulation, tics, sleep
    → = Addresses NE component that stimulants may not fully cover
```

### §10.5 — Medication ≠ Cure: Framework perspective

```
QUAN TRỌNG:

  Medication COMPENSATES, không THAY ĐỔI hardware tuning:
    → DAT density = genetic (SLC6A3 gene)
    → COMT activity = genetic (Val158Met polymorphism)
    → DRD4 sensitivity = genetic
    → Medication active → symptoms managed
    → Medication stops → symptoms return

  Framework parallel:
    → Levodopa cho Parkinson: replace missing dopamine (source dead)
    → Stimulants cho ADHD: slow clearance of existing dopamine (source alive)
    → KHÁC: Parkinson = progressive (medication less effective over time)
    → ADHD = STABLE (medication consistently effective — no progression)

  Environment FIT vẫn critical:
    → Medication + wrong environment = limited benefit
    → Right environment (novelty, autonomy, immediate feedback)
      + medication = compound benefit
    → Some ADHD adults: right environment → medication unnecessary
    → = Environment IS part of "treatment" (reduce demands on regulation)
```
