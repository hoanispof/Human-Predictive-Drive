## §1 — AUTISM: FRAMING + HETEROGENEITY

### §1.1 — Spectrum, không phải binary

```
🟢 DSM-5 ASD (APA 2013):

  2 DOMAINS, 3 LEVELS:

  Domain A — Social Communication + Social Interaction:
    A1: Deficits in social-emotional reciprocity
    A2: Deficits in nonverbal communicative behaviors
    A3: Deficits in developing, maintaining, and understanding relationships
    → TẤT CẢ 3 criteria required

  Domain B — Restricted, Repetitive Behaviors (RRBs):
    B1: Stereotyped/repetitive motor movements, use of objects, or speech
    B2: Insistence on sameness, inflexible adherence to routines
    B3: Highly restricted, fixated interests
    B4: Hyper- or hypo-reactivity to sensory input ← MỚI trong DSM-5
    → ÍT NHẤT 2/4 criteria required

  3 Levels (support needs):
    Level 1: "Requiring support"
    Level 2: "Requiring substantial support"
    Level 3: "Requiring very substantial support"

  ⚠️ "Level" ≠ "severity" ≠ "functioning"
  → Level phản ánh SUPPORT NEEDS, không phải intelligence hay capability
  → Cùng 1 người có thể Level 1 ở Domain B, Level 2 ở Domain A
```

### §1.5 — Brain: IDIOSYNCRATIC connectivity

```
🟢 ORIGINAL model — underconnectivity:
  → Just et al. 2004 (Brain 127(8):1811-1821):
    Functional connectivity between cortical areas consistently LOWER in ASD
    Proposed frontal-posterior underconnectivity

🟡 REFINED — local over + long-range under:
  → Courchesne & Pierce 2005 (Current Opinion Neurobiology 15(2)):
    Frontal cortex EXCESSIVELY connected locally
    POORLY synchronized with distant regions

🟢 CURRENT — IDIOSYNCRATIC patterns:
  → Hahamy et al. 2015 (Nature Neuroscience 18(2):302-309):
    ASD brains show IDIOSYNCRATIC distortions of connectivity
    Increased INTER-INDIVIDUAL VARIABILITY is the core feature
    Previous over/under findings = this heterogeneity mischaracterized

  → Current consensus:
    Simple "long-range under + local over" = OUTDATED
    Actual finding = EACH autistic brain = unique connectivity pattern
    = CONSISTENT với framework: mỗi autistic person = unique configuration

🟡 Early brain overgrowth:
  → Courchesne et al. 2001, 2003 (JAMA 290(3)):
    Brain enlargement 2-4 yo. Sudden increase 1-14 months.
    Cerebral white matter +18%, cerebellar +39%
  → Current: real nhưng MORE NUANCED and SUBGROUP-SPECIFIC
    Overgrowth ≠ universal to all autistic individuals

🟡 Amygdala:
  → Faster growth 6-24 months (Cohen's d = 0.56 by 12 months)
  → Right amygdala significantly larger in ASD (Hedges' g = 0.44)
  → Significant HETEROGENEITY across studies
```

### §2.3 — Reduced HABITUATION = key mechanism

```

  Finding: ASD+SOR group showed DECREASED NEURAL HABITUATION

  WHAT THIS MEANS:

    → Tiếng máy lạnh: first 10 seconds = notice. 5 minutes = gone.
    → Autistic brain (with SOR): response DOES NOT DECREASE
    → Tiếng máy lạnh: 5 minutes = STILL firing. 30 minutes = STILL FIRING.

  FRAMEWORK MAPPING:

    → Typical: sensory input → body-feedback → HABITUATE → signal fades
    → ASD+SOR: sensory input → body-feedback → NO HABITUATION → ACCUMULATES

    → Over time: body-feedback OVERLOAD → PFC capacity exhausted → meltdown/shutdown

## §3 — MULTISENSORY INTEGRATION

### §3.1 — Temporal Binding Window: gấp đôi

```
🟢 Foss-Feig et al. 2010 (Experimental Brain Research 203(2):381-389):

  FLASH-BEEP ILLUSION paradigm:
    → Ánh sáng flash + beep sound presented at varying time intervals
    → Brain "binds" chúng thành 1 event nếu within temporal window

  Result:
    Typical development: TBW ≈ 300ms
    Autism: TBW ≈ 600ms

    = GẤP ĐÔI temporal window

  WHAT THIS MEANS:
    → ASD brain treats 2 separate events as "together" even when
      separated by up to 600ms (instead of 300ms)
    → OR: ASD brain FAILS to separate events that occur within 600ms
    → = More events get "bound" together → more cross-modal interference
    → = Harder to parse rapid sequences (like speech + lip movement)
```

### §3.2 — Speech-specific deficit

```
🟢 Stevenson et al. 2014 (Journal of Neuroscience 34(3):691-697):

  KEY FINDING: TBW enlargement KHÔNG uniform across all stimuli:
    → Simple stimuli (flash-beep): moderate enlargement
    → SPEECH stimuli: LARGEST enlargement
    → = Complex, social-relevant stimuli suffer MOST

  Perceptual binding strength correlated with:
    → Low-level temporal processing abilities
    → = Foundation matters — low-level timing → high-level integration

  IMPLICATION:
    → Speech = inherently audiovisual (lip movement + sound + facial expression)
    → Wider TBW for speech → audiovisual speech binding DIFFERENT
    → = Difficulty following conversation in group (multiple audio-visual streams)
    → = Preference for text-based communication (removes temporal binding requirement)
    → = Consistent with Sasson 2017: bias disappears with TEXT-only

🟢 Baum, Stevenson & Wallace 2015 (Progress in Neurobiology 134:140-160):
  Comprehensive review: ROBUST and CONSISTENT multisensory integration impairment
  Neuroimaging: failure to activate large-scale cortical networks
  Reduced long-range functional connectivity supports integration differences
```

### §3.3 — Framework: Social = inherently multi-modal

```
🟡 07-Social-Recognition-Arc.md §1.1:

  "There is no meaningful single-channel social event."

  Social event = MULTIPLE channels fire SIMULTANEOUSLY:
    → Visual (face + posture + gesture)
    → Auditory (voice + intonation + content)
    → Tactile (handshake, hug, proximity)
    → Interoceptive (emotional state mirroring)
    → Olfactory (pheromones, familiar scent)

  4 binding mechanisms (07-Social-Recognition-Arc §6):
    ① Temporal co-occurrence + Hebbian binding
    ② Multisensory neurons (superior colliculus + STS)
    ③ Default Mode Network scaffolding
    ④ Amygdala emotional tag as binding cue

  NẾU 1+ mechanism DIFFERENT (not absent — DIFFERENT):
    → Social chunk compilation CASCADE different
    → Social chunks compile with DIFFERENT temporal resolution
    → = "Social difficulty" = DOWNSTREAM of integration difference
    → = NOT "social brain broken" — integration parameters DIFFERENT
```

### §3.4 — From integration to social: the cascade

```
🟡 FRAMEWORK SYNTHESIS:

  STEP 1: TBW wider (~600ms vs ~300ms)
    → Brain binds more events into single percept
    → OR fails to separate rapidly alternating events

  STEP 2: Social events = rapid multi-modal streams
    → Conversation: words at ~150ms intervals + facial expressions at ~200ms
    → Group conversation: multiple speakers overlapping
    → Each stream requires temporal binding PRECISION

  STEP 3: Wider TBW → DIFFERENT social chunk compilation
    → Nuances that depend on precise timing → missed or blurred
    → "Sarcasm" = requires detecting MISMATCH between tone (audio) and content
      within very narrow temporal window → harder with wider TBW
    → "Social cues" = often multi-modal and time-dependent

  STEP 4: Different social chunks → different Self-Pattern-Modeling predictions (§5)
    → Self-Pattern-Modeling runs on OWN chunk library
    → Different social chunks → different predictions about others
    → Neurotypical predicts using neurotypical social chunks → fails for autistic person
    → Autistic predicts using own social chunks → fails for neurotypical
    → = BIDIRECTIONAL failure (double empathy, §5.4)

  → "Social deficit" reframe: INTEGRATION DIFFERENCE → COMPILATION DIFFERENCE
    → DOWNSTREAM cascade, not primary "social brain" failure
```

## §4 — PREDICTIVE PROCESSING: 3 THEORIES × FRAMEWORK

### §4.1 — Hypo-priors: world "too real"

```
🟢 Pellicano & Burr 2012 (TiCS 16(10):504-510):

  CORE CLAIM:
    → Typical brain: strong PRIORS (predictions based on past experience)
      → Perception = prior + sensory input (Bayesian brain)
      → Strong priors → SMOOTH experience (fill in gaps, predict next event)
    → Autistic brain: WEAKER PRIORS
      → Sensory data DOMINATES → world perceived "more raw"
      → Less smoothing → more detail perceived → BUT more overwhelming

  EXPLAINS:
    → Enhanced perceptual abilities (§9): weaker priors = less top-down smoothing
      → SEE MORE DETAIL (not filtered by expectation)
    → Sensory sensitivity: raw data without dampening = LOUDER, BRIGHTER, MORE
    → Resistance to optical illusions (some studies)
    → Difficulty with social prediction (social = heavily prior-dependent)

  FRAMEWORK MAPPING:
    → prediction-delta = difference between PREDICTED and ACTUAL
    → Weak priors → fewer/weaker predictions → MORE prediction-delta per event
    → = Body-feedback fires MORE OFTEN, for MORE things
    → = Consistent with sensory overload (§2) and reduced habituation (§2.3)
```

### §4.2 — HIPPEA: inflexible precision of prediction errors

```
🟢 Van de Cruys et al. 2014 (Psychological Review 121(4):649-675):

  CORE CLAIM:
    → Typical brain: FLEXIBLY adjusts precision of prediction errors
      → "Noisy context" → turn DOWN precision (tolerate more mismatch)
      → "Precise context" → turn UP precision (catch small errors)
    → Autistic brain: prediction errors given inflexibly HIGH precision
      → EVERY mismatch treated as IMPORTANT → cannot ignore minor deviations
      → = OVERFITTING to details

  EXPLAINS:
    → Insistence on sameness: changing routine = prediction error
      → HIGH precision → error treated as SIGNIFICANT → distress
    → Detail focus (§9): high precision on sensory errors → notice fine details
    → Difficulty with ambiguity: ambiguous situations = many prediction errors
      → all treated as important → OVERWHELMING
    → Skill with rule-based systems: clear rules → few prediction errors → comfortable

  FRAMEWORK MAPPING:
    → prediction-delta AMPLIFIED
    → Same mismatch → LARGER body-feedback response in autism
    → = Consistent with insistence on sameness (IS factor from §1.7)
    → = IS = strategy to REDUCE prediction errors (control environment → predictable)
```

### §4.3 — Aberrant Precision: WHERE matters

```
🟢 Lawson, Rees & Friston 2014 (Frontiers in Human Neuroscience 8:302):

  CORE CLAIM:
    → Precision allocation ATYPICAL across cortical hierarchy
    → NOT uniformly hypo or hyper — DIFFERENT at different levels
    → Low-level sensory: TOO MUCH precision (over-weighted sensory details)
    → High-level social/abstract: TOO LITTLE precision (weak top-down models)
    → = BRIDGE between Pellicano (weak high-level priors)
        and Van de Cruys (inflexible low-level precision)

  EXPLAINS:
    → BOTH enhanced perception (low-level precision high)
        AND social difficulty (high-level precision low)
    → Simultaneously: SEE details + MISS social gist
    → = NOT paradox — DIFFERENT levels of hierarchy

  FRAMEWORK MAPPING:
    → Prediction model operates at multiple levels (Core-v7.8-Draft.md)
    → Autism: low-level prediction (sensory) = overweighted
    → High-level prediction (social, abstract) = underweighted
    → = Consistent with cascade model (§2.5):
        sensory → rich/detailed chunks | social → sparser chunks
```

### §4.4 — Framework position: cả 3 consistent

```
🟡 FRAMEWORK SYNTHESIS:

  Plan gốc question: "prediction-delta threshold LOWER vs AMPLIFIED?"
  Answer sau research: CẢ 3 theories consistent với framework:

  ┌────────────────────┬──────────────────────────┬───────────────────────┐
  │ Theory             │ Mechanism                │ Framework mapping     │
  ├────────────────────┼──────────────────────────┼───────────────────────┤
  │ Hypo-priors        │ Weak predictions →       │ prediction-delta      │
  │ (Pellicano 2012)   │ more surprise per event  │ FIRES MORE OFTEN     │
  ├────────────────────┼──────────────────────────┼───────────────────────┤
  │ HIPPEA             │ Inflexible high-precision │ prediction-delta     │
  │ (Van de Cruys 2014)│ → every mismatch large   │ AMPLIFIED per event  │
  ├────────────────────┼──────────────────────────┼───────────────────────┤
  │ Aberrant Precision │ Precision atypical across │ prediction-delta     │
  │ (Lawson 2014)      │ hierarchy levels         │ DIFFERS by level     │
  └────────────────────┴──────────────────────────┴───────────────────────┘

  ALL 3 → prediction-delta LARGER than typical → larger body-feedback

### §6.2 — Hull et al. 2017: 3-stage model

## §10 — MELTDOWN/SHUTDOWN QUA BODY-FEEDBACK LENS

### §10.1 — Meltdown: OVERFLOW, not tantrum

```
🟡 FRAMEWORK ANALYSIS:

  MELTDOWN defined:
    → Body-feedback OVERLOAD → PFC goes OFFLINE → motor/emotional DISCHARGE
    → Crying, screaming, hitting, throwing, running
    → INVOLUNTARY — person cannot "choose" to stop

  MELTDOWN ≠ TANTRUM:
    → Tantrum = GOAL-DIRECTED (want something → escalate to get it)
    → Meltdown = OVERFLOW (body-feedback exceeds PFC capacity → discharge)
    → Tantrum: audience changes behavior. Remove audience → often stops.
    → Meltdown: audience irrelevant. Remove audience → continues.
    → = Fundamentally different mechanism — same behavioral surface

  MECHANISM (Body-Feedback-Mechanism.md):
    → Sensory input → body-feedback (§2: no habituation → ACCUMULATES)
    → Prediction-delta → body-feedback (§4: amplified in autism)
    → Social demands → body-feedback (§5: Self-Pattern-Modeling mismatch → dissonance)
    → Cumulative load → EXCEEDS PFC processing capacity
    → PFC goes offline (Threat.md: NE α1 cascade, PFC disconnect)
    → Body-feedback DISCHARGES through motor/emotional output
```

### §10.2 — Shutdown: WITHDRAWAL, not avoidance

```
🟡 FRAMEWORK ANALYSIS:

  SHUTDOWN defined:
    → Body-feedback OVERLOAD → PFC WITHDRAWAL → freeze/collapse
    → Going silent, becoming non-verbal, "spacing out," lying down
    → INVOLUNTARY — person cannot "choose" to engage

  SHUTDOWN = DIFFERENT EXPRESSION, SAME CAUSE:
    → Same trigger: body-feedback exceeds PFC capacity
    → Different output: discharge (meltdown) vs withdrawal (shutdown)
    → May depend on:
      → Individual nervous system profile (fight/flight vs freeze)
      → Context (public → shutdown, private → meltdown)
      → Current arousal level (moderate → meltdown, extreme → shutdown)

  PFC-Configuration.md:
    → Mode ④ Disconnected: PFC partially offline
    → Mode ⑤ Dissociated: PFC withdrawal, emotional numbness
    → Shutdown ≈ Mode ⑤ — defense mechanism, not "not caring"
```

### §10.3 — Neuroscience evidence

```
🟢 Green et al. 2013 (JAACAP 52(11)):
  → Amygdala HYPERACTIVATION to mildly aversive sensory stimuli
  → Even controlling for anxiety → sensory over-reactivity = DISTINCT
  → Severity correlated with parent-reported sensory difficulties

🟢 Green et al. 2015 (JAMA Psychiatry 72(8)):
  → REDUCED NEURAL HABITUATION in ASD+SOR
  → Brain does NOT "get used to" repeated stimuli
  → = CUMULATIVE: each additional input ADDS to load (no decay)
  → = "Fine in the morning, meltdown by afternoon" = ACCUMULATION, not trigger

  FRAMEWORK:
    → Reduced habituation + amplified prediction-delta + sensory gain high
    → = TRIPLE load on body-feedback system
    → Throughout the day: load ACCUMULATES (no habituation to reduce)
    → Final straw: seemingly minor event → capacity EXCEEDED → meltdown/shutdown
    → Observers: "They overreacted to nothing" → WRONG — overreacted to EVERYTHING
      accumulated over HOURS, final event was just the ONE that tipped the scale
```

### §10.4 — Stimming = REGULATION mechanism

```
🟢 Kapp et al. 2019 (Autism 23(7):1782-1792):

  Study: autistic adults' OWN perspectives on stimming
  → Self-regulatory mechanism (soothing, communicating, processing)
  → Suppressing stimming = EFFORTFUL + DISTRESSING
  → Stimming serves MULTIPLE functions:
    → Self-soothing (reduce body-feedback load)
    → Communication (express internal state)
    → Processing aid (rhythmic movement supports thinking)
    → Enjoyment (pleasant sensory input)

  FRAMEWORK:
    → Fidgeting-Analysis.md v1.0: fidgeting = body-feedback regulation MILD
    → Stimming = body-feedback regulation STRONGER
      (same mechanism, higher regulation NEED due to higher sensory load)
    → Stimming = body-feedback DRAIN mechanism:
      → Rhythmic movement → predictable sensory input → prediction-delta LOW
      → = Creates "safe" sensory stream that reduces cumulative load
      → = Like background music that masks unpredictable noise

  SUPPRESSING STIMMING:
    → Remove regulation mechanism WHILE load continues building
    → = INCREASES risk of meltdown/shutdown (no drain + accumulating load)
    → = Forcing "quiet hands" = removing fire extinguisher during fire
    → Correct approach: allow stimming + reduce ENVIRONMENTAL load
```

### §10.5 — Recovery

```
🟡 FRAMEWORK ANALYSIS:

  Body-feedback must DRAIN before PFC re-engage:
    → After meltdown: cortisol + NE + body-feedback ELEVATED
    → PFC recovery = not instant (needs body-feedback to DECREASE first)
    → Body-first invariant (Feeling.md): body state → PFC capacity
      NOT PFC decision → body state

  WHAT HELPS:
    → Low-stimulation environment (reduce INPUT)
    → Time (body-feedback naturally drains if no new input)
    → Predictable, safe context (reduce prediction-delta)
    → Familiar sensory input (stimming, comfort objects, familiar textures)
    → NO additional demands (adding demands = adding load to already-overloaded system)

  WHAT HARMS:
    → Forcing engagement DURING overload → extends recovery
    → Demanding eye contact → adds amygdala activation
    → Bright/loud environment → adds sensory input
    → Social pressure ("calm down") → adds social prediction demand
    → "Processing" the event immediately → PFC not yet online
    → = All of above = ADDING LOAD to system that needs DRAINING
```
