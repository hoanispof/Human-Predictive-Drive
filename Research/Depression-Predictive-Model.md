## 1. CƠ CHẾ CORE: LEARNED PREDICTION SUPPRESSION

### 1.1 Bối cảnh — 3 đường từ cùng gốc

```
Cortisol mãn tính từ open loop không resolve
                    │
     ┌──────────────┼──────────────┐
     │              │              │
  Loop CỤ THỂ   Loop ÍT +     Loop TRỪU TƯỢNG +
  + não active   cortisol CAO   cortisol TB-thấp
     │              │              │
   RITUAL        SHUTDOWN      SUPPRESSION
     │              │              │
    OCD          BURNOUT      TRẦM CẢM
  (mắc kẹt)    (sụp → hồi)  (phẳng dần)
```

**OCD**: não predict MẠNH nhưng SAI CHỖ → tạo ritual → mắc kẹt trong loop
**Burnout**: não KHÔNG THỂ predict → PFC hardware shutdown → nghỉ → hồi
**Trầm cảm**: não CÓ THỂ nhưng KHÔNG predict → learned suppression → phẳng dần

### 1.2 Cơ chế chi tiết

```
BÌNH THƯỜNG:
  Predict → mismatch → điều chỉnh → thử lại → match → đóng loop → satisfaction
  (Cycle hoạt động → motivation duy trì → cuộc sống có hướng)

ĐƯỜNG VÀO TRẦM CẢM:
  Predict → mismatch → điều chỉnh → thử lại → MISMATCH LẠI
  → điều chỉnh → thử lại → MISMATCH LẠI
  → lặp N lần...
  → Não encode pattern: "predict KHÔNG DẪN TỚI match"
  → Giảm prediction activity (KHÔNG tắt hẳn — giảm DẦN)
  → Ít predict = ít expectation = ít motivation = ít action
  → Kết quả: PHẲNG cảm xúc, mất hứng thú, mất động lực
```

**Điểm then chốt**: đây KHÔNG phải PFC sụp (burnout). PFC vẫn hoạt động.
Người trầm cảm vẫn CÓ THỂ làm việc, suy nghĩ, quyết định.
Nhưng não đã học rằng prediction không hiệu quả → KHÔNG MUỐN predict.

### 1.3 Tại sao nghỉ ngơi KHÔNG đủ hồi

```
Burnout:
  PFC sụp (hardware) → nghỉ → PFC repair → predict lại → hồi
  Giống circuit breaker: tắt → reset → bật lại

Trầm cảm:
  Suppression = LEARNED pattern (encode trong synaptic weights)
  Nghỉ ngơi không xóa learned pattern
  → Não vẫn "biết" rằng predict vô ích → vẫn suppress
  → Cần RE-TEACH: tạo trải nghiệm predict → match mới
     để não CẬP NHẬT pattern cũ
```

Đây là lý do:
- **CBT hiệu quả**: thay đổi belief = thay đổi prediction pattern → não học lại
- **SSRIs hiệu quả**: có thể hạ ngưỡng để não bắt đầu predict lại
  → predict nhỏ → match → positive feedback → dần hồi
- **Chỉ nghỉ**: không đủ — suppression vẫn còn

## 2. ĐIỀU KIỆN: TẠI SAO ĐƯỜNG TRẦM CẢM THAY VÌ OCD/BURNOUT

### 2.1 Open loop TRỪU TƯỢNG

OCD cần open loop có "handle" — thứ cụ thể để ritual hóa:
  - Khóa cửa → kiểm tra lại
  - Bẩn → rửa tay
  - Sai thứ tự → sắp lại

Trầm cảm = open loop KHÔNG CÓ handle:
  - "Mình có giá trị không?" → làm gì để ritual hóa câu hỏi này?
  - "Tương lai sẽ ra sao?" → không có action cụ thể nào đóng được
  - "Mình có đang sống đúng cách?" → không có tiêu chí rõ ràng
  - Kỳ vọng xã hội mơ hồ → không biết đáp ứng bao nhiêu là đủ

### 2.2 Không đóng được bằng 4 cách (Block 5b)

```
① Giải thật sự  — Không giải được (câu hỏi không có đáp án duy nhất)
② Chunk/integrate — Quá trừu tượng để chunk
③ PFC release    — Quá quan trọng (identity, survival) → không release được
④ Ritual          — Không có handle cụ thể → không ritual hóa được
```

→ Loop mở VĨNH VIỄN → drain liên tục → suppression dần dần

### 2.3 Cortisol ở mức "vừa đủ hại"

```
Cortisol CAO + đột ngột     → PFC sụp → BURNOUT (circuit breaker)
Cortisol CAO + loop cụ thể  → ritual phát triển → OCD
Cortisol TB-THẤP + kéo dài  → không đủ sụp, không đủ ritual
                             → drain chậm → SUPPRESSION → trầm cảm
```

Đây là lý do trầm cảm thường "đến từ từ" — không có sự kiện rõ ràng,
không có điểm sụp, chỉ phẳng dần theo thời gian.

## 3. CÁC ĐƯỜNG VÀO TRẦM CẢM (Prediction-Based)

### 3.1 Đường xã hội — Open loop từ môi trường

```
Xã hội hiện đại tạo open loop trừu tượng LIÊN TỤC:

  Mạng xã hội:
    → So sánh xã hội (predict "mình cần như họ" → mismatch liên tục)
    → Validation loop (predict "post này sẽ được like" → không kiểm soát được)

  Kỳ vọng thành công:
    → "Tuổi này phải có nhà/xe/gia đình" → open loop không rõ endpoint
    → Predict → mismatch → điều chỉnh → vẫn mismatch (vì tiêu chí dịch chuyển)

  Bất định kinh tế/tương lai:
    → Predict tương lai → không đủ data → mismatch → predict lại → vẫn mismatch
    → Não không thể đóng loop về tương lai (chưa xảy ra)

  Thông tin quá tải (news cycle):
    → Mỗi tin = 1 open loop nhỏ (threat assessment)
    → Tổng cộng = drain liên tục ở background
```

### 3.2 Đường identity — Open loop về bản thân

```
  "Mình là ai?" / "Mình muốn gì thật sự?"
    → Predict identity → mismatch với hành vi thực tế → điều chỉnh → vẫn mismatch
    → Đặc biệt phổ biến ở giai đoạn chuyển tiếp (tốt nghiệp, đổi nghề, ly hôn)

  "Mình có đủ tốt không?"
    → Predict tiêu chuẩn → so sánh → mismatch
    → Tiêu chuẩn DI CHUYỂN (moving goalpost) → KHÔNG BAO GIỜ match
    → = Open loop vĩnh viễn
```

### 3.3 Đường mất mát — Prediction network bị phá vỡ

```
  Mất người quan trọng / mất công việc / mất mối quan hệ:
    → Não có HỆ THỐNG prediction dựa trên thứ đó
      (VD: "sáng dậy sẽ thấy người đó" → prediction fails)
    → Hàng trăm micro-prediction bị mismatch đồng thời
    → Quá nhiều để re-learn → suppression hàng loạt

  ⚠️ Đây là nơi prediction pathway GIAO với grief reaction.
  Grief bình thường: dần re-learn prediction mới → hồi.
  Grief → depression: suppression thay vì re-learn → phẳng kéo dài.
  (Cần thêm data để phân biệt rõ hơn — xem Research Gaps)
```

### 3.4 Đường kiệt sức quyết định — Decision fatigue pathway

```
  Quá nhiều quyết định nhỏ mỗi ngày (hiện đại > quá khứ)
    → Mỗi quyết định = 1 prediction + evaluation
    → PFC workload cao liên tục (không sụp — chỉ mệt)
    → Predict → chọn → kết quả không rõ tốt/xấu → không có feedback rõ
    → Dần dần: "chọn gì cũng vậy" → suppress prediction cho quyết định
    → Lan ra: "làm gì cũng vậy" → phẳng
```

## 5. COMORBIDITY VỚI OCD VÀ BURNOUT

### 5.1 OCD → Trầm cảm (đường tự nhiên)

```
OCD: ritual hoạt động → predict + ritual + check → loop giữ não active
     │
     ritual BẮT ĐẦU THẤT BẠI (tolerance — cần nhiều hơn nhưng không đủ)
     │
     predict + ritual → MISMATCH LẠI (ritual không giảm anxiety nữa)
     │
     Não học: "ritual cũng vô ích" → suppress prediction ở domain đó
     │
     OCD + DEPRESSION (comorbid)
```

🟢 Literature: ~67% OCD patients có depressive episodes (Ruscio et al., 2010)
Framework giải thích: đây không phải 2 bệnh ngẫu nhiên trùng — mà có ĐƯỜNG ĐI
từ cái này sang cái kia khi ritual thất bại.

### 5.2 Trầm cảm → OCD (rất hiếm)

```
Trầm cảm: prediction SUPPRESSED → não không active predict
  → Không đủ prediction activity để TẠO ritual mới
  → Ritual cần: predict + check + repeat — cả 3 cần não active

Để trầm cảm → OCD cần:
  ① Bật lại prediction trước (thuốc/therapy/sự kiện mạnh)
  ② Gặp open loop CỤ THỂ có handle ở thời điểm đó
  ③ Hardware OFC/dlPFC thiên về ritual
  → Xác suất 3 điều kiện đồng thời: THẤP
```

### 5.3 Burnout → Trầm cảm (có thể)

```
Burnout: PFC sụp → nghỉ → PFC hồi → quay lại → sụp lại (recurrence)
  │
  Sau nhiều cycle burnout-recovery-burnout:
  Não có thể học: "hồi cũng vô ích, sẽ sụp lại"
  → Từ SHUTDOWN (hardware) chuyển sang SUPPRESSION (learned)
  → Burnout chuyển thành trầm cảm mãn tính
```

⚠️ Đây là prediction chưa verify — cần data.

### 5.4 Trầm cảm → Burnout (khó)

```
Trầm cảm: prediction suppressed → workload GIẢM (vì không làm gì)
  → Cortisol không đủ cao để sụp PFC
  → Burnout cần cortisol CAO + workload CAO — trầm cảm không tạo điều kiện này
```

### 5.5 Bảng tóm tắt chuyển đổi

```
  Từ \ Sang     OCD          Burnout      Trầm cảm
  ─────────────────────────────────────────────────
  OCD           —            Hiếm         CÓ (~67%)
  Burnout       Hiếm         —            CÓ THỂ (sau nhiều cycle)
  Trầm cảm     Rất hiếm     Khó          —
```

## 6. VULNERABILITY THEO ENCODING MODALITY

```
Somatic-dominant:
  Loop trừu tượng = khó cảm nhận rõ → có thể IGNORE được
  → Có thể ÍT vulnerable hơn với depression pathway xã hội
  → NHƯNG: mất mát cảm xúc (người quan trọng) = hit RẤT MẠNH
  → 🟡 Cần data

Verbal-dominant:
  Loop trừu tượng = CHÍNH LÀ verbal territory
  "Mình có giá trị không?" = câu hỏi verbal, lặp verbal, drain verbal
  → Có thể NHIỀU vulnerable hơn với depression từ rumination
  → Pure-O + depression comorbidity có thể cao nhất ở nhóm này
  → 🟡 Cần data

Spatial-dominant:
  → Chưa đủ data để predict

Motor-dominant:
  Nếu vẫn hoạt động thể chất → cortisol management tự nhiên
  → Có thể protective factor
  → NHƯNG: nếu bị CHẶN hoạt động (chấn thương, bệnh) → risk tăng mạnh
  → 🟡 Cần data
```

## 7. CAN THIỆP — PHÂN TÍCH CHUYÊN SÂU THEO CƠ CHẾ

### 7.1 Nguyên tắc gốc

```
Burnout can thiệp:    NGHỈ    (PFC cần repair hardware)
Depression can thiệp: RE-TEACH (não cần LEARN LẠI rằng predict có ích)
→ Nghỉ ngơi KHÔNG đủ cho depression — suppression đã encode, cần override.
```

### 7.2 Bốn cơ chế can thiệp

```
Framework identify 4 CƠ CHẾ khác nhau mà các phương pháp hiện tại sử dụng:

  CƠ CHẾ A — TRÁNH LOOP (Avoidance)
    Đổi domain/môi trường → không chạm open loop trừu tượng nữa
    → Loop cũ dần fade (không được reinforce) → suppression giảm
    ⚠️ Hiệu quả nhưng TẠM — hardware không đổi, gặp loop tương tự = tái phát

  CƠ CHẾ B — VIẾT LẠI PREDICTION (Rewrite)
    Identify prediction pattern sai → test → thay bằng pattern mới
    → Não re-learn ở level learned → thay đổi BỀN
    ⚠️ Chậm, cần kỹ năng therapist, nhưng bền nhất

  CƠ CHẾ C — HẠ NGƯỠNG PREDICT (Lower threshold)
    Dùng hóa chất/kích thích để não DỄ bắt đầu predict lại
    → Không chữa — TẠO ĐIỀU KIỆN để re-learn
    ⚠️ Phụ thuộc thuốc nếu không kết hợp B

  CƠ CHẾ D — RESET (Pattern dissolution)
    Phá vỡ toàn bộ prediction pattern (kể cả suppression)
    → Não BUILD LẠI từ đầu → cơ hội re-learn
    ⚠️ Mạnh nhất nhưng rủi ro — phá CẢ pattern tốt lẫn xấu
       Nếu không kết hợp re-teach (B) → não có thể re-learn suppression

  ╔══════════════════════════════════════════════════════════════╗
  ║ INSIGHT QUAN TRỌNG:                                        ║
  ║ Phương pháp BỀN nhất ≠ phương pháp MẠNH nhất.              ║
  ║ Bền nhất = VIẾT LẠI (B) + tạo điều kiện (C nếu cần).      ║
  ║ Reset (D) mạnh nhưng nếu không re-teach → tái phát.        ║
  ║                                                             ║
  ║ Phương pháp lý tưởng = C (bật predict) + B (dạy predict    ║
  ║ đúng) + A (giảm input loop mới) = 3 cơ chế phối hợp.      ║
  ╚══════════════════════════════════════════════════════════════╝
```

```
   Framework predict: đặc biệt hiệu quả cho SOMATIC-DOMINANT
     (tương tác vật lý, cảm giác, không cần verbal processing)
   Giới hạn: không đủ 1 mình cho phổ vừa+, là bổ trợ

   Giới hạn: cần CONSISTENCY, khó duy trì khi suppress nặng
   Framework predict: hiệu quả GIẢM khi phổ nặng hơn
     (suppression sâu → micro-predict không đủ mạnh để override)

④ SOCIAL CONNECTION
   Cơ chế: A (tránh — đổi context) + B nhẹ (predict xã hội → match)
   Framework: giao tiếp = predict phản ứng người khác → match/mismatch
     Nếu giao tiếp tích cực → match → positive feedback
     + Oxytocin giảm cortisol (tương tác trực tiếp)
   Evidence: 🟢 Mạnh — social isolation = risk factor lớn
     Holt-Lunstad et al. (2015): social isolation ~ 15 cigarettes/day mortality
   Giới hạn: depression gây thu mình → khó tiếp cận social
     = Catch-22: cần social để hồi, nhưng suppress cản social

══════════════════════════════════════════════════════════════════════════
MỨC 2 — CẦN HỖ TRỢ CHUYÊN GIA (phổ vừa)
══════════════════════════════════════════════════════════════════════════

④b IPT (Interpersonal Therapy)
   Cơ chế: B (viết lại prediction XÃ HỘI)
   Framework:
     IPT tập trung vào 4 vấn đề: grief, role disputes, role transitions, deficits
     → TẤT CẢ đều là open loop XÃ HỘI trừu tượng
     IPT = identify open loop xã hội cụ thể → tạo plan đóng loop
     → Predict "relationship sẽ thay đổi nếu làm X" → test → feedback
     = Viết lại prediction pattern ở DOMAIN XÃ HỘI cụ thể
   Evidence: 🟢 Mạnh — ngang CBT cho moderate depression
     Cuijpers et al. (2011): IPT = CBT = medication cho moderate
   Framework giải thích TẠI SAO IPT hiệu quả NHƯ CBT:
     Cả hai đều là B (rewrite), chỉ khác DOMAIN target.
     CBT: rewrite prediction TỔNG QUÁT. IPT: rewrite prediction XÃ HỘI.
     → Hiệu quả tương đương vì CƠ CHẾ giống nhau.
   Framework predict: IPT > CBT cho depression từ ĐƯỜNG XÃ HỘI (§3.1)
     CBT > IPT cho depression từ ĐƯỜNG IDENTITY (§3.2)
   Giới hạn: cần therapist trained, ít phổ biến hơn CBT

④c ACT (Acceptance and Commitment Therapy)
   Cơ chế: A+B hybrid (release loop + tạo prediction mới từ values)
   Framework:
     ACT gồm 2 phần:
     ① Acceptance = dạy não BUÔNG loop không giải được (= PFC release, cách đóng ③)
       → Giảm drain → giải phóng bandwidth → giống mindfulness nhưng structured
     ② Committed action based on VALUES = tạo PREDICTION MỚI
       "Tôi sẽ hành động theo giá trị X" → hành động → kết quả
       → Predict → match (vì values = tự chọn, dễ match hơn kỳ vọng bên ngoài)
     = Buông loop CŨ + build loop MỚI có match rate cao
   Evidence: 🟢 TB-Mạnh — growing evidence base, RCTs positive
     A-Tjak et al. (2015): ACT hiệu quả cho depression, effect size TB
   Framework giải thích TẠI SAO values quan trọng:
     Values = direction (không phải endpoint) → KHÔNG BAO GIỜ là open loop
     (không có "đến nơi" hay "chưa đến nơi" → không thể mismatch)
     = Cách predict mà LUÔN CÓ match → antidote cho suppression
   Giới hạn: khái niệm trừu tượng, cần therapist giỏi giải thích

④d ART / MUSIC THERAPY
   Cơ chế: B qua NON-VERBAL channel
   Framework:
     Tạo nghệ thuật = predict ("mình muốn tạo cái gì đó") → process → output
     → Match: sản phẩm HIỆN DIỆN, cụ thể, có thể thấy/nghe
     = Micro-predict nhưng qua KÊNH KHÔNG LỜI
   ⚡ INSIGHT QUAN TRỌNG:
     Art/Music therapy có thể là PHƯƠNG PHÁP B DUY NHẤT cho non-verbal dominant
     CBT = B qua verbal. Art therapy = B qua visual/somatic/auditory.
     Nếu người bệnh là SOMATIC-DOMINANT hoặc SPATIAL-DOMINANT:
       → CBT verbal có thể KHÔNG CHẠM được prediction system chính
       → Art therapy chạm TRỰC TIẾP
   Evidence: 🟢 TB — meta-analyses cho thấy hiệu quả, đặc biệt kết hợp
     Uttley et al. (2015): art therapy = significant effect cho depression
   Framework predict:
     Art therapy > CBT cho somatic/spatial-dominant (chưa verify)
     CBT > Art therapy cho verbal-dominant
   Giới hạn: ít RCTs lớn, khó standardize

④e PSYCHODYNAMIC THERAPY
   Cơ chế: B sâu (viết lại prediction VÔ THỨC)
   Framework:
     Psychodynamic = identify prediction pattern KHÔNG NHẬN BIẾT
     → Pattern được encode từ nhỏ (attachment, early experience)
     → Người bệnh không BIẾT mình đang predict gì → không tự sửa được
     → Therapist giúp SURFACE hidden prediction → test → rewrite
     = Giống CBT nhưng ở level SÂU HƠN (unconscious vs conscious prediction)
   Evidence: 🟢 TB — long-term outcomes tốt
     Leichsenring & Rabung (2008): long-term psychodynamic = effective
   Framework giải thích TẠI SAO mất thời gian:
     Hidden prediction KHÓA hơn conscious prediction
     → Cần nhiều thời gian hơn để surface + rewrite
     Nhưng khi rewrite xong → BỀN hơn (vì thay đổi ở level sâu nhất)
   Giới hạn: chậm (months-years), cần therapist rất giỏi, đắt

⑤ BEHAVIORAL ACTIVATION (với therapist)
   Cơ chế: B (viết lại) — phiên bản có cấu trúc của ③
   Framework: therapist BUỘC tạo hoạt động dù không muốn
     → Hoạt động → predict → feedback → dần hồi prediction activity
     Khác ③ vì có NGƯỜI HỖ TRỢ khi suppress quá nặng để tự bắt đầu
   Evidence: 🟢 Mạnh — ngang CBT cho moderate depression
     Dimidjian et al. (2006): BA = CBT = medication cho moderate
   Framework giải thích TẠI SAO effective:
     Phải có prediction ACTIVITY thì mới có gì để re-learn.
     Không hoạt động = không predict = không feedback = không thay đổi.

⑥ CBT (Cognitive Behavioral Therapy)
   Cơ chế: B (viết lại) — trực tiếp nhất
   Framework:
     CBT identify: "predict A dẫn tới expect B" (VD: "mình thử = mình thất bại")
     CBT test: tạo experiment nhỏ → kiểm tra prediction có đúng không
     CBT rewrite: prediction sai → thay prediction mới → não encode
     = TRỰC TIẾP thay đổi prediction pattern ở level learned
   Evidence: 🟢 Rất mạnh — decades of RCTs, gold standard
     Cuijpers et al. (2019): CBT effective cho mọi mức độ depression
   Framework giải thích TẠI SAO CBT là gold standard:
     Vì nó trực tiếp tác động vào CƠ CHẾ GỐC (prediction pattern).
     Các phương pháp khác tác động GIÁN TIẾP.
   Giới hạn: cần therapist giỏi, mất thời gian (12-20 sessions)

⑦ MINDFULNESS / MEDITATION
   Cơ chế: Đặc biệt — học PFC RELEASE loop (cách đóng ③ Block 5b)
   Framework:
     Không giải loop — nhưng BUÔNG được → giảm drain
     = Dạy não: "loop này KHÔNG CẦN giữ active"
     → Giải phóng WM resources → PFC có bandwidth predict lại
   Evidence: 🟢 TB-Mạnh — MBCT (Mindfulness-Based CBT) hiệu quả
     đặc biệt cho PREVENTION relapse
     Kuyken et al. (2016): MBCT giảm relapse 31%
   Framework giải thích: buông loop = giảm background drain
     → Ít drain = PFC khỏe hơn = predict dễ hơn = ít suppress
   Giới hạn: khó học khi suppress nặng (cần PFC hoạt động để learn)

⑦b EMDR (Eye Movement Desensitization and Reprocessing)
   Cơ chế: D CỤC BỘ (reset TARGETED — chỉ phá pattern liên quan)
   Framework:
     EMDR ≠ reset toàn bộ (như psychedelic/ECT)
     EMDR = bilateral stimulation → disrupt SPECIFIC memory-prediction link
     → Pattern "sự kiện X → predict tai họa Y" bị PHÁ cục bộ
     → Re-consolidation: não BUILD LẠI prediction mới cho sự kiện đó
     = "D chính xác" — chỉ phá cái cần phá, giữ nguyên phần còn lại
   Evidence: 🟢 Mạnh cho PTSD, 🟡 TB cho depression
     Cuijpers et al. (2020): EMDR hiệu quả cho depression có trauma component
   Framework giải thích TẠI SAO EMDR mạnh cho trauma nhưng TB cho depression:
     Trauma = open loop CỤ THỂ (sự kiện rõ ràng) → D cục bộ = chính xác
     Depression = open loop TRỪU TƯỢNG (nhiều, mơ hồ) → D cục bộ = không đủ scope
     NHƯNG: depression có NGUỒN GỐC từ sự kiện cụ thể → EMDR hiệu quả
   Framework predict: EMDR hiệu quả nhất cho depression ĐƯỜNG MẤT MÁT (§3.3)
     (mất mát = sự kiện cụ thể → có target rõ cho D cục bộ)
   Giới hạn: cần therapist certified, không phù hợp depression không có trauma

MỨC 3 — CẦN BÁC SĨ (phổ vừa-nặng)
══════════════════════════════════════════════════════════════════════════

⑧ SSRIs / SNRIs (thuốc chống trầm cảm thế hệ mới)
   Cơ chế: C (hạ ngưỡng predict)
   Framework:
     Serotonin ↑ → ngưỡng để não bắt đầu predict GIẢM
     → Predict nhỏ bắt đầu xuất hiện → match → positive feedback
     = Không CHỮA — TẠO ĐIỀU KIỆN để predict system hoạt động lại
     Giống "khởi động xe chết máy" — cần ai đó quay chìa khóa
   Evidence: 🟢 Mạnh — decades of evidence, FDA approved
   Framework predict quan trọng:
     SSRIs + CBT > SSRIs alone > CBT alone (cho moderate-severe)
     Vì: C (bật predict) + B (dạy predict đúng) = phối hợp tối ưu
   🟢 Literature confirm: combination therapy > monotherapy
     Cuijpers et al. (2014): combination = most effective
   Giới hạn: side effects, 2-4 tuần mới có tác dụng
     Framework giải thích delay: cần thời gian để predict system
     TÍCH LŨY đủ micro-matches mới thay đổi rõ

⑨ TMS (Transcranial Magnetic Stimulation)
   Cơ chế: C (hạ ngưỡng) + D nhẹ (kích thích PFC)
   Framework:
     Kích thích từ trường nhắm vào dlPFC
     → TĂNG PFC activity trực tiếp → tăng khả năng predict
     = "Sạc pin" cho PFC, không phải reset toàn bộ
   Evidence: 🟢 TB-Mạnh — FDA approved cho treatment-resistant
   Giới hạn: cần thiết bị, nhiều sessions (20-30)

⑨b VNS (Vagus Nerve Stimulation)
   Cơ chế: C (hạ ngưỡng qua vagal pathway)
   Framework:
     Kích thích dây thần kinh vagus → tăng vagal tone
     → HPA axis ổn định → cortisol giảm → ngưỡng predict HẠ
     = C qua đường "bottom-up" (từ body → brain, khác SSRIs từ brain)
     + Vagal tone cao → parasympathetic ↑ → trạng thái "an toàn"
       → Não DỄ predict hơn (neuroception of safety — Porges)
   Evidence: 🟢 TB — FDA approved cho treatment-resistant (2005)
   Framework giải thích TẠI SAO bottom-up C có thể khác top-down C:
     SSRIs: thay đổi chemistry → predict threshold hạ
     VNS: thay đổi physiological state → cảm giác "an toàn" → predict threshold hạ
     → Có thể bổ sung cho nhau (khác pathway)
   Framework predict: VNS có thể đặc biệt hiệu quả cho SOMATIC-DOMINANT
     (vì tác động qua body → trùng kênh encoding chính)
   Giới hạn: cần phẫu thuật cấy thiết bị, đắt, hiệu quả chậm (months)

⑩ KETAMINE / ESKETAMINE
   Cơ chế: D (reset) — nhanh, mạnh
   Framework:
     Tác dụng trong GIỜ (không phải tuần như SSRIs)
     Có thể: phá suppression pattern TẠM THỜI → cửa sổ re-learn
     NMDA receptor → synaptogenesis nhanh → kết nối mới
     = Không phải "chữa" — TẠO CỬA SỔ để re-learn
   Evidence: 🟢 Mạnh cho treatment-resistant, FDA approved (esketamine)
   Framework predict: hiệu quả GIẢM nếu không kết hợp therapy (B)
     trong cửa sổ sau ketamine
   🟡 Đang nghiên cứu: optimal timing therapy post-ketamine
   Giới hạn: tác dụng tạm (ngày-tuần), cần lặp lại, addiction potential

⑪ PSYCHEDELIC-ASSISTED THERAPY (Psilocybin, LSD)
   Cơ chế: D (reset sâu) — phá toàn bộ prediction pattern
   Framework:
     Psychedelic → neural connectivity TĂNG MẠNH tạm thời
     → Default Mode Network (DMN) DISSOLUTION
     → DMN = nơi chứa self-referential prediction ("mình là ai",
       "mình có giá trị không") — chính là nơi chứa open loop trừu tượng
     → DMN tan → TẤT CẢ prediction pattern (kể cả suppression) bị PHÁ
     → Não ở trạng thái "tabula rasa" tạm thời
     → Khi hồi: BUILD LẠI prediction từ đầu
     → NẾU có therapist hướng dẫn: build pattern MỚI, lành mạnh
     → NẾU không có hướng dẫn: có thể re-build pattern CŨ hoặc TỆ HƠN
   Evidence: 🟡 Rất hứa hẹn, đang Phase 2/3 trials
     Carhart-Harris et al. (2021): psilocybin = escitalopram cho moderate
     FDA breakthrough therapy designation cho psilocybin
   Framework giải thích TẠI SAO cần "integration session" sau trip:
     Reset phá pattern → cần ai đó HƯỚNG DẪN build lại (B)
     Không có integration = reset lãng phí hoặc nguy hiểm
   ⚠️ Framework predict: psychedelic KHÔNG HIỆU QUẢ nếu dùng alone
      (Steve Jobs dùng LSD — nhưng ông đã có purpose rõ ràng,
       = có sẵn prediction pattern mới để build sau reset)
   Giới hạn: chưa legal ở hầu hết nơi, rủi ro psychological,
     cần setting an toàn + therapist kinh nghiệm

⑫ ECT (Electroconvulsive Therapy — Sốc điện)
   Cơ chế: D (reset hardware) — mạnh nhất, cuối cùng
   Framework:
     Kích thích điện → reset toàn bộ neural activity
     → Suppression bị XÓA cùng với nhiều pattern khác
     = "Tắt máy tính bật lại" — mọi thứ reload
   Evidence: 🟢 Mạnh cho treatment-resistant, decades of use
     Remission rate ~50-70% cho treatment-resistant
   Framework predict: relapse rate CAO nếu không maintenance
     Vì: reset xóa suppression nhưng KHÔNG dạy predict mới
     → Gặp cùng open loop → re-learn suppression → tái phát
   🟢 Literature confirm: ECT relapse rate cao without maintenance
   Giới hạn: side effects (mất trí nhớ tạm), stigma, cần gây mê
```

### 7.5 Lộ trình can thiệp — theo phổ

```
╔═══════════════════════════════════════════════════════════════════════╗
║ NGUYÊN TẮC: bắt đầu NHẸ NHẤT, leo thang khi không đủ.             ║
║ Ở MỌI mức: ưu tiên cơ chế B (viết lại) làm NỀN.                   ║
║ Cơ chế khác = HỖ TRỢ để B hoạt động được.                          ║
╚═══════════════════════════════════════════════════════════════════════╝

PHỔ NHẸ ("meh, kệ" — suppress ở 1-2 domain):
  → ① Exercise + ② Giảm input + ③ Micro-predict
  → Nếu đủ: duy trì. Nếu không đủ sau 4-6 tuần: lên mức 2.

PHỔ VỪA ("không thấy ý nghĩa" — suppress nhiều domain):
  → ⑤ Behavioral Activation + ⑥ CBT
  → Kết hợp ①②③ ở mức 1
  → Nếu không đủ sau 8-12 tuần: thêm mức 3.

PHỔ NẶNG ("không thấy gì cả" — suppress gần toàn bộ):
  → ⑧ SSRIs (C: bật predict) + ⑥ CBT (B: dạy predict đúng)
  → Medication TẠO ĐIỀU KIỆN để therapy HOẠT ĐỘNG
  → Kết hợp ①④ khi có thể
  → Nếu treatment-resistant sau 2+ medication trials: lên mức 4.

TREATMENT-RESISTANT:
  → ⑩ Ketamine (cửa sổ nhanh) + CBT trong cửa sổ
  → Hoặc ⑪ Psychedelic-assisted (nếu available) + integration
  → Hoặc ⑫ ECT + maintenance therapy
  → ⚠️ MỌI phương pháp mức 4 CẦN kết hợp B để bền.
```

```
  🟡 Novel prediction — chưa có study chia theo encoding modality

PRED-TX-10: EMDR hiệu quả nhất cho depression có NGUỒN GỐC SỰ KIỆN CỤ THỂ
  Framework: D cục bộ cần TARGET rõ. Depression từ mất mát (§3.3) = có target.
    Depression từ xã hội (§3.1) hoặc identity (§3.2) = không có target rõ → kém.
  🟡 Partially supported: EMDR evidence mạnh cho PTSD+depression comorbid

PRED-TX-11: IPT > CBT cho depression từ đường xã hội (§3.1)
  Framework: IPT rewrite prediction XÃ HỘI trực tiếp. CBT rewrite tổng quát.
    Nếu gốc là open loop xã hội → IPT chính xác hơn.
  🟡 Cần head-to-head study chia theo entry pathway

PRED-TX-12: PURPOSE (mục đích sống) = "natural Behavioral Activation"
  Framework: người có purpose = có prediction direction SẴN
    → Predict → hành động → feedback → match/adjust → predict tiếp
    → Prediction system LUÔN ACTIVE → khó suppress
    = Purpose NGĂN suppression tự nhiên, không cần therapist
  🟢 Partial confirm: logotherapy (Frankl), ikigai research,
    purpose in life ↔ lower depression rates (Kim et al., 2014)

PRED-TX-13: High PE sensitivity = CẦN QUẢN LÝ thay vì CHỮA
  Framework: nếu PE sensitivity là TRAIT (hardware) →
    người high-sensitivity sẽ LUÔN vulnerable với open loop trừu tượng
    → Goal không phải "hết trầm cảm" mà là "quản lý tốt prediction system"
    = Trang bị kỹ năng đóng loop + quản lý drain, không phải loại bỏ trait
  🟡 Consistent with: chronic depression management literature,
    temperament-based approaches (Cloninger)
```

### 9.1 Case 1 — IT worker: Burnout → Depression (PHỔ NHẸ-VỪA)

```
Bối cảnh: Nhân viên IT, làm việc cường độ cao 2+ năm, burnout → nghỉ →
  quay lại → burnout lại → dần "kệ hết".

Framework analysis:
  ① Giai đoạn 1: Open loop nhiều (deadlines, bugs) → cortisol cao → burnout (PFC sụp)
  ② Nghỉ → PFC hồi (hardware repair) → quay lại → CÒN LOOP CŨ → sụp lại
  ③ Sau 2+ burnout cycles: não ENCODE pattern: "hồi cũng vô ích, sẽ sụp lại"
  ④ Chuyển từ SHUTDOWN (hardware) → SUPPRESSION (learned)
  → Burnout → Depression (§5.3 pathway confirmed)

Consistent? ✅
  - Entry: burnout recurrence → learned suppression (§5.3)
  - Đường vào: §3.4 (decision fatigue) + §3.2 (identity: "mình có phù hợp IT?")
  - Phổ: nhẹ-vừa (vẫn làm việc, nhưng "kệ", mất hứng)
  - Can thiệp phù hợp: đổi môi trường (A) + CBT (B) → rewrite "predict vô ích"
```

### 9.2 Case 2 — Postpartum Depression (PHỔ VỪA-NẶNG)

```
Bối cảnh: Phụ nữ sau sinh, hormone thay đổi mạnh, mất ngủ,
  vai trò mới hoàn toàn, cảm giác "không đủ tốt".

Framework analysis:
  ① Hormonal shift → ngưỡng predict THAY ĐỔI (C tự nhiên nhưng ngược — threshold ↑)
  ② Vai trò mẹ = prediction system HOÀN TOÀN MỚI, chưa có pattern
     → Predict "em bé sẽ ngủ" → không ngủ → mismatch
     → Predict "mình sẽ biết phải làm gì" → không biết → mismatch
     → Hàng trăm micro-prediction FAIL đồng thời
  ③ Identity loop: "mình có là mẹ tốt?" → open loop trừu tượng, no endpoint
  ④ Mất ngủ = PFC performance ↓↓ → gần burnout threshold
  ⑤ Xã hội: "mẹ phải hạnh phúc" → predict "mình nên hạnh phúc" → mismatch

  → MULTI-PATHWAY: hormonal (ngoài scope) + prediction pathway (trong scope)
  → Prediction component: §3.2 (identity) + §3.1 (social pressure) + mismatch hàng loạt

Consistent? ✅
  ⚠️ Với caveat: framework giải thích PREDICTION COMPONENT, không giải thích
     hormonal component. Nhưng 2 components TƯƠNG TÁC:
     Hormonal ↑ threshold → prediction fails dễ hơn → suppression nhanh hơn
  - Can thiệp: SSRIs (C — bù hormonal + hạ threshold) + CBT/IPT (B — rewrite)
    + social support (A+B nhẹ) → consistent với clinical guidelines
```

### 9.3 Case 3 — William Styron: Severe Depression (PHỔ NẶNG)

```
Bối cảnh: Nhà văn nổi tiếng, viết "Darkness Visible" (1990).
  Mô tả depression nặng: "không thể cảm nhận gì", "bóng tối hoàn toàn",
  "mọi thứ mất ý nghĩa". Đã thử nhiều phương pháp, cuối cùng hồi qua
  hospitalization + time + medication.

Framework analysis:
  ① Styron: creative identity RẤT MẠNH → open loop "mình còn viết được không?"
     = Loop trừu tượng, identity-level, impossible endpoint
  ② Alcohol withdrawal → biochemical disruption → threshold predict ↑↑
     (= C ngược — hardware barrier tăng đột ngột)
  ③ Mô tả "không cảm nhận gì" = prediction system GẦN TẮT HOÀN TOÀN
     → Phổ nặng: suppress ở hầu hết domain
  ④ Hồi qua: medication (C) + hospitalization (A — thoát context)
     + time (slow re-learning) → dần predict lại

Consistent? ✅
  - Phổ: nặng (gần tắt prediction) — khớp với mô tả "darkness"
  - Entry: §3.2 (identity) + biochemical (alcohol withdrawal)
  - Recovery: C + A + dần B → consistent
  - Framework explain "darkness" = prediction system OFF → không có expectation
    → không có gì "phía trước" → darkness = absence of prediction về tương lai
```

### 9.4 Case 4 — Abraham Lincoln: Chronic Depression + Functioning (PHỔ VỪA MÃN TÍNH)

```
Bối cảnh: Lincoln có "melancholia" suốt đời, nhưng vẫn function ở level CAO.

Framework analysis:
  ① Hardware: có thể PE sensitivity CAO (trait) → dễ suppress
  ② Mất mát nhiều (mẹ, chị, bạn, con) = §3.3 pathway, NHIỀU LẦN
  ③ NHƯNG: Lincoln có PURPOSE CỰC MẠNH (abolition, preserve Union)

  ④ = Phổ vừa MÃN TÍNH nhưng FUNCTIONAL

    Purpose ở 1 domain GIỮCHO prediction system hoạt động ở domain đó
  - Confirm PRED-TX-12: purpose = protective factor
```

### 9.5 Case 5 — Winston Churchill: "Black Dog" (PHỔ NHẸ-VỪA DAO ĐỘNG)
