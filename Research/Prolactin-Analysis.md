## 3. Cơ Chế Chính Xác — Satisfaction Signal

```
LUỒNG:

  1. Body CÓ nhu cầu (body-need active ở 1+ nhóm)
  2. Schema-Drive phục vụ: kích hoạt hành vi đáp ứng nhu cầu
  3. Hành vi → body NHẬN reward (opioid/oxytocin channel fire)
  4. Body EVALUATE: "nhóm NÀY đã đủ chưa?"
     → Chưa đủ: prolactin KHÔNG release → schema TIẾP TỤC phục vụ → loop 2-3
     → ĐỦ RỒI: prolactin RELEASE = "Satisfaction Signal"
  5. Schema DETECT prolactin → "body ĐÃ ĐỦ nhóm này" → TỰ NGUYỆN dừng drive nhóm này
  6. Body: bình an (nhóm này fulfilled) → resource FREED cho nhóm khác (nếu cần)

  ĐẶC BIỆT:
    → Prolactin = PER NHÓM body-need, KHÔNG phải toàn bộ
    → Ăn đủ (Nhóm 1 fulfilled) → prolactin cho Nhóm 1 → schemas ăn DỪNG
    → Nhưng: chưa gặp bạn (Nhóm 2 chưa fulfilled) → schemas connection VẪN drive
    → = SELECTIVE satisfaction signal → per group → không phanh TẤT CẢ cùng lúc

  VÀ: prolactin RELEASE phụ thuộc CÁCH fulfill:
    Multi-channel (E+C đồng thời): prolactin MẠNH → signal RÕ → dừng DỄ
    Single-channel (chỉ dopamine): prolactin YẾU → signal MỜ → dừng KHÓ
    → Vì: multi-channel = opioid + oxytocin → body respond ĐẦY ĐỦ → "thật sự đủ"
    → Single: chỉ dopamine → body: "có gì đó nhưng CHƯA ĐỦ thật" → prolactin ÍT
```

### 3.1 Schema Drive Bằng Gì? — 4 Loại Drive (Không Chỉ Cortisol)

```
⚠️ Giả thiết ban đầu: "cortisol = duy nhất drive schema" → SAI
   Cortisol = MẠNH NHẤT ngắn hạn → nhưng KHÔNG phải duy nhất.

⚠️ Refine quan trọng: SCHEMA = TRIGGER, HORMONE = ENERGY + SUSTAIN
   Drive luôn BẮT ĐẦU từ schema fire (trừ exception baseline lệch).
   Hormone KHÔNG khởi động drive — hormone CUNG CẤP energy + duy trì drive.

   Ví dụ chứng minh: đang đi chơi vui → bỗng nhớ "hôm nay thi!"
     → Schema fire: "thi = quan trọng + trễ = hậu quả"
     → NGAY LẬP TỨC biết phải chạy — cortisol LÚC NÀY VẪN THẤP
     → NE spike (0.5-2s): energy cho hành động tức thì
     → Cortisol peak SAU (5-20 phút): DUY TRÌ alert
     → = Schema trigger TRƯỚC, hormone đến SAU

4 LOẠI SCHEMA DRIVE (trigger) + HORMONE tương ứng (energy/sustain):

  THREAT DRIVE: "CẦN phải làm vì NGUY HIỂM nếu không"
    → Schema predict: nguy hiểm / mất mát / hậu quả xấu
    → Hormone energy: NE + Adrenaline (tức thì) + Cortisol (sustain)
    → 3 nguồn trigger khác nhau:
      Physical threat: đau, nguy hiểm vật lý → reflex + NE + cortisol
      Social threat: bị quát, bị từ chối → schema + NE + cortisol (không endorphin)
      Anticipation: imagine hậu quả tương lai → PFC draft + cortisol NHẸ
    → MẠNH + TỐN (body stress) + ngắn hoặc mãn tính tùy nguồn

  DOPAMINE DRIVE: "MUỐN vì thú vị/mới"
    → Schema predict: có cái mới / improvement / thú vị phía trước
    → Hormone energy: Dopamine (motivation seek + focus)
    → VỪA + BỀN + NHẸ
    → Ví dụ: "cái gì đây?" → schema predict novelty → dopamine → drive explore

  OPIOID DRIVE: "MUỐN vì sướng"
    → Schema predict: trải nghiệm tốt phía trước (ăn ngon, nhạc hay, mastery)
    → Hormone energy: Dopamine (seek) + Opioid khi nhận (enjoy)
    → VỪA + BỀN + TÍCH CỰC
    → Ví dụ: nhớ vị phở → schema predict sướng → drive tìm quán

  OXYTOCIN DRIVE: "MUỐN vì ấm áp"
    → Schema predict: connection / gần gũi phía trước
    → Hormone energy: Oxytocin (pull toward) + Dopamine nhẹ (seek)
    → NHẸ + RẤT BỀN
    → Ví dụ: nhớ bạn → schema predict ấm áp → drive gọi điện

  VÀ: để HÀNH VI thực sự XẢY RA (từ "muốn" → "làm"):
    → CẦN chút ACTIVATION ENERGY (NE + cortisol nhẹ)
    → Nằm "muốn ăn phở" (opioid drive) → nhưng ĐỨNG DẬY đi ăn = cần NE nhẹ
    → = NE là ACTIVATION ENERGY phổ biến nhất (không phải cortisol)
    → Thiếu NE hoàn toàn → "muốn nhưng không làm" → depression pattern

  EXCEPTION — hormone drive KHÔNG CẦN schema:
    → Cortisol baseline CAO mãn tính → body LUÔN alert → anxiety không rõ lý do
    → Dopamine hijack (addiction) → seeking không có schema "muốn" rõ ràng
    → = Khi hormone baseline BỊ LỆCH → hormone CÓ THỂ drive trực tiếp
    → = Bình thường: Schema → Hormone → Behavior
    → = Bất thường: Hormone (lệch) → Behavior (không có schema rõ)

  → LIÊN QUAN PROLACTIN:
    Prolactin fire cho TẤT CẢ 4 loại drive (khi body-need fulfilled)
    KHÔNG chỉ fire cho cortisol drive
    Ăn no (cortisol drive fulfilled) → prolactin ✅
    Hiểu rồi (dopamine drive fulfilled) → prolactin ✅
    Nghe đủ nhạc (opioid drive fulfilled) → prolactin ✅
    Gặp bạn đủ (oxytocin drive fulfilled) → prolactin ✅
```

### 3.2 Prolactin CHỈ Fire Khi Body REQUEST Thật

```
⚠️ INSIGHT QUAN TRỌNG: prolactin KHÔNG fire cho mọi hoạt động:

  Prolactin fire KHI VÀ CHỈ KHI:
    1. Body CÓ nhu cầu THẬT (body-need active — body REQUEST)
    2. Schema serve nhu cầu đó → hành vi fulfill
    3. Body NHẬN reward THẬT (opioid/oxytocin fire — Lớp 1)
    4. Body: "đủ" → prolactin release

  Prolactin KHÔNG fire khi:
    → Body KHÔNG request (habit tự chạy, không từ body-need)
    → Body KHÔNG nhận reward thật (dopamine only, Lớp 1 = 0)
    → = Không có "đủ cho cái gì" → prolactin không có gì để signal

  VÍ DỤ — SCROLL MXH:

    Lần 1 (lần đầu tiên bao giờ):
      → Post hay → opioid THẬT ("hay!") + dopamine ("mới!")
      → Body: reward thật → prolactin CÓ THỂ fire nhẹ
      → Body learn: "scroll = reward" → schema HÌNH THÀNH

    Lần 100 (đã quen):
      → Post → dopamine NHẸ ("mới!" micro) → nhưng opioid GẦN ZERO
      → Vì: đọc post VỀ food ≠ ĂN food → nhìn ≠ trải nghiệm
      → Body: "nhận GÌ thật?" → "KHÔNG" → prolactin KHÔNG fire
      → → Schema tiếp tục drive (habit compiled) + dopamine duy trì loop
      → → KHÔNG CÓ endpoint tự nhiên vì: KHÔNG CÓ "đủ cho cái gì"
      → → "Scroll mãi" = body KHÔNG BAO GIỜ nói "đủ" (vì chưa bao giờ NHẬN gì thật)

  VÍ DỤ — ĂN CÙNG GIA ĐÌNH vs ĂN MỘT MÌNH SCROLL:

    Ăn cùng gia đình:
      Body REQUEST: đói (body-need Nhóm 1) + connection (Nhóm 2)
      Ăn: opioid (ngon) + oxytocin (gia đình) → multi-channel → body reward THẬT
      → Prolactin MẠNH (multi-channel) → "no VÀ ấm" → dừng THOẢI MÁI
      → = Body request → fulfill THẬT → prolactin → satisfied → bình an

    Ăn một mình scroll MXH:
      Body REQUEST: đói (Nhóm 1 only)
      Ăn: opioid (ngon) → single-channel → prolactin YẾU hơn
      + Scroll: dopamine (micro) → opioid ≈ 0 → prolactin = 0 cho scroll
      → "No bụng" (prolactin nhẹ cho ăn) + "VẪN scroll" (prolactin = 0 cho scroll)
      → = Ăn DỪNg → scroll KHÔNG dừng → vì: 2 hoạt động, 2 prolactin KHÁC

  → "KHÔNG DỪNG ĐƯỢC" CÓ 3 LÝ DO KHÁC NHAU:
    A: Body KHÔNG request → không "đủ" cho cái gì → prolactin = 0 (scroll MXH)
       FIX: nhận ra "body KHÔNG cần cái này" → switch sang cái body CẦN
    B: Body request → fulfill → prolactin CÓ → BỊ LẤN (cortisol/dopamine quá mạnh)
       FIX: giảm cortisol/dopamine → prolactin nổi lên → nghe được
    C: Body request → fulfill → prolactin CÓ → schema OVERRIDE ("phải hơn!")
       FIX: fix schema ("đủ là ok" thay "phải hơn")
```

## 7. Hardware Ảnh Hưởng — Prolactin Sensitivity

```
Cùng fulfill → KHÁC prolactin response → vì hardware:

  1. Prolactin RECEPTOR sensitivity (gen):
     → Receptor nhạy: ÍT prolactin → ĐÃ nghe → dừng DỄ → "dễ thỏa mãn"

     → Dopamine SUPPRESS prolactin release (tonic inhibition)
     → Dopamine system ACTIVE (DRD4 7-repeat?): prolactin BỊ suppress MẠNH → signal YẾU
       → = Improviser: dopamine cao → prolactin yếu → khó nghe "đủ" → chán nhanh, chuyển tiếp
     → Dopamine system ÍT active (DRD4 4-repeat): prolactin ĐƯỢC release → signal RÕ
       → = Specialist: dopamine vừa → prolactin ok → nghe "đủ" → ở lại lâu, kiên trì

  3. Multi-channel BOOST prolactin:
     → Multi-channel fulfill → opioid + oxytocin → prolactin AMPLIFIED (vượt suppress)
     → = CÁCH vượt hardware limit: đủ channels → prolactin MẠNH đủ dù receptor kém
     → → Improviser: cần MULTI-CHANNEL hơn specialist để nghe "đủ"
     → → Specialist: single-channel ĐÃ đủ nghe "đủ"

  ⚠️ RESEARCH STATUS:
    Dopamine suppress prolactin: 🟢 proven (Ben-Jonathan 2001)
    DRD4 → dopamine → prolactin link: 🟡 inferred (chưa ai test trực tiếp)
    Multi-channel boost prolactin: 🟢 proven (partnered > solo: Brody 2006)
    Prolactin receptor genetic variants: 🟡 exists nhưng chưa linked rõ tới behavior
```
