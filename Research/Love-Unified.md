    smoothing_capacity ≈ |Valence-Structural depth| — max(Schema_A, competing_schemas)

    IF smoothing_capacity ≈ 0: body OSCILLATES, conflicted
    IF smoothing_capacity < 0: attribute VISIBLE → PFC flags it

      |Valence-Structural depth| = compiled body-base extension depth for that agent
      Schema_A = strength of the attribute-evaluation schema

      → Competing schemas may be CONTEXT-DEPENDENT

    ② Valence-Structural decreases → smoothing lost → attributes SURFACE

    ⑤ Competing schemas → reduce smoothing
