## 1. Nguyên Lý: Mismatch, Không Phải Bug

```
BUG      = code SAI  → output sai ở MỌI môi trường.
MISMATCH = code ĐÚNG → output "sai" vì MÔI TRƯỜNG thay đổi.

Não người: code viết cho savanna (200.000 năm).
Môi trường hiện đại: 250 năm (công nghiệp), 20 năm (MXH).
→ Code KHÔNG KỊP UPDATE cho environment mới.

CAN THIỆP KHÁC NHAU:
  Bug      → sửa code (thuốc, thay đổi phần cứng)
  Mismatch → sửa environment HOẶC cung cấp input đúng spec
  → Phần lớn patterns dưới đây cần SỬA ENVIRONMENT, không cần sửa não.

MỖI PATTERN = 1+ cơ chế cụ thể đang chạy đúng spec nhưng sai context.
Chi tiết evolutionary mismatch cốt lõi: Neurochemistry.md §6.6.
```

## P1: Stress Addiction

```
NHÃN PHỔ BIẾN:  "Workaholic" / "nghiện drama" / "không biết nghỉ"
FRAMEWORK LABEL: Cortisol→DA loop tự duy trì (Cortisol Treadmill as Addiction)
```

```
🟡 CƠ CHẾ:

  Cortisol moderate → GR kích thích VTA → DA TĂNG (Neurochemistry.md §6.4).
  → Cortisol moderate = NGUỒN DOPAMINE GIÁN TIẾP.

  VÒNG LẶP:
  1. Stress/deadline → cortisol moderate → DA↑ → PE ("productive!", "alive!")
  2. Stress qua → cortisol HẠ → DA HẠ → "trống rỗng" (PE deficit)
  3. PE deficit → TÌM stress mới → cortisol↑ → DA↑ → loop
  4. Baseline cortisol DẦN TĂNG (allostatic load, McEwen 1998)
  5. Cần stress MẠNH HƠN cho cùng DA hit = TOLERANCE

  CẤU TRÚC GIỐNG NGHIỆN CHẤT (Koob model, Neurochemistry.md §6.10):
    Chất:   substance → DA spike → withdrawal → seek substance → DA spike
    Stress: deadline → DA spike → calm = "withdrawal" → seek stress → DA spike
    → Cùng allostatic shift, cùng tolerance, cùng cơ chế.

  HAI BIẾN THỂ:
  A. WORK VARIANT: deadline→deadline→deadline. "Đam mê" hay "sợ dừng"?
  B. DRAMA VARIANT: xung đột→hòa giải→xung đột. "Tình yêu mãnh liệt" hay "cortisol hit"?
     Variant B thêm: oxytocin (makeup) + opioid (relief) → 3 PE sources cùng lúc.

  V5.5 LENS — DOPAMINE INVERTED-U (§6.6):
    Stress addiction thường ở VÙNG MODERATE-CAO trên dopamine inverted-U.
    Cortisol moderate → DA tăng → kẹt ở "hơi cao" trên DA curve.
    → Đủ DA để "muốn làm" nhưng phasic > tonic → explore stress mới > deepen work.
    → Nếu không dừng: baseline cortisol leo dần → đổ sang descending limb → burnout.
```

```
THIẾT KẾ GỐC (savanna):
  Moderate cortisol → DA → hành động nhanh khi cần (săn mồi, di cư).
  SAU ĐÓ: recovery tự nhiên (đêm, mùa nghỉ). Chu kỳ tự reset.
  → Thiết kế cho BURST, không cho LIÊN TỤC.

MISMATCH:
  Xã hội hiện đại: deadline 24/7, KPI liên tục, always-on culture.
  → Không có recovery window → chu kỳ KHÔNG RESET → allostatic shift.
  → Xã hội THƯỞNG pattern này ("chăm chỉ", "committed") → không nhận ra là nghiện.

VULNERABILITY (Mô Hình Vuông):
  🟡 Dễ nhất: vùng cạnh trên-phải (external deep + high threshold).
    Threshold cao → PE ngắn không đủ → cần cortisol boost → treadmill.
    External deep (Soldier-Deep) → institution reward "chăm chỉ" → reinforce loop.
  Cũng dễ: cạnh trên-trái (Architect threshold cao) nhưng biểu hiện khác:
    Architect: "project sắp xong, cần thêm chút" → tự tạo deadline nội bộ.
  Ít dễ: cạnh dưới (threshold thấp) → PE ngắn đủ → ít cần cortisol boost.

DỄ NHẦM:
  ✗ "Đam mê thật" (dopamine-driven + cortisol optimal + CÓ recovery)
  ✓ Stress addiction (cortisol-driven + KHÔNG recovery + tolerance tăng)
  TEST: bỏ deadline 2 tuần →
    "Trống rỗng / lo lắng / tạo deadline giả" = stress addiction.
    "Tự do, tự tìm project vì thích" = đam mê thật.

CAN THIỆP:
  1. Nhận diện: "productive feeling" đến từ deadline (cortisol) hay nội dung (dopamine)?
  2. Xây RECOVERY WINDOWS có chủ đích (giống thiết kế gốc)
  3. Tìm PE source dopamine-driven thay thế (không qua cortisol)
  4. Chấp nhận "trống rỗng" ban đầu = withdrawal, KHÔNG PHẢI lười
```

## P2: Self-Harm

```
NHÃN PHỔ BIẾN:  "Tự hủy" / "tìm sự chú ý" / "yếu đuối"
FRAMEWORK LABEL: Triple PE mechanism — opioid + controllability + cycle restoration
```

```
🟡 CƠ CHẾ — 3 hệ thống fire ĐỒNG THỜI:

  MECHANISM 1 — OPIOID RELEASE:
  🟢 Đau vật lý → endorphin (opioid nội sinh) release.
    Cùng cơ chế runner's high: đau → endorphin → relief.
    Người cortisol mãn tính: DA blunted → anhedonia (không vui từ gì cả).
    Nhưng opioid system VẪN HOẠT ĐỘNG (khác pathway).
    → Đau vật lý = MỘT TRONG ÍT CÁCH CÒN LẠI trigger PE giữa numbness.
    🟢 Nock & Prinstein (2004): "automatic positive reinforcement" — tạo cảm giác.

  MECHANISM 2 — CONTROLLABILITY RESTORATION:
  🟡 Chronic uncontrollable stress → vmPFC teo → learned helplessness.
    → "Không kiểm soát được GÌ CẢ trong cuộc sống."
    Self-harm = MỘT THỨ kiểm soát được: khi nào, ở đâu, mạnh bao nhiêu.
    → vmPFC brief activation → controllability signal restored TẠM THỜI.
    → "Đó là thứ DUY NHẤT tôi kiểm soát được" = chính xác controllability theory.
    🟢 Nock & Prinstein (2004): "automatic negative reinforcement" — giảm cảm xúc tiêu cực.

  MECHANISM 3 — CORTISOL CYCLE RESTORATION: 🔴
    Cortisol mãn tính: chu kỳ savanna BỊ PHÁ (Neurochemistry.md §6.6).
    Cortisol cao liên tục, KHÔNG resolution, KHÔNG recovery.
    Self-harm TẠO chu kỳ nhân tạo:
      Cắt/cào (SPIKE cấp tính)
      → Đau (ACTION — cụ thể)
      → Endorphin (RESOLUTION — reward rõ)
      → Calm (RECOVERY — cortisol tạm hạ)
      → Chu kỳ HOÀN CHỈNH mà não thiếu từ lâu.

  → 3 drive equations ĐỒNG THỜI dương trong 1 hành vi
    = TẠI SAO self-harm CỰC KỲ KHÓ BỎ — không phải 1 reward mà 3.
```

```
THIẾT KẾ GỐC:
  Opioid release sau đau = analgesic tự nhiên (bị thương khi chạy → endorphin → tiếp tục).
  Controllability = vmPFC đánh giá "kiểm soát được" → ức chế bất lực (sinh tồn).
  Cortisol cycle = spike → resolution → recovery (thiết kế chuẩn).
  → CẢ 3 mechanism đều ĐÚNG THIẾT KẾ. Không có cái nào "hỏng."

MISMATCH:
  Cortisol mãn tính HIỆN ĐẠI phá chu kỳ + phá controllability + blunt DA.
  → 3 hệ thống bị "thiếu input" cùng lúc.
  → Self-harm = não TỰ TẠO input cho cả 3 bằng cách duy nhất còn biết.
  → KHÔNG PHẢI "tự hủy" — là CỐ TỰ CHỮA bằng shortcut nguy hiểm.

VULNERABILITY (Mô Hình Vuông):
  🟡 Không phụ thuộc mạnh vào vị trí trên phổ — phụ thuộc CORTISOL MÃN TÍNH.
  Bất kỳ vị trí nào + cortisol mãn tính + DA blunted → vulnerable.
  Nhưng threshold CAO + sensitivity CAO → dễ hơn:
    PE từ nguồn bình thường KHÔNG ĐỦ (threshold) + cạn nhanh (sensitivity)
    → opioid pathway trở thành "lối cuối" sớm hơn.
  Vùng cạnh dưới (Drifter): chunks chưa xây → ít PE source thay thế → vulnerable hơn.

DỄ NHẦM:
  ✗ "Tìm sự chú ý" (CÓ THỂ là yếu tố PHỤ, KHÔNG phải cơ chế chính)
  ✗ "Yếu đuối" (judgment, không cơ sở sinh học)
  ✓ Triple PE mechanism dưới chronic cortisol

CAN THIỆP (PHẢI thay thế TỪNG cơ chế):
  Mechanism 1 (opioid): exercise, cold exposure, cắn đá lạnh
    (đau KHÔNG tổn thương nhưng VẪN trigger endorphin)
  Mechanism 2 (controllability): xây lại action path bất kỳ
    (journal, sắp xếp đồ, chọn bài hát — BẤT CỨ GÌ kiểm soát được)
  Mechanism 3 (cycle): tạo acute controllable stress LÀNH MẠNH
    (exercise, box breathing, cold shower = spike → resolution → recovery)
  Nền tảng: HẠ cortisol mãn tính (bước 1 luôn, Core.md §11.3)
  ⚠️ Bỏ self-harm MÀ KHÔNG thay thế 3 PE source = bỏ PE duy nhất còn lại → nguy hiểm hơn.
```

## P3: Trauma Bonding

```
NHÃN PHỔ BIẾN:  "Yêu sai người" / "ngu" / "thích bị đánh"
FRAMEWORK LABEL: Oxytocin × cortisol co-fire + intermittent reinforcement
```

```
🟡 CƠ CHẾ:

  ĐIỀU KIỆN: người GÂY ĐAU + người AN ỦI = CÙNG 1 NGƯỜI.

  VÒNG LẶP:
  1. Abuse phase: đau → cortisol SPIKE → PFC offline → fear
  2. Reconciliation phase: xin lỗi/yêu thương → oxytocin FLOOD → relief
  3. PE = KHỔNG LỒ (uncertainty cực cao → resolution → PE cực lớn)
  4. Oxytocin + cortisol ĐỒNG THỜI → bond CỰC MẠNH + CỰC ĐỘC
  5. Intermittent reinforcement: không biết lúc nào đau, lúc nào yêu
     → Prediction KHÔNG THỂ ổn định → PE liên tục → bond tăng liên tục

  🟢 Variable ratio reinforcement = mạnh nhất (Skinner).
  Slot machine dùng cùng cơ chế: không biết khi nào thắng → không dừng được.

  THÊM — SCHEMA + THRESHOLD LOCK:
  Cycle lặp → schema "yêu = đau" HÌNH THÀNH.
  → Quan hệ SAU bị filter qua schema:
    Người tốt + ổn định = "nhàm" (PE thấp so với threshold đã calibrate CAO bởi drama).
    Người toxic + drama = "intense" (PE match threshold).
  → Threshold BỊ CALIBRATE bởi trauma → cần "liều" tương đương.

  V5.5 LENS — HABITUATION BLINDNESS (§6.7):
    Intermittent reinforcement = CHỐNG habituation hoàn hảo.
    PE KHÔNG BAO GIỜ → 0 (vì prediction không bao giờ ổn định).
    → Đây là lý do trauma bond "mãnh liệt hơn" tình yêu bình thường:
      tình yêu bình thường → predicted → PE → 0 → "hết yêu" (Habituation Blindness).
      trauma bond → KHÔNG BAO GIỜ predicted → PE KHÔNG BAO GIỜ → 0 → "mãnh liệt mãi."
```

```
THIẾT KẾ GỐC:
  Oxytocin bond sau stress CHUNG = survival (bộ lạc gắn bó sau chiến đấu CÙNG NHAU).
  Cortisol + oxytocin co-fire = "đồng đội" bonding (shared adversity).
  🟢 Dutton & Painter (1993): trauma bonding model.
  → Thiết kế cho: CÙNG CHIẾN ĐẤU VỚI threat bên ngoài.

MISMATCH:
  Khi threat đến TỪ CHÍNH người gắn bó → cơ chế "đồng đội" fire SAI ĐỐI TƯỢNG.
  → Bond MẠNH HƠN (vì cortisol cao hơn) thay vì yếu hơn. Ngược trực giác.
  → Savanna: người gây đau = kẻ thù = KHÔNG gắn bó.
     Hiện đại: người gây đau = người yêu/bố mẹ = GẮN BÓ + ĐAU cùng lúc.

VULNERABILITY (Mô Hình Vuông):
  🟡 Dễ nhất: vùng cạnh phải (external source dominant).
    Chunks external → PE phụ thuộc reference (partner) → partner = PE source chính.
    Mất partner = mất PE source → drive âm → quay lại dù toxic.
  Thêm risk: serotonin sensitivity thấp + schema "mình không xứng hơn" (schema bản thân thấp).
  Ít dễ hơn: vùng cạnh trái-trên (internal deep) → PE source đa dạng + internal
    → partner = 1 source trong nhiều, mất = đau nhưng KHÔNG identity collapse.

DỄ NHẦM:
  ✗ "Yêu thật sự nhưng khó khăn" (bond = fear + oxytocin, không phải tình yêu lành mạnh)
  ✗ "Thích bị đánh" (KHÔNG thích đau — thích RELIEF SAU đau = opioid + oxytocin)
  ✗ Love Addiction Type A (hardware-driven, không cần abuse cycle)

CAN THIỆP:
  1. TÁCH khỏi cycle (cắt intermittent reinforcement — khó nhất nhưng quan trọng nhất)
  2. Hạ cortisol nền → PFC online → NHẬN RA pattern
  3. Sửa schema "yêu = đau" (CBT, schema therapy)
  4. Hạ threshold (đã bị calibrate cao bởi drama) → dopamine fasting khỏi drama
  5. Xây oxytocin source MỚI (an toàn): bạn bè, nhóm hỗ trợ, pet
```

## P4: Love Addiction — 3 Types

```
NHÃN PHỔ BIẾN:  "Lụy tình" / "không sống thiếu người yêu"
FRAMEWORK LABEL: 3 cơ chế khác nhau, cùng biểu hiện bề mặt

⚠️ "Lụy tình" BỀ MẶT giống nhau nhưng CƠ CHẾ khác → CAN THIỆP khác.
   Nhầm type = can thiệp sai = không hiệu quả.
```

```
🔴 TYPE A — HARDWARE-DRIVEN (Oxytocin Dominant):

  Kênh gốc: Oxytocin = nguồn PE CHÍNH.
  + Prolactin thấp (không phanh) + vasopressin cao (bảo vệ mạnh).
  Gen: OXTR nhạy + AVPR1A variants.

  Cơ chế: PE LỚN NHẤT trong đời = từ kết nối. Mất = mất PE source chính.
  Biểu hiện: yêu mãnh liệt BẤT KỂ AI (pattern lặp lại mọi mối quan hệ).
  ≠ bệnh lý NẾU prolactin bình thường. = vấn đề NẾU prolactin thấp (không dừng được).

  Thiết kế gốc: pair bonding MẠNH = nuôi con TỐT HƠN → survival.
  Mismatch: xã hội nhiều chia tay → hệ thống thiết kế cho BOND LÂU DÀI
    bị trigger nhiều lần → mỗi lần mất = đau như thiết kế (cực mạnh).

  Can thiệp: KHÔNG CẦN sửa hardware. Cần: đa dạng PE source (đừng all-in oxytocin),
    adjust prolactin (thiền, exercise) nếu cần, chọn partner phù hợp.
```

```
🔴 TYPE B — ANXIOUS ATTACHMENT (Cortisol-Driven):

  Attachment style: anxious (bối cảnh sớm: bố mẹ không nhất quán).
  Oxytocin system HYPERACTIVE — luôn tìm kiếm, KHÔNG BAO GIỜ "đủ."

  Cơ chế: cortisol + oxytocin ĐỒNG THỜI (giống trauma bonding nhẹ).
    "Lụy" = KHÔNG PHẢI yêu nhiều — mà SỢ MẤT nhiều.
    Yêu = cortisol-driven (tránh mất) > dopamine-driven (hướng kết nối).
  Biểu hiện: lo lắng liên tục, kiểm tra, cần reassurance không ngừng.

  Thiết kế gốc: trẻ cần GẮN BÓ để sống. Bố mẹ không nhất quán
    → tăng cường tìm kiếm gắn bó = ADAPTIVE trong savanna (kiên trì → mẹ phản hồi).
  Mismatch: người lớn VẪN chạy "tìm kiếm mẹ" program → áp lên partner.

  Can thiệp: attachment therapy, sửa schema "mình không đủ tốt",
    HẠ CORTISOL TRƯỚC → anxiety giảm → oxytocin system bớt hyperactive.
```

```
🔴 TYPE C — SCHEMA OVERRIDE:

  Schema: "giá trị bản thân = được yêu." Identity GẮN vào relationship.
  Oxytocin có thể BÌNH THƯỜNG — vấn đề ở SCHEMA, không phải hardware hay attachment.

  Cơ chế: mất relationship = mất IDENTITY (schema override, Core.md §7.4).
    Giống identity fusion (Swann et al., 2012) nhưng với partner thay vì nhóm.
    Drive equation: reward từ relationship = identity maintenance → cost = INFINITE nếu mất.
  Biểu hiện: "không biết mình là ai nếu không yêu ai."

  Thiết kế gốc: identity schemas = adaptive (biết mình là ai → predict tốt hơn).
  Mismatch: identity QUÁ HẸP (chỉ 1 nguồn) → mất nguồn đó = identity collapse.

  Can thiệp: xây identity rộng hơn (nhiều domain), sửa schema "giá trị = được yêu"
    → KHÔNG cần động vào oxytocin hay attachment.
```

```
BẢNG PHÂN BIỆT 3 TYPE:

              │ Cơ chế chính  │ Tầng       │ Test nhanh                       │
──────────────┼───────────────┼────────────┼──────────────────────────────────┤
Type A (hw)   │ Oxytocin gen  │ Tầng 0     │ Yêu AI cũng intense (mọi mối QH)│
Type B (att)  │ Cortisol/att  │ Tầng -1, 1 │ Lo SỢ MẤT > vui khi CÓ          │
Type C (sch)  │ Schema        │ Tầng 1     │ "Không biết mình là ai" khi lẻ   │

→ Cùng "lụy tình" bề mặt, 3 can thiệp HOÀN TOÀN KHÁC.
  Type A: đa dạng PE source + adjust prolactin.
  Type B: attachment therapy + hạ cortisol.
  Type C: mở rộng identity + sửa schema.

  → Trừng phạt bất công = PE THẬT (dopamine từ "công lý được thực thi").

  UNRESOLVED PREDICTION:
  Bị bất công → prediction model bị BREAK ("thế giới công bằng" → SAI).
  → Model cần UPDATE: hoặc "thế giới KHÔNG công bằng" (khó) hoặc "SỬA lại cho công bằng."
  → Não CHỌN "sửa" (trừng phạt/trả thù) vì dễ hơn update worldview.
  → Khi CHƯA trả thù: prediction VẪN unresolved → rumination (giống Habituation Blindness §6.7
    nhưng NGƯỢC: prediction KHÔNG update vì KHÔNG chấp nhận, không phải vì habituation).

  VASOPRESSIN + CORTISOL LOOP:
  Vasopressin: "bảo vệ cái MÌnh coi là đúng" (Core.md §4).
  Cortisol: bất công = threat → cortisol → "wanting" trừng phạt.
  → Vasopressin DUY TRÌ mục tiêu (không quên). Cortisol DUY TRÌ drive (muốn hành động).
  → Kết hợp: "nhớ mãi + muốn trả mãi" = grudge.
```

```
THIẾT KẾ GỐC (savanna):
  Nhóm nhỏ (50-150): kẻ gian lận = threat cho HỢP TÁC NHÓM.
  → Trừng phạt kẻ gian = duy trì reciprocal altruism = nhóm SỐNG.
  → Cost: trừng phạt = effort + risk → CHỈ KHI đáng (sẽ gặp lại, cần hợp tác tiếp).
  → Grudge RESOLVE: trừng phạt xong → kẻ gian sửa → prediction update → xong.
    HOẶC: kẻ gian bị đuổi → threat removed → xong.

MISMATCH:
  Hiện đại: bị bất công bởi người KHÔNG GẶP LẠI (online, institution, xa lạ).
  → Trừng phạt KHÔNG KHẢ THI (không gặp, không power, hệ thống bảo vệ kẻ gian).
  → Prediction KHÔNG THỂ resolve (vì action path bị chặn).
  → Nhưng vasopressin + cortisol VẪN loop → ruminate → grudge VÔ HẠN.
  → Thiết kế cho "giải quyết trong nhóm nhỏ" bị chạy "giữ mãi vì không giải quyết được."

VULNERABILITY (Mô Hình Vuông):
  🟡 Dễ nhất: vasopressin CAO + schema "công bằng" mạnh + internal deep.
    Vasopressin cao: không buông (hardware).
    Internal deep: "hệ thống đúng/sai" riêng → vi phạm hệ thống = PE âm CỰC MẠNH.
    Schema "thế giới phải công bằng" → mỗi bất công = prediction error lớn.
  Ít dễ: vasopressin thấp + schema "thế giới không công bằng, bình thường."
    → Bất công = expected → PE nhỏ → không giữ lâu.

DỄ NHẦM:
  ✗ "Đòi công lý" (CÓ THỂ đúng NẾU dẫn đến action resolve. Grudge thuần = PE loop)
  ✗ "Nhỏ nhen" (judgment. Cơ chế = fairness enforcement + unresolved prediction)
  ✗ Outrage Loop P6 (outrage = phản ứng TỨC THÌ với sự kiện; grudge = DÀI HẠN với đối tượng)

CAN THIỆP:
  1. Resolve prediction: chấp nhận "thế giới không hoàn toàn công bằng" → update model
     (KHÓ nhưng duy nhất CẮT ĐƯỢC loop)
  2. Action path: nếu CÓ THỂ hành động → hành động → resolve → xong
  3. Reframe: "giữ grudge = CHO kẻ gian THUÊ não mình miễn phí"
     → PFC reframe cost → giảm drive giữ
  4. Giảm cortisol: bất công cũ → cortisol RESIDUAL → hạ cortisol → giảm "wanting" trả thù
  5. Chuyển energy: từ "trả thù" → "xây dựng" (PE từ deepen domain → thay thế PE từ revenge)
```

## P11: Hoarding

```
NHÃN PHỔ BIẾN:  "Đồ đạc bừa bãi" / "tham lam" / "bệnh tích trữ"
FRAMEWORK LABEL: Scarcity prediction + opioid comfort + loss aversion mismatch
```

```
🟡 CƠ CHẾ:

  SCARCITY PREDICTION:
  Brain predict "tương lai THIẾU" → dự trữ = GIẢM uncertainty → cortisol HẠ.
  Schema thế giới "khan hiếm / không an toàn" (thường từ childhood poverty, deprivation).
  → Mỗi vật sở hữu = 1 prediction chunk "mình sẽ ok nếu có cái này."
  → VỨT = xóa prediction chunk = TĂNG uncertainty = cortisol SPIKE.

  OPIOID COMFORT:
  Sở hữu + chạm/thấy đồ → opioid nhẹ (comfort, quen thuộc, "mình có").
  Giống emotional eating (P5): opioid system dùng làm THUỐC GIẢM ĐAU cho cortisol.
  Nhưng thay vì ăn → sở hữu.

  LOSS AVERSION AMPLIFIED:
  🟢 Kahneman & Tversky (1979): mất đau GẤP ĐÔI được vui.
  Mỗi đồ vật = chunk prediction đã có → vứt = prediction BREAK → PE âm.
  → Cost vứt > benefit giữ (trong model của người hoarding).
  → Tích trữ = DRIVE EQUATION DƯƠNG: reward (giảm uncertainty + opioid) − cost (không gian) > 0.
  → Chỉ khi cost VƯỢT threshold (hết chỗ, sức khỏe, bị ép) → conflict.
```

```
THIẾT KẾ GỐC (savanna):
  Tích trữ thức ăn, công cụ, vật liệu = SURVIVAL trong môi trường KHAN HIẾM.
  → Sóc tích trữ hạt = cùng cơ chế: uncertainty mùa đông → dự trữ → sống.
  → Savanna: dự trữ GIỚI HẠN (mang theo khi di cư = cost vật lý cao → tự giới hạn).

MISMATCH:
  Hiện đại: vật dụng GIÁ RẺ + KHÔNG cần mang theo (nhà cố định) + mua dễ.
  → Cost tích thêm = GẦN ZERO (đặt hàng 1 click). Cost vứt = PE âm CỰC MẠNH.
  → Hệ thống "dự trữ" KHÔNG CÓ PHANH TỰ NHIÊN (savanna có: mang nặng → tự dừng).
  → Schema "khan hiếm" KHÔNG UPDATE vì thế giới VẪN CÓ uncertainty (dù dồi dào).
  → "Biết đâu cần" = prediction hợp lý theo brain, KHÔNG HỢP LÝ theo thực tế.

VULNERABILITY (Mô Hình Vuông):
  🟡 Dễ nhất: cortisol mãn tính + opioid primary + schema "thế giới khan hiếm."
    Cortisol mãn tính → scarcity signal LIÊN TỤC.
    Opioid primary → sở hữu = PE kênh gốc match → double reward.
    Schema từ childhood poverty/deprivation → prediction "sẽ thiếu" CỰC MẠNH.
  Vị trí trên phổ: không rõ ràng per-pattern — phụ thuộc schema + hardware hơn position.
  Nhưng: depth CAO trong "domain sở hữu" → chunks prediction về đồ vật CỰC SÂU
    → vứt = break NHIỀU chunks → đau hơn.

DỄ NHẦM:
  ✗ "Sưu tầm" (sưu tầm = curated, có hệ thống, PE từ domain depth.
     Hoarding = KHÔNG curated, PE từ giảm uncertainty. TEST: vứt 1 món → anxiety hay không?)
  ✗ "Tham lam" (judgment. Không muốn THÊM — sợ MẤT. Loss aversion ≠ greed)
  ✗ "Lười dọn" (CÓ THỂ yếu tố phụ. Nhưng cơ chế gốc = cortisol + loss aversion)

CAN THIỆP:
  1. Hạ cortisol TRƯỚC → giảm scarcity signal → giảm drive tích trữ TỪ GỐC
  2. Sửa schema: "tương lai = đủ" (exposure: vứt nhỏ → không chết → update prediction)
  3. Opioid thay thế: PE từ trải nghiệm (không phải sở hữu) — exercise, nature, social
  4. Gradual exposure: bắt đầu từ đồ THẤP attachment → brain encode "vứt = ok" → scale lên
  5. Reframe: "giữ = cost (không gian, stress, hỗn loạn)" → PFC add cost vào equation
  ⚠️ ÉP vứt nhanh = cortisol SPIKE CỰC MẠNH → phản tác dụng. Gradual LUÔN.
```
