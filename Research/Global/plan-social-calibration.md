### §2 — 7 FUNCTIONS CALIBRATION (chi tiết từng function)

```
NỘI DUNG CẦN TRIỂN KHAI:

  Mỗi function = 1 sub-section với:
    - FUNCTION: calibrate cái gì
    - MECHANISM: framework mapping
    - TRADITIONAL: xã hội cũ provide thế nào
    - MODERN: đang vỡ thế nào
    - EVIDENCE: data/research

  ─────────────────────────────────────────────
  ① DIRECTION — "Tôi nên đi đâu?"
     → Traditional: nghề gia đình, con đường rõ ràng, guild/caste/role
     → Mechanism: Imagine-Final pre-installed (Loại C trust-compile)
     → Modern breakdown: infinite options → paradox of choice
     → 🟢 Evidence: Shepard 2023 (guilds = identity lifecycle)
     → 🟢 Evidence: De la Croix et al. 2017 (apprenticeship institutions)

  ② PUSH (threat) — "Nếu không làm thì sao?"
     → Traditional: đói, boss, deadline, xã hội phán xét, mùa vụ
     → Mechanism: External threat → cortisol → drive (Self-Created-Threat §5)
     → Modern breakdown: safety net tốt, AI automate, external threat GIẢM
     → Evidence: "quiet quitting", "lying flat" (躺平), NEET rates

  ③ PULL (reward) — "Làm thì được gì?"
     → Traditional: lương, quán bia, lễ hội, cộng đồng acknowledge
     → Mechanism: Social reward + body-base reward (opioid per domain)
     → Modern breakdown: micro-novelty thay thế (TikTok, games, dopamine scroll)
     → Evidence: attention economy, reward substitution

  ④ REPAIR — "Khi nào nghỉ?"
     → Traditional: weekend, Sabbath, lễ hội, mùa nghỉ, về quê
     → Mechanism: cortisol reset + connection + body-input replenishment
     → Modern breakdown: always-on, no forced rest, blurred boundaries
     → 🟢 Evidence: circadian disruption (Cho et al. 2018)
     → 🟢 Evidence: shift work → cancer, psychiatric conditions (Walker & Davis 2023)

  ⑤ CHECK — "Tôi đang đúng hướng không?"
     → Traditional: xã hội feedback tự nhiên (hàng xóm, cộng đồng, elders)
     → Mechanism: domain feedback qua social agents (Connection.md)
     → Modern breakdown: echo chambers, AI confirm, anonymous urban life
     → 🟢 Evidence: Putnam 2000 — trust 58%→37%, civic org decline

  ⑥ IDENTITY — "Tôi là ai?"
     → Traditional: rites of passage, guild membership, family role, religious identity
     → Mechanism: Anchor-Schema compile (Meaning.md v2.0), Loại C trust-install
     → Modern breakdown: identity fluidity, no structured transitions
     → 🟢 Evidence: Van Gennep 1909 (3-phase rites: separation→liminality→incorporation)
     → 🟢 Evidence: Turner 1967 (liminality = identity reconstruction)

  ⑦ CONNECTION — "Tôi thuộc về đâu?"
     → Traditional: tribe ~150 (Dunbar), extended family, village, religious community
     → Mechanism: Connection 3 Generative Primitives, co-regulation
     → Modern breakdown: nuclear family, urban anonymity, digital replacement
     → 🟢 Evidence: Dunbar 1993 (group size ~150, hierarchical layers 5/15/50/150)
     → 🟢 Evidence: Holt-Lunstad 2010 (social relationships +50% survival)
     → 🟢 Evidence: US Surgeon General 2023 (loneliness = 15 cigarettes/day)
```

### §3 — HISTORICAL EVOLUTION: 3 ÊRAs

```
NỘI DUNG CẦN TRIỂN KHAI:

  ① HUNTER-GATHERER (~300,000 years):
     → 7 functions TỰ NHIÊN nhất (all built-in)
     → Group size 30-150 → natural check + connection
     → Daily walking 8-15km → body-input optimal
     → Roles clear → direction set by observation
     → Seasonal cycles → natural repair
     → = Body-Input-Enumeration.md ancestral baseline
     → = "Hệ thống calibration GỐC mà body calibrated FOR"

  ② AGRICULTURAL (~10,000 years):
     → Functions SCALE UP nhưng vẫn intact
     → Village replaced tribe → 150 rule still holds locally
     → Religion formalize 7 functions:
       → Direction (faith path), Push (sin/punishment), Pull (salvation/community),
       → Repair (Sabbath/festivals), Check (confession/elders),
       → Identity (baptism/confirmation), Connection (congregation)
     → 🟢 Religion.md v2.2: 7 functions = melody technology cover 7/7
     → Guilds formalize Direction + Identity (apprentice→journeyman→master)
     → = Culture CODIFY calibration → transmit qua thế hệ = Tier 3 calibration

  ③ INDUSTRIAL (~250 years):
     → Functions START TO BREAK:
     → Urban migration → Connection THIN (village→city, tribe→strangers)
     → Factory schedule → Repair EXTERNALLY IMPOSED (weekend invented 1908)
     → Mass education → Direction STANDARDIZED (one-size-fits-all)
     → But STILL FUNCTIONAL: company loyalty, unions, churches, community orgs
     → = Calibration system STRAINED but HOLDING

  ④ INFORMATION/AI ERA (~30 years → now):
     → Functions BREAKING SIMULTANEOUSLY:
     → All 7 under pressure AT ONCE (not sequential)
     → Tipping point: first time MULTIPLE functions fail ĐỒNG THỜI
     → = §5 deep analysis
```
