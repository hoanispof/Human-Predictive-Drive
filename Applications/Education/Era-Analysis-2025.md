## 2. CÁI ĐANG THAY ĐỔI TRONG NHU CẦU SKILLS

```
🟡 KNOWLEDGE ACCESS THAY ĐỔI:

  TRƯỚC (pre-AI):
    → "Biết" = giá trị → vì ACCESS khó (sách, trường, expert)
    → Memorization = useful → vì "nhớ" = nhanh hơn "tìm"
    → Teacher = knowledge source → vì teacher BIẾT, student chưa

  BÂY GIỜ (AI era):
    → "Biết" ≠ giá trị → vì ACCESS gần instant (AI trả lời trong giây)
    → "Biết HỎI đúng" = giá trị → vì AI cần ĐÚNG CÂU HỎI để cho ĐÚNG CÂU TRẢ LỜI
    → "Biết ĐÁNH GIÁ" = giá trị → vì AI có thể SAI (hallucination, bias)
    → Teacher ≠ knowledge source → AI biết NHIỀU hơn teacher
      → Teacher = CALIBRATOR, MENTOR (ref: Education-System.md §7)

  CHUNK THRESHOLD GIẢM ~4x 🟡:
    (ref: backup Education-AI-Era-Draft.md §1)
    → Trước: vào flow trong 1 domain = cần TÍCH LŨY đủ chunks (years)
    → Bây giờ: AI cung cấp ~80% context chunks ON-DEMAND
    → Người cần: ~20% foundation chunks + biết hỏi + biết evaluate
    → = Threshold vào flow GIẢM → accessible cho NHIỀU người hơn
    → = "Biết hỏi" + "biết feel" (body-base check) > "biết nhiều"
    → ⚠️ Nhưng: 20% foundation VẪN CẦN human build (NL2 confirmed)


🟡 JOB MARKET THAY ĐỔI:

  AUTOMATION tăng:
    → Routine cognitive tasks: AI làm được (data entry, analysis, report)
    → Routine physical tasks: robotics tiến bộ (nhưng chậm hơn cognitive AI)
    → = Nhiều việc "trước cần người" → AI/robot làm được

  HUMAN-VALUE skills tăng giá trị:
    → Creativity: tạo cái MỚI THẬT SỰ (AI = remix, human = truly novel)
    → Judgment: quyết định trong uncertainty, ethics, trade-offs
    → Empathy: human connection, care, emotional support
    → Somatic skills: surgery, craft, physical care, sports, performance
    → Leadership: lead humans through change, inspire, align
    → = Cái AI KHÔNG LÀM ĐƯỢC (hiện tại) = human value

  HYBRID skills mới:
    → Human + AI collaboration = new skill type
    → "Prompt engineering", "AI-augmented research", "AI-assisted creation"
    → = Biết DÙNG AI effectively = competitive advantage
    → ⚠️ Nhưng: specific AI tools = thay đổi nhanh → meta-skill (learn to use new tools) > specific tool skill


🟡 LEARNING THAY ĐỔI:

  AI TUTOR — potential per-individual at scale:
    → NL3 (per-hardware calibration) = historically EXPENSIVE (1-on-1 human tutor)
    → AI tutor: per-pace, per-level, instant feedback, 24/7 available
    → = NL3 AT SCALE cho first time in history
    → ⚠️ Nhưng: AI = no emotion reading, no body-base sensing, no social context
    → = AI tutor + human teacher = COMPLEMENT (ref: Education-System §9)

  SELF-LEARNING resources unlimited:
    → YouTube, courses, tutorials, AI explanations = FREE, per-pace
    → = "Curation" = new foundation skill → lọc, đánh giá, chọn sources
    → = Self-directed learning CÓ THỂ ở scale → nhưng CẦN meta-learning (NL7)

  KNOWLEDGE OBSOLESCENCE nhanh hơn:
    → Specific knowledge = outdated NHANH (coding languages, tools, methods)
    → = Adaptability càng quan trọng (NL7 confirmed by era)
    → = "Learn to learn" > "learn specific content"

  ATTENTION / INFORMATION OVERLOAD:
    → Unlimited content → brain KHÔNG designed cho unlimited input
    → Social media + AI content = dopamine loop risk → attention fragmentation
    → = Recovery, sleep, offline time = QUAN TRỌNG HƠN (NL6)
    → = Education must ALSO teach: how to MANAGE information, not just consume
```

     → 2010: "learn to code" = future-proof → 2025: AI codes → "learn to code" ≠ enough

     → = STRENGTHEN case cho NL7 (adaptability > specific knowledge)
     → = Build người CÓ THỂ learn bất kỳ job mới → không train cho 1 job cụ thể
     🔴 Prediction confidence: RẤT THẤP

     → Children growing up WITH AI + unlimited screen content
     → Impact on: attention span, social development, body-base, sleep?
     → 🟢 Some data: excessive screen time → attention issues (correlation)
     → 🔴 Long-term impact of AI companion during childhood = UNKNOWN
     → = Possibly THE most important unknown for education
     → = Precautionary: moderate screen time, preserve human interaction
       (may be WRONG — but safer than "unlimited AI from age 3")
     🔴 Prediction confidence: THẤP — research đang chạy

  ④ EDUCATION TECHNOLOGY — AI TUTOR: REPLACE? SUPPLEMENT?
     → AI tutor CÓ THỂ: per-pace, per-level, instant feedback, 24/7
     → AI tutor KHÔNG THỂ (hiện tại): read emotion, sense body, social context
     → = Current: SUPPLEMENT (AI + human teacher > either alone)
     → = Future: AI thêm emotion reading? AI thêm social? → UNKNOWN
     → Nếu AI + emotion + social → teacher role THAY ĐỔI lần nữa
     → = Plan for "AI as supplement" → nhưng READY cho "AI role expanding"
     🔴 Prediction confidence: TRUNG BÌNH (near-term) → THẤP (5+ years)

  ⑤ GEOPOLITICAL — FRAGMENTATION?
     → Current: knowledge = somewhat global (internet, English-dominant)
     → Trend: "data sovereignty", AI regulation per country, tech wars
     → = Knowledge systems CÓ THỂ fragment → "internet" → "internets" per region
     → Impact on education: access, language, cultural isolation
     → = Uncertain → but: foundation + adaptability = ROBUST against fragmentation
     🔴 Prediction confidence: THẤP

  ⑥ SOCIETAL — INEQUALITY GAP
     → AI era = CÓ THỂ DEMOCRATIZE (AI tutor for all)
     → HOẶC: AI era = WIDEN gap (rich access AI early → compound advantage)
     → Digital divide: urban + wealthy + connected ≠ rural + poor + disconnected
     → Education implications: per-individual at scale = ONLY IF access = universal
     → = Access = PREREQUISITE → technology ≠ solution nếu thiếu access
     🔴 Prediction confidence: TRUNG BÌNH
