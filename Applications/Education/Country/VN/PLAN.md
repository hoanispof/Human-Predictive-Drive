## FILE 2: VN-Cultural-Factors.md

```
MỤC ĐÍCH:
  Đặc tính văn hóa VN ảnh hưởng education (positive + negative)
  = "Cùng 10 NL, cách apply ở VN KHÁC vì văn hóa KHÁC"

RESEARCH:
  → Ít cần web search (cultural observation + framework analysis)
  → NHƯNG: có thể search VN education research papers
  → KEY: creator (Vietnamese) review cultural accuracy

VĂN HÓA VN CẦN PHÂN TÍCH (initial list — có thể thêm/bớt):

  ① "Học để đổi đời" / "Học tài thi phận"
     → Imagine-Final phổ biến NHẤT ở VN
     → Strength: DRIVE mạnh → nhưng NARROW (chỉ = "thi đỗ + việc tốt")
     → Risk: Forced Soldier (hardware ≠ "đổi đời" direction)

  ② Filial piety / Hiếu thảo
     → Bridge type: social + identity + guilt
     → "Con phải giỏi vì bố mẹ vất vả" = MẠNH nhưng = guilt bridge
     → Risk: cortisol chronic + Forced Soldier

  ③ "Thi đua" / Competitive culture
     → Ranking, xếp hạng, so sánh = EMBEDDED
     → Cortisol source mạnh → NL6 risk
     → PISA: competitive → scores cao → nhưng wellbeing?

  ④ "Thầy nói trò nghe" / Authority-based
     → Teacher = authority → lecture-dominant
     → Risk: suppress meta-learning (NL7), suppress creativity (NL6)
     → Nhưng: structure cho external-tendency students (positive side)

  ⑤ Gia đình mở rộng / Family network
     → Support network MẠNH → emotional safety net
     → Nhưng: nhiều "advisors" = nhiều áp lực = cortisol source
     → Double-bind risk: gia đình + trường ÉP cùng hướng

  ⑥ "Biết đủ" vs "Vươn lên"
     → Cultural tension → ảnh hưởng Imagine-Final scope
     → Rural: "biết đủ" → limited Imagine-Final
     → Urban: "vươn lên" → sometimes excessive pressure

  ⑦ Học thêm culture (private tutoring)
     → VN đặc thù: học thêm = norm, not exception
     → = THÊM ECP ngoài school → cortisol tích lũy
     → = Nhưng cũng = per-individual bổ sung (nếu đúng method)

  ⑧ "Bằng cấp" / Credential culture
     → Bằng = proxy cho ability → system REWARD bằng, không depth
     → Risk: surface assessment entrenched (NL8 violate)

OUTLINE:
  §0 — Tại sao văn hóa quan trọng cho education
  §1 — 8 cultural factors (list trên) — phân tích mỗi factor
  §2 — Map: cultural factor × NL → positive / negative / neutral (TABLE)
  §3 — Leverage points: văn hóa NÀO là STRENGTH có thể dùng
  §4 — Risk points: văn hóa NÀO đang CẢN TRỞ
  §5 — Honest Assessment (outsider bias, stereotype risk)

PHASES:
  Phase A: §0 + §1 (purpose + 8 cultural factors analysis)
  Phase B: §2 + §3 + §4 + §5 (map + leverage + risk + honest)

Target: ~300-500 dòng
Confidence: 🟡 (cultural observation + framework analysis)
⚠️ Cần creator review cultural accuracy
```
