# Curriculum Framework — What to Teach, When, and For Whom v7.8

> **Hardware = ENGINE — always running, source of ALL drives.**
> **Collective-Arc = ROAD — infrastructure for individual fill drives.**
> **Compilation = VEHICLE — compiled chains the individual builds over 15-20 years.**
>
> **Curriculum = ROAD MAP — decides which ROAD to build, at what DEPTH.**
> **Domain-Knowledge-Map (WHAT) + Mechanism (HOW) + Connection-Education (WHO) → this file = DELIVERY PLAN.**
> **Letting ENGINE (student) know which ROADs exist → helping them build the right VEHICLE.**
>
> **Good ROAD = many directions, per-ENGINE calibrated, deep enough for VEHICLE to run.**
> **Bad ROAD = 1 direction, forcing every ENGINE to build the same VEHICLE on the same ROAD.**
>
> **Foundation Matrix (§2) = UNIVERSAL ROAD — every ENGINE needs it. Durable for decades.**
> **Era-specific (§3) = ERA-SPECIFIC ROAD — update when the era changes.**
> **Per-hardware (§4) = VEHICLE CUSTOMIZATION — varies per individual.**
>
> **"WHAT to teach changes every era. WHEN to teach is more stable — because the brain is stable."**

    → Higher Self-Observation Level → better self-directed learning → faster in ALL domains

    → Together: knowing your LIMITS (PFC Budget) + knowing your STATE (Self-Observation)
    → Self-Observation = TOOL, NOT VIRTUE (Self-Observation.md v1.0 §9)

    Per-stage DEPTH TARGET for Self-Observation capacity:

        → "Are you sad right now?" = naming FOR the child → builds Level 1→2
        → Target: child CAN communicate basic states to teacher
        → Curriculum implication: NO explicit training → SCAFFOLD through daily interaction

      Stage 2 (6-12y): Level 2→3 — Body-Predict

        → Curriculum implication: INTEGRATE into ALL domains (not standalone "mindfulness hour")

          Self-Observation capacity INCREASES + regulation capacity NOT YET SUFFICIENT = frustration
        → Reward OBSERVATION, not REGULATION:
          "I notice I often worry before exams" = SUCCESS (Level 4 observation)
          "But still worry" = NORMAL (regulation ≠ observation)
          Punishing for noticing-but-not-controlling → KILLS observation drive
        → Target: student CAN analyze own learning patterns across domains
        → Meta-learning × Self-Observation convergence (§2 ⑤ + cross-cutting)

      Stage 4 (18+): Level 4→5 — Meta-Observe
