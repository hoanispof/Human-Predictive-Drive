tổng hợp từ 3 File-Index.md để tạo file global-index.json tổng hợp.
global-index.json chứa toàn bộ danh sách các file: [item1]tên file, [item2]nhóm chuyên gia tương ứng, [item3]mô tả ngắn, [item4]mô tả đầy đủ([What] + [mô tả đầy đủ chi tiết đơn giản, giúp người đọc biết sơ lược file sẽ trình bày về cái gì(nội dung trình bày như 1 bài viết ngắn gọn đầy đủ, đẹp trong HTML frame)]

như vậy. file global-index sẽ tổng hợp 4 lượt:
lượt 1: tổng hợp danh sách [item1] file-name
lượt 2: đọc từng file thật kỹ, tạo tóm tắt gọn nhẹ để mọi người hiểu được ý nghĩa toàn bộ file, thành [item4]
lượt 3: tổng hợp [item2] "chuyên ngành tương ứng" theo từng file
Core(thuộc core human architecture), N/A, psychology, neuroscience, behaviromental, human-development,.... <= thống kê danh sách này trước đã nhỉ
lượt 4: tạo [item3] mô tả ngắn gọn.

như vậy. chúng ta sẽ có file global-index.json thông tin toàn diện về framework.
Sau đó sẽ tạo file HTML-File-Viewer.html. để trình bày mọi nội dung trong global-index.json.

người dùng trỏ chuột vào file nào sẽ hiện toolbox mô tả dài về file đó.
người dùng click vào file nào thì sẽ redirect họ tới thẳng file .md tương ứng.
