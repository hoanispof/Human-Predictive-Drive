## 6. Limitations and Falsifiability (~1 page)

```
SOURCE: Blackbox-Map.md + individual §7 Honest Assessment

CONTENT:
  6.1 — 5 Blackbox Gaps
    → Gap 1: Body-Input completeness
    → Gap 2: Signal Transduction "Driver Layer" (LARGEST)
    → Gap 3: Chunk Substrate (FUNDAMENTAL — what IS a chunk physically?)
    → Gap 4: Integration → Feeling → Consciousness
    → Gap 5: Hormone Combinatorics
    → Source: Blackbox-Map.md

  6.2 — How to falsify
    → Show a behavior the framework CANNOT explain
       using its own mechanisms (not by adding new ones)
    → Show a departure claim is wrong
       (e.g., prove dopamine IS directly reward, not salience)
    → Show the body-first architecture is wrong
       (e.g., prove PFC decides BEFORE body fires)
    → Show observation parameters ARE hardwired modules
       (e.g., find a "novelty module" in the brain)

  6.3 — Known biases
    → Single author (Creator-Lens.md)
    → Game developer lens (systems thinking bias)
    → Somatic processor bias (body-first may be projection)
    → No peer review yet
    → Survivorship: examples chosen AFTER framework built
```
