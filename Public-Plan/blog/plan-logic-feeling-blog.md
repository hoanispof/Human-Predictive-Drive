## §3 — CẤU TRÚC BLOG (DỰ KIẾN)

```
  ① HOOK: The misconception everyone carries
     → "Logic vs Feeling = 2 fundamentally different cognitive modes"
     → Belief này ở đâu: education, pop psychology, productivity culture
     → Kahneman's System 1/2 (dù Kahneman nuanced hơn) reinforced framing
     → "Trust your gut" vs "Think rationally" = false dichotomy
     → Tại sao belief này persistent: shareability difference looks like mechanism difference

  ② EVIDENCE: Same mechanism inside
     → Einstein doing math = Body-Knowing (compiled, automatic for him)
        Observer says "logic" vì math = shareable, reproducible
     → Therapist reading patient = Body-Knowing (compiled, automatic for expert)
        Observer says "intuition" vì clinical judgment = non-shareable
     → Chef tasting = Body-Knowing. Senior dev seeing bug = Body-Knowing.
     → CÙNG mechanism, KHÁC domain shareability

     Research:
       🟢 Klein 1998 — recognition-primed decisions: experts recognize, don't analyze
       🟢 Chase & Simon 1973 — chess expertise = chunking (compiled patterns)
       🟢 Kahneman & Klein 2009 — conditions for reliable intuition
       🟢 Ericsson — deliberate practice builds compiled domain expertise
       🟢 Einstein letters to Hadamard 1945 — "visual + muscular" thinking, not verbal logic

  ③ REFRAME: The real axis — Compiled vs Fresh
     → Compiled (Body-Knowing): automatic, body-direct, cost ≈ 0, domain-specific
     → Fresh (PFC draft): deliberate, PFC-mediated, ~4 working memory slots, costly
     → Key insight: speed is CONSEQUENCE of compilation, not defining feature
     → Kahneman reframe: System 1 with compiled expertise = accurate AND fast
       System 2 without compiled base = expensive AND unreliable
     → Senior dev vs junior dev: same codebase, different compiled base

  ④ WHY OBSERVER SEES "TWO MODES"
     → Shareability determines label:
       Math: deterministic → Body-Knowing converges across people → "logic"
       Psychology: probabilistic → Body-Knowing diverges → "intuition"
       Body-state: private → Body-Knowing cannot be shared → "feeling"
     → Observer CAN follow math steps → trust HIGH → "logic"
     → Observer CANNOT follow clinical judgment → trust LOW → "intuition"
     → Labels = f(observer's ability to verify), NOT mechanism difference
     → Labels STILL USEFUL for communication — just not mechanism descriptions

  ⑤ PFC = LAWYER, NOT JUDGE
     → 🟢 Gazzaniga split-brain: left hemisphere CONFABULATES reason for right hemisphere's action
     → 🟢 Haidt 2001: moral judgment = intuition first, reasoning post-hoc
     → 🟢 Nisbett & Wilson 1977: people cannot report actual causes of their behavior
     → "I quit for career growth" (actual: escape toxic boss)
     → "I bought this car for practical reasons" (actual: status body-need)
     → "Logical reasoning" CAN BE post-hoc lawyering for body-need
     → This doesn't mean logic is useless — Fresh processing is essential for
       novel domains, formalization, teaching, knowledge accumulation
     → It means: "logic" is not a neutral truth-finder, and "feeling" is not random noise

  ⑥ WHEN BOTH ARE WRONG — AND THE ONLY ARBITER
     → Body-Knowing wrong: Self-Referencing Trap (compiled on wrong basis),
       evolution lag (ancestral patterns in modern context), shelf-life expired
     → Fresh wrong: PFC lawyering (post-hoc narrative), insufficient working memory,
       premise from hidden body-need
     → NEITHER can verify itself:
       "Logic check" = PFC verify PFC = biased
       "Body check" = body verify body = potentially Self-Referencing
     → Domain feedback = ONLY calibration mechanism = trọng tài duy nhất
     → You CANNOT prescribe "trust logic" or "trust feeling" — because both can be wrong,
       and you can't tell which from inside

  ⑦ IMPLICATIONS
     → "Trust your gut" = incomplete (gut can be Self-Referencing)
     → "Think rationally" = incomplete (PFC can be Lawyer)
     → Expert intuition reliable BECAUSE of domain compilation, not magical body wisdom
     → "Knowing but not doing" reframe: 2 systems reaching different conclusions,
       not willpower failure
     → For AI/ML: reward modeling assumes rational agent — PFC = Lawyer complicates this
     → For education: teach domain compilation, not "logic vs creativity"

  ⑧ FALSIFICATION: Framework sai nếu...
     → Logic và intuition use fundamentally different neural substrates
       (not just different domains on same substrate)
     → Expert intuition accuracy UNRELATED to domain compilation
       (experts accurate for reasons other than compiled patterns)
     → PFC deliberation CONSISTENTLY outperforms compiled processing
       in domains where expert has compiled base
       (would mean "rational override" IS superior, not just different)
     → Content type (analytical vs emotional) predicts decision quality
       BETTER than compilation level
       (would mean logic/feeling IS the real axis after all)

  ⑨ CALL TO VERIFY
     → Link tới framework source files
     → "If you have expertise in cognitive science, decision-making research,
        or philosophy of mind — test this claim"
     → Counter-evidence > confirmation
```
