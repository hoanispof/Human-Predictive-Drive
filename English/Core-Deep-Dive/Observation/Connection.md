## §0 — WHY CONNECTION EXISTS: EVOLUTIONARY FOUNDATION

```
⭐ THE CORE QUESTION: Why does evolution select for a connection drive?

  PRINCIPLE: Connection exists because OTHER agents = EXTERNAL TOOLS
  that help body-base meet needs a SINGLE body CANNOT meet on its own.
  Body gives REAL REWARD when connection is present because connection =
  ACTUAL body-base feeding across multiple channels simultaneously.

  CAUSAL CHAIN:

    1 agent alone:
      → Knows what it knows → uses that → THAT'S IT
      → Faces threat alone → handles it alone → overload
      → Gets sick → no one cares → dies
      → = Low survival ceiling

    2+ agents CONNECTED:
      → A knows fire + B knows medicine = BOTH live better
      → Face threat → take turns watching → metabolic cost DROPS
      → A gets sick → B cares → A survives
      → = Much higher survival ceiling

    → Evolution selects for the mechanism to DETECT + MAINTAIN access to agents
    → = CONNECTION DRIVE
    → Body gives reward when connection is present (agent input feeds body-base)
    → Body gives pain when connection is lost (agent input cut off)


  ⭐ AGENTS = EXTERNAL TOOLS FOR BODY-BASE:

    (Agent-Mechanism.md §12: "Agent as body-need feeder")

    Body NEEDS social presence like it needs food, water, and sleep.

    ⑤ Social Brain Hypothesis (Dunbar 1998):
       Neocortex size ~ group size across primates.
       Large brain evolved NOT to solve math — but to MANAGE social agents.


  ⭐ REAL BODY REWARD — NOT ABSTRACT:

    Connection present:
      → Oxytocin release: trust increases, cortisol drops, warmth
```
