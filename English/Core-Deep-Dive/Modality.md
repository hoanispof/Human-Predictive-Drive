## §0 — WHAT IS MODALITY

```
Modality = ENCODING CHANNEL — how the brain stores information into chunks.

Chunk-system (Core-v7.8-Draft.md §4) = sole substrate.
Chunks are NOT stored on 1 single channel — they are stored on MANY channels SIMULTANEOUSLY.
Each channel = 1 modality.

EXAMPLE — chunk "dog":
  Visual:    image of a dog (4 legs, tail, fur)
  Auditory:  barking sound
  Somatic:   sensation of fur against the hand
  Motor:     motion of petting the dog's head
  Emotional: fear (if bitten before) or joy (if raised from puppy)
  Communication: label "dog" / "chó"

→ 1 chunk = COMBINATION of many modalities
→ Deep chunk = many modalities
→ Shallow chunk = few modalities (typically verbal only)

MODALITY IN V7.8:

  Hardware Map (§1.2):
    Zone B = CORTICAL MODALITY — PFC trainable
    Visual cortex, Auditory cortex, Somatosensory, Motor, Insula,...
    → Modality channels = physical brain regions (T1 hardware)

  Software Map (§1.3):
    Chunks compile WITHIN modalities (T2 software)
    → Same hardware (zone B), different compiled chunks = different skills

  Observation:
    Modality Balance = hardware profile (Core-v7.8-Draft.md §6.3)
    → Affects how chunks compile (multi-modal richness)
    → Affects how PFC observes (which modality is dominant)
    → Brain-wide property, NOT specific to PFC or Unconscious

KEY DISTINCTIONS:
  → Modality channels = T1 (hardware, brain regions, changes slowly)
  → Chunks in modalities = T2 (software, changes through experience)
  → Hardware sets RANGE → chunks select POSITION within range
```

## §1 — 6 MODALITIES (5 Experience + 1 Communication)

```
NOT 6 equal modalities — but 2 TYPES that differ fundamentally in nature:

  5 EXPERIENCE MODALITIES (encode DIRECT EXPERIENCE — primary DATA):
    ① Visual:    images, shapes, space          (Visual cortex V1→IT, FFA)
    ② Auditory:  sounds, voices, rhythm         (Auditory cortex, Heschl's)
    ③ Somatic:   body sensations, gut feeling   (Insula + Somatosensory)
    ④ Motor:     movement, muscle memory        (Motor cortex + Cerebellum + BG)
    ⑤ Emotional: emotions, fear, joy, love      (Amygdala + Insula + Limbic)

    = ALL creatures with nervous systems have these (varying degrees)
    = Encode experience directly — body senses → brain stores

  1 COMMUNICATION MODALITY (encode LABELS + TRANSFER — INDEX):
    ⑥ Verbal / Gesture / Writing / Code /...
    = Does NOT encode experience → encodes REFERENCES to experience
    = ONLY HUMANS have this at complex levels (language, writing, symbols)

    Brain areas: Broca's (production) + Wernicke's (comprehension) + Arcuate fasciculus
    🟢 Deaf-mute people using sign language: ALSO use Broca + Wernicke
       (Petitto et al. 2000) → brain areas for LANGUAGE, not just for SPEAKING

    4 FUNCTIONS:
      LABEL:    assigns names to multi-modal chunks → PFC holds more easily (WM compression)
      TRANSFER: transmits knowledge to others WITHOUT requiring direct experience
      COMBINE:  combines labels → rapidly expands domain WITHOUT experience
      SEQUENCE: processes SEQUENTIAL + HIERARCHICAL structure (Broca's, see below)


WHY SEPARATE INTO 2 TYPES:

  🟢 Strongest evidence — Fedorenko et al. (MIT, 2011-2024):
    Brain has 2 SEPARATE networks (dissociable):
      Language network (Broca + Wernicke): processes LANGUAGE
      Multiple demand network (PFC + parietal): processes THINKING / REASONING
    → Language damage: cannot speak, STILL calculates, solves puzzles ✅
    → Multiple demand damage: cannot reason, STILL speaks fluently ✅
    → THINKING ≠ LANGUAGE — 2 DIFFERENT systems

  Additional evidence:
    → Children 0-2: NO verbal yet → STILL chain (160+ strategies)
    → Aphasia: language lost → STILL calculates
    → Meditation silences inner voice: STILL thinks

  → Verbal = COMMENTATOR: describes the match, does NOT play football
  → "Thinking in words" = verbal NARRATES results the unconscious already created

  BUT — Broca's has 1 SMALL function related to thinking:
    🟢 Broca's area ALSO processes SEQUENTIAL / HIERARCHICAL structure:
      Action sequences (Fadiga et al.)
      Musical structure (musical syntax — Maess et al. 2001)
      → Broca's does NOT reason → but DOES process SEQUENCE
      → Language = 1 type of sequence (sentence = ordered word chain)


COMMUNICATION MODALITY — POWERFUL EVEN WITHOUT BEING A "REASONER":

  ✅ TRANSFER: "fire is hot" → transmits knowledge WITHOUT being burned
  ✅ COMBINE: "forest fire" = combined labels → NEW schema no one has experienced
  ✅ WM BOOST: label compresses chunk → PFC holds MORE → WIDER workspace
  ✅ SEQUENCE: orders steps, plans sequence, hierarchical structure

  ❌ BUT encoding = SHALLOW (only labels, not experience):
    "Knowing fire is hot" (verbal) ≠ "FEELING fire is hot" (somatic + emotional)
    Verbal schema = PREVIEW (direction correct, low fidelity)
    Experience schema = CONFIRM (high fidelity, body truly responds)
    "Knowing but unable to do" = verbal HAS labeled → body has NOT yet experienced


═══════════════════════════════════════════════════════
§1.1 — 5 Experience Modalities — Detail
═══════════════════════════════════════════════════════

① VISUAL — Images
  Brain areas: V1-V5, Fusiform (faces), Parietal (spatial)
  Encodes: images, colors, shapes, space, motion
  2 sub-types:
    Visual-sequential: "flow chart in the mind" → VERTICAL
    Visual-spatial: "3D model in the mind" → HORIZONTAL (cross-domain structural match)
  Cross-domain: YES but FILTERED (requires structural similarity)
    "Atom looks like solar system" = YES → match ✅
    "Team conflict like code bug" = NO structural similarity → MISS
  ⚠️ Most vivid in children age 4-7 → decreases when verbal takes over (school)
  ⚠️ Aphantasia ~2-5% have no visual imagination

② AUDITORY — Sound
  Brain areas: Primary auditory cortex, Heschl's gyrus, Wernicke (speech)
  Encodes: sounds, voices, music, rhythm, tone
  2 sub-types:
    Inner voice: speaking to oneself internally → VERTICAL (sequential self-talk)
    Pattern listening: hearing PATTERNs → HORIZONTAL (rhythm, melody across contexts)
  Cross-domain: light (pattern match via rhythm)
  Advantage: continuous (ears always open), strong emotional power (music → body responds)
  → Deep dive 6 unique hardware properties: Auditory-Hardware.md

③ SOMATIC — Body Sensation
  Brain areas: Insula (interoception) + Somatosensory + body-wide receptors
  Encodes: body sensations, gut feeling, tension, warmth, pain
  Characteristics:
    → NON-VERBAL: "feel it but cannot articulate clearly"
    → CROSS-DOMAIN: body detects patterns in ANY domain
    → FAST: body responds ~50ms vs PFC ~200ms
    → HONEST: somatic response is hard to fake
  Cross-domain: EXTREMELY HORIZONTAL (unfiltered body state match)
    Body does not encode "team" or "code" → encodes SENSATION
    Sensation MATCHES = "MATCHES" → regardless of domain
    = Universal matching key → cross-domain insight engine
  ⚠️ vmPFC = bridge somatic → conscious. Weak vmPFC → alexithymia
  ⚠️ Somatic → verbal is difficult: language LACKS WORDS for body states
    Insula sits DEEP → connection to Broca/Wernicke is LONG + WEAK

④ MOTOR — Movement
  Brain areas: Motor cortex + Premotor + Cerebellum (timing) + BG (habit)
  Encodes: movements, skills, procedures, muscle memory
  Characteristics:
    → PROCEDURAL: "hands know how, mind cannot explain"
    → COMPILED: repeat enough → BG takes over → automatic → PFC freed
    → DURABLE: cycling, swimming, typing → not forgotten for years
  Cross-domain: VERY LOW (domain-specific, little transfer)
  Direction: VERTICAL (sequential skills within domain → drills DEEP)

⑤ EMOTIONAL — Emotion
  Brain areas: Amygdala (threat/reward) + Insula (feeling state) + Limbic
  Encodes: emotional associations, fear, joy, disgust, love
  Characteristics:
    → FASTEST: Amygdala ~12ms (fastest in the brain)
    → STRONGEST: emotional encoding = strong + long-lasting (flashbulb)
    → SOCIAL-DOMINANT: primarily encodes social/relational patterns
    → CONTAGIOUS: detects + mirrors others' emotions
  Cross-domain: HORIZONTALLY BIASED SOCIAL (feeling match across social contexts)
  ⚠️ Differs from somatic:
    Somatic = body PATTERN in any domain (can be non-emotional)
    Emotional = FEELING VALENCE (always has +/−)
    Overlap: emotional HAS somatic component (fear → heart pounds)
    But: somatic CAN BE non-emotional ("this structure feels similar")
  ⚠️ Trauma = EXTREMELY STRONG emotional encoding → does not fade → intrusive
```

## §2 — HARDWARE BASIS (Sensor → Cortex)

```
⭐ REFRAME: Sensor → Modality mapping is NOT 1:1. It is MANY-TO-MANY.

  1 SENSOR → MANY cortical areas:
    Eyes → Visual cortex (image) + Amygdala (threat, ~12ms) + SCN (circadian)
    Ears → Auditory cortex + Amygdala (emotional) + Wernicke (speech)
    Skin → Somatosensory (where/what) + Insula (pleasant/unpleasant)
    Nose → Olfactory cortex + Amygdala + Hippocampus (BYPASS thalamus!)

  1 CORTICAL AREA ← MANY sensors:
    Amygdala ← eyes + ears + skin + nose + gut + hormones + mirror

  → "Modality" = PROCESSING AREA, not "input source"
  → Visual modality = Visual cortex processing (receives primarily from eyes + top-down PFC)

HARDWARE:
  Most input passes through THALAMUS (gateway + gate) before reaching cortex.
  Exceptions: olfaction bypasses thalamus, amygdala subcortical (~12ms).
  PFC controls attention via TRN (Thalamic Reticular Nucleus) — opens/closes gate.
  The entire neocortex = SAME 6-LAYER STRUCTURE (Mountcastle 1957),
  differs in wiring + layer thickness + receptor density.

  Hardware Map (Core-v7.8-Draft.md §1.2):
    Zone B = CORTICAL MODALITY — PFC trainable
    Includes: Visual, Auditory, Language, Somatosensory, Motor,
              Cerebellum, Insula, Parietal, Temporal
    → PFC (Zone A) reach Zone B: STRONG (Layer 2/3 direct, top-down bias)

BINDING — How modalities sync (5 concurrent mechanisms):
  ① Gamma oscillation ~40Hz (fire same phase = same object — Singer 1995)
  ② Multisensory neurons (hardware, present from birth — Stein & Meredith 1993)
  ③ Convergence zones (Parietal + STS + Insula — Damasio 1989)
  ④ Amygdala affective tagging (emotional → strong binding — McGaugh 2004)
  ⑤ DMN scaffolding (long-range connectivity — Doria 2010)

  → NO single binder. Binding = emergent from 5 mechanisms.
  → Detail: Neural-Architecture.md

🟢 Many-to-many mapping: established neuroanatomy
🟢 Thalamic gating: established (TRN, Sherman & Guillery)
🟢 6-layer cortex: Mountcastle 1957
🟢 Binding mechanisms: Singer 1995, Stein 1993, Damasio 1989, McGaugh 2004
🟡 "Modality = processing area" framing = framework synthesis
```

## §3 — CHUNK × MODALITY (Depth = Modality Count)

```
CHUNK DEPTH = NUMBER OF MODALITIES IN WHICH THE CHUNK IS ENCODED.

This is NOT a metaphor — it is LITERAL multi-modal encoding:
  Chunk stored in MANY brain areas = many modalities = DEEPER chunk.

  1 modality  → ★☆☆☆☆ SURFACE
    "Water boils at 100°C" (verbal only)
    Learn: read once. Forget: quickly. Change: change verbal → done.

  2 modalities → ★★☆☆☆
    "Red light = stop" (verbal + visual)
    Learn: see + hear. Forget: slower. Change: 2 modalities.

  3 modalities → ★★★☆☆
    "Guitar Am chord" (motor + auditory + visual)
    Learn: look + listen + hands practice. Forget: fairly slowly.

  4 modalities → ★★★★☆
    "Fear of dogs" (visual + auditory + somatic + emotional)
    Learn: 1 bite = sufficient. Forget: very slowly.

  5-6 modalities → ★★★★★ DEEP
    "I am worthless" (ALL modalities)
    Learn: many years of experience. Forget: near impossible. Change: years of therapy.


CHUNK DEPTH × FRAMEWORK DYNAMICS:

  COMPILE (Chunk.md v2.0 §2):
    Multi-modal encoding = 1 of 5 compile mechanisms
    → Chunks fire in MANY brain areas simultaneously → strong binding
    → Emotional peak + multi-modal = FASTEST compilation (1 occurrence = sufficient)

  RESISTANCE TO CHANGE:
    Fix 1 modality → 4-5 other modalities STILL hold the old version → pull back
    → = Why therapy produces "I know this already, so why is nothing changing":
      verbal fixed (1 modality), somatic + emotional not yet (3-4 modalities)
    → Fix: fix ENOUGH modalities past majority → behavior changes

  FEELING THRESHOLD (Core-v7.8-Draft.md §5):
    magnitude × clarity → PFC-observable
    → Deep chunk (many modalities) = STRONGER signal
    → Expert domain (high clarity) = detect signal MORE SUBTLY
    → Modality richness DIRECTLY affects feeling threshold

  BODY-FEEDBACK:
    Chunks stored in somatic + emotional modalities → body CAN EVALUATE
    Chunks stored verbal-only → body CANNOT evaluate → no body-confirm
    → Verbal-only chunks = "false explicit" (Somatic-Articulation-Loop §1.3)
    → Body-confirmed chunks = "true explicit"

  LABEL ≠ 6th MODALITY:
    🟡 Label = retrieval path, NOT modality (Chunk.md §6 Label-As-Handle)
    Label "dog" = INDEX to multi-modal chunk {visual+auditory+somatic+...}
    Label does NOT contain experience → contains REFERENCE to experience
    → WM compress: 1 label = 1 slot → hold MORE chunks than raw experience

🟢 Multi-modal encoding strengthens memory: Shams & Seitz 2008
🟢 Emotional encoding = strongest: McGaugh 2004
🟢 Resistance to change via modalities: implicit learning research
🟡 Depth = modality count framing = framework synthesis
🟡 "True vs false explicit" = framework extension (Somatic-Articulation-Loop §1.3)
```

## §4 — MODALITY INTERACTION (3 Types)

```
When a chunk is stored in MANY modalities, the modalities can:

① REINFORCEMENT (same direction → chunk STRENGTHENS):

  Verbal "dog is dangerous" + Visual dog baring teeth + Emotional fear
  = 3 modalities all saying "DANGEROUS" → EXTREMELY STRONG chunk

  → Multi-sensory learning is effective BECAUSE: many modalities reinforce
  → Why experience > lecture: experience = multi-modal, lecture = verbal only

② CONFLICT (opposite direction → "knowing but unable to do"):

  Verbal "dog is friendly" (fixed by CBT)
  BUT: Emotional STILL scared + Somatic STILL tense + Visual STILL sees threat
  = 1 modality says A, 3 say B → B WINS (stronger signal wins)

  → "Knowing but not changing" = MODALITY CONFLICT
  → Verbal (1) vs Somatic+Emotional+Motor (3-4) → majority wins
  → Fix: fix ENOUGH modalities past majority → behavior changes
  → = Body-Feedback dissonance (Core-v7.8-Draft.md §4):
    body evaluates conflict → fires DISSONANCE signal

③ COMPENSATION (1 modality compensates for a weak modality):

  Aphantasia (visual = 0) → COMPENSATES with verbal + somatic
  Deaf (auditory = 0) → COMPENSATES with visual + somatic
  → Neuroplasticity: resources reallocate to available modality
  → = Capability is not lost, only the CHANNEL changes


⭐ MODALITY CONFLICT × THERAPY (preview §7):

  "Therapy not working" COULD = correct therapy, WRONG modality target:
    CBT targets verbal → verbal-only chunk → effective ✅
    CBT targets somatic+emotional chunk → verbal fixed, body NOT YET → ❌

🟢 Multi-sensory integration: Shams & Seitz 2008
🟢 Modality conflict in therapy: exposure therapy research
🟢 Compensation/neuroplasticity: Merabet & Pascual-Leone 2010
🟡 "Modality majority wins" = framework synthesis
```

## §5 — MODALITY × THINKING DIRECTION (Vertical / Horizontal)

```
Dominant modality → THINKING DIRECTION → DEVELOPMENT DIRECTION:

  EXTREME VERTICAL ←————————————————————→ EXTREME HORIZONTAL
  Verbal    Motor   Visual-seq  Auditory  Emotional  Visual-spatial  Somatic

  VERTICAL (specialist — deep within 1 domain):
    Verbal + Motor dominant → chain SEQUENTIALLY within domain
    → Step 1 → step 2 → step 3 → drills DEEP
    → School trains verbal for 12-16 years → majority = vertical

  HORIZONTAL (improviser — wide across many domains):
    Somatic + Visual-spatial dominant → pattern match CROSS-DOMAIN
    → "This FEELS like that" (somatic)
    → "This LOOKS like that" (visual-spatial)
    → Cross-domain insight, framework building, intuition


═══════════════════════════════════════════════════════
§5.1 — Concept-First vs Label-First
═══════════════════════════════════════════════════════

Depending on somatic vs verbal dominance, the ORDER of understanding DIFFERS:

SOMATIC-DOMINANT (concept first, label second):
  ① FEELS the concept FIRST (body/pattern)
  ② Label = found AFTER, or not needed
  ③ Wrong label → body says "not matching yet" → finds different label
  ④ Many concepts have NO label → understands but hard to EXPLAIN
  → = Somatic-Articulation Loop (Somatic-Articulation-Loop §2): body knows → words come after

VERBAL-DOMINANT (label first, concept through label):
  ① RECEIVES label FIRST (word, definition)
  ② Understands concept THROUGH label (label = gateway)
  ③ Wrong label → concept is also wrong → must fix label first
  ④ Every concept HAS a label → easy to explain, but misses nuance

BOTH work. Only differs in order. Everyone has both, only the ratio differs.


═══════════════════════════════════════════════════════
§5.2 — Different Encode Types → Different Cross-Domain
═══════════════════════════════════════════════════════

  MODALITY       │ ENCODE TYPE               │ CROSS-DOMAIN
  ═══════════════╪═══════════════════════════╪═══════════════════════
  Visual         │ Discrete points + spatial  │ Filtered (structural match)
  Auditory       │ Temporal patterns (rhythm) │ Light (pattern match)
  Motor          │ Action sequences          │ Very low (domain-specific)
  Emotional      │ Valence + intensity       │ Horizontal-social (feeling match)
  Somatic        │ Continuous field (body)   │ EXTREME HORIZONTAL (unfiltered)
  Communication  │ Labels + references       │ Horizontal via label combining

Why SOMATIC cross-domain is STRONGEST:
  → Body state = ABSTRACT representation (no fixed structure)
  → Body encodes SENSATION, not domain-label
  → Sensation MATCHES = "MATCHES" → regardless of domain
  → + Body-wide = MOST data (whole body vs 1 brain area)
  → + Continuous analog field = RICHER than discrete points
  → BUT: NOISY — body feeling similar ≠ actually similar → NEEDS verification

Why VISUAL cross-domain EXISTS but is FILTERED:
  → Visual encodes DISCRETE structures → requires structural similarity
  → "Atom LOOKS like solar system" → YES structural match ✅
  → "Team conflict like code bug" → NO structural match → MISS
  → Somatic CATCHES (body state match) but visual MISSES (no structure match)
  → Visual = PRECISION horizontal (when structures match)
  → Somatic = BREADTH horizontal (any domain)


═══════════════════════════════════════════════════════
§5.3 — Expert = Balanced Modalities Within Domain
═══════════════════════════════════════════════════════

"Expert" is NOT strong in 1 modality → BALANCED across RELEVANT modalities:

  Each domain requires a DIFFERENT modality mix:
    Surgery:  visual-spatial + motor + somatic + emotional regulation
    Coding:   sequential + visual-spatial + somatic gut feel
    Therapy:  emotional + somatic + communication
    Music:    auditory + motor + emotional

  Expert = ALL relevant modalities are STRONG ENOUGH:
    → Skilled doctor: sees + does + feels + stays calm = ALL strong
    → Strong visual but trembling hands = NOT an expert surgeon

  DOMAIN FIT = modality profile MATCHES domain requirements:
    → "Poor performance" could = modalities NOT matching domain (fish trying to climb trees)
    → Career guidance: find domain MATCHING modality profile → natural advantage

  5 THINKING STYLES (from 5 experience modalities):
    Visual-spatial:  "structure matches" → architect, physicist
    Somatic:         "feels similar" → cross-domain theorist, therapist
    Motor:           "hands know how" → craftsman, athlete, surgeon
    Emotional:       "emotions match" → leader, artist, counselor
    Sequential:      "next step" → analyst, programmer, manager

🟡 Thinking direction framework = framework synthesis
🟡 Expert = balanced modalities = framework synthesis
🟢 Cross-domain analogy research: Gentner & Markman 1997
🟢 Implicit learning cross-domain: Reber 1967
```

## §6 — MODALITY × DEVELOPMENT (By Age)

```
Modality development is NOT uniform — each stage has a dominant modality:

  Age 0-2:  Somatic●●●●● + Emotional●●●●●
    → Body feel + attachment = foundation
    → Attachment chunks encoded in 2 STRONGEST modalities at this point → EXTREMELY DEEP
    → Verbal: not yet. Visual: rudimentary. Motor: reflex.

  Age 2-4:  + Motor●●●● + Visual●●●●
    → Running/jumping + pretend play + imagination
    → Multi-modal explosion → EXTREMELY FAST learning
    → Verbal begins (age 2-3)

  Age 4-7:  + Verbal●●●
    → Visual STILL strong + Verbal INCREASING → Creativity PEAK
    → Rich visual + verbal not yet dominant → NOT filtered

  Age 7-12: Verbal TAKES OVER●●●●●
    → School training: reading, writing, testing, memorizing
    → Visual, Somatic, Motor gradually SUPPRESSED (school does not use them)
    → "Losing imagination" = verbal dominates, visual decreases

  Age 13-18: Emotional SPIKE●●●●●
    → Hormones → emotional encoding increases strongly
    → Verbal still strong + Emotional INCREASES → conflict
    → "Knowing it's wrong but doing it anyway" = verbal says A, emotional says B (§4 conflict)

  Age 18+:  Verbal DOMINANT for most people
    → 12+ years school training → verbal leads
    → Except: people who train other modalities (artist, athlete, meditator)

⚠️ SCHOOL = VERBAL TRAINING MACHINE:
  12-16 years × verbal → majority = verbal dominant → specialist mode
  Other modalities ATROPHY (used less → weaken)
  CAN re-train: art → visual↑, meditation → somatic↑, sports → motor↑
  Neuroplasticity: "use it or lose it" — but RECOVERABLE with re-training

CONNECTION WITH V7.8:
  Development = SAME architecture, chunk density INCREASES gradually (Core §9)
  Modality development = chunks compile MORE modalities over time
  School bias = chunks compile BIASED toward verbal → other modalities lack chunks

🟢 Attachment encoding: Bowlby 1969, Ainsworth 1978
🟢 Visual-verbal shift in school: educational development research
🟢 Neuroplasticity: Merabet & Pascual-Leone 2010
🟡 School = verbal machine framing = framework synthesis
```

```
EACH therapy approach targets a DIFFERENT modality:

  THERAPY              │ MODALITY TARGET         │ SUITED FOR
  ═════════════════════╪═════════════════════════╪═══════════════════════════
  CBT                  │ Verbal                  │ Verbal-dominant chunks
  Exposure therapy     │ Emotional + Somatic     │ Phobia, anxiety (body)
  EMDR                 │ Visual + Emotional      │ Trauma (flashbulb)
  Somatic Experiencing │ Somatic                 │ Body-stored trauma
  Art therapy          │ Visual + Emotional      │ Non-verbal processing
  Music therapy        │ Auditory + Emotional    │ Emotional regulation
  Dance/Movement       │ Motor + Somatic + Emot  │ Body-embedded patterns
  Meditation           │ Somatic + Emotional     │ Body awareness
  Focusing (Gendlin)   │ Somatic                 │ Felt sense → articulation

PRINCIPLE: Match therapy modality WITH chunk modality profile.

  Verbal-only chunk → CBT effective ✅
  Somatic+emotional chunk → CBT NOT effective ❌ → needs body-based therapy
  "Therapy not working" COULD = correct therapy, WRONG modality target

  Framework prediction:
    Identify chunk modality profile FIRST → choose CORRECT therapy → effective

CONNECTION WITH FRAMEWORK:
  → Somatic-Articulation-Loop (Somatic-Articulation-Loop.md):
    Stage 5 body-confirm = somatic modality evaluates articulation
    Effective therapy = therapy reaches the CORRECT modalities where chunk is encoded
  → Feeling (Feeling.md v2.0):
    Feeling threshold (magnitude × clarity) → modality richness affects magnitude
    Deep chunks (many modalities) → stronger signal → easier for PFC to observe
  → Cortisol (Cortisol-Amplifier-Not-Cause.md):
    Chronic cortisol → overrides body signals → somatic modality muffled
    → Therapy must address cortisol BEFORE somatic work becomes effective

🟢 Therapy modality matching: exposure therapy research
🟢 CBT limitations for body-encoded trauma: van der Kolk 2014
🟢 EMDR for visual+emotional: Shapiro 2001
🟢 Focusing effectiveness: Hendricks 2001 meta-analysis
```
