### §1.2 — Distribution, not a single point

```
⭐ EACH PERSON = 1 DISTRIBUTION, NOT 1 POINT:

  SHOULD NOT say: "This person is in the body-domain" (oversimplification)
  SHOULD say: "This person has gaps DISTRIBUTED with a center in the body-domain"

  Why:
    → EVERYONE has Tier 1 gaps (hungry, thirsty, hot, cold) — sensory domain
    → EVERYONE has social gaps (connection, status) — social domain
    → Differences = RELATIVE, not ABSOLUTE
    → A CEO who is abstract-domain dominant still enjoys good food (sensory gap present)
    → A blue-collar worker who is body-domain dominant still has curiosity (small abstract gap)

  Distribution has a "CENTER OF MASS":
    → Where gaps are DENSEST
    → Where body-need fires STRONGEST
    → Where activities CONCENTRATE most
    → = Center of mass = proxy for "where this person is on the spectrum"

  Distribution has "TAILS":
    → Body-domain person still has an abstract tail (mild curiosity)
    → Abstract-domain person still has a sensory tail (still eats, sleeps)
    → Tails = secondary gaps — lower priority, but STILL FIRE

  Distribution CHANGES OVER TIME:
    → Chunks accumulate → new gaps form → distribution SHIFTS
    → Chunks decay (disuse) → old gaps weaken → distribution NARROWS
    → Major life events → distribution CAN shift dramatically
```
