## §1 — CORE: 2 TYPES OF DISSONANCE

### §1.1 — The core distinction

```
EVALUATIVE DISSONANCE:
  → Requires compiled chunks
  → Can develop only with age + experience
  → Processed via dACC + anterior insula
  → PFC CAN modulate (reframing, therapy, schema challenge)
  → CAN be numbed (dissociation, suppression)
  → Examples: betrayal, career anxiety, identity crisis, social rejection,
    moral injury, existential dread
  = THE MAJORITY OF ADULT HUMAN SUFFERING

DIRECT-STATE DISSONANCE:
  → Hardware pathways — present from birth
  → No compiled chunks required
  → No evaluation needed
  → PFC: minimal modulation
  → NUMBNESS-PROOF (see §1.5)
  → Examples: needle, itch, hunger, dizziness, loud noise, extreme temperature
  = EVOLUTIONARY FLOOR — always present

KEY: Orthogonal dimension. NOT a spectrum from one to the other.
  A single experience can have BOTH simultaneously.
```

### §1.2 — Orthogonal, not a spectrum

```
"Orthogonal" = the two types can vary independently:

  High Direct-State + Low Evaluative:
  → Needle from trusted doctor in calm context
  → Physical pain present, minimal anxiety/dread

  Low Direct-State + High Evaluative:
  → Fearing bankruptcy — no tissue damage
  → Pure compiled pattern generates suffering

  High BOTH:
  → Enemy stabbing you = Direct-State pain + Evaluative threat/betrayal
  → Tiger attack = startle (Direct-State) + survival dread (Evaluative)

  Low BOTH:
  → Pleasant afternoon, no gap, no signal
```

### §1.3 — Precondition mapping

```
Reward-Signal-Architecture §1.3 established Precondition mapping for reward.
SAME logic applies to dissonance:

  P1 Nociceptor baseline → Direct-State Dissonance threshold
  P2 Homeostatic systems → Direct-State mismatch sensitivity
  P3 Compiled chunks → Evaluative Dissonance REQUIRE this
  P4 Evaluative gate function → Evaluative Gates Direct-State (§3)
  P5 Social/narrative baseline → Evaluative Dissonance context

  = Same Precondition structure serves BOTH directions.
```

### §1.4 — Direct-State = evolutionary floor

```
Direct-State Dissonance = parallel to Direct-State Reward (Reward-Signal-Architecture §1.4):

  Direct-State Reward:
  → Burnout-proof: VTA dopamine fires regardless of compiled exhaustion
  → Body-oriented therapy exploits this (§10.5)

  Direct-State Dissonance:
  → NUMBNESS-PROOF: nociceptors fire regardless of Evaluative shutdown
  → This is what dissociation CANNOT block (§10.1)
  → Evolutionary logic: organism MUST respond to tissue damage regardless
    of cognitive/emotional state

  Both = hardware-level, cannot be compiled away.
```

### §1.5 — "Numbness-Proof" defined

```
"Numbness" in everyday use = often Evaluative shutdown:
  → Emotional blunting after trauma
  → Dissociation reducing emotional pain
  → Compiled layer going offline

NUMBNESS-PROOF = Direct-State Dissonance persists through all of the above:
  → Even in dissociative states: pain still registers
  → Even in emotional blunting: hunger, itch, temperature still signal
  → Even in meditation: A-delta and C fibers still fire

= Structural fact about hardware architecture.
= Why "just don't feel it" fails for genuine physical discomfort.
```

### §1.6 — Neurochemical multi-role

```
DIRECT-STATE DISSONANCE neurochemistry:
  → Substance P + glutamate + CGRP: nociception
  → Ghrelin: hunger
  → Histamine: itch
  → TRPV1 / TRPM8: temperature (hot/cold receptors)
  → CRH → cortisol: Evaluative sustainer (see §7.3)
  → dACC: social pain (Eisenberger 2003 — Evaluative territory)

BRIDGE MOLECULE: Opioid = bidirectional
  → Blocks Direct-State Dissonance = analgesia
  → Creates Evaluative Reward = hedonic effect
  = One molecule crossing both directions.
  = Why opioid abuse: Direct-State pain blocked + Evaluative reward created simultaneously.
```

### §1.7 — 2-source mapping

```
Body-Feedback-Mechanism.md: 2 signal sources.
  → Sensory-Driven ≈ Direct-State territory
  → Pattern-Driven ≈ Evaluative territory

Applied to dissonance:
  → Sensory-Driven signal → Direct-State Dissonance (tissue damage, homeostatic deviation)
  → Pattern-Driven signal → Evaluative Dissonance (prediction violation, schema mismatch)

IMPORTANT — two cascades possible:
  → Evaluative Dissonance CAN cascade → Direct-State body symptoms
    (anxiety → heart race, sweating, nausea = Evaluative triggering hardware)
  → Direct-State Dissonance CAN trigger Evaluative assessment
    (needle → "why is this happening?" = Direct-State prompting Evaluative)
  = Two-way influence, not one-way.
```

## §2 — E₀→E₃ COMPLEXITY GRADIENT APPLIED TO DISSONANCE

### §2.1 — Core finding

```
Reward-Signal-Architecture §2 established E₀→E₃ for reward.
IDENTICAL gradient applies to dissonance direction:

  E₀: Direct-State / minimal compilation → hardwired aversion
  E₁: Minimal compiled context → stranger-based discomfort
  E₂: Moderate compiled patterns → social rejection, failure experiences
  E₃: Deep compiled schemas → moral injury, existential dread, identity crisis

The E-complexity axis measures EVALUATION DEPTH, not reward-vs-dissonance.
= E₀→E₃ is a meta-dimension that spans both directions.
```

### §2.2 — E₀: bitter aversion

```
E₀ DISSONANCE EXAMPLE: Bitter aversion

  Research anchor: Steiner 1973 — gustofacial reflexes in neonates
  Anencephalic infants (no cortex) show disgust response to bitter taste.
  = Response exists without cortex, without compiled experience.

  Evaluative component: taste receptor → template match → "reject" signal
  → MINIMAL evaluation — receptor-level comparison, not compiled schema
  → This is WHY E₀, not purely Direct-State

  Direct-State component: TRPV1 involvement in some bitter compounds

  = E₀ sits at the boundary: template match without deep compilation.
  (See §11.3 Q4 for open question on exact classification.)
```

### §2.3 — E₁ → E₃ examples

```
E₁ DISSONANCE: Minimal compiled context
  → Stranger anxiety (~8 months developmental onset)
  → Requires: compiled "familiar face" template + mismatch detection
  → Infant without face templates CANNOT have stranger anxiety
  → Direct-State PARALLEL TRACK still present (hunger, pain = simultaneous)

E₂ DISSONANCE: Moderate compiled patterns
  → Social rejection (Eisenberger 2003: dACC activation, Cyberball)
  → Failure at valued task
  → Interpersonal conflict with meaningful other
  → Requires: compiled relationship schemas + compiled self-competence schemas

E₃ DISSONANCE: Deep compiled schemas — decades of compilation required
  → Moral injury: "I violated my core values"
  → Existential dread: "my life is meaningless"
  → Identity dissolution: "I don't know who I am anymore"
  → These CANNOT occur in children — requires decades of schema compilation
```

### §2.4 — Stacking development

```
LAYERS STACK — Direct-State always present as parallel track:

  Neonate: E₀ + Direct-State only
    → Bitter aversion + nociception + hunger + temperature
    → NO Evaluative layer yet

  Infant (~8-12 months): E₀ + E₁ + Direct-State
    → Adds stranger anxiety, separation distress

  Child (3-7 years): E₀ + E₁ + E₂ + Direct-State
    → Adds social rejection sensitivity, failure discomfort

  Adult: E₀ + E₁ + E₂ + E₃ + Direct-State
    → All layers active simultaneously
    → Tiger attack activates ALL layers at once

  Direct-State = parallel track throughout. Never replaced by higher layers.
  Higher E-levels ADD complexity, they do not eliminate the lower floor.
```

## §3 — EVALUATIVE GATES DIRECT-STATE: DISSONANCE VERSION

### §3.1 — Same 5-step insula process, different direction

```
Reward-Signal-Architecture §3 established:
  Evaluative Gates Direct-State FOR REWARD (pleasure direction).

SAME MECHANISM applies to dissonance direction:

  Step 1: Direct-State signal → posterior insula (raw, unmodified)
  Step 2: Evaluative assessment runs IN PARALLEL
  Step 3: Anterior insula INTEGRATES Direct-State + Evaluative
  Step 4: ACC runs conflict check
  Step 5: OFC/vmPFC assigns value + behavioral priority

= Same 5-step gradient. Same insula architecture. Different direction.
```

### §3.2 — 4 modulation outcomes

```
4 OUTCOMES based on Evaluative state:

  1. EVALUATIVE AMPLIFIES Direct-State Dissonance:
     → Enemy administers the needle → same tissue damage, MORE pain
     → Nocebo effect: "this will hurt" → pain increases
     → Mechanism: OFC/vmPFC sends amplifying signal to anterior insula

  2. EVALUATIVE SUPPRESSES Direct-State Dissonance:
     → Trusted doctor administers same needle → less pain
     → Placebo effect: "this will reduce pain" → pain decreases
     → Loved one holding hand → pain reduces (Master 2009, Younger 2010)

  3. EVALUATIVE ABSENT (infant, early development):
     → Raw Direct-State signal, no modulation
     → Pure hardware response, no gating layer yet

  4. EVALUATIVE CONFLICTED:
     → Ambiguous context ("is this person friend or threat?")
     → ACC conflict → pain experience fluctuates, unstable
```

### §3.3 — Placebo/Nocebo as structural proof

```
PLACEBO = proof that Evaluative gates Direct-State Dissonance:
  → Wager et al. 2004: placebo reduces pain via PFC → rostral ACC → PAG pathway
    = Top-down Evaluative signal modifying Direct-State pain channel
  → Zubieta et al. 2005: placebo activates ENDOGENOUS opioid system
    = Evaluative "it will hurt less" triggers actual neurochemistry change

NOCEBO = proof of Evaluative AMPLIFICATION:
  → Colloca & Benedetti 2005: nocebo hyperalgesia
  → Benedetti et al. 2007: nocebo effect mediated by cholecystokinin
    = "This will hurt more" → actual pain increase via specific neurochemistry

SOCIAL SUPPORT = natural Evaluative suppressor:
  → Eisenberger et al. 2006: social support reduces pain perception
  → Master et al. 2009: photo of romantic partner reduces pain (fMRI)
  → Younger et al. 2010: romantic love reduces pain
  = "Someone safe is here" = Evaluative signal → Direct-State modulation

= Placebo/Nocebo research = prior art for Evaluative Gates Direct-State
  in the dissonance direction, before framework named it.
```

### §3.4 — Insula development implication

```
DEVELOPMENT GRADIENT:
  → Young infant: posterior insula active, anterior insula develops with age
  → Evaluative gate capacity builds during development (see §9)
  → Children show LESS placebo modulation than adults = less developed gate
  → Adolescents: gate active but anterior insula still maturing
    (Blakemore 2008: adolescent social sensitivity = Evaluative layer still calibrating)

CLINICAL IMPLICATION:
  → Measuring "pain" without measuring Evaluative context = incomplete
  → Same tissue damage → different pain experience depending on Evaluative state
  → Pain research prior art: Melzack & Casey 1968 (sensory-discriminative vs
    affective-motivational), Rainville 1997 (hypnosis separates 2 components),
    Price 2000 (fMRI confirms different substrates)
```

## §4 — MISMATCH SPLITTING: HARDWARE VS COMPILED

### §4.1 — Why splitting is necessary

```
02-Dissonance.md established: 3 genuine sources of dissonance.
  Source ②: MISMATCH (prediction error, expectation violation)

Problem: "Mismatch" covers TWO radically different phenomena:
  → Homeostatic mismatch: body temperature off → Direct-State signal
  → Compiled expectation mismatch: betrayal → Evaluative signal

Lumping them under "mismatch" obscures the mechanism.
= Mismatch splitting is necessary for clinical accuracy.
```

### §4.2 — Hardware mismatch → Direct-State

```
HARDWARE MISMATCH = Direct-State territory:

  → Temperature deviation from homeostatic set-point
  → Vestibular mismatch (motion sickness: visual vs vestibular conflict)
  → Homeostatic deviation: blood glucose drop, osmolarity shift, CO₂ rise
  → Respiratory mismatch: air hunger when breath is held

  Mechanism: hardware sensor → deviation detected → Direct-State signal
  → No compiled experience required
  → Present from birth
  → Infant CAN experience all of these

  Examples: dizziness, hunger, nausea from motion, breathlessness
```

### §4.3 — Compiled mismatch → Evaluative

```
COMPILED MISMATCH = Evaluative territory:

  → Betrayal: compiled "this person is trustworthy" + behavior contradicts
  → Career failure: compiled self-competence schema + reality contradicts
  → Identity crisis: compiled "who I am" + life events contradict
  → Moral injury: compiled "I am someone who doesn't do this" + action contradicts

  Mechanism: compiled prediction → reality mismatch → dACC + anterior insula
  → REQUIRES compiled chunks — infant CANNOT experience these
  → Develops with age as compilation deepens
  → PFC CAN modulate (reframe, challenge schemas)

  Key: the mismatch is between COMPILED MODELS, not hardware sensors.
```

### §4.4 — Impact on 3 genuine sources (from 02-Dissonance.md)

```
02-Dissonance.md: 3 genuine sources.
Mismatch splitting REFINES the mapping:

  Source ①: Nociception
  → Pure Direct-State territory
  → No Evaluative component needed for signal
  → (Evaluative CAN modulate it via §3 — but signal exists without modulation)

  Source ②: Mismatch → NEEDS SPLIT:
  → Hardware mismatch → Direct-State
  → Compiled mismatch → Evaluative
  → SAME source label, DIFFERENT type

  Source ③: Recalibration
  → Mostly Direct-State (homeostatic recalibration)
  → Some Evaluative component when schemas must update
  → (e.g., grief = recalibration after loss includes deep Evaluative layer)

= Mismatch is the one source that requires explicit splitting.
```

## §5 — CHUNK DYNAMICS × 2 DISSONANCE TYPES

### §5.1 — Overview

```
Body-Feedback-Mechanism.md: 3 Chunk Dynamics.
  → Chunk-Shift: experience flips from one valence to opposite
  → Chunk-Miss: expected experience doesn't arrive
  → Chunk-Gap: gap between current state and compiled baseline

Applying 2 Dissonance Types to chunk dynamics reveals:
  → NOT all chunk dynamics affect both types equally
  → The DISTRIBUTION itself is structural proof for 2-type architecture
```

### §5.2 — Chunk-Shift = Evaluative-only proof

```
CHUNK-SHIFT = PRIMARILY EVALUATIVE:

  Example 1: Food + cockroach
  → Food chunk: compiled as "good/safe" → reward signal
  → Cockroach discovered → INSTANT CHUNK-SHIFT → nausea, disgust
  → Same food, same physical substance → experience flips completely
  = Evaluative layer flipped. Direct-State layer (taste receptors) unchanged.

  Example 2: Loved one's touch + betrayal revelation
  → Touch compiled as "safe/warm" → Direct-State Reward (CT afferents) +
    Evaluative Reward (entity-compiled layer)
  → Betrayal revelation: same touch now feels uncomfortable → CHUNK-SHIFT
  → Direct-State component (CT afferents firing) = UNCHANGED
  → Evaluative component = FLIPPED to dissonance
  = Structural proof: 2 components can shift independently.

  DIRECT-STATE CANNOT CHUNK-SHIFT:
  → Needle still hurts regardless of who holds it or how you think about it
  → Temperature still signals regardless of context
  → Itch still fires regardless of meaning assigned
  = Hardware doesn't flip. Only compiled layers flip.

  CONCLUSION: Chunk-Shift in dissonance direction = Evaluative-only phenomenon.
  = This is one of the strongest structural proofs for 2-type architecture.
```

### §5.3 — Chunk-Miss = both types

```
CHUNK-MISS = BOTH TYPES:

  Evaluative Chunk-Miss:
  → Expected approval doesn't arrive → Evaluative Dissonance
  → Expected connection absent → Evaluative Dissonance
  → Meaning expected from work → didn't come → Evaluative Dissonance

  Direct-State Chunk-Miss:
  → Expected food doesn't arrive → hunger intensifies (Direct-State)
  → Expected touch absent → Direct-State signal (social touch deprivation)
  → Expected sensory reward absent → Direct-State gap

  = Both types can generate Chunk-Miss dissonance.
  = Chunk-Miss is the most type-neutral of the 3 chunk dynamics.
```

### §5.4 — Chunk-Gap = primarily Evaluative

```
CHUNK-GAP = PRIMARILY EVALUATIVE:

  Chunk-Gap = gap between current state and compiled baseline.
  → "Something is missing but I can't name what" = Evaluative territory
  → Gap from compiled baseline requires: baseline exists in compiled form
  → Direct-State doesn't "miss" a baseline — it signals current deviation

  Direct-State component exists but secondary:
  → Homeostatic setpoint deviation = Direct-State gap signal
  → But the "background dissatisfaction" of Chunk-Gap = Evaluative layer

  = Chunk-Gap is primarily how Evaluative Dissonance manifests as diffuse,
    ongoing low-level suffering without clear identifiable source.
```

## §7 — FRAMEWORK INTERACTIONS

### §7.1 — Simulation-Engine × 2 Dissonance Types

```
Simulation-Engine.md: 1 engine with 3 components.
= Evaluative Dissonance GENERATOR.

Simulation-Engine GENERATES:
  → Preview of bankruptcy: body pre-feels → Evaluative Dissonance
  → Anticipation of rejection: compiled social schema → Evaluative signal
  → "What if" scenarios: Evaluative Dissonance from imagined futures
  = ALL Simulation-Engine output = Evaluative territory

Simulation-Engine CANNOT:
  → Generate Direct-State Dissonance
  → Imagining needle does not activate A-delta/C fibers at full intensity
  → Imagining hunger does not trigger ghrelin at full level
  (Some somatic amplification possible but NOT hardware-level Direct-State)

Simulation-Engine CAN cascade:
  → Evaluative Dissonance (anxiety) → HPA activation → cortisol → 
    → body symptoms (heart rate, sweating, nausea) = Evaluative-triggered Direct-State
  = Not Direct-State from Simulation, but Evaluative triggering hardware cascade.
```

### §7.2 — PFC Operations × 2 Dissonance Types

```
PFC-Operations.md: HOLD and SUPPRESS operations.

PFC HOLD × Dissonance Types:
  → HOLD effective on EVALUATIVE: deliberate rumination holds dissonance active
    (worry, replaying painful events = HOLD on Evaluative Dissonance)
  → HOLD PARTIAL on Direct-State: can attend to pain but cannot amplify hardware
    (attention to pain = HOLD on Evaluative layer assessing pain, not Direct-State itself)

PFC SUPPRESS × Dissonance Types:
  → SUPPRESS temporary on EVALUATIVE: Wegner 1994 suppression effect
    (active suppression of thoughts → rebound effect; cortisol drain)
  → SUPPRESS VERY LIMITED on Direct-State: cannot override nociceptors for long
    (stoic endurance = Evaluative override, limited; eventually Direct-State breaks through)

CLINICAL:
  → CBT targets EVALUATIVE (challenge schemas) = appropriate mechanism
  → Cannot CBT away Direct-State Dissonance
  → Medication for Direct-State (analgesia) ≠ medication for Evaluative (anxiolytics)
```

### §7.3 — Cortisol × 2 Dissonance Types

```
Cortisol-Baseline.md: Cortisol = amplifier, direction > level.

Cortisol × Dissonance Types:
  → Cortisol = sustainer for BOTH types of dissonance
  → Stronger effect on EVALUATIVE (HPA axis response to Evaluative threat)
  → Cortisol AMPLIFIES Evaluative Dissonance in loop:
    Evaluative threat → cortisol → sensitized threat detection → more Evaluative dissonance

  Cortisol half-life: 60-90 minutes
  → Once elevated by Evaluative Dissonance event:
  → Body remains sensitized for ~60-90 minutes
  → = WHY calming down takes time after argument/bad news
  → = WHY "just think positive thoughts" fails immediately after cortisol spike:
    hardware still sensitized regardless of Evaluative reframing

  Chronic cortisol elevation:
  → Sensitizes both types (lowers threshold for both)
  → Primarily Evaluative driver (rumination, chronic stress = sustained HPA activation)
```

### §7.4 — Asymmetric Transition Speed

```
CORE FINDING: Reward → Dissonance = FAST. Dissonance → Reward = SLOW.

FAST direction (reward → dissonance):
  → Amygdala fast path: ~12ms (LeDoux 1996)
  → Bad news: immediate valence shift
  → Betrayal: instant Evaluative flip
  → Jump scare: direct Hardware activation
  = MILLISECONDS

SLOW direction (dissonance → reward):
  → Cortisol clearance: 60-90 minutes minimum
  → Emotional memory consolidation: hours
  → Neural recalibration of Evaluative layer: hours to days
  = HOURS TO DAYS

RESEARCH ANCHORS:
  → Baumeister et al. 2001: "Bad is stronger than good" — processing asymmetry
  → Kahneman & Tversky 1979: loss aversion ~2x (losses loom larger than gains)
  → Rozin & Royzman 2001: negativity dominance + contamination principle

MECHANISM:
  → Reward → Dissonance FAST: because survival = detect threat immediately
  → Dissonance → Reward SLOW: because recovery = requires evidence of safety,
    which requires TIME to accumulate (Evaluative system needs sustained safety signal)
    
  = Asymmetry resides in EVALUATIVE LAYER primarily:
    Direct-State transitions faster (pain stops when stimulus removed)
    Evaluative dissonance persists through cortisol and memory consolidation window
```

## §8 — CASES

### §8.1 — Direct-State pure cases

```
NEEDLE:
  → A-delta fibers (sharp, fast) + C fibers (slow, aching)
  → No compiled chunks required
  → Infant with zero life experience = same neural pathway
  → Doctor vs enemy administers: SAME Direct-State signal (see §8.3 for Evaluative difference)

ITCH:
  → Histamine → C fibers → pruritogenic pathway
  → Hardware signal, no evaluation required
  → "Thinking the itch away" = Evaluative modulation, limited effectiveness

HUNGER:
  → Blood glucose drop → ghrelin release → hypothalamic hunger circuit
  → Direct homeostatic signal
  → Fasting meditators report hunger Direct-State persists regardless of Evaluative state

LOUD NOISE STARTLE:
  → Cochlear → brainstem → startle circuit (Koch 1999)
  → Sub-cortical pathway — faster than Evaluative processing
  → Habituates over time (Evaluative learns "not dangerous") but Direct-State still fires initially

DIZZINESS / MOTION SICKNESS:
  → Vestibular-visual mismatch → hardware conflict
  → No Evaluative component needed
  → Cannot think your way out of motion sickness (Direct-State dominates)
```

### §8.2 — Evaluative pure cases

```
THINKING OF ENEMY (no contact):
  → Pattern-Driven E₂
  → Compiled "threat/untrustworthy" schema activated by thought alone
  → No sensory input, no tissue damage → pure Evaluative signal

FEARING BANKRUPTCY:
  → Simulation-Engine E₂-E₃
  → Financial schema compilation required (children cannot fear bankruptcy)
  → Body responds as if threat is present (cortisol, tension)
  → = Evaluative → body cascade, but NOT Direct-State in origin

"MY LIFE IS MEANINGLESS":
  → E₃ deep
  → Requires decades of Meaning compilation to have assembled, then violated
  → No physical correlate — pure schema collapse
  → Existential dread = architecture-level Evaluative Dissonance

REMEMBERING EMBARRASSING MOMENT:
  → E₂ replay via Simulation-Engine
  → Original event may be years past, body still responds
  → Direct-State component = zero (no current physical input)
  → Evaluative = full activation through memory replay
```

### §8.3 — Mixed cases (compound)

```
TIGER ATTACK — 3-phase compound:
  Phase 1 (~12ms): Direct-State startle
  → Amygdala fast path (LeDoux 1996)
  → Subcortical, no cortex required
  → Instant adrenaline, freeze/flee impulse
  
  Phase 2 (~200ms): Evaluative assessment
  → "What is this? How dangerous? Any escape?"
  → Compiled threat schemas activated
  → Evaluative Dissonance: survival dread
  
  Phase 3: Compound sustained
  → Both types active simultaneously, amplifying each other
  → Hardware-Subsidy × "maximum threat" Evaluative context = peak dissonance

PARENT SCOLDING — layered compound:
  → Mild Direct-State: auditory startle component (volume)
  → Evaluative Layer 1: Entity-Compiled × parent (hardware-subsidy level 4-5)
    "attachment figure disapproves" = deep Evaluative
  → Evaluative Layer 2: content evaluation ("am I actually wrong?")
  → Evaluative Layer 3: anticipatory ("what happens next?") via Simulation-Engine
  → = One event generates Direct-State (mild) + 3 Evaluative layers simultaneously

DOCTOR INJECTION VS ENEMY STAB:
  → SAME needle, SAME tissue damage, SAME A-delta/C fiber activation
  → DIFFERENT Evaluative context:
    Doctor: "healing, I chose this, safe" → Evaluative SUPPRESSES Direct-State
    Enemy: "threat, betrayal, danger" → Evaluative AMPLIFIES Direct-State
  → Result: enemy stab = more painful than medically identical doctor injection
  = Structural proof that §3 modulation is real and clinically significant.
```

## §10 — CLINICAL IMPLICATIONS

### §10.1 — Dissociation: selective Evaluative shutdown

```
🟡 DISSOCIATION = SELECTIVE EVALUATIVE SHUTDOWN:

  Mechanism:
  → Evaluative layer goes offline or reduces activity significantly
  → Direct-State Dissonance: PERSISTS (numbness-proof = §1.5)
  → Evaluative Dissonance: BLOCKED or severely blunted

  Observable:
  → Person reports "not feeling emotions" (Evaluative offline)
  → Person CAN still feel physical pain, hunger, temperature (Direct-State intact)
  → = PROOF that 2 systems can dissociate independently

  Structural dissociation (Van der Hart et al. 2006):
  → Aparently Normal Part (ANP): Evaluative system functional, Direct-State suppressed
  → Emotional Part (EP): opposite pattern
  → = Two distinct systems, separately activatable

  Clinical implication:
  → Emotional blunting ≠ "pain free"
  → Dissociative patients still need Direct-State pain management
  → Therapy must address Evaluative layer (trauma processing)
    to restore full signal function — NOT just Direct-State symptom management
```

### §10.2 — Alexithymia: Evaluative labeling deficit

```
🟡 ALEXITHYMIA = EVALUATIVE DISSONANCE PROCESSING IMPAIRED:

  Mechanism:
  → Evaluative Dissonance signals exist (dACC + anterior insula fire)
  → Labeling/identification step fails
  → Person CANNOT identify "what kind of discomfort this is"

  Diagnostic precision:
  → CAN identify Direct-State Dissonance: "I feel pain here" ✓
  → CANNOT label Evaluative Dissonance: "I feel... something... I don't know" ✗

  = NOT a signal deficit. = Evaluative LABELING deficit.
  Signal is present; the interpretation/identification layer underperforms.

  Research anchor: Taylor et al. 1997 — alexithymia assessment.

  Clinical implication:
  → Treatment = develop Evaluative labeling capacity (not Direct-State management)
  → Body-scan practices help bridge Direct-State sensation → Evaluative label
  → Psychotherapy goal: build vocabulary + recognition for Evaluative signals
```

### §10.3 — Anxiety: excessive Evaluative Dissonance

```
🟡 ANXIETY = EXCESSIVE EVALUATIVE DISSONANCE:

  Mechanism:
  → Excessive Evaluative Dissonance (worry, anticipation, rumination)
  → Cascades → Direct-State body symptoms (heart racing, sweating, nausea)
  = Evaluative Dissonance TRIGGERING Direct-State responses.

  This is the REVERSE of §3 (Evaluative Gates Direct-State):
  → §3: Evaluative MODULATES incoming Direct-State signal
  → Anxiety: Evaluative GENERATES new Direct-State responses (via HPA cascade)
  = Two-way street: gating AND generating.

  Compiled Quality matters:
  → Schema-compiled fear (old schemas) vs genuine threat (current reality)
  → CBT target: reduce EVALUATIVE component (reframe schemas, challenge predictions)

  Clinical implication:
  → Direct-State symptoms (racing heart, sweating) = DOWNSTREAM of Evaluative
  → Treating body symptoms alone = treating downstream, not source
  → Evaluative layer = primary treatment target
  → Medication (beta-blockers for body symptoms) = symptom management only

  🟢 Barlow 2002: anxiety = anticipatory component + somatic symptoms.
```

### §10.4 — Chronic pain: compound architecture

```
🟡 CHRONIC PAIN = COMPOUND (NOT SINGLE TYPE):

  Original injury: Direct-State Dissonance (tissue damage — clear, initial)
  + Fear of pain: Evaluative Dissonance (anticipation via Simulation-Engine)
  + Social withdrawal: Evaluative Dissonance (isolation from compiled social baseline)
  + Identity shift: Evaluative Dissonance ("I am a pain patient" = identity chunk installed)
  = COMPOUND: 1 Direct-State origin + multiple Evaluative layers accumulated over time

  Why chronic pain ≠ acute pain in treatment response:
  → Acute pain: primarily Direct-State → analgesia works well
  → Chronic pain: Direct-State + multiple Evaluative layers
  → Evaluative layers PERSIST even when tissue heals
  → Treating only Direct-State (medication alone) = incomplete
  → Evaluative layers require Evaluative intervention (CBT, schema work, social reconnection)

  🟢 Gatchel et al. 2007: biopsychosocial model of chronic pain.
```

### §10.5 — Body-oriented therapy: cross-direction insight

```
🟡 BODY-ORIENTED THERAPY WORKS BIDIRECTIONALLY:

  REWARD DIRECTION (Reward-Signal-Architecture §10 established):
    Evaluative Reward exhausted (burnout)
    → Activate Direct-State Reward (touch, exercise, warmth)
    → "Backdoor" past exhausted Evaluative gate
    → Re-entry via hardware floor

  DISSONANCE DIRECTION (this file — new insight):
    Evaluative Dissonance overactive (anxiety, rumination, chronic stress)
    → Activating Direct-State REWARD can counterbalance Evaluative DISSONANCE:
      Exercise → endocannabinoids → Direct-State Reward → COMPETES with Evaluative Dissonance
      Touch (CT afferents) → Direct-State Reward → suppresses anxiety signal
    → = Direct-State Reward as antidote to Evaluative Dissonance (CROSS-DIRECTION)
    → = Not just "activate reward when reward is stuck"
    → = ALSO "activate Direct-State Reward to compete with Evaluative Dissonance"

  Mechanism: different valence types can compete in anterior insula integration
  → Direct-State Reward signal "fills" anterior insula → less room for Evaluative Dissonance

  🟢 Van der Kolk 2014: The Body Keeps the Score.
```
