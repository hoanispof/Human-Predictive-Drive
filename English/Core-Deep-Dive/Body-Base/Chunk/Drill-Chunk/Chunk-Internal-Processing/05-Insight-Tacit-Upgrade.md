## §1 — Insight / Aha Moment

### §1.1 — Position in 4-type taxonomy

```
Type 2 Connection (01-Chunk-Connection §1.2):
  = 2+ OLD chunks suddenly LINK in a NEW way → sudden burst
  = MOSTLY UNCONSCIOUS (DMN, incubation), emerges suddenly

Aha = Type 2 connection producing NOVEL CHUNK:
  → Not just link noted (Type 4) — entire new pattern compiles
  → Emotional peak mechanism → fast compile
  → Gamma burst + right temporal lobe activation
  → 🟢 Kounios & Beeman 2009: neural basis of insight

⭐ POSITION IN FRAMEWORK:
  → F1: HOW chunks compile from raw input
  → F3: HOW external chunks install
  → F4 01: HOW PFC deliberately connects (Type 4)
  → F4 THIS: HOW insight SPONTANEOUSLY connects (Type 2 → novel chunk)
```

### §1.2 — Insight mechanism (detailed)

```
🟡 FRAMEWORK SYNTHESIS — insight as probability distribution event:

STAGE 1 — PREPARATION (conscious):
  → PFC holds problem chunks in WM
  → Type 4 deliberate search: hold A+B → check → hold A+C → check...
  → Body votes: smooth / resist / empty for each connection
  → SOME connections found but INCOMPLETE (problem not solved)
  → = "I have tried everything" → conscious search exhausted

STAGE 2 — INCUBATION (unconscious):
  → PFC releases problem → DMN (Default Mode Network) continues
  → DMN: free association, memory replay, mind-wandering
  → 🟢 DMN and creativity (Beaty et al. 2016, Andrews-Hanna 2012)
  → LOW-PROBABILITY links explored (01b §5.2):
    → Conscious search → only explores HIGH-probability links
    → DMN → explores LOW-probability links (weaker, distant associations)
    → = "Creativity = accessing low-probability pathways" (01b)
  → Sleep: hippocampal replay explores combinations
  → 🟢 Sleep and insight (Wagner et al. 2004: sleep improves insight solutions)

STAGE 3 — ILLUMINATION (sudden):
  → Low-probability link suddenly EXCEEDS threshold
  → → Gamma burst (~30-100 Hz) in right temporal cortex
  → → 🟢 Kounios & Beeman 2009: gamma activity 300ms before insight report
  → Body response: SUDDEN coherence → dopamine + mild cortisol drop
  → Phenomenology: "EUREKA!" / "Aha!" / "That's it!"
  → = Type 2 connection fires — novel chunk compiles via emotional peak mechanism

STAGE 4 — VERIFICATION (conscious):
  → PFC re-engages → check insight against existing knowledge
  → Type 4 deliberate: "Does this new connection actually hold?"
  → Body vote: smooth → accepted / resist → maybe wrong insight
  → = Not all insights are correct → verification essential
  → Einstein: insight (body simulation wrongness) → years of math verification

  ⭐ INSIGHT LIFECYCLE = preparation→incubation→illumination→verification
  → 🟢 Wallas 1926: 4-stage model of creativity (classic, still valid)
  → Framework adds: MECHANISM at each stage (probability, DMN, gamma, body vote)


WHY LOW-PROBABILITY LINKS PRODUCE INSIGHT:

  From 01b §2 — activation probability distribution:
  → Most activation goes to HIGH-probability links (routine, expected)
  → LOW-probability links rarely fire (distant, cross-domain)
  → But when they DO fire → NOVEL connection
  → Novel = potentially valuable IF coherent
  → Cross-domain links = LARGEST aha (01 §1.2: Type 2)

  → CREATIVITY = SYSTEMATIC access to low-probability pathways:
    → Diverse experience → more low-probability links available
    → Incubation → DMN explores these pathways
    → Novel link fires → insight
    → = "More diverse chunks = more creative potential"
```

## §2 — Tacit Knowledge

### §2.1 — Polanyi definition + framework interpretation

```
🟢 ESTABLISHED — Polanyi 1966:
  "We can know more than we can tell."

CLASSIC EXAMPLES:
  → Expert cyclist: CAN ride → CANNOT explain how to balance
  → Expert surgeon: CAN perform → CANNOT fully articulate decisions
  → Expert chef: CAN taste "needs salt" → CANNOT specify exact chemical detection
  → Expert parent: CAN detect child distress → CANNOT articulate which cues

FRAMEWORK INTERPRETATION:

  TACIT KNOWLEDGE = chunks compiled SO DEEP → PFC cannot observe process:

  ① Chunks compiled via thousands of repetitions (F1 Convergence Point 2 channel 1)
  ② Chunks compiled across multiple modalities (F1 Convergence Point 2 channel 3)
  ③ Meta-chunks formed: A+B+C → [ABC] → fires AS ONE (Type 3 compiled)
  ④ Meta-meta-chunks: [ABC]+[DEF]+[GHI] → [ABCDEFGHI] → fires AS ONE
  ⑤ Process becomes INVISIBLE to PFC:
     → PFC sees: INPUT → OUTPUT
     → PFC does NOT see: the dozens of intermediate chunk firings
     → = "Automatic" / "intuitive" / "just know"

  WHY PFC CANNOT SEE PROCESS:
  → PFC observes via Feeling model Feel-Observation (observation of own states)
  → When process runs AS ONE meta-chunk → no intermediate states to observe
  → = Compiled chain runs SUB-PFC (below observation threshold)
  → = Like compiled subroutine in programming: call function → get result → don't see internals
  → 🟡 Framework: tacit = compiled below PFC observation threshold
```

### §2.3 — Tacit knowledge articulation (Somatic-Articulation Loop)

```
TACIT → EXPLICIT transition (when possible):

  Somatic-Articulation Loop §2 provides the mechanism:
  
  ① TACIT state: expert "just knows" → cannot explain
  ② DWELL: PFC maintains attention on tacit output → body responds
  ③ PARTIAL ARTICULATION: fragments emerge
     → "It's something about the angle..."
     → "The color is slightly off..."
     → "The rhythm feels wrong..."
  ④ BODY CONFIRMATION: each fragment → body votes smooth/resist
     → Smooth: "yes, that captures part of it"
     → Resist: "no, that's not quite right"
  ⑤ ITERATE: refine → new fragment → body confirm → refine
  ⑥ EXPLICIT (partial): some tacit knowledge becomes articulable
     → Expert chef: "I can now explain 70% of why 'needs salt'"
     → Remaining 30%: still tacit (too deep for decomposition)

  ⚠️ NOT ALL TACIT CAN BECOME EXPLICIT:
  → Some meta-chunks compiled SO deep → truly irreducible
  → Bicycle balance: vestibular + motor → no verbal decomposition possible
  → Some expert knowledge: pattern recognition across too many variables
  → = Polanyi's point: there IS a permanent tacit residue
  → Framework: agrees — PFC observation has RESOLUTION LIMIT
  → Below resolution limit → permanently tacit

  ⭐ AI ERA IMPLICATION:
  → AI can help ARTICULATE tacit knowledge:
    → Expert: "I just know this design is wrong"
    → AI: "Is it the proportions? The color? The layout?"
    → Expert body votes each suggestion
    → = AI extends Somatic-Articulation Loop
    → = AI catalyzes tacit → explicit transition
  → But AI CANNOT access the tacit knowledge DIRECTLY
  → AI can only PROMPT the expert → expert's body provides answer

## §3 — Gap 2 + H12: Language Evolution

### §3.1 — Gap 2 mechanism (formalized)

```
🟡 FRAMEWORK CONCEPT — Gap 2 (Post-Verbal Bypass):

DEFINITION:
  Gap 2 = situation where INTERNAL PROCESSING creates novel chunk
  BUT existing verbal library has NO FITTING LABEL.

  → Distinct from Gap 1 (pre-verbal: infant has experience, no words yet)
  → Gap 2 = ADULT has extensive vocabulary BUT novel chunk exceeds it
  → = "I know something new but have no words to express it"

MECHANISM:

  ① Internal processing creates novel chunk:
     → Insight (§1) → new connection
     → Tacit accumulation (§2) → new compiled pattern
     → Experience combination → unprecedented situation

  ② PFC searches verbal library:
     → 🟢 Lexical access (Levelt 1989): PFC retrieves verbal labels
     → Search through available words/phrases
     → Attempt to match: "concept X is... like... it's... hmm..."

  ③ NO EXACT MATCH found — 3 possible outcomes:

     (a) APPROXIMATE MATCH:
         → Existing word PARTIALLY fits → use with qualifier
         → "It is like [existing word] but not quite"
         → = Metaphor, simile, analogy as BRIDGE
         → = "That feeling is like being lost but not exactly"

     (b) DESCRIPTION:
         → Use combination of existing words to DESCRIBE
         → "When you scroll through bad news continuously and cannot stop"
         → = Description WORKS but is NOT a single label
         → = Not efficient for repeated use

     (c) NEW WORD / LABEL COINED:
         → Community creates new word for shared chunk
         → "Doomscroll" = single label for Gap 2 concept
         → = Language EVOLVES
         → = H12 prediction

  ④ New word enters vocabulary → becomes verbal anchor (F4 03 Type A)
     → Now: concept has label → shareable → Direction B install possible
     → Others can receive label → compile their own version
     → = Language grows BECAUSE internal processing OUTPACES vocabulary
```

      → = H12 predicts EXACTLY this cross-linguistic variation

### §4.2 — Piaget reconciliation

```
🟡 FRAMEWORK REFRAME — Piaget stages = chunk database thresholds:

PIAGET'S 4 STAGES:
  ① Sensorimotor (0-2): body + object interaction
  ② Preoperational (2-7): symbolic play, language
  ③ Concrete Operational (7-11): logical operations within domain
  ④ Formal Operational (11+): abstract logic across domains

FRAMEWORK REFRAME:
  NOT hardware maturation stages → CHUNK DATABASE THRESHOLDS:

  ① SENSORIMOTOR = F1 territory:
     → Body-input chunks compiling (sensory, motor, proprioceptive)
     → Object permanence = enough object chunks compiled → schema stable
     → = "Compilation phase" — raw input → first chunks
     → Threshold: enough body + object chunks → symbolic capability unlocks

  ② PREOPERATIONAL = F3 begins:
     → Enough LABELED chunks for symbolic play
     → Language chunks accumulating → verbal-as-handle (F1 Convergence Point 6)
     → Can USE symbols but logic WITHIN domain only
     → Threshold: enough labeled chunks → concrete logic capability unlocks

  ③ CONCRETE OPERATIONAL = F4 begins:
     → Enough domain-specific chunks → can CONNECT them (Type 4)
     → Logic works WITHIN domain (sufficient chunks in that domain)
     → = "Conservation" = enough chunks about quantity to detect equivalence
     → Threshold: enough domain chunks → meta-chunk compilation → abstract unlocks

  ④ FORMAL OPERATIONAL = F4 mature:
     → Enough META-CHUNKS across domains → cross-domain reasoning
     → Can hold ABSTRACT relationships (Type 4 on meta-chunks)
     → = "Hypothetico-deductive" = Type 4 chain on abstract meta-chunks
     → Threshold: enough cross-domain meta-chunks → full abstract reasoning

  ⭐ KEY REFRAME:
  → Piaget saw STAGES (discrete jumps) → Framework sees GRADIENT (continuous accumulation)
  → F1 Convergence Point 1 confirmed: chunk compilation is gradient, not discrete
  → "Stage transition" = gradient crossed BEHAVIORAL THRESHOLD
  → Parent observes: "today my child UNDERSTANDS" → actually: gradient crossed threshold TODAY
  → = "Aha at individual level, stage transition at population level"

  ⭐ INDIVIDUAL VARIATION:
  → Different children accumulate chunks at DIFFERENT RATES
  → Environment (F3: education quality, cultural input) affects accumulation rate
  → "Advanced for age" = faster chunk accumulation (richer input environment)
  → "Delayed" = slower chunk accumulation (impoverished input)
  → = NOT hardware difference (in most cases) → INPUT difference
  → 🟢 Consistent with: environmental effects on cognitive development
