## §1 — What is F4: Internal Processing definition

### §1.1 — Core definition

```
F4 = HOW INTERNAL PROCESSING TRANSFORMS EXISTING CHUNKS INTO NOVEL CHUNKS
     — without direct external shared template
     — using PFC hold + body feedback + chunk association

Primary mechanism:
  ① PFC holds 2+ chunks in Working Memory (~3-5 slots)
  ② Check relationships: overlap? conflict? complement?
  ③ Body feedback: "connects" / "feels off" / "aha!"
  ④ Novel chunk emerges: new pattern compiled from combination
  ⑤ Seek label: articulate → anchor → use in plan
```

### §1.2 — Distinguishing F3 vs F4

```
F3 (EXTERNAL INSTALL):
  Source:    people/books/culture/education provide
  Direction: TOP-DOWN (receive label → compile chunk)
  Example:   learn label "gravity" → drop object → see it fall → chunk compiles
  Key:       chunk template exists OUTSIDE
  = Direction B (Logic-Planning.md §7.4)

F4 (INTERNAL TRANSFORM):
  Source:    created by oneself, no external template
  Direction: BOTTOM-UP (feel something → find label)
  Example:   "something's off" about the code → articulate → "ah, God Object!"
  Key:       NOVEL chunk generated INTERNALLY
  = Direction A (Logic-Planning.md §7.4)

OVERLAP:
  The same case can be BOTH F3 and F4:
  Einstein READS Newton (F3: external install of Newtonian chunks)
  → YEARS of thought (F4: internal transform → aha relativity)
  → F3 provides MATERIAL, F4 TRANSFORMS the material

⭐ F4 CANNOT HAPPEN WITHOUT F1 + F3:
  → F1 compiles raw chunks (substrate)
  → F3 installs labels + external knowledge (library)
  → F4 transforms the library into novel insights
  → "Creativity = novel recombination of existing chunks"
    (nothing comes from nothing — every insight builds on existing chunks)
```

### §1.3 — Age-independence

```
F4 mechanism IS THE SAME at every age:

  Age 2:    2 chunks "dog" + "bite" → novel chunk "dogs are dangerous"
  Student:  "triangle" + "Pythagorean theorem" → novel chunk "calculating the hypotenuse"
  Einstein: "Maxwell" + "Galilean transform" → "conflict!" → novel chunk "spacetime"
  Composer: melody A + harmony B → novel chunk "new composition"

  = SAME mechanism: PFC hold → check → body vote → novel emerge
  = DIFFERS: number of chunks + depth + domain sophistication
  = "Intelligence" ≠ better mechanism — "Intelligence" = more + deeper chunks
```
