## §1 — DEFINITION: RESONANCE CORE + BY-PRODUCT MATCH

### §1.1 — 3 principles — "100% separate"

```
🟡 FOUNDATION — everything INSIDE each entity is SEPARATE:

  ① EMOTIONS = 100% SEPARATE:
    Body-feedback = f(hardware × chunk library × current state × context)
    These 4 variables NEVER perfectly match between 2 entities → output is NEVER identical.
    "Both happy playing soccer": the defender feels NE-resolve, the striker VTA-dopamine, the goalkeeper opioid.
    "Happy" = PFC's common label for DIFFERENT body-feedback patterns.
    🟢 Bouchard 1990: identical twins → nearly similar but NOT identical.

  ② GAP = 100% SEPARATE:
    Gap = f(surrounding chunks) — Gap-Direction §1.
    Different chunks → different gaps. Same class: A curious, B corrective, C bored.

  ③ DRIVE = 100% SEPARATE:
    Drive = f(gap + body-feedback + current state).
    Parent's drive "protect" ≠ child's drive "explore" → DIFFERENT but fit each other.

  → Each entity = 1 SEPARATE WORLD. Applies CROSS-SPECIES.
```

### §1.3 — By-Product Match: THE mechanism

```

## §2 — SCOPE: 4 CONDITIONS + OWN DRIVES + WHAT ISN'T RESONANCE

### §2.1 — 4 Conditions REQUIRED SIMULTANEOUSLY (★ NEW v1.1)

```
⭐⭐ 4 CONDITIONS REQUIRED SIMULTANEOUSLY:

  ① POSITIVE VALENCE — approach dominant:
    Both entities fill gaps through approach (novelty, interest, curiosity).
    Approach × Approach = self-reinforcing → highest quality.
    Avoidance × Avoidance = breaks when threat removed → lowest quality.
    Mixed = most common → quality depends on ratio.
    🟢 Elliot 2006: approach/avoidance motivation.

  ② MUTUAL AGENT-MODE — Self-Pattern-Modeling ON for BOTH parties:
    Each entity predicts the other's STATE, not just their output.
    Tool-mode (predict action→outcome) → dissolves when goal ends → 0 resonance.
    Agent-mode (predict entity state→response) → CAN compile → sustainable.
    = Self-Pattern-Modeling must be ON. (Self-Pattern-Modeling §1: Modeling = active forward simulation.)

  ③ GAP OVERLAP — gap distributions overlap sufficiently:
    By-product match REQUIRES gap direction overlap.
    High overlap → high match probability → resonance EASY to emerge.
    Low overlap → still possible but requires PFC effort.
    Gap overlap = Axis ① in 5 Axes (§5).

  ④ OWN DRIVES MAINTAINED — entity must HAVE their own gap + drive:
    Resonance = mutual by-product match.
    By-product = output when entity fills THEIR OWN gap.
    → Entity MUST HAVE their own gap + drive → only then CAN there be a by-product.
    Own drives suppressed → by-product LOST → resonance DIES.
    = WHY Compiled-Suppress = LEVERAGE POINT in Resonance Decline (§10).

  MISSING ANY 1 CONDITION → NOT RESONANCE:
    Positive but one-way → parasocial.
    Mutual but Tool-mode → transactional.
    Agent-mode but gap DOESN'T overlap → mismatch.
    Overlap but drives are suppressed → over-clone → decay.

🟡 4 conditions framework = synthesis from multiple drills.
🟢 Individual conditions: established (per-citation).
```

    By-products CONFLICT gap direction → active friction → costly.

    Self-selection protects: gravitate AWAY. Forced proximity sustains.

## §3 — 2 RESONANCE SCENARIOS + TEMPORAL DYNAMICS

### §3.1 — Scenario ① Direct (compiled, cost ≈ 0)

```
⭐ BY-PRODUCT MATCH NATURAL, NO PFC EFFORT REQUIRED:

  2 SUB-TYPES:

  ①a — HARDWARE BY-PRODUCT (from PROPERTIES / EXISTENCE):
    Entity B EXISTS → A perceives → matches A's hardware gap → A receives reward.
    B DOES NOT NEED TO DO ANYTHING. Unidirectional per person.
    HABITUATES (Weber-Fechner): perception adjusts → normalizes.
    EXCEPTION: evolution-protected bonds (parent-infant) → habituation EXTREMELY SLOW.
    Examples: partner → attractive appearance, parent → newborn (oxytocin), person → cute dog.

  ①b — COMPILED Self-Pattern-Modeling BY-PRODUCT (from mutual Compiled Self-Pattern-Modeling):
    A's Compiled Self-Pattern-Modeling fires → body simulates B's state → responds appropriately.
    → B FEELS understood → B's Self-Pattern-Modeling fires → responds → FEEDBACK LOOP.
    ANTI-HABITUATES: Hebbian each use → stronger → MORE automatic.
    → Cost DECREASES, accuracy INCREASES, reward INCREASES → POSITIVE SPIRAL.
    Example: "long-married couple understands each other in half a sentence" = maximum compiled ①b.
    = DEEPEST + MOST SUSTAINABLE form of resonance.

  ①a + ①b RELATIONSHIP:
    ①a INITIATES + serves as SAFETY CUSHION for ①b (beauty → proximity → Self-Pattern-Modeling begins).
    ①b REPLACES ①a as ①a habituates.
    HEALTHY = ①b grows LARGE ENOUGH to replace declining ①a.
    "The spark is gone" = ①a exhausted + ①b not yet developed.
    "Mature love" = ①b dominant, ①a = bonus.

  🟢 Feldman 2007: interpersonal synchrony.
  🟢 Coan & Sbarra 2015: social co-regulation.
  🟢 Hedonic adaptation: established psychology.
```
