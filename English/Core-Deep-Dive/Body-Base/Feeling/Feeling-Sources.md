# Feeling Sources — Where Feelings Come From by Body Channel

> **"Hungry" comes from the stomach. "Lonely" comes from agent input deficit.**
> **"She's not being honest" comes from the mirror system.**
> **"Peaceful when I get home" comes from schema. "It will rain" comes from prediction.**
> **"I love you" comes from 9+ channels firing simultaneously.**

> multi-source integration makes it hard for PFC to decompose.
>
> This file: the MAP from body channels → the feelings they produce.
> 10+ sources, 50+ examples, case study "love" fully decomposed.

## §1 — L0 Threat Channel Feelings

```
🟢 L0 — THREAT CHANNEL:

  Mechanism:
    → Amygdala threat detection (🟢 LeDoux 1996)
    → HPA axis activation → cortisol release
    → Sympathetic nervous system activation
    → Fast pathway: thalamus → amygdala (~12ms)
    → Slow pathway: thalamus → cortex → amygdala (~40ms+)

  Purpose:
    → Survival — detect danger BEFORE conscious thought
    → Evolution: "better safe than sorry" bias
    → False positive tolerable; false negative lethal

  Feeling characteristics:
    → FAST (milliseconds)
    → LOCATION UNCLEAR (PFC struggles to identify the body source)
    → HIGH INTENSITY (body treats as priority signal)
    → HARD TO VERBALIZE (often no clear "why")
    → OVERRIDES other feelings when active


  EXAMPLES:

    "Something feels wrong" (walking alone at night)
      → L0 fires from: lighting, silence, spatial compression
      → Body knows before PFC can articulate anything
      → Typical response: "I don't know why, but I want to leave"

    "This person feels dangerous"
      → L0 reads: micro-expressions, posture, proximity, eye contact
      → Often accurate; PFC often cannot explain source
      → 🟢 Thin-slice detection (Ambady 1993)

    "I feel like I'm being watched"
      → Sensory pickup of directed attention from others
      → Pre-verbal, fast, survival-grade signal

    "I shudder when I think of that place"
      → Schema-triggered L0 (memory → L0 activation)
      → Note: schema channel can feed L0 retrospectively
      → = Thinking generates body threat response as if real

    "I don't want to take that route today"
      → Predictive L0 — source unclear to PFC
      → 🟡 Gavin de Becker "Gift of Fear" (1997): trust these


  PFC ACCURACY BY LAYER:
    Feel-Consciousification:  HIGH — body reliable, signal crosses
    Feel-Observation:         MEDIUM — often overridden or dismissed
    Feel-Location:            LOW — "whole body" sensation
    Feel-Labeling:            MEDIUM — "fear", "unease", "bad feeling"
    Feel-Explanation:         LOW — confabulation high at this layer


  LITERACY IMPLICATION:
    → Trust Feel-Consciousification–Feel-Observation L0 signals,
      especially in unfamiliar contexts with strangers or new places
    → Don't rationalize away "bad feeling" about a person or place
    → But do verify when possible — L0 also produces false positives
    → Anxiety disorders = L0 chronic over-activation
      (signal accurate MECHANISM, but context no longer warrants it)


  Research:
    🟢 LeDoux 1996 "The Emotional Brain"
    🟢 Amygdala threat detection (extensive literature)
    🟡 de Becker 1997 (practical, not peer-reviewed)
    🟢 Ambady & Rosenthal 1993 thin slices
```

## §2 — L1 Physiological / Interoceptive Feelings

```
🟢 L1 — PHYSIOLOGICAL / INTEROCEPTIVE CHANNEL:

  Mechanism:
    → Interoception afferent pathways (🟢 Craig 2002, 2013)
    → Insula as integration hub (Feeling-Mechanism-Deep §2)
    → Vagal afferents + spinal afferents
    → Homeostatic monitoring of organ states (continuous)

  Purpose:
    → Maintain homeostasis — keep body within viable range
    → Signal deficits (hunger, thirst, fatigue)
    → Signal damage (pain, illness)
    → Continuous body ↔ brain communication

  Feeling characteristics:
    → MEDIUM-SLOW timescale (minutes to hours typically)
    → LOCATION ACCURACY VARIES (somatic HIGH, visceral LOW)
    → Graduated intensity: mild → acute → severe
    → Chronically ignored in modern life — until critical


  EXAMPLES BY SUBCATEGORY:

  ── HUNGER / THIRST / SATIETY ──
    "Hungry" (gastric signals, ghrelin)
    "Thirsty" (osmotic + HR change, vasopressin)
    "Full" (stretch receptors in stomach)
    "Craving something specific" (sweet, salty, fat, protein)
    "Satiated for this food type" (taste-specific fullness)

  ── FATIGUE / REST ──
    "Tired" (systemic fatigue, multi-source)
    "Exhausted" (severe depletion, distinct from tired)
    "Sleepy" (circadian + homeostatic sleep pressure)
    "Groggy" (morning sleep inertia)
    "Energized" (well-rested, post-exercise state)

  ── PAIN / DISCOMFORT ──
    "Stomach ache somewhere in here" (visceral — LOW location accuracy)
    "My foot is asleep" (somatic — HIGH location accuracy)
    "Headache" (vascular / muscular / tension subtypes)
    "Muscle soreness after workout" (DOMS)
    "Itching right here" (local somatic, precise location)
    "Throat burns" (mucosal)

  ── TEMPERATURE ──
    "Cold" (hypothalamus → shivering, vasoconstriction)
    "Hot" (sweating, vasodilation)
    "Chilled through" (core temperature drop)
    "Feverish" (hyperthermia, interleukin signaling)

  ── RESPIRATION / CARDIOVASCULAR ──
    "Can't breathe" (chemoreceptors, CO2 buildup)
    "Heart pounding" (sympathetic activation)
    "Breathless" (exertion)
    "Chest tight" (anxiety-related or physical)

  ── SEXUAL / AROUSAL ──
    "Aroused" (sexual arousal)
    "Stimulated" (sensory/touch response)
    Note: highly cultural in LABELING, mechanism = universal)


  LOCATION ACCURACY SUBTLETY:
    Somatic pain (skin, muscle): HIGH accuracy
      → "Sharp pain in my right pinky toe" = precise
    Visceral pain (organs): LOW accuracy → referred pain common
      → "Stomach ache" = approximate, location fuzzy
      → Heart attack → arm/jaw pain (referred)
      → Appendicitis → initially generalized, then localizes
    → Expert (doctor, experienced patient) learns to discriminate
    → This matters for self-diagnosis and communication


  PFC ACCURACY BY LAYER:
    Feel-Consciousification:  HIGH — body state generates clear signal
    Feel-Observation:         HIGH if attention is allocated
    Feel-Location:            VARIES (somatic HIGH, visceral LOW)
    Feel-Labeling:            HIGH for common states, LOW for subtle
    Feel-Explanation:         MEDIUM — often confused with emotional states
      (e.g.: hunger → irritability misread as anger;
             thirst → fatigue misread as laziness)


  LITERACY IMPLICATION:
    → Before labeling an emotion, check physiological basics:
      "Am I hungry? Thirsty? Tired? In pain? Need to move?"
    → This simple check prevents many mislabels at Feel-Labeling
    → Pain literacy: describe quality + location + timing + intensity
    → "Hangry" = classic L1 misread as emotional state


  Research:
    🟢 Craig 2002 "How do you feel? Interoception: the sense of the physiological condition of the body"
    🟢 Craig 2013 synthesis
    🟢 Sherrington proprioception (classical foundation)
    🟢 Pain research — nociception + central processing
```

## §3 — L2 Experience / Novelty Feelings

```
🟢 L2 — EXPERIENCE / NOVELTY CHANNEL:

  Mechanism:
    → VTA dopaminergic SEEKING system (🟢 Panksepp)
    → Novelty detection circuits → habituation curves
    → Reward prediction error (🟢 Schultz 1997)
    → Delta detection: change from baseline → dopamine

  Purpose:
    → Drive exploration of the environment
    → Expand behavioral repertoire
    → Maintain cognitive engagement
    → Detect change = signal something worth attending

  Feeling characteristics:
    → MEDIUM timescale (seconds to hours)
    → Positive valence typically
    → Context-dependent intensity
    → Habituation-sensitive: same thing → signal fades


  EXAMPLES:

    "Bored" (habituation, SEEKING drive not engaged)
      → Same stimulus, novelty depleted
      → Body signal: seek new input
      → = System working correctly: flagging need for variation

    "Interesting!" (novelty reward fires)
      → New pattern detected above threshold
      → Engagement increases automatically

    "Curious" (SEEKING drive active)
      → Information gap detected
      → Motivation to close the gap

    "Drawn to this" (approach drive)
      → Novelty + positive prediction → body moves toward

    "Excited" (high arousal + positive valence)
      → Novelty at peak + anticipation of reward
      → Dopamine + anticipatory opioid

    "Bored AND unhappy" (negative valence when nothing novel)
      → L2 deficit signal + negative valence overlay
      → = Not just "bored" — body signals meaningful deprivation


  HABITUATION DYNAMICS:
    → First exposure: high novelty reward
    → Repeated exposure: adaptation curve
    → Asymptote: minimal reward from same stimulus
    → "Bored" = body correctly signaling: need new input

    Counter: deliberate novelty injection
    → Routine daily life → habituation → "why do I feel flat?"
    → New activity / travel / learning → novelty refresh
    → = L2 explanation for why breaks, learning, and variety feel good


  PFC ACCURACY BY LAYER:
    Feel-Consciousification:  MEDIUM — subtle, often background hum
    Feel-Observation:         MEDIUM — easy to miss ("I'm just bored")
    Feel-Labeling:            HIGH — common vocabulary exists
    Feel-Explanation:         MEDIUM — causes often misattributed


  LITERACY IMPLICATION:
    → Recognize "bored" as a body signal, not a personal failing
    → Respond to boredom with deliberate novelty (not passive distraction)
    → "Curious" as a reliable compass for learning and creative work
    → Vary contexts deliberately to counter habituation


  Research:
    🟢 Panksepp SEEKING system (1998, 2012)
    🟢 Schultz reward prediction error (1997)
    🟢 Berridge wanting vs liking distinction (2009)
```

## §4 — Mastery / Meaning / Status (Evaluative) Feelings

```
🟢 MASTERY / MEANING / STATUS — EVALUATIVE OBSERVATION PARAMETERS:

  Mechanism:
    → Ventral striatum + mPFC integration
    → Self-schema firing (compiled self-models)
    → Social comparison circuits
    → Long-timescale reward encoding

  Purpose:
    → Drive competence development
    → Maintain sense of meaning and purpose
    → Navigate social hierarchy
    → Long-term life optimization (decades-scale)

  Feeling characteristics:
    → LONG timescale (hours to years)
    → Schema-mediated: compiled, not raw input
    → Self-concept dependent — who you think you are shapes these
    → Culturally shaped at Feel-Labeling level


  EXAMPLES:

  ── MASTERY ──
    "Confident" (compiled mastery history fires)
      → Schema: "I've succeeded at this kind of thing before"
    "Proud" (achievement recognized, by self or others)
      → Reward system + self-schema fire together
    "Useless" (mastery deficit schema)
      → Self-schema: "I can't do anything right"
    "Capable" (skill ownership felt)
      → Mastery history compiled into identity
    "Helpless" (learned helplessness)
      → Chronic failure + schema calcified: "nothing I do matters"

  ── MEANING ──
    "This matters" (meaning schema fires)
      → Current activity aligns with deep compiled schema
    "Pointless" (meaning deficit)
      → Activity disconnected from schema: "why am I doing this?"
    "I'm going in the right direction" (path alignment)
      → Long-term trajectory feels coherent
    "Lost" (meaning disorientation)
      → Schema about future unclear or shattered
    "Just surviving" (bare meaning)
      → Survival mode: meaning thin, reserve depleted

  ── STATUS ──
    "Respected" (status received)
      → Others' behavior signals status adequate or high
    "Diminished" (status violation)
      → Others' behavior signals low regard
    "On equal footing" (status parity)
      → Comfortable baseline, no monitoring required
    "Ashamed" (public status loss)
      → Social signal of status threat, body responds viscerally
    "Chronically insecure" (status deficit calcified)
      → Self-schema about status locked in negative position


  SCHEMA-MEDIATED NATURE:
    → Evaluative feelings are NOT "raw input"
    → They are schema FIRING ABOUT BODY STATE
    → Schema = built from years of accumulated history
    → Changes slowly, feels deeply — harder to update than L1 states

  Decomposition example:
    "Useless" (feeling) =
      + self-schema ("I am someone who fails")
      + history compiled ("I've failed many times before")
      + current context (just failed at task)
      + valence: negative
    → Fires as 1 feeling "useless" — but IS multi-channel


  PFC ACCURACY BY LAYER:
    Feel-Consciousification:  HIGH — body deeply engaged when these fire
    Feel-Observation:         HIGH
    Feel-Location:            HARD — often "whole body" or "chest/stomach"
    Feel-Labeling:            HIGH for common states, LOW for subtle
    Feel-Explanation:         OFTEN WRONG — rationalization is heavy here


  LITERACY IMPLICATION:
    → Evaluative feelings encode long-term patterns, don't dismiss them
    → But also: don't react to them as immediate situational truth
    → Verify schema: "Is this schema still accurate, or am I reacting
      to a pattern from years ago?"
    → Therapy often targets evaluative schema updates


  Research:
    🟢 Ventral striatum long-term reward encoding
    🟢 mPFC self-referential processing
    🟢 Festinger 1954 social comparison theory
    🟢 Seligman 1967 learned helplessness
```

## §5 — Agent Input Channel Feelings

```
🟢 AGENT INPUT — A BODY-BASE NEED (independent channel):

  Framework insight:
    Agent input = body-base need REAL, not just a "social nice-to-have"
    Body requires the presence of other agents to regulate.

  Research support:
    🟢 Social Baseline Theory (Coan 2015) — brain defaults to social
    🟢 Social buffering (Kiyokawa 2004) — presence reduces stress response
    🟢 Eisenberger 2003 — social pain = physical pain circuits (fMRI)
    🟢 Bucharest Early Intervention — agent deprivation damages development
    🟢 Solitary confinement research — extreme deprivation = severe harm


  Mechanism:
    → Oxytocin (bonding, trust, co-regulation)
    → Co-regulation of heart rate, cortisol, autonomic state
    → Mirror system activation (continuous background)
    → Ventral vagal (Porges polyvagal 🟡)

  Purpose:
    → Body needs other bodies to thrive
    → Social presence recalibrates internal state
    → Agent input = primary regulation channel in humans


  FEELING MANIFESTATIONS:

  ── AGENT PRESENT (need met) ──
    "Warm when near mom"
      → Parasympathetic dominance + oxytocin
    "Safe when with a friend"
      → Co-regulation: cortisol suppressed
    "Connected to that person"
      → Mirror active + schema: "trusted person"
    "Peaceful being together"
      → Deep regulation state
    "Not alone"
      → Baseline social need met: body settles

  ── AGENT ABSENT (deficit) ──
    "Lonely" (acute agent deficit)
      → Body signals need for presence
    "Isolated" (chronic deficit)
      → Schema forms around aloneness: "this is my life"
    "Craving company" (active seeking drive)
      → Drive to find contact
    "Empty" (deep agent deficit)
      → Chronic lack calcified into baseline state
    "Abandoned" (expected agent missing)
      → Schema violation + L0 threat fires simultaneously
    "Missing someone specific"
      → Particular agent needed, no substitute

  ── AGENT PARTIAL / MEDIATED ──
    "Lonely in a crowd"
      → People present but no real connection registers
    "Close but distant" (physical proximity, no psychic presence)
    "Being ignored"
      → Agent present but denying input
    "Lonely online"
      → Mediated agents (social media) not satisfying the real need


  ⭐ CRITICAL INSIGHT:
    Agent input is NOT a "want" — it is a biological NEED.
    Body suffers real damage from agent deprivation,
    just as it does from water or food deprivation.

    → Reframes loneliness: not "weakness" — it is a body distress signal
    → Reframes love: not "luxury" — it is regulation infrastructure
    → Reframes solitary confinement: physiologically, it is torture


  MODERN ERA COMPLICATION:
    → Agent input evolved for in-person, face-to-face presence
    → Modern substitutes: text, video call, para-social, crowds
    → Mediated agents: partial substitute, not full match
    → "Lonely in a crowd" = real and common phenomenon
    → Social media can paradoxically DEEPEN the deficit
      (dopamine-stimulating, connection-substituting, but not meeting need)


  PFC ACCURACY BY LAYER:
    Feel-Consciousification:  MEDIUM — often normalized as "life"
    Feel-Observation:         VARIES — cultures differ in attending to loneliness
    Feel-Labeling:            HIGH for acute, LOW for chronic
    Feel-Explanation:         OFTEN WRONG — "I just need more work" /
                             "I'm fine alone" / "I need space"


  LITERACY IMPLICATION:
    → Recognize agent deficit as body signal, not personal defect
    → Distinguish QUANTITY (number of contacts) from QUALITY (depth)
    → "Lonely" = valid body signal deserving a real response
    → Don't rationalize the need for presence away


  Research:
    🟢 Coan 2015 Social Baseline Theory
    🟢 Kiyokawa 2004 social buffering
    🟢 Eisenberger 2003 social pain
    🟢 Bucharest Early Intervention Project (Nelson et al.)
    🟢 Holt-Lunstad 2015 — loneliness mortality meta-analysis
```

## §6 — Mirror-Generated Feelings

```
🟢 MIRROR — FEELINGS ABOUT OTHERS' STATES:

  Mechanism:
    → Mirror = byproduct of agent processing
    → Pattern matching + own-body simulation + schema activation
    → 🟢 Gallese 2007 embodied simulation
    → 🟢 Rizzolatti 1996 mirror neurons discovery

  Purpose:
    → Predict agent behavior (primary evolutionary function)
    → Coordinate actions with others
    → Social learning by observation
    → Emotional contagion (intentional or not)

  Feeling characteristics:
    → FAST (milliseconds to seconds)
    → Source OFTEN MISATTRIBUTED to self
    → Thin-slice accuracy can be surprisingly high
    → Contagious — spreads through social groups


  EXAMPLES:

  ── READING OTHERS ──
    "I don't trust him"
      → Mirror reads micro-tension, eye avoidance, body posture
      → PFC often cannot explain the specific cues
    "She's hiding something"
      → Incongruence detection (verbal vs somatic mismatch)
      → 🟢 Ambady thin-slice research
    "He's kind"
      → Positive behavioral pattern recognized, mirrored
    "He says he's fine but he's not"
      → Mirror reads body truth over verbal claim
    "She's about to blow up"
      → Pre-anger somatic cues detected before she knows herself

  ── GROUP / ATMOSPHERE ──
    "Tension in the room"
      → Multi-agent mirror integration → collective state
    "Something's off in here"
      → Similar to above; body reads group before PFC articulates
    "Good energy in this room"
      → Group regulated state read collectively

  ── EMOTIONAL CONTAGION ──
    "I feel good because they're happy"
      → Mirror + own reward system fire together
    "I feel sad being around them"
      → Contagious sadness (mirror mechanism)
    "Their stress is infecting me"
      → Cortisol mirror response
    "Laughing because they're laughing"
      → Deep mirror activation (involuntary, hard to suppress)

  ── MIRROR REWARD / CARE ──
    "I suffer when I see them suffer"
      → Ventral empathy fires (🟢 Singer 2004 pain empathy circuits)
    "I feel good when they succeed"
      → Vicarious reward (mirror opioid)
    "Baby smiling → warmth in me"
      → CARE system activation (🟢 Panksepp)


  ⭐ MECHANISM (from Domain-Mechanisms.md):
    Mirror feelings do NOT come from a separate "empathy system."
    Mirror = 3 layered mechanisms:
      ① Pattern matching (pre-mirror, limbic — fast, automatic)
      ② Agent modeling (mirror proper — from 6-24 months development)
      ③ Schema simulation (post-mirror, compiled — richest fidelity)

    Mirror feelings = YOUR OWN BODY FIRING as if you were
    experiencing what the other person is experiencing — but WEAKER.
    = Embodied simulation. = You feel a "copy" of their state.


  PFC ACCURACY BY LAYER:
    Feel-Consciousification:  HIGH — own body simulation is real
    Feel-Observation:         OFTEN MISSED — background, automatic
    Feel-Location:            HARD — "whole body" or "chest"
    Feel-Labeling:            MEDIUM — may be mislabeled as own feeling
    Feel-Explanation:         OFTEN WRONG — "I'm anxious because of X"
                             (when actually catching someone else's anxiety)


  MIRROR MISATTRIBUTION (common everyday error):
    → Enter a room where group is anxious → feel anxious
    → PFC: "I must be anxious about something in MY life"
    → Actually: mirroring the group's state
    → Not a bug — mirror evolved precisely for co-regulation
    → Bug: when PFC confabulates an internal explanation


  LITERACY IMPLICATION:
    → When a feeling arises in social context, ask first:
      "Is this mine, or am I picking it up from someone else?"
    → This distinction matters for response:
      own feeling → address own needs
      mirrored feeling → may belong to the other person
    → Therapists learn counter-transference = trained mirror awareness
    → "I feel bad around this person" may = their state, not your problem


  Research:
    🟢 Gallese 2007 embodied simulation
    🟢 Rizzolatti 1996 mirror neurons
    🟢 Ambady 1993 thin slices
    🟢 Hatfield 1994 emotional contagion
    🟢 Singer 2004 pain empathy circuits
    🟡 Heyes ASL theory (2010+) — alternative framework
```

## §7 — Schema-Triggered Feelings

```
🟢 SCHEMA — FEELINGS FROM COMPILED KNOWLEDGE:

  Mechanism:
    → Compiled knowledge fires when pattern matches
    → Schema stores EMOTIONAL WEIGHT, not just facts
    → Memory reactivation includes somatic marker (🟢 Damasio)
    → Long-term potentiation + reconsolidation

  Purpose:
    → Fast response to familiar situations — no recomputation needed
    → Emotional regulation based on accumulated history
    → Rapid meaning assignment when context matches pattern

  Feeling characteristics:
    → Can fire INSTANTLY when pattern matches
    → Often feels like "from nowhere" — schema source is pre-verbal
    → Body responds AS IF real input is present
    → Schema and current reality can diverge


  EXAMPLES:

  ── POSITIVE SCHEMA ──
    "Peaceful when I get home"
      → Home schema fires → body relaxes
    "Nostalgia hearing that old song"
      → Memory schema + original feeling reactivated
    "Safe with religious faith"
      → Belief schema fires → body settles into trust state
    "Warm thinking of childhood"
      → Memory schema compiled positive
    "I trusted him immediately"
      → Pattern matched a trusted-person schema

  ── NEGATIVE SCHEMA ──
    "I shudder thinking of that place"
      → Trauma schema fires + L0 activation
    "I don't want to go in that room"
      → Schema associated with a bad memory
    "I go cold when I hear that name"
      → Schema + L0 threat fire simultaneously
    "My stomach hurts when I remember that"
      → Schema-generated physical response (real pain, not imagined)

  ── ABSTRACT SCHEMA ──
    "Justice was done" (after a verdict)
      → Justice schema fires → satisfaction
    "That's deeply unfair"
      → Schema violation + intense reaction
    "Guilt"
      → Moral schema + self-judgment pattern fires
    "Sacred" (in a cathedral, a mountain, a moment)
      → Transcendence schema fires


  ⭐ BODY CANNOT DISTINGUISH SCHEMA FROM REAL INPUT:
    Critical framework point:
    → Body fires when schema fires
    → Body fires when real input fires
    → SAME physiological response to both
    → = Schema-generated feeling is PHYSICALLY REAL

    Examples:
    "Thinking of someone who died" → body grieves
      (schema fire — grief response is real)
    "Thinking about tomorrow's deadline" → cortisol rises
      (schema fire — stress response is real)
    "Reading a scripture or inspiring text" → body finds peace
      (schema fire — parasympathetic response is real)

    = This is why placebo and nocebo are medically real.
    = This is why PTSD is physically real, not "in the head."


  ACCURACY SUBTLETY:
    Schema fires based on PATTERN MATCH — but the match can be WRONG:
    → Current situation ≠ past situation exactly
    → Schema fires reactive feeling anyway
    → Person reacts to the PAST, not the present

    → PTSD: old trauma pattern fires at safe context
    → Prejudice: old category schema fires at new individual
    → Triggered reactions: schema overrides current evidence

    Wise approach: notice when schema is firing → check current reality
    → Is the pattern match accurate HERE, NOW?


  PFC ACCURACY BY LAYER:
    Feel-Consciousification:  HIGH — body reacts fully
    Feel-Observation:         VARIES — often unnoticed until strong
    Feel-Location:            HARD — "whole body" typical
    Feel-Labeling:            HIGH if schema has a common name
    Feel-Explanation:         OFTEN WRONG — attributes to current
                             context, not to the schema firing


  LITERACY IMPLICATION:
    → When strong feeling arises "from nowhere," ask:
      "What schema is firing? What does this situation remind me of?"
    → Is the current situation matching ACCURATELY, or is it template-matching?
    → Trauma work = updating schema responses (reconsolidation)
    → Therapy is often = help schema update with current reality


  Research:
    🟢 Damasio somatic marker — emotional memory in body
    🟢 Long-term potentiation / long-term depression
    🟢 Nader 2000 reconsolidation research
    🟢 PTSD research — trauma schema activation mechanism
    🟢 Placebo / nocebo research — schema effect on body state
```

## §8 — Imagine-Final Preview Feelings

```
🟢 IMAGINE-FINAL — FEELINGS FROM FUTURE PREVIEW:

  Mechanism (Imagine-Final.md for full detail):
    → PFC simulates possible future → body responds to simulation
    → "As if" feelings — weaker than real but physiologically present
    → Core to 🟢 Damasio's somatic marker hypothesis

  Purpose:
    → Preview outcomes BEFORE committing to action
    → Guide decisions by felt comparison of options
    → Anticipatory planning with body input
    → Risk assessment through felt sense — not just logic

  Feeling characteristics:
    → VOLUNTARY or AUTOMATIC (can be deliberately run or spontaneous)
    → Strength varies with simulation vividness and expertise
    → Often rejected by analytical thinking ("that's just imagination")
    → BUT validated by wise decision-making (Damasio insight)


  EXAMPLES:

  ── POSITIVE PREVIEWS ──
    "This plan will work" (felt conviction, not reasoned certainty)
      → Preview run → positive outcome felt in body
    "This is a good idea"
      → Coherence detected in simulation → body smooth
    "It's going to be great"
      → Anticipation preview → reward fires in advance
    "I miss them and I want to see them"
      → Preview reunion → mirror reward felt in advance
    "I'll succeed"
      → Multiple simulation paths converge positive

  ── NEGATIVE PREVIEWS ──
    "I'll be rejected"
      → Preview of rejection → body contracts
    "This is going to go wrong"
      → Multiple simulations find problems → body resists
    "I'll regret this"
      → Post-hoc preview of regret (retrospective future)
    "I can't live without X"
      → Preview of absence → loss response fires
    "It will hurt to lose this"
      → Preview of loss → anticipatory grief

  ── ENVIRONMENTAL / PREDICTIVE ──
    "It's going to rain"
      → Body reads pressure, humidity, sky patterns
      → Prediction feeling — often accurate
    "I shouldn't take that route today"
      → Predictive feeling; source often unclear to PFC
    "Something's about to happen"
      → Vague anticipation — L0 + cognitive mixed

  ── DECISION PREVIEW ──
    "Option A feels right"
      → Preview A results better than B
    "Option B actually fits better"
      → Body votes via somatic marker (Damasio)
    "Neither option feels right"
      → Preview shows no path satisfies current needs


  ⭐ DAMASIO'S KEY INSIGHT:
    VMPFC patients (Phineas Gage type) retain logic but LOSE preview feelings.
    → Can reason about options perfectly well
    → Cannot FEEL the difference between options
    → Decision-making becomes PARALYZED or random
    → = Preview feeling is NECESSARY for decision, not optional flavor

    Implication: "purely rational decisions" excluding feelings are IMPOSSIBLE.
    Wise decisions INTEGRATE preview feelings WITH logical analysis.


  ACCURACY NOTE:
    → Depends on simulation accuracy — expertise improves this
    → 🟢 Klein 1998: expert practitioners trust preview feelings; novices don't
    → Novice simulations often miss key variables → unreliable preview
    → = Preview feeling reliability = function of expertise depth


  PFC ACCURACY BY LAYER:
    Feel-Consciousification:  HIGH when simulation is vivid
    Feel-Observation:         Depends on attention allocation
    Feel-Location:            Often whole-body
    Feel-Labeling:            Usually labeled "gut feeling", "intuition", "sense"
    Feel-Explanation:         Can be explained by retrieving the simulation


  LITERACY IMPLICATION:
    → Don't dismiss "gut feeling" about future options
    → Verify simulation quality: what expertise do I have here?
    → Deliberately run previews before important decisions
    → Notice preview feelings even in small choices — body is always running them


  Research:
    🟢 Damasio 1994 somatic marker hypothesis
    🟢 Klein 1998 naturalistic decision making
    🟢 Kahneman 2011 System 1 = fast preview-based assessment
    🟢 Gilbert 2007 "Stumbling on Happiness" (affective forecasting)
```

## §9 — Valence Compiled Feelings

```
🟢 VALENCE — FEELINGS FROM COMPILED ASSESSMENTS:

  Mechanism (Valence.md for detail):
    → Repeated interactions compile into valence profiles
    → Entity evaluated across multiple channels over time
    → Compiled assessment fires as single, fast feeling
    → Updated slowly by new experience

  Purpose:
    → Fast evaluation of familiar entities — no recomputation
    → Efficient domain navigation (what to approach / avoid)
    → Compressed meaning assignment

  Feeling characteristics:
    → Entity-specific (about a thing, person, place, idea)
    → Stable over time — slow to update
    → Single-word label typically
    → Multi-channel state compressed


  EXAMPLES:

  ── POSITIVE VALENCE ──
    "I like that tree" (positive compile: aesthetic + memory + schema)
    "I love this house" (multi-channel positive: safety + history + aesthetic)
    "I'm obsessed with this cat" (agent input + novelty + warmth)
    "I love this book" (cognitive fit + meaning + resonance)

  ── NEGATIVE VALENCE ──
    "I hate that name" (compiled negative: association + phonetic + history)
    "I can't warm up to that person" (mirror + schema + valence)
    "I'm over this food" (habituation + taste saturation)
    "That room makes me uncomfortable" (threat + disgust + aesthetic)

  ── COMPLEX VALENCE ──
    "Love" (case study §12 below) — 9+ channels converged
    "I love them but also resent them" (mixed valence — Valence.md §7)
    "I respect them and also fear them" (dual compile: admiration + threat)
    "I want it but also don't" (approach-avoid conflict, unresolved)


  SINGLE-WORD COMPRESSION:
    Valence feelings compress a multi-channel state → 1 word.
    Compression ratio can be 9:1 or higher.
    → Information lost → functionality preserved
    → PFC accesses the "love" label; body knows the full state

  Example:
    "Love" →
      + L0: safe (no threat activation)
      + L1: calm regulated body
      + L2: novelty still present (SEEKING engaged)
      + Meaning schema active (deep significance)
      + Agent input fully met
      + Mirror + mutual positive state
      + Imagine-Final: positive future preview
      + Schema: "this person is mine, I am theirs"
      + Compiled valence: strongly positive
    → Compressed to: "love" (one syllable)


  WHY MULTI-CHANNEL FEELINGS FEEL "INEFFABLE":
    When asked "why do you love them?" — honest answer is hard:
    → Any single reason understates radically
    → List of reasons feels mechanical, reductive
    → "Because they are them" = closest HONEST answer
    → = Acknowledging compression loss, not forcing false decomposition

    → Framework: "love feels ineffable" = compression phenomenon.
    Not mysticism. Not romantic vagueness.
    Body knows 9+ channels. PFC approximates.


  PFC ACCURACY BY LAYER:
    Feel-Consciousification:  HIGH — strong signal
    Feel-Observation:         HIGH if attending
    Feel-Location:            Whole-body typical
    Feel-Labeling:            Single word (compression lossy but functionally strong)
    Feel-Explanation:         Hard to explain fully; any attempt is partial


  LITERACY IMPLICATION:
    → Valence feelings encode a lot of information — don't dismiss "I just don't like this"
    → But: valences can become outdated or built on biased history
    → Verify periodically: "Is this valence still accurate, or is it obsolete?"


  Research:
    🟢 Classical + operant conditioning
    🟢 Ventral striatum compiled reward encoding
    🟢 Somatic marker — Damasio
    🟢 Attitude research (Fiske, Cuddy et al.)
```

## §10 — Cognitive / Prediction Feelings

```
🟢 COGNITIVE / PREDICTION — FEELINGS FROM THINKING:

  Mechanism:
    → Pattern matching + prediction-delta detection
    → ACC conflict / error signals (🟢 Botvinick 2004)

    → Insight neural signature (🟢 Kounios 2014)

  Purpose:
    → Signal cognitive rightness / wrongness
    → Guide learning: what deserves more attention?
    → Enable expert intuition — accumulated patterns fire fast
    → Alert to inconsistencies before explicit detection

  Feeling characteristics:
    → Domain-specific: only reliable within areas of expertise

  ── WRONGNESS FEELINGS ──

      → Pattern mismatch felt before reasoning through it

      → Coherence check failed

    "Brain fog"

      → Massive prediction-delta overwhelming integration

  ── COMPREHENSION FEELINGS ──

      → 🟢 Kounios insight research: distinct neural signature

      → Same phenomenon, different vocabulary
    "Grokking" (Heinlein)

    → Trust cognitive feelings INSIDE your domain of expertise
```

## §11 — 7 Categories of Feelings (Regrouped by Phenomenal Type)

```
🟡 PHENOMENAL REGROUPING — A COMPLEMENTARY VIEW:

  Sections §1-10 mapped feelings by CHANNEL / SOURCE (mechanism-first).
  Section §11 regroups by PHENOMENAL TYPE — how the feeling presents.

  The same feeling can draw from multiple channels.
  The same channel can produce multiple phenomenal types.
  Both views are useful; neither is complete alone.


  7 CATEGORIES:


  11.1 — BODY STATE FEELINGS
  ───────────────────────────
    Pure physiological reports, primarily L1.
    → "Hungry", "thirsty", "tired", "in pain", "cold", "itchy"
    → Clearest PFC access at Feel-Location → Feel-Labeling
    → Natural entry point for feeling literacy training
    → Easy to verify and act on directly


  11.2 — EMOTIONAL FEELINGS (NARROW SENSE)
  ─────────────────────────────────────────
    What mainstream calls "the emotions."
    → "Happy", "sad", "angry", "afraid", "loving", "disgusted"
    → 🟢 Panksepp 7 primary systems (SEEKING / FEAR / RAGE / LUST / CARE / PANIC / PLAY)
    → Cultural labeling varies significantly
    → = Subset of feeling — NOT all of feeling
    → Framework: emotions are 1 category inside the larger feeling space


  11.3 — SOCIAL READING FEELINGS (MIRROR-BASED)
  ──────────────────────────────────────────────
    Feelings about others, from mirror mechanism.
    → "She doesn't trust me"
    → "Tension in the room"
    → "He's hiding something"
    → "Bad vibe"
    → Often misattributed to self — treated as own feelings
    → = Social intelligence = literacy for these feelings


  11.4 — COGNITIVE FEELINGS
  ──────────────────────────
    Rightness / wrongness from cognitive processing.
    → "Got it!"
    → "Something's wrong here"
    → "Stuck"
    → "I understand now"
    → Expert intuition lives here
    → = Domain-specific, trained, compiled over time


  11.5 — PREDICTIVE / INTUITIVE FEELINGS
  ───────────────────────────────────────
    About future outcomes, from Imagine-Final preview mechanism.
    → "This will work"
    → "This is going to go wrong"
    → "I have a feeling about this"
    → "Gut feeling about the decision"
    → = Decision-critical; Damasio: necessary, not optional


  11.6 — VALENCE FEELINGS
  ────────────────────────
    Like / dislike about entities — compiled over history.
    → "Love", "hate"
    → "Like", "dislike"
    → "Into it", "over it"
    → Fast, stable, entity-specific
    → = Compressed history: changing them requires new experiences


  11.7 — META FEELINGS
  ─────────────────────
    Feelings ABOUT feelings.
    → "Sad that I'm sad again" (meta-sadness)
    → "Angry at myself for getting angry" (meta-anger)
    → "I hate that I feel this way"
    → "Guilty for feeling..." (moral meta-feeling)
    → Can spiral (depression pattern)
    → Or stabilize (mindful awareness: "I notice I feel sad — that's okay")
    → = The upper layer in the 7-layer model applied reflexively


  WHY 2 VIEWS MATTER:
    → Channel view (§1-10): diagnostic — trace where the feeling comes from
    → Phenomenal view (§11): recognition — vocabulary for what it FEELS LIKE
    → Together: complete map of the feeling landscape
    → Literacy = fluency with both views
```

## §12 — Case Study: "Love" Decomposed

```
⭐ CASE STUDY — WHY A SINGLE WORD CAPTURES A MULTI-CHANNEL STATE:

  Framework claim: feelings are multi-source integrated.
  Best demonstration: "love" — notoriously hard to explain.

  Let's decompose it.


  WHAT "I LOVE YOU" TYPICALLY INVOLVES (ideal case):

  ── L0 THREAT CHANNEL ──
     "You are safe" — body does not activate threat around you
     "I'm not afraid when I'm with you"
     → L0 baseline: minimal activation
     → Security enables all other channels to deepen


  ── L1 PHYSIOLOGICAL CHANNEL ──
     "My body relaxes around you"
     → Parasympathetic dominance (calm, settled)
     → Heart rate slows; background tension normalizes
     → Cortisol low
     → Mild arousal if romantic (HPA + sexual systems)
     → Comfort touch → oxytocin release
     → Co-regulation of physiological rhythms (🟢 Coan 2015)


  ── L2 EXPERIENCE / NOVELTY CHANNEL ──
     "You always have something new"
     → Not fully habituated — SEEKING remains engaged
     → Interactions retain novelty dimension
     → Discoveries, surprises, learning WITH you


  ── MASTERY / MEANING / STATUS (EVALUATIVE) CHANNEL ──
     "You make my life feel meaningful"
     → Meaning schema fires strongly
     → "We together" = identity component
     → Status: mutual respect sustained
     → Long-term narrative centers you
     → Self-schema includes you as permanent feature


  ── AGENT INPUT CHANNEL ──
     "You fill my agent need deeply"
     → Body's agent input channel fully met
     → Co-regulation present (body regulated by presence)
     → Baseline loneliness suppressed
     → Without you: immediate agent input deficit fires
     → = Biological, not metaphorical


  ── MIRROR CHANNEL ──
     "I feel you love me too"
     → Mirror reading your body back to you
     → Detects safety, trust, love returned
     → Mutual mirror = positive feedback loop
     → Your happiness → my happiness (emotional contagion)
     → Your sadness → my sadness


  ── IMAGINE-FINAL PREVIEW CHANNEL ──
     "The future with you feels right"
     → Preview of future together: positive valence
     → Preview of future without you: negative valence (loss)
     → Preview runs constantly in background
     → Guides commitment behaviors without explicit reasoning


  ── SCHEMA CHANNEL ──
     "You are mine, I am yours"
     → Relationship schema formed (compiled knowledge about you)
     → Fires fast when you're present
     → Updates with new experiences
     → Can be shaken by violations (betrayal, prolonged distance)


  ── COMPILED VALENCE CHANNEL ──
     "Strong positive valence on you as entity"
     → Multi-channel converged over time → compiled
     → Fires fast as single positive signal
     → Stable, resistant to small fluctuations
     → = Core "feeling of loving you"


  ── COGNITIVE / PREDICTION CHANNEL (often present) ──
     "You are right for me"
     → Cognitive pattern-matching over life history
     → Coherence with deep life schema: "this fits"
     → Pre-verbal sense of fit


  TOTAL: 9+ channels firing simultaneously.


  PFC COMPRESSION:
    9+ channels → single word "love" (one syllable)
    Compression ratio: 9:1 at minimum
    Information loss: ~80-90%
    Functionality preserved: 100% (body knows; system works)


  WHY "WHY DO YOU LOVE THEM?" IS HARD:
    Q: "Why do you love them?"
    A attempts:
      → "Because they're beautiful" — reductive: love them without beauty, still love
      → "Because they're brilliant" — reductive: many brilliant people not loved
      → "Because they understand me" — closer but still partial
      → "Because they are them" — closest HONEST answer

    "Because they are them" = acknowledge compression, don't force false decomposition.

    This is NOT romantic mysticism. This is a COMPRESSION PHENOMENON.
    PFC cannot retrieve a 9-channel state perfectly.
    Body knows. PFC approximates.


  WHY LOVE FEELS "INEFFABLE":
    Framework answer: love = 9+ channels converged.
    Single word compresses them all.
    Decomposition is always lossy.
    → "Ineffable" = correctly acknowledging the compression limit.


  PRACTICAL IMPLICATION:
    → Don't demand perfect verbal justification for deep feelings
    → "Because they are them" is intellectually honest
    → Trust the compiled valence over Feel-Explanation attempts
    → BUT: verify channels over time
      Is meaning still firing? Still agent input? Still mirror?
      Lost channels = weakening compile — even if the word "love" persists


  CONTRAST CASE: "Hate" Decomposed
    Similar multi-channel structure, negative valence:
    + L0: threat (this person is dangerous to me)
    + L1: body tenses when near
    + Agent input: NEGATIVE (drains, doesn't fill)
    + Mirror: reads hostility back
    + Preview: future negative (dread)
    + Schema: "this person is bad for me"
    + Compiled valence: strongly negative
    → "Hate" = same compression phenomenon
    → Love and hate are both multi-source compressed states
    → Both feel "ineffable" for the same reason


  OTHER FRAMEWORK EXAMPLES:

    "Missing someone gone" = agent input + schema + memory + valence
      = 4+ channels compressed

    "Afraid of something irrational" = L0 + schema (trauma) + preview
      = 3 channels compressed

    "Jealous of a friend" = status (evaluative) + compiled valence + schema
      = 3 channels

    "Compassion for yourself" = mirror (reflexive) + meta-feeling + evaluative
      = 3 channels

    "Proud of your child" = meaning (evaluative) + mirror + agent input + schema
      = 4 channels


  GENERALIZED PRINCIPLE:
    If a feeling is hard to explain → multi-channel confirmed
    If a feeling is easy to explain → usually single-channel dominant
    Emotional intelligence = ability to notice which channels are firing
    Wisdom = trust the compiled signal + verify channels over time
```
