### §1.1 — [Feeling.md](../Feeling.md) (1262 lines) notes

**Role**: Foundation file — defines Feeling + 7-layer structure + Core Loop +
Evolutionary framing.

#### §1.1.1 — Formal Definition (Feeling.md §1.1)

> **FEELING** = feedback signal that has been **INTEGRATED** about the state of body
> systems (body-base (L0+L1) + agent input + mirror + schema + imagine-final + valence
> + cognitive prediction) → **VERBALIZABLE** at PFC level.
>
> Condensed: Feeling is HOW the body makes itself **noticeable** to PFC.

**4 key phrases**:
1. "feedback signal" — feedback, not creating new information. Body = source, feeling
   = messenger.
2. "has been INTEGRATED" — unified, not a list of sensations. Multi-source → single
   stream.
3. "state of body systems" — multi-source (many systems firing simultaneously).
4. "VERBALIZABLE at PFC level" — PFC must be able to interact with it (observe,
   locate, label, explain — imperfectly).

**Formula**: `Body fire + Integrated + PFC-noticeable = FEELING`

**Critical disclaimer (§1.2)**: Feeling ≠ Emotion (narrow). Emotion (joy/sadness/
anger/fear) is only **1 subset** of feeling. Feeling encompasses:
- Body-physical feelings ("hungry", "numb leg")
- Emotional narrow ("happy", "sad")
- Social reading ("feel that the other person doesn't trust me")
- **Cognitive/prediction** ("feel like this answer is right", "feel like something is
  wrong") ← **Theme D key**
- Predictive/intuition ("feel like I shouldn't trust this person")
- Valence ("like that tree", "love her")
- Schema-triggered ("feel that the Virgin Mary is protecting me", "feel peaceful
  arriving home")

→ **All 7 types = same feeling system**. Mainstream only calls the emotional narrow
type "emotion".

#### §1.1.2 — 3 Critical Characteristics (§1.3)

1. **MULTI-SOURCE**: Feeling = convergence of many body systems firing simultaneously.
   E.g. "love her" = 9+ channels.
2. **INTEGRATED**: Not a list. Integration at insula + ACC + VMPFC (🟢 established).
3. **VERBALIZABLE**: Does NOT mean "has a ready word". Can be "feeling somehow",
   "something feels off", "don't know but feeling..." = felt sense (Gendlin).

#### §1.1.3 — 7-Layer Structure (§2) — **CRITICAL for Theme A**

```
Feel-Explanation: EXPLANATION    "Why does this feeling exist"    20-70% fidelity
Feel-Labeling: LABELING          "What is this feeling called"    40-80%
Feel-Location: LOCATION          "Where in the body"              70-90%
Feel-Observation: OBSERVATION    "Something is happening"         85%
═══════════════════════════════════════════════════════════════════
Feel-Consciousification: CONSCIOUSIFICATION "Body state becomes noticeable" 90%
─────────────────────────────────────────────────────────────────
Feel-Integration: INTEGRATION    Insula + ACC + VMPFC              95%
Feel-RawSignals: RAW SIGNALS     Multi-system fire                 100%
```

**Key insight**: Fidelity DECREASES going up. Feel-RawSignals = 100% body truth;
Feel-Explanation = PFC's best guess.

**Feel-RawSignals–Feel-Consciousification = BODY-based** (pre-PFC or PFC-adjacent)
**Feel-Observation–Feel-Explanation = PFC PROCESSING** of feeling (high fidelity-loss
zone)

**Critical quote for Theme A**:
> "Wise people TRUST Feel-Consciousification–Feel-Observation (body noticeable, not
> yet labeled). Naive people TRUST Feel-Labeling–Feel-Explanation (label +
> explanation). Training (meditation, focusing) = shifting attention from
> Feel-Explanation BACK to Feel-Consciousification–Feel-Observation."

**Feel-Consciousification = "felt sense" (Gendlin)**:
> "Feel-Consciousification is the point where the body has finished computing — but
> PFC has NOT yet applied any label. 'Something', 'feels somehow', 'don't know but
> feel...' = TRUTH already ready. PFC doesn't have words yet."

**Feel-Labeling compression**:
> "Multi-source feeling → 1 word. Compression ratio: can be 9:1 or more. A great
> deal of information is LOST at this layer. 'Love her' → 9+ channels compressed to
> 1 morpheme. Lose 80%+ information."

**Feel-Explanation risk**:
> "Feel-Explanation = PFC narrative-making engine activated. Confabulation risk HIGH.
> Rationalization common. Displacement frequent. E.g.: 'I'm angry at my husband for
> not washing the dishes' → Actually: tired + hungry + stressed → any stimulus would
> trigger anger → PFC creates a post-hoc story."

#### §1.1.4 — Feeling Loop (§3) — **CRITICAL for Theme A**

```
BODY SYSTEMS fire (body-base + mirror + schema + imagine-final + valence + ...)
  ↓
INTEGRATION (insula + ACC + VMPFC)
  ↓
FEELING (Feel-Consciousification: becomes noticeable)
  ↓
PFC OBSERVE (Feel-Observation–Feel-Explanation: locate, label, explain)
  ↓
DRIVE (motivational direction)
  ↓
ACTION
  ↓
NEW BODY STATE → loop continues
```

**Key quote**:
> "Feeling loop = MOTOR of meaningful action. No feeling = no PFC motivation to act
> (Damasio VMPFC patients: reasoning intact, decision broken)."

#### §1.1.5 — §5 Evolutionary Sufficiency — **Relevant Theme A + D**

**Ancestral 2M years**: Binary feeling sufficient (~20-50 feelings). Simple
environment, body-environment aligned, feeling signals reliable, PFC role minimal.

**Modern era 5 mismatches**:
1. Environment mismatch (1 stress signal → 8+ modern causes)
2. Decision timescale mismatch (seconds → decades)
3. Chronic vs. acute mismatch (cortisol damage)
4. Social complexity explosion (Dunbar 150 → 1000s weak ties)
5. Information overload (distant threats fire same circuits)

**Modern PFC role EXPANDED**: Observer → Interpreter → Calibrator → Mismatch
detector → Meta-feeling.

**Framework positioning**: Feeling framework = **INTERPRETER BRIDGE** between ancestral
body + modern environment. Body is still right — it is simply speaking an ancient
language that PFC needs to learn to read.

#### §1.2.2 — §6 Mirror-Generated Feelings — **CRITICAL for Theme E**

**Mechanism** (per Domain-Mechanisms.md §1):
> "Mirror feelings do NOT come from a 'separate empathy system'. Mirror = 3 mechanisms:

> ② Agent modeling (mirror proper, 6-24m+)

**Key quote**:
> "Feelings from mirror = OWN body firing AS IF experiencing what the other person

**Mirror reward/punishment**:
- "Seeing someone else suffer → I suffer" (ventral empathy fire)
- "Seeing someone else happy → I am rewarded" (vicarious reward)
- "Seeing a baby smile → warm feeling" (CARE system activation)

**Mirror misattribution** (common error):
> "Enter room, catch group anxiety → feel anxious → PFC: 'I must be anxious about
> something' → Actually: mirroring others. Not a bug — mirror evolved for
> co-regulation."

**Research**: Gallese 2007 embodied simulation, Rizzolatti 1996 mirror neurons,
Ambady 1993 thin slices, Hatfield 1994 emotional contagion, Singer 2004 pain empathy.

#### §1.2.3 — §7 Schema-Triggered Feelings — **CRITICAL for Theme D**

**Key claim**:
> "**BODY CANNOT DISTINGUISH SCHEMA vs. REAL INPUT**. Body fires when schema fires.
> Body fires when real input fires. SAME physiological response. → Schema-generated
> feeling is PHYSICALLY REAL."

**Examples**:
- "Thinking about someone who has passed" → body grieves (schema fire, real grief
  response)
- "Thinking about a deadline" → body stresses (schema fire, real cortisol)
- "Reading scripture" → body reaches peaceful state (schema fire, real
  parasympathetic)

**Schema-triggered pain (psychogenic)**:
> "Psychogenic pain = real pain, not 'fake'. Mechanism: schema fires pain circuits.
> Same neural pathways as 'real' pain."

**Accuracy subtlety**:
> "Schema fires based on pattern match — but pattern match can be WRONG. Current
> situation ≠ past situation exactly. Schema fires reactive feeling anyway. Person
> reacts to the past, not the present. → PTSD, triggered responses, prejudice."

### §1.3 — [Feeling-Accuracy.md](../Feeling-Accuracy.md) (1874 lines) notes

**Role**: HOW PFC reads feelings, error modes, pain special case, literacy training.

#### §1.3.1 — §1 Temporal Order Invariant — **CORE for Theme A**

**The core invariant**:
> "Body computes → Feeling emerges → PFC observes. This order is **NEVER REVERSED**.
> PFC does NOT generate feeling. PFC cannot create body state."

**4 evidences**:
1. **Libet research** 🟢: Unconscious brain activity precedes conscious decision by
   300-500ms
2. **Damasio VMPFC patients** 🟢: VMPFC damage → logic + memory + language intact,
   feeling-based decision LOST → paralyzed
3. **Interoception** 🟢: Insula integrates body state BEFORE cortical processing.
   PFC does NOT access raw body signals directly.
4. **Gendlin felt sense** 🟢: 50+ years clinical — "felt sense cannot be generated
   by thinking. Must be ACCESSED, not created."

**Implications**:
- A. "Think yourself into feeling" = **impossible**. Can trigger schema → schema
  triggers body state (indirect). But cannot directly will feeling into existence.
- B. PFC can INHIBIT feeling expression, not generate its source.
- C. Awareness matters: Body computes whether PFC notices or not.

→ **CRITICAL for Theme A**: Framework claim is clear — Feeling is fully body-based,
PFC OBSERVES. Answer to user's meta-question: NO, we are NOT forcing feeling into
body-base — the framework's stance is that feeling GENUINELY belongs to body-base,
PFC observes + (indirectly) triggers via schema.

**Philosophical note**:
> "Hard problem of consciousness still open. Framework does NOT claim to solve it.
> Framework claims: at FUNCTIONAL level, body-first is observable."

#### §1.3.2 — §3 6 PFC Error Modes — **Theme A + Theme E relevant**

1. **RATIONALIZATION** — Body fires, PFC creates plausible story. E.g.: "Feel
   irritated → coworker is annoying" vs. actually "hungry + slept badly + over-
   stimulated". Nisbett & Wilson 1977 🟢.

2. **MISATTRIBUTION** — Physical ↔ emotional label swap. E.g.: hungry → "angry"
   (hangry), anxiety → "stomach problems". Schachter & Singer 1962 🟢.

3. **DISPLACEMENT** — Feeling about A → directed at B (safer target). E.g.:
   "Stressed from boss → yell at spouse". ← **Theme E relevance**: "When helping
   feels uncomfortable" can be displacement.

4. **OVER-COMPLEX LABELING** — Simple body need → grand narrative. E.g.:
   "existential dread" when actually bad sleep + skipped breakfast. Intellectual trap
   for the over-educated.

5. **UNDER-LABELING / DENIAL** — Strong feeling → "I'm fine". Alexithymia +
   cultural training (stoic, male socialization).

6. **CULTURAL SCRIPT OVERRIDE** — Actual feeling overridden by culturally acceptable
   label. E.g.: men "tender longing" → forced into "physical want". Barrett
   granularity insight.

**Root causes** (§3.7):
- Compression mismatch (multi → single)
- Linguistic structure (SVO grammar forces single-cause)
- Cultural script limits
- Self-concept protection
- Social pressure to explain
- Single-cause bias (evolutionary efficiency)
- Temporal myopia
- Agency preservation
- Feel-Explanation bias

#### §1.3.3 — §4 Pain Special Case — **relevant Theme D (if pain as right/wrong
signal)**

**4.1 Location accuracy**:
- **Somatic pain** (skin, muscles, joints): HIGH accuracy (well-mapped somatosensory
  cortex, precise within cm)
- **Visceral pain** (organs): LOW accuracy (convergent afferents → referred pain:
  heart attack → left arm, appendicitis → initially periumbilical, diaphragm →
  shoulder)

**4.3 Emotional pain = physical pain circuits**:
> "Eisenberger 2003: social pain activates ACC, same as physical pain. Subsequent:
> rejection, grief, heartbreak all real pain circuits. 'Broken heart syndrome'
> (Takotsubo) is real."

**4.4 Psychogenic pain is REAL pain**:
> "'Psychogenic' does NOT mean 'fake'. Same neural pathways as 'real' pain."

→ Theme D connection: "Feeling that something is wrong" might trigger same circuits.
Both wrongness feelings (body Feel-Consciousification) + cognitive error detection
(ACC) = body signals.

#### §1.3.4 — §5-7 Feeling Literacy Training

**Definition**: Feeling literacy = skill of accurately reading own feelings from body
signal to explanation, with calibrated confidence per layer.

**Training methods**:
- Meditation / Mindfulness (Vipassana, MBSR) — Farb 2013 🟢 improved interoceptive

- **Focusing (Gendlin)** — 6 steps, clinical effectiveness 50 years
- Body awareness practices (yoga, Feldenkrais, Alexander)
- Therapy modalities (EFT, IFS, AEDP, Hakomi)

**Gendlin's KEY observation**:
> "Those who paused and CHECKED with body sense ('Hmm, wait... it's more like...')
> improved more. Those who rationalized quickly did NOT improve."

→ Direct evidence for body-first + skepticism about Feel-Labeling–Feel-Explanation
labels.

### §1.5 — [Self-Pattern-Modeling.md](../Self-Pattern-Modeling.md) (377 lines,
OUTLINE status) notes

**Status**: Phase 1 OUTLINE, content not yet filled. But outline already reveals
critical architectural insights.

**Core concept**: Self-Pattern Match = **intrapersonal mechanism** — PFC observes
own internal state by activating own learned pattern chunks.

**Key distinction from Resonance** (sibling):
- Self-Pattern Match = library used to **LABEL own state** (inward)
- Resonance = library used to **SIMULATE other's state** (outward)
- Same chunk library, 2 different applications

**Central claim**:
> "Self-Pattern Match may be the **core mechanism of Feeling itself**, not a separate
> channel."

**Core definition**:
> "Intrapersonal mechanism by which PFC observes own internal state (body +
> unconscious processing) through activation of self-built pattern chunks."

**What it is NOT**:
- NOT direct raw body sensing (PFC has no raw access)
- NOT external influence
- NOT automatic hardware trigger (requires learned chunks first)
- NOT single-shot (stage-gated, state-gated, probabilistic)

**Mechanism** (§2):
1. Input: channels feed signals upstream
2. Chunk library: built through experience + repetition, operant + classical
   conditioning, cross-channel scaffolding
3. Match operation: fuzzy pattern matching, **state-gated** (current state modulates
   activation), **near-miss triggers strongest** (predictive coding amplifies
   dissonance), exact match weakest
4. PFC observation: read access to integrated outputs (Feel-Consciousification–
   Feel-Observation "felt sense")
5. Labeling: chunks provide labels; missing chunks → diffuse feeling ("vaguely happy
   without knowing why"); rich chunks → specific verbalization

**8 developmental stages** (§3):
- Stage 0 Crude Distress (0-3m)
- Stage 1 Behavioral Discrimination (3-12m)
- Stage 2 Body Location (12-24m)
- Stage 3 Verbal Labels Primitive (2-3y)
