## §2 — LOGIC WRONG: PARADIGM SHIFTS IN HISTORY

Every case below shares 1 pattern:
- Logic chain **coherent** at that moment
- **Supporting** evidence real (correlation)
- Community **consensus** (social validation)
- **STILL WRONG** — because scope was not wide enough

### ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
### CASE L1: THE SUN ORBITS THE EARTH
### (Geocentric → Heliocentric)
### ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

① CONTEXT:
  Ptolemy (2nd century) built the geocentric model.
  This model dominated for ~1,400 years.

② WHAT LOGIC SAYS:
  → Look up: the sun MOVES from east to west (observation CORRECT)
  → Earth: DOES NOT FEEL like it is moving (observation CORRECT)
  → Ptolemy added epicycles → predicted planetary positions ACCURATELY
  → Logic chain: observation + math model + prediction accuracy = "correct"

③ REALITY:
  Copernicus (1543): heliocentric model — simpler, explains better
  Galileo (1610): telescope → sees Jupiter's moons orbiting Jupiter
  Kepler: elliptical orbits → predicts MORE ACCURATELY
  Newton: gravity → MECHANISM explains why

④ PATTERN:
  → Logic correct WITHIN the scope of naked-eye observation
  → Wrong when scope EXPANDED (telescope, more precise measurement)
  → 1,400 years coherent + wrong
  (Logic-Feeling-Balance.md §5.1: "Logic only sees WITHIN current scope")

  🟢 Kuhn 1962: Structure of Scientific Revolutions — paradigm shift archetype

### ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
### CASE L2: DISEASE FROM TOXIC AIR
### (Miasma → Germ Theory)
### ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

① CONTEXT:
  From Hippocrates (5th century BCE) to the late 19th century.
  Miasma theory: disease comes from "bad air" produced by decaying organic matter.
  Dominated for ~2,300 years.

② WHAT LOGIC SAYS:
  → Foul-smelling places = more disease (correlation CORRECT — because dirty places also have germs)
  → Swamps → malaria (malaria = "mal aria" = "bad air" — the disease name
    preserved the flawed logic in language)
  → Avoid foul smells → fewer illnesses (works — because avoiding foul = avoiding germ-heavy places)
  → Logic chain: observation + correlation + practical result = "correct"

③ REALITY:
  Pasteur (1860s): bacteria cause disease, not air
  Koch (1876): proved Bacillus anthracis causes anthrax — ended miasma
  Semmelweis (1847): handwashing reduced mortality from ~18% to ~2%
    → Rejected by colleagues, dismissed from position
    → Died in a psychiatric hospital (1865)
    → Decades later recognized as correct

④ PATTERN:
  → Correlation (foul smell → disease) CORRECT — but mechanism WRONG
  → Logic failed by confusing correlation with causation
  → Semmelweis: evidence CLEAR (9× mortality reduction) but STILL REJECTED
    because it did not fit the current framework
  (Logic-Feeling-Balance.md §5.2: "Correlation ≠ Mechanism")

  🟢 Pasteur 1860s: germ theory experiments
  🟢 Semmelweis 1847: handwashing data (mortality 18% → 2%)
  🟢 Koch 1876: anthrax proof — definitive end of miasma

### ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
### CASE L3: OBJECTS BURN BY RELEASING "PHLOGISTON"
### (Phlogiston → Oxygen)
### ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

① CONTEXT:
  Stahl (1697) proposed: objects burn by releasing "phlogiston" (fire substance).
  Dominated chemistry for ~100 years.

② WHAT LOGIC SAYS:
  → Wood burns → ash is lighter than wood → "phlogiston was released" (observation fits)
  → Heated metal → calx (oxide) → "phlogiston escaped" (consistent)
  → Explained MOST combustion phenomena observed
  → Chemistry community consensus ~100 years

③ REALITY:
  Lavoisier (1770s): WEIGHED metals before and after heating
  → Metal GAINED weight after burning — opposite of "losing phlogiston"
  → Discovered oxygen: burning = combining with oxygen, not releasing anything
  1783: Lavoisier declared "phlogiston is an imaginary substance"
  ~1800: most chemists accepted oxygen theory

④ PATTERN:
  → Logic explained phenomena (lighter ash) but MISSED other phenomena
    (heavier metal)
  → 100 years with no precise measurement → contradiction not discovered
  → 1 simple measurement (weighing before/after) → overturned the entire theory
  (Logic-Feeling-Balance.md §5.1: scope limit — cannot see beyond scope)

  🟢 Lavoisier 1770s-1783: oxygen theory
  🟢 Stahl 1697: phlogiston theory (dominated ~100 years)

### ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
### CASE L4: ULCERS CAUSED BY STRESS
### (Stress → H. pylori)
### ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

① CONTEXT:
  For most of the 20th century: medical consensus was that peptic ulcers were caused by stress + acid.
  Treatment: reduce stress, acid-reducing medication, surgical stomach removal.

② WHAT LOGIC SAYS:
  → Stressed people → more ulcers (correlation CORRECT — stress does worsen them)
  → High acid → mucosal damage (mechanism PARTIALLY CORRECT)
  → Reduce acid → symptoms decrease (clinical result = confirming)
  → "No bacterium can survive in stomach acid" (logic chain coherent)

③ REALITY:
  Marshall & Warren (Perth, Australia, 1982): discovered H. pylori bacterium
  Medical community rejected: "bacteria cannot survive in acid"
  Marshall (1984): DRANK the bacteria himself → developed gastritis after 3 days
    → Endoscopy on day 8: severe inflammation, H. pylori colonizing
  2005: Marshall & Warren awarded the Nobel Prize
  → Treatment: 1-2 weeks of antibiotics — instead of surgical stomach removal

④ PATTERN:
  → "Bacteria cannot survive in acid" = logic correct WITHIN the understanding of the time
  → Reality: H. pylori HAS a mechanism to neutralize acid (logic did not know this)
  → Decades of unnecessary surgeries
  → 1 experiment (self-ingestion) → overturned consensus
  (Logic-Feeling-Balance.md §5.2: correlation ≠ mechanism)

  🟢 Marshall & Warren 1982-1984: H. pylori discovery
  🟢 Nobel Prize 2005: "for discovery of bacterium H. pylori and its role in gastritis and peptic ulcer disease"

### CASE L5: BLOODLETTING AS TREATMENT
### (Bloodletting — Humoral Theory)
### ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

① CONTEXT:
  From ancient Greece to the 19th century.
  Humoral theory: disease from imbalance of 4 humors (blood, yellow bile, black bile, phlegm).
  Bloodletting = "rebalancing" → common medical practice for ~2,000 years.

② WHAT LOGIC SAYS:
  → Fever → flushed face → "too much blood" → drain some (logic chain fits observation)
  → After bloodletting: patient weakened → "calmer" → "improved"
    (reality: weak from blood loss, misread as "calmer")
  → 2,000 years of practice → "tradition proves it works"

③ REALITY:
  George Washington (1799): severe throat inflammation
  → Doctors removed ~80 ounces of blood (~2.4 liters) in 12 hours
  → = ~40% of total body blood volume
  → Washington died — largely from excessive blood loss
  → Modern medicine: bloodletting CAUSES HARM, does not cure disease
    (except in rare cases like hemochromatosis)

④ PATTERN:
  → 2,000 years of practice = evidence "it works" (survivorship bias: only remember those who survived)
  → Logic fit observation (flushed face → too much blood) but mechanism COMPLETELY WRONG
  → Consequence: countless people died from "treatment"
  → Strongest case for: coherent logic + long practice ≠ correct

  🟢 Washington's death 1799: ~80 oz blood removed, ~40% total blood volume
  🟢 Humoral theory: dominated Western medicine ~2,000 years

### CASE L6: LOBOTOMY TO TREAT MENTAL ILLNESS
### (Lobotomy — Nobel Prize 1949)
### ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

① CONTEXT:
  Egas Moniz (Portugal) proposed leucotomy (lobotomy) for mental illness.
  Awarded the Nobel Prize in Physiology or Medicine 1949.

② WHAT LOGIC SAYS:
  → Brain area causing problems → cut/destroy → problem gone (simple logic chain)
  → Patients after lobotomy: "calmer" (observation correct — because brain was damaged)
  → Nobel Prize = highest validation in science
  → Moniz reported: "acceptable" results in first 40 patients

③ REALITY:
  ~20,000 lobotomies in the US (through 1951), higher rate than in the UK
  Patients: lost concentration, lost emotion, lost personality
  Moniz criticized: understated complications, did not follow up on patients
  Decades later: recognized as harmful — lobotomy now nearly completely banned
  Many patients' families called for the Nobel Prize to be revoked

④ PATTERN:
  → Logic correct at a LOW level: cutting neural connections = reducing signal (mechanism correct)
  → Logic wrong at a HIGH level: brain ≠ machine — cutting 1 part ≠ "fixing"
  → Nobel Prize = EXTREMELY STRONG social validation → reinforced the error
  → "Calmer" = correct observation, wrong interpretation
    (calm because of damage, not because of "cure")

  🟢 Moniz 1949: Nobel Prize for leucotomy
  🟢 ~20,000 lobotomies in US by 1951
  🟢 Now recognized as harmful; calls to revoke Nobel Prize

### CASE L7: CONTINENTS CANNOT MOVE
### (Rejection of Continental Drift → Plate Tectonics)
### ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

① CONTEXT:
  Wegener (1912): proposed that continents were once joined and had drifted apart.
  Evidence: matching coastline shapes, similar fossils on distant continents.
  Rejected by the geological community — especially in North America.

② WHAT THE REJECTORS' LOGIC SAYS:
  → "Rock cannot move" (correct within the understanding of the time)
  → "No mechanism strong enough to push continents" (correct — mantle convection not yet known)
  → "Wegener is a meteorologist, not a geologist" (ad hominem but powerful)
  → One paleontologist called it "German pseudo-science"

③ REALITY:
  1950s: paleomagnetism data → support Wegener
  1953: samples from India → confirm it was once in the Southern Hemisphere (exactly as Wegener predicted)
  1960s: plate tectonics theory — mechanism (mantle convection) discovered
  Wegener: died in 1930 on a Greenland expedition, never lived to see his vindication

④ PATTERN:
  → Logic rejected because "no mechanism" — correct that none was known, but WRONG to conclude
    "no mechanism = does not happen"
  → Evidence (matching shapes, fossils) IGNORED because it did not fit the framework
  → 50 years of delay because logic-of-the-time had insufficient scope
  (Logic-Feeling-Balance.md §5.1: logic only sees within current scope)

  🟢 Wegener 1912: continental drift hypothesis
  🟢 1960s: plate tectonics confirmed
  🟢 Wegener died 1930: never saw vindication

### CASE L8: LIFE SPONTANEOUSLY GENERATES FROM DEAD MATTER
### (Spontaneous Generation → Biogenesis)
### ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

① CONTEXT:
  Since Aristotle: life can spontaneously arise from non-living matter.
  "Meat left out → produces flies." This was "scientific fact" for ~2,000 years.

② WHAT LOGIC SAYS:
  → Meat left out → flies APPEAR (observation CORRECT)
  → No flies seen arriving to lay eggs (observation CORRECT — eggs are tiny, hard to see)
  → Conclusion: flies generate spontaneously from meat
    (logic chain: A → B observed, C not observed → A causes B)

③ REALITY:
  Redi (1668): covered meat with gauze → no maggots in meat
  → Flies laid eggs ON the gauze, could not get through → maggots on gauze, not on meat
  Pasteur (1859): swan-neck flask → boiled water did NOT become contaminated
  → Break the neck → bacteria enter → water turns cloudy
  → Proved: organisms come from OUTSIDE, do not generate spontaneously

④ PATTERN:
  → Observation correct (meat → maggots) but MISSED hidden variable (flies laying eggs)
  → 2,000 years: nobody TRIED covering the meat — because "we already know"
  → 1 simple experiment (covering with gauze) → overturned 2 millennia of logic

  🟢 Redi 1668: meat experiment with gauze
  🟢 Pasteur 1859: swan-neck flask experiment

### CASE L9: LIGHT NEEDS A MEDIUM TO TRAVEL
### (Luminiferous Ether → No Medium Needed)
### ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

① CONTEXT:
  19th century: light had been shown to be a wave.
  Logic: waves require a medium (sound needs air, water waves need water).
  → "Luminiferous ether" must exist — the medium for light.

② WHAT LOGIC SAYS:
  → Sound waves need air (CORRECT)
  → Water waves need water (CORRECT)
  → Light = wave → needs medium = ether (analogy logic SOUND)
  → Ether must fill the universe, weightless, invisible

③ REALITY:
  Michelson-Morley (1887): experiment measuring speed of light in 2 directions
  → If ether exists → light speed MUST DIFFER by direction
  → Result: NO difference → ether does not exist
  Einstein (1905): special relativity — light needs no medium
  1920s: physics community accepted — ether abandoned

④ PATTERN:
  → Logic by analogy: "A needs medium, B needs medium, C is like A and B → C needs medium"
  → Analogy = powerful tool BUT does not guarantee truth
  → Light ≠ mechanical wave → analogy BREAKS at a fundamental level
  → 1 experiment (Michelson-Morley) → destroyed the underlying assumption

  🟢 Michelson-Morley 1887: null result for ether
  🟢 Einstein 1905: special relativity — no medium needed

### CASE F1: EATING SUGAR — STRONG BODY REWARD, WRONG FOR DOMAIN
### (Evolution Lag — Intensely pleasant)
### ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

① CONTEXT:
  Humans eat sweets → clear body reward: "tasty," "pleasant," "want more."
  This is a universal experience — all cultures, all eras.

② WHAT FEELING SAYS:
  → VTA fires positive prediction-delta when sugar touches taste receptors
  → Opioid release → "genuinely pleasant" (STRONG liking signal)
  → Dopamine → "seek more" (STRONG wanting signal)
  → Body says: "THIS IS GOOD. GET MORE."

③ REALITY:
  Ancestral environment: sugar was EXTREMELY RARE (only seasonal fruit, honey)
  → Body rewarding "find sugar" = REAL survival advantage
  Modern environment: sugar is EXTREMELY EASY TO GET (everywhere, cheap)
  → Body STILL rewards as if sugar is scarce → overconsume → obesity, diabetes

  Humans are running a reward system firmware ~200,000 years old
  in an environment ~200 years new.
  = Body has not yet updated.

④ PATTERN:
  → Feeling CORRECT about mechanism (body needs energy → sugar = fast energy)
  → Feeling WRONG about context (sugar is no longer scarce → reward disproportionate)
  → Body has NO mechanism to self-detect "the environment has changed"
  (Why-Body-Knows.md §6: Evolution Lag)

  🟢 Evolutionary mismatch: PMC (Frontiers in Psychiatry 2018) — sugar addiction
      from evolution to revolution
  🟢 5/11 substance use disorder criteria met for sugar
  🟢 Berridge: opioid system responds to sugar similarly to drugs

### CASE F2: BEING PROVOKED — BODY FIRES IN THE RIGHT DIRECTION, WRONG MAGNITUDE
### (Magnitude Mismatch — Extreme anger)
### ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

① CONTEXT:
  Someone deliberately provokes you → body reacts immediately: heart racing,
  face flushing, muscles tensing, urge to react.

② WHAT FEELING SAYS:
  → Amygdala: THREAT detected → cortisol + norepinephrine
  → Body: "DANGER. REACT NOW."
  → Intensity: STRONG — proportional to ancestral threat level
  → Wants: to respond, strike back, defend position

③ REALITY:
  Ancestral environment: being provoked = challenge to social standing
  → Losing position = losing resource access = SURVIVAL danger
  → Body fires STRONGLY = reasonable (proportional response for survival)

  Modern environment: being provoked = discomfort
  → Just walking away = sufficient (no one dies from being teased)
  → Body STILL fires at "survival" intensity for a "discomfort" situation
  → Overreaction → regret → unnecessary social cost

④ PATTERN:
  → Feeling CORRECT in direction (genuine threat — someone deliberately causes harm)
  → Feeling WRONG in magnitude (ancestral intensity for a modern context)
  → Body has NO mechanism to auto-scale magnitude to context
  → PFC arrives AFTER body ~200ms → window in which overreaction occurs
  (Logic-Feeling-Balance.md §3.5: magnitude mismatch)

  🟢 Damasio: body fires T=0ms, PFC observes T=200ms+
  🟢 Amygdala response: ~12ms (LeDoux 1996) — faster than PFC

## §4 — FEELING WRONG: SUBTLE CASES

### ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
### CASE F3: HEARING SOMEONE ELSE'S SUCCESS STORY
### (External Inject — Subtly pleasant)
### ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

① CONTEXT:
  You encounter someone telling you about their wonderful job and life.
  May be a friend, colleague, or someone on social media.
  The story is good, authentic, inspiring.

② WHAT FEELING SAYS:
  → Mirror: body simulates STORYTELLER's state → "feels good"
  → Imagine-Final: PFC previews "if I did the same..." → body pre-feels positive
  → Schema: links to existing aspirations → "matches my dreams"
  → Valence: combined → positive → "I want this"
  → Multiple channels fire simultaneously → feeling STRONG, CONVINCING

③ REALITY:
  → THEIR story ≠ YOUR domain
  → They have different context, chunks, history → their path was calibrated FOR THEM
  → Your body checks coherence with THE STORY (coherent!)
    but DOES NOT check against YOUR OWN DOMAIN
  → Imagine-Final formed from Self-Referencing — body checking internal loop
  → Anchor-Schema locks → PFC spotlight narrows → alternatives not scanned
  → Possibly: a DIFFERENT path fits better — but it has already been overlooked

④ PATTERN:
  → There is nothing "obviously wrong" here:
    good story, genuine emotion, reasonable logic, multiple channels confirm
  → Only 1 problem: NOT CERTAIN IT FITS YOU
  → Subtle because: no red flag, no dissonance
  → Only domain feedback (TRY → result) can reveal it
  (Anchor-Schema.md §3 ④: External Inject bypasses PFC)
  (Feeling.md §8: Domain-Checked vs Self-Referencing)

### ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
### CASE F4: DISCOMFORT WHEN RECEIVING FEEDBACK FROM SOMEONE WITH EXPERTISE
### (Recalibration Discomfort — Subtly uncomfortable)
### ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

① CONTEXT:
  You are working on something — a job, a creative project, a course of study.
  Someone with expertise (mentor, senior, specialist) gives negative feedback:
  "This approach isn't right." "This section needs reworking." "You're heading in the wrong direction."

② WHAT FEELING SAYS:
  → Amygdala: threat to self-schema → mild cortisol
  → Status shift: "my position is being challenged" → discomfort
  → Body: "DISLIKE THIS. WANT TO PUSH BACK. OR RETREAT."
  → PFC Feel-Labeling—Feel-Explanation may:
    - Rationalize: "they don't understand my context"
    - Displacement: "they just want to show off"
    - Denial: "the feedback doesn't matter"
  → Feeling: mild discomfort, wanting to dismiss, wanting to defend what one is doing

③ REALITY:
  → If dismissed immediately by feeling → miss VALUABLE information
  → A person with expertise = larger chunk library in that domain
  → Their feedback CAN BE correct — based on patterns you DO NOT YET HAVE

  BUT (genuine complexity):
  → Feedback CAN ALSO BE wrong — expertise ≠ infallible
  → Or correct for the general domain but wrong for one's specific context
  → NO way to know for certain immediately

  With enough accumulated chunks (experience in the domain):
  → Body gradually shifts from "threat" to "recalibration"
  → Recognize: initial discomfort is Chunk-Shift (valence flip)
  → Good feedback → new schema forms → better direction emerges
  → Body-Feedback.md: recalibration = the 3rd source of discomfort (temporary, like muscle soreness after gym)

④ PATTERN:
  → Initial feeling: "threat" → want to reject
  → Reality: CAN BE recalibration → better direction
  → Subtle because: the discomfort is CORRECT (being genuinely challenged)
    BUT the behavior driven by feeling (rejecting) MAY miss something valuable
  → Need domain feedback (TRY applying the feedback → see result)
  (Body-Feedback.md §1: 3 real sources of discomfort — recalibration = 3rd)

## §5 — FEELING WRONG: SUPPLEMENTARY ANGLES

### ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
### CASE F5: BODY NEUTRAL — HABITUATING TO A BAD SITUATION
### (Baseline Shift — Feeling ABSENT)
### ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

① CONTEXT:
  You are in a relationship, a job, or a situation
  that has GRADUALLY become unhealthy — but the change is too slow to notice.

② WHAT FEELING SAYS:
  → Body: ... (silence. no clear signal.)
  → "It's normal." "Everyone's like this." "Nothing special."
  → No dissonance. No alarm. No motivation to change.

③ REALITY:
  Mechanism: VTA habituates → quality X compiles as BASELINE
  → When quality decreases GRADUALLY (very slowly, day by day):
    → Each small decrease → delta BELOW threshold → VTA does not fire
    → Baseline self-adjusts downward → "new normal" = lower than "old normal"
    → After 1 year: quality decreased 50% but body says "normal"
      (⚠️ 50% = illustrative, not a precise measurement)

  When you NOTICE:
  → Encountering the OLD environment again (visiting hometown, meeting old friends, changing context)
  → Body compares current vs OLD reference → delta EXCEEDS threshold → "why is it so different"
  → Or: someone OUTSIDE the situation comments → external mirror

④ PATTERN:
  → Feeling is not WRONG — feeling is ABSENT
  → Body operates correctly by mechanism (habituation, baseline shift)
  → But this mechanism DOES NOT DISTINGUISH "habituating to good" vs "habituating to bad"
  → Unlike F1-F4: here feeling says nothing → THAT is what is dangerous
  → Needs external reference (other people, different environment) to detect
  (Body-Feedback-Mechanism.md: Quality Baseline Shift + Chunk-Miss variant 2 "fade")

### ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
### CASE F6: BODY MIXED — APPROACH + AVOIDANCE SIMULTANEOUSLY
### (Conflict Signal — Feeling CONFUSED)
### ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

① CONTEXT:
  A new opportunity appears: new job, new project, new relationship, major decision.
  Both attractive and frightening. Body fires in BOTH directions simultaneously.

② WHAT FEELING SAYS:
  → Approach signal: "Exciting! Want to try!" (novelty + positive Imagine-Final)
  → Avoidance signal: "Risk! Fear of failure!" (threat + loss aversion)
  → Combined: CONFUSED. "Want it but scared." "Like it but don't dare."
  → People DISLIKE this state → want to resolve it QUICKLY
  → Typically: force a choice in 1 direction → lose information from the other

③ REALITY:
  → Mixed signal = body is ACCURATELY REFLECTING the complexity
  → The situation GENUINELY has both opportunity and risk
  → Body fires both = HONEST response
  → But people interpret "confused = wrong" → force simplification

  Two types of forced resolution, both wrong:
    → Force approach: ignore risk → may encounter problems later
    → Force avoidance: ignore opportunity → may regret later
    → BOTH types of forcing LOSE INFORMATION

④ PATTERN:
  → Feeling is not WRONG — feeling ACCURATELY reflects complexity
  → Wrong to interpret "confused = bad" → force resolution
  → Reality: confused = body is saying "genuinely complex, need more information"
  → Need: domain feedback (explore more, try small, gather data)
    rather than forcing an immediate decision
  (Drive.md §4: 4 types of drive conflict)

## §6 — BOTH WRONG: WHEN LOGIC + FEELING AGREE AND STILL FAIL

### ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
### CASE B1: BLOODLETTING — LOGIC + FEELING BOTH SAY "CORRECT"
### ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Logic: "Disease from fluid imbalance → bloodletting = rebalancing"
  Doctor's Feeling: "Patient is calmer after bloodletting" (body observes: CORRECTLY — because weakened)
  Patient's Feeling: "The doctor says so + tradition + everyone does it"
    → social validation → body: "reassured"

  Both AGREE: "bloodletting = treatment"
  Both WRONG: bloodletting CAUSES HARM

  Why both were wrong simultaneously:
  → Logic: wrong framework (humoral theory)
  → Feeling: social validation + authority + tradition = powerful inject
  → Logic confirms feeling, feeling confirms logic = mutual reinforcement
  → NOBODY checked domain feedback (how many died from bloodletting?)
  → Survivorship bias: only remembered those who survived (= "bloodletting works")

### ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
### CASE B2: LOBOTOMY — NOBEL PRIZE + "PATIENT IMPROVED"
### ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Logic: "Problem area of brain → cut → problem gone. Nobel Prize confirms."
  Doctor's Feeling: "Patient is calmer." (body observes: CORRECTLY)
  Community's Feeling: "Nobel Prize = pinnacle of science → must be right"

  Both AGREE: "lobotomy = breakthrough"
  Both WRONG: lobotomy destroys personality

  Why:
  → Logic oversimplified: brain ≠ machine (cutting one part ≠ fixing it)
  → Feeling: "calmer" = correct observation, wrong interpretation (calm because of damage)
  → Nobel Prize = ultimate social validation → both logic and feeling amplified
  → Lack of follow-up (domain feedback) → long-term consequences not visible

### ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
### CASE B3: EVERYDAY — "EVERYONE DOES IT"
### ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Situation: A decision that "everyone around agrees with" —
  may be a career path, a way of living, values, an investment, etc.

  Logic: "Many people do it → must be right. Evidence: they're doing fine."
  Feeling: "Everyone does it → body reassured (social validation). I want it too."

  BOTH CAN BE WRONG because:
  → "Many people do it" = social proof, NOT domain truth
  → "They're doing fine" = present snapshot, NOT trajectory
  → Logic: survivorship bias (only sees successes, not failures)
  → Feeling: mirror + external inject (body simulates THE STATE OF OTHERS)
  → Mutual reinforcement: logic confirms feeling → "certainly right"

  NOT always wrong:
  → Social proof is SOMETIMES right (the majority are right in many contexts)
  → But NOT ALWAYS → this is the most subtle case
  → Only YOUR OWN domain feedback can tell you whether it fits

### 7.2 Why This Pattern Cannot Be Eliminated

```
4 structural reasons:

  ① Domain is infinite → there is always something unknown
     → Logic scope is always < domain → always has blind spots
     → Feeling calibration is always < domain → always has lag

  ② Each person has different chunks → different blind spots
     → No "universal warning system"
     → My error ≠ your error

  ③ Both check COHERENCE, not TRUTH
     → Logic: coherent rules ≠ correct rules (Cases L1-L9)
     → Feeling: coherent body signal ≠ correct body signal (Cases F1-F6)
     → Coherence = necessary, not sufficient

  ④ Mutual reinforcement creates false confidence
     → Logic confirms feeling + feeling confirms logic
     → = Feeling of "certainty" INCREASES but accuracy does NOT increase
     → Cases B1-B3: both agree = MOST CONFIDENT + MOST WRONG
```

### 7.3 What CAN Be Done

```
CANNOT: eliminate error
CANNOT: know in advance which is wrong
CANNOT: create rules to avoid error

CAN:
  → RECOGNIZE that this pattern exists (this file helps with that)
  → REDUCE ABSOLUTE CERTAINTY (awareness → humility)
  → INCREASE DOMAIN FEEDBACK (try → observe → adjust)
  → ACCEPT TRIAL-ERROR (not a weakness — the only mechanism)
  → EACH PERSON SELF-BALANCES (no one can do it for you)

  (Detail: Logic-Feeling-Balance.md §6-§8)
```
